一、坐标、长度与面积里的绝对值

核心问题

点 $A(a,0)$ 在 x 轴上，$OA$ 的长度等于多少？

先把题目拆开

已知条件$A(a,0)$ 在 x 轴上
要求目标线段 $OA$ 的长度
关键词长度——长度永远非负
容易忽略$a$ 可以是负数，但 $OA$ 不能是负数

关键想法

坐标分量可正可负，但长度永远非负。所以要把坐标变成长度，必须加绝对值：
$A(a,0)$ 时，$OA = |a|$；$B(0,b)$ 时，$OB = |b|$。

标准解法

第 1 步：看具体数值

若 $A(-3,0)$，坐标分量 $a=-3$。

$OA = |-3| = 3$，不是 $-3$。

为什么：长度不可能为负数。$-3$ 到原点的距离是 $3$。

第 2 步：推广到字母

$A(a,0)$ 的坐标分量是 $a$（可能是正、负或零）。

$OA = |a|$。

同理，$B(0,b)$ 时，$OB = |b|$。

第 3 步：面积公式

一次函数与坐标轴交于 $A(a,0)$ 和 $B(0,b)$，则：

$$S_{\triangle AOB} = \frac{1}{2} \cdot OA \cdot OB = \frac{1}{2} |a| \cdot |b|$$

为什么：面积 $= \frac{1}{2} \times \text{底} \times \text{高}$，底和高都是长度，必须非负。

例题：$y = 2x - 6$ 求三角形面积

x 轴交点：令 $y=0$，$2x-6=0$，$x=3$，$A(3,0)$。

y 轴交点：令 $x=0$，$y=-6$，$B(0,-6)$。

$OA = |3| = 3$，$OB = |-6| = 6$。

$S = \frac{1}{2} \times 3 \times 6 = 9$。

易错点：$S = \frac{1}{2} a \cdot b$（错！）。坐标 $a$ 或 $b$ 为负时，面积会算成负数。必须写 $S = \frac{1}{2} |a| \cdot |b|$。

一句话总结

坐标到长度，必须加绝对值：$OA = |a|$，$OB = |b|$，$S = \frac{1}{2}|a| \cdot |b|$。

二、一次函数求坐标轴交点与面积

方法引入

一次函数 $y=kx+b$ 与坐标轴围成的三角形面积怎么算？只需要三步。

三步法：
① 令 $y=0$，解出 $x$，得到 x 轴交点 $A(a,0)$
② 令 $x=0$，代入得 $y$，得到 y 轴交点 $B(0,b)$
③ 面积 $S = \dfrac{1}{2} \cdot |a| \cdot |b|$

例题：$y = \dfrac{1}{2}x - 2$

第 1 步：求 x 轴交点

令 $y = 0$：

$\dfrac{1}{2}x - 2 = 0$

$x = 4$

所以 $A(4,\, 0)$。

为什么令 $y=0$？x 轴上所有点的纵坐标都是 0，直线与 x 轴的交点就是同时满足 $y=kx+b$ 和 $y=0$ 的点。

第 2 步：求 y 轴交点

令 $x = 0$：

$y = -2$

所以 $B(0,\, -2)$。

y 轴上所有点的横坐标都是 0，直接把 $x=0$ 代入函数即可。

第 3 步：计算面积

线段长度：

$OA = |4| = 4$，$OB = |-2| = 2$

面积：

$S = \dfrac{1}{2} \times 4 \times 2 = 4$

为什么用 $|a|$ 而不是 $a$？因为 $a$ 是坐标，可正可负；而线段长度一定非负。$A(-4,0)$ 到原点的距离也是 4。

三、一次函数面积反求参数

例题

一次函数 $y = kx - 2$ 与坐标轴围成的面积为 $5$，求 $k$。

核心流程：先求交点（含 $k$）→ 写线段长度（含绝对值）→ 列面积方程 → 解绝对值方程 → 得 $\pm k$
第 1 步：求 y 轴交点（截距已知）

令 $x=0$：$y = -2$，$B(0, -2)$，$OB = |-2| = 2$。

第 2 步：求 x 轴交点（含 $k$）

令 $y=0$：

$kx - 2 = 0 \Rightarrow x = \dfrac{2}{k}$

$A\!\left(\dfrac{2}{k},\, 0\right)$，所以 $OA = \left|\dfrac{2}{k}\right|$。

$A$ 的横坐标是 $\dfrac{2}{k}$，但 $OA$ 是距离，必须加绝对值。不管 $k$ 正负，距离都是非负的。

第 3 步：列面积方程

$S = \dfrac{1}{2} \cdot \left|\dfrac{2}{k}\right| \cdot 2 = 5$

$\left|\dfrac{2}{k}\right| = 5$

第 4 步：解绝对值方程

$\dfrac{|2|}{|k|} = 5 \Rightarrow |k| = \dfrac{2}{5}$

$k = \dfrac{2}{5}$  或  $k = -\dfrac{2}{5}$

例题 3：$y = \dfrac{k}{2}x + 2$，面积 $S = 2$

分析：$k$ 在斜率位置，截距固定

$B(0, 2)$，$OB = 2$（不含 $k$，已知量）。

令 $y=0$：$\dfrac{k}{2}x + 2 = 0 \Rightarrow x = -\dfrac{4}{k}$

$OA = \left|-\dfrac{4}{k}\right| = \dfrac{4}{|k|}$

列方程并求解

$S = \dfrac{1}{2} \cdot \dfrac{4}{|k|} \cdot 2 = 2$

$\dfrac{4}{|k|} = 2 \Rightarrow |k| = 2 \Rightarrow k = 2$ 或 $k = -2$

例题 4：$y = -2x + b$，面积 $S = 4$

分析：$b$ 在截距位置，斜率固定

$x$ 轴交点：令 $y=0$，$-2x+b=0 \Rightarrow x=\dfrac{b}{2}$，$OA = \dfrac{|b|}{2}$

$y$ 轴交点：$B(0,b)$，$OB = |b|$

列方程并求解

$\dfrac{1}{2} \cdot \dfrac{|b|}{2} \cdot |b| = 4$

$\dfrac{|b|^2}{4} = 4 \Rightarrow |b|^2 = 16 \Rightarrow |b| = 4 \Rightarrow b = 4$ 或 $b = -4$

四、一次函数与正半轴 + 角度

核心条件解读

"与 x 轴正半轴交于 $A$"：$A(a,0)$，$a > 0$。
"与 y 轴正半轴交于 $B$"：$B(0,b)$，$b > 0$。
正半轴条件意味着坐标为正，可以直接去掉绝对值！

解题流程：
① 正半轴 → $a > 0,\, b > 0$ → $OA = a$，$OB = b$（不用绝对值）
② 面积 → $\dfrac{1}{2}ab = S$
③ 角度 → 在直角 $\triangle AOB$ 中，用特殊角的 tan 找 $a$ 与 $b$ 的关系
④ 解出 $a,\, b$ 的具体值
⑤ 代入 $y = kx + b$，由 $0 = ka + b$ 求 $k$

例题 1：$S = 2$，$\angle ABO = 30°$

第 1 步：正半轴条件

$A(a, 0)$，$a > 0$；$B(0, b)$，$b > 0$。

面积：$\dfrac{1}{2}ab = 2$，即 $ab = 4$。

第 2 步：角度 → 边比

在直角 $\triangle AOB$ 中（$\angle AOB = 90°$）：

$\tan \angle ABO = \dfrac{\text{对边 } OA}{\text{邻边 } OB} = \dfrac{a}{b}$

$\tan 30° = \dfrac{\sqrt{3}}{3}$，所以 $\dfrac{a}{b} = \dfrac{\sqrt{3}}{3}$，即 $a = \dfrac{\sqrt{3}}{3}\, b$

第 3 步：联立求解

$ab = 4$，$a = \dfrac{\sqrt{3}}{3}\, b$

$\dfrac{\sqrt{3}}{3}\, b^2 = 4 \Rightarrow b^2 = 4\sqrt{3} \Rightarrow b = 2\sqrt[4]{3}$

$a = \dfrac{2\sqrt[4]{3}}{\sqrt{3}} = \dfrac{2}{\sqrt[4]{3}}$

第 4 步：求 $k$

直线过 $(0, b)$，所以截距就是 $b$。又过 $(a, 0)$：

$0 = ka + b \Rightarrow k = -\dfrac{b}{a} = -\sqrt{3}$

所以 $k = -\sqrt{3}$，$b = 2\sqrt[4]{3}$。

例题 2：$S = 4$，$\angle OAB = 45°$

正半轴 + 面积

$A(a,0),\, a>0$；$B(0,b),\, b>0$。$\dfrac{1}{2}ab = 4$，即 $ab = 8$。

45° → 等腰直角

$\tan \angle OAB = \dfrac{OB}{OA} = \dfrac{b}{a} = \tan 45° = 1$

所以 $b = a$。这是等腰直角三角形。

联立求解

$a^2 = 8 \Rightarrow a = 2\sqrt{2}$（$a > 0$），$b = 2\sqrt{2}$

$k = -\dfrac{b}{a} = -1$

所以 $k = -1$，$b = 2\sqrt{2}$。

易错提醒

易错点：正半轴条件不能滥用。只有题目明确说"正半轴"时，才能写 $OA = a$（不加绝对值）。如果没说，就必须写 $OA = |a|$。

一句话总结

正半轴可以去绝对值 → 角度定边比 → 面积定坐标 → 代入求 $k,b$。正半轴是"去绝对值的许可证"。

四个知识点的关系

这四个知识点是一条链：

第一节：绝对值是基础——坐标变长度必须加绝对值。

第二节：三步法是流程——令 $y=0$、令 $x=0$、$\frac{1}{2}|a||b|$。

第三节：反求参数是逆向——已知面积倒推 $k$ 或 $b$，绝对值方程出 $\pm$。

第四节：正半轴+角度是综合——正半轴去绝对值，角度定边比，面积定坐标。

四者共同训练的核心动作：**算之前先想清楚符号**。
