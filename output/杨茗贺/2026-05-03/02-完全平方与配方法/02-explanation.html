一、完全平方公式——正用与反用

核心公式

$$\boxed{(x+a)^2 = x^2 + 2ax + a^2}$$

正用：左边 → 右边，叫做展开。
反用：右边 → 左边，叫做凑平方（因式分解的一种）。

先把题目拆开

已知条件一个平方形式或一个三项式
要求目标展开结果或平方形式
关键词首² ± 2·首·尾 + 尾²
容易忽略中间项的系数是 $2 \times \text{首} \times \text{尾}$，不是 $\text{首} \times \text{尾}$

关键想法

完全平方公式只有一种结构：首² ± 2·首·尾 + 尾²。
正用时，把首和尾代入公式，中间项自动生成；
反用时，先找首平方和尾平方，再验证中间项是否匹配。

标准解法

正用例1：展开 $(x+4)^2$

首 $= x$，尾 $= 4$。

$(x+4)^2 = x^2 + 2 \cdot x \cdot 4 + 4^2 = x^2 + 8x + 16$

为什么：中间项是 $2 \times x \times 4 = 8x$，不是 $4x$。

正用例2：展开 $(x-5)^2$

首 $= x$，尾 $= -5$。

$(x-5)^2 = x^2 + 2 \cdot x \cdot (-5) + (-5)^2 = x^2 - 10x + 25$

为什么：尾是 $-5$，所以中间项为负，尾平方 $(-5)^2 = 25$ 仍为正。

正用例3：展开 $(2x+3)^2$

首 $= 2x$，尾 $= 3$。

$(2x+3)^2 = (2x)^2 + 2 \cdot (2x) \cdot 3 + 3^2 = 4x^2 + 12x + 9$

为什么：首平方是 $(2x)^2 = 4x^2$，不是 $2x^2$。

反用例：$x^2 + 10x + 25$ 写成平方形式

第1步：找首平方——$x^2$，所以首 $= x$。

第2步：找尾平方——$25 = 5^2$，所以尾 $= 5$。

第3步：验证中间项——$2 \cdot x \cdot 5 = 10x$ ✓，与原式中间项一致。

所以 $x^2 + 10x + 25 = (x+5)^2$。

改错例：$x^2 + 6x + 9 = (x+6)^2$ 对吗？

展开右边验证：$(x+6)^2 = x^2 + 12x + 36$。

左边 $= x^2 + 6x + 9 \neq x^2 + 12x + 36$。不对！

正确做法：首 $= x$，尾 $= 3$，中间项 $2 \cdot x \cdot 3 = 6x$ ✓。

所以 $x^2 + 6x + 9 = (x+3)^2$。

为什么错：把尾 $3$ 当成了 $6$。$(x+6)^2$ 的中间项是 $12x$，不是 $6x$。

易错点：展开 $(2x+3)^2$ 时首平方写成 $2x^2$。正确：$(2x)^2 = 4x^2$。
易错点：反用时不验证中间项，直接猜。比如看到 $x^2 + 6x + 9$ 就猜 $(x+6)^2$。必须验证 $2 \times x \times 6 = 12x \neq 6x$，所以猜错了。

二、配方法解一元二次方程

核心方法

用配方法解 $x^2 + 6x + 5 = 0$。

先把题目拆开

已知条件一元二次方程 $ax^2 + bx + c = 0$（这里 $a=1$）
要求目标方程的解
关键词配完全平方、两边同时加、开方写 ±
容易忽略配方的数必须"两边同时加"；开方必须写 ±

配方法四步流程

  1. 移常数：把常数项移到等号右边
  2. 配完全平方：两边同时加上"一次项系数的一半"的平方
  3. 化简右边：算出右边的数值
  4. 开方：若右边 $\ge 0$，写 $x + a = \pm\sqrt{m}$，解出 $x$；若右边 $< 0$，无解

关键想法

配方法的目标是：把 $x^2 + px + c = 0$ 变成 $(x+a)^2 = m$ 的形式。
这样就可以直接开方得到两个根。
配方秘诀：一次项系数除以 $2$，再平方，就是需要加的数。

标准解法（完整演示）

第 1 步：移常数

$x^2 + 6x + 5 = 0$

$x^2 + 6x = -5$

为什么：把常数移走，只留含 $x$ 的项在左边。

第 2 步：配完全平方

一次项系数是 $6$，一半是 $3$，$3^2 = 9$。

两边同时加 $9$：

$x^2 + 6x + 9 = -5 + 9$

为什么：只加左边不加右边，等号就不成立了！"两边同时加"是配方法最关键的纪律。

第 3 步：化简右边

$(x+3)^2 = 4$

第 4 步：开方

$x + 3 = \pm 2$

$x + 3 = 2 \Rightarrow x = -1$

$x + 3 = -2 \Rightarrow x = -5$

为什么写 ±：因为 $(+2)^2 = 4$ 且 $(-2)^2 = 4$，开方有两个结果。

三种情况演示

情况一：右边为正数（正常两个解）

$x^2 + 4x - 5 = 0$

$x^2 + 4x = 5$

加 $(4 \div 2)^2 = 4$：$x^2 + 4x + 4 = 9$

$(x+2)^2 = 9$，$x+2 = \pm 3$

$x = 1$ 或 $x = -5$

情况二：右边为零（重根）

$x^2 - 6x + 9 = 0$

$x^2 - 6x = -9$

加 $(6 \div 2)^2 = 9$：$(x-3)^2 = 0$

$x - 3 = 0$，$x = 3$（只有一个解，重根）

情况三：右边为负数（无解）

$x^2 + 2x + 5 = 0$

$x^2 + 2x = -5$

加 $(2 \div 2)^2 = 1$：$(x+1)^2 = -4$

任何实数的平方都不可能等于 $-4$，所以无解。

易错提醒

易错点：配方时只加左边不加右边。比如 $x^2 + 6x = -5$，写成 $x^2 + 6x + 9 = -5$，右边忘了加 $9$。正确：$x^2 + 6x + 9 = 4$。
易错点：开方时漏写 ±。$(x+2)^2 = 9$ 只写 $x+2 = 3$，漏了 $x+2 = -3$。正确：$x+2 = \pm 3$。
易错点：遇到 $(x+1)^2 = -4$ 时不判断无解，硬写 $x+1 = \pm\sqrt{-4}$。实数范围内 $\sqrt{-4}$ 不存在，必须写"无解"。

一句话总结

配方法四步：移常数 → 两边加（一次系数一半的平方）→ 化简右边 → 开方写 ±。别忘了"两边同时加"！

三、完全平方公式求 $a+b$、$a-b$

核心问题

已知 $a^2+b^2=17$，$ab=4$，求 $a+b$ 和 $a-b$。

先把题目拆开

已知条件$a^2+b^2=17$，$ab=4$
要求目标$a+b$ 和 $a-b$
关键词完全平方公式、代入、开方
容易忽略从 $(a+b)^2=25$ 到 $a+b=\pm 5$，必须写 $\pm$

解题路线

  1. 写公式 $(a+b)^2=a^2+2ab+b^2$
  2. 把已知值代入
  3. 计算，开方加 $\pm$
  4. 同理求 $(a-b)^2$

关键想法

不需要知道 $a$ 和 $b$ 各是多少！只需要把 $a^2+b^2$ 和 $ab$ 代入完全平方公式，就能直接算出 $(a+b)^2$。

标准解法

求 $a+b$
第 1 步：写公式

$$(a+b)^2 = a^2 + 2ab + b^2$$

完全平方公式：和的平方等于两数平方和加二倍乘积。

第 2 步：代入

$$(a+b)^2 = (a^2+b^2) + 2ab = 17 + 2\times 4 = 17 + 8 = 25$$

把 $a^2+b^2=17$ 和 $ab=4$ 分别代入对应位置。

第 3 步：开方

$$a+b = \pm 5$$

$(a+b)^2=25$，所以 $a+b=5$ 或 $a+b=-5$，合写 $\pm 5$。

求 $a-b$
第 1 步：写公式

$$(a-b)^2 = a^2 - 2ab + b^2$$

差的平方：中间项变成 $-2ab$。

第 2 步：代入

$$(a-b)^2 = 17 - 8 = 9$$

第 3 步：开方

$$a-b = \pm 3$$

易错提醒

易错点：从 $(a+b)^2=25$ 直接写 $a+b=5$，漏了 $-5$。
易错点：公式记错，$(a+b)^2=a^2+b^2$ 漏了 $2ab$。
易错点：求 $a-b$ 时用错公式，把 $-2ab$ 写成 $+2ab$。

一句话总结

写公式、代入已知值、开方加 $\pm$。$(a+b)^2$ 用 $+$，$(a-b)^2$ 用 $-$。

三个知识点的关系

这三个知识点是一条链：

完全平方公式是工具——记住 $(x+a)^2 = x^2 + 2ax + a^2$，会正用也会反用。

配方法是用工具——把方程配成完全平方形式后开方求解，是反用的直接应用。

求 $a+b$、$a-b$是综合应用——用完全平方公式把已知条件组合起来，不求具体值直接求和与差。

三者共同训练的核心动作：对公式既会正着用，也会反着用，还能灵活代入。
