学生版讲解：两直线交点与割补法求动点坐标

题目：在平面直角坐标系 $xOy$ 中，已知一次函数 $y=2x+m$ 与 $y=-\dfrac{1}{2}x+n$ 的图象都经过点 $A(-2,0)$，且分别与 $y$ 轴交于点 $B$ 和点 $C$。
(1) 求 $B$、$C$ 两点的坐标；
(2) 设点 $D$ 在直线 $y=-\dfrac{1}{2}x+n$ 上，且在 $y$ 轴右侧，当 $\triangle ABD$ 的面积为 $15$ 时，求点 $D$ 的坐标。

一、这题先看懂什么

这道题的场景其实很简单：

  1. 两条直线，一个公共点。 直线 $l_1$：$y=2x+m$，直线 $l_2$：$y=-\dfrac{1}{2}x+n$。它们都经过同一点 $A(-2,0)$。
  2. 每条直线各碰一次 $y$ 轴。 $l_1$ 碰到 $y$ 轴的地方叫 $B$，$l_2$ 碰到 $y$ 轴的地方叫 $C$。
  3. 有一个动点 $D$。 $D$ 在 $l_2$ 上，且在 $y$ 轴右边，$\triangle ABD$ 的面积恰好是 $15$。
想一想：如果 $m$、$n$ 的值还不知道，$A$ 点"在两条直线上"这个信息能帮你做什么？

二、第一步该做什么

第一步不是画图，也不是套面积公式——而是把 $m$、$n$ 算出来。

为什么？因为第(2)问要用到 $B$ 的坐标和 $l_2$ 的解析式，而这两样东西都取决于 $m$ 和 $n$。

怎么算？方法叫待定系数法，核心动作就一句：

点在图象上 $\;\Longrightarrow\;$ 把坐标代入解析式，得到关于系数的方程

$A(-2,0)$ 在 $l_1$ 上：把 $x=-2, y=0$ 代入 $y=2x+m$，解出 $m$。
$A(-2,0)$ 在 $l_2$ 上：把 $x=-2, y=0$ 代入 $y=-\dfrac{1}{2}x+n$，解出 $n$。

三、关键转化怎么想

第(1)问是"热身"，第(2)问才是重点。第(2)问的关键动作是：

面积 $\to$ 方程：把 $\triangle ABD$ 的面积等于 $15$ 这个条件，转化成一个关于 $D$ 坐标的方程

具体来说，你需要一个"把坐标变成面积"的公式。最直接的做法是坐标三角形面积公式（也叫行列式面积公式）：

$$S_{\triangle P_1P_2P_3} = \frac{1}{2}\left|x_1(y_2-y_3) + x_2(y_3-y_1) + x_3(y_1-y_2)\right|$$

这个公式的好处：知道三个顶点的坐标，就能直接算面积，不需要画辅助线、不需要选底和高。

想一想：你之前学过的割补法（选一条边为底，找对应的高）和这个公式，结果一样吗？为什么这里用公式更方便？

四、标准解法

第(1)题：求 $B$、$C$ 坐标

步骤1：求 $m$

把 $A(-2,0)$ 代入 $y=2x+m$：

$0 = 2 \times (-2) + m$  →  $0 = -4 + m$  →  $\boxed{m = 4}$

所以 $l_1$：$y = 2x + 4$

为什么这么做："图象经过点"就是"坐标满足方程"，代入就得到关于 $m$ 的方程。

步骤2：求 $n$

把 $A(-2,0)$ 代入 $y=-\dfrac{1}{2}x+n$：

$0 = -\dfrac{1}{2} \times (-2) + n$  →  $0 = 1 + n$  →  $\boxed{n = -1}$

所以 $l_2$：$y = -\dfrac{1}{2}x - 1$

步骤3：求 $B$、$C$ 坐标

$B$ 是 $l_1$ 与 $y$ 轴的交点，令 $x=0$：$y = 2 \times 0 + 4 = 4$，$\boxed{B(0,4)}$

$C$ 是 $l_2$ 与 $y$ 轴的交点，令 $x=0$：$y = -\dfrac{1}{2} \times 0 - 1 = -1$，$\boxed{C(0,-1)}$

为什么这么做：$y$ 轴上所有点的 $x$ 坐标都是 $0$，直接令 $x=0$ 就是求 $y$ 轴交点。

第(2)题：求 $D$ 坐标

步骤4：设 $D$ 坐标

$D$ 在 $l_2$：$y = -\dfrac{1}{2}x - 1$ 上，设 $D(t,\, -\dfrac{1}{2}t - 1)$。

题目说 $D$ 在 $y$ 轴右侧，所以 $t > 0$。

为什么这么做：动点在某条直线上，就设参数表示坐标，用一个字母表示横坐标，纵坐标用直线解析式表达。

步骤5：用坐标面积公式列方程

将 $A(-2,0)$，$B(0,4)$，$D(t, -\frac{1}{2}t - 1)$ 代入面积公式：

$$S = \frac{1}{2}\left|x_A(y_B - y_D) + x_B(y_D - y_A) + x_D(y_A - y_B)\right|$$

$$15 = \frac{1}{2}\left|(-2)\!\left(4 - \left(-\frac{1}{2}t - 1\right)\right) + 0\!\left(\left(-\frac{1}{2}t - 1\right) - 0\right) + t(0 - 4)\right|$$

$$15 = \frac{1}{2}\left|(-2)\!\left(5 + \frac{1}{2}t\right) + 0 - 4t\right|$$

$$15 = \frac{1}{2}\left|-10 - t - 4t\right|$$

$$15 = \frac{1}{2}\left|-10 - 5t\right|$$

$$30 = |-10 - 5t| = |-5(t + 2)| = 5(t + 2)$$

（因为 $t > 0$，所以 $t + 2 > 0$，绝对值直接去掉）

$$30 = 5(t + 2)$$

$$t + 2 = 6$$

$$\boxed{t = 4}$$

为什么这么做：坐标面积公式把三个顶点坐标和面积直接联系起来，一步列方程。

步骤6：写出 $D$ 坐标并验证

$y_D = -\dfrac{1}{2} \times 4 - 1 = -2 - 1 = -3$

$\boxed{D(4, -3)}$

验证：$t = 4 > 0$，满足"$y$ 轴右侧"的条件。验证通过。

五、边讲边问

  1. "图象经过点 $A(-2,0)$"这句话，翻译成数学语言是什么意思？
  2. 求 $y$ 轴交点时，为什么要令 $x=0$？
  3. 设 $D(t, -\frac{1}{2}t - 1)$ 时，纵坐标为什么不用另一个新字母 $k$ 来设？
  4. 面积公式里有绝对值号，去掉绝对值时需要判断什么？
  5. 如果算出来的 $t = -2$，这个结果能要吗？为什么？

六、易错提醒

  • 代入算 $m$ 时算错负号。$y = 2 \times (-2) + m = -4 + m$，不是 $4 + m$。负数乘正数是负数！
  • 面积公式里漏掉 $\dfrac{1}{2}$。三角形面积 = $\dfrac{1}{2} \times$ 底 $\times$ 高，别忘了除以 $2$。
  • 忘记检查 $t > 0$。题目明确说 "$D$ 在 $y$ 轴右侧"，$t$ 必须大于 $0$。如果算出 $t \le 0$，说明无解或题目有误。

七、一句话总结

"点在直线上 → 代入求参数；面积给条件 → 坐标公式列方程；算完要验证 → 条件约束不能忘。"
