自适应练习：一次函数与正比例函数的图象符号分析

教师判断

当前档位：C 档。判断依据：初二学生已掌握基本函数图象性质，能看懂 y=ax+b 的斜率和截距含义，但对"用同一组参数同时约束两个函数"这一思路尚需训练。本组练习侧重符号分类讨论与排除法的专项训练。

练习说明

本组练习只练一个动作：根据 a、b 的符号，同时写出两条直线的走向和截距，再用排除法选答案。不需要画精确图象，只需要判断走向和截距的正负。

第 1 题

已知一次函数 y = ax + b 和正比例函数 y = abx（a, b 为常数，ab ≠ 0）。若 y = ax + b 的图象经过第一、二、三象限，y = abx 的图象经过第一、三象限。下列关于 a、b 的判断，正确的是（　　）

A. a > 0, b > 0　　B. a > 0, b < 0　　C. a < 0, b > 0　　D. a < 0, b < 0

训练目标：从图象特征反推 a、b 的符号，并验证两条线是否自洽。

预期卡点：能推出 y=ax+b 经过一二三象限 ⇒ a>0, b>0，但忘记检验 y=abx 是否匹配。

提示一

y=ax+b 经过第一、二、三象限，说明它不经过第四象限。走向是什么？截距在上方还是下方？

提示二

从"经过一二三象限"推出 a>0, b>0 后，算一下 ab 的符号，再看 y=abx 是否经过第一、三象限。

第 2 题

已知一次函数 y = ax + b 和正比例函数 y = abx（a, b 为常数，ab ≠ 0）。下列四个选项各给出了两条直线的描述，其中不可能同时成立的是（　　）

A. y=ax+b 走向↗，截距在上方；y=abx 走向↗

B. y=ax+b 走向↗，截距在下方；y=abx 走向↘

C. y=ax+b 走向↘，截距在上方；y=abx 走向↗

D. y=ax+b 走向↘，截距在下方；y=abx 走向↘

训练目标：对每条直线的特征，反推 a、b 的符号要求，检验两条线是否使用同一组 (a,b)。

预期卡点：每条线单独看都合理，容易忽略"同一组 a、b"的约束；可能需要逐个验证四组符号。

提示一

对每个选项，先根据 y=ax+b 的走向确定 a 的正负，再根据截距确定 b 的正负，然后算 ab 的符号，看 y=abx 的走向是否匹配。

提示二

重点检查选项 C：走向↘ ⇒ a<0，截距上方 ⇒ b>0，那么 ab<0 ⇒ y=abx 应该走向↘。但选项说 y=abx 走向↗，矛盾！

第 3 题

已知一次函数 y = ax + b 和正比例函数 y = (a-1)x（a, b 为常数，a ≠ 0, a ≠ 1, b ≠ 0）。若两条直线的图象都经过第一、三象限，且 y = ax + b 与 y 轴交于正半轴，则 a 的取值范围是______。

训练目标：将符号分析方法迁移到参数表达式发生变化的情况（a-1 代替 ab）。

预期卡点：不习惯"斜率是一个表达式"的情况；可能忘记 a-1>0 这一条件的求解。

提示一

写出两个条件：① y=ax+b 经过一二三象限 ⇒ a>0, b>0；② y=(a-1)x 经过一三象限 ⇒ a-1 > 0。联立解 a 的范围。

提示二

由 ② 得 a > 1；结合 ① 的 a > 0，取交集得 a > 1。

参考答案与升级/降级判断

第 1 题

答案：A（a > 0, b > 0）

y=ax+b 经过一二三象限 ⇒ a>0, b>0。此时 ab>0 ⇒ y=abx 斜率为正，经过一三象限，与题意吻合。

判断：

  • 学生选 A 且能说出检验过程 → 正常，进入第 2 题
  • 学生选 A 但没检验 y=abx → 提醒"别忘了检验第二条线"，仍进入第 2 题
  • 学生选错 → 降级到 B 档，先补"系数与图象走向对应关系"的基本训练

第 2 题

答案：C

选项 C：y=ax+b 走向↘ ⇒ a<0；截距上方 ⇒ b>0。则 ab<0，y=abx 应走向↘。但选项说 y=abx 走向↗，矛盾。其余三个选项均可找到对应的 (a,b) 符号组。

判断：

  • 学生选 C 且过程完整 → 升级到 E 档，下次可做结构隐藏型变式
  • 学生选 C 但过程不完整（如跳过验证） → 保持 C 档，进入第 3 题
  • 学生选错 → 降级到 B 档，回到"分情况列表"的基本训练

第 3 题

答案：a > 1

y=ax+b 经过一二三象限且交 y 轴正半轴 ⇒ a>0, b>0。y=(a-1)x 经过一三象限 ⇒ a-1>0 ⇒ a>1。取交集得 a>1。

判断：

  • 学生做对 → 确认 E 档，下次做结构隐藏变式（如将参数关系嵌入实际情境）
  • 学生在 a-1>0 的求解上卡住 → 保持 C 档，加强"表达式符号判断"训练
  • 学生完全不会 → 降级到 B 档，退回基础
