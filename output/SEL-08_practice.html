自适应练习：分段函数——水费计费

教师判断

当前档位：D档（会列式，但可能在反向求解时漏验证区间，或第二段化简出错）。

判断依据：学生为初二水平，能解一次函数方程，但建模能力需要提升。基于结构分析，中等学生最易在第二段列式和区间判断上出错。本次练习定位在"巩固分段列式"和"强化先判段再求解"两个核心动作上，不升级到结构隐藏型变式。

练习说明

本组共3题，训练两个动作：

  1. 从新情境中正确列出分段函数解析式——重点是第二段"前段+超出"的结构。
  2. 反向求解时先判断落在哪一段——算完回头检查。

建议：每题完成后对照参考答案，记录卡点，判断是否需要降级或升级。

第1题 巩固列式

某地出租车收费标准如下：
① 不超过3公里时，收费8元；
② 超过3公里的部分，每公里收费2.4元。
设行驶里程为 x 公里（x > 0），费用为 y 元。
(1) 写出 y 与 x 的函数解析式，并指出定义域；
(2) 某次乘车费用为24.8元，求行驶里程。

训练目标：从新情境中提取"起步价+超出单价"的分段结构，正确列出第二段解析式（不是 y = 2.4x）。

预期卡点：第二段写成 y = 2.4x（忘了前3公里的起步价8元）。

提示一

分界点在哪里？超过3公里以后，费用由哪两部分组成？

提示二

第二段：前3公里费用 = 8 元，超出部分 = (x − 3) 公里，超出部分费用 = 2.4 × (x − 3)。所以 y = 8 + 2.4(x − 3) = 2.4x + 0.8。

第2题 强化判段

某快递公司收费规则：首重1公斤以内收费10元；超过1公斤的部分，每公斤收费3.5元。设包裹重量为 x 公斤（x > 0），运费为 y 元。
(1) 写出 y 与 x 的函数解析式；
(2) 小明寄一个包裹付了38.5元，求这个包裹的重量。

训练目标：第(2)问——先算分界点的费用，判断38.5元落在哪一段，再解方程。强化"先判段"意识。

预期卡点：直接用 y = 3.5x + 6.5 = 38.5 算出 x = 9.14，但不验证 x = 9.14 > 1 是否合理（虽然这题碰巧对了，但养成验证习惯很重要）。

提示一

分界点是 x = 1。当 x = 1 时 y 等于多少？38.5 元和这个数比一比，落在哪一段？

提示二

x = 1 时 y = 10 元。38.5 > 10，所以用第二段 y = 3.5x + 6.5 解方程：3.5x + 6.5 = 38.5，得 x = 9.14。检查：9.14 > 1，在第二段定义域内，合理。注意：重量通常取一位小数，x = 9.14 ≈ 9.1 公斤。

第3题 轻度迁移

某城市居民用电实行阶梯电价：
① 全年用电量不超过2760度时，每度电0.52元；
② 超过2760度但不超过4800度的部分，每度电0.57元；
③ 超过4800度的部分，每度电0.82元。
设全年用电量为 x 度（x > 0），电费为 y 元。
(1) 写出 y 与 x 的函数解析式（三段）；
(2) 某户全年电费为1600元，求全年用电量。

训练目标：从两段扩展到三段——体会结构不变、段数增加的规律。训练"逐段累加"的列式方法。

预期卡点：第三段列式时只加"超出4800的部分 × 0.82"，忘记加上前两段的总额。

提示一

分界点是 x = 2760 和 x = 4800。第二段和第三段的费用都由"前面所有段的费用之和"加上"本段超出部分 × 本段单价"构成。先算各段在分界点的累计费用。

提示二

第一段 y = 0.52x（0 < x ≤ 2760）
第二段 y = 0.52 × 2760 + 0.57(x − 2760) = 0.57x − 138（2760 < x ≤ 4800）
第三段 y = 0.52 × 2760 + 0.57 × 2040 + 0.82(x − 4800) = 0.82x − 1338（x > 4800）
第(2)问：分界点处 y(2760) = 1435.2，y(4800) = 2598。1600 介于两者之间 → 第二段。0.57x − 138 = 1600，x = 3048.77。验证：2760 < 3048.77 ≤ 4800，合理。

参考答案与升级/降级判断

第1题

答案：

(1) 当 0 < x ≤ 3 时，y = 8
当 x > 3 时，y = 8 + 2.4(x − 3) = 2.4x + 0.8
(2) x = 3 时 y = 8。24.8 > 8 → 用第二段：2.4x + 0.8 = 24.8，x = 10。验证 10 > 3，合理。行驶里程 10 公里。
判断：
· 第二段正确写出 y = 2.4x + 0.8 → 核心动作掌握，可继续；
· 第二段写成 y = 2.4x（漏起步价）→ 降级：回到讲解页，重新看"转化2"部分，再做类似一题；
· 第(2)问没验证区间 → 提醒，不降级，下一题重点关注。

第2题

答案：

(1) 当 0 < x ≤ 1 时，y = 10
当 x > 1 时，y = 10 + 3.5(x − 1) = 3.5x + 6.5
(2) x = 1 时 y = 10。38.5 > 10 → 第二段：3.5x + 6.5 = 38.5，x = 9.14。验证 9.14 > 1，合理。包裹重约 9.1 公斤。
判断：
· 第(2)问先判断区间再解方程 → "先判段"意识已建立，可升级；
· 没有先判断就直接解方程 → 需要提醒，但答案碰巧正确时不降级，下一题继续观察；
· 计算错误 → 说明基础运算需巩固，不降级但记录。

第3题

答案：

(1) 当 0 < x ≤ 2760 时，y = 0.52x
当 2760 < x ≤ 4800 时，y = 0.52 × 2760 + 0.57(x − 2760) = 0.57x − 138
当 x > 4800 时，y = 0.52 × 2760 + 0.57 × 2040 + 0.82(x − 4800) = 0.82x − 1338
(2) y(2760) = 1435.2，y(4800) = 2598。1600 介于两者之间 → 第二段。
0.57x − 138 = 1600，x = 1738 ÷ 0.57 ≈ 3049 度。验证：2760 < 3049 ≤ 4800，合理。
判断：
· 三段全部正确列出，第(2)问正确判断区间 → 掌握，可以进入E档（同源平级变式，换情境但保留两段结构，例如电费/水费/出租车混合题）；
· 第二段或第三段列式错误 → 退回C档，做"选关系、找关键量"的专项训练；
· 三段列式正确但第(2)问区间判断错误 → 退回D档，加强"先判段"训练。
