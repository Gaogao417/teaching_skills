学生版讲解：用待定系数法求一次函数解析式

一、这题先看懂什么

题目说："有一个一次函数"，什么是一次函数？就是y = kx + b（k ≠ 0）这样的函数，画出来是一条直线。

题目又给了两条信息：

  • 当 x = 3 时，y = −1，说明点 (3, −1) 在这条直线上；
  • 当 x = 2 时，y = 8，说明点 (2, 8) 也在这条直线上。

我们的任务：找到 k 和 b 分别等于多少，把解析式写完整。

二、第一步该做什么

不要急着算！先想一想：我们不知道 k 和 b，但它们是固定的两个数。

既然不知道，那就设出来：

设 y = kx + b（k ≠ 0）

这里"设"就是把未知数摆出来，等一下用条件来求它们。这个方法叫做待定系数法——"待定"就是"等着确定"的意思。

三、关键转化怎么想

知道了两个点在直线上，怎么利用呢？

关键转化：如果点 (x₀, y₀) 在直线 y = kx + b 上，那么把 x₀ 和 y₀ 代进去，等式一定成立。

想一想：如果点 (3, −1) 在直线 y = kx + b 上，把 x = 3、y = −1 代入，得到什么？

答案：3k + b = −1  ……①

同理，把 (2, 8) 代入：

2k + b = 8  ……②

这样我们就得到了两个方程、两个未知数（k 和 b）的方程组，可以解了！

四、标准解法

第1步：设解析式

设 y = kx + b（k ≠ 0）

为什么：k 和 b 是未知的，先写出来才知道要算什么。

第2步：代入已知点，列方程组

将 (3, −1) 代入：3k + b = −1  ……①

将 (2, 8) 代入：2k + b = 8   ……②

为什么：点在图像上，坐标就满足关系式，每个点给一个方程。

第3步：解方程组（加减消元法）

① − ②： (3k + b) − (2k + b) = −1 − 8

           k = −9

将 k = −9 代入②：2 × (−9) + b = 8

           −18 + b = 8

           b = 26

为什么选消 b：两个方程里 b 的系数都是 1，直接相减就能消掉 b，一步求出 k。

第4步：写出解析式并检验

所以 y = −9x + 26

检验：把 (3, −1) 代入：−9 × 3 + 26 = −27 + 26 = −1 ✓

把 (2, 8) 代入：−9 × 2 + 26 = −18 + 26 = 8 ✓

为什么检验：代入时容易写错位置或算错符号，检验一遍能确保答案正确。

五、边讲边问

  1. 待定系数法的第一步是什么？（提示：把 k 和 b 怎么办？）
  2. "点 (3, −1) 在直线上"这句话，用数学语言怎么翻译成方程？
  3. 解方程组时，① − ② 消去的是 k 还是 b？为什么选消这个？
  4. 如果 k = −9，b = 26，那么当 x = 0 时 y 等于多少？这个数在图像上代表什么？
  5. 为什么两个点就能确定一条直线，一个点行不行？

六、易错提醒

  • 代入时 x、y 位置写反：把 (3, −1) 代入写成 k × (−1) + b = 3，就全错了。记住：括号里第一个数是 x，第二个数是 y。
  • 解方程组正负号出错：① − ② 时，右边是 −1 − 8 = −9，不是 −1 + 8。减号要分配到整个式子。
  • 忘记写 k ≠ 0：一次函数的定义要求 k ≠ 0，虽然本题解出来 k = −9 ≠ 0 自然满足，但设的时候要写上。

七、一句话总结

设 → 代 → 列 → 解 → 写（并检验）

待定系数法五步走：设出解析式，代入已知点，列出方程组，解出系数，写出解析式并检验。
