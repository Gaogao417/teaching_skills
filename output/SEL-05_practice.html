自适应练习：一次函数增减性分类讨论

教师判断

当前档位：D档（会列式，但可能漏掉分类讨论或忘记验证）。

判断依据：初二学生已掌握一次函数基本性质和待定系数法，但分类讨论是薄弱环节。本次练习聚焦"识别是否需要分类讨论"和"正确配对端点"两个关键动作。

练习说明

本组练习只做一件事：判断 k 的正负是否确定，再决定需要列几组方程。 请在每道题中先写出你的判断理由，再列方程求解。

第1题（诊断分类讨论意识）

已知一次函数 y = kx + b，当 x = 1 时 y = 5，当 x = 3 时 y = 9。

（1）求这个一次函数的解析式。

（2）本题需要分类讨论吗？为什么？

训练目标：识别"k 的符号已由数据隐含确定"的情形，不需要分类讨论。

预期卡点：学生可能不假思索地分两类讨论，或无法从数据中看出 k 的符号。

提示一

看看 x 从 1 变到 3，y 从 5 变到 9——y 是变大了还是变小了？这说明 k 是正还是负？

提示二

x 增大时 y 也增大，所以 k > 0。既然 k 的正负已经确定了，还有必要讨论 k < 0 的情况吗？

第2题（巩固分类讨论配对）

已知一次函数 y = kx + b 的自变量 x 的取值范围是 1 到 4，相应的函数值 y 的取值范围是 −1 到 5。求这个一次函数的解析式。

训练目标：在"需要分类讨论"的场景中正确配对端点并列方程组。

预期卡点：k < 0 时端点配反，或忘记讨论第二种情况。

提示一

题目告诉了 k 的正负吗？没有。所以分 k > 0 和 k < 0 两种情况。先想想：k > 0 时，x 最小值对应 y 的哪个值？

提示二

k > 0 时：x = 1 对应 y = −1，x = 4 对应 y = 5。k < 0 时反过来：x = 1 对应 y = 5，x = 4 对应 y = −1。分别列方程组。

第3题（条件限制型变式）

已知一次函数 y = kx + b（k < 0），自变量 x 的取值范围是 −2 到 5，相应的函数值 y 的取值范围是 −8 到 1。求这个一次函数的解析式。

训练目标：当 k 的符号已经给定时，只列一组方程——体会"条件限制简化分类讨论"。

预期卡点：虽然知道 k < 0，但仍可能配错端点；或虽然不再需要讨论，但学生在解完后又怀疑自己漏了解。

提示一

题目明确说了 k < 0（递减），所以只需要考虑一种配对方式。x 最小值对应 y 的哪个值？

提示二

k < 0 → 递减 → x = −2 对应 y = 1，x = 5 对应 y = −8。列一组方程就够了。

参考答案与升级/降级判断

第1题

（1）

k + b = 5   ①
3k + b = 9   ②

② − ①：2k = 4，k = 2

代入①：b = 5 − 2 = 3

解析式：y = 2x + 3

（2）不需要分类讨论。因为题目给了两个确定的对应点（x = 1, y = 5）和（x = 3, y = 9），可以直接确定 k 和 b。从数据也能看出 x 增大时 y 增大，k > 0 已经隐含确定。

升级/降级判断：如果学生正确答出"不需要"并给出合理理由 → 维持 D 档，进入第 2 题。如果学生答"需要分类讨论" → 降至 C 档，回到讲解阶段复习"分类讨论的触发条件"。

第2题

情况一：k > 0（递增）

k + b = −1   ①
4k + b = 5   ②

② − ①：3k = 6，k = 2，b = −3

解析式：y = 2x − 3

情况二：k < 0（递减）

k + b = 5   ③
4k + b = −1   ④

④ − ③：3k = −6，k = −2，b = 7

解析式：y = −2x + 7

升级/降级判断：如果学生两解都正确 → 升至 E 档，可做迁移型变式。如果只得到一解 → 维持 D 档，重点关注"端点配对"步骤。如果配对错误（k 符号与解矛盾）→ 降至 C 档。

第3题

k < 0，只列一组方程：

x = −2 对应 y = 1（最小 x 配最大 y）

x = 5 对应 y = −8（最大 x 配最小 y）

−2k + b = 1   ①
5k + b = −8   ②

② − ①：7k = −9，k = −9/7

代入①：b = 1 + 2 × (−9/7) = 1 − 18/7 = −11/7，b = −11/7

解析式：y = −(9/7)x − 11/7

升级/降级判断：如果学生正确且只列了一组方程 → 升至 E 档。如果列了两组方程（不必要地讨论）但最终结果正确 → 维持 D 档，需明确"已知 k 符号时不需讨论"。如果端点配反 → 降至 C 档。
