自适应练习：等腰梯形存在性

教师判断

当前档位：E 档（会做原题，但需要迁移巩固）

判断依据：学生已掌握平行四边形存在性，理解坐标几何基本工具（斜率、距离公式、平移）。通过讲解阶段学习了等腰梯形的三种分类讨论方法，但尚未独立完成过此类问题。需要同源平级变式来巩固"枚举平行对边 → 列方程 → 验证退化"的流程。

练习说明

本组共 3 题，每题都练习同一个动作：找到所有可能的平行对边组合，列方程求出第四个顶点，然后验证不退化为平行四边形。

请按顺序完成，每题写出完整的枚举、列式和验证过程。

第 1 题（巩固：正向应用）

在平面直角坐标系中，已知 A(0, 0)、B(2, 4)、C(6, 0) 三个点。若点 D 在坐标平面内，且以 A、B、C、D 为顶点的四边形是等腰梯形，求所有满足条件的点 D 的坐标。

训练目标：独立完成三种情况的枚举，每步写出"哪两条边平行"和"哪两条边相等"。

预期卡点：① 漏掉 AB ∥ CD 的情况（因为 AB 是斜线不直观）；② 解方程时计算出错；③ 忘记验证退化。

提示一

先算出 AB、BC、AC 三条线段的斜率和长度平方，备用。然后列出三组可能的平行对边组合。

提示二

三组组合分别是：① AB ∥ CD，AC = BD；② AC ∥ BD，AB = CD；③ BC ∥ AD，AB = CD。每组用斜率相等列一个方程，用距离相等列另一个方程。

答题区域

第 2 题（变式：换情境，结构不变）

抛物线 y = −x² + 4x 与 x 轴交于 O(0, 0) 和 A(4, 0)，与 y 轴交于 B(0, 4)。若点 D 在坐标平面内，且以 O、B、A、D 为顶点的四边形是等腰梯形，求所有满足条件的点 D 的坐标。

训练目标：将同样的分类讨论方法应用到新图形（抛物线）上，体会方法的通用性。

预期卡点：虽然图形变了，但核心方法完全一样。学生可能被"抛物线"吓到，但其实只需要 O、B、A 三个点的坐标。

提示一

这道题的关键信息和上一题完全一样——三个已知点的坐标。不需要用抛物线的任何其他性质。直接算 OB、BA、OA 的斜率和长度。

提示二

注意：OA 在 x 轴上（斜率 0），OB 在 y 轴上（斜率不存在/无穷大）。三组平行对边中，OA ∥ BD 的情况下 BD 斜率 = 0 → D 的 y 坐标和 B 相同（即 y = 4）。

答题区域

第 3 题（提升：结构隐藏）

在平面直角坐标系中，A(−1, 0)、B(0, 2)、C(3, 0) 三个点。若四边形 ABCD 是等腰梯形，且 AD ∥ BC，求点 D 的坐标。（注意：本题已指定哪两条边平行。）

训练目标：在指定平行边的情况下，专注于"等腰 + 不退化"两个条件的列式和验证。

预期卡点：① AD ∥ BC 的斜率方程可能有两个解；② 需要判断哪个解构成梯形（而不是平行四边形）。

提示一

既然已知 AD ∥ BC，先算 BC 斜率：(0−2)/(3−0) = −2/3。设 D(a,b)，则 (b−0)/(a+1) = −2/3，得到 b = −2(a+1)/3。

提示二

等腰条件：AB = CD 或 AC = BD。题目说"等腰梯形"，两腰相等。既然 AD ∥ BC，两腰就是 AB 和 CD（或 AC 和 BD，需看哪个更合理）。先试 AB = CD，列出距离等式。

答题区域

参考答案与升级/降级判断

第 1 题

已知：A(0,0), B(2,4), C(6,0)

备算：

线段斜率长度²
AB220
BC−132
AC036

情况一：AB ∥ CD，AC = BD

kCD = 2 → b/(a−6) = 2 → b = 2a − 12

|AC|² = |BD|² → 36 = (a−2)² + (b−4)²

代入 b = 2a − 12：(a−2)² + (2a−16)² = 36 → 5a² − 68a + 224 = 0 → Δ = 68² − 4×5×224 = 4624 − 4480 = 144 = 12² > 0 → a = (68±12)/10，a = 8 或 a = 28/5

  • a = 8 → b = 4 → D(8, 4)。验证 AB ∥ CD ✓。检查 AD ∥ BC：kAD = 1/2 ≠ −1 ✓，但 AB = CD（kAB = kCD = 2 且 |AB|² = |CD|² = 20），一组对边平行且相等 → 退化为平行四边形，舍去。
  • a = 28/5 → b = −4/5 → D(28/5, −4/5)。验证 AB ∥ CD：kCD = (−4/5)/(28/5 − 6) = (−4/5)/(−2/5) = 2 ✓。检查 AD ∥ BC：kAD = (−4/5)/(28/5) = −1/7 ≠ −1 ✓。AC = BD 已满足。D(28/5, −4/5)。

情况二：AC ∥ BD，AB = CD

AC 斜率 0 → BD 斜率 0 → (b−4)/(a−2) = 0 → b = 4

|AB|² = |CD|² → 20 = (a−6)² + (4)² → (a−6)² = 4 → a = 8 或 a = 4

  • D(8, 4)：验证 AC ∥ BD（都水平 ✓），检查 AB ∥ CD：kAB = 2，kCD = (4−0)/(8−6) = 2 → AB ∥ CD，退化为平行四边形，舍去。
  • D(4, 4)：kCD = (4−0)/(4−6) = −2 ≠ 2 ✓。答案 D(4, 4)。

情况三：BC ∥ AD，AB = CD

kAD = kBC = −1 → b/a = −1 → b = −a

|AB|² = |CD|² → 20 = (a−6)² + (−a)² → 2a² − 12a + 16 = 0 → a² − 6a + 8 = 0 → a = 2 或 a = 4

  • D(2, −2)：验证 BC ∥ AD（斜率都是 −1 ✓），检查 AB ∥ CD：kCD = (−2−0)/(2−6) = 1/2 ≠ 2 ✓。答案 D(2, −2)。
  • D(4, −4)：kCD = (−4)/(4−6) = 2 = kAB，退化为平行四边形，舍去。

第 1 题答案：D(28/5, −4/5)、D(4, 4) 或 D(2, −2)

升级判断：若学生三组都列出但情况一未正确求出两个解（尤其是未舍去退化解 D(8, 4)） → 说明解方程和验证退化需加强，保持 E 档。若三组都正确完成 → 升至 F 档。

第 2 题

已知：O(0,0), B(0,4), A(4,0)

备算：

线段斜率长度²
OB不存在（竖直）16
BA−132
OA0（水平）16

情况一：OB ∥ AD，OA = BD

OB 竖直 → AD 竖直 → a − 4 = 0 → a = 4（D 的 x 坐标等于 A 的 x 坐标）

设 D(4, b)，|OA|² = |BD|² → 16 = (4−0)² + (b−4)² = 16 + (b−4)² → (b−4)² = 0 → b = 4

  • D(4, 4)：OB ∥ AD（都竖直 ✓），检查 OA ∥ BD：kBD = (4−4)/(4−0) = 0 = kOA，两组对边分别平行 → 退化为平行四边形，舍去。此情况无有效解。

情况二：BA ∥ OD，OB = AD

kOD = kBA = −1 → b/a = −1 → b = −a

|OB|² = |AD|² → 16 = (a−4)² + (−a−4)² = (a²−8a+16) + (a²+8a+16) = 2a² + 32 → 2a² + 16 = 0 → a² = −8，无实数解，此情况不成立。

情况三：OA ∥ BD，OB = AD

kBD = 0 → b = 4

|OB|² = |AD|² → 16 = (a−4)² + (4)² → (a−4)² = 0 → a = 4

D(4, 4) = (A 的 x, B 的 y)。验证：OA ∥ BD 都水平 ✓，检查 OB ∥ AD：kAD = (4−4)/(4−4) 不确定 → 实际上 A(4,0) 和 D(4,4)，AD 竖直 = OB 竖直，退化为平行四边形，舍去。

第 2 题答案：不存在满足条件的点 D

升级判断：此题三种情况均无有效解。若学生正确完成全部推导并得出"无解"结论 → 升至 F 档。若学生在某步卡住或未发现退化 → 保持 E 档。

第 3 题

已知：A(−1,0), B(0,2), C(3,0)，AD ∥ BC

kBC = (0−2)/(3−0) = −2/3

kAD = −2/3 → (b−0)/(a−(−1)) = −2/3 → b = −2(a+1)/3 …… ①

两腰是 AB 和 CD：|AB|² = |CD|²

|AB|² = (0+1)² + (2−0)² = 5

|CD|² = (a−3)² + (b−0)² = 5 …… ②

代入 ① 到 ②：(a−3)² + 4(a+1)²/9 = 5

两边乘 9：9(a−3)² + 4(a+1)² = 45

9(a²−6a+9) + 4(a²+2a+1) = 45

13a² − 46a + 85 = 45

13a² − 46a + 40 = 0

Δ = 46² − 4×13×40 = 2116 − 2080 = 36

a = (46 ± 6) / 26 → a = 2 或 a = 20/13

  • a = 2 → b = −2(3)/3 = −2 → D(2, −2)。验证：kAD = (−2)/(3) = −2/3 = kBC ✓。检查另一组对边 AB 和 CD：kAB = 2，kCD = (−2−0)/(2−3) = 2 → AB ∥ CD，退化为平行四边形，舍去。
  • a = 20/13 → b = −2(33/13)/3 = −22/13 → D(20/13, −22/13)。验证 AD ∥ BC ✓。kCD = (−22/13)/(20/13−3) = (−22/13)/(−19/13) = 22/19 ≠ 2 ✓。D(20/13, −22/13)。

再试 AC = BD 作为等腰条件：

|AC|² = (3+1)² + 0 = 16

|BD|² = (a−0)² + (b−2)² = 16

代入 b = −2(a+1)/3：a² + (−2(a+1)/3 − 2)² = 16

a² + (−2a−2−6)²/9 = 16 → a² + (−2a−8)²/9 = 16

9a² + 4(a+4)² = 144

9a² + 4a² + 32a + 64 = 144

13a² + 32a − 80 = 0

Δ = 1024 + 4160 = 5184 = 72²

a = (−32 ± 72) / 26 → a = 40/26 = 20/13 或 a = −104/26 = −4

  • a = 20/13 → 已得到，对应 AB = CD 的情况。
  • a = −4 → b = −2(−3)/3 = 2 → D(−4, 2)。验证 AD ∥ BC：kAD = (2−0)/(−4+1) = −2/3 ✓。kAB = 2，kCD = (2−0)/(−4−3) = −2/7 ≠ 2 ✓。D(−4, 2)。

第 3 题答案：D(20/13, −22/13) 或 D(−4, 2)

升级判断：若学生只试了一种等腰条件就结束 → 提醒"两腰不一定是 AB 和 CD"，保持 E 档。若两种等腰条件都试了并正确舍去退化 → 升至 F 档。
