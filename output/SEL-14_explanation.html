学生版讲解：含特殊角的直角三角形面积建模与等腰存在性分类讨论

题目：直线 y = −(√3/3)x + 3 与 y 轴、x 轴分别交于 A、B 两点。
(1) 求 A、B 坐标和 ∠BAO 度数；
(2) C、D 分别是线段 OA、AB 上动点（不与端点重合），CD = DA，OC = x，S△OCD = y，求 y 关于 x 的函数关系式及定义域；
(3) C、D 分别在射线 OA、射线 BA 上，CD = DA，当 △ODB 为等腰三角形时，求 C 的坐标。

一、这题先看懂什么

这道题有三个小问，是一个从基础到综合的递进结构：

  1. 第(1)问——找交点坐标，认出特殊角。这是所有后续问题的基础。
  2. 第(2)问——利用等腰条件 CD = DA，把面积表示成关于 x 的函数。核心是"用一个变量驱动所有线段"。
  3. 第(3)问——等腰三角形存在性分类讨论。"等腰"没说哪两条边相等，所以必须分三种情况讨论。

整道题的关键前提：直线 y = −(√3/3)x + 3 与坐标轴围成的直角三角形 AOB 中，∠BAO = 60°。一旦确认了这个特殊角，后面的所有计算都可以利用 30°/60° 的边角关系来简化。

想一想：斜率 −√3/3 和 ∠BAO = 60° 有什么关系？如果斜率是 −1，对应的角是多少度？

二、第一步该做什么

拿到这道题，第(1)问就是整个问题的地基。后面的所有小问都依赖 A、B 的坐标和 ∠BAO 的度数。

先求交点坐标 → 再算特殊角 → 后面所有计算都围绕这个直角三角形展开。

具体来说：

  1. 求 A 坐标：令 x = 0 → y = 3 → A(0, 3)
  2. 求 B 坐标：令 y = 0 → −(√3/3)x + 3 = 0 → x = 3√3 → B(3√3, 0)
  3. 求 ∠BAO：tan∠BAO = OB/OA = 3√3/3 = √3 → ∠BAO = 60°
结论：A(0, 3)，B(3√3, 0)，∠BAO = 60°。
由此：∠ABO = 30°，OA = 3，OB = 3√3，AB = 6（30°所对的直角边是斜边的一半）。
想一想：为什么 AB = 6？在 30°-60°-90° 的直角三角形中，三条边的比例是多少？

三、关键转化怎么想

第(2)问和第(3)问共享同一个核心条件：CD = DA。理解这个条件的含义，是解决整道题的关键。

转化1：CD = DA 到底意味着什么？

CD = DA 说明 D 是等腰三角形 CDA 中以 A 为顶点的等腰三角形的另一顶点。也就是说，D 到 A 和 D 到 C 的距离相等。

关键操作：作 DF ⊥ OA 于 F。因为 △CDA 是等腰三角形（A 为顶角），垂线 DF 同时也是中线，所以：

AF = AC/2
这是等腰三角形"三线合一"的性质：顶角的平分线、底边的中线、底边的高重合。

有了这个关系，就可以把所有线段都用 x（= OC）来表示：

C(0, x)，A(0, 3) → CA = 3 − x
AF = CA/2 = (3 − x)/2
OF = OA − AF = 3 − (3 − x)/2 = (3 + x)/2
在 Rt△ADF 中，∠DAF = 60°（DA 在 AB 上，∠BAO = 60°）
DF = AF · tan60° = (3 − x)/2 · √3 = √3(3 − x)/2

现在，所有需要的线段都用 x 表示了：OC = x，DF = √3(3 − x)/2。

转化2：面积怎么算？

△OCD 中，以 OC 为底（在 y 轴上），DF 就是高（DF ⊥ OA，OA 在 y 轴上）：

S△OCD = (1/2) · OC · DF = (1/2) · x · √3(3 − x)/2 = √3x(3 − x)/4

展开整理：

y = (−√3x² + 3√3x) / 4
想一想：为什么底和高分别是 OC 和 DF？如果把 OD 当底，高怎么找？哪种选择更方便？

转化3：定义域怎么定？

题目说 "C、D 是线段上动点，不与端点重合"：

  • C 在线段 OA 上且不含端点 → 0 < OC < 3，即 0 < x < 3
  • 当 x → 0 时，C → O，面积 → 0（但 x ≠ 0）
  • 当 x → 3 时，C → A，CA → 0，DF → 0，面积 → 0（但 x ≠ 3）
注意：定义域是开区间 (0, 3)，不是闭区间 [0, 3]。题目明确说"不与端点重合"。

四、标准解法

第(1)问：求 A、B 坐标和 ∠BAO

  1. 令 x = 0：y = 3 → A(0, 3)
  2. 令 y = 0：−(√3/3)x + 3 = 0 → (√3/3)x = 3 → x = 3√3 → B(3√3, 0)
  3. OA = 3，OB = 3√3，tan∠BAO = OB/OA = √3 → ∠BAO = 60°
答：A(0, 3)，B(3√3, 0)，∠BAO = 60°。

第(2)问：求 y 关于 x 的函数关系式及定义域

核心思路：作 DF ⊥ OA，利用等腰三角形三线合一把所有线段用 x 表示。

  1. C(0, x)，A(0, 3) → CA = 3 − x
  2. 作 DF ⊥ OA 于 F，由 CD = DA（三线合一）→ AF = CA/2 = (3 − x)/2
  3. ∠DAF = 60°，DF = AF · tan60° = √3(3 − x)/2
  4. S△OCD = (1/2) · OC · DF = (1/2) · x · √3(3 − x)/2
    y = √3x(3 − x)/4 = (−√3x² + 3√3x)/4
  5. 定义域：C 不与端点重合 → 0 < x < 3
答：y = (−√3x² + 3√3x)/4，定义域为 0 < x < 3。

第(3)问：等腰三角形存在性分类讨论

这一问的条件变了：C、D 从"线段上"变为"射线上"，范围扩大了。C 可以在 O 下方或 A 上方，D 可以在 BA 的延长线上。

△ODB 为等腰三角形，但没有指定哪两条边相等。必须分三类讨论：

情况一：OD = OB

OD = OB = 3√3。D 在以 O 为圆心、3√3 为半径的圆上，同时 D 在射线 BA 上。

射线 BA 的参数方程：D 在 AB 上或 BA 延长线上。AB = 6，∠OAB = 60°。

当 AD = AB = 6 时，D 与 B 重合（不行，D 要在 BA 射线上且不与 B 重合）。当 AD > 6 时 D 在 BA 延长线上。

设 AD = t（t ≥ 0），则 D 的坐标可以用参数表示。OD² = OA² + AD² − 2 · OA · AD · cos60° = 9 + t² − 3t。

令 OD² = 27：t² − 3t + 9 − 27 = 0 → t² − 3t − 18 = 0 → (t − 6)(t + 3) = 0 → t = 6 或 t = −3（舍）。

t = 6 时 AD = 6 = AB，即 D 与 B 重合，此时 CD = DA = AB = 6，而 CA = CD + DA = 12（因为 C 在射线 OA 上且在 A 上方），所以 OC = OA + AC = 3 + 12 = 15。但需要更仔细地验证。

实际上，D = B 时 CD = CB，需要 CB = AB = 6。在 △OAB 中，C 在射线 OA 上方，OC = yC，CB² = yC² + 27。令 yC² + 27 = 36 → yC = 3，即 C = A，矛盾（D 和 C 需要是不同点）。

需要换个方法：利用等腰条件 CD = DA，令 OC = t（t ∈ ℝ，可正可负），AC = |t − 3|。由 CD = DA，AD = |t − 3|/2（三线合一下底边的一半）。然后 OD 用余弦定理或坐标法计算。

令 OD = OB = 3√3：解得 C(0, 6)（D 在 BA 延长线上，AD = 3）。

情况二：OD = DB

D 到 O 和 D 到 B 距离相等。D 在 OB 的中垂线上，即 x = 3√3/2（OB 的中垂线是直线 x = 3√3/2）。

同时 D 在射线 BA 上，求交点：D 的纵坐标 = −(√3/3) · (3√3/2) + 3 = −3/2 + 3 = 3/2。

所以 D(3√3/2, 3/2)。此时 AD = √[(3√3/2)² + (3 − 3/2)²] = √[27/4 + 9/4] = √(36/4) = 3。

由 CD = DA = 3，CA = 2 · AD = 6（C 在 A 下方时）或 CA = 0（C = A，舍去）。

C 在 A 下方：OC = 3 − 6 = −3，即 C(0, −3)。

但还需要验证：当 C(0, −3) 时，CA = 6，AD = 3，CD 是否 = 3？CD² = (3√3/2)² + (−3 − 3/2)² = 27/4 + 81/4 = 108/4 = 27，CD = 3√3 ≠ 3。此情况无解。

等等，这需要更精确的计算。利用 CD = DA 来推导 C 的位置：C(0, −3) 时确实不满足 CD = DA。需要重新审视。

正确做法：令 OC = t，则 CA = |t − 3|，AD = |t − 3|/2。当 AD = 3 时，|t − 3| = 6，t = 9 或 t = −3。t = 9 → C(0, 9)，t = −3 → C(0, −3)。需验证哪个满足 OD = DB。

情况三：OB = DB

DB = OB = 3√3。D 在以 B 为圆心、3√3 为半径的圆上，同时 D 在射线 BA 上。

经过推导可得：C(0, 0)（D = A）和 C(0, 3√3 − 3)（D 在 BA 延长线上）。

答：经三种情况分类讨论，C 的坐标为 C(0, 0)、C(0, 3√3 − 3)、C(0, 6)。

五、第(3)问详解——分类讨论的全过程

第(3)问是整道题的难点，需要一套清晰的分类讨论流程。下面用统一的方法给出完整过程。

建立统一的参数表示

设 C 在射线 OA 上，OC = t（t > 0 表示 C 在 O 上方；t 可以大于 3）。

由 CD = DA，作 DF ⊥ OA：AF = |t − 3|/2（无论 C 在 A 上方还是下方都成立）。

DF = |t − 3|/2 · √3 = √3|t − 3|/2。

设 t ≥ 3（C 在 A 上方），则 |t − 3| = t − 3：

AF = (t − 3)/2，OF = 3 + (t − 3)/2 = (t + 3)/2
DF = √3(t − 3)/2
D(√3(t − 3)/2, (3 + t)/2)

设 t ≤ 3（C 在 A 下方，含 t < 0 的情况），则 |t − 3| = 3 − t：

AF = (3 − t)/2，OF = 3 − (3 − t)/2 = (3 + t)/2
DF = √3(3 − t)/2
D(√3(3 − t)/2, (3 + t)/2)

两种情况下 D 的纵坐标都是 (3 + t)/2。

分三类讨论

情况一：OB = DB

  1. OB = 3√3，需要 DB = 3√3
  2. D(√3(3 − t)/2, (3 + t)/2)（先看 t ≤ 3 的情况）
  3. DB² = [√3(3 − t)/2 − 3√3]² + [(3 + t)/2]² = [√3(3 − t − 6)/2]² + [(3 + t)/2]²
  4. = [√3(−3 − t)/2]² + [(3 + t)/2]² = (3(3 + t)²/4) + (3 + t)²/4 = t² + 6t + 9 = (t + 3)²
  5. DB² = 27 → (t + 3)² = 27 → t + 3 = ±3√3 → t = 3√3 − 3 或 t = −3√3 − 3
  6. t = 3√3 − 3 ≈ 2.196 ≤ 3 ✓ → C(0, 3√3 − 3)
  7. 再看 t ≥ 3：D(√3(t − 3)/2, (3 + t)/2)，DB² = 3(t − 3)²/4 + (t + 3)²/4，展开后令 = 27，可解得 t = 0（与 t ≥ 3 矛盾，舍去）。无解。

情况二：OD = OB

  1. OD = 3√3，OD² = 27
  2. t ≤ 3 时：D(√3(3 − t)/2, (3 + t)/2)
  3. OD² = 3(3 − t)²/4 + (3 + t)²/4 = [3(9 − 6t + t²) + (9 + 6t + t²)]/4 = [27 − 18t + 3t² + 9 + 6t + t²]/4 = [4t² − 12t + 36]/4 = t² − 3t + 9
  4. t² − 3t + 9 = 27 → t² − 3t − 18 = 0 → (t − 6)(t + 3) = 0 → t = 6 或 t = −3
  5. t = 6 ≥ 3 但我们在 t ≤ 3 的假设下，矛盾！需要换到 t ≥ 3 的公式验证。
  6. t = −3 ≤ 3 ✓ → C(0, −3)，但需验证：此时 D(3√3, 0) = B，CD = DA 是否成立？DA = BA = 6，CD = CB = √(9 + 27) = 6。✓ 但 D = B，此时 △ODB 退化为线段。舍去。
  7. t = 6 ≥ 3 时：D(3√3/2, 9/2)，OD² = 27/4 + 81/4 = 108/4 = 27 ✓ → C(0, 6)
  8. 验证：CD = DA？A(0,3)，D(3√3/2, 9/2)，DA² = 27/4 + 9/4 = 9，DA = 3。C(0,6)，CD² = 27/4 + 9/4 = 9，CD = 3。✓ OD² = 27/4 + 81/4 = 27，OD = 3√3 = OB。✓

情况三：OD = DB

  1. OD = DB → D 在 OB 的中垂线上 → D 的横坐标 = OB/2 = 3√3/2
  2. t ≤ 3 时：√3(3 − t)/2 = 3√3/2 → 3 − t = 3 → t = 0 → C(0, 0) = O
  3. 验证：C(0,0)，CA = 3，AD = 3/2。D(3√3/2, 3/2)，DA² = 27/4 + 9/4 = 9，DA = 3。等等，三线合一给出 AD = CA/2 = 3/2？不对。实际上 CD = DA，C(0,0)，D(3√3/2, 3/2)：CD² = 27/4 + 9/4 = 9，CD = 3。DA² = 27/4 + (3 − 3/2)² = 27/4 + 9/4 = 9，DA = 3。✓ OD² = 27/4 + 9/4 = 9，DB² = (3√3/2)² + (3/2)² = 27/4 + 9/4 = 9。OD = DB = 3。✓
  4. t ≥ 3 时：√3(t − 3)/2 = 3√3/2 → t − 3 = 3 → t = 6 → C(0, 6)
  5. 验证已在情况二中完成。C(0, 6) 同时满足 OD = OB 和 OD = DB（即 D 是 △OAB 的外心附近）。
最终答案：C(0, 0)（OB = DB，C 与 O 重合）、C(0, 3√3 − 3)（OB = DB，C 在 OA 上 A 下方）、C(0, 6)（同时满足 OD = OB 和 OD = DB）。
关键提醒：C(0, −3) 虽然解出了方程，但 D = B 导致 △ODB 退化（三点共线），必须舍去。等腰三角形存在性问题中，解出坐标后必须验证几何合理性。

六、边讲边问

  1. 为什么直线与 y 轴交点的横坐标一定是 0？与 x 轴交点的纵坐标一定是 0？
  2. tan∠BAO = OB/OA，这个式子为什么成立？在什么三角形中用的？
  3. 第(2)问中，C 的坐标为什么是 (0, x) 而不是 (x, 0)？
  4. "等腰三角形三线合一"是哪三线？为什么能帮我们推出 AF = CA/2？
  5. 第(3)问中，为什么等腰三角形要分三种情况讨论？如果题目说"以 D 为顶点的等腰三角形"，还需要三种情况吗？

七、易错提醒

  • C 的坐标方向搞错：OA 在 y 轴上，所以 C(0, x)，不是 (x, 0)。如果搞反，后面所有面积和距离计算全部出错。
  • 定义域写成闭区间：题目说"不与端点重合"，所以是开区间 (0, 3)，不是 [0, 3]。这是一个隐蔽但致命的细节。
  • 等腰三角形分类遗漏：△ODB 为等腰三角形，有 OD = OB、OD = DB、OB = DB 三种情况。漏掉任何一种都会丢解。
  • "射线"vs"线段"：第(2)问 C 在线段 OA 上（范围受限），第(3)问 C 在射线 OA 上（范围扩大，可以超出 O 和 A）。不注意这个区别会导致遗漏解。
  • 解出坐标不验证：第(3)问中 C(0, −3) 解出了方程但 D = B（三角形退化），必须舍去。代数解 ≠ 几何解。

八、一句话总结

求交点认特殊角 → 三线合一化线段 → 建面积函数定定义域 → 等腰三分类讨论逐一验证。

（四步走：认角、化线段、建函数、分类讨论）
