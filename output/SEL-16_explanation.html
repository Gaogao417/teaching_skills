视角：

学生版讲解：等腰梯形存在性——直线平移与分类讨论

题目：直线 l1 经过 O(0,0)、A(1,2) 两点，将直线 l1 向下平移 6 个单位得到直线 l2，交 x 轴于点 C，B 是直线 l2 上一点，且四边形 ABCO 是平行四边形。
(1) 求直线 l2 的表达式及 B 点的坐标；
(2) 若 D 是平面直角坐标系内的一点，且以 O、A、C、D 四个点为顶点的四边形是等腰梯形，求点 D 的坐标。

一、先补哪几个动作

做这道题之前，确认以下动作是否熟练：

  1. 直线平移：y = 2x 向下平移 6 个单位 → y = 2x − 6（上加下减，减的是常数项）。
  2. 求交点坐标：令 y = 0 代入直线方程，直接解出 x。
  3. 平行四边形的坐标判定：对角线中点相同（最常用的方法），或一组对边平行且相等。
  4. 斜率公式：k = (y₂ − y₁) / (x₂ − x₁)，用来判断两条边是否平行。
  5. 距离平方公式：|PQ|² = (x₂ − x₁)² + (y₂ − y₁)²，用来判断两条边是否等长。
小测：如果 y = 3x + 1 向上平移 4 个单位，方程怎么写？

二、这题先看懂什么

这道题分两步走：

第(1)问是基础——求直线方程、找交点、用平行四边形条件求点坐标，这些你都学过。

第(2)问是今天的新内容——给你三个点的坐标，让你找第四个点 D，使得 O、A、C、D 构成等腰梯形。

关键问题是：等腰梯形有几种不同的摆法？

三、第(1)问——先热个身

3.1 求 l1 和 l2

l1 过 O(0,0) 和 A(1,2)，两点确定一条直线：

        k = (2 − 0) / (1 − 0) = 2，所以 l1：y = 2x        

向下平移 6 个单位，就是在 y 后面减 6：

        l2：y = 2x − 6        

3.2 求 C 点坐标

C 是 l2 与 x 轴的交点。令 y = 0：

0 = 2x − 6 → x = 3，所以 C(3, 0)

3.3 求 B 点坐标（平行四边形条件）

四边形 ABCO 是平行四边形。平行四边形的对角线互相平分，即两条对角线的中点重合：

对角线 AC 的中点 = 对角线 BO 的中点

        AC 中点：((1+3)/2, (2+0)/2) = (2, 1)        

        BO 中点：((xB+0)/2, (yB+0)/2) = (xB/2, yB/2)        

令两者相等：xB/2 = 2, yB/2 = 1，所以 B(4, 2)

同时验证：B 在 l2 上吗？y = 2 × 4 − 6 = 2 ✓

四、第(2)问——等腰梯形的核心思路

4.1 等腰梯形是什么？

等腰梯形 = 一组对边平行 + 另一组对边相等（且不平行）。

4.2 已知什么、求什么？

已知三个点：(O(0,0)、A(1,2)、C(3,0))

求第四个点 D 的坐标，使 OACD 是等腰梯形。

4.3 最关键的一步——分类讨论

梯形有两条互相平行的边（叫做底），还有两条不平行但相等的边（叫做腰）。

我们有 4 个顶点 O、A、C、D，一共 3 种选择哪一对边平行：

情况平行的边（底）相等的边（腰）
情况一OA ∥ CDOC = AD
情况二AC ∥ ODOA = CD
情况三OC ∥ ADOA = CD
想一想：为什么不是 6 种？因为"OA ∥ CD"和"CD ∥ OA"是同一种情况。4 个顶点中选 2 条对边，只有 C(4,2) = 3 种组合。

五、标准解法——三种情况逐一求解

准备工具：斜率公式和距离公式

两点 P(x1, y1) 和 Q(x2, y2) 的：

斜率：k = (y2 − y1) / (x2 − x1)

距离：|PQ| = √[ (x2 − x1)² + (y2 − y1)² ]

已知关键值（先算好备用）

线段斜率长度（平方）
OA(2−0)/(1−0) = 21² + 2² = 5
OC(0−0)/(3−0) = 03² + 0² = 9
AC(0−2)/(3−1) = −12² + (−2)² = 8

情况一：OA ∥ CD，且 OC = AD

①平行条件：kCD = kOA = 2

设 D(a, b)，则：kCD = (b − 0) / (a − 3) = 2

得到：b = 2(a − 3) …… ①

②等腰条件：|OC|² = |AD|²

|OC|² = 9（已算好）

|AD|² = (a − 1)² + (b − 2)² = 9 …… ②

③把 ① 代入 ②：

(a − 1)² + (2a − 6 − 2)² = 9

(a − 1)² + (2a − 8)² = 9

a² − 2a + 1 + 4a² − 32a + 64 = 9

5a² − 34a + 56 = 0

(5a − 14)(a − 4) = 0

a = 14/5 或 a = 4

④验证：

  • a = 14/5 → b = 2(14/5 − 3) = −2/5 → D(14/5, −2/5)
  • a = 4 → b = 2(4 − 3) = 2 → D(4, 2) = B，此时 OACD 变成平行四边形（OA ∥ CD 且 OC ∥ AD），不是梯形，舍去。

情况一结果：D(14/5, −2/5)

情况二：AC ∥ OD，且 OA = CD

①平行条件：kOD = kAC = −1

b / a = −1 → b = −a …… ①

②等腰条件：|OA|² = |CD|²

|OA|² = 5，|CD|² = (a − 3)² + (b)² = 5 …… ②

③把 b = −a 代入 ②：

(a − 3)² + a² = 5

2a² − 6a + 4 = 0

a² − 3a + 2 = 0

(a − 1)(a − 2) = 0

a = 1 或 a = 2

④验证：

  • a = 1 → b = −1 → D(1, −1)。验证另一组对边：OA 斜率 2，CD 斜率 = (−1−0)/(1−3) = 1/2 ≠ 2 ✓（不平行，是梯形）
  • a = 2 → b = −2 → D(2, −2)。验证另一组对边：OA 斜率 2，CD 斜率 = (−2−0)/(2−3) = 2 = OA 斜率 → OA ∥ CD，退化为平行四边形，舍去。

情况二结果：D(1, −1)

情况三：OC ∥ AD，且 OA = CD

①平行条件：kAD = kOC = 0

A(1,2) 和 D(a,b) 的斜率 = 0 → b − 2 = 0 → b = 2 …… ①

②等腰条件：|OA|² = |CD|²

(a − 3)² + (2 − 0)² = 5

(a − 3)² + 4 = 5

(a − 3)² = 1

a − 3 = ±1 → a = 4 或 a = 2

③验证：

  • a = 4 → D(4, 2) = B，又变成平行四边形，舍去。
  • a = 2 → D(2, 2)。检查：OC 斜率 0，AD 斜率 = (2−2)/(2−1) = 0 → OC ∥ AD ✓。两腰 OA 斜率 2，CD 斜率 = (2−0)/(2−3) = −2 ≠ 2 ✓（不平行，是梯形）。

情况三结果：D(2, 2)

最终答案

第(1)问：l2: y = 2x − 6，B(4, 2)

第(2)问：D(14/5, −2/5) 或 D(2, 2) 或 D(1, −1)

六、边讲边问

  1. 直线 l1: y = 2x 向下平移 6 个单位后，方程怎么写？为什么不是 y = 2(x − 6)？
  2. 四边形 ABCO 是平行四边形，你能说出两种不同的判定方法吗？（提示：对角线中点 / 平行且相等）
  3. 等腰梯形的两条"底"必须平行，两条"腰"必须相等但不平行。如果我们有 4 个顶点，一共能选出多少种不同的"哪两条边是底"的组合？
  4. 在情况一中，a = 4 对应的 D 点为什么被舍去了？
  5. 如果这道题问的是"直角梯形"而不是"等腰梯形"，分类讨论的方法有什么不同？

七、易错提醒

  • 平移方向搞反：向上平移是 y = 2x + 6，向下才是 −6。记住"上加下减"。
  • 漏掉情况：梯形的三种平行对边组合都要讨论。最容易漏掉的是"AC ∥ OD"这一种，因为 OA 和 OC 是坐标轴方向比较直观，而 AC 是斜线。
  • 忘记验证退化：解出来的 D 点可能让四边形变成平行四边形（两组对边都平行），这种情况不是梯形，必须舍去。
  • 距离公式中忘记平方：等腰条件列方程时，两边都要用"距离的平方"，避免出现根号使计算变复杂。

八、一句话总结

梯形存在性三步走：① 列出三组可能的平行对边 → ② 每组列"平行 + 等腰"两个方程 → ③ 解出后验证不是平行四边形。
