学生版讲解：分段函数——水费计费

一、这题先看懂什么

这道题说的是水费怎么收。你一年用了 x 立方米的水，要交 y 元水费。关键是：用的多，后面的水更贵。

规则其实就一句话：前面220立方米以内，一个价；超过220的部分，另一个（更高的）价。

题目里还有一笔污水处理费——不管哪个档次，每用1立方米都要额外加1.53元。这笔钱要叠加上去。

想一想：如果一户人家一年只用了100立方米水，应该按哪个档次收费？如果用了300立方米呢？

二、第一步该做什么

拿到这类题，不要急着算数字。第一步是：

找分界点，画"分段线"。

这道题的分界点很明确：220立方米。

区间 收费方式
0 < x ≤ 220（第一段） 每立方米按"正常价"收
x > 220（第二段） 前220按正常价，超出的按"加价"收

有了分界点，你就知道要写两个函数解析式——这就是"分段函数"。

想一想：分界点 x=220 这个位置，应该归到哪一段？为什么？

三、关键转化怎么想

把题目的话变成数学式子，需要两步转化：

转化1：把"每项费用"加起来，得到一个"合计单价"

题目说每立方米收水费1.92元，还要加污水处理费1.53元。合起来：

合计单价 = 1.92 + 1.53 = 3.45（元/立方米）

转化2：第二段不能直接用高单价乘全部水量

第二段的水费由两部分组成：

部分A：前220立方米，按正常价 3.45 元/立方米
    → 费用 = 3.45 × 220 = 759 元

部分B：超出220的部分，共 (x − 220) 立方米，按加价收
    → 超出部分单价 = 3.30 + 1.53 = 4.83 元/立方米
    → 费用 = 4.83 × (x − 220)
想一想：如果不把"部分A"和"部分B"分开算，直接写成 y = 4.83x，结果会怎样？和正确答案差多少？

四、标准解法

第(1)问：写出函数解析式

第一段（0 < x ≤ 220）：

  1. 合计单价 = 1.92 + 1.53 = 3.45（元/立方米）
  2. y = 3.45x，定义域 0 < x ≤ 220

第二段（x > 220）：

  1. 前220立方米费用 = 3.45 × 220 = 759 元
  2. 超出部分费用 = 4.83 × (x − 220)
  3. y = 759 + 4.83(x − 220)
  4. 展开化简：y = 759 + 4.83x − 1062.6 = 4.83x − 303.6
  5. 定义域 x > 220
最终答案：
当 0 < x ≤ 220 时，y = 3.45x
当 x > 220 时，y = 4.83x − 303.6

第(2)问：已知水费1000.5元，求用水量

关键一步：先判断落在哪一段！

  1. 算分界点的水费：当 x = 220 时，y = 3.45 × 220 = 759 元
  2. 比较：1000.5 > 759 → 说明用水量超过了220立方米 → 用第二段
  3. 代入第二段解方程：
    4.83x − 303.6 = 1000.5
    4.83x = 1304.1
    x = 1304.1 ÷ 4.83 = 270
  4. 检验：270 > 220，在第二段定义域内，正确。
答：这户全年用水量为 270 立方米。
注意：如果直接用第一段反除：x = 1000.5 ÷ 3.45 = 290，但 290 > 220，不在第一段定义域内！所以必须先判断落在哪一段。

五、边讲边问

  1. 如果某户用了恰好220立方米水，水费是多少？
  2. 在第二段中，4.83 和 303.6 这两个数字分别是怎么来的？
  3. 如果一户人家的水费恰好是759元，能确定用水量吗？用哪一段来算？
  4. 为什么第(2)问不能直接用"水费 ÷ 单价"得到答案？
  5. 如果把污水处理费去掉（只有水费1.92元和加价3.30元），解析式会怎么变？

六、易错提醒

  • 忘加污水处理费：题目说"加收每立方米1.53元"，这是每立方米都要加的，不是一笔固定费用。不加上去，整道题都不对。
  • 第二段直接 y = 4.83x：这是最常见的错误。超出部分的加价只适用于"超出220以后的水量"，不适用于全部水量。正确做法是：前段总额 + 超出部分单价 × (x − 220)。
  • 第(2)问不验证区间：算出 x = 290 后没有回头检查 290 > 220，就把答案写上去了。分段函数问题中，算完必须回头检查结果是否在所选区间内。

七、一句话总结

找分界 → 分段列式 → 反代先判段 → 解完回头看。

（四步走：分界点、分段解析式、判断区间、验证结果）

教师说明：本讲解面向初二学生，假设学生已掌握一次函数的基本概念和方程求解。重点训练"从文字规则中提取分段结构"和"反向代入时先判断区间"两个动作。建议在"边讲边问"环节逐题检查，重点关注学生对第二段列式逻辑的理解。
