学生版讲解：一次函数最值——购机利润问题

题目：某商店销售A型和B型两种电脑，销售一台A型获利120元，B型获利140元。计划一次购进两种电脑共100台，B型不超过A型的3倍。设购进A型 x 台，总利润为 y 元。
(1) 求 y 与 x 的关系式；(2) 购进A型、B型各多少台利润最大？(3) 若最多购进A型60台，总利润能否为13600元？若不能，求利润范围。

一、这题先看懂什么

这道题说的是开店进货怎么赚最多。两种电脑，A型每台赚120元，B型每台赚140元。一共进100台，但有限制条件：B型的数量不能超过A型的3倍。

我们需要解决三个问题：

  1. 列关系式——利润 y 和A型台数 x 之间是什么函数？
  2. 求最值——在限制条件下，怎么进负能赚最多？
  3. 判断可达性——再加一个限制后，能不能赚到某个特定金额？

本质上，这就是在限定范围内找一个一次函数的最大值和值域。

想一想：如果没有任何限制条件，进哪种电脑更赚钱？为什么还要有限制条件？

二、第一步该做什么

拿到这类题，不要急着列式。第一步是：

把所有文字条件翻译成数学语言——变量、关系式、不等式。

具体来说，要搞清楚三件事：

  1. 变量是什么？ → A型 x 台，B型就是 (100 − x) 台
  2. 目标函数是什么？ → 总利润 = A型利润 + B型利润
  3. 限制条件是什么？ → "B型不超过A型的3倍"翻译成不等式

这三件事搞清楚了，后面的求解就像走流程一样简单。

想一想："B型不超过A型的3倍"这句话，用数学式子怎么写？（提示：B型数量 = 100 − x，A型数量 = x）

三、关键转化怎么想

这道题有两步关键转化：

转化1：把"利润"变成一个函数

总利润 = A型赚的钱 + B型赚的钱，每一项都是"单价 × 数量"：

A型利润：120 × x
B型利润：140 × (100 − x)
总利润：y = 120x + 140(100 − x)

展开化简：y = 120x + 14000 − 140x = −20x + 14000

这是一个一次函数，而且 k = −20 < 0，说明它是递减的——A型进得越多，利润反而越少（因为B型单台更赚钱）。

转化2：把"B不超过A的3倍"变成不等式

逐字翻译：

"B型" → 100 − x
"不超过" → ≤
"A型的3倍" → 3x

所以：100 − x ≤ 3x
解得：100 ≤ 4x，即 x ≥ 25

再加上隐含条件（不能买负数台）：x ≥ 0，100 − x ≥ 0（即 x ≤ 100）。

综合起来，x 的取值范围是：

25 ≤ x ≤ 100
想一想：如果题目说"B型至少是A型的2倍"，不等式应该怎么写？解出来 x 的范围是什么？

四、标准解法

第(1)问：求 y 与 x 的关系式

  1. A型 x 台，B型 (100 − x) 台
  2. y = 120x + 140(100 − x)
  3. 化简：y = 120x + 14000 − 140x = −20x + 14000
答：y = −20x + 14000（其中 25 ≤ x ≤ 100）

第(2)问：购进各多少台利润最大？

这是这道题的核心。关键就一句话：

看 k 的符号，决定取哪个端点。
  1. y = −20x + 14000，k = −20 < 0 → 函数递减
  2. 递减函数：越往左（x 越小），y 越大
  3. 所以最大值在 x = 25（范围的左端点）
  4. 代入：ymax = −20 × 25 + 14000 = −500 + 14000 = 13500 元
  5. 此时A型25台，B型 100 − 25 = 75 台
答：购进A型25台、B型75台时，利润最大，为13500元。
注意：如果忘了"xB≤3A"这个约束条件，会以为 x 的范围是 0 ≤ x ≤ 100，然后在 x = 0 处取最大值。但 x = 0 意味着不进A型全进B型，B型 = 100台，A型 = 0台，此时"0的3倍 = 0"，而100 > 0，不满足约束！

第(3)问：最多进A型60台，利润能否为13600元？

这里多了一个限制：x ≤ 60。和原来的 x ≥ 25 合在一起：

25 ≤ x ≤ 60
  1. 先解方程：令 −20x + 14000 = 13600
    −20x = −400
    x = 20
  2. 验证范围：x = 20，但范围是 25 ≤ x ≤ 60。
    x = 20 不在范围内！
  3. 结论：不能达到13600元
  4. 求利润范围：函数仍然递减，在 x = 25 时最大，x = 60 时最小
    x = 25 时：y = −20 × 25 + 14000 = 13500 元
    x = 60 时：y = −20 × 60 + 14000 = −1200 + 14000 = 12800 元
答：不能达到13600元。利润范围为 12800 ≤ y ≤ 13500（单位：元）。

五、边讲边问

  1. 为什么 y = −20x + 14000 中 k 是负数？这和A型、B型谁的利润更高有什么关系？
  2. 如果题目改成"A型不超过B型的3倍"，不等式应该怎么写？解出来范围是什么？
  3. 第(2)问中，如果函数是递增的（k > 0），最大值应该取哪个端点？
  4. 第(3)问中，如果不验证范围直接说"能，x = 20"，哪里出了问题？
  5. 如果题目把"最多购进A型60台"改成"最多购进B型80台"，新的 x 范围是什么？

六、易错提醒

  • 漏翻译约束条件："B型不超过A型的3倍"必须翻译成 100 − x ≤ 3x。很多同学直接用 0 ≤ x ≤ 100 当范围，导致最值取错位置。记住：每个文字限制都要变成一个不等式。
  • 递减取端点方向搞反：k < 0 时函数递减，最大值在最小的 x处（左端点）。容易习惯性地以为"最大值在最大的地方"。
  • 第(3)问解完方程不验证：解出 x = 20 就下结论"能"，但20不在范围 [25, 60] 内。凡是涉及不等式约束的问题，算完必须回头检查范围。

七、一句话总结

列函数 → 翻约束定范围 → 看 k 的符号 → 端点取最值。

（四步走：建模、定域、看单调性、取端点）

教师说明：本讲解面向初二学生，假设学生已掌握一次函数基本概念和一元一次不等式求解。核心训练目标：(1) 将文字约束翻译为不等式；(2) 利用 k 的符号判断单调性，在闭区间端点处取最值；(3) 解方程后验证范围。建议在"边讲边问"环节重点关注学生对"递减函数最大值在左端点"的理解。
