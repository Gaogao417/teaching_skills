学生版讲解：三定一动平行四边形存在性问题

一、这题先看懂什么

题目给了我们一条直线 $y=-2x+4$，它和 $x$ 轴交于 $A$，和 $y$ 轴交于 $B$。

想象一下：这条直线从 $y$ 轴正半轴的某个位置出发，往右下方走，穿过 $x$ 轴正半轴。

第(1)问让你找到这两个"着陆点"的坐标——这是后面所有问题的地基。

第(2)问在 $AB$ 上找一点 $C$，使得 $BCOD$（$O$ 是原点）拼成一个菱形，然后求 $D$ 的坐标。

第(3)问是最关键的——已知 $B$、$D$、$A$ 三个点的坐标，要你找出所有可能的 $P$，使四边形 $BDAP$ 是平行四边形。注意"所有"两个字——不止一个答案。

想一想：你能先在草稿纸上画出 $y=-2x+4$ 的大致图像吗？标出 $A$ 和 $B$ 大概在什么位置。

二、第一步该做什么

第(1)问是整道题的入口。不要跳过它直接看后面——后面所有坐标都依赖 $A$ 和 $B$ 的值。

找直线与 $x$ 轴交点：令 $y=0$，解 $-2x+4=0$，得 $x=2$，所以 $A(2,\ 0)$。
找直线与 $y$ 轴交点：令 $x=0$，得 $y=4$，所以 $B(0,\ 4)$。

记住这个操作：与 $x$ 轴交点令 $y=0$；与 $y$ 轴交点令 $x=0$。

想一想：为什么 $A$ 的纵坐标一定是 $0$？$B$ 的横坐标一定是 $0$？

三、关键转化怎么想

第(2)问：菱形怎么求？

菱形的定义是四条边都相等的四边形。

题目说 $BCOD$ 是菱形，其中 $O(0,0)$、$B(0,4)$ 已知，$C$ 在 $AB$ 上，$D$ 在第二象限。

关键动作：四条边相等 → 先用两个条件确定 $C$，再用剩余条件确定 $D$。

步骤一：确定 $C$。
$C$ 在线段 $AB$ 上，设 $C$ 把 $AB$ 分成某个比例。菱形要求 $|BC|=|CO|$：
$B(0,4)$，$O(0,0)$，$C$ 在从 $A(2,0)$ 到 $B(0,4)$ 的线段上。
用参数法：$C = (2-2s,\ 4s)$，其中 $s \in [0,1]$。
$|BC|^2 = (2-2s)^2 + (4s-4)^2 = 4(1-s)^2 + 16(1-s)^2 = 20(1-s)^2$
$|CO|^2 = (2s-2)^2 + (-4s)^2 = 4(1-s)^2 + 16s^2$
令 $|BC|=|CO|$：$20(1-s)^2 = 4(1-s)^2 + 16s^2$
化简：$16(1-s)^2 = 16s^2$，即 $(1-s)^2 = s^2$
解得 $s = \dfrac{1}{2}$（$s=1$ 时 $C=B$，四边形退化，舍去）。
所以 $C\!\left(1,\ 2\right)$，即 $C$ 是 $AB$ 的中点。
步骤二：确定 $D$。
菱形边长 $|BC| = \sqrt{20 \times \dfrac{1}{4}} = \sqrt{5}$。
$|OD| = |BC| = \sqrt{5}$，所以 $D_x^2 + D_y^2 = 5$。（①）
$|DB| = |BC| = \sqrt{5}$，所以 $D_x^2 + (4-D_y)^2 = 5$。（②）
由 ②-①：$(4-D_y)^2 - D_y^2 = 0$，即 $16 - 8D_y = 0$，$D_y = 2$。
代入 ①：$D_x^2 + 4 = 5$，$D_x^2 = 1$，$D_x = \pm 1$。
$D$ 在第二象限 → $D_x < 0$，所以 $D(-1,\ 2)$。
想一想：为什么 $s=1$ 要舍去？如果 $C=B$，四边形 $BCOD$ 会变成什么形状？

第(3)问：三定一动平行四边形——这道题的核心

这是本讲的重中之重。已知三个顶点 $B(0,4)$、$D(-1,2)$、$A(2,0)$，求所有使 $BDAP$ 为平行四边形的 $P$。

核心方法：中点坐标法。

平行四边形的对角线互相平分，所以两条对角线的中点是同一个点。

中点坐标法：若 $M(x_1,y_1)$ 和 $N(x_2,y_2)$ 的中点为 $\left(\dfrac{x_1+x_2}{2},\ \dfrac{y_1+y_2}{2}\right)$。
平行四边形的两条对角线中点重合 → 列方程组，解出未知顶点。

关键动作：三个定点，有几种"配对"方式？

从 $B$、$D$、$A$ 中选两个作为对角线的端点，第三个定点和 $P$ 作为另一条对角线的端点。

选 2 个点的方式有 $C_3^2 = 3$ 种，所以 $P$ 有 3 个可能的值：

情况对角线 1 的端点对角线 2 的端点
情况一$B(0,4)$ 和 $D(-1,2)$$A(2,0)$ 和 $P(x,y)$
情况二$D(-1,2)$ 和 $A(2,0)$$B(0,4)$ 和 $P(x,y)$
情况三$B(0,4)$ 和 $A(2,0)$$D(-1,2)$ 和 $P(x,y)$
想一想：为什么不可能是 $BD$ 和 $PA$ 以外的组合？比如 $BP$ 和 $DA$？

四、标准解法

第(1)问

  1. 令 $y=0$：$-2x+4=0$，$x=2$，所以 $A(2,\ 0)$。
  2. 令 $x=0$：$y=4$，所以 $B(0,\ 4)$。

第(2)问

  1. 设 $C$ 在 $AB$ 上：$C = (2-2s,\ 4s)$。
  2. 菱形条件 $|BC|=|CO|$：解得 $s=\dfrac{1}{2}$，$C(1,2)$。
  3. 边长 $|BC|=\sqrt{5}$。由 $|OD|=\sqrt{5}$ 且 $|DB|=\sqrt{5}$ 且 $D$ 在第二象限，解得 $D(-1,2)$。

第(3)问（中点坐标法，三步走）

情况一：$BD$ 与 $AP$ 为对角线
$BD$ 中点：$\left(\dfrac{0+(-1)}{2},\ \dfrac{4+2}{2}\right) = \left(-\dfrac{1}{2},\ 3\right)$
$AP$ 中点：$\left(\dfrac{2+x}{2},\ \dfrac{0+y}{2}\right)$
中点相等：$\dfrac{2+x}{2} = -\dfrac{1}{2} \Rightarrow x = -3$；$\dfrac{y}{2} = 3 \Rightarrow y = 6$
$\boxed{P(-3,\ 6)}$
情况二：$DA$ 与 $BP$ 为对角线
$DA$ 中点：$\left(\dfrac{-1+2}{2},\ \dfrac{2+0}{2}\right) = \left(\dfrac{1}{2},\ 1\right)$
$BP$ 中点：$\left(\dfrac{0+x}{2},\ \dfrac{4+y}{2}\right)$
中点相等：$\dfrac{x}{2} = \dfrac{1}{2} \Rightarrow x = 1$；$\dfrac{4+y}{2} = 1 \Rightarrow y = -2$
$\boxed{P(1,\ -2)}$
情况三：$AB$ 与 $DP$ 为对角线
$AB$ 中点：$\left(\dfrac{0+2}{2},\ \dfrac{4+0}{2}\right) = (1,\ 2)$
$DP$ 中点：$\left(\dfrac{-1+x}{2},\ \dfrac{2+y}{2}\right)$
中点相等：$\dfrac{-1+x}{2} = 1 \Rightarrow x = 3$；$\dfrac{2+y}{2} = 2 \Rightarrow y = 2$
$\boxed{P(3,\ 2)}$

答案汇总：$P(-3,6)$、$P(1,-2)$、$P(3,2)$。

五、边讲边问

  1. 求直线与坐标轴的交点时，分别令什么等于 $0$？为什么？
  2. 菱形的四条边都相等，为什么我们只用了 $|BC|=|CO|$ 这一个条件就能确定 $C$？（提示：如果 $|BC|=|CO|$ 且四边形四边相等，还需要什么？）
  3. 中点坐标公式是什么？如果有两点 $(x_1,y_1)$ 和 $(x_2,y_2)$，中点坐标怎么写？
  4. 三个定点、一个动点组成平行四边形，一共有几种情况？为什么是 3 种而不是 4 种？
  5. 如果这道题只要求你画图找 $P$，你觉得最容易漏掉哪个 $P$？为什么？

六、易错提醒

易错 1：第(3)问只找到一个或两个 $P$。
画图时，如果 $P$ 落在图象之外（比如 $P(-3,6)$ 在第二象限比较远的位置），很容易漏掉。必须用中点坐标法穷举所有 3 种配对，不要只靠画图。
易错 2：中点公式写错。
中点是 $\left(\dfrac{x_1+x_2}{2},\ \dfrac{y_1+y_2}{2}\right)$，注意是坐标相加除以 2，不是坐标相减。写错公式会导致所有答案都错。
易错 3：忘记检查象限条件。
第(2)问 $D$ 在第二象限，所以 $D_x = -1$ 而不是 $D_x = 1$。如果忽略象限条件，会多出一个错误答案。

七、一句话总结

三定一动平行四边形：列出 $C_3^2 = 3$ 种对角线配对 → 对每种情况用中点坐标法列方程 → 解出 3 个 $P$ 点，不遗漏不重复。
