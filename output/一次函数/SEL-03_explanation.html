学生版讲解：一次函数与坐标轴围成面积

题目：已知一次函数 $y = kx + b$（$k$、$b$ 是常数）的图像平行于直线 $y = -3x$，且经过点 $(2,-3)$。

（1）求这个一次函数的解析式；

（2）求这个一次函数与两坐标轴所围成的图形面积。

一、这题先看懂什么

这道题给了我们两个信息：

  • 信息一：函数图像平行于 $y = -3x$。意思是：它和 $y = -3x$ 一样"陡"，斜率相同，只是位置不同。
  • 信息二：图像经过点 $(2,-3)$。意思是：把 $x=2$ 代进去，算出来的 $y$ 必须等于 $-3$。

问题分两步：先求出函数式子，再算它和坐标轴围出来的面积。

二、第一步该做什么

第（1）问的目标是把 $k$ 和 $b$ 都确定下来。

"平行"这个词直接告诉我们一件事：两条直线的 $k$（斜率）相等。$y = -3x$ 的 $k$ 就是 $-3$，所以我们要找的直线也是 $k = -3$。

想一想：如果两条直线平行，它们的斜率是什么关系？

知道了 $k = -3$，还差 $b$。这时用第二个信息——经过 $(2,-3)$：

把 $x = 2$、$y = -3$ 代入 $y = -3x + b$：

$$-3 = -3 \times 2 + b \implies -3 = -6 + b \implies b = 3$$

所以解析式为 $y = -3x + 3$。

三、关键转化：面积怎么算？

第（2）问要求"与两坐标轴所围成的图形面积"。这是什么图形？

  • 直线 $y = -3x + 3$ 和 $x$ 轴交于一点，和 $y$ 轴交于一点。
  • $x$ 轴和 $y$ 轴本身互相垂直。
  • 所以围成的图形是一个直角三角形！

要算面积，只要找到三角形的底和高。底和高分别就是两个交点的坐标值。

面积 $S = \dfrac{1}{2} \times \text{底} \times \text{高} = \dfrac{1}{2} \times |x_{\text{截距}}| \times |y_{\text{截距}}|$

四、标准解法

第（1）问：求解析式

  1. 确定 $k$：平行于 $y = -3x$，所以 $k = -3$。
    为什么？平行直线的斜率相等。
  2. 代入定点求 $b$：把 $(2, -3)$ 代入 $y = -3x + b$：
    $-3 = -3 \times 2 + b = -6 + b$，所以 $b = 3$。
  3. 写出解析式：$\boxed{y = -3x + 3}$

第（2）问：求面积

  1. 求与 $y$ 轴的交点：令 $x = 0$，得 $y = 3$，交点为 $(0, 3)$。
  2. 求与 $x$ 轴的交点：令 $y = 0$，得 $0 = -3x + 3$，解得 $x = 1$，交点为 $(1, 0)$。
  3. 算面积：
    $S = \dfrac{1}{2} \times 1 \times 3 = \dfrac{3}{2}$。
  4. 答：围成图形的面积为 $\boxed{\dfrac{3}{2}}$。

拓展：通用公式

对于一般一次函数 $y = kx + b$（$k \neq 0$），与坐标轴交点为：

  • $y$ 轴交点：$(0,\, b)$，截距长度 $= |b|$
  • $x$ 轴交点：令 $y = 0$ 得 $x = -\dfrac{b}{k}$，截距长度 $= \left|\dfrac{b}{k}\right|$

所以面积通用公式为：

$S = \dfrac{1}{2} \times \left|\dfrac{b}{k}\right| \times |b| = \dfrac{b^2}{2|k|}$

验证本题：$S = \dfrac{3^2}{2 \times |-3|} = \dfrac{9}{6} = \dfrac{3}{2}$，和之前算的一样！

五、边讲边问

  1. 两条直线平行，它们的斜率有什么关系？
  2. 已知直线 $y = 2x - 1$，把它向上平移 3 个单位后，新直线怎么写？
  3. 求一次函数与 $y$ 轴的交点，只需要令什么等于 0？
  4. 一次函数 $y = 4x - 8$ 与坐标轴围成的面积是多少？（用通用公式试试）
  5. 如果一次函数 $y = kx + 5$ 与坐标轴围成的面积是 $\dfrac{25}{4}$，求 $k$ 的值。

六、易错提醒

  • 忘记取绝对值：截距可能是负数（比如交在 $x$ 轴负半轴），但面积要用正的长度来算。公式里 $|b|$ 和 $\left|\dfrac{b}{k}\right|$ 的绝对值不能省！
  • 底和高对应错误：$x$ 轴截距是"底"方向的长度，$y$ 轴截距是"高"方向的长度，别搞混。记住：令 $x=0$ 得 $y$ 截距，令 $y=0$ 得 $x$ 截距。
  • 漏乘 $\dfrac{1}{2}$：面积 $= \dfrac{1}{2} \times \text{底} \times \text{高}$，这个 $\dfrac{1}{2}$ 经常被忘记，尤其是在心算的时候。

七、一句话总结

平行定 $k$，过点定 $b$；令 $x=0$、$y=0$ 找截距；面积 $= \dfrac{1}{2} \times |x_{\text{截}}| \times |y_{\text{截}}|$，记住公式 $S = \dfrac{b^2}{2|k|}$ 更快。
