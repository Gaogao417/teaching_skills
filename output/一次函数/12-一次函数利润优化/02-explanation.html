一次函数与实际应用（利润优化）

一、先把题目拆开

已知条件 数学含义
销售一台A型获利120元，B型获利140元 A型单价利润 $a=120$，B型单价利润 $b=140$
计划购进两种电脑共100台 $x + (100 - x) = 100$，即A型 $x$ 台，B型 $100 - x$ 台
B型不超过A型的3倍 $100 - x \leq 3x$ → $x \geq 25$
第(3)问附加：最多购进A型60台 $x \leq 60$，与 $x \geq 25$ 合并为 $25 \leq x \leq 60$
求解目标 对应操作
(1) 求 $y$ 与 $x$ 的关系式 用总利润公式列函数，化简为一次函数标准形式
(2) 各购进多少台利润最大 确定 $x$ 的范围，判断单调性，端点取最值
(3) 总利润能否为13600元？ 代入解方程，验证 $x$ 是否在范围内；求值域

二、这题准备怎么走

  1. 列函数——设A型 $x$ 台，用总利润 = A型利润 + B型利润列出 $y$ 与 $x$ 的关系式并化简。
  2. 定范围——把所有文字约束（总数100、B≤3A）翻译为不等式，求交集得到 $x$ 的取值区间。
  3. 看单调性——判断一次函数 $k$ 的正负，确定递增还是递减。
  4. 取端点——递减函数的最大值在左端点（最小的 $x$），直接代入即可。
  5. 验范围——第(3)问解方程后，一定要回头检验 $x$ 是否落在允许范围内。

三、关键想法

核心策略：建模 → 定域 → 看单调性取端点。

这类利润优化问题的本质是：在闭区间上求一次函数的最值。只要完成三步：

1. 把文字翻译成函数 $y = -20x + 14000$；

2. 把文字约束翻译成不等式，确定 $x \in [25,\, 100]$；

3. 因为 $k = -20 < 0$（递减），最大值就在最左端 $x = 25$ 处。

不需要逐个代入试算，看一眼单调性就能锁定答案。

四、标准解法

第(1)问：求 $y$ 与 $x$ 的关系式

Step 1：设变量、列式

设购进A型电脑 $x$ 台，则B型电脑 $100 - x$ 台。

总利润 $y$ = A型利润 + B型利润：

$$y = 120x + 140(100 - x)$$

Step 2：化简

$$y = 120x + 14000 - 140x = -20x + 14000$$

即 $y = -20x + 14000$（$x$ 为正整数）。

第(2)问：各购进多少台利润最大？

Step 1：翻译约束条件，确定 $x$ 的范围

隐含条件：$x \geq 0$，$100 - x \geq 0$，即 $0 \leq x \leq 100$。

显式条件：B型不超过A型的3倍：

$$100 - x \leq 3x \;\Longrightarrow\; 4x \geq 100 \;\Longrightarrow\; x \geq 25$$

取交集：$25 \leq x \leq 100$。

Step 2：判断单调性

$y = -20x + 14000$ 中，$k = -20 < 0$，函数在 $[25,\, 100]$ 上单调递减。

Step 3：取端点求最大值

递减函数：$x$ 越小，$y$ 越大。所以最大值在左端点 $x = 25$。

$$y_{\max} = -20 \times 25 + 14000 = -500 + 14000 = 13500 \text{（元）}$$

此时 A型 $x = 25$ 台，B型 $100 - 25 = 75$ 台。

答：购进A型25台、B型75台时利润最大，最大利润为13500元。

第(3)问：总利润能否为13600元？

Step 1：附加约束，缩小区间

新增条件：最多购进A型60台，即 $x \leq 60$。

与原范围 $x \geq 25$ 取交集：$25 \leq x \leq 60$。

Step 2：代入目标值，解方程

$$-20x + 14000 = 13600$$ $$-20x = -400$$ $$x = 20$$

Step 3：验证范围（关键！）

$x = 20 < 25$，不在范围 $[25,\, 60]$ 内，所以总利润不能达到13600元。

Step 4：求利润范围

函数仍为递减，在 $[25,\, 60]$ 上：

$$y(25) = -20 \times 25 + 14000 = 13500 \text{（元）}$$ $$y(60) = -20 \times 60 + 14000 = -1200 + 14000 = 12800 \text{（元）}$$

利润范围为 $12800 \leq y \leq 13500$（元）。

五、边讲边问

Q1

"B型不超过A型的3倍"写成不等式，应该是 $100 - x \leq 3x$ 还是 $100 - x \geq 3x$？为什么？

Q2

$y = -20x + 14000$ 中的 $k = -20 < 0$，说明函数图像从左到右是上升还是下降？$y$ 随 $x$ 增大而______。

Q3

递减函数在闭区间 $[25,\, 100]$ 上，最大值在 $x =$ ______ 处取得，最小值在 $x =$ ______ 处取得。

Q4

第(3)问解出 $x = 20$，如果不去验证范围就直接下结论，会犯什么错误？

Q5

第(3)问利润范围是 $12800 \leq y \leq 13500$。如果函数是递增的（$k > 0$），同样的区间 $[25,\, 60]$，利润范围的端点会怎么变？

六、易错提醒

错误1：漏掉"B ≤ 3A"的约束

只看到"共100台"，就把范围写成 $0 \leq x \leq 100$，导致最大值取在 $x = 0$ 处，答案错误。

对策：逐条圈出题目中的约束条件，每一条都要翻译为不等式。

错误2：化简 $y = 120x + 140(100 - x)$ 时算错

常见错误：忘记展开括号，或 $120x - 140x$ 的符号搞反。

对策：分两步——先展开 $140 \times 100 = 14000$ 和 $140 \times (-x) = -140x$，再合并同类项 $120x - 140x = -20x$。

错误3：递减函数的端点取反

知道函数递减，却误以为最大值在右端点 $x = 100$ 处。

对策：记住口诀——"递减往左取最大，递增往右取最大"。

错误4：第(3)问解出 $x = 20$ 后不验证范围

直接回答"能"，忽略了 $x$ 必须在 $[25,\, 60]$ 内这一前提。

对策：凡解方程得到的值，必须回头检查是否落在自变量的允许范围内。

错误5：附加约束后范围漏掉下界

第(3)问看到"最多购进A型60台"，就把范围写成 $0 \leq x \leq 60$，漏掉了 $x \geq 25$。

对策：附加约束是与原有约束取交集，不是替换。原来的下界 $x \geq 25$ 仍然有效。

七、一句话总结

利润优化题就是线性约束下一次函数最值问题，万能三步走：

建模（列函数）→ 定域（翻译不等式）→ 看单调性取端点。

解出答案后，别忘了回头验证范围——这是拿满分的最后一步。
