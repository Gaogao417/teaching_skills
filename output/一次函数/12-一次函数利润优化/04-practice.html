一次函数与实际应用（利润优化）

练习说明

本练习围绕一次函数建模与最值优化展开，共 3 题，难度递进。

每道题包含：题目 → 提示 → 解答区。请先独立完成，再看提示或答案。

核心方法口诀：建模 → 定域 → 看单调性取端点。

当前档位：E档（基础巩固） 完成全部练习后可进行自检。

练习一

第 1 题：服装店进货利润问题

某服装店销售甲款和乙款两种外套。甲款每件获利 80 元，乙款每件获利 120 元。计划一次购进两种外套共 50 件，且乙款件数不超过甲款件数的 2 倍。

设购进甲款外套 $x$ 件，总利润为 $y$ 元。

(1) 求 $y$ 与 $x$ 的函数关系式；

(2) 购进甲款、乙款各多少件时总利润最大？最大利润是多少？

建模定域递减取左端点

提示

第(1)问：总利润 $y$ = 甲款利润 + 乙款利润。甲款 $x$ 件，乙款 $50 - x$ 件。

第(2)问：先把"乙款不超过甲款的2倍"翻译成关于 $x$ 的不等式，再与隐含条件 $0 \leq x \leq 50$ 取交集。

练习二

第 2 题：水果批发采购问题

水果批发商采购苹果和橙子两种水果，苹果每箱获利 50 元，橙子每箱获利 90 元。计划共采购 80 箱，其中苹果箱数不少于橙子箱数。

设采购苹果 $x$ 箱，总利润为 $y$ 元。

(1) 写出 $y$ 关于 $x$ 的函数关系式；

(2) 如何采购才能使总利润最大？最大利润是多少？

(3) 若苹果最多采购 60 箱，总利润能否达到 6200 元？若能，求苹果的箱数；若不能，求总利润的取值范围。

建模定域递增取右端点验证范围

提示

第(1)问：$y = 50x + 90(80 - x)$，化简后注意 $k$ 的正负。

第(2)问："苹果不少于橙子"即 $x \geq 80 - x$，解得 $x \geq 40$。范围是 $40 \leq x \leq 80$。

第(3)问：附加 $x \leq 60$ 后范围变为 $40 \leq x \leq 60$。先解方程再验证范围。

练习三

第 3 题：文具店进货方案问题

文具店销售A型和B型两种书包。A型每个获利 60 元，B型每个获利 100 元。计划一次购进共 40 个，B型不超过A型的 3 倍，且A型至少购进 8 个。

设购进A型 $x$ 个，总利润为 $y$ 元。

(1) 求 $y$ 与 $x$ 的函数关系式；

(2) 购进A型、B型各多少个时利润最小？最小利润是多少？

(3) 若要求总利润不低于 2800 元，求A型书包至少购进多少个。

多个约束递减求最小值反向问题

提示

第(1)问：$y = 60x + 100(40 - x)$，化简得 $y = -40x + 4000$。

第(2)问：约束有三个——$x \geq 0$，$40 - x \geq 0$ 即 $x \leq 40$；"B不超过A的3倍"即 $40 - x \leq 3x$ 得 $x \geq 10$；"A至少8个"即 $x \geq 8$。综合得 $10 \leq x \leq 40$。注意本题求最小值。

第(3)问："利润不低于 2800 元"即 $y \geq 2800$，利用递减性转化为 $x$ 的不等式。

答案与解析

第 1 题参考答案

第(1)问

设购进甲款 $x$ 件，则乙款 $50 - x$ 件。

$$y = 80x + 120(50 - x) = 80x + 6000 - 120x = -40x + 6000$$

即 $y = -40x + 6000$。

第(2)问

隐含条件：$0 \leq x \leq 50$。

乙款不超过甲款的2倍：$50 - x \leq 2x$，即 $3x \geq 50$，$x \geq \dfrac{50}{3} \approx 16.67$。

因为 $x$ 为正整数，所以 $x \geq 17$。范围：$17 \leq x \leq 50$。

$k = -40 < 0$，函数递减，最大值在左端点 $x = 17$ 处。

$$y_{\max} = -40 \times 17 + 6000 = -680 + 6000 = 5320 \text{（元）}$$

答：购进甲款 17 件、乙款 33 件时利润最大，最大利润为 5320 元。

第 2 题参考答案

第(1)问

设采购苹果 $x$ 箱，则橙子 $80 - x$ 箱。

$$y = 50x + 90(80 - x) = 50x + 7200 - 90x = -40x + 7200$$

即 $y = -40x + 7200$。

第(2)问

隐含条件：$0 \leq x \leq 80$。

苹果不少于橙子：$x \geq 80 - x$，即 $2x \geq 80$，$x \geq 40$。

范围：$40 \leq x \leq 80$。

$k = -40 < 0$，函数递减，最大值在左端点 $x = 40$ 处。

$$y_{\max} = -40 \times 40 + 7200 = -1600 + 7200 = 5600 \text{（元）}$$

答：采购苹果 40 箱、橙子 40 箱时利润最大，最大利润为 5600 元。

第(3)问

附加约束 $x \leq 60$，范围缩小为 $40 \leq x \leq 60$。

令 $-40x + 7200 = 6200$，解得 $40x = 1000$，$x = 25$。

$x = 25 < 40$，不在范围 $[40,\, 60]$ 内，所以总利润不能达到 6200 元。

利润范围：递减函数在 $[40,\, 60]$ 上——

$$y(40) = 5600 \text{（元）}，\quad y(60) = -40 \times 60 + 7200 = 4800 \text{（元）}$$

利润范围为 $4800 \leq y \leq 5600$（元）。

第 3 题参考答案

第(1)问

设购进A型 $x$ 个，则B型 $40 - x$ 个。

$$y = 60x + 100(40 - x) = 60x + 4000 - 100x = -40x + 4000$$

即 $y = -40x + 4000$。

第(2)问

列出所有约束：

• $x \geq 0$，$40 - x \geq 0$ 即 $x \leq 40$；

• B型不超过A型的3倍：$40 - x \leq 3x$，即 $4x \geq 40$，$x \geq 10$；

• A型至少8个：$x \geq 8$。

取交集：$10 \leq x \leq 40$。（$x \geq 10$ 已经包含 $x \geq 8$）

$k = -40 < 0$，函数递减。最小值在右端点 $x = 40$ 处。

$$y_{\min} = -40 \times 40 + 4000 = -1600 + 4000 = 2400 \text{（元）}$$

答：购进A型 40 个、B型 0 个时利润最小，最小利润为 2400 元。

第(3)问

$y \geq 2800$，即 $-40x + 4000 \geq 2800$。

$$-40x \geq -1200$$

$$x \leq 30$$

结合范围 $10 \leq x \leq 40$，得到 $10 \leq x \leq 30$。

答：A型书包至少购进 10 个时，总利润不低于 2800 元。

注意：这里是"至少购进多少个"使得利润不低于 2800 元。因为函数递减，$x$ 越小利润越大，所以在范围 $[10, 30]$ 内，A型至少 10 个（此时利润恰好 3600 元 ≥ 2800）。实际上 $x \leq 30$ 都满足，所以"A型至少 10 个"这一说法是在完整范围内的回答；若问"A型至多多少个还能保证利润 ≥ 2800"，答案则是 $x \leq 30$。

教师备注

  • 第 1 题——基础建模 + 单约束。训练"翻译不等式 + 递减取左端点"的基本功。注意 $50/3 \approx 16.67$，需取整为 17，考察整数约束意识。
  • 第 2 题——三问结构，与原题高度同构。第(3)问的 $x = 25$ 不在范围 $[40, 60]$ 内，检验学生是否验证范围。与原题第(3)问"陷阱"完全对应。
  • 第 3 题——多约束 + 求最小值 + 反向不等式。三个约束取交集时，需识别 $x \geq 10$ 吸收 $x \geq 8$；第(2)问求最小值考验逆向思维；第(3)问将 $y \geq 2800$ 转化为 $x$ 的不等式是不等式方向的处理训练。
  • 档位说明：默认 E 档（基础巩固）。若学生第 1 题顺利、第 2 题第(3)问也正确，可升级至 D 档（增加非整数端点讨论或二元一次方程组约束）。若第 1 题不等式翻译出错，建议回退复习不等式基础。
  • 计算量控制：三题系数均为整十或整百，化简和端点代入均为心算级别，避免计算干扰思路。

  • 我能从文字条件中正确列出总利润的函数关系式并化简

  • 我能求多个不等式的交集，得到 $x$ 的完整取值范围
  • 我能通过 $k$ 的正负判断函数的增减性
  • 我能根据增减性在端点处正确取最大值或最小值
  • 解方程后我会回头验证 $x$ 是否在允许范围内
