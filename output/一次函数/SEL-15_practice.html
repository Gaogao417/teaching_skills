自适应练习：三定一动平行四边形存在性问题

教师判断

当前档位：E 档。判断依据：学生已学过平行四边形判定，能理解基本图形关系，但需要加强存在性问题的系统训练——尤其是三定一动模型的配对方法和中点坐标法的熟练运用。本次练习定位为"同源平级变式"，保留核心结构，换表层情境，逐步建立配对直觉。

练习说明

本组共 3 题，全部围绕同一个核心动作：已知三个顶点，用中点坐标法找平行四边形的第四个顶点。

每道题都是独立的小情境，但解题步骤完全相同。做完后对比三道题的过程，你会发现它们用的是同一套方法。

每道题后面有 2 个提示，卡住时再看提示。请先独立尝试。

第 1 题（基础：坐标轴上的定点）

在平面直角坐标系中，$A(3,0)$、$B(0,2)$、$C(1,0)$。点 $P$ 为平面内一点，以 $A$、$B$、$C$、$P$ 为顶点的四边形是平行四边形，求所有满足条件的 $P$ 点坐标。

训练目标：会用 $C_3^2=3$ 列出三种对角线配对，并用中点坐标法逐一求解。

预期卡点：忘记 $AB$、$AC$、$BC$ 三种配对，漏掉解。

提示一

平行四边形的对角线互相平分，所以两条对角线的中点坐标相等。从 $A$、$B$、$C$ 三个点中选两个作为一条对角线的端点，有几种选法？

提示二

三种配对：（1）$AB$ 与 $CP$ 为对角线；（2）$AC$ 与 $BP$ 为对角线；（3）$BC$ 与 $AP$ 为对角线。分别列中点方程。例如配对（1）：$AB$ 中点 $\left(\dfrac{3+0}{2},\dfrac{0+2}{2}\right)=(1.5,1)$，$CP$ 中点 $\left(\dfrac{1+x}{2},\dfrac{0+y}{2}\right)$，令二者相等解 $(x,y)$。

第 2 题（进阶：定点都不在坐标轴上）

在平面直角坐标系中，$A(1,1)$、$B(3,3)$、$C(4,0)$。点 $P$ 为平面内一点，以 $A$、$B$、$C$、$P$ 为顶点的四边形是平行四边形，求所有满足条件的 $P$ 点坐标。

训练目标：在定点不在坐标轴上的情况下，依然能正确列出三种配对的中点方程。

预期卡点：计算中点时坐标相加出错，或忘记除以 2。

提示一

方法与第 1 题完全相同。先写出三种配对，再逐一列方程。注意坐标相加时不要算错。

提示二

配对（1）$AB$ 与 $CP$：$AB$ 中点 $=\left(\dfrac{1+3}{2},\dfrac{1+3}{2}\right)=(2,2)$，$CP$ 中点 $=\left(\dfrac{4+x}{2},\dfrac{0+y}{2}\right)$，令 $(2,2)=\left(\dfrac{4+x}{2},\dfrac{y}{2}\right)$，得 $x=0, y=4$。类似地完成其余两种配对。

第 3 题（迁移：结合直线方程）

在平面直角坐标系中，一次函数 $y=x-1$ 的图象与 $x$ 轴交于点 $A$，与 $y$ 轴交于点 $B$。点 $C(3,2)$。点 $P$ 为平面内一点，以 $A$、$B$、$C$、$P$ 为顶点的四边形是平行四边形，求所有满足条件的 $P$ 点坐标。

训练目标：先求出 $A$、$B$ 坐标（需要令 $y=0$ 和 $x=0$），再用中点坐标法求 $P$。训练"先找定点，再配对"的完整流程。

预期卡点：第一步就求错 $A$ 或 $B$ 的坐标，导致后续全错。

提示一

先求 $A$ 和 $B$：令 $y=0$ 得 $A$；令 $x=0$ 得 $B$。求出后再用中点坐标法对 $A$、$B$、$C$ 三点进行配对。

提示二

$A(1,0)$、$B(0,-1)$、$C(3,2)$。三种配对：（1）$AB$ 中点 $(0.5, -0.5)$ 与 $CP$ 中点 $\left(\dfrac{3+x}{2},\dfrac{2+y}{2}\right)$ 相等；（2）$AC$ 中点 $(2,1)$ 与 $BP$ 中点 $\left(\dfrac{x}{2},\dfrac{-1+y}{2}\right)$ 相等；（3）$BC$ 中点 $(1.5, 0.5)$ 与 $AP$ 中点 $\left(\dfrac{1+x}{2},\dfrac{y}{2}\right)$ 相等。

参考答案与升级/降级判断

第 1 题答案

$A(3,0)$，$B(0,2)$，$C(1,0)$。

配对一（$AB$ 与 $CP$）：$AB$ 中点 $(1.5,1) = CP$ 中点 $\left(\dfrac{1+x}{2},\dfrac{y}{2}\right)$ → $x=2, y=2$ → $P(2,2)$。

配对二（$AC$ 与 $BP$）：$AC$ 中点 $(2,0) = BP$ 中点 $\left(\dfrac{x}{2},\dfrac{2+y}{2}\right)$ → $x=4, y=-2$ → $P(4,-2)$。

配对三（$BC$ 与 $AP$）：$BC$ 中点 $(0.5,1) = AP$ 中点 $\left(\dfrac{3+x}{2},\dfrac{y}{2}\right)$ → $x=-2, y=2$ → $P(-2,2)$。

答案：$P(2,2)$、$P(4,-2)$、$P(-2,2)$。

判断：若学生三解全对且过程规范 → 升级至 F 档，下一轮可做结构隐藏变式。若只找到 1~2 个解 → 仍留在 E 档，重点练配对穷举。若配对正确但计算出错 → 补充中点坐标公式的计算练习。

第 2 题答案

$A(1,1)$，$B(3,3)$，$C(4,0)$。

配对一（$AB$ 与 $CP$）：$AB$ 中点 $(2,2) = CP$ 中点 $\left(\dfrac{4+x}{2},\dfrac{y}{2}\right)$ → $x=0, y=4$ → $P(0,4)$。

配对二（$AC$ 与 $BP$）：$AC$ 中点 $(2.5,0.5) = BP$ 中点 $\left(\dfrac{3+x}{2},\dfrac{3+y}{2}\right)$ → $x=2, y=-2$ → $P(2,-2)$。

配对三（$BC$ 与 $AP$）：$BC$ 中点 $(3.5,1.5) = AP$ 中点 $\left(\dfrac{1+x}{2},\dfrac{1+y}{2}\right)$ → $x=6, y=2$ → $P(6,2)$。

答案：$P(0,4)$、$P(2,-2)$、$P(6,2)$。

判断：若三解全对 → 确认 E→F 档，说明定点位置不影响配对能力。若计算错误但配对正确 → 说明方法掌握但基本功需加强，可回到 D 档做计算专项。

第 3 题答案

$A(1,0)$，$B(0,-1)$，$C(3,2)$。

配对一（$AB$ 与 $CP$）：$AB$ 中点 $(0.5,-0.5) = CP$ 中点 $\left(\dfrac{3+x}{2},\dfrac{2+y}{2}\right)$ → $x=-2, y=-3$ → $P(-2,-3)$。

配对二（$AC$ 与 $BP$）：$AC$ 中点 $(2,1) = BP$ 中点 $\left(\dfrac{x}{2},\dfrac{-1+y}{2}\right)$ → $x=4, y=3$ → $P(4,3)$。

配对三（$BC$ 与 $AP$）：$BC$ 中点 $(1.5,0.5) = AP$ 中点 $\left(\dfrac{1+x}{2},\dfrac{y}{2}\right)$ → $x=2, y=1$ → $P(2,1)$。

答案：$P(-2,-3)$、$P(4,3)$、$P(2,1)$。

判断：若 $A$、$B$ 坐标求错但后续方法正确 → 说明存在性问题方法已掌握，但基础求交点能力需巩固，建议补充直线与坐标轴交点的专项练习。若全部正确 → 确认 F 档，可进入结构隐藏型变式（如将平行四边形条件改为面积条件或其他约束）。

本组练习核心口诀：三定一动找 $P$ 点，先选两点连对角，中点相等列方程，三种配对不能少。
