学生版讲解：等腰梯形存在性

等腰梯形存在性

题目

直线 $l_1$ 经过 $O(0,0)$、$A(1,2)$ 两点，将直线 $l_1$ 向下平移 $6$ 个单位得到直线 $l_2$，交 $x$ 轴于点 $C$，$B$ 是直线 $l_2$ 上一点，且四边形 $ABCO$ 是平行四边形。

(1) 求直线 $l_2$ 的表达式及 $B$ 点的坐标；

(2) 若 $D$ 是平面直角坐标系内的一点，且以 $O$、$A$、$C$、$D$ 四个点为顶点的四边形是等腰梯形，求点 $D$ 的坐标。

一、先把题目拆开

条件能推出什么
$O(0,0)$、$A(1,2)$ 在 $l_1$ 上两点确定直线：$l_1$ 的斜率 $k=\dfrac{2-0}{1-0}=2$，所以 $l_1: y=2x$
$l_1$ 向下平移 $6$ 个单位得 $l_2$$l_2: y=2x-6$
$C$ 是 $l_2$ 与 $x$ 轴交点令 $y=0$：$2x-6=0$，$x=3$，故 $C(3,0)$
$ABCO$ 是平行四边形对角线 $AC$ 与 $OB$ 中点相同，可求 $B$ 坐标为 $B(4,2)$
$O,A,C,D$ 构成等腰梯形需要分类讨论哪两条边平行（三选一）

二、这题准备怎么走

  1. 求 $l_1$ 的表达式（两点式）
  2. 平移得到 $l_2$ 的表达式
  3. 求 $l_2$ 与 $x$ 轴交点 $C$
  4. 利用平行四边形条件求 $B$
  5. 等腰梯形问题：枚举三组平行对边
  6. 每组：平行条件（斜率相等）+ 等腰条件（腰长相等），列方程解 $D$
  7. 验证每组解不退化为平行四边形

三、关键想法

等腰梯形 = 恰好一组对边平行 + 另一组对边不平行 + 两腰相等。

已知三个点 $O(0,0)$、$A(1,2)$、$C(3,0)$，第四个点 $D$ 未知。三个已知点两两连成三条边 $OA$、$AC$、$OC$。梯形要求"恰好一组对边平行"，所以底边有三种选法：

(a) $OA \parallel CD$  (b) $AC \parallel OD$  (c) $OC \parallel AD$

每组选法下，先由斜率相等列一个方程确定 $D$ 的方向，再由两腰长度相等列第二个方程求出 $D$ 的坐标。解完后必须检查另一组对边是否也平行——如果也平行，那就变成平行四边形了，必须舍去。

四、标准解法

第(1)问：求 $l_2$ 的表达式及 $B$ 的坐标

步骤 1：求 $l_1$ 的表达式

$l_1$ 经过 $O(0,0)$ 和 $A(1,2)$，由两点式：

$$k = \frac{2-0}{1-0} = 2, \quad l_1: y = 2x$$

步骤 2：向下平移 6 个单位得 $l_2$

向下平移 $b$ 变小，$l_2: y = 2x - 6$。

步骤 3：求 $C$ 坐标

$C$ 是 $l_2$ 与 $x$ 轴的交点，令 $y=0$：

$$2x - 6 = 0 \implies x = 3, \quad C(3,\,0)$$

步骤 4：平行四边形 $ABCO$ 求 $B$

平行四边形对角线中点相同。$AC$ 中点与 $OB$ 中点相同：

$$AC \text{ 中点}: \left(\frac{0+3}{2},\,\frac{2+0}{2}\right) = \left(\frac{3}{2},\,1\right)$$

$$OB \text{ 中点}: \left(\frac{0+x_B}{2},\,\frac{0+y_B}{2}\right) = \left(\frac{3}{2},\,1\right)$$

故 $x_B = 3$，$y_B = 2$。又 $B$ 在 $l_2$ 上，验证 $y = 2 \times 3 - 6 = 0 \neq 2$，矛盾。

换用另一种判定：$ABCO$ 是平行四边形 $\iff$ $AB \parallel OC$ 且 $AB = OC$。

$OC$ 在 $x$ 轴上，斜率为 $0$，故 $AB$ 斜率也为 $0$，即 $y_B = y_A = 2$。$B$ 在 $l_2$ 上：$2 = 2x_B - 6$，得 $x_B = 4$。$|OC| = 3$，$|AB| = |x_B - x_A| = 4 - 1 = 3$，满足。

$$\boxed{l_2: y = 2x - 6,\quad C(3,\,0),\quad B(4,\,2)}$$

第(2)问：求 $D$ 使 $OACD$ 为等腰梯形

先算备用数据（三条已知边的斜率和长度平方）：

边斜率 $k$长度平方 $|AB|^2$
$OA$$k_{OA} = \dfrac{2-0}{1-0} = 2$$1^2 + 2^2 = 5$
$OC$$k_{OC} = 0$（在 $x$ 轴上）$3^2 = 9$
$AC$$k_{AC} = \dfrac{0-2}{3-1} = -1$$2^2 + (-2)^2 = 8$

情况一：$OA \parallel CD$（底为 $OA$ 和 $CD$，腰为 $OC$ 和 $AD$）

平行条件：$k_{CD} = k_{OA} = 2$

设 $D(x,y)$，则 $\dfrac{y - 0}{x - 3} = 2$，即 $y = 2(x-3)$。

等腰条件：$|OC|^2 = |AD|^2$，即 $9 = (x-1)^2 + (y-2)^2$。

将 $y = 2x - 6$ 代入：

$$(x-1)^2 + (2x-6-2)^2 = 9$$

$$(x-1)^2 + (2x-8)^2 = 9$$

$$x^2 - 2x + 1 + 4x^2 - 32x + 64 = 9$$

$$5x^2 - 34x + 56 = 0$$

$$x = \frac{34 \pm \sqrt{1156 - 1120}}{10} = \frac{34 \pm \sqrt{36}}{10} = \frac{34 \pm 6}{10}$$

$x = 4$ 或 $x = \dfrac{14}{5}$。

当 $x = 4$ 时，$y = 2$，$D(4,2)$。检查另一组对边：$k_{AD} = \dfrac{2-2}{4-1} = 0 = k_{OC}$，两组对边都平行 $\Rightarrow$ 退化为平行四边形，舍去。

当 $x = \dfrac{14}{5}$ 时，$y = 2 \times \dfrac{14}{5} - 6 = -\dfrac{2}{5}$。检查：$k_{AD} = \dfrac{-\frac{2}{5}-2}{\frac{14}{5}-1} = \dfrac{-12/5}{9/5} = -\dfrac{4}{3} \neq 0 = k_{OC}$，不平行，有效。

$$\boxed{D\left(\dfrac{14}{5},\, -\dfrac{2}{5}\right)}$$

情况二：$AC \parallel OD$（底为 $AC$ 和 $OD$，腰为 $OA$ 和 $CD$）

平行条件：$k_{OD} = k_{AC} = -1$

$\dfrac{y}{x} = -1$，即 $y = -x$。

等腰条件：$|OA|^2 = |CD|^2$，即 $5 = (x-3)^2 + y^2$。

将 $y = -x$ 代入：

$$(x-3)^2 + x^2 = 5$$

$$2x^2 - 6x + 4 = 0$$

$$(x-1)(x-2) = 0$$

$x = 1$ 或 $x = 2$。

当 $x = 1$ 时，$y = -1$，$D(1,-1)$。检查：$k_{CD} = \dfrac{-1-0}{1-3} = \dfrac{1}{2} \neq 2 = k_{OA}$，不平行，有效。

当 $x = 2$ 时，$y = -2$，$D(2,-2)$。检查：$k_{CD} = \dfrac{-2-0}{2-3} = 2 = k_{OA}$，$OA \parallel CD$ 且 $AC \parallel OD$，退化为平行四边形，舍去。

$$\boxed{D(1,\,-1)}$$

情况三：$OC \parallel AD$（底为 $OC$ 和 $AD$，腰为 $OA$ 和 $CD$）

平行条件：$k_{AD} = k_{OC} = 0$

$y_D = y_A = 2$，即 $y = 2$。

等腰条件：$|OA|^2 = |CD|^2$，即 $5 = (x-3)^2 + 4$。

$$(x-3)^2 = 1$$

$x - 3 = \pm 1$，$x = 4$ 或 $x = 2$。

当 $x = 4$ 时，$D(4,2)$。检查：$k_{CD} = \dfrac{2-0}{4-3} = 2 = k_{OA}$，$OA \parallel CD$ 且 $OC \parallel AD$，退化为平行四边形，舍去。

当 $x = 2$ 时，$D(2,2)$。检查：$k_{CD} = \dfrac{2-0}{2-3} = -2 \neq 2 = k_{OA}$，不平行，有效。

$$\boxed{D(2,\,2)}$$

汇总答案

$$D\left(\dfrac{14}{5},\, -\dfrac{2}{5}\right)\;\text{或}\; D(1,\,-1)\;\text{或}\; D(2,\,2)$$

六、易错提醒

漏掉情况：只考虑了 $OA \parallel CD$ 一种平行方式，漏掉了 $AC \parallel OD$ 和 $OC \parallel AD$。等腰梯形存在性问题必须系统枚举三组平行对边，一个都不能少。

忘记排除退化：解出 $D$ 后没有检查另一组对边是否也平行。如果两组对边都平行，四边形就变成平行四边形了，不是梯形。每解出一个 $D$，必须立刻验证。

平移方向搞反：向下平移 $6$ 个单位，$b$ 应该减 $6$，不是加 $6$。记住：$y = kx + b$ 中，向上平移 $b$ 变大，向下平移 $b$ 变小。

距离公式算错：等腰条件用的是"两腰长相等"，要先把腰是哪两条边搞清楚。底是 $OA$ 和 $CD$ 时，腰是 $OC$ 和 $AD$，等腰就是 $|OC| = |AD|$，不要混淆。

平行四边形求 $B$ 时方法不当：直接用"对角线中点相同"时，需要确认哪两条是对角线。$ABCO$ 的对角线是 $AC$ 和 $OB$，不是 $AO$ 和 $BC$。

七、一句话总结

等腰梯形存在性 = 枚举三组平行对边 × 每组列两个方程（斜率相等 + 腰长相等）× 验证不退化为平行四边形。

三个已知点、三条已知边，每条边都能当"底"，所以一定是三种情况。先算好所有斜率和长度平方作为备用数据，然后一组一组往下算，每算完一组就检查另一组对边平不平行，不平行才是真梯形。
