学生版讲解：直线分割三角形面积求解析式

一、这题先看懂什么

题目说了一条直线 $y=x+3$，它和 $x$ 轴、$y$ 轴各交一个点，分别叫 $A$ 和 $B$。这样 $O$（原点）、$A$、$B$ 就围成了一个直角三角形 $\triangle AOB$。

现在有另一条直线 $l$，它经过原点 $O$，和线段 $AB$ 交于点 $C$。这条 $l$ 把 $\triangle AOB$ 一刀切成两个小三角形，面积比是 $2:1$。

我们要做的，就是求出这条"刀"——直线 $l$ 的解析式。

想一想：如果 $C$ 靠近 $B$，$\triangle AOC$ 大还是 $\triangle BOC$ 大？

二、第一步该做什么

求任何和图形有关的问题，第一步都是把图上的关键点坐标算出来。不先算坐标，后面的一切都无法进行。

这里 $A$ 在 $x$ 轴上（$y=0$），$B$ 在 $y$ 轴上（$x=0$），直接代入：

  • 令 $y=0$：$0=x+3$，得 $A(-3,\ 0)$
  • 令 $x=0$：$y=0+3=3$，得 $B(0,\ 3)$
想一想：$|OA|$ 和 $|OB|$ 分别是多少？它们有什么关系？

三、关键转化怎么想

题目给了"面积比 $2:1$"，但我们要找的是"直线的斜率 $k$"。怎么从面积变到斜率？中间需要一个"桥"。

桥就是：面积比 → 高之比 → 坐标比 → 斜率。

具体来说：

  • $\triangle AOC$ 的底是 $OA$，高就是 $C$ 的纵坐标的绝对值 $|y_C|$
  • $\triangle BOC$ 的底是 $OB$，高就是 $C$ 的横坐标的绝对值 $|x_C|$
  • 因为 $|OA|=|OB|=3$（都是 $3$），所以面积比 直接等于 $|y_C|:|x_C|$

这一步是全题最关键的一步，我们把面积比变成了坐标比。

四、标准解法

第 1 步：求 $A$、$B$ 坐标（已完成）

$A(-3,0)$，$B(0,3)$。注意 $|OA|=|OB|=3$。

第 2 步：设 $C$ 的坐标

$C$ 在线段 $AB$ 上，直线 $AB$ 的方程是 $y=x+3$。设 $C(t,\ t+3)$，其中 $-3 \le t \le 0$。

第 3 步：利用面积比列式

$\triangle AOC$ 以 $OA$ 为底，高 $= |t+3|$；$\triangle BOC$ 以 $OB$ 为底，高 $= |t|$。

$$S_{\triangle AOC} = \frac{1}{2} \times 3 \times |t+3|, \quad S_{\triangle BOC} = \frac{1}{2} \times 3 \times |t|$$

第 4 步：分两种情形讨论

题目说"分为 $2:1$ 的两部分"，没说谁大谁小，所以要讨论。

情形一：$S_{\triangle AOC} : S_{\triangle BOC} = 2 : 1$

$$\frac{|t+3|}{|t|} = 2$$

因为 $-3 \le t \le 0$，所以 $t+3 \ge 0$，$|t+3| = t+3$；$t \le 0$，$|t| = -t$。

$$\frac{t+3}{-t} = 2 \implies t+3 = -2t \implies 3t = -3 \implies t = -1$$

所以 $C(-1,\ 2)$，斜率 $k = \dfrac{2-0}{-1-0} = -2$。

直线 $l$：$y = -2x$。

情形二：$S_{\triangle AOC} : S_{\triangle BOC} = 1 : 2$

$$\frac{|t+3|}{|t|} = \frac{1}{2}$$

$$\frac{t+3}{-t} = \frac{1}{2} \implies 2(t+3) = -t \implies 3t = -6 \implies t = -2$$

所以 $C(-2,\ 1)$，斜率 $k = \dfrac{1-0}{-2-0} = -\dfrac{1}{2}$。

直线 $l$：$y = -\dfrac{1}{2}x$。

第 5 步：检验

$t=-1$ 和 $t=-2$ 都在 $[-3,\ 0]$ 范围内，符合"点 $C$ 在线段 $AB$ 上"的条件。两个解都有效。

答：直线 $l$ 的解析式为 $y = -2x$ 或 $y = -\dfrac{1}{2}x$。

五、边讲边问

  1. $\triangle AOC$ 和 $\triangle BOC$ 有一个共同的顶点 $O$。$\triangle AOC$ 的"底"如果选 $OA$，"高"怎么表示？
  2. 为什么 $S_{\triangle AOC} : S_{\triangle BOC}$ 可以直接等于 $|y_C| : |x_C|$？$|OA|$ 和 $|OB|$ 起了什么作用？
  3. 如果题目改为"把面积分为 $3:1$ 的两部分"，$|y_C| : |x_C|$ 应该等于多少？
  4. 为什么需要分两种情形讨论？如果只算一种，可能丢掉什么？
  5. 如果 $C$ 不在线段 $AB$ 上，而在射线 $BA$ 的延长线上，上面的解还成立吗？为什么？

六、易错提醒

  • 漏掉分类讨论："分为 $2:1$ 的两部分"有 $S_{AOC}:S_{BOC}=2:1$ 和 $1:2$ 两种情况，必须都算。
  • 绝对值处理错误：$|t+3|$ 和 $|t|$ 在 $-3 \le t \le 0$ 范围内要分别去掉绝对值符号，不能直接 $\frac{t+3}{t}=2$（$t$ 是负数！）。
  • 忽略 $C$ 在线段上：求出 $t$ 值后要检查是否在 $[-3,\ 0]$ 范围内，否则点 $C$ 不在线段 $AB$ 上。

七、一句话总结

面积比 → 高之比 → 坐标比 → 斜率，四步转化是解题核心。别忘了分类讨论和范围检查！
