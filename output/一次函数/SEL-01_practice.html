自适应练习：识别一次函数与正比例函数

教师判断

当前档位：C档（会找关键量，但选关系不熟练）

判断依据：学生为初二刚开始学习一次函数，尚无实际答题记录，按默认较低档位处理。本组练习聚焦"化简为标准形式后比对定义"的核心动作，不引入参数或复杂变形。

升级条件：3 题全部正确，可升至 E 档做同源变式

降级条件：第 1 题出错，降至 B 档做低门槛识别题

练习说明

本组练习只练一个动作：把"非标准长相"的式子化成 $y = kx + b$ 的标准形式，然后判断是不是一次函数。

共 3 题，每题 2-3 个小问。做完后对照参考答案自批，记录错在哪里。

第 1 题

下列各式中，哪些是一次函数？哪些是正比例函数？写出化简过程。

  1. $y = \dfrac{2x - 6}{3}$
  2. $y = \dfrac{2x}{3} - 2$
  3. $y = \dfrac{2}{x} + 1$

训练目标化简分式形式，对比一次函数与反比例函数

预期卡点第(3)小题 $x$ 在分母，学生可能忽略

提示一

一次函数的标准形式是 $y = kx + b$，其中 $x$ 只能出现在分子上且指数为 1。

提示二

第(1)小题：分母 3 分别除分子中的 $2x$ 和 $-6$。第(3)小题：$x$ 在分母里，$y = \dfrac{2}{x} + 1$ 能写成 $y = kx + b$ 吗？

第 2 题

判断下列式子是不是一次函数，写出化简过程。

  1. $y = 5 - x$
  2. $y = (x + 1)^2 - x^2 - 2x$
  3. $y = 0 \cdot x + 7$

训练目标常数写在前面时的识别；展开化简；$k \neq 0$ 条件检查

预期卡点第(2)小题需要展开才能判断；第(3)小题 $k = 0$ 易被忽略

提示一

一次函数要求 $x$ 的系数 $k \neq 0$。如果 $k = 0$ 呢？

提示二

第(2)小题：$(x+1)^2 = x^2 + 2x + 1$，展开后 $y = x^2 + 2x + 1 - x^2 - 2x = 1$。这还是一次函数吗？

第 3 题

小明说："$y = 3x + 0$ 和 $y = 3x$ 是两个不同的一次函数。"他的说法对吗？请说明理由。

训练目标正比例函数是一次函数的特例；等价表达式的理解

预期卡点学生可能觉得"长得不一样"就是不同的函数

提示一

想想看，$b = 0$ 的一次函数叫什么？$3x + 0$ 和 $3x$ 在数学上相等吗？

提示二

$y = 3x + 0$ 就是 $y = 3x$，$b = 0$，这是正比例函数，也是一次函数。它们是同一个函数，不是两个。

参考答案与升级/降级判断

第 1 题

  1. $y = \dfrac{2x}{3} - \dfrac{6}{3} = \dfrac{2}{3}x - 2$。$k = \dfrac{2}{3} \neq 0$，$b = -2 \neq 0$ → 是一次函数，不是正比例函数。
  2. $y = \dfrac{2}{3}x - 2$，与第(1)小题等价 → 是一次函数，不是正比例函数。
  3. $y = \dfrac{2}{x} + 1$，$x$ 在分母，不能写成 $y = kx + b$ → 不是一次函数（是反比例函数加常数）。

判断：

  • 3 小题全对 → 维持 C 档，可尝试第 2 题
  • 第(3)小题判断为一次函数 → 降级至 B 档，需专门练习反比例函数识别

第 2 题

  1. $y = -x + 5$。$k = -1 \neq 0$，$b = 5 \neq 0$ → 是一次函数，不是正比例函数。
  2. $(x+1)^2 = x^2 + 2x + 1$，所以 $y = x^2 + 2x + 1 - x^2 - 2x = 1$。这没有 $x$ 项（$k = 0$）→ 不是一次函数（是常数函数）。
  3. $y = 0 \cdot x + 7 = 7$。$k = 0$，不满足 $k \neq 0$ → 不是一次函数（是常数函数）。

判断：

  • 3 小题全对 → 升至 D 档
  • 第(2)小题没展开就直接判断 → 维持 C 档，需强化"先化简再判断"的习惯
  • 第(3)小题判断为一次函数 → 降级至 C 档，$k \neq 0$ 条件需反复强调

第 3 题

小明的说法不对。

$y = 3x + 0$ 和 $y = 3x$ 是同一个式子，因为 $+0$ 不改变值。它们都满足 $y = kx + b$ 且 $b = 0$，所以它们是同一个正比例函数，当然也是同一个一次函数。

判断：

  • 答对并说清理由 → 升至 E 档，可做同源变式练习
  • 答对但理由不完整 → 维持 C 档，需强化正比例函数是特例的理解
  • 答错 → 降级至 B 档，需回到讲解页重新理解一次函数与正比例函数的关系
