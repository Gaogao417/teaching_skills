自适应练习：两直线交点与割补法求动点坐标

教师判断

当前档位：D档（会列式，但可能漏条件检查/绝对值处理）

判断依据：学生已掌握待定系数法和面积公式，能完成基本计算。原题的难点在于"面积条件→方程"的转化以及对"$D$在$y$轴右侧"这一约束条件的检查。设计三道题：第1题巩固基础流程，第2题引入绝对值分支判断，第3题变换固定顶点测试迁移。

练习说明

本组共 3 道题，只练一个动作："面积条件 → 坐标方程 → 求动点坐标"。

每道题都给你一个三角形面积和部分已知点，你需要用坐标面积公式列方程，解出动点的坐标。

做完后对照参考答案，看看自己有没有漏掉条件检查。

第1题 巩固

在平面直角坐标系中，直线 $y = x - 1$ 与 $y$ 轴交于点 $A$，与 $x$ 轴交于点 $B$。点 $P$ 在直线 $y = x - 1$ 上，且在第一象限内，$\triangle OAB$ 的面积为 $\dfrac{1}{2}$。

（注意：这里$\triangle OAB$的面积是固定的，你需要先确认这个面积值是否正确——如果正确，则说明直线确定；如果有问题，请修正。）

修正后的题目：在平面直角坐标系中，直线 $y = 2x - 4$ 与 $y$ 轴交于点 $A$，与 $x$ 轴交于点 $B$。点 $P$ 在直线 $AB$ 上，且 $\triangle OAP$ 的面积为 $6$（$O$ 为坐标原点）。求点 $P$ 的坐标。

训练目标：巩固"设参数坐标 → 坐标面积公式列方程 → 解方程"的基本流程。

预期卡点：设 $P(t, 2t-4)$ 后代入面积公式，绝对值的处理。

提示一

先求 $A$ 的坐标（令 $x=0$），$A$ 在 $y$ 轴上。$\triangle OAP$ 的三个顶点是 $O(0,0)$、$A$、$P(t, 2t-4)$。

提示二

$A(0,-4)$，代入面积公式：$\frac{1}{2}|0 \cdot (0 - (2t-4)) + 0 \cdot ((2t-4) - 0) + t(0 - (-4))| = 6$，化简得 $|4t| = 12$，$t = \pm 3$。

第2题 升级

已知直线 $y = -x + 3$ 与 $x$ 轴交于点 $M$，与 $y$ 轴交于点 $N$。点 $Q$ 在直线 $MN$ 上，$\triangle OMQ$ 的面积为 $4$（$O$ 为坐标原点）。

训练目标：绝对值产生两个解，需要根据几何意义判断取舍。

预期卡点：算出两个 $t$ 值后，不知道该不该保留——这里题目没有区域限制，所以两个解都可能有效。

提示一

先求 $M$（令 $y=0$）和 $N$（令 $x=0$）的坐标。$M(3,0)$，$N(0,3)$。

提示二

设 $Q(t, -t+3)$，用坐标面积公式列方程：$\frac{1}{2}|0 \cdot (0 - (-t+3)) + 3 \cdot ((-t+3) - 0) + t(0 - 0)| = 4$。注意这里用 $O(0,0)$、$M(3,0)$、$Q(t,-t+3)$ 三个顶点。

第3题 迁移

在平面直角坐标系中，一次函数 $y = -\dfrac{3}{2}x + 6$ 的图象与 $x$ 轴交于点 $E$，与 $y$ 轴交于点 $F$。点 $R$ 在直线 $EF$ 上，且在 $x$ 轴下方，$\triangle EFR$ 的面积为 $9$。求点 $R$ 的坐标。

训练目标：核心结构不变（面积求动点），但变换了固定顶点和区域约束，测试迁移能力。

预期卡点："$R$ 在 $x$ 轴下方"这个条件需要转化为 $y_R < 0$ 的约束，用来从绝对值产生的解中筛选。

提示一

先求 $E$（令 $y=0$）和 $F$（令 $x=0$）：$E(4,0)$，$F(0,6)$。

提示二

设 $R(t, -\frac{3}{2}t + 6)$，"在 $x$ 轴下方"意味着 $-\frac{3}{2}t + 6 < 0$，即 $t > 4$。用坐标面积公式列方程后，取满足 $t > 4$ 的解。

参考答案与升级/降级判断

参考答案

第1题

$A(0, -4)$，设 $P(t, 2t - 4)$。

$$S_{\triangle OAP} = \frac{1}{2}|0 \cdot (0 - (2t-4)) + 0 \cdot ((2t-4) - 0) + t(0 - (-4))| = \frac{1}{2}|4t| = 6$$

$|4t| = 12$，$t = 3$ 或 $t = -3$。

$P_1(3, 2)$，$P_2(-3, -10)$。

判断：若学生只写一个解，说明绝对值处理有遗漏 → 降级到C档，强化绝对值意识。若两个解都正确 → 正常，进入第2题。

第2题

$M(3, 0)$，$N(0, 3)$。设 $Q(t, -t + 3)$。

$$S_{\triangle OMQ} = \frac{1}{2}|0 \cdot (0 - (-t+3)) + 3 \cdot ((-t+3) - 0) + t(0 - 0)| = \frac{1}{2}|3(-t+3)| = 4$$

$\frac{1}{2}|-3t + 9| = 4$，$|-3t + 9| = 8$，$-3t + 9 = 8$ 或 $-3t + 9 = -8$。

$t = \frac{1}{3}$ 或 $t = \frac{17}{3}$。

$Q_1\!\left(\frac{1}{3}, \frac{8}{3}\right)$，$Q_2\!\left(\frac{17}{3}, -\frac{8}{3}\right)$。

判断：本题无区域约束，两个解都有效。若学生能正确写出两个解 → 升级，说明绝对值分支已掌握。若计算出错 → 降级到D档，检查代数运算。

第3题

$E(4, 0)$，$F(0, 6)$。设 $R\!\left(t, -\frac{3}{2}t + 6\right)$，$t > 4$。

$$S_{\triangle EFR} = \frac{1}{2}\left|4\!\left(6 - \left(-\frac{3}{2}t+6\right)\right) + 0\!\left(\left(-\frac{3}{2}t+6\right) - 0\right) + t(0 - 6)\right| = 9$$

$$\frac{1}{2}\left|4 \cdot \frac{3}{2}t - 6t\right| = \frac{1}{2}|6t - 6t| = \frac{1}{2} \cdot 0 = 0$$

面积为 $0$——说明 $E$、$F$、$R$ 共线，不能构成三角形！原题设计有误。

教学说明：这道题暴露了一个重要问题——当动点在两定点所在的直线上时，三角形退化为线段，面积恒为 $0$。这说明出题时需要动点不在两定点连线上。

修正版第3题：一次函数 $y = -\dfrac{3}{2}x + 6$ 的图象与 $x$ 轴交于点 $E$，与 $y$ 轴交于点 $F$。点 $R$ 在直线 $y = 2x - 6$ 上，且 $\triangle EFR$ 的面积为 $20$。求点 $R$ 的坐标。

修正版第3题 答案

$E(4, 0)$，$F(0, 6)$。设 $R(t, 2t - 6)$。

$$S = \frac{1}{2}\left|4\!\left(6 - (2t-6)\right) + 0\!\left((2t-6) - 0\right) + t(0 - 6)\right| = 20$$

$$\frac{1}{2}\left|4(12 - 2t) - 6t\right| = 20$$

$$\frac{1}{2}|48 - 8t - 6t| = 20$$

$$\frac{1}{2}|48 - 14t| = 20$$

$|48 - 14t| = 40$

$48 - 14t = 40$ → $14t = 8$ → $t = \dfrac{4}{7}$

$48 - 14t = -40$ → $14t = 88$ → $t = \dfrac{44}{7}$

$R_1\!\left(\dfrac{4}{7}, -\dfrac{34}{7}\right)$，$R_2\!\left(\dfrac{44}{7}, \dfrac{46}{7}\right)$。

判断：若学生能完成本题（包括识别出修正版题目与原结构的差异）→ 升级到E档，已掌握迁移能力。若学生在绝对值或计算上出错 → 停留在D档。
