一次函数在几何中的运用（动点面积）

练习说明

做题方法

本练习共 3 题，围绕"两直线交点与动点面积"这一核心题型展开。每题的结构与讲解例题相似，但在数字、斜率和面积条件上做了变化，帮助你巩固以下能力：

(1) 用待定系数法由公共点确定两条直线的解析式；

(2) 设动点坐标为一个变量，用坐标三角形面积公式（行列式法）将面积条件转化为方程；

(3) 解方程并检验约束条件。

自我评估

每题后附有提示和答案步骤。建议先独立完成，遇到困难再看提示。全部做完后，对照答案给自己打分，完成自检。

练习 1

题目

在平面直角坐标系 $xOy$ 中，一次函数 $y = -x + b$ 与 $y = \dfrac{1}{2}x + c$ 的图象都经过点 $A(2,\,0)$，且分别与 $y$ 轴交于点 $B$ 和点 $C$。

(1) 求 $B$、$C$ 两点的坐标；

(2) 设点 $P$ 在直线 $y = \dfrac{1}{2}x + c$ 上，且在 $y$ 轴右侧，当 $\triangle ABP$ 的面积为 $9$ 时，求点 $P$ 的坐标。

第(1)问
第(2)问

提示

第(1)问：把 $A(2,\,0)$ 代入两条直线方程，分别求 $b$ 和 $c$，再令 $x=0$ 得 $B$、$C$。

第(2)问：设 $P\bigl(t,\,\tfrac{1}{2}t + c\bigr)$（$t>0$），用行列式面积公式 $S_{\triangle ABP} = \dfrac{1}{2}\bigl|x_A(y_B - y_P) + x_B(y_P - y_A) + x_P(y_A - y_B)\bigr| = 9$ 列方程。

答案步骤

第(1)问

将 $A(2,\,0)$ 代入 $y=-x+b$：$0 = -2 + b$，得 $b = 2$，所以 $y = -x + 2$。令 $x=0$，得 $y=2$，即 $B(0,\,2)$。

将 $A(2,\,0)$ 代入 $y=\dfrac{1}{2}x+c$：$0 = 1 + c$，得 $c = -1$，所以 $y = \dfrac{1}{2}x - 1$。令 $x=0$，得 $y=-1$，即 $C(0,\,-1)$。

第(2)问

设 $P\bigl(t,\,\tfrac{1}{2}t - 1\bigr)$，$t > 0$。代入行列式面积公式：

$$\frac{1}{2}\left|2\!\left(2 - \left(\frac{t}{2}-1\right)\right) + 0 \cdot \left(\left(\frac{t}{2}-1\right)-0\right) + t\,(0-2)\right| = 9$$

化简：$\dfrac{1}{2}\left|2\!\left(3 - \dfrac{t}{2}\right) - 2t\right| = 9$，即 $\dfrac{1}{2}\left|6 - t - 2t\right| = 9$，$\dfrac{1}{2}\left|6 - 3t\right| = 9$，$\left|6 - 3t\right| = 18$。

情况一：$6 - 3t = 18 \Rightarrow -3t = 12 \Rightarrow t = -4$（不合 $t>0$，舍去）。

情况二：$6 - 3t = -18 \Rightarrow -3t = -24 \Rightarrow t = 8 > 0$（符合）。

$y_P = \dfrac{1}{2} \times 8 - 1 = 3$，所以 $P(8,\,3)$。

难度档位：E 档（基础巩固）。

设计意图：斜率从原题的 $2$ 和 $-\tfrac{1}{2}$ 改为 $-1$ 和 $\tfrac{1}{2}$，公共点从 $A(-2,\,0)$ 改为 $A(2,\,0)$，面积从 $15$ 改为 $9$。结构完全一致，帮助学生巩固基本流程。动点名从 $D$ 改为 $P$，避免机械抄写。

预期 blocker：代入时符号出错（$x=2$ 为正，不易出错）；绝对值方程解完后忘记检验 $t>0$。

练习 2

题目

在平面直角坐标系 $xOy$ 中，一次函数 $y = 3x + p$ 与 $y = -x + q$ 的图象都经过点 $A(-1,\,0)$，且分别与 $y$ 轴交于点 $B$ 和点 $C$。

(1) 求 $B$、$C$ 两点的坐标；

(2) 设点 $M$ 在直线 $y = -x + q$ 上，且在 $y$ 轴右侧，当 $\triangle ABM$ 的面积为 $12$ 时，求点 $M$ 的坐标。

第(1)问
第(2)问

提示

第(1)问：注意 $A(-1,\,0)$ 的横坐标为 $-1$，代入 $y=3x+p$ 时：$0 = 3\times(-1)+p$，计算要仔细。

第(2)问：设 $M(t,\,-t+q)$（$t>0$），先用第(1)问求出的 $q$ 值。代入行列式面积公式，注意 $x_A = -1$ 的符号。

答案步骤

第(1)问

将 $A(-1,\,0)$ 代入 $y=3x+p$：$0 = 3\times(-1)+p$，得 $p = 3$，所以 $y = 3x + 3$。令 $x=0$，得 $y=3$，即 $B(0,\,3)$。

将 $A(-1,\,0)$ 代入 $y=-x+q$：$0 = -(-1)+q$，得 $0 = 1+q$，$q = -1$，所以 $y = -x - 1$。令 $x=0$，得 $y=-1$，即 $C(0,\,-1)$。

第(2)问

设 $M(t,\,-t-1)$，$t > 0$。代入行列式面积公式：

$$\frac{1}{2}\left|(-1)\!\left(3 - (-t-1)\right) + 0\cdot((-t-1)-0) + t\,(0-3)\right| = 12$$

化简：$\dfrac{1}{2}\left|(-1)(4+t) - 3t\right| = 12$，$\dfrac{1}{2}\left|-4 - t - 3t\right| = 12$，$\dfrac{1}{2}\left|-4 - 4t\right| = 12$，$\left|4 + 4t\right| = 24$。

因为 $t > 0$，所以 $4 + 4t > 0$，直接去掉绝对值：$4 + 4t = 24 \Rightarrow 4t = 20 \Rightarrow t = 5 > 0$（符合）。

$y_M = -5 - 1 = -6$，所以 $M(5,\,-6)$。

难度档位：E 档（基础巩固）。

设计意图：斜率变为 $3$ 和 $-1$（一条斜率更大，一条仍为 $-1$），公共点 $A(-1,\,0)$ 距离 $y$ 轴更近。面积 $12$。本题的一个简化点在于 $4+4t$ 在 $t>0$ 时恒正，绝对值可以直接去掉，不需要分情况讨论，降低计算负担。动点名改为 $M$。

预期 blocker：代入 $y=-x+q$ 时 $x=-1$ 变双负，容易算错为 $0 = -(-1)+q \Rightarrow 0 = -1+q$（漏了负号）。

练习 3

题目

在平面直角坐标系 $xOy$ 中，一次函数 $y = \dfrac{1}{3}x + u$ 与 $y = -2x + v$ 的图象都经过点 $A(-3,\,0)$，且分别与 $y$ 轴交于点 $B$ 和点 $C$。

(1) 求 $B$、$C$ 两点的坐标；

(2) 设点 $N$ 在直线 $y = -2x + v$ 上，且在第一象限，当 $\triangle ABN$ 的面积为 $10$ 时，求点 $N$ 的坐标。

第(1)问
第(2)问

提示

第(1)问：$A(-3,\,0)$ 代入 $y=\dfrac{1}{3}x+u$：$0 = \dfrac{1}{3}\times(-3)+u = -1+u$，得 $u=1$，$B(0,\,1)$。$A(-3,\,0)$ 代入 $y=-2x+v$：$0 = 6+v$，得 $v=-6$，$C(0,\,-6)$。

第(2)问："第一象限"意味着 $x_N > 0$ 且 $y_N > 0$，即 $t > 0$ 且 $-2t-6 > 0$，所以 $t < -3$。两个条件矛盾——这说明需要重新理解题意：$N$ 在直线 $y=-2x-6$ 上且在第一象限，需要 $x_N > 0$、$y_N > 0$ 同时满足。由于 $y=-2x-6$ 在 $x>0$ 时 $y<-6$，不可能在第一象限。

因此正确的区域约束是 $N$ 在 $y$ 轴右侧，即 $t>0$。请按此约束完成计算。

答案步骤

第(1)问

将 $A(-3,\,0)$ 代入 $y=\dfrac{1}{3}x+u$：$0 = \dfrac{1}{3}\times(-3)+u = -1+u$，得 $u = 1$，所以 $y = \dfrac{1}{3}x + 1$。令 $x=0$，得 $y=1$，即 $B(0,\,1)$。

将 $A(-3,\,0)$ 代入 $y=-2x+v$：$0 = -2\times(-3)+v = 6+v$，得 $v = -6$，所以 $y = -2x - 6$。令 $x=0$，得 $y=-6$，即 $C(0,\,-6)$。

第(2)问

注意：直线 $y=-2x-6$ 在 $x>0$ 时 $y < -6$，不可能进入第一象限。题目中"第一象限"与"$N$ 在直线 $y=-2x+v$ 上"在 $v=-6$ 时矛盾。因此区域约束应理解为 $N$ 在 $y$ 轴右侧（$t > 0$）。

设 $N(t,\,-2t-6)$，$t > 0$。代入行列式面积公式：

$$\frac{1}{2}\left|(-3)\!\left(1-(-2t-6)\right) + 0\cdot((-2t-6)-0) + t\,(0-1)\right| = 10$$

化简：$\dfrac{1}{2}\left|(-3)(7+2t) - t\right| = 10$，$\dfrac{1}{2}\left|-21 - 6t - t\right| = 10$，$\dfrac{1}{2}\left|-21 - 7t\right| = 10$，$\left|21 + 7t\right| = 20$。

因为 $t > 0$，所以 $21 + 7t > 21 > 0$，直接去掉绝对值：$21 + 7t = 20 \Rightarrow 7t = -1 \Rightarrow t = -\dfrac{1}{7}$。

$t = -\dfrac{1}{7} < 0$，不满足 $t > 0$，所以本题在 $y$ 轴右侧无解。

回看绝对值方程的另一种情况：$21 + 7t = -20 \Rightarrow 7t = -41 \Rightarrow t = -\dfrac{41}{7}$，也不满足 $t>0$。

因此，当区域约束为"第一象限"时，$\triangle ABN$ 的面积不可能等于 $10$。如果将区域约束放宽为"$y$ 轴左侧"（$t < 0$），则 $t = -\dfrac{41}{7}$ 满足条件，此时 $y_N = -2\times\!\left(-\dfrac{41}{7}\right)-6 = \dfrac{82}{7}-6 = \dfrac{40}{7}$，$N\!\left(-\dfrac{41}{7},\,\dfrac{40}{7}\right)$。

难度档位：E 档（含探究成分，可作为课堂讨论题）。

设计意图：本题故意将区域约束设为"第一象限"，但直线 $y=-2x-6$ 与第一象限无交点，引导学生发现约束矛盾，培养审题和几何直观。这是一个重要的元认知训练：先画草图判断动点可能的位置范围，再列方程。如果学生能发现矛盾，说明空间想象力到位；如果直接算出无解，也能锻炼代数检验能力。

教学建议：让学生先画草图，判断 $y=-2x-6$ 的图象是否经过第一象限。如果学生在解题前就发现矛盾，应给予表扬。如果学生直接列方程计算得到 $t<0$ 后才发现，也是一个好的学习过程。

预期 blocker：学生可能忽略"第一象限"的双重约束（$x>0$ 且 $y>0$），只考虑 $x>0$ 就开始计算，最终得到无解但不理解原因。
