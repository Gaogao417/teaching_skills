自适应练习：双动点运动中特殊四边形存在性

教师判断

当前档位：E 档。判断依据：学生为初二尖子生，已能理解等腰三角形分类讨论和平行四边形判定，但双动点运动中四边形存在性是中考压轴新题型。核心训练目标是将"速度→坐标"的参数化表达与四边形条件结合，建立系统化的分析框架。本次练习定位为"同源平级变式"——保留速度参数化 + 四边形存在性的核心结构，换情境、换四边形类型，逐步建立"动点坐标化 → 向量/中点列方程 → 分类检验"的完整思维链。

练习说明

本组共 3 题，全部围绕同一个核心动作：用速度写出动点的参数坐标，再用坐标法判断四边形类型。

每道题都是独立的情境，但解题流程相同：(1) 参数化坐标；(2) 列四边形条件方程；(3) 解方程并检验。做完后对比三道题的过程，总结通法。

每道题后面有 2 个提示，卡住时再看提示。请先独立尝试。

第 1 题（基础：单动点 + 平行四边形存在性）

在平面直角坐标系中，$O$ 为原点，$A(0,3)$，$B(4,0)$。动点 $P$ 从 $O$ 出发，沿 $x$ 轴正方向以每秒 2 个单位速度运动。设运动时间为 $t$ 秒（$t > 0$）。当 $t$ 为何值时，以 $O$、$A$、$B$、$P$ 为顶点的四边形是平行四边形？

训练目标：会用速度写出 $P(t)$ 的坐标，并用向量法检验不同顶点排列下的平行四边形条件。

预期卡点：只尝试一种顶点排列就下结论，遗漏其他可能的排列。

提示一

先写出 $P$ 的坐标：$P$ 在 $x$ 轴上，速度为 2，所以 $P(2t, 0)$。然后思考：四个点 $O(0,0)$、$A(0,3)$、$B(4,0)$、$P(2t,0)$ 排列成四边形，有哪些排列方式能使两组对边分别平行且相等？

提示二

注意 $A$ 在 $y$ 轴上，$O$、$B$、$P$ 都在 $x$ 轴上。所以 $OA$ 是竖直的，$OB$ 和 $OP$ 是水平的。尝试排列 $OABP$：$OA = (0,3)$，$BP = (2t-4, 0)$。$OA$ 竖直、$BP$ 水平，不平行。尝试 $OBPA$：$OB = (4,0)$，$PA = (-2t, 3)$，不平行。尝试 $OAPB$：$OA = (0,3)$，$PB = (4-2t, 0)$。需要第三边 $AP = (2t, -3)$，$OB = (4,0)$。检查 $OA \parallel PB$ 吗？$(0,3)$ 与 $(4-2t, 0)$ 不平行。再尝试 $OPAB$：$OP = (2t,0)$，$AB = (4,-3)$，不平行。关键：由于 $O$、$P$、$B$ 都在 $x$ 轴上，任何包含 $OP$ 和 $AB$ 作为对边的排列都需要 $OP \parallel AB$，即 $y$ 分量都为零，但 $AB$ 的 $y$ 分量为 $-3 \neq 0$。唯一的可能是让 $OA$（竖直）与某条也竖直的边平行。只有 $P$ 到 $B$ 的水平距离等于 $O$ 到 $A$ 时...重新用中点坐标法：以 $OA$ 和 $BP$ 为对角线。

第 2 题（进阶：双动点 + 矩形存在性）

在平面直角坐标系中，$O$ 为原点，$A(0,2)$。动点 $P$ 从 $O$ 出发，沿 $x$ 轴正方向以每秒 1 个单位速度运动；动点 $Q$ 从 $A$ 出发，沿 $y$ 轴负方向以每秒 1 个单位速度运动。$P$、$Q$ 同时出发，运动时间为 $t$ 秒。当 $t$ 为何值时，$\triangle OPQ$ 的面积等于 $1$？能否使四边形 $OPQA$ 成为矩形？

训练目标：写出双动点参数坐标 $P(t)$ 和 $Q(t)$，结合面积公式列方程；进一步判断矩形存在性。

预期卡点：面积公式中三角形的底和高选错；矩形的判定条件（邻边垂直 + 平行四边形）漏掉其中之一。

提示一

先写坐标：$P(t, 0)$，$Q(0, 2-t)$。$\triangle OPQ$ 以 $OP$ 为底、$Q$ 的纵坐标为高（$Q$ 在 $y$ 轴上），面积 $= \dfrac{1}{2} \times |OP| \times |y_Q| = \dfrac{1}{2} \times t \times |2-t|$。令其等于 1，解 $t$。注意讨论 $t < 2$ 和 $t > 2$ 时绝对值的去法。

提示二

矩形 = 平行四边形 + 一个角为直角。四边形 $OPQA$ 中 $O(0,0)$、$P(t,0)$、$Q(0,2-t)$、$A(0,2)$。注意 $O$、$A$ 都在 $y$ 轴上，$O$、$P$ 都在 $x$ 轴上。所以 $\angle POA = 90°$。如果 $OPQA$ 是平行四边形，则它自动有直角，就是矩形。用向量法检验平行四边形条件。

第 3 题（迁移：双动点 + 菱形否定论证）

在平面直角坐标系中，$O$ 为原点，$C(0,3)$，$D(4,3)$。动点 $P$ 从 $O$ 出发，沿 $x$ 轴正方向以每秒 2 个单位速度运动；动点 $Q$ 从 $C$ 出发，沿射线 $CD$（向右）以每秒 1 个单位速度运动。$P$、$Q$ 同时出发，运动时间为 $t$ 秒。

(1) $t$ 为何值时，以 $O$、$P$、$Q$、$D$ 为顶点的四边形是平行四边形？
(2) 能否使该四边形成为菱形？请说明理由。

训练目标：在不同定点配置下完成参数化 + 平行四边形检验的完整流程；掌握"否定存在性"的论证方法。

预期卡点：否定菱形存在性时论证不完整，只试了一种排列就下结论。

提示一

先写坐标：$P(2t, 0)$，$Q(t, 3)$（$Q$ 在 $CD$ 上，$CD$ 是 $y=3$ 的水平线段）。然后对 $O(0,0)$、$P(2t,0)$、$Q(t,3)$、$D(4,3)$ 四个点，用向量法尝试不同排列。注意 $O$ 和 $P$ 在 $x$ 轴上，$Q$ 和 $D$ 在 $y=3$ 上——这和原题的结构很像。

提示二

平行四边形：尝试排列 $OPQD$，其中 $OP = (2t,0)$，$DQ = (4-t, 0)$。$OP \parallel DQ$（都在水平方向）。$OQ = (t,3)$，$PD = (4-2t, 3)$。$OQ \parallel PD$（$y$ 分量相同）。令 $OP = DQ$：$2t = 4-t \Rightarrow 3t = 4 \Rightarrow t = 4/3$。再验证 $OQ = PD$：$t = 4/3$ 时 $OQ = (4/3, 3)$，$PD = (4 - 8/3, 3) = (4/3, 3)$ ✓。菱形需要四边相等。平行四边形 OPQD 中 $OP = 8/3$，$OQ = 5/3$，不相等，所以不是菱形。还需要检查其他排列能否成为菱形。

参考答案与升级/降级判断

第 1 题答案

$O(0,0)$，$A(0,3)$，$B(4,0)$，$P(2t,0)$。

方法：中点坐标法。
以 $OA$ 和 $BP$ 为对角线：$OA$ 中点 $(0, 1.5)$，$BP$ 中点 $\left(\dfrac{4+2t}{2},\ 0\right)$。
中点相等：$0 = \dfrac{4+2t}{2} \Rightarrow 4+2t=0 \Rightarrow t=-2$（舍去）。

以 $OB$ 和 $AP$ 为对角线：$OB$ 中点 $(2, 0)$，$AP$ 中点 $\left(t,\ 1.5\right)$。
中点相等：$2 = t$ 且 $0 = 1.5$，矛盾，无解。

以 $AB$ 和 $OP$ 为对角线：$AB$ 中点 $(2, 1.5)$，$OP$ 中点 $(t, 0)$。
中点相等：$t = 2$ 且 $0 = 1.5$，矛盾，无解。

以 $OA$ 和 $BP$ 为对角线但换配对方式：以 $O$ 和 $P$ 为对角线两端、$A$ 和 $B$ 为另一对角线两端：$OP$ 中点 $(t,0)$，$AB$ 中点 $(2,1.5)$。需要 $t=2$ 且 $0=1.5$，矛盾。

结论：本题不存在 $t > 0$ 使四边形 OABP 为平行四边形。原因是 $A$ 在 $y$ 轴上而 $O, B, P$ 都在 $x$ 轴上，$OA$ 竖直但找不到另一条竖直且等长的对边来配对。

判断：若学生正确得出"不存在"并论证完整 → 升级至 F 档，说明否定论证能力已初步建立。若只试了一种排列就下结论 → 仍留在 E 档，需加强"穷举排列"的意识。若中点坐标法使用正确但计算出错 → 回到 D 档补充中点公式计算练习。

第 2 题答案

$P(t, 0)$，$Q(0, 2-t)$。

面积 $= 1$：
$S_{\triangle OPQ} = \dfrac{1}{2} \times t \times |2-t| = 1$。
当 $t < 2$：$\dfrac{1}{2} t(2-t) = 1 \Rightarrow t(2-t) = 2 \Rightarrow -t^2 + 2t - 2 = 0 \Rightarrow t^2 - 2t + 2 = 0$，$\Delta = 4-8 < 0$，无解。
当 $t \geq 2$：$\dfrac{1}{2} t(t-2) = 1 \Rightarrow t(t-2) = 2 \Rightarrow t^2 - 2t - 2 = 0 \Rightarrow t = \dfrac{2 \pm \sqrt{12}}{2} = 1 \pm \sqrt{3}$。
取 $t = 1 + \sqrt{3} \approx 2.73 > 2$ ✓。$t = 1 - \sqrt{3} < 0$，舍去。
$t = 1 + \sqrt{3}$ 时面积等于 1。
矩形判断：
四边形 $OPQA$：$O(0,0)$，$P(t,0)$，$Q(0,2-t)$，$A(0,2)$。
$OP = (t,0)$，$AQ = (0, -t)$（$A$ 到 $Q$）。$OP \perp AQ$ 且 $|OP| = |AQ| = t$。
$OA = (0,2)$，$PQ = (-t, 2-t)$。$OA \parallel PQ$ 需 $\dfrac{0}{-t} = \dfrac{2}{2-t}$，即 $0 = 2$，矛盾。
所以 OPQA 不能成为矩形。

判断：若面积方程解对且矩形否定论证完整 → 确认 E 档偏上，下一轮可做原题第(3)问的完整训练。若面积方程的绝对值讨论出错 → 回到 D 档，补充含绝对值的方程专项。若矩形判断方法混乱 → 仍留在 E 档，重点练"矩形 = 平行四边形 + 直角"的判定流程。

第 3 题答案

$O(0,0)$，$C(0,3)$，$D(4,3)$，$P(2t,0)$，$Q(t,3)$。

(1) 平行四边形：
排列 OPQD：$OP=(2t,0)$，$QD=(4-t,0)$，$OP \parallel QD$。
$OQ=(t,3)$，$PD=(4-2t,3)$，$OQ \parallel PD$。
$OP = QD \Rightarrow 2t = 4-t \Rightarrow t = \dfrac{4}{3}$。
验证 $OQ = PD$：$OQ = \left(\dfrac{4}{3}, 3\right)$，$PD = \left(4-\dfrac{8}{3}, 3\right) = \left(\dfrac{4}{3}, 3\right)$ ✓。
$t = \dfrac{4}{3}$ 秒时，四边形 OPQD 为平行四边形。
(2) 菱形判断：
菱形需要四边相等。OPQD 中 $OP = 2 \times \dfrac{4}{3} = \dfrac{8}{3}$，$OQ = \sqrt{\left(\dfrac{4}{3}\right)^2 + 9} = \sqrt{\dfrac{16}{9} + 9} = \sqrt{\dfrac{97}{9}} = \dfrac{\sqrt{97}}{3}$。
$\dfrac{8}{3} \neq \dfrac{\sqrt{97}}{3}$，所以 OPQD 不是菱形。
检查其他排列能否成为菱形：由于 $O,P$ 在 $x$ 轴上、$Q,D$ 在 $y=3$ 上，任何排列的对边中至少有一组方向为水平。要让四边相等，需要竖直方向的边长等于水平方向的边长，这要求 $|OQ_y| = |OP_x|$ 或类似条件。穷举所有排列后均不满足。
结论：不能成为菱形。

判断：若平行四边形和菱形两部分都做对 → 升级至 F 档，可进入结构隐藏型变式（如将坐标条件改为面积或比例条件）。若平行四边形做对但菱形否定论证不完整 → 仍留在 E 档，重点训练否定存在性的穷举方法。若整体困难 → 降级至 D 档，回到单动点 + 特殊三角形问题夯实基础。

本组练习核心口诀：动点先标坐标（速度 × 时间），四边形条件列方程（向量或中点），分类讨论要穷举，否定结论也要证。
