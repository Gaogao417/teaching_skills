待定系数法求解析式

练习说明

下面有三道练习题，考查用待定系数法求一次函数 $y = kx + b$（$k \neq 0$）的解析式。

每题给出一次函数图像上两个点的坐标，请你：

  1. 设解析式为 $y = kx + b$（$k \neq 0$）；
  2. 将两个点的坐标分别代入，得到关于 $k$、$b$ 的方程组；
  3. 解方程组，求出 $k$ 和 $b$ 的值；
  4. 写出完整的解析式，并检验。

遇到困难时，请参考每题下方的提示。完成后可对照参考答案进行自评。

练习题

第 1 题

已知一次函数 $y = kx + b$（$k \neq 0$）的图像经过点 $A(1,\, 5)$ 和点 $B(-2,\, -4)$，求这个一次函数的解析式。

提示：将 $A(1,\, 5)$ 代入 $y = kx + b$ 得到 $k + b = 5$；再将 $B(-2,\, -4)$ 代入得到 $-2k + b = -4$。两式相减即可消去 $b$。

教师批注：

  • 设 $y = kx + b$，代入 $A(1,5)$：$k + b = 5$ 〖①〗；代入 $B(-2,-4)$：$-2k + b = -4$ 〖②〗。
  • ② - ① 得 $-3k = -9$，即 $k = 3$。
  • 代入 ① 得 $b = 2$。
  • 解析式为 $y = 3x + 2$。检验：$A(1,5)$ 代入得 $3 \times 1 + 2 = 5$ ✔；$B(-2,-4)$ 代入得 $3 \times (-2) + 2 = -4$ ✔。
  • 易错点：代入 $(-2)$ 时漏写负号；消元时两式相减的方向搞反导致 $k$ 的符号错误。

第 2 题

已知一次函数的图像经过点 $(4,\, 0)$，且当 $x = 1$ 时，$y = 9$。求这个一次函数的解析式。

提示：注意 "$x = 1$ 时 $y = 9$" 表示图像经过点 $(1,\, 9)$。因此图像经过 $(4,\, 0)$ 和 $(1,\, 9)$ 两个点，按待定系数法步骤操作即可。

教师批注：

  • 图像经过 $(4, 0)$ 和 $(1, 9)$。
  • 设 $y = kx + b$，代入 $(4,0)$：$4k + b = 0$ 〖①〗；代入 $(1,9)$：$k + b = 9$ 〖②〗。
  • ① - ② 得 $3k = -9$，即 $k = -3$。
  • 代入 ② 得 $b = 12$。
  • 解析式为 $y = -3x + 12$。检验：$x=4$ 时 $y = -3 \times 4 + 12 = 0$ ✔；$x=1$ 时 $y = -3 \times 1 + 12 = 9$ ✔。
  • 本题条件表述方式不同于第 1 题，考验学生将文字条件转化为点坐标的能力。

第 3 题

一个一次函数的图像与 $x$ 轴交于点 $(2,\, 0)$，与 $y$ 轴交于点 $(0,\, -6)$。求这个一次函数的解析式。

提示：图像与 $y$ 轴的交点为 $(0,\, -6)$，即当 $x = 0$ 时 $y = -6$，直接代入 $y = kx + b$ 可得 $b = -6$。这样可以跳过解方程组，直接得到 $b$ 的值，再用另一个点求出 $k$。

教师批注：

  • 设 $y = kx + b$，代入 $(0, -6)$：$b = -6$。
  • 代入 $(2, 0)$：$2k - 6 = 0$，解得 $k = 3$。
  • 解析式为 $y = 3x - 6$。检验：$(2,0)$ 代入得 $3 \times 2 - 6 = 0$ ✔；$(0,-6)$ 代入得 $-6 = -6$ ✔。
  • 本题体现"与 $y$ 轴交点的纵坐标即为 $b$ 值"这一重要性质，是待定系数法的简化运用。
  • 进阶：学生会发现 $y$ 轴截距直接就是 $b$，这是后续学习"截距式"的基础。

参考答案

第 1 题

设 $y = kx + b$（$k \neq 0$），分别代入 $A(1,\,5)$ 和 $B(-2,\,-4)$：

$$\begin{cases} k + b = 5 \quad \text{①} \\ -2k + b = -4 \quad \text{②} \end{cases}$$

② - ① 得：$-3k = -9$，解得 $k = 3$。

将 $k = 3$ 代入 ① 得：$b = 2$。

所以这个一次函数的解析式为 $\boxed{y = 3x + 2}$。

第 2 题

条件转化为：图像经过 $(4,\,0)$ 和 $(1,\,9)$。

设 $y = kx + b$（$k \neq 0$），分别代入两点：

$$\begin{cases} 4k + b = 0 \quad \text{①} \\ k + b = 9 \quad \text{②} \end{cases}$$

① - ② 得：$3k = -9$，解得 $k = -3$。

将 $k = -3$ 代入 ② 得：$b = 12$。

所以这个一次函数的解析式为 $\boxed{y = -3x + 12}$。

第 3 题

设 $y = kx + b$（$k \neq 0$），代入 $(0,\,-6)$：

$b = -6$。

代入 $(2,\,0)$：$2k + (-6) = 0$，解得 $k = 3$。

所以这个一次函数的解析式为 $\boxed{y = 3x - 6}$。

教师评判要点

  • 是否正确设出 $y = kx + b$ 并注明 $k \neq 0$。
  • 代入点坐标时 $x$、$y$ 的位置是否正确，正负号是否准确。
  • 方程组是否正确列出并能用消元法（加减消元或代入消元）正确求解。
  • 最终解析式是否写为 $y = kx + b$ 的标准形式，系数是否正确。
  • 第 3 题是否注意到 $y$ 轴交点的纵坐标直接等于 $b$，能简化计算。
  • 是否有检验意识（将已知点代入解析式验证）。

学生自评

  • 我能独立设出 $y = kx + b$ 并代入两个点坐标。
  • 我能正确列出关于 $k$、$b$ 的方程组并求解。
  • 我注意到了不同条件的表述方式（点坐标、"当 $x = a$ 时 $y = b$"、"与坐标轴交于"）。
  • 我写出了解析式并进行了检验。

如果以上全部能独立完成，说明你已经掌握了待定系数法求一次函数解析式的基本方法。

衔接预告

本次练习你掌握了：

  • 核心方法：待定系数法 —— 设 → 代 → 列 → 解 → 写
  • 关键转化：点在函数图像上 ⟺ 坐标满足函数关系式
  • 实用技巧：$y$ 轴截距直接等于 $b$，可简化计算

下一步我们将学习：

  • 用待定系数法求其他类型函数（如反比例函数 $y = \dfrac{k}{x}$、二次函数 $y = ax^2 + bx + c$）的解析式
  • 根据实际问题中的条件确定函数关系式
