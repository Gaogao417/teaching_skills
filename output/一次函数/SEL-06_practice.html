自适应练习：直线分割三角形面积求解析式

教师判断

当前档位：D → E 档过渡。

判断依据：学生函数基础扎实（已能求交点、设参数），需要提升的核心是"面积比 → 坐标比"的转化链和分类讨论的完整性。本组练习保留核心结构，换表层情境，训练转化链在不同情境下的应用，并逐步隐藏辅助线。

练习说明

本组练习围绕同一个核心动作：面积比 → 高之比 → 坐标比 → 斜率。每道题换了图形背景，但解法骨架一样。做完后对比三道题，看看"四步转化"怎么在不同题目里反复出现。

第 1 题 E 档·巩固

已知直线 $y = -2x + 6$ 与 $x$ 轴交于点 $A$，与 $y$ 轴交于点 $B$。直线 $l$ 经过原点 $O$，与线段 $AB$ 交于点 $C$，且 $S_{\triangle AOC} : S_{\triangle BOC} = 1:1$。求直线 $l$ 的解析式。

训练目标：在新的函数表达式中执行完整的四步转化，体会"面积比 $1:1$ 简化为 $|y_C|=|x_C|$"。

预期卡点：$|OA|$ 和 $|OB|$ 不再相等，需要认真算；面积比 $1:1$ 时绝对值方程 $\frac{|t \text{ 对应的量}|}{|t \text{ 对应的量}|}=1$ 的列法。

提示一

先求 $A$、$B$ 坐标。$A(3,0)$，$B(0,6)$。$|OA|=3$，$|OB|=6$。

提示二

$C$ 在 $AB$ 上，设 $C(t,\ -2t+6)$，$0 \le t \le 3$。$S_{\triangle AOC} = \frac{1}{2} \times 3 \times |-2t+6|$，$S_{\triangle BOC} = \frac{1}{2} \times 6 \times |t|$。面积相等意味着 $3|-2t+6| = 6|t|$。

第 2 题 E 档·迁移

在平面直角坐标系中，直线 $y = \frac{1}{2}x + 4$ 与 $x$ 轴交于点 $A$，与 $y$ 轴交于点 $B$。点 $C$ 是线段 $AB$ 上一点，直线 $OC$ 把 $\triangle AOB$ 的面积分为 $3:1$ 的两部分。求直线 $OC$ 的解析式。

训练目标：面积比变为 $3:1$，但转化链不变；体会"分母不变，分子变"的列式方式。

预期卡点：$|OA| = 8$、$|OB| = 4$，比例 $8:4=2:1$ 需要带入计算，不能简单认为面积比 = 高之比；绝对值处理时 $t$ 的范围变了。

提示一

$A(-8,0)$，$B(0,4)$。$|OA|=8$，$|OB|=4$。设 $C(t,\ \frac{1}{2}t+4)$，$-8 \le t \le 0$。

提示二

$S_{\triangle AOC} = \frac{1}{2}\times 8 \times |\frac{1}{2}t+4| = 4|\frac{1}{2}t+4|$，$S_{\triangle BOC} = \frac{1}{2}\times 4 \times |t| = 2|t|$。分两种情形：$4|\frac{t}{2}+4|:2|t| = 3:1$ 或 $1:3$。

第 3 题 F 档·结构隐藏

如图（请自画），在 $\triangle AOB$ 中，$A(0,6)$，$B(4,0)$，$O$ 为原点。点 $P$ 在线段 $AB$ 上，$\triangle AOP$ 的面积是 $\triangle BOP$ 面积的 $\dfrac{1}{3}$。求直线 $OP$ 的解析式。

训练目标：$A$ 在 $y$ 轴、$B$ 在 $x$ 轴（与原题相反），但解法不变；需要自己判断谁是"底"、谁是"高"。

预期卡点：$A$、$B$ 位置与原题对称，学生会把 $\triangle AOP$ 的底误选为 $OA$（应选 $OA$，但高是 $|x_P|$）；或者混淆 $\dfrac{1}{3}$ 对应哪个三角形。

提示一

画图：$A$ 在 $y$ 轴正半轴，$B$ 在 $x$ 轴正半轴。$|OA|=6$，$|OB|=4$。$AB$ 方程：$y = -\frac{3}{2}x + 6$。设 $P(t,\ -\frac{3}{2}t+6)$，$0 \le t \le 4$。

提示二

$\triangle AOP$ 以 $OA$ 为底，高 $= |x_P| = t$；$\triangle BOP$ 以 $OB$ 为底，高 $= |y_P| = -\frac{3}{2}t+6$。$\frac{\frac{1}{2}\times 6 \times t}{\frac{1}{2}\times 4 \times (-\frac{3}{2}t+6)} = \frac{1}{3}$。化简：$\frac{6t}{4(-\frac{3}{2}t+6)} = \frac{1}{3}$。别忘了还有一种情形。

参考答案与升级/降级判断

第 1 题

$A(3,0)$，$B(0,6)$。$|OA|=3$，$|OB|=6$。

设 $C(t,\ -2t+6)$，$0 \le t \le 3$。

$S_{\triangle AOC} = \frac{1}{2}\times 3 \times |-2t+6| = \frac{3}{2}|-2t+6|$

$S_{\triangle BOC} = \frac{1}{2}\times 6 \times |t| = 3|t|$

面积相等：$\frac{3}{2}|-2t+6| = 3|t|$，即 $|-2t+6| = 2|t|$。

在 $0 \le t \le 3$ 范围内：$-2t+6 > 0$，$t > 0$，所以 $-2t+6 = 2t \implies 4t = 6 \implies t = \frac{3}{2}$。

$C\!\left(\frac{3}{2},\ 3\right)$，$k = \dfrac{3}{3/2} = 2$。

答：$y = 2x$。（因为面积比 $1:1$ 对称，此题只有一解。若学生写了两解，说明分类讨论时没注意到 $1:1$ 本身对称。）

升级/降级判断：若独立完成 → 进入第 2 题；若卡在绝对值去符号 → 回到讲解页重新看"情形一"部分；若不会求 $A$、$B$ → 降回 C 档，补交点求法专项。

第 2 题

$A(-8,0)$，$B(0,4)$。$|OA|=8$，$|OB|=4$。

设 $C(t,\ \frac{t}{2}+4)$，$-8 \le t \le 0$。

$S_{\triangle AOC} = \frac{1}{2}\times 8 \times |\frac{t}{2}+4| = 4|\frac{t}{2}+4|$

$S_{\triangle BOC} = \frac{1}{2}\times 4 \times |t| = 2|t|$

情形一：$S_{\triangle AOC}:S_{\triangle BOC} = 3:1$

$4|\frac{t}{2}+4| = 3 \times 2|t| \implies |\frac{t}{2}+4| = \frac{3}{2}|t|$

$-8 \le t \le 0$：$\frac{t}{2}+4 \ge 0$，$|t| = -t$。$\frac{t}{2}+4 = -\frac{3}{2}t \implies 2t = -4 \implies t = -2$。

$C(-2,\ 3)$，$k = \dfrac{3}{-2} = -\dfrac{3}{2}$。$l: y = -\dfrac{3}{2}x$。

情形二：$S_{\triangle AOC}:S_{\triangle BOC} = 1:3$

$4|\frac{t}{2}+4| = \frac{1}{3} \times 2|t| \implies |\frac{t}{2}+4| = \frac{1}{6}|t|$

$\frac{t}{2}+4 = \frac{1}{6}(-t) \implies \frac{t}{2}+4 = -\frac{t}{6} \implies \frac{2t}{3} = -4 \implies t = -6$。

$C(-6,\ 1)$，$k = \dfrac{1}{-6} = -\dfrac{1}{6}$。$l: y = -\dfrac{1}{6}x$。

答：$y = -\dfrac{3}{2}x$ 或 $y = -\dfrac{1}{6}x$。

升级/降级判断：若两种情形都算对 → 进入第 3 题；若漏一种情形 → 强调"分为 $3:1$ 的两部分"有两种分法；若计算错误 → 检查绝对值去符号步骤。

第 3 题

$A(0,6)$，$B(4,0)$。$|OA|=6$，$|OB|=4$。$AB$ 方程：$y = -\frac{3}{2}x+6$。

设 $P(t,\ -\frac{3}{2}t+6)$，$0 \le t \le 4$。

$\triangle AOP$ 以 $OA$（$y$ 轴上）为底，高 $= |x_P| = t$；$\triangle BOP$ 以 $OB$（$x$ 轴上）为底，高 $= |y_P| = -\frac{3}{2}t+6$。

$S_{\triangle AOP} = \frac{1}{2}\times 6 \times t = 3t$，$S_{\triangle BOP} = \frac{1}{2}\times 4 \times (-\frac{3}{2}t+6) = 2(-\frac{3}{2}t+6) = -3t+12$。

情形一：$S_{\triangle AOP}:S_{\triangle BOP} = 1:3$（$S_{AOP} = \frac{1}{3}S_{BOP}$）

$3t = \frac{1}{3}(-3t+12) \implies 9t = -3t+12 \implies 12t = 12 \implies t = 1$。

$P(1,\ \frac{9}{2})$，$k = \dfrac{9/2}{1} = \dfrac{9}{2}$。$OP: y = \dfrac{9}{2}x$。

情形二：$S_{\triangle AOP}:S_{\triangle BOP} = 3:1$（$S_{BOP} = \frac{1}{3}S_{AOP}$）

$3t = 3(-3t+12) \implies 3t = -9t+36 \implies 12t = 36 \implies t = 3$。

$P(3,\ \frac{3}{2})$，$k = \dfrac{3/2}{3} = \dfrac{1}{2}$。$OP: y = \dfrac{1}{2}x$。

答：$y = \dfrac{9}{2}x$ 或 $y = \dfrac{1}{2}x$。

升级/降级判断：若独立完成并两解 → F 档掌握，下次可给结构隐藏更强的变式（如分点不在坐标轴围成的三角形中）；若底高选错 → 回到讲解页复习"谁做底、谁做高"；若只做一解 → 提醒分类讨论。

下一轮：根据本页完成情况，更新学生画像；必要时回到讲解页，或生成下一组变式。
三道题的递进设计：第 1 题（等分，熟悉流程）→ 第 2 题（非等分，$|OA|\neq|OB|$）→ 第 3 题（结构隐藏，$A$、$B$ 位置互换，底高需自行判断）。
