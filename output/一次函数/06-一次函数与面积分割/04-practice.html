一次函数与几何综合（面积分割）

难度默认：E档 适用：九年级一次函数复习

练习说明

本练习围绕"直线分割三角形面积求解析式"这一核心题型展开。所有题目都遵循同一套路：

  1. 求一次函数与坐标轴的交点坐标；
  2. 利用面积比转化为坐标分量之比；
  3. 求得分点坐标，进而确定分割直线的斜率；
  4. 分两种情形讨论，注意分点在线段上的范围约束。

每道题都配有分步提示和详细解答，请先独立思考，遇到困难再参考提示。完成全部题目后填写自评表。

核心转化链：面积比 $\longrightarrow$ 高之比（坐标分量之比）$\longrightarrow$ 分点参数 $\longrightarrow$ 斜率

第1题（基础巩固）

已知直线 $y = 2x + 6$ 与 $x$ 轴交于点 $A$，与 $y$ 轴交于点 $B$。直线 $l$ 经过原点 $O$，与线段 $AB$ 交于点 $C$，且把 $\triangle AOB$ 的面积分为 $1:2$ 的两部分。求直线 $l$ 的解析式。

分步提示
  1. 求交点：令 $y = 0$，得 $2x + 6 = 0$，所以 $A(-3,\,0)$；令 $x = 0$，得 $B(0,\,6)$。
  2. 面积比与坐标关系：$S_{\triangle AOC} = \dfrac{1}{2}\,|OA| \cdot |y_C|$，$S_{\triangle BOC} = \dfrac{1}{2}\,|OB| \cdot |x_C|$。
  3. 参数化分点：设 $C(t,\, 2t+6)$，其中 $-3 \le t \le 0$。
  4. 列比求解：分两种情形讨论面积比 $1:2$ 和 $2:1$，解出 $t$。
  5. 求斜率：$k = \dfrac{2t+6}{t}$，代入 $t$ 的值。
解答步骤

解：

第一步：求交点坐标

令 $y = 0$：$2x + 6 = 0 \Rightarrow x = -3$，所以 $A(-3,\,0)$。

令 $x = 0$：$y = 6$，所以 $B(0,\,6)$。

$|OA| = 3$，$|OB| = 6$。

第二步：参数化分点 $C$

因为 $C$ 在线段 $AB$ 上，设 $C(t,\, 2t+6)$，其中 $-3 \le t \le 0$。

第三步：面积比转化为坐标关系

$$S_{\triangle AOC} = \frac{1}{2} \times 3 \times |2t+6| = \frac{3|2t+6|}{2}$$

$$S_{\triangle BOC} = \frac{1}{2} \times 6 \times |t| = 3|t|$$

第四步：分两种情形讨论

情形一：$S_{\triangle AOC} : S_{\triangle BOC} = 1 : 2$

$$\frac{3|2t+6|}{2} : 3|t| = 1 : 2$$

$$\frac{|2t+6|}{2|t|} = \frac{1}{2} \Rightarrow |2t+6| = |t|$$

因为 $-3 \le t \le 0$，所以 $2t + 6 \ge 0$，$t \le 0$，去绝对值：

$$2t + 6 = -t \Rightarrow 3t = -6 \Rightarrow t = -2$$

验证：$t = -2 \in [-3,\,0]$，符合条件。$C(-2,\,2)$。

$k = \dfrac{2}{-2} = -1$，所以 $l:\ y = -x$。

情形二：$S_{\triangle AOC} : S_{\triangle BOC} = 2 : 1$

$$\frac{3|2t+6|}{2} : 3|t| = 2 : 1$$

$$\frac{|2t+6|}{2|t|} = 2 \Rightarrow |2t+6| = 4|t|$$

去绝对值：$2t + 6 = -4t \Rightarrow 6t = -6 \Rightarrow t = -1$。

验证：$t = -1 \in [-3,\,0]$，符合条件。$C(-1,\,4)$。

$k = \dfrac{4}{-1} = -4$，所以 $l:\ y = -4x$。

综上，直线 $l$ 的解析式为 $y = -x$ 或 $y = -4x$。

教师批注：

  • 本题 $|OA| \ne |OB|$（3 与 6），不能直接用坐标比等于面积比的简捷路径，需完整代入面积公式。
  • 易错点：绝对值展开时忘记 $t \le 0$ 导致正负号出错。
  • 若学生漏掉情形二，说明分类讨论意识不足，建议回看提示第4步。

第2题（对称巧解）

已知直线 $y = -x + 4$ 与 $x$ 轴交于点 $A$，与 $y$ 轴交于点 $B$。直线 $l$ 经过原点 $O$，与线段 $AB$ 交于点 $C$，把 $\triangle AOB$ 的面积分为 $1:1$ 的两部分（即等分）。求直线 $l$ 的解析式。

分步提示
  1. 求交点：$A(4,\,0)$，$B(0,\,4)$。注意 $|OA| = |OB| = 4$。
  2. 利用对称性：因为 $|OA| = |OB|$，所以 $S_{\triangle AOC} = S_{\triangle BOC}$ 等价于 $|y_C| = |x_C|$。
  3. 求分点：设 $C(t,\, -t+4)$，由 $|y_C| = |x_C|$ 且 $C$ 在第一象限，得 $-t+4 = t$。
  4. 求斜率：$k = \dfrac{-t+4}{t}$，代入求值。
解答步骤

解：

第一步：求交点坐标

令 $y = 0$：$-x + 4 = 0 \Rightarrow x = 4$，所以 $A(4,\,0)$。

令 $x = 0$：$y = 4$，所以 $B(0,\,4)$。

$|OA| = |OB| = 4$。

第二步：利用对称性

因为 $|OA| = |OB|$，面积等分意味着：

$$S_{\triangle AOC} = \frac{1}{2}|OA| \cdot y_C, \quad S_{\triangle BOC} = \frac{1}{2}|OB| \cdot x_C$$

由 $S_{\triangle AOC} = S_{\triangle BOC}$ 且 $|OA| = |OB|$，得 $y_C = x_C$。

第三步：求分点 $C$

设 $C(t,\, -t+4)$，其中 $0 \le t \le 4$。

由 $y_C = x_C$：$-t + 4 = t \Rightarrow 2t = 4 \Rightarrow t = 2$。

验证：$t = 2 \in [0,\,4]$，符合条件。$C(2,\,2)$。

第四步：求斜率

$$k = \frac{2}{2} = 1$$

所以直线 $l$ 的解析式为 $y = x$。

说明：本题面积比为 $1:1$（等分），由于图形关于 $y = x$ 对称，结果恰好为 $y = x$，只有一解。

教师批注：

  • 本题设计意图：让学生体会 $|OA| = |OB|$ 时，面积比直接退化为坐标分量之比的简捷路径。
  • 延伸提问：如果面积比改为 $1:3$，结果是什么？（$y = \dfrac{1}{3}x$ 或 $y = 3x$）
  • 部分学生可能疑惑"等分为什么只有一解"——因为等分线恰好是 $y = x$，两种情形重合。

第3题（变式提升）

已知直线 $y = \dfrac{1}{2}x + 2$ 与 $x$ 轴交于点 $A$，与 $y$ 轴交于点 $B$。直线 $l$ 经过点 $P(0,\,1)$，与线段 $AB$ 交于点 $C$，且 $\triangle APC$ 与 $\triangle BPC$ 的面积比为 $3:1$。求直线 $l$ 的解析式。

分步提示
  1. 求交点：$A(-4,\,0)$，$B(0,\,2)$。注意 $P(0,\,1)$ 在 $y$ 轴上，位于 $O$ 与 $B$ 之间。
  2. 画图理解：$C$ 在线段 $AB$ 上，$P$ 在 $y$ 轴上。$\triangle APC$ 和 $\triangle BPC$ 可以分别以 $PA$ 和 $PB$ 为底（在 $y$ 轴上），高为点 $C$ 到 $y$ 轴的水平距离 $|x_C|$。
  3. 面积关系：$S_{\triangle APC} = \dfrac{1}{2} \cdot |PA| \cdot |x_C|$，$S_{\triangle BPC} = \dfrac{1}{2} \cdot |PB| \cdot |x_C|$。因为高相同（都是 $|x_C|$），面积比 $= |PA| : |PB| = 1 : 1$……等等，题目给的是 $3:1$，这说明底不能都用 $y$ 轴上的线段。
  4. 换底重新分析：分别以 $x$ 轴上 $OA$ 段为公共参照。$S_{\triangle APC} = \dfrac{1}{2}|x_C| \cdot |y_P - y_A| = \dfrac{1}{2}|x_C| \cdot 1$，$S_{\triangle BPC} = \dfrac{1}{2}|x_C| \cdot |y_B - y_P| = \dfrac{1}{2}|x_C| \cdot 1$。
  5. 重新审视：更直接的方法——设 $l: y = kx + 1$，联立 $y = \dfrac{1}{2}x + 2$ 求交点 $C$，再用面积公式列方程。
  6. 参数化法：设 $C\!\left(t,\,\dfrac{t}{2}+2\right)$，$-4 \le t \le 0$。用 $P(0,1)$ 分别与 $A$、$B$、$C$ 构成三角形，通过面积比列方程。
解答步骤

解：

第一步：求交点坐标

令 $y = 0$：$\dfrac{1}{2}x + 2 = 0 \Rightarrow x = -4$，所以 $A(-4,\,0)$。

令 $x = 0$：$y = 2$，所以 $B(0,\,2)$。

已知 $P(0,\,1)$。

第二步：设分点 $C$ 并参数化

设 $C\!\left(t,\,\dfrac{t}{2}+2\right)$，其中 $-4 \le t \le 0$。

第三步：计算面积

以 $PC$ 为底边方向，将 $\triangle APC$ 和 $\triangle BPC$ 都拆为"以 $y$ 轴上的线段为底、以水平距离为高"：

$$S_{\triangle APC} = \frac{1}{2}|x_C| \cdot |y_P - y_A| = \frac{1}{2}(-t) \cdot 1 = -\frac{t}{2}$$

$$S_{\triangle BPC} = \frac{1}{2}|x_C| \cdot |y_B - y_P| = \frac{1}{2}(-t) \cdot 1 = -\frac{t}{2}$$

面积比始终为 $1:1$，与题目 $3:1$ 矛盾——说明需要换一种分割方式。

重新分析：直接用坐标公式计算三角形面积

$$S_{\triangle APC} = \frac{1}{2}\left|x_A(y_P - y_C) + x_P(y_C - y_A) + x_C(y_A - y_P)\right|$$

$$= \frac{1}{2}\left|(-4)\!\left(1 - \frac{t}{2}-2\right) + 0 + t(0-1)\right|$$

$$= \frac{1}{2}\left|(-4)\!\left(-1 - \frac{t}{2}\right) - t\right| = \frac{1}{2}\left|4 + 2t - t\right| = \frac{1}{2}|t + 4|$$

因为 $-4 \le t \le 0$，所以 $t + 4 \ge 0$：

$$S_{\triangle APC} = \frac{t + 4}{2}$$

$$S_{\triangle BPC} = \frac{1}{2}\left|x_B(y_P - y_C) + x_P(y_C - y_B) + x_C(y_B - y_P)\right|$$

$$= \frac{1}{2}\left|0 + 0 + t(2-1)\right| = \frac{1}{2}|t| = -\frac{t}{2}$$

第四步：分两种情形列方程

情形一：$S_{\triangle APC} : S_{\triangle BPC} = 3 : 1$

$$\frac{t+4}{2} : \left(-\frac{t}{2}\right) = 3 : 1$$

$$\frac{t+4}{-t} = 3 \Rightarrow t + 4 = -3t \Rightarrow 4t = -4 \Rightarrow t = -1$$

验证：$t = -1 \in [-4,\,0]$，符合。$C\!\left(-1,\,\dfrac{3}{2}\right)$。

$$k = \frac{\frac{3}{2} - 1}{-1 - 0} = \frac{\frac{1}{2}}{-1} = -\frac{1}{2}$$

所以 $l:\ y = -\dfrac{1}{2}x + 1$。

情形二：$S_{\triangle APC} : S_{\triangle BPC} = 1 : 3$

$$\frac{t+4}{2} : \left(-\frac{t}{2}\right) = 1 : 3$$

$$\frac{t+4}{-t} = \frac{1}{3} \Rightarrow 3(t+4) = -t \Rightarrow 4t = -12 \Rightarrow t = -3$$

验证：$t = -3 \in [-4,\,0]$，符合。$C\!\left(-3,\,\dfrac{1}{2}\right)$。

$$k = \frac{\frac{1}{2}-1}{-3-0} = \frac{-\frac{1}{2}}{-3} = \frac{1}{6}$$

所以 $l:\ y = \dfrac{1}{6}x + 1$。

综上，直线 $l$ 的解析式为 $y = -\dfrac{1}{2}x + 1$ 或 $y = \dfrac{1}{6}x + 1$。

教师批注：

  • 本题关键变化：定点从原点移到 $P(0,1)$，不再有 $|OA|=|OB|$ 的对称性，且不能用简单的"底 $\times$ 高"直接做。
  • 能力分层：E档学生重点掌握设参数、列方程的框架；D档以上应体会坐标公式法的普适性。
  • 易错：面积比的方向（哪个三角形在前）容易混淆，提醒学生标注清楚 $S_{\triangle APC}$ 和 $S_{\triangle BPC}$ 各自的面积表达式。

学习交接

通过本练习，你应掌握以下内容：

  • 题型识别：看到"直线分割三角形面积"立即想到：面积比 $\to$ 坐标比 $\to$ 分点 $\to$ 斜率。
  • 分类讨论：面积比有两种分配方向，务必分别讨论。
  • 范围约束：分点必须在线段上，解出的参数要检验范围。
  • 对称性利用：$|OA| = |OB|$ 时可以大幅简化计算。

下一阶段预告：

  • 分割直线不再过 $y$ 轴上的定点，而是过一般位置的定点——需要更灵活地选择"底和高"的组合。
  • 面积比从直接给定变为间接给出（如一个三角形面积是另一个的2倍少2等）。
  • 综合压轴题中，面积分割往往与动点问题结合出现。
