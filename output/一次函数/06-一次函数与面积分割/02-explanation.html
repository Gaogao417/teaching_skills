一次函数与几何综合（面积分割）

题目

已知直线 \(y=x+3\) 的图象与 \(x\) 轴、\(y\) 轴交于 \(A\)、\(B\) 两点，直线 \(l\) 经过原点，与线段 \(AB\) 交于点 \(C\)，把 \(\triangle AOB\) 的面积分为 \(2:1\) 的两部分，求直线 \(l\) 的解析式。

一、先把题目拆开

项目内容
已知条件 直线 \(y=x+3\) 与 \(x\) 轴交于 \(A\)、与 \(y\) 轴交于 \(B\)；直线 \(l\) 过原点 \(O\)；直线 \(l\) 与线段 \(AB\) 交于点 \(C\)；\(\triangle AOB\) 被分成的两块面积之比为 \(2:1\)
要求目标 求直线 \(l\) 的解析式（即求斜率 \(k\)）
关键词 面积比、过原点、线段上的交点
容易忽略 \(C\) 必须在线段 \(AB\) 上（不是延长线）；面积比 \(2:1\) 有两个方向（\(\triangle AOC\) 大还是 \(\triangle BOC\) 大？），需要分类讨论

二、这题准备怎么走

  1. 求出 \(A\)、\(B\) 的坐标
  2. 利用面积比 \(= |OA| \cdot |y_C| : |OB| \cdot |x_C|\) 建立方程
  3. 发现 \(|OA|=|OB|=3\)，化简为 \(|y_C|:|x_C|=2:1\) 或 \(1:2\)
  4. 因为 \(C\) 在直线 \(AB\) 上，用参数表示 \(C(t,\,t+3)\)，代入坐标比解出 \(t\)
  5. 分两种情形讨论，算出 \(k=\dfrac{t+3}{t}\)，写出两条直线 \(l\) 的解析式

三、关键想法

两个三角形 \(\triangle AOC\) 和 \(\triangle BOC\) 共用顶点 \(O\)，它们分别以 \(OA\) 和 \(OB\) 为底、以点 \(C\) 的 \(y\) 坐标和 \(x\) 坐标为高。因此

\[ \frac{S_{\triangle AOC}}{S_{\triangle BOC}} = \frac{\tfrac{1}{2}\cdot|OA|\cdot|y_C|}{\tfrac{1}{2}\cdot|OB|\cdot|x_C|} = \frac{|y_C|}{|x_C|} \]

而本题恰好 \(|OA|=|OB|=3\)（对称！），所以面积比 直接等于坐标分量之比，不需要解方程组求交点——一步到位。

四、标准解法

第 1 步：求 A、B 坐标

求交点

直线 \(y=x+3\)：

令 \(y=0\)，则 \(x+3=0\)，得 \(A(-3,\,0)\)；

令 \(x=0\)，则 \(y=3\)，得 \(B(0,\,3)\)。

为什么：一次函数与坐标轴的交点，就是分别令另一个变量为零。

第 2 步：参数化 C 并列方程

设 C 坐标

因为 \(C\) 在线段 \(AB\) 上，而直线 \(AB\) 的方程是 \(y=x+3\)，所以可设 \(C(t,\,t+3)\)，其中 \(-3 \le t \le 0\)。

为什么：参数化比分段讨论简洁，一个字母 \(t\) 就确定了 \(C\) 的位置。

列面积比方程

\[ \frac{S_{\triangle AOC}}{S_{\triangle BOC}} = \frac{\tfrac{1}{2}\cdot 3 \cdot |t+3|}{\tfrac{1}{2}\cdot 3 \cdot |t|} = \frac{|t+3|}{|t|} \]

为什么：底 \(OA=OB=3\) 约掉了，面积比只剩下 \(|y_C|:|x_C|\)。

第 3 步：分两种情形求解

情形一：\(S_{\triangle AOC} : S_{\triangle BOC} = 2 : 1\)

\[ \frac{|t+3|}{|t|} = 2 \]

因为 \(-3 \le t \le 0\)，所以 \(t \le 0\)（即 \(|t|=-t\)），\(t+3 \ge 0\)（即 \(|t+3|=t+3\)）：

\[ \frac{t+3}{-t} = 2 \;\Longrightarrow\; t+3 = -2t \;\Longrightarrow\; 3t = -3 \;\Longrightarrow\; t = -1 \]

此时 \(C(-1,\,2)\)，斜率 \(k = \dfrac{2}{-1} = -2\)，直线 \(l\)：\(y = -2x\)。

情形二：\(S_{\triangle AOC} : S_{\triangle BOC} = 1 : 2\)

\[ \frac{|t+3|}{|t|} = \frac{1}{2} \]

同理，\(t+3 \ge 0\)，\(t \le 0\)：

\[ \frac{t+3}{-t} = \frac{1}{2} \;\Longrightarrow\; 2(t+3) = -t \;\Longrightarrow\; 3t = -6 \;\Longrightarrow\; t = -2 \]

此时 \(C(-2,\,1)\)，斜率 \(k = \dfrac{1}{-2} = -\dfrac{1}{2}\)，直线 \(l\)：\(y = -\dfrac{1}{2}x\)。

第 4 步：写出答案

最终结果

\[ \boxed{y = -2x \quad \text{或} \quad y = -\frac{1}{2}x} \]

为什么：面积比没有指定哪部分大，所以两个方向都要考虑，两条直线都符合题意。

五、边讲边问

问题 1

直线 \(y=x+3\) 与 \(x\) 轴的交点 \(A\) 坐标是什么？与 \(y\) 轴的交点 \(B\) 呢？

 

问题 2

\(\triangle AOB\) 的面积是多少？底和高分别是谁？

 

问题 3

\(\triangle AOC\) 以 \(OA\) 为底时，高是点 \(C\) 的哪个坐标？\(\triangle BOC\) 以 \(OB\) 为底时，高又是哪个坐标？

 

问题 4

为什么本题面积比可以直接写成 \(|y_C|:|x_C|\)？（提示：看看 \(|OA|\) 和 \(|OB|\) 的关系）

 

问题 5

如果题目说"靠近 \(A\) 的那块面积是靠近 \(B\) 的两倍"，还需要分两种情形吗？为什么？

 

问题 6

求出 \(t=-1\) 后，怎么从 \(C\) 的坐标得到直线 \(l\) 的斜率？写出你的计算过程。

 

六、易错提醒

易错 1：只算了一种情形。 题目说"面积分为 2:1"，没有说哪部分是 2、哪部分是 1，所以必须分 \(\triangle AOC\) 大和 \(\triangle BOC\) 大两种情况讨论，否则会漏掉一条直线。

易错 2：忘记绝对值。 在线段 \(AB\) 上，\(t\) 是负数（\(-3 \le t \le 0\)），所以 \(|t|=-t\) 而不是 \(t\)。如果直接写 \(\dfrac{t+3}{t}=2\)，右边是正数、左边是负数，会得出矛盾。

易错 3：不考虑 C 在线段上的范围。 解完方程后要检查 \(t\) 是否在 \([-3,0]\) 内。如果题目条件稍有变化，解出的 \(t\) 可能不满足范围约束，此时该情形无解。

易错 4：底和高选错。 \(\triangle AOC\) 的底是 \(OA\)（在 \(x\) 轴上），对应的高是点 \(C\) 到 \(x\) 轴的距离，即 \(|y_C|\)，不是 \(|x_C|\)。两者不要搞混。

七、一句话总结

核心转化链：面积比 → 高之比（坐标分量之比）→ 分点参数 → 斜率。共顶点的两个三角形面积比 = 底 × 高之比；底相等时，面积比 = 高之比 = 坐标分量之比，一步到位。
