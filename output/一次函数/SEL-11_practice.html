自适应练习：一次函数与反比例函数交点——中点关系

练习说明

本组练习训练两个核心动作：

  1. 共线等距 → 中点关系：看到"三点共线 + 两段等长"，直接用中点公式
  2. 平行坐标轴 → 坐标相同：平行 $x$ 轴则纵坐标相同，平行 $y$ 轴则横坐标相同

共 3 题，做完后对照参考答案自批，记录错在哪里。

第 1 题

在平面直角坐标系中，直线 $l$ 与 $x$ 轴交于 $A(-4, 0)$，与 $y$ 轴正半轴交于 $B$，与反比例函数 $y = \dfrac{k}{x}\ (x > 0)$ 的图象交于 $C$。已知 $AB = BC$，$C$ 的纵坐标为 6。

  1. 求 $B$、$C$ 的坐标；
  2. 求直线 $l$ 的表达式；
  3. 求反比例函数的表达式。

训练目标共线等距→中点公式求坐标 + 待定系数法

预期卡点看不出 AB=BC 就是中点条件；中点公式应用出错

提示一

$A$、$B$、$C$ 都在直线 $l$ 上，且 $AB = BC$。$B$ 和 $A$、$C$ 是什么关系？

提示二

$B$ 是 $AC$ 中点。$x_B = 0$（$B$ 在 $y$ 轴上），所以 $0 = \dfrac{-4 + x_C}{2}$ → $x_C = 4$。$y_B = \dfrac{0 + 6}{2} = 3$。得 $B(0, 3)$，$C(4, 6)$。

第 2 题

已知 $A(1, 2)$、$B(3, y)$、$C(5, 6)$ 三点在同一条直线上。

  1. 若 $AB = BC$，求 $B$ 的坐标；
  2. 若 $AB : BC = 1 : 3$（$B$ 在 $AC$ 上），求 $B$ 的坐标。

训练目标中点公式的巩固；分点公式的初步接触

预期卡点第(2)小题比例不是 1:1，需推广中点公式为分点公式

提示一

第(1)小题：$AB = BC$ → $B$ 是中点。第(2)小题：$AB : BC = 1 : 3$ 意味着 $B$ 把 $AC$ 分成 $1:3$，可以用分点公式或"从 A 出发走 $\frac{1}{1+3}$ 的路"来想。

提示二

第(1)小题：$B\left(\dfrac{1+5}{2}, \dfrac{2+6}{2}\right) = (3, 4)$。第(2)小题：$x_B = 1 + \dfrac{1}{4}(5-1) = 2$，$y_B = 2 + \dfrac{1}{4}(6-2) = 3$，即 $B(2, 3)$。

第 3 题

直线 $AB$ 与 $x$ 轴交于 $A(-4, 0)$，与 $y$ 轴正半轴交于 $B$，与反比例函数 $y = \dfrac{n}{x}\ (x > 0)$ 的图象交于 $C$，且 $AB = BC$，点 $C$ 的纵坐标为 $6$。

  1. 求直线 $AB$ 的表达式与反比例函数的表达式；
  2. 过点 $B$ 作 $BD \parallel x$ 轴，交反比例函数 $y = \dfrac{n}{x}$ 的图象于 $D$，求线段 $CD$ 的长度。

训练目标原题结构的巩固：中点 + 平行坐标轴 + 距离公式完整流程

预期卡点换数字后中点公式计算出错；BD // $x$ 轴后 $y_D = y_B$ 设错

提示一

第(1)小题和第 1 题思路完全一样：$AB = BC$ 共线 → $B$ 是 $AC$ 中点。第(2)小题也和原题第(2)问思路一致。

提示二

$x_B = 0 = \dfrac{-4 + x_C}{2}$ → $x_C = 4$；$y_B = \dfrac{0+6}{2} = 3$。$B(0,3)$，$C(4,6)$。直线过 $A(-4,0)$ 和 $B(0,3)$：$k = \dfrac{3}{4}$，$y = \dfrac{3}{4}x + 3$。$n = 4 \times 6 = 24$。$BD \parallel x$ 轴：$y_D = 3$，$x_D = \dfrac{24}{3} = 8$，$D(8,3)$。$CD = \sqrt{(8-4)^2 + (3-6)^2} = \sqrt{16+9} = 5$。

参考答案与升级/降级判断

第 1 题

  1. $B$ 是 $AC$ 中点。$x_B = 0 = \dfrac{-4 + x_C}{2}$ → $x_C = 4$；$y_B = \dfrac{0 + 6}{2} = 3$。$B(0, 3)$，$C(4, 6)$。
  2. 直线 $l$ 过 $A(-4, 0)$ 和 $B(0, 3)$：斜率 $k = \dfrac{3-0}{0-(-4)} = \dfrac{3}{4}$，截距 $b = 3$。$y = \dfrac{3}{4}x + 3$。
  3. $C(4, 6)$ 在 $y = \dfrac{k}{x}$ 上：$k = 4 \times 6 = 24$。$y = \dfrac{24}{x}$。

判断：

  • 3 小题全对 → 升至 D 档
  • 第(1)小题出错（看不出中点）→ 降至 B 档，需回到讲解页复习"共线等距"概念
  • 第(2)(3)小题出错 → 维持 C 档，待定系数法需巩固

第 2 题

  1. $B$ 是 $AC$ 中点：$B\left(\dfrac{1+5}{2}, \dfrac{2+6}{2}\right) = \mathbf{(3, 4)}$。
  2. $AB : BC = 1 : 3$，$B$ 将 $AC$ 分为 $1:3$。从 $A$ 出发走 $\dfrac{1}{1+3} = \dfrac{1}{4}$ 的路到达 $B$：

    $x_B = 1 + \dfrac{1}{4}(5 - 1) = 1 + 1 = 2$

    $y_B = 2 + \dfrac{1}{4}(6 - 2) = 2 + 1 = 3$

    $B(2, 3)$

判断：

  • 2 小题全对 → 升至 D 档，具备推广到分点公式的能力
  • 第(1)小题出错 → 维持 C 档，中点公式基础不牢
  • 第(2)小题答不出 → 维持 C 档，但已达到中点掌握水平

第 3 题

  1. $B$ 是 $AC$ 中点：$x_B = 0 = \dfrac{-4 + x_C}{2}$ → $x_C = 4$；$y_B = \dfrac{0+6}{2} = 3$。

    $B(0, 3)$，$C(4, 6)$。直线 $AB$ 过 $A(-4, 0)$ 和 $B(0, 3)$：$k = \dfrac{3-0}{0-(-4)} = \dfrac{3}{4}$，$y = \dfrac{3}{4}x + 3$。

    $n = x_C \cdot y_C = 4 \times 6 = 24$，$y = \dfrac{24}{x}$。

  2. $BD \parallel x$ 轴 → $y_D = y_B = 3$。$D$ 在 $y = \dfrac{24}{x}$ 上：$x_D = \dfrac{24}{3} = 8$，$D(8, 3)$。

    $C(4, 6)$，$D(8, 3)$。$CD = \sqrt{(8-4)^2 + (3-6)^2} = \sqrt{16 + 9} = \sqrt{25} = \mathbf{5}$。

判断：

  • 2 小题全对 → 升至 E 档，原题结构已巩固，可做变式
  • 第(1)小题出错 → 维持 C 档，中点公式或待定系数法需强化
  • 第(2)小题出错 → 维持 C 档，平行坐标轴的坐标转化需专项练习
