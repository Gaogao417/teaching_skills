一次函数与不等式

练习说明：本练习共 3 题，围绕"一次函数值与不等式的关系"展开。请先独立完成每题的解答，再对照答案检查。遇到困难时可参考提示，但尽量先自己尝试。

难度定位：基础巩固（E 档）

练习 1

训练目标：将函数值条件转化为关于 \(x\) 的方程与不等式，利用增减性判断方向

已知一次函数 \(y = 2x - 5\)：

(1) 当 \(x\) 取何值时，\(y = 3\)？

(2) 当 \(x\) 取何值时，\(y > 3\)？

(3) 当 \(x\) 取何值时，\(y < 3\)？

提示

将 \(y = 3\) 代入解析式，先解方程 \(2x - 5 = 3\)，得到分界点 \(x = 4\)。再观察 \(k = 2 > 0\)，函数递增，即可直接写出不等式的解集。

教师备注：本题为基础题，\(k = 2 > 0\) 为正，学生需掌握"先求分界点，再由增减性定方向"的基本流程。常见卡点：部分学生会忘记判断 \(k\) 的符号，直接默认不等号方向不变。

练习 2

训练目标：递减函数中不等号方向反转；复合不等式拆分与交集

已知一次函数 \(y = -x + 8\)：

(1) 当 \(x\) 取何值时，\(y = 2\)？

(2) 当 \(x\) 取何值时，\(y > 2\)？

(3) 当 \(x\) 取何值时，\(y \leqslant 6\)？

提示

(1) 代入解方程 \(-x + 8 = 2\)，得 \(x = 6\)。
(2) 注意 \(k = -1 < 0\)，函数递减！\(y\) 越大，\(x\) 越小，所以 \(y > 2\) 对应 \(x < 6\)。
(3) 先求 \(y = 6\) 时的分界点：\(-x + 8 = 6\)，得 \(x = 2\)。递减函数，\(y \leqslant 6\) 对应 \(x \geqslant 2\)。

教师备注：本题核心变化在于 \(k < 0\)（递减），不等号方向需要反转。中等学生容易在此出错——习惯性地认为"大于就对应大于"。第 (3) 问使用 \(\leqslant\) 而非 \(<\)，训练学生对等号边界的敏感度。

练习 3

训练目标：复合不等式 \(a < y < b\) 的拆分与交集运算

已知一次函数 \(y = \dfrac{1}{2}x + 1\)：

(1) 当 \(x\) 取何值时，\(y = 4\)？

(2) 当 \(x\) 取何值时，\(y < 4\)？

(3) 当 \(x\) 取何值时，\(1 < y < 4\)？

提示

(1) 解方程 \(\dfrac{1}{2}x + 1 = 4\)，两边减 1 得 \(\dfrac{1}{2}x = 3\)，所以 \(x = 6\)。
(2) \(k = \dfrac{1}{2} > 0\)，递增，\(y < 4\) 对应 \(x < 6\)。
(3) 复合不等式拆分为两部分：左半 \(y > 1\) 即 \(\dfrac{1}{2}x + 1 > 1\)，得 \(x > 0\)；右半 \(y < 4\) 即 \(x < 6\)。取交集：\(0 < x < 6\)。

教师备注：第 (3) 问是本题的难点。常见错误：只解一边忘另一边；或者解出 \(x > 0\) 和 \(x < 6\) 后写成并集 \(x < 0\) 或 \(x > 6\)。教师应强调"取交集"而非"取并集"。\(y = 1\) 对应 \(x = 0\) 是一个自然数分界点，学生容易验证。

答案与解析

练习 1 答案

  1. 代入 \(y = 3\)：\(\quad 2x - 5 = 3\)

    \(\quad 2x = 8\)

    \(\quad \boxed{x = 4}\)

  2. 因为 \(k = 2 > 0\)，函数递增，\(y\) 随 \(x\) 增大而增大。

    \(y > 3\) 时，\(x > 4\)。

    \(\quad \boxed{x > 4}\)

  3. 同理，递增函数中 \(y < 3\) 对应 \(x < 4\)。

    \(\quad \boxed{x < 4}\)

练习 2 答案

  1. 代入 \(y = 2\)：\(\quad -x + 8 = 2\)

    \(\quad -x = -6\)

    \(\quad \boxed{x = 6}\)

  2. 因为 \(k = -1 < 0\)，函数递减，\(y\) 随 \(x\) 增大而减小。

    \(y > 2\) 时，\(x < 6\)（方向反转！）。

    \(\quad \boxed{x < 6}\)

  3. 先求分界点：\(-x + 8 = 6\)，解得 \(x = 2\)。

    递减函数中 \(y \leqslant 6\) 对应 \(x \geqslant 2\)。

    \(\quad \boxed{x \geqslant 2}\)

练习 3 答案

  1. 代入 \(y = 4\)：\(\quad \dfrac{1}{2}x + 1 = 4\)

    \(\quad \dfrac{1}{2}x = 3\)

    \(\quad \boxed{x = 6}\)

  2. \(k = \dfrac{1}{2} > 0\)，递增函数。

    \(y < 4\) 对应 \(x < 6\)。

    \(\quad \boxed{x < 6}\)

  3. 拆分复合不等式 \(1 < y < 4\)：

    \(\quad\)左半：\(\dfrac{1}{2}x + 1 > 1 \;\Longrightarrow\; \dfrac{1}{2}x > 0 \;\Longrightarrow\; x > 0\)

    \(\quad\)右半：\(\dfrac{1}{2}x + 1 < 4 \;\Longrightarrow\; \dfrac{1}{2}x < 3 \;\Longrightarrow\; x < 6\)

    \(\quad\)取交集：\(\boxed{0 < x < 6}\)

教师评判标准

练习 1：全对 → 保持当前档位；第 (2)(3) 问方向写反 → 提醒增减性判断，不降档。

练习 2：全对（含不等号反转）→ 可升级；第 (2) 或 (3) 问方向未反转 → 降一档，回到递增函数基础练习。

练习 3：全对（含交集）→ 确认达标；复合不等式只解一边或取并集 → 重点复习交集概念，不降档但需订正。

自我检查

完成全部练习后，请对照答案逐题检查：

  1. 方程求解是否正确？（代入验证）
  2. 不等式方向是否与 \(k\) 的正负一致？
  3. 复合不等式是否分别解了两边并取交集？
  4. 端点的开闭（等号有无）是否正确？
