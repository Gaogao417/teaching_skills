一次函数的性质（分类讨论）

练习说明

本练习围绕一次函数的分类讨论展开。核心方法：

  1. 识别关键参数（通常为斜率 $k$）的符号是否确定；
  2. 若不确定，分别按 $k>0$ 和 $k<0$ 两种情况建立方程组；
  3. 求出所有可能的解析式，并验证是否符合题目条件。

每道题后附有提示、解题步骤和答案，供你自主检查。

第 1 题

已知一次函数 $y=kx+b$ 的自变量的取值范围是 $-2$ 到 $4$，相应的函数值的取值范围是 $-1$ 到 $5$。求这个一次函数的解析式。

提示

$k$ 的正负未知，需要分两种情况讨论。

  • $k>0$（递增）：$x$ 最小对应 $y$ 最小，$x$ 最大对应 $y$ 最大。
  • $k<0$（递减）：反过来配对。
解题步骤

情况一：$k>0$（递增函数）

$x$ 从小到大对应 $y$ 从小到大：

$$\begin{cases} -2k + b = -1 \\ 4k + b = 5 \end{cases}$$

两式相减：$6k = 6$，解得 $k = 1$；代入得 $b = 1$。

解析式为 $y = x + 1$。

情况二：$k<0$（递减函数）

$x$ 从小到大对应 $y$ 从大到小：

$$\begin{cases} -2k + b = 5 \\ 4k + b = -1 \end{cases}$$

两式相减：$-6k = 6$，解得 $k = -1$；代入得 $b = 3$。

解析式为 $y = -x + 3$。

验证：$y = x + 1$ 在 $x \in [-2, 4]$ 时，$y \in [-1, 5]$，符合。$y = -x + 3$ 在 $x \in [-2, 4]$ 时，$y \in [-1, 5]$，符合。

教师备注（E档）：本题数值较简单，$k = \pm 1$，$b$ 为整数。适合作为入门练习，帮助学生建立分类讨论的意识。重点观察学生是否主动分两种情况，还是只写出一个解。

答案：$y = x + 1$ 或 $y = -x + 3$

第 2 题

已知一次函数 $y=kx+b$ 中，当 $x$ 的取值范围是 $1$ 到 $5$ 时，$y$ 的取值范围是 $-3$ 到 $9$。求这个一次函数的解析式。

提示

仍然需要分 $k>0$ 和 $k<0$ 讨论。注意这里 $x$ 的范围都是正数，但讨论方法不变。

  • $k>0$：$(1, -3)$、$(5, 9)$
  • $k<0$：$(1, 9)$、$(5, -3)$
解题步骤

情况一：$k>0$（递增函数）

$$\begin{cases} k + b = -3 \\ 5k + b = 9 \end{cases}$$

两式相减：$4k = 12$，解得 $k = 3$；代入得 $b = -6$。

解析式为 $y = 3x - 6$。

情况二：$k<0$（递减函数）

$$\begin{cases} k + b = 9 \\ 5k + b = -3 \end{cases}$$

两式相减：$-4k = -12$，解得 $k = -3$；代入得 $b = 12$。

解析式为 $y = -3x + 12$。

验证：$y = 3x - 6$ 在 $x \in [1, 5]$ 时，$y \in [-3, 9]$，符合。$y = -3x + 12$ 在 $x \in [1, 5]$ 时，$y \in [-3, 9]$，符合。

教师备注（E档）：与第 1 题结构完全相同，但 $x$ 范围改为正数区间，避免负数代入带来的额外出错点。$k = \pm 3$，$b$ 为整数，计算量仍然很小。可观察学生是否形成了解题模板意识。

答案：$y = 3x - 6$ 或 $y = -3x + 12$

第 3 题

已知一次函数 $y=kx+b$ 的图像经过点 $(-1, 6)$ 和 $(2, -3)$。求这个一次函数的解析式。

提示

本题已经明确给出了两个点，不需要分类讨论！直接用待定系数法即可。

将 $(-1, 6)$ 和 $(2, -3)$ 分别代入 $y = kx + b$，得到关于 $k$、$b$ 的方程组。

解题步骤

将两个点代入 $y = kx + b$：

$$\begin{cases} -k + b = 6 \\ 2k + b = -3 \end{cases}$$

两式相减：$3k = -9$，解得 $k = -3$。

代入第一式：$3 + b = 6$，解得 $b = 3$。

解析式为 $y = -3x + 3$。

教师备注（E档）：本题作为对比题，故意不给范围而给两个确定的点。目的是让学生区分"需要分类讨论"和"不需要分类讨论"的情境。观察学生是否仍然画蛇添足地分两种情况。若学生能直接待定系数法求解，说明已真正理解分类讨论的触发条件。

答案：$y = -3x + 3$

自我检查

完成全部练习后，请对照以下清单核对：

  • 是否对每一道范围型题目都分了 $k>0$ 和 $k<0$ 两种情况？
  • $k<0$ 时，端点的对应关系是否正确（反向配对）？
  • 解方程组时，计算是否正确（可以用代入法验证）？
  • 最终结果是否验证了符合原题的取值范围？
  • 第 3 题是否避免了不必要的分类讨论？

答案汇总

第 1 题：$y = x + 1 \quad$ 或 $\quad y = -x + 3$

第 2 题：$y = 3x - 6 \quad$ 或 $\quad y = -3x + 12$

第 3 题：$y = -3x + 3$

教学交接说明

当前档位：E 档（基础入门）

练习目标：建立分类讨论意识，掌握"给范围求解析式"的基本流程。

第 1 题：原题结构，数值简单，检验学生是否知道需要分类讨论。

第 2 题：结构复现，正数区间，巩固解题模板。

第 3 题：对比题，已知两点不涉及分类讨论，检验学生能否正确判断何时需要讨论、何时不需要。

升级条件：若学生三题全部正确且用时合理，可升至 D 档（引入非整数 $k$ 或增加验证步骤的复杂度）。

降级信号：若学生在第 1 题就漏解（只写出一个解析式），需回到讲解阶段，强化分类讨论的必要性。
