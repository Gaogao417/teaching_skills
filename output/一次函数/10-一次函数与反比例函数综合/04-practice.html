第 1 题

已知直线 $l$ 与 $x$ 轴交于点 $A(-3, 0)$，与 $y$ 轴的正半轴交于点 $B$，与反比例函数 $y=\dfrac{k}{x}\;(x>0)$ 的图象交于点 $C$，且 $AB=BC$。点 $C$ 的纵坐标为 $6$。

(1) 求直线 $l$ 的表达式；

(2) 过点 $B$ 作 $BD \parallel x$ 轴，交反比例函数 $y=\dfrac{k}{x}$ 的图象于点 $D$，求线段 $CD$ 的长度。

点击查看提示
  1. $A$、$B$、$C$ 三点共线且 $AB=BC$，说明 $B$ 是 $AC$ 的什么？
  2. 利用中点坐标公式：$x_B = \dfrac{x_A + x_C}{2}$，$y_B = \dfrac{y_A + y_C}{2}$。注意 $B$ 在 $y$ 轴上，所以 $x_B = 0$。
  3. 第 (2) 问：$BD \parallel x$ 轴意味着 $D$ 与 $B$ 的纵坐标相同。
点击查看解答步骤

(1)

因为 $A$、$B$、$C$ 共线且 $AB=BC$，所以 $B$ 是 $AC$ 的中点。

中点公式：$x_B = \dfrac{x_A + x_C}{2} = \dfrac{-3 + x_C}{2} = 0$，解得 $x_C = 3$。

$y_B = \dfrac{y_A + y_C}{2} = \dfrac{0 + 6}{2} = 3$，所以 $B(0, 3)$，$C(3, 6)$。

$C(3, 6)$ 在 $y = \dfrac{k}{x}$ 上，所以 $k = 3 \times 6 = 18$。

直线 $l$ 过 $A(-3, 0)$ 和 $B(0, 3)$，斜率 $= \dfrac{3 - 0}{0 - (-3)} = 1$，所以直线 $l$ 的表达式为 $\boxed{y = x + 3}$。

(2)

$BD \parallel x$ 轴，所以 $y_D = y_B = 3$。$D$ 在 $y = \dfrac{18}{x}$ 上，$x_D = \dfrac{18}{3} = 6$，所以 $D(6, 3)$。

$CD = \sqrt{(6-3)^2 + (3-6)^2} = \sqrt{9 + 9} = \boxed{3\sqrt{2}}$。

教师备注：本题与原题结构完全一致，仅将 $A$ 从 $(-2,0)$ 改为 $(-3,0)$，$C$ 的纵坐标从 $4$ 改为 $6$。训练目标：巩固"共线等距即中点"的识别与中点公式的运用。常见障碍（低）：看不出 $AB=BC$ 即中点。常见障碍（中）：不会同时利用 $x_B=0$ 求 $x_C$。

第 2 题

已知直线 $l$ 与 $x$ 轴交于点 $A(-4, 0)$，与 $y$ 轴的正半轴交于点 $B$，与反比例函数 $y=\dfrac{k}{x}\;(x>0)$ 的图象交于点 $C$，且 $AB = 2BC$。点 $C$ 的纵坐标为 $3$。

(1) 求直线 $l$ 的表达式；

(2) 过点 $C$ 作 $CD \parallel y$ 轴，交反比例函数 $y=\dfrac{k}{x}$ 的图象于点 $D$，求线段 $CD$ 的长度。

点击查看提示
  1. $AB = 2BC$ 且 $A$、$B$、$C$ 共线，说明 $B$ 分 $AC$ 的比为 $2:1$。试试分点公式：$x_B = \dfrac{2 \cdot x_C + 1 \cdot x_A}{2+1}$。
  2. 因为 $B$ 在 $y$ 轴上，所以 $x_B = 0$，代入上式即可求出 $x_C$。
  3. 第 (2) 问：$CD \parallel y$ 轴意味着 $D$ 与 $C$ 的横坐标相同，但 $D$ 在反比例函数的另一支上。
点击查看解答步骤

(1)

$A$、$B$、$C$ 共线且 $AB = 2BC$，所以 $B$ 分 $AC$ 的比为 $2:1$（$B$ 靠近 $C$）。

分点公式：$x_B = \dfrac{2 \cdot x_C + 1 \cdot x_A}{2+1} = \dfrac{2x_C + (-4)}{3}$。

因为 $B$ 在 $y$ 轴上，$x_B = 0$，所以 $2x_C - 4 = 0$，$x_C = 2$，即 $C(2, 3)$。

同样 $y_B = \dfrac{2 \cdot y_C + 1 \cdot y_A}{2+1} = \dfrac{2 \times 3 + 0}{3} = 2$，所以 $B(0, 2)$。

$C(2, 3)$ 在 $y = \dfrac{k}{x}$ 上，$k = 2 \times 3 = 6$。

直线 $l$ 过 $A(-4, 0)$ 和 $B(0, 2)$，斜率 $= \dfrac{2-0}{0-(-4)} = \dfrac{1}{2}$，所以直线 $l$ 的表达式为 $\boxed{y = \dfrac{1}{2}x + 2}$。

(2)

$CD \parallel y$ 轴，所以 $x_D = x_C = 2$。$D$ 在 $y = \dfrac{6}{x}$ 上，$y_D = \dfrac{6}{2} = 3$。

此时 $D(2, 3)$ 与 $C(2, 3)$ 重合，这是因为 $C$ 本身就在 $x=2$ 处与反比例函数只有一个交点。

本题的 $C$ 恰好是直线与反比例函数在第一象限的唯一交点。若考虑反比例函数在 $x>0$ 的定义域，$x_D = 2$ 时 $y_D = 3 = y_C$，因此 $\boxed{CD = 0}$。

教师备注：本题设计意在引导学生发现——$CD \parallel y$ 轴时，$D$ 与 $C$ 可能重合，这是一个有意义的讨论点。课堂上可追问：如果要使 $CD > 0$，$CD$ 应该平行于哪条轴？

教师备注：本题是中点关系的推广：从 $AB=BC$（中点）变为 $AB=2BC$（分点）。训练目标：将分点坐标公式（定比分点）作为中点公式的自然推广。常见障碍（中）：不熟悉分点公式，或混淆内分/外分。本题保持了简洁的计算，斜率为 $\frac{1}{2}$，方便心算。

第 3 题

已知直线 $l$ 与 $x$ 轴交于点 $A(-6, 0)$，与 $y$ 轴的正半轴交于点 $B$，与反比例函数 $y=\dfrac{k}{x}\;(x>0)$ 的图象交于点 $C$，且 $AB=BC$。点 $C$ 的纵坐标为 $2$。

(1) 求直线 $l$ 的表达式及反比例函数的表达式；

(2) 过点 $B$ 作 $BD \parallel x$ 轴，交反比例函数 $y=\dfrac{k}{x}$ 的图象于点 $D$。再过点 $D$ 作 $DE \parallel y$ 轴，交直线 $l$ 于点 $E$。求线段 $DE$ 的长度。

点击查看提示
  1. 第 (1) 问的核心方法与第 1 题完全相同——先识别中点关系。
  2. $B$ 在 $y$ 轴上，$x_B = 0$。用中点公式 $x_B = \dfrac{x_A + x_C}{2} = 0$ 求 $x_C$。
  3. 第 (2) 问分两步：先求 $D$（$BD \parallel x$ 轴 → $y_D = y_B$），再求 $E$（$DE \parallel y$ 轴 → $x_E = x_D$，且 $E$ 在直线 $l$ 上）。
点击查看解答步骤

(1)

$A$、$B$、$C$ 共线且 $AB=BC$，$B$ 是 $AC$ 的中点。

$x_B = \dfrac{-6 + x_C}{2} = 0$，解得 $x_C = 6$。

$y_B = \dfrac{0 + 2}{2} = 1$，所以 $B(0, 1)$，$C(6, 2)$。

$C(6, 2)$ 在 $y = \dfrac{k}{x}$ 上，$k = 6 \times 2 = 12$，所以反比例函数为 $\boxed{y = \dfrac{12}{x}}$。

直线 $l$ 过 $A(-6, 0)$ 和 $B(0, 1)$，斜率 $= \dfrac{1-0}{0-(-6)} = \dfrac{1}{6}$，所以直线 $l$ 的表达式为 $\boxed{y = \dfrac{1}{6}x + 1}$。

(2)

$BD \parallel x$ 轴，$y_D = y_B = 1$。$D$ 在 $y = \dfrac{12}{x}$ 上，$x_D = \dfrac{12}{1} = 12$，所以 $D(12, 1)$。

$DE \parallel y$ 轴，$x_E = x_D = 12$。$E$ 在直线 $l$ 上，$y_E = \dfrac{1}{6} \times 12 + 1 = 2 + 1 = 3$，所以 $E(12, 3)$。

$DE = |y_E - y_D| = |3 - 1| = \boxed{2}$。

教师备注：本题在第 1 题基础上增加了一次平行线构造（$DE \parallel y$ 轴），形成"两步坐标追踪"。训练目标：在多步平行构造中保持坐标的有序传递。$A(-6,0)$ 与 $C$ 的纵坐标 $2$ 的选取确保了 $x_C=6$，$D(12,1)$ 与 $E(12,3)$ 的纵坐标差恰好为整数 $2$，避免出现无理数。常见障碍（中）：在多步构造中混淆 $x$、$y$ 坐标。
