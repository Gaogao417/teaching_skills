自适应练习：一次函数值与不等式

教师判断

当前档位：E档（会做原题，但不会迁移）。

判断依据：初二学生已学一次函数和一元一次不等式，题目结构清晰、系数简单，预计能完成原题。但将系数改为负数（递减函数）后，不等号方向的处理是典型的迁移卡点，需要针对性训练。

练习策略：同源平级变式，保留"代入转化"核心结构，换系数正负，训练学生对增减性与不等式方向的敏感度。

练习说明

本组共 3 题，只练一个动作：把 y 的条件代回函数式子，变成方程或不等式，再求解。注意观察系数正负的变化对不等号方向的影响。

第 1 题

已知一次函数 \(y = -2x + 10\)。

(1) 当 \(x\) 取何值时，\(y = 0\)？

(2) 当 \(x\) 取何值时，\(y > 0\)？

(3) 当 \(x\) 取何值时，\(y < 0\)？

训练目标：系数为负数时，增减性改变，不等式方向与直觉相反。

预期卡点：第(3)问容易直接写 \(x < 5\)，实际应该是 \(x > 5\)。

提示一

先看系数：\(-2 < 0\)，所以这个函数是递减的。y 大了，x 反而小。

提示二

第(1)问解方程 \(-2x + 10 = 0\)，得 \(x = 5\)。因为递减，所以 \(y > 0\) 时 \(x < 5\)，\(y < 0\) 时 \(x > 5\)。

第 2 题

已知一次函数 \(y = \dfrac{1}{3}x - 4\)。

(1) 当 \(x\) 取何值时，\(y = -1\)？

(2) 当 \(x\) 取何值时，\(-2 < y < -1\)？

训练目标：复合不等式的拆分与交集，且上下界都是负数。

预期卡点：拆成 \(\dfrac{1}{3}x - 4 > -2\) 和 \(\dfrac{1}{3}x - 4 < -1\) 后，解的过程容易算错常数项的移项。

提示一

先解(1)得 \(y = -1\) 对应 \(x = 9\)，这是上界分界点。再求 \(y = -2\) 对应的 \(x\)，作为下界分界点。

提示二

\(y = -2\) 代入：\(\dfrac{1}{3}x - 4 = -2\)，得 \(\dfrac{1}{3}x = 2\)，即 \(x = 6\)。系数 \(\frac{1}{3} > 0\) 递增，所以 \(-2 < y < -1\) 对应 \(6 < x < 9\)。

第 3 题

已知一次函数 \(y = -\dfrac{1}{2}x + 5\)。

(1) 当 \(x\) 取何值时，\(1 < y < 3\)？

(2) 当 \(x\) 取何值时，\(y \leq 1\)？

训练目标：负系数 + 复合不等式 + 含"等于"的不等式，综合迁移。

预期卡点：负系数递减，\(y\) 从 1 到 3 对应的 \(x\) 范围容易写反；\(\leq\) 在递减函数下对应的边界判断。

提示一

系数 \(-\frac{1}{2} < 0\)，函数递减。先求 \(y = 1\) 对应 \(x = 8\)，\(y = 3\) 对应 \(x = 4\)。

提示二

递减时，\(y\) 大对应 \(x\) 小：\(1 < y < 3\) 对应 \(4 < x < 8\)。\(\;y \leq 1\) 时 \(x \geq 8\)。

参考答案与升级/降级判断

第 1 题

  1. \(-2x + 10 = 0 \Longrightarrow -2x = -10 \Longrightarrow x = 5\)。
  2. \(-2x + 10 > 0 \Longrightarrow -2x > -10\)。两边除以 \(-2\)（负数），不等号翻转：\(x < 5\)。
  3. \(-2x + 10 < 0 \Longrightarrow -2x < -10\)。两边除以 \(-2\)，不等号翻转：\(x > 5\)。

升级/降级判断：若学生三问全对，保持 E 档进入第 2 题。若第(2)(3)问不等号方向写反（写成 \(x>5\)、\(x<5\)），降级至 D 档——需加强"负系数时不等号翻转"的意识。

第 2 题

  1. \(\dfrac{1}{3}x - 4 = -1 \Longrightarrow \dfrac{1}{3}x = 3 \Longrightarrow x = 9\)。
  2. 复合不等式 \(0 < \dfrac{1}{3}x - 4 < -1\)（注：应为 \(-2 < \dfrac{1}{3}x - 4 < -1\)）
    左半：\(\dfrac{1}{3}x - 4 > -2 \Longrightarrow \dfrac{1}{3}x > 2 \Longrightarrow x > 6\)
    右半：\(\dfrac{1}{3}x - 4 < -1 \Longrightarrow \dfrac{1}{3}x < 3 \Longrightarrow x < 9\)
    取交集：\(6 < x < 9\)。

升级/降级判断：若学生正确拆分并取交集得 \(6 < x < 9\)，保持 E 档进入第 3 题。若只解了一边或交集取错（写成并集），降级至 D 档。

第 3 题

  1. \(-\dfrac{1}{2}x + 5 = 1 \Longrightarrow -\dfrac{1}{2}x = -4 \Longrightarrow x = 8\)；
    \(-\dfrac{1}{2}x + 5 = 3 \Longrightarrow -\dfrac{1}{2}x = -2 \Longrightarrow x = 4\)。
    系数为负，递减函数，\(y\) 从 1 到 3 对应 \(x\) 从 8 到 4：\(4 < x < 8\)。
  2. \(y \leq 1 \Longrightarrow -\dfrac{1}{2}x + 5 \leq 1 \Longrightarrow -\dfrac{1}{2}x \leq -4 \Longrightarrow x \geq 8\)。

升级/降级判断：若学生三题全对，升级至 F 档——已掌握，下次可以做结构隐藏型变式（如函数式含参）。若第(2)问写反（写成 \(x \leq 8\)），保持 E 档，需要再练负系数含等号的情况。

综合判断：

  • 三题全对且过程规范 → F档
  • 复合不等式或负系数出错 → D档或E档（视具体错误）
  • 基础代入都不会 → C档（回到讲解页重新理解"把 y 换成条件数"）
