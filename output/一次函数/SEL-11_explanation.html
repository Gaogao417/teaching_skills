学生版讲解：一次函数与反比例函数交点——中点关系

一、这题先看懂什么

题目给出了几个关键信息，我们先把它们"翻译"成坐标：

题目信息 坐标含义
直线 AB 与 $x$ 轴交于 $A(-2, 0)$ $A$ 的坐标已经告诉你了：$A(-2, 0)$
与 $y$ 轴正半轴交于 $B$ $B$ 在 $y$ 轴上，所以 $x_B = 0$，$y_B > 0$
与 $y = \dfrac{m}{x}\ (x>0)$ 交于 $C$，$y_C = 4$ $C$ 的纵坐标已知为 4，横坐标未知
$AB = BC$ 线段 AB 和 BC 长度相等（关键条件！）
想一想：$AB = BC$ 只告诉你"长度相等"，但题目还有一个隐藏条件——$A$、$B$、$C$ 都在同一条直线上。这个"共线 + 等距"意味着什么？

提示：如果 $A$、$B$、$C$ 在同一条线上，且 $AB = BC$，那么 $B$ 就是 $A$ 和 $C$ 的正中间——$B$ 是 $AC$ 的中点。

二、第一步该做什么

很多同学看到这道题，第一反应是"设直线方程，列方程组求解"。但这样做计算量很大，容易出错。

更好的第一步是：

核心动作：识别"共线等距 → 中点"关系

$A$、$B$、$C$ 都在直线 $AB$ 上（共线），又已知 $AB = BC$（等距），所以 $B$ 是 $AC$ 的中点。直接用中点坐标公式求坐标，不需要列距离方程。

中点坐标公式：如果 $M$ 是 $P(x_1, y_1)$ 和 $Q(x_2, y_2)$ 的中点，那么

$M\left(\dfrac{x_1 + x_2}{2},\ \dfrac{y_1 + y_2}{2}\right)$

三、关键转化怎么想

转化 1：$AB = BC$ → $B$ 是 $AC$ 中点

这是整道题最关键的一步。我们已知：

  • $A(-2, 0)$
  • $B(0, y_B)$（$y_B > 0$）
  • $C(x_C, 4)$

$B$ 是 $AC$ 中点，所以：

$x_B = \dfrac{x_A + x_C}{2}$，即 $0 = \dfrac{-2 + x_C}{2}$ → $x_C = 2$

$y_B = \dfrac{y_A + y_C}{2}$，即 $y_B = \dfrac{0 + 4}{2} = 2$

一步就得到了两个关键坐标：$B(0, 2)$、$C(2, 4)$。

想一想：如果不利用中点关系，而是用距离公式 $\sqrt{(x_C+2)^2+(4-0)^2}=\sqrt{(0-x_C)^2+(y_B-4)^2}$ 来做，你觉得计算会怎样？

会变成一个含两个未知数的方程，还得和其他条件联立，非常繁琐。中点公式一步搞定，这就是"看对关系"的价值。

转化 2：$C$ 在反比例函数上 → 求 $m$

反比例函数 $y = \dfrac{m}{x}$ 上的点，坐标满足 $xy = m$。所以：

$m = x_C \times y_C = 2 \times 4 = 8$，即反比例函数为 $y = \dfrac{8}{x}$

转化 3：$A$、$B$ 确定直线 → 待定系数法

两个点确定一条直线。已知 $A(-2, 0)$ 和 $B(0, 2)$：

斜率 $k = \dfrac{2 - 0}{0 - (-2)} = 1$

直线方程 $y = x + 2$

验证：$C(2, 4)$ 代入 $y = x + 2$ → $4 = 2 + 2$ ✓

四、标准解法

第 (1) 问：求直线 AB 的表达式

  1. 识别中点关系：$AB = BC$，$A$、$B$、$C$ 共线 → $B$ 是 $AC$ 中点
  2. 用中点公式求坐标：

    $x_B = \dfrac{-2 + x_C}{2} = 0$ → $x_C = 2$

    $y_B = \dfrac{0 + 4}{2} = 2$

    所以 $B(0, 2)$，$C(2, 4)$

  3. 待定系数法求直线：设 $y = kx + b$，代入 $A(-2, 0)$ 和 $B(0, 2)$：

    $0 = -2k + b$，$2 = 0 + b$ → $b = 2$，$k = 1$

    $y = x + 2$

第 (2) 问：求 CD 的长度

  1. 理解"BD // $x$ 轴"：平行于 $x$ 轴的线上，所有点的纵坐标相同。所以 $y_D = y_B = 2$
  2. 求 D 的坐标：$D$ 在 $y = \dfrac{8}{x}$ 上，$y_D = 2$，所以 $x_D = \dfrac{8}{2} = 4$，即 $D(4, 2)$
  3. 用距离公式求 CD：

    $C(2, 4)$，$D(4, 2)$

    $CD = \sqrt{(4 - 2)^2 + (2 - 4)^2} = \sqrt{4 + 4} = \sqrt{8} = 2\sqrt{2}$

    $CD = 2\sqrt{2}$

五、边讲边问

问 1：$AB = BC$ 能说明 $B$ 是 $AC$ 中点吗？什么情况下不能？

问 2：如果题目改成 $AB : BC = 1 : 2$（共线），你能用类似的方法求 $B$ 的坐标吗？

问 3：反比例函数 $y = \dfrac{m}{x}$ 上的点，横纵坐标的积是多少？为什么？

问 4："BD // $x$ 轴"这个条件，在坐标系里意味着 D 和 B 的什么坐标相同？

问 5：如果第 (2) 问把 BD 改成 BD // $y$ 轴，D 的坐标会变成什么？CD 还是一样的吗？

六、易错提醒

易错 1：看不出 AB = BC 就是中点条件

有些同学看到 $AB = BC$ 就想用距离公式。记住：如果 A、B、C 在同一条直线上，$AB = BC$ 就意味着 B 是中点，直接用中点公式，快得多。

易错 2：BD // $x$ 轴时设错 D 的坐标

平行于 $x$ 轴 → 纵坐标相同（$y$ 不变），横坐标变化。平行于 $y$ 轴 → 横坐标相同（$x$ 不变），纵坐标变化。不要搞反了。

易错 3：距离公式中坐标相减的顺序和符号

$CD = \sqrt{(x_D - x_C)^2 + (y_D - y_C)^2}$，顺序不影响结果（因为平方了），但要小心不要漏掉负号导致算错。养成习惯：用"终点减起点"。

七、一句话总结

共线等距找中点，中点公式求坐标；平行坐标轴，对应坐标不变；距离公式算长度。
