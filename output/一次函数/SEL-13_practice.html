自适应练习：一次函数最值——购机利润问题

教师判断

当前档位：D档（会列式，但可能在翻译约束条件时遗漏不等式、端点取反、或解完不验证范围）。

判断依据：学生为初二水平，能解一次函数方程，但综合建模能力需要提升。基于结构分析，中等学生最易在"漏翻约束为不等式"和"递减函数端点方向搞反"上出错。本次练习定位在巩固"列函数+翻约束定范围+看k取端点"三个核心动作上，情境迁移但结构不变。

练习说明

本组共3题，训练三个动作：

  1. 从新情境中列函数并化简——总利润 = 各部分之和，化成 y = kx + b 的形式。
  2. 把文字限制翻译为不等式——确定 x 的取值范围。
  3. 看 k 的符号，在端点处取最值——递减取左端，递增取右端。

建议：每题完成后对照参考答案，记录卡点，判断是否需要降级或升级。

第1题 巩固建模

某工厂生产甲、乙两种零件，生产一个甲零件获利30元，乙零件获利50元。计划一次生产两种零件共200个，乙零件不超过甲零件的4倍。设生产甲零件 x 个，总利润为 y 元。
(1) 求 y 与 x 的关系式；
(2) 生产甲、乙各多少个利润最大？

训练目标：从新情境中正确列出函数关系式，化简为 y = kx + b 的形式，并将"乙不超过甲的4倍"翻译为不等式，求 x 的范围。

预期卡点：不等式写成 200 − x ≤ 4x，但解错（应该得到 x ≥ 40）；或者直接用 0 ≤ x ≤ 200 而忽略约束条件。

提示一

乙零件数量 = 200 − x。"乙不超过甲的4倍"这句话中，乙对应什么式子？甲对应什么式子？"不超过"用什么符号？

提示二

y = 30x + 50(200 − x) = 30x + 10000 − 50x = −20x + 10000。不等式：200 − x ≤ 4x → 200 ≤ 5x → x ≥ 40。范围：40 ≤ x ≤ 200。k = −20 < 0，递减，最大值在 x = 40，y = −800 + 10000 = 9200 元。甲40个，乙160个。

第2题 强化定域

某商店购进A、B两种商品，A每件获利80元，B每件获利100元。共购进50件，要求A的件数不少于B的件数。设购进A x 件，总利润为 y 元。
(1) 求 y 与 x 的关系式，并写出 x 的取值范围；
(2) 求利润的最大值和最小值。

训练目标：本题 k > 0（函数递增），与原题 k < 0 相反。训练学生"看 k 符号决定取哪个端点"的能力，同时训练"不少于"这类约束的翻译。

预期卡点："A不少于B"翻译为 x ≥ 50 − x → x ≥ 25，但可能写成 x ≤ 25；递增函数最大值取错端点（取了 x = 25 而非 x = 50）。

提示一

"A不少于B"意思是 A ≥ B，即 x ≥ 50 − x。化简后 x 的范围是什么？另外别忘了 x ≥ 0 和 50 − x ≥ 0。

提示二

y = 80x + 100(50 − x) = 80x + 5000 − 100x = −20x + 5000。k = −20 < 0，递减！最大值在最小 x 处：x = 25 时 y = −500 + 5000 = 4500 元。最小值在最大 x 处：x = 50 时 y = −1000 + 5000 = 4000 元。注意：虽然B利润更高，但 k < 0 说明整体还是递减的。

第3题 轻度迁移

某校组织春游，租用大客车和中巴车共8辆。大客车每辆可载40人，租金800元；中巴车每辆可载20人，租金500元。要求中巴车不超过大客车的2倍，且参加春游的师生共240人。
设租用大客车 x 辆，总租金为 y 元。
(1) 根据师生人数求 x 的值，并验证是否满足车辆约束；
(2) 如果师生人数改为230人，能否恰好租满8辆车？若能，求 y；若不能，说明理由并求 y 的范围。

训练目标：在原结构基础上增加一个"恰好满足"的约束（等式约束），训练"先解方程再验证不等式"的综合能力。第(2)问训练"判断可达性+求值域"的复合动作。

预期卡点：第(1)问只列车的数量关系 40x + 20(8−x) = 240，解出 x = 4，但忘记验证"中巴不超过大客车2倍"（4 ≤ 2×4 = 8，满足）；第(2)问解出 x 不是整数后不知如何处理。

提示一

第(1)问：总人数 = 大客车载人 + 中巴车载人 = 40x + 20(8−x)，令其等于240，解出 x。然后验证：中巴车数 = 8−x，大客车数 = x，检查 8−x ≤ 2x 是否成立。第(2)问：令 40x + 20(8−x) = 230，看看解出的 x 是否在有效范围内。

提示二

(1) 40x + 20(8−x) = 240 → 40x + 160 − 20x = 240 → 20x = 80 → x = 4。中巴 = 4辆，大客车 = 4辆。验证：4 ≤ 2×4 = 8，满足。y = 800×4 + 500×4 = 3200 + 2000 = 5200 元。
(2) 40x + 20(8−x) = 230 → 20x + 160 = 230 → 20x = 70 → x = 3.5。但车辆数必须为整数，所以不能恰好租满。y = 800x + 500(8−x) = 300x + 4000。约束：8−x ≤ 2x → x ≥ 3（取整后x≥3），且 x ≤ 8。x 只能取 3,4,5,6,7,8。当 x=3 时 y=4900，x=8 时 y=6400。但人数限制：40x+20(8−x)=20x+160=230 → x=3.5，说明实际需要3.5辆大客车。若取x=3，人数=220<230不够；取x=4，人数=240>230超了。所以230人无法恰好用整数辆满足，y 范围为 4900 ≤ y ≤ 6400（x=3到x=8）。

参考答案与升级/降级判断

第1题

答案：

(1) y = 30x + 50(200 − x) = 30x + 10000 − 50x = −20x + 10000
约束：200 − x ≤ 4x → x ≥ 40，且 0 ≤ x ≤ 200，综合 40 ≤ x ≤ 200
(2) k = −20 < 0，递减，最大值在 x = 40
ymax = −20 × 40 + 10000 = 9200 元
甲 40 个，乙 160 个
判断：
· 函数、范围、最值全部正确 → 核心动作掌握，可继续；
· 函数正确但范围写成 0 ≤ x ≤ 200（漏约束）→ 降级：回到讲解页"转化2"部分，重点练习"文字→不等式"的翻译；
· 范围正确但最值取了 x = 200（端点方向反了）→ 提醒递减函数"最小x处取最大值"，不降级但记录。

第2题

答案：

(1) y = 80x + 100(50 − x) = 80x + 5000 − 100x = −20x + 5000
约束：x ≥ 50 − x → x ≥ 25，且 0 ≤ x ≤ 50，综合 25 ≤ x ≤ 50
(2) k = −20 < 0，递减
最大值：x = 25 时 y = −20×25 + 5000 = 4500 元
最小值：x = 50 时 y = −20×50 + 5000 = 4000 元
判断：
· 三问全部正确 → "看k取端点"动作稳固，可升级；
· "不少于"翻译错误（写成 x ≤ 25）→ 退回D档，做约束翻译专项训练；
· 最值端点取反（最大值取了x=50）→ 退回D档，需要反复练习"k<0递减，最大值在左端点"。

第3题

答案：

(1) 40x + 20(8−x) = 240 → 20x = 80 → x = 4
验证约束：中巴 = 8−4 = 4，大客车 = 4，4 ≤ 2×4 = 8 → 满足
y = 800×4 + 500×4 = 5200 元
(2) 40x + 20(8−x) = 230 → 20x = 70 → x = 3.5
x = 3.5 不是整数，不能恰好租满
y = 800x + 500(8−x) = 300x + 4000
约束：x ≥ 3（取整），x ≤ 8
x = 3 时 y = 4900（最小），x = 8 时 y = 6400（最大）
但 x=3 时只能载 220 人 < 230 人，不满足人数需求
x=4 时载 240 人 > 230 人，满足
所以实际上 x ∈ {4, 5, 6, 7, 8}，y ∈ {5200, 5500, 5800, 6100, 6400}
判断：
· 第(1)问正确解出并验证 → 方程+不等式复合能力具备；
· 第(2)问判断"不能"并给出合理分析 → 掌握，可进入E档（同源平级变式，换情境如生产配料、运输调度等）；
· 第(2)问不会处理非整数解 → 退回D档，需要做"整数解"的专项训练；
· 第(1)问忘记验证约束 → 退回D档，强化"解完回头看"的习惯。
