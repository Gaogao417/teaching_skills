\begin{eduReadTipBox}
已知一次函数 $y=kx+b$ 的图像经过点 $(2,5)$ 和点 $(-1,-1)$，求 $k$ 和 $b$ 的值。\end{eduReadTipBox}





\begin{eduRouteFlow}

  \node[edu route node] (route0) {已知：两个点的坐标 $(2,5)$ 和 $(-1,-1)$};
  \node[edu route node, right=10mm of route0] (route1) {要求：求 $k$ 和 $b$};
  \node[edu route node, right=10mm of route1] (route2) {关键词：一次函数、经过点、待定系数法};
  \node[edu route node, right=10mm of route2] (route3) {忽略：无};
  \draw[edu route arrow] (route0.east) -- (route1.west);
  \draw[edu route arrow] (route1.east) -- (route2.west);
  \draw[edu route arrow] (route2.east) -- (route3.west);
\end{tikzpicture}
\end{eduRouteFlow}





\begin{eduRouteFlow}
\begin{tikzpicture}
  \node[edu route node] (route0) {将两个点代入 $y=kx+b$，列方程组};
  \node[edu route node, right=10mm of route0] (route1) {解方程组，求出 $k$ 和 $b$};
  \node[edu route node, right=10mm of route1] (route2) {写出最终解析式并检验};
  \draw[edu route arrow] (route0.east) -- (route1.west);
  \draw[edu route arrow] (route1.east) -- (route2.west);
\end{tikzpicture}
\end{eduRouteFlow}





\begin{eduReadTipBox}
一次函数有两个未知系数 $k$ 和 $b$，两个独立条件可以确定它们。将点的坐标代入，得到关于 $k, b$ 的二元一次方程组。\end{eduReadTipBox}





\begin{eduSoftBox}
\eduSubTitle{代入列方程}将 $(2,5)$ 代入：$5 = 2k + b$。将 $(-1,-1)$ 代入：$-1 = -k + b$。\end{eduSoftBox}




\begin{eduSoftBox}
\eduSubTitle{解方程组}两式相减：$6 = 3k$，所以 $k = 2$。代回：$5 = 4 + b$，所以 $b = 1$。\par{\small\color{edu-muted}待定系数法的核心是：用已知条件确定未知系数。}\end{eduSoftBox}




\begin{eduSoftBox}
\eduSubTitle{写出答案}$k = 2, b = 1$，即 $y = 2x + 1$。    \par\hspace{5mm}{\small — 检验：$(2, 2\times2+1) = (2,5)$ ✓}
    \par\hspace{5mm}{\small — 检验：$(-1, 2\times(-1)+1) = (-1,-1)$ ✓}
\end{eduSoftBox}





\begin{eduMistakeBox}
\eduSubTitle{代入时符号出错}将 $(-1,-1)$ 代入 $y=kx+b$ 时，容易写成 $-1 = k + b$（漏掉负号）。正确应为 $-1 = -k + b$。\end{eduMistakeBox}




\begin{eduMistakeBox}
\eduSubTitle{忘记检验}求出 $k, b$ 后应代回原点检验，避免计算错误。\end{eduMistakeBox}





\begin{eduReadTipBox}
  \textbf{提示 1：}
如果有三个点都满足 $y=2x+1$，还需要全部代入吗？\end{eduReadTipBox}




\begin{eduReadTipBox}
  \textbf{提示 1：}
如果只给你一个点，能唯一确定 $k$ 和 $b$ 吗？\end{eduReadTipBox}





\begin{eduReadTipBox}
待定系数法 = 代入已知点 → 列方程组 → 解出系数 → 写出解析式。\end{eduReadTipBox}
