# 结构分析：一次函数过两点求解析式与坐标轴面积

## 原题

已知一次函数 y = kx + b 的图像经过点 A(1, 3) 和点 B(−2, −6)。

（1）求这个一次函数的解析式；
（2）判断点 C(3, 9) 是否在这个函数的图像上；
（3）求该函数图像与 x 轴、y 轴的交点坐标，并计算图像与坐标轴围成的三角形面积。

## 一、题目场景

- 数学对象：一次函数 y = kx + b 的图像（一条直线）
- 变量/参数：k（斜率）、b（截距）
- 函数/图形：直线 y = 3x，过原点
- 已知条件：直线经过 A(1, 3) 和 B(−2, −6)
- 要求目标：解析式、点是否在直线上、与坐标轴交点及围成面积

## 二、核心结构

- 表面考点：待定系数法求一次函数解析式、代入检验、求交点坐标
- 本质考点：二元一次方程组的建立与求解；**退化情形识别**——b = 0 时直线过原点，与两轴交点重合，不围成三角形
- 一句话问题模式：给出两点确定直线参数，再利用解析式做代入检验和交点计算，关键在于识别正比例函数的退化面积

## 三、关键转化

- 最关键的转化：将"图像经过点"转化为方程组——每个点给出一个关于 k、b 的方程
- 为什么降低计算量：两个方程直接相减消 b，一步得 k，再回代得 b，无需复杂消元
- 不转化时的低效路径：用两点式斜率公式求 k 再代入，多一步且容易符号出错

## 四、标准路径骨架

1. 先做什么：将 A(1, 3) 代入 y = kx + b 得 k + b = 3（方程①）
2. 再做什么：将 B(−2, −6) 代入得 −2k + b = −6（方程②）
3. 建立什么关系：①−② 得 3k = 9
4. 如何求解：k = 3，回代得 b = 0，解析式 y = 3x
5. 需要检查什么：第（3）问中 b = 0 时两个交点重合于原点，面积为 0，不能套用 S = ½|a|·|b| 公式

## 四点五、标准完整解与验算

- 关键交点/关键量：k = 3, b = 0, 与 x 轴交点 (0, 0), 与 y 轴交点 (0, 0)
- 面积/方程/关系式：
  - 方程组：{ k + b = 3; −2k + b = −6 }
  - 面积：交点重合于原点，面积为 0
- 完整求解过程：
  - （1）代入两点建立方程组，消元求解得 y = 3x
  - （2）将 x = 3 代入 y = 3x 得 y = 9 = 9，点 C 在图像上
  - （3）令 y = 0 得 x = 0（x 轴交点 (0,0)）；令 x = 0 得 y = 0（y 轴交点 (0,0)）。两交点重合，围不成三角形，面积为 0
- 最终答案：
  - （1）y = 3x
  - （2）点 C(3, 9) 在函数图像上
  - （3）与两坐标轴交点均为 (0, 0)，面积为 0
- 排除值：无（本题无分母、无根号约束）
- 退化情形：**b = 0 是本题核心退化**，直线过原点，正比例函数
- 验算：k = 3 时 A: 3×1 = 3 ✓; B: 3×(−2) = −6 ✓; C: 3×3 = 9 ✓
- 本题最短可靠路径：代入两点 → 方程组相减消 b → 得 k = 3, b = 0 → 识别 b = 0 即过原点

## 五、出题人逻辑

- 诱导学生硬算的位置：第（3）问诱导学生机械套用 S = ½|x₀|·|y₀|，忽略两交点重合的退化情形
- 真正的捷径：求出 b = 0 后立刻识别"正比例函数过原点"，直接判断面积为 0
- 训练的可迁移能力：待定系数法 + 退化情形意识 + "公式适用条件"的检验习惯

## 六、学生卡点预测

- 基础薄弱学生（L0-L1）：不理解"图像经过点"意味着坐标满足解析式，不会将条件转化为方程
- 中等学生（L2-L3）：能求出解析式，但第（3）问机械套面积公式 S = ½|a|·|b| 得出错误答案（如 S = ½×0×0 = 0 虽然结果碰巧对，但思维是错的；或混淆公式写成非零值）
- 较强学生（L4-L5）：基本无卡点，但可能不主动思考退化情形的一般性（什么条件下面积才非零）

## 七、变式原则

- 核心不变量：待定系数法求一次函数解析式（两点 → 方程组 → 参数）
- 表层特征：点的坐标数值、问法排列
- 可变维度：
  1. 点的坐标数值（整数/分数/含参数）
  2. 所求目标（求解析式 / 判断点 / 求面积 / 求交点距离）
  3. b 是否为 0（是否出现退化）
  4. 表征方式（解析式 / 图像 / 表格 / 文字描述）
  5. 条件包装（"过原点"直接给出 vs 通过计算发现）
- 深化阶梯：
  1. 同结构换数：改为过 A(2, 5) 和 B(−1, −1)，b ≠ 0，面积非零
  2. 换问法：已知 y = 2x + 1 与坐标轴围成三角形，求面积
  3. 换表征：给图像读截距，再求解析式
  4. 条件包装：已知直线将某矩形分成面积为 2:1 的两部分，求解析式
  5. 反向构造：已知直线与坐标轴围成面积为 6 的三角形，且过点 (1, 2)，求解析式
- 允许的变换：改数字、改所求、增加/减少小问、换表征
- 禁止的变换：引入二次函数或反比例函数、引入不等式组、需要相似三角形知识
- 表征切换：解析式 ↔ 图像读图 ↔ 两点坐标 ↔ 应用题情境
- 包装方式：几何面积题、实际应用（路程-时间、价格-数量）、图像判断题
- 近迁移例子：
  - 过 A(0, 2) 和 B(3, 0) 求解析式及面积（b ≠ 0，标准情形）
  - 过 A(1, 4) 和 B(3, 10) 求 k 和 b
- 远迁移例子：
  - 已知直线 y = kx + b 与 x 轴交于 (a, 0)，与 y 轴交于 (0, b)，面积 S = ½|ab|，若 S = 12 且直线过 (2, 3)，求解析式
- 反例/伪变式：
  - 给出三个点但其中两个在同一条直线上，第三个不在——考查"共线"判断
  - 给出"过原点"和一个点——虽然也是两点确定直线，但直接可判断 b = 0

## 八、计算复杂度预算

- 原题计算层级：二元一次方程组消元（1步消元 + 1步回代），整数运算
- 允许小步上升到：含分数坐标的方程组求解（如过 (½, 1) 和 (−1, −2)）
- 禁止引入的计算负担：含字母参数的讨论、联立两个一次函数求交点、绝对值不等式
- 必须保留的可见支架：待定系数法"代入→列方程→消元→回代"四步

## 九、推荐讲题任务包

- 适合的学习层级：L1-L4
- 本题讲解目标：掌握待定系数法，建立"坐标满足解析式 = 方程"的转化意识，识别 b = 0 的退化情形
- 不要直接讲的抽象话："数形结合"、"待定系数法就是解方程组"
- 必须先问的问题：
  - "图像经过一个点"意味着这个点的坐标满足什么？
  - 一次函数有两个参数，至少需要几个点才能确定？
  - 求出 b = 0 后，这条直线有什么特殊之处？
- 关键讲解顺序：
  1. 用"经过点 = 坐标满足解析式"代入两点
  2. 方程组相减消 b（不是代入消元）
  3. 解出后先观察 b = 0 的含义
  4. 第（3）问画图验证"两个交点重合"
- 最适合的具体数值例子：y = 2x + 4（不过原点，与两轴交于不同点，面积 = ½×2×4 = 4，作为对比）
- 讲到哪里停下来让学生回答：代入两点列方程组后，让学生自己选择消元方法

## 十、推荐练题任务包

- 若学生在 L0-L1，出什么题：
  - 给出 y = 2x + 1 和一个点 (1, 3)，判断是否在图像上（训练"代入检验"）
  - 给出直线过 (0, 3) 和 (1, 5)，求 k 和 b（训练代入列方程）
- 若学生在 L2，出什么题：
  - 标准待定系数法：过两个整数点求解析式，b ≠ 0
  - 过原点和另一个点求解析式（训练 b = 0 的识别）
- 若学生在 L3，出什么题：
  - 含退化情形的面积计算（本题本身）
  - 换问法：已知解析式和面积反求参数
- 若学生在 L4，如何迁移：
  - 表征切换：给图像读截距再求解析式
  - 条件包装：应用题情境中的一次函数
- 若学生达到 L5-L6，如何深化/抽象/包装：
  - 反向构造：已知面积和过某点，求解析式（可能有两组解）
  - 含参数讨论：过 (a, 2a) 和 (−1, 3) 的直线何时 b = 0
- 禁止出的跑偏变式：引入二次函数、需要不等式组求解、涉及三角函数
