## 训练目标

本专题是整月训练的"第 1 周地基"，目标不是做出多复杂的截面，而是把后续平行/垂直章节反复依赖的两个底层习惯先立住：

1. **把公理变成画图动作**：会从"两个公共点定交线"直接落到笔上的连线动作，而不是停在文字默写。
2. **截面边必须落在几何体表面上**：学生能说清"截面的每一条边都是截面与某个表面（面/棱所在面）的交线"，不再把截面画到几何体外面。
3. **会用延長线找隐藏交点**：在某个表面所在的平面内，延长两条已知线找到落在这个面上的交点，再倒推交线。
4. **共面证明有规范路径**：知道两条相交直线共面、两条平行直线共面是"定面"的两条主路，证明两线共面时优先往这两条上靠，而不是凭感觉。
5. **几何语言过关**：交线、共面、异面的符号书写（$\alpha \cap \beta = l$、$a \cap b = P$、$a, b$ 共面于 $\alpha$ 等）格式正确，为课时 4 起的平行证明铺路。

> 与总览"四层过关标准"对应：本专题主攻**第三层（作图层）**和**第四层（论证层）**的前半段；第一、二层（文字、符号）由每天默写支撑。

## 知识脉络与定理关系

> 这一节回答两个问题：本模块**到底教哪几条定理**，以及这些定理**彼此是什么关系**——不是把定理摊成一张平铺清单，而是把它们的因果链条讲清。讲课时按这条脉络推进，学生卡壳时也能定位到"卡在哪一环"。

### 本模块的定理与公理（及其关系对象）

| 定理/公理 | 内容（符号） | 关系对象 / 在脉络里的位置 |
|---|---|---|
| **基本事实1（三点定面）** | 过不在一条直线上的三点，有且只有一个平面。 | 关系：3 个不共线点 → 唯一平面。**这是"定面"的根基公理**，推论①②③都由它导出。 |
| **基本事实2（线在面内）** | $A \in l,\ B \in l,\ A \in \alpha,\ B \in \alpha \Rightarrow l \subset \alpha$。 | 关系：点∈线 + 点∈面 ⇒ 线⊂面。**这是"从属关系"的桥梁**——截面作图里"截面边必须落在表面上"就是反复用这条。 |
| **基本事实3（两面交线）** | $P \in \alpha,\ P \in \beta \Rightarrow \alpha \cap \beta = l$（且 $P \in l$）。 | 关系：公共点 ⇒ 交线。**这是截面作图的核心定理**——找两个公共点就定了整条交线。 |
| **推论①** | 一条直线 + 直线外一点确定一个平面。 | 关系：由基本事实1+2导出。"线+点定面"的主路。 |
| **推论②** | 两条相交直线确定一个平面。 | 关系：由基本事实1+2导出。"两相交线定面"的主路——共面证明优先往这条上靠。 |
| **推论③** | 两条平行直线确定一个平面。 | 关系：由基本事实1+2导出。"两平行线定面"的主路——共面证明的第二条主路。 |

### 定理关系图（文字版）

```
                         [基本事实1：三点定面]   ←── 最根本的定面公理
                                  │
                          导出三条实用化主路
                  ┌───────────────┼───────────────┐
              [推论①]          [推论②]          [推论③]
             线+点定面       两相交线定面      两平行线定面
                  │               │               │
                  └─────── 共面证明的"定面"出口 ───────┘
                                  │（对立面）
                          [异面判定：不共面即异面]

  [基本事实2：两点∈线∈面 ⇒ 线⊂面]   ←── 从属关系的桥梁
              （判断"一条线是否落在某个面上"的唯一判据）
                                  │
                                  ▼ 反复使用
  [基本事实3：两面有一个公共点 ⇒ 一条过该点的交线]   ←── 截面作图的核心
              （找两个公共点 ⇒ 定整条交线）
```

### 知识脉络的四条逻辑链（讲解时按此推进）

1. **从属关系链**（基本事实2 承担）：建立"点→线→面"的从属传递——两点在线上、两点又在面上，则整条线在面上。这是判断"一条线是否落在某个面上"的**唯一判据**。截面作图里反复出现的"截面边必须落在表面上"，每一步都是在用基本事实2 落实"这条边属于这个面"。

2. **定面链**（基本事实1 + 推论①②③）：基本事实1（三点定面）是最根本的定面公理；推论①②③是它的实用化（线+点、两相交线、两平行线都能定面）。**共面证明的本质就是"找到一个能把这两条线都装进去的平面"**——优先用推论②（相交）或推论③（平行）定面，这就是"相交/平行两直线共面"两条主路的来源。

3. **交线链**（基本事实3 承担）：基本事实3 说"两面有一个公共点就有一条过该点的交线"。截面作图的核心动作"**找两个公共点定交线**"，就是把基本事实3 用两次：先找一个公共点（知道交线必过它），再在某表面所在平面内延长线找到第二个公共点（确定整条交线）。没有第二个公共点时，就回到从属关系链，用基本事实2 说明边落表面。

4. **异面判定链**（与共面对立）：不共面即异面。判定两条直线异面：找一个平面装下其中一条线和另一条线上的两个点，若第三个点不在这个平面内、且不在这条线上，则两线异面——本质上是"定面链的反向应用"。

> 这四条链不是平行的：**定面链（证明线在面里）** 与 **交线链（找两面的公共线）** 共同支撑截面作图；**从属关系链** 是两者的公共工具；**异面判定** 是定面链的对立面，用来排除。讲解时把这条脉络点明，学生才不会把"三点定面""两点定交线""相交平行共面"当成互不相干的三句口诀。

## 讲解方法

本专题讲解统一围绕四句话展开，板书时把这四句当作"操作口令"逐条落在一道例题上，学生卡壳时就指其中一句追问：

- **两个公共点定交线**：两个平面相交，交线由两个公共点唯一确定。讲截面第一步永远是"先找截平面与几何体表面的公共点"。
- **同一平面内延长找交点**：当两个公共点不直接可见时，在某个表面所在的平面内，延长两条已有线找到它们的交点，这个交点一定落在该表面上，可作为定交线的第二个公共点。
- **截面边落在表面上**：截面与几何体表面的交线就是截面的边。每画一条边都要追问"这条边落在哪个表面上"，落在空中的线一律判错。
- **相交/平行两直线共面**：两条相交直线共面、两条平行直线共面。证明两线共面时，要么说明它们相交、要么说明它们平行，再由推论定面；这是共面证明的两条主路。

每道讲解例题统一按总览第四节的三步板书推进：**判对象（这是截面作图还是共面证明）→ 转化目标（找几个公共点 / 往相交还是平行上靠）→ 书写格式（边落表面、符号完整）**。

## 图形设计说明

本轮只做设计说明，不画正式图（不写 TikZ、不渲染）。图形分三类几何体作底：

- **正方体（最常用底）**：用于课时 1–3 大部分作图题。关键设计：已知点取在棱中点或顶点上；截面边要分别落在上下面（$ABCD$ / $A_1B_1C_1D_1$）、前后侧面、左右侧面上；延長线一律在某个面所在的平面内画，落点要标在该面的棱上。
- **棱锥 / 棱柱**：用于课时 3 第二类作图题。关键设计：三棱锥、四棱锥作底，截面过棱上一点；强调"截面边落在侧面上"，避免学生把截面画穿顶点外侧。
- **组合体（圆柱挖去圆锥）**：仅 1 道拓展题，取选填综合第 11 题图源改写。设计要点：竖直截面与圆锥母线、圆柱母线的交线分别说明；不要求精确计算，只辨析截面形状。

所有图的统一约定：被遮挡线画虚线、可见线画实线；截面区域用浅色面标注，便于学生分辨"截面边"和"几何体棱"。延長线单独用一种线型（点划线或浅色细线）区分，避免和实棱混淆。

## 题型梯度

练习分四档，由易到难，与课时 1–3 顺序对齐：

- **第 1 档 · 判定确定平面**：给几个点/几条线，判断能否确定一个平面，依据公理 3 及三个推论。重点：相交/平行两直线定面，三点不共线才定面。
- **第 2 档 · 画简单截面**：截面与几何体表面的公共点直接可见，连线即得交线（如过正方体三顶点作截面）。重点：截面边落表面，实虚线正确。
- **第 3 档 · 延長线找交点画复杂截面**：公共点不直接可见，需要在某表面所在平面内延长两条线找交点（题集第 25 题类型）。重点：延長线必须画在"同一个面所在平面内"，落点在棱上。
- **第 4 档 · 证明两线共面**：先画截面/给图，再解释理由，最后证明指定两线共面（往相交/平行两直线上靠）。重点：规范书写"因为 $a \cap b = P$（或 $a \parallel b$），所以 $a, b$ 共面"。

> 每档配 3–4 题；第 1 档基础、第 2–3 档中档（课内做）、第 4 档提高（课后）。

## 常见错误

批改与讲评时重点盯这 6 条（每条对应讲解方法里的一句话）：

1. **把截面画到几何体外面**：截面边没有落在表面上，画穿了顶点或棱的外侧。→ 追问"这条边落在哪个表面"，回扣"截面边落在表面上"。
2. **漏画延長线**：复杂截面直接连两个不该连的点，跳过"在同一面内延长找交点"的步骤，导致交线错位。→ 强制要求先画延長线再定交线。
3. **把异面直线当成相交线找交点**：误以为两条空间直线一定有交点，在错误的平面内硬找交点。→ 先判两线是相交、平行还是异面，只有相交/共面才能找交点。
4. **共面证明没用"相交/平行两直线定面"**：凭直觉说"它们在一个面里"，没写相交或平行前提，缺判定依据。→ 强制走"相交定面"或"平行定面"两条主路。
5. **公理 3 三点共线误用**：把"不共线三点定面"记成"任意三点定面"，或反过来在三点共线时仍说能定面。→ 默写"不共线"前提。
6. **实虚线不分**：被遮挡的棱/线画成实线，可见截面边画成虚线，图与实际几何体不符。→ 讲评时按"被面挡住=虚"统一纠正。
