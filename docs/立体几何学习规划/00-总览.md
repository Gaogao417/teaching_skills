## 一、定位与目标学生

- **目标学生**：高一第二学期，立体几何初轮专项训练。
- **起点诊断**：能用文字复述"平行/垂直"定理，但几何语言书写不规范；做截面题不会找交线；证明题"会想不会写"，跳步严重。
- **训练周期**：一个月（四周），每周 3 课时，共 **12 课时**。
- **本轮产出**：只交付"可审阅的方案大纲"，不含完整题库、不含 PDF、不含 assignment YAML。

## 二、每天默写（贯穿全月）

立体几何的根本是"定理 + 几何语言"。每天默写是整轮训练的底盘，具体题库见 `01-默写题作业方案.md`。

- **频率**：每周 3 次正课默写 + 1 次周末滚动小测。
- **形式**：给定理文字版 + 一张空间图，学生在空白处写几何语言（符号 + 字母），而不是抄定理。
- **四份正式作业主题**：
  1. 点/线属于面与确定平面（公理 1、2、3 及推论）。
  2. 平行定理（线线、线面、面面平行判定与性质）。
  3. 垂直定理（线线、线面、面面垂直判定与性质）。
  4. 平行 / 垂直混合调用（最容易漏写"相交"前提）。
- **规模建议**：每份 10–12 题；另设滚动小测模板（从四份里抽 6 题循环）。

## 四、每节课固定结构

每节正课（课时 1–11）一律按下面 5 段式推进，便于学生形成稳定预期，也便于批改归因。

1. **入门默写（5–8 分钟）**：从当周默写库里抽 3–4 题，符号 + 字母，不写文字。
2. **定理回顾（5 分钟）**：用"几何语言 + 一张图"复习上节课核心定理，口头点名说"为什么需要相交"。
3. **讲解例题（15–20 分钟）**：每个专题 3–5 个讲解例题（见各分支文件），统一按"判对象 → 转化目标 → 书写格式"三步板书。
4. **分层练习（20–25 分钟）**：每个专题 8–10 题，分基础 / 中档 / 提高三档；课内只做基础 + 中档，提高题留作课后。
5. **出门测（5 分钟）**：1 道中档题，限时独立完成，作为本节是否过关的判据。

## 五、四层过关标准

学生是否"过关"用四层判据，而不是单纯正确率。每层都要求**几何语言完整**（相交/平行/垂直前提不能漏）。

- **第一层 · 文字层**：能用自然语言说清定理内容（含"相交""平行""平面内"等限定词）。
- **第二层 · 符号层**：能把定理翻成几何语言（$a \subset \alpha$、$a \cap b = P$、$a \perp \alpha$ 等符号书写正确）。
- **第三层 · 作图层**：能画出对应的空间图（截面、辅助线、垂足、二面角平面角位置），图与符号一致。
- **第四层 · 论证层**：能在证明里写清"因为—所以"链，标注每一步用的是哪条判定/性质，不跳步。

> 默写作业负责第一、二层；讲义练习负责第三、四层。某一层反复出错，回到对应分支文件补训。

## 六、总口令

整轮训练反复强调一句口令，用于学生卡住时的自我提示，也用于批改归因：

> **"判对象、找目标、写格式。"**
>
> - 判对象：这条线/这个角，是线线、线面还是面面？
> - 找目标：要证平行就找面内平行线；要证垂直就找两条相交线；要算角就先找截面/垂线。
> - 写格式：相交/平行/垂直的前提写全，每一步标注定理。
