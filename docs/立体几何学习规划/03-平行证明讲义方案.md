## 训练目标

本专题承接课时 1–3 的"确定平面 + 共面/截面"，把学生从"会背平行定理"推进到"会写平行证明"。三条具体目标：

1. **判对象**：拿到一道平行题，第一句话能说出"要证的是线线、线面还是面面平行"，而不是直接动手找线。
2. **目标倒推**：能把目标"翻译"成一个可操作的动作——要证线面平行就去平面里找一条与已知直线平行的直线；要证面面平行就去一个平面里找两条相交直线分别与另一平面平行。
3. **书写格式**：判定/性质定理的几何语言写全，尤其"相交""平面内"两个前提不漏，每一步标注用的是判定定理还是性质定理。

> 过关判据沿用总览的"四层标准"。本专题重点管第三、四层（作图层、论证层）；第一、二层（文字层、符号层）由 `01-默写题作业方案.md` 的平行定理默写兜底。

## 知识脉络：平行关系的相互转化

> 本节是本专题的"脊梁"。线线、线面、面面三种平行**不是三个孤立定理**，而是一条**有向转化链**：判定定理"向上凑"出平行，性质定理"向下用"掉平行。讲清这条链，学生才能在综合题里知道"下一步往哪走"。这正是学生最缺的那条"知识脉络"——三个层级两两之间是怎样转化的。

### 三条平行定理的转化链（必须画成关系图）

讲课时务必把下面这张关系图画在黑板上，贯穿课时 4–6：

```
        判定定理(凑)              判定定理(凑)
   线线 ──────────────> 线面 ──────────────> 面面
   平行                 平行                 平行
     <──────────────       <──────────────
        性质定理(用)              性质定理(用)
```

- **判定定理方向是"向上凑"**：线线⇒线面（线面平行判定：在面内找一条平行线）；线面⇒面面（面面平行判定：面内两条相交线分别与另一面平行）。
- **性质定理方向是"向下用"**：面面⇒线线（面面平行性质：两面与第三个面相交，两条交线平行）；线面⇒线线（线面平行性质：过该直线作面，交线与之平行）。
- **核心口诀**："判定是凑，性质是用。判定用来凑出平行，性质用来用掉平行。"——这句话要反复说，每讲一道题先问学生"这一步是凑还是用？"

### 复杂综合题：判定与性质交替

综合题不是一条线走到底，而是**反复在判定与性质之间切换**（判定⇒性质⇒判定⇒…）。每一步先判对象（证的是线线、线面还是面面），再选定理方向：要"得到"一个更高层的平行就上判定，要"打开"一个已知平行就上性质。这就是课时 6"平行综合 + 目标倒推"专题的核心训练点。

### 所有平行定理清单（含关系对象与方向）

| 定理 | 符号 | 关系对象 | 方向 |
|---|---|---|---|
| 基本事实 4（平行传递性） | $a\parallel b,\ b\parallel c \Rightarrow a\parallel c$ | 线线→线线 | — |
| 线面平行判定 | $a\not\subset\alpha,\ b\subset\alpha,\ a\parallel b \Rightarrow a\parallel\alpha$ | 线线→线面 | 判定（向上） |
| 线面平行性质 | $a\parallel\alpha,\ a\subset\beta,\ \alpha\cap\beta=b \Rightarrow a\parallel b$ | 线面→线线 | 性质（向下） |
| 面面平行判定 | $a\subset\alpha,\ b\subset\alpha,\ a\cap b=A,\ a\parallel\beta,\ b\parallel\beta \Rightarrow \alpha\parallel\beta$ | 线面→面面 | 判定（向上） |
| 面面平行性质 | $\alpha\parallel\beta,\ \alpha\cap\gamma=a,\ \beta\cap\gamma=b \Rightarrow a\parallel b$ | 面面→线线 | 性质（向下） |
| 面面平行其他性质 | $\alpha\parallel\beta,\ a\subset\alpha \Rightarrow a\parallel\beta$ 等 5 条 | 面面→线面 | 性质 |

> **等角定理**（两边分别对应平行的两角相等或互补）：是线线平行传递性的延伸，主要用于求异面直线所成角，不参与"判定/性质"的上下转化，但属于平行知识体系的一环，课时 4 回顾定理时一并带过。

### 一句话总结

本专题的主线就是教学生**在这条转化链上来回走**：会判对象（处在哪一层）、会选方向（凑还是用）、会写前提（相交、平面内不漏）。后面"讲解方法"的三步板书、"定理×例题对照"里的每道题，都是这条链的具体走法。

## 讲解方法

统一用**"先判对象，再转化目标"**的讲法，所有讲解例题和板书都按下面三步走，让学生形成稳定的解题肌肉记忆。这其实是总览总口令"判对象、找目标、写格式"在本专题的具体落地。

### 第一步 · 判对象（这是什么平行？）

读题后先锁定待证结论里的两个对象：两个都是直线 → 线线平行；一个直线一个平面 → 线面平行；两个都是平面 → 面面平行。判错对象，后面全错，所以这步必须让学生口头说出来。

### 第二步 · 转化目标（倒推找一个可证的中间结论）

核心是**目标倒推**，按结论类型分别"翻译"：

- **要证线面平行**（$a \not\subset \alpha$，证 $a \parallel \alpha$）→ 在平面 $\alpha$ 内找一条直线 $b$，使 $a \parallel b$（线面平行**判定**）。这是本专题最高频的动作。
- **要证面面平行**（证 $\alpha \parallel \beta$）→ 在一个平面（如 $\alpha$）内找**两条相交直线** $a,b$，分别证 $a \parallel \beta$、$b \parallel \beta$（面面平行**判定**）。强调"相交"二字。
- **已知平行，要推线线平行**（已知 $a \parallel \alpha$，$a \subset \gamma$，$\gamma \cap \alpha = b$，证 $a \parallel b$）→ 用线面平行**性质**定理。性质定理的方向是"平行 → 线线平行"，和判定定理方向相反，务必分清。

> 一句话总结给学生：**判定定理用来"凑出"平行，性质定理用来"用掉"平行。**

### 第三步 · 书写格式（三段式板书）

每个例题板书统一写成三段：①写前提（点明 $a \subset \alpha$、$a \cap b=P$ 等所有条件）；②写中间平行（怎么得到 $a \parallel b$，标定理名）；③写结论（$\therefore a \parallel \alpha$，标判定定理名）。辅助线/辅助面一律用虚线并在图上标字母，落笔前先在图上指认。

## 图形设计说明

本轮只做设计说明，不画正式图（不写 TikZ/TikZ 代码、不渲染）。正式生成阶段再按本说明出图。

- **载体选择**（按难度递进）：
  - **正方体 / 长方体**：用于基础档，棱、面对角线天然平行，学生一眼能看出"面内平行线"。
  - **棱柱（三棱柱、四棱柱）**：中档主力载体，上下面平行、侧棱平行，适合练面面平行判定与线面平行性质。
  - **棱台 / 棱锥**：提高档载体。棱台上下底面平行、侧棱延长交于一点，天然带"分点/比例"条件，适合练辅助线。
- **中点 / 分点条件**：提高档大量使用中点、三分点等比例条件，目的是逼出辅助线（取中点连线构造平行四边形或三角形中位线），练"目标倒推"里"找面内平行线"这一步。
- **辅助线画法约定**：
  - 辅助线一律用**虚线**，并在图上标字母（如取 $M$ 为 $AB$ 中点，连 $CM$）。
  - 辅助线要明确画在"哪个平面内"——这是学生最容易出错的地方（见"常见错误"），图上用阴影或标注点出辅助线所在平面。
  - 构造平行四边形：连两条中点，强调"先证平行四边形，再得线线平行"。
- **字母约定**：几何体顶点用大写 $A,B,C,D$ / $P$（顶）；中点/分点用 $M,N,E,F$；平面用 $\alpha,\beta,\gamma$。全专题统一，避免一题一换。

## 题型梯度

练习与例题分三档，对应总览"四层标准"的第三、四层。

### 基础档（约 3–4 题）· 直接套判定

- 题面给好"面内的那条平行线"，学生只需判定 + 写格式。
- 载体：正方体 / 长方体。
- 考查：线面平行判定定理的套用 + 几何语言书写是否完整（"相交""平面内"前提）。
- 目的：把"判定定理"和书写格式钉死。

### 中档档（约 3–4 题）· 需要作辅助线 / 辅助面

- 题面不给现成平行线，学生需要自己**取中点 / 取分点、连线**构造出面内的平行线（平行四边形或中位线）。
- 载体：棱柱、棱台。
- 考查：目标倒推的"找面内平行线"动作 + 辅助线作在正确平面内。
- 目的：练"转化目标"那一步，能自己造出中间平行。

### 提高档（约 2–4 题）· 面面与线面混用 + 中点/比例条件

- 题目同时要求面面平行与线面平行的相互转化（判定 ↔ 性质来回切），或给较隐蔽的中点/比例条件，需要先判对象、再倒推选哪条路。
- 来源：平行讲义第 4 节综合应用 + 几何法综合讲义第 1 节。
- 考查：跨定理调用 + 长链路书写格式，对应"论证层"过关。
- 目的：能独立完成"判对象 → 选定理 → 写全前提"的全流程，作为课后与拔高。

## 常见错误

批改归因时重点盯这几条，每条都对应总览口令里漏掉的某一环：

1. **漏写"相交"前提**：证面面平行时只写"两条直线分别与另一平面平行"，不交代两条直线"相交"，直接用判定定理。这是最高频失分点。
2. **把线面平行直接当线线平行用**：已知 $a \parallel \alpha$ 就直接说 $a$ 平行于 $\alpha$ 内任意直线，忽略了"性质定理需要过 $a$ 作一个平面与 $\alpha$ 相交"这一步。
3. **辅助线作在错误的平面内**：取中点连线时，把辅助线画到了不在所需平面的位置，导致"面内平行线"根本不在目标平面里，整条证明链断掉。
4. **用性质定理当判定定理（或反之）**：方向搞反——该用判定（凑平行）时去用性质（用平行），或已知平行想推线线平行却去套判定定理。
5. **判错对象**：题目要证线面平行，学生当成线线平行去做，找错了目标；或把面面平行和线面平行混淆。
6. **书写跳步、不标定理名**：从"中点"直接跳到"平行"，中间"证平行四边形 / 中位线"那一步省略，也不标注每步用哪条定理，批改无法归因。
