## 引言 · 本轮的核心重构

上一版的方案是"**4 份固定主题的默写作业**"（点线面 / 平行 / 垂直 / 混合），每周一份、各 10–12 题。这一版推翻这个模型，换成"**定理库 + 抽题**"模型：

- **不再按周固定主题**，而是先建立一个"立体几何定理库"，每次作业从库里按需抽题。
- **不再用同一种形式对待所有定理**。定理库里的定理天然分成两类，训练形式完全不同：
  - **A 类 · 点线面关系 = 整理**：这类定理不需要"默写文字"，而是要整理成一张**逻辑脉络清晰的结构图/表**——核心是"点∈线、点∈面、线⊂面"之间的从属关系，以及位置关系的分类。
  - **B 类 · 平行/垂直定理 = 默写填空**：这类定理要的不是背文字，而是"**给图、一步推理、补中间条件**"——给一个平行/垂直关系，让学生推另一个平行/垂直关系，中间的关键步骤由学生补全。

> 一句话：**默写题不再是"4 份固定作业"，而是"从定理库抽题，按定理类型用不同形式"**。A 类整理结构，B 类填空推理。

## 训练目标

新模型下，两类定理的目标分开定：

1. **A 类目标 · 建立结构化整理能力**：让学生能把"空间中点、线、面的从属关系"整理成一张有逻辑主线的图/表——能说清"基本事实 2 是点∈线+点∈面⇒线⊂面的桥梁"、"基本事实 3 是公共点⇒交线"、"三个推论都讲怎么定面"。训练的是**结构与分类**，不是逐字背诵。
2. **B 类目标 · 把定理从"背文字"推进到"看图写一步推理 + 补中间条件"**：学生看到一张带标注的空间图 + 一个已知关系 + 一个要证的结论，能独立补出中间那一步（多是用某个判定/性质定理搭桥）。这才是定理真正"会用"的标志。
3. **前提写全（B 类）**：训练学生在写判定定理时，**相交/平行/平面内**这些限定条件一个不漏——这是整轮批改归因的头号考点，在"补中间条件"这一步暴露得最彻底。
4. **判对象先于写公式（B 类）**：每道填空题先让学生判"这条定理是线线、线面还是面面"，再补中间步骤，避免对象混淆。
5. **滚动抗遗忘**：通过每周从定理库滚动抽题（A 类整理项 + B 类填空题），让所有定理循环重现，防止"这周背了下周忘"。

> A 类只负责"整理出结构"；B 类负责"一步推理填空"；多步作图与论证由讲义练习承接，不在本文件展开。

## Category A · 点线面关系整理

A 类不考"默写"，考"**整理得有没有逻辑主线**"。目标产物是一张结构图或一张分类表，能让人一眼看出"点、线、面之间的从属关系是怎么搭起来的"。

### 要整理成什么样

整理要覆盖三条逻辑主线，而不是把定理一条条平铺罗列：

1. **从属关系主线（核心）**：点、线、面之间靠"∈"和"⊂"连起来。这里的关键桥梁是 **基本事实 2**——
   - 符号：$A \in l$，$B \in l$，$A \in \alpha$，$B \in \alpha \Rightarrow l \subset \alpha$。
   - 逻辑：**点在线上 + 点在面上 ⇒ 线在面内**。这是把"点"层层升级到"线"、再升级到"面"的那一步桥。
   - 整理时要让学生点明："为什么两条直线上各一点在面内，整条线就进了面？"——靠的就是这条桥。
2. **公共点 → 交线主线**：基本事实 3——
   - 符号：$P \in \alpha$，$P \in \beta \Rightarrow \alpha \cap \beta = l$，$P \in l$。
   - 逻辑：**两个不重合的面有一个公共点 ⇒ 有且只有一条过该点的交线**。这是"面和面怎么发生关系"的入口。
3. **定面主线**：基本事实 1 + 三个推论，统一归到"**多少东西能确定一个平面**"这一问下——
   - 基本事实 1：过不在一条直线上的三点，有且只有一个平面（3 个不共线点 ⇒ 平面）。
   - 推论①：一条直线和直线外一点确定平面。
   - 推论②：两条相交直线确定平面。
   - 推论③：两条平行直线确定平面。
   - 整理时四条要并列对比，让学生看出"它们其实都在回答同一个问题"。

### 整理的组织原则

整理图/表要让学生自己分清两件事，不能混：

- **从属关系**：谁属于谁（$A \in l$，$l \subset \alpha$）——讲的是"归属"。
- **位置关系分类**：线和线（相交 / 平行 / 异面）、线和面（在面内 / 相交 / 平行）、面和面（平行 / 相交）——讲的是"怎么摆"。

> 整理类作业的判分标准不是"定理抄全没抄全"，而是"**逻辑主线清不清楚、分类乱没乱**"。本轮只定整理应覆盖的内容和主线，不替学生生成整张图。

## Category B · 平行垂直默写填空

B 类是真正意义上的"默写"，但形式不是"背定理文字"，而是"**给图、一步推理、补中间条件**"。

### 题型结构

每道 B 类填空题统一是这样一个结构：

1. **给一张带标注的空间图**：图中画出定理涉及的所有对象，标好字母。
2. **给一个已知关系**（平行或垂直，已知成立的）。
3. **给一个要证的结论**（另一个平行或垂直关系）。
4. **中间一步留空**：让学生补出那一步关键推理——通常是用某个判定/性质定理搭的桥。

> 核心是第 4 步：要补的不是结论，也不是已知，而是**把已知和结论连起来的那座桥**。这才是定理"会不会用"的真正考点。

### 标准例题（用户给的样板）

**线面平行性质定理**的填空：

- 已知：$AB \parallel \alpha$。
- 求证：$AB \parallel CD$。
- 图：画出 $AB$、平面 $\alpha$、以及一条在 $\alpha$ 内、与 $AB$ 平行的 $CD$。
- **中间留空**：学生要补出"过 $AB$ 作平面 $ABCD$，使 $ABCD \cap \alpha = CD$"这一步。

这就是"给线面平行，让补交线，推线线平行"——线面平行性质定理的典型一步推理。

### 题型模板（每个定理怎么变成填空）

下面给 4 个模板，示范"一条定理 → 一道一步推理填空"的改写方式。正式生成时按此模式扩成完整题库。

1. **线面平行性质（线面⇒线线）**：
   - 已知 $a \parallel \alpha$，求证 $a \parallel b$。
   - 留空：补出"$a \subset \beta$，$\alpha \cap \beta = b$"（即构造辅助面、找到交线）。
   - 符号：$a \parallel \alpha$，$a \subset \beta$，$\alpha \cap \beta = b \Rightarrow a \parallel b$。
2. **线面平行判定（线线⇒线面）**：
   - 已知 $a \parallel b$，$b \subset \alpha$，求证 $a \parallel \alpha$。
   - 留空：补出"$a \not\subset \alpha$"（线在面外这一前提，最易漏）。
   - 符号：$a \not\subset \alpha$，$b \subset \alpha$，$a \parallel b \Rightarrow a \parallel \alpha$。
3. **线面垂直判定（线线⇒线面）**：
   - 已知 $l \perp m$，$l \perp n$，$m \subset \alpha$，$n \subset \alpha$，求证 $l \perp \alpha$。
   - 留空：补出"$m \cap n = A$"（两条线相交这一前提，头号失分点）。
   - 符号：$l \perp m$，$l \perp n$，$m \subset \alpha$，$n \subset \alpha$，$m \cap n = A \Rightarrow l \perp \alpha$。
4. **面面垂直性质（面面⇒线面）**：
   - 已知 $\alpha \perp \beta$，$\alpha \cap \beta = l$，$AB \subset \alpha$，$AB \perp l$，求证 $AB \perp \beta$。
   - 留空：补出"$AB \perp l$"已在已知中时，让学生补"这条性质定理的完整前提组合"，强化"面面垂直 + 交线 + 面内 + 垂直于棱"四件套缺一不可。
   - 符号：$\alpha \perp \beta$，$\alpha \cap \beta = l$，$AB \subset \alpha$，$AB \perp l \Rightarrow AB \perp \beta$。

> 还有线面垂直性质（$a \perp \alpha$，$b \perp \alpha \Rightarrow a \parallel b$，跨类结论）、面面平行判定/性质、面面垂直判定、二面角平面角三要素，正式生成时各配一道同结构填空。每道图必须画出所有对象并标好字母。

## 讲解方法

两类定理讲法不同，分开固定：

1. **B 类 · 三段式填空**：每题统一按"判对象 → 补中间 → 写结论"三步。先判"这条定理是线线、线面还是面面"（一行），再补中间条件（核心，若干行符号），最后写结论（$\Rightarrow$ 后一行）。空白按这三段排版。
2. **B 类 · 给图不给文字**：题目只给空间图 + 已知/求证，定理文字版不给，符号全部由学生补。绝不让学生抄现成几何语言。
3. **B 类 · 批改三秒归因**：每题按"对象错 / 中间漏 / 符号错"三类标红。对象错最严重（整题重做），中间漏次之（补一条即可），符号错最轻（只改符号）。
4. **B 类 · 错题回炉**：当天错的题，第二天滚动里必带一道同定理变式，连错两次升级到讲义练习补训。
5. **A 类 · 一起搭图，不填空**：A 类不搞填空，而是**师生一起把整理图/表搭出来**——老师引导逻辑主线（从属关系 / 公共点→交线 / 定面三条），学生在自己的图上落位。判分看"主线清不清、分类乱没乱"。
6. **口令复述（B 类）**：每次填空前口头复述总口令"判对象、找桥、写格式"，强化肌肉记忆。

## 抽题与滚动练习机制

定理库抽题是滚动练习的核心，机制如下：

1. **每次作业的配比**：从 A 类抽 **1–2 个整理项**（按三条主线轮换），从 B 类抽 **3–4 道填空题**（判定/性质交替、不同关系对象混抽）。总量控制在 5 分钟内完成。
2. **滚动重现**：每个定理进库后，至少在随后两周内各被抽到一次。错过的定理优先回炉，连错两次的升级到讲义补训。
3. **周节奏**：每周 3 次正课抽题 + 1 次周末滚动小测。周末小测从全库抽，A 类 1 项 + B 类 5 题（判定 2、性质 2、前提辨析 1），确保每周都碰最易错的"中间条件漏写"。
4. **进度标记**：定理库每条标注"已抽次数 / 上次出错主题"，作为下次抽题的依据。

> 单次时长 5 分钟内完成。保持轻量，宁可少抽一题，也不要让默写挤占正课时间。

## 常见错误

以下是立体几何默写/整理里的典型错误，批改时按这些归因，错题回炉也按这些主题分类补训。

1. **漏写"相交"前提**：线面平行判定、线面垂直判定都要求面内"两条相交线"，学生常只写 $a \parallel b$、$a \parallel c$ 而漏掉 $b \cap c = P$。这是 B 类头号失分点，在"补中间条件"这一步暴露最彻底。
2. **补不出中间那座桥**：B 类学生常把已知和结论直接用 $\Rightarrow$ 连起来，跳过"构造辅助面/找交线"这一步。典型如线面平行性质里漏掉"$a \subset \beta$，$\alpha \cap \beta = b$"。根因是没真正理解定理是"怎么搭桥的"。
3. **$a \parallel b$ 与 $a \parallel \alpha$ 混淆**：把"线线平行"符号写成"线面平行"，或反之。根因是没先判对象。
4. **垂足位置写错**：线面垂直里垂足应标在面 $\alpha$ 上，学生常画在线上或图外；面面垂直里二面角平面角的顶点位置也常错。
5. **"平面内"前提漏写**：$a \subset \alpha$ 这一条常被省略，导致判定不成立。尤其线面平行判定必须写 $a \not\subset \alpha$ 与 $b \subset \alpha$ 两条。
6. **判定与性质写反**：判定是"由条件推位置关系"，性质是"由位置关系推新结论"，学生常把因果箭头方向写反，或把性质当判定用。
7. **A 类整理分类混乱**：把"从属关系（∈/⊂）"和"位置关系分类（相交/平行/异面）"混在一起，看不出基本事实 2 是从属关系的桥、基本事实 3 是交线的入口。这是 A 类的主要失分点。
8. **符号书写不规范**：$a \subset \alpha$ 写成 $a \in \alpha$，$a \cap b = P$ 写成 $a \times b$，$\Rightarrow$ 写成 $=$。这类最轻，但量大，要早纠正。
