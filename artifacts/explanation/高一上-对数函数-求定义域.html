学生版讲解：y = ln(x - 1) 的定义域

主题：对数函数的定义域

原题

求函数 y = ln(x - 1) 的定义域。

一、先把题目拆开

已知条件函数表达式为 y = ln(x - 1)，这是一个对数函数
要求目标求定义域——哪些 x 值能让这个式子算得出来
关键词ln（自然对数）、定义域、真数
容易忽略对数括号里的东西（真数）必须大于 0

二、这题准备怎么走

  1. 看清楚：ln 后面括号里是 x - 1，这就是"真数"
  2. 想条件：真数必须大于 0，所以 x - 1 > 0
  3. 解不等式：移项得 x > 1
  4. 写答案：用区间表示法写成 (1, +∞)

三、关键想法

核心动作：看到 ln( )，第一步就是把括号里的东西设为大于 0。
不用想别的，直接写：括号里的式子 > 0，然后解这个不等式。

四、标准解法

求 y = ln(x - 1) 的定义域
第 1 步：找到真数

函数 y = ln(x - 1) 中，ln 后面括号里的 x - 1 就是真数。

为什么要找真数？因为对数函数只接受正数输入，负数和 0 都不行。

第 2 步：令真数大于 0

x - 1 > 0

ln 的括号里必须填正数。填 0 → ln(0) 算不出来；填负数 → ln(-1) 也算不出来。只有正数才行。

第 3 步：解不等式

x - 1 > 0，两边同时加 1，得 x > 1

一元一次不等式，移项即解。

第 4 步：用区间表示法写出定义域
定义域为 (1, +∞)

圆括号表示不包含端点 1。x = 1 代入得 ln(0)，无意义。

五、边讲边问

想一想 1

想一想 3

六、易错提醒

易错点 1：写成 [1, +∞)（用了方括号）。
x = 1 时 ln(0) 无意义，端点 1 不能包含在内。必须用圆括号 (1, +∞)。
易错点 2：解不等式时方向搞反，写成 x < 1。
x - 1 > 0 移项后是 x > 1，不是 x < 1。可以代值验证：x = 2 时 ln(1) = 0 没问题。

七、一句话总结

求对数函数的定义域，就三步：找真数 → 令真数 > 0 → 解不等式写区间。
