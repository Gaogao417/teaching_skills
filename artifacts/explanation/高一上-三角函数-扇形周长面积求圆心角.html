学生版讲解：扇形周长面积求圆心角

主题：扇形公式 · 联立消元 · 一元二次方程 | 适合高中数学

原题

扇形周长为 6，面积为 2，求圆心角的弧度数。

零、先复习三个扇形公式

弧长公式：l = rθ
面积公式：S = (1/2)rl = (1/2)r²θ
周长公式：C = 2r + l

其中 r = 半径，l = 弧长，θ = 圆心角（弧度）。周长 = 两条半径 + 一段弧。

一、先把题目拆开

已知条件周长 2r + l = 6，面积 (1/2)rl = 2
要求目标圆心角 θ 的弧度数
关键词扇形、周长、面积、弧度
容易忽略周长包含两条半径（2r），不只是弧长 l

二、这题准备怎么走

  1. 从面积公式算出 rl 的值，把 l 用 r 表示
  2. 代入周长公式，得到只含 r 的方程
  3. 解方程求出 r
  4. 用弧长公式 θ = l/r 求圆心角

三、关键想法

两个方程，三个未知数，怎么办？先从面积算出 rl = 4，就把 l = 4/r。这样周长方程里就只剩 r 一个未知数了。

四、标准解法

1从面积公式出发

面积公式：(1/2)rl = 2

两边乘 2：

rl = 4， 即 l = 4/r

为什么：面积直接把 r 和 l 乘在一起了。知道乘积，就能用一个表示另一个。

2代入周长公式

周长 = 2r + l = 6，把 l = 4/r 代入：

2r + 4/r = 6

为什么：两个方程里都有 r 和 l。把 l 消掉，只剩 r，就能解了。

3解关于 r 的方程

两边乘 r（r > 0）：

2r² − 6r + 4 = 0

两边除以 2：

r² − 3r + 2 = 0

因式分解：

(r − 1)(r − 2) = 0， 所以 r = 1 或 r = 2

为什么：r 是扇形半径，一定大于 0，所以乘 r 不会改变等号方向。因式分解：找两个数，乘积 = 2，和 = 3，即 1 和 2。

4分别求圆心角

用 l = 4/r 和 θ = l/r：

情形一：r = 1

l = 4/1 = 4，θ = l/r = 4/1 = 4

情形二：r = 2

l = 4/2 = 2，θ = l/r = 2/2 = 1

为什么：两个 r 都是正数，都是合法的扇形半径，所以两个解都要保留。小半径对应大弧度（"扁扇形"），大半径对应小弧度（"瘦扇形"）。

5验算

r = 1, l = 4：周长 = 2×1 + 4 = 6 ✓，面积 = (1/2)×1×4 = 2 ✓

r = 2, l = 2：周长 = 2×2 + 2 = 6 ✓，面积 = (1/2)×2×2 = 2 ✓

最终答案：圆心角为 1 弧度或 4 弧度。

五、边讲边问

想一想 1

扇形的周长包含哪几部分？如果只写了 l，会怎样？

想一想 2

面积公式 (1/2)rl = 2 乘以 2 以后得到 rl = 4。这个 rl = 4 告诉你 l 等于多少（用 r 表示）？

想一想 3

解出 r = 1 和 r = 2 两个解之后，能不能只保留一个？为什么两个都要？

想一想 4

你能从几何上解释为什么这道题有两个答案吗？（提示：想想什么样的扇形半径小、弧长大。）

六、易错提醒

易错点 1：周长漏写 2r。扇形周长 = 两条半径 + 弧长 = 2r + l，不是只有弧长。如果写成 l = 6，整个题就做错了。
易错点 2：解二次方程出错。2r² − 6r + 4 = 0 两边要先除以 2 变成 r² − 3r + 2 = 0，再因式分解。直接分解 2r² − 6r + 4 = 0 时系数容易看错。
易错点 3：只写一个解。r = 1 和 r = 2 都是正数，都对应合法的扇形，两个圆心角 4 和 1 都要写上。

七、一句话总结

面积 → 得 rl → 用 r 表示 l → 代入周长 → 解二次方程 → θ = l/r。遇到扇形给周长和面积，第一步永远是：从面积算出 rl，把一个变量消掉。
