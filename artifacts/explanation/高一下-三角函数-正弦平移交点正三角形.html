学生版讲解：sin函数平移交点正三角形

主题标签：三角函数 · 图像平移 · 交点 · 正三角形 · 参数方程

原题

将函数 y = sin(πx) 的图像向左平移 m（m > 0）个单位后，所得图像与原函数图像的相邻三个交点恰好构成一个正三角形，求 m 的值。

一、先把题目拆开

已知条件① y = sin(πx)；② 左移 m 个单位（m > 0）；③ 两个图像的相邻三个交点构成正三角形
要求目标求 m 的值
关键词左移、交点、相邻、正三角形
容易忽略① sin A = sin B 有两种情形；② 正三角形要求三边都相等，不只是两边

二、这题准备怎么走

  1. 写出左移后的函数表达式
  2. 联立两个函数，求出交点横坐标的通项公式
  3. 写出相邻三个交点的坐标，发现纵坐标的规律
  4. 用正三角形"三边相等"的条件列方程
  5. 解方程，求出 m

三、关键想法

把 sin A = sin B 的两种情形分别列出来，化简后会发现交点的横坐标成等差数列，公差为 1。这意味着相邻交点的横坐标差恒为 1，问题从"求无数个交点"一下子变成"只看一个 y 值"。

四、标准解法

第 1 步：写出左移后的函数

左移 m 个单位，x 变成 x + m：

y = sin(π(x + m))
第 2 步：联立求交点

两个图像的交点满足：

sin(πx) = sin(π(x + m))

设 A = πx，B = π(x + m)，则 sin A = sin B，有两种情形：

情形一：A = B + 2kπ，即 πx = π(x + m) + 2kπ

化简：0 = πm + 2kπ，即 m = −2k。这只给出 m 的特殊值，不是我们要找的一般交点。

情形二：A = π − B + 2kπ，即 πx = π − π(x + m) + 2kπ

化简：2πx = π(1 − m) + 2kπ

两边除以 2π：

x = 1 − m2 + k   (k ∈ Z)

为什么：这是交点横坐标的通项公式，相邻两个 k 值对应的 x 差 1。

第 3 步：写出相邻三个交点的坐标

记 x0 = (1−m)/2，则第 k 个交点：

Pk = (x0 + k,   sin(π(x0 + k)))

计算纵坐标：

sin(π(x0 + k)) = sin(πx0 + kπ) = sin(πx0) · cos(kπ) + cos(πx0) · sin(kπ)

因为 sin(kπ) = 0，cos(kπ) = (−1)k：

yk = (−1)k · y0  ，其中 y0 = sin（π(1−m)2）

为什么：纵坐标交替取 y0 和 −y0。

相邻三个交点为：

P0 = (x0, y0)，  P1 = (x0+1, −y0)，  P2 = (x0+2, y0)
第 4 步：列正三角形条件

正三角形要求三边相等。先算三条边的长度：

|P0P2| = √[(x0+2 − x0)2 + (y0 − y0)2] = √[22 + 0] = 2

|P0P1| = √[(x0+1 − x0)2 + (−y0 − y0)2] = √[1 + 4y02]

令 |P0P1| = |P0P2| = 2：

√(1 + 4y02) = 2  ⟹  1 + 4y02 = 4  ⟹  y02 = 34

为什么：|P1P2| 的计算与 |P0P1| 完全对称，所以只需一个等式。

第 5 步：解 m

由 y02 = 3/4 得 |y0| = √3/2：

|sin(π(1−m)2)| = √32

因为 m > 0，令 π(1−m)2 = π3 或 −π3（取主值范围内）：

1−m2 = 13  ⟹  1−m = 23  ⟹  m = 13

1−m2 = −13  ⟹  1−m = −23  ⟹  m = 53

两个值都满足 m > 0。

答：m = 13 或 53

想一想 3

想一想 4

验证：把 m = 1/3 代入，算出 P0、P1、P2 的坐标，然后算 |P0P1| 和 |P0P2|，看它们是否相等。

六、易错提醒

易错点 1：漏掉 sin A = sin B 的第二种情形 A = π − B + 2kπ。如果只写 A = B，只能得到 m = −2k，完全找不到交点。
易错点 2：左移的方向搞错。左移 m 个单位是 x → x + m，不是 x → x − m。
易错点 3：算距离时纵坐标的符号弄错。P1 的纵坐标是 −y0，所以 |P0P1| 中纵坐标差是 (−y0) − y0 = −2y0，平方后是 4y02，不是 y02。

七、一句话总结

sin A = sin B 两种情形化简 → 交点横坐标等差（公差 1） → 纵坐标交替 ±y0 → 三边相等列方程 → 解参数。
