学生版讲解：求 tan(2x+1) 的最小正周期

三角函数 · 周期变换 · tan 型函数

原题

求函数 y = tan(2x + 1) 的最小正周期。

一、先把题目拆开

已知条件函数解析式 y = tan(2x + 1)，其中自变量为 x
要求目标最小正周期 T
关键词tan、周期、最小正
容易忽略tan 的基础周期是 π，不是 2π；平移量 +1 不影响周期

二、这题准备怎么走

  1. 回忆 tan x 的基础周期是多少
  2. 找到自变量 x 前面的系数 ω
  3. 代入周期公式 T = π/|ω|
  4. 验证答案是否正确

三、关键想法

周期只和自变量 x 前面的系数 ω 有关，和后面的加数（平移量）无关。
只看系数，不看加数。

四、标准解法

第 1 步：回忆基础周期

tan x 的最小正周期是 π。

注意：sin x 和 cos x 的周期是 2π，但 tan x 的周期是 π——别搞混！

为什么：tan 的渐近线间距是 π，图像每 π 重复一次。

第 2 步：读出自变量系数

y = tan(2x + 1)，自变量 x 前面的系数 ω = 2。

为什么：周期公式 T = 基础周期 / |ω|，需要先找到 ω。

第 3 步：代入公式

T = π / |2| = π/2

T = π / |ω| = π / 2
第 4 步：验证

把 T = π/2 代回原函数，检查是否满足 f(x + T) = f(x)：

tan(2(x + π/2) + 1) = tan(2x + π + 1)

因为 tan 的周期是 π，所以 tan(2x + π + 1) = tan(2x + 1) ✓

注意：+1 始终跟着 x 走，加上 π/2 后变成了 +π + 1，但 tan 看到的是整体增加了 π，正好一个周期。

最终答案
Tmin = π/2

五、边讲边问

想一想 1

tan x 的最小正周期是多少？sin x 呢？它们为什么不一样？

想一想 2

如果题目变成 y = tan(2x + 100)，周期会变成多少？100 这个数会影响周期吗？

想一想 3

y = tan(2x + 1) 和 y = tan(2x) 的周期相同吗？图像有什么区别？

六、易错提醒

易错点 1：把 tan 的基础周期记成 2π。
tan x 的周期是 π，不是 2π。sin 和 cos 才是 2π。
易错点 2：认为 +1 会改变周期。
加号后面的数是"平移量"，只让图像左右移动，不改变重复的间距。
易错点 3：忘记公式里要取绝对值。
如果 ω = -2，T = π/|-2| = π/2，结果一样。取绝对值保证周期为正。

七、一句话总结

找到 x 前面的系数 ω，用 T = 基础周期 / |ω| 算。
tan 型：T = π / |ω|；sin/cos 型：T = 2π / |ω|。
