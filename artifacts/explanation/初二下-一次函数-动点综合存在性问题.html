一次函数与平行四边形中的动点问题

一次函数 · 平行四边形 · 动点参数化 · 等腰直角三角形

原题

直线 y = 2x + 6 的图像交 x 轴于点 A，交 y 轴于点 B，点 D 在 x 轴正半轴上，且 OD = 8/3·OA，以 AB、AD 为边作平行四边形 ABCD。

(1) 求点 A、B 的坐标

(2) 求出直线 BD 的函数解析式

(3) 点 E 从 O 点出发，以每秒一个单位的速度沿 x 轴正方向移动，运动时间记为 t，点 F 是线段 BD 的中点，连接 EF 并延长，交直线 BC 于点 H。请直接写出：当 t 为何值时，四边形 ABHE 是平行四边形 / 等腰梯形？

(4) 点 Q 是直线 BD 上的一个动点，在 y 轴上找一点 P，连接 AP、AQ、PQ，当 △APQ 是以 PQ 为斜边的等腰直角三角形时，请直接写出点 P 的坐标。

一、先把题目拆开

已知条件直线 y = 2x + 6；OD = 8/3·OA；AB、AD 是平行四边形的边；E 以 1 单位/秒在 x 轴正方向移动；F 是 BD 中点
要求目标(1) A、B 坐标 (2) BD 解析式 (3) t 取何值使 ABHE 为平行四边形/等腰梯形 (4) P 的坐标
关键词"交 x 轴""交 y 轴" → 令 y=0 / x=0；"OD = 8/3·OA" → OA 是长度不是坐标；"以 AB、AD 为边" → AB 和 AD 是邻边
容易忽略OA 是长度 = |−3| = 3，不是 −3；平行四边形的 C 点要自己算出来

二、这题准备怎么走

  1. 求 A、B 坐标（令 y=0，令 x=0）
  2. 算 OA 长度 → 算 OD → 确定D坐标
  3. 用平行四边形条件确定 C 坐标
  4. 待定系数法求 BD 直线
  5. 把 E 的运动参数化：E = (t, 0)
  6. 求直线 EF 与直线 BC 的交点 H，得到 H = (8−t, 6)
  7. 用向量条件列方程，求平行四边形和等腰梯形的 t 值
  8. 用"旋转法"求第(4)问的 P 坐标

三、关键想法

动点问题的核心：把"运动的点"用一个字母 t 表示坐标，然后所有运算都带着 t 做，最后把 t 算出来。
等腰直角三角形的快捷方法：从两个非直角顶点向公共直线作垂线，利用"一线三垂直"得到两个直角三角形全等（AAS），从而把几何条件变成简单的线段等式。P 在上、P 在下各算一次。

四、标准解法

第 (1) 问：求点 A、B 的坐标
求 A（交 x 轴）

交 x 轴 → y = 0：

0 = 2x + 6 → 2x = −6 → x = −3

所以 A(−3, 0)

交 x 轴的点的 y 坐标一定等于 0，只需令 y = 0 代入直线方程。

求 B（交 y 轴）

交 y 轴 → x = 0：

y = 2×0 + 6 = 6

所以 B(0, 6)

交 y 轴的点的 x 坐标一定等于 0，只需令 x = 0 代入直线方程。

想一想 1

如果直线 y = −x + 4，它交 x 轴于哪个点？交 y 轴于哪个点？

预备：确定 D、C 坐标（为后面做准备）
求 D 坐标

OD = 8/3 × OA。OA 是长度 = |−3 − 0| = 3（不是 −3！）

OD = 8/3 × 3 = 8

D 在 x 轴正半轴 → D(8, 0)

OA 是"距离"，永远是正数。A 的坐标是 −3，但 OA 的长度是 |−3| = 3。

求 C 坐标（平行四边形）

平行四边形 ABCD 中，AB 和 AD 是邻边。所以 C 的位置 = A + AB方向 + AD方向。

用坐标计算：从 A 到 B，x 走了 0−(−3) = 3，y 走了 6−0 = 6。

从 A 到 D，x 走了 8−(−3) = 11，y 走了 0−0 = 0。

从 B 继续走 AD 的方向：C = (0+11, 6+0) = (11, 6)

平行四边形中，AD ∥ BC 且 AD = BC，所以从 B 走和 AD 相同的方向和距离就到 C。

四个关键点的坐标表（后面反复用到）：

A(−3, 0)、B(0, 6)、C(11, 6)、D(8, 0)

第 (2) 问：求直线 BD 的解析式
待定系数法

BD 过 B(0, 6) 和 D(8, 0)。

斜率 k = (0 − 6) / (8 − 0) = −6/8 = −3/4

过 B(0, 6) → b = 6

所以 y = −3/4 x + 6

两点确定一条直线。斜率公式 k = (y₂−y₁)/(x₂−x₁)，然后代入任一点求 b。

第 (3) 问：动点问题——求 t 使 ABHE 为平行四边形 / 等腰梯形
第 1 步：参数化点 E

E 从原点出发，沿 x 轴正方向，速度 1 单位/秒。

所以 E 的坐标 = (t, 0)，其中 t > 0

这就是"参数化"——用一个字母 t 代替一个不断变化的量。后面所有关于 E 的计算都带着 t。

第 2 步：确定 F 坐标

F 是 BD 的中点。B(0, 6)，D(8, 0)。

F = ((0+8)/2, (6+0)/2) = (4, 3)

中点坐标 = 两个端点坐标的平均值。

第 3 步：求直线 EF 与 BC 的交点 H（关键步骤）

直线 BC 是 y = 6（水平线，因为 B 和 C 的 y 坐标都是 6）。

直线 EF 过 E(t, 0) 和 F(4, 3)。

求 EF 与 y = 6 的交点：

  EF 斜率 = (3−0)/(4−t) = 3/(4−t)

  EF 方程：y = 3/(4−t) · (x − t)

  令 y = 6：6 = 3(x−t)/(4−t)

  化简：2(4−t) = x − t → 8 − 2t = x − t → x = 8 − t

所以 H = (8 − t, 6)

这一步是整道题的关键。一旦写出 H = (8−t, 6)，后面所有问题都变成了代数方程。

现在四边形 ABHE 的四个点：

A(−3, 0)、B(0, 6)、H(8−t, 6)、E(t, 0)

注意：B 和 H 的 y 坐标都是 6，A 和 E 的 y 坐标都是 0。所以 BH 和 AE 都是水平线，永远平行。

第 4 步：平行四边形条件

ABHE 是平行四边形 → AB ∥ HE 且 BH ∥ AE

BH ∥ AE 已经永远成立（都是水平线）。

只需要 AB ∥ HE：

  AB 方向：从 A(−3,0) 到 B(0,6) → x 走 3，y 走 6

  HE 方向：从 H(8−t,6) 到 E(t,0) → x 走 t−(8−t) = 2t−8，y 走 0−6 = −6

  AB ∥ HE → 3/(2t−8) = 6/(−6) = −1

  2t − 8 = −3 → t = 5/2

平行四边形的判定：两组对边分别平行。一组已经天然平行，只需让另一组也平行。

第 5 步：等腰梯形条件

等腰梯形 = 只有一组对边平行 + 不平行的两边长度相等。

BH ∥ AE 永远成立 → 这是"平行的那一组"。

不平行的两边是 AB 和 HE，需要 |AB| = |HE|。

  |AB| = √(3² + 6²) = √45 = 3√5

  |HE| = √((2t−8)² + (−6)²) = √((2t−8)² + 36)

  令 |AB| = |HE|：

  45 = (2t−8)² + 36

  (2t−8)² = 9

  2t − 8 = 3 或 2t − 8 = −3

  t = 11/2 或 t = 5/2

  t = 5/2 时 ABHE 是平行四边形（不是梯形），排除！

所以 t = 11/2

平行四边形是特殊的梯形（两组对边都平行），题目要求等腰梯形，必须排除平行四边形的情况。

第 (3) 问答案：
平行四边形：t = 5/2
等腰梯形：t = 11/2
第 (4) 问：等腰直角三角形存在性——一线三垂直全等法
先理解条件

△APQ 以 PQ 为斜边的等腰直角三角形 → 直角在 A 点。

这意味着：

  (1) AP = AQ（两条直角边相等）

  (2) ∠PAQ = 90°（直角在 A）

等腰直角三角形中，斜边对着直角。PQ 是斜边 → 直角在 A。

作辅助线（一线三垂直）

核心辅助线：

  ① 过 P 向 x 轴作垂线，垂足为 O(0, 0)（因为 P 在 y 轴上，垂足恰好是原点）

  ② 过 Q 向 x 轴作垂线，垂足为 M(q, 0)

这样就有三个直角：

  ∠AOP = 90°（PO ⊥ x 轴）

  ∠QMA = 90°（QM ⊥ x 轴）

  ∠PAQ = 90°（题目已知）

这就是"一线三垂直"——一条线（x 轴）上出现了三个直角关系。这种构造能把 ∠PAQ = 90° 的条件"传递"到两个直角三角形之间，建立角相等的关系。

找角关系（关键一步）

A(−3, 0)、O(0, 0)、M(q, 0) 都在 x 轴上，且 O 和 M 都在 A 的右边。

所以射线 AO 和射线 AM 方向相同（都向右）。

因此 ∠MAQ = ∠OAQ（同一个角）。

因为 ∠PAQ = 90°：

  ∠PAO + ∠OAQ = 90°

在直角三角形 APO 中：

  ∠PAO + ∠APO = 90°

所以 ∠APO = ∠MAQ

两个角都等于 90° 减去 ∠PAO，所以它们相等。这个角相等关系是一线三垂直的核心——它把 x 轴一侧的角"搬"到了另一侧。

设未知数

P 在 y 轴上 → P = (0, p)

Q 在 BD 上（BD: y = −3/4 x + 6）→ Q = (q, −3/4 q + 6)

A = (−3, 0)

各线段长度：AO = 3，PO = |p|，QM = |−3/4 q + 6|，AM = |q + 3|

情况一：P 在 x 轴下方，Q 在 x 轴上方
AAS 全等

P 在 x 轴下方 → p < 0，PO = −p

Q 在 x 轴上方 → −3/4 q + 6 > 0，QM = −3/4 q + 6

在 △APO 和 △QMA 中：

  ∠AOP = ∠QMA = 90° ✓

  ∠APO = ∠MAQ（上面已证）✓

  AP = AQ（等腰条件，这是两个三角形中 90° 角对面的边）✓

所以 △APO ≅ △QMA（AAS）

对应边相等：AO = QM，PO = AM

  AO = 3 → QM = 3 → −3/4 q + 6 = 3 → q = 4

  PO = AM → −p = q + 3 = 7 → p = −7

所以 P(0, −7)

AAS 全等需要两角一边。90° 是公共的，∠APO = ∠MAQ 是我们证的，AP = AQ 是题目给的等腰条件。注意 AP 对应的是 △QMA 中 QM 对面的 QA，对应关系是 A↔Q、P↔M、O↔A。

情况二：P 在 x 轴上方，Q 在 x 轴下方
换一组全等

P 在 x 轴上方 → p > 0，PO = p

Q 在 x 轴下方 → −3/4 q + 6 < 0，QM = 3/4 q − 6

角关系变为：∠QAP = 90° → ∠QAO + ∠OAP = 90°

直角三角形 APO 中：∠OAP + ∠APO = 90°

所以 ∠QAO = ∠APO，即 ∠QAM = ∠APO

在 △OAP 和 △MAQ 中：

  ∠AOP = ∠AMQ = 90° ✓

  ∠APO = ∠QAM（上面已证）✓

  AP = AQ（等腰条件）✓

所以 △OAP ≅ △MAQ（AAS）

对应边相等：OA = MQ，OP = AM

  OA = 3 → MQ = 3 → 3/4 q − 6 = 3 → q = 12

  OP = AM → p = q + 3 = 15

所以 P(0, 15)

注意情况二的全等对应关系和情况一不同！情况一是 △APO ≅ △QMA，情况二是 △OAP ≅ △MAQ。顶点的顺序变了，因为 P 和 Q 上下交换了位置。

第 (4) 问答案：P(0, −7) 或 P(0, 15)

六、易错提醒

易错 1：OA 当成坐标
A 的坐标是 (−3, 0)，但 OA 是长度 = 3，不是 −3。OD = 8/3 × 3 = 8，不是 8/3 × (−3) = −8。
易错 2：平行四边形 C 点搞反
AB 和 AD 是邻边，C 在 B 和 D 的"对面"。C = B + AD方向 = (0+11, 6+0) = (11, 6)。如果写成 C = D + AB方向 也能得到同样的结果，但方向不能搞反。
易错 3：等腰梯形忘了排除平行四边形
t = 5/2 和 t = 11/2 都满足 |AB| = |HE|，但 t = 5/2 时四边形是平行四边形（两组对边都平行），不是梯形。必须排除。
易错 4：全等对应关系搞混
情况一 △APO ≅ △QMA 和 情况二 △OAP ≅ △MAQ 的对应关系不同，因为 P 和 Q 在 x 轴的上下位置换了。必须重新画图找对应关系，不能照搬。
