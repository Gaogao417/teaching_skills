学生版讲解：小于 π 的正整数集合

主题标签：集合 · 正整数 · 无理数大小比较 · 列举法

原题

写出小于 π 的正整数集合。

一、先把题目拆开

已知条件上界是 π（一个无理数，约等于 3.14）
要求目标写出满足条件的集合，用列举法
关键词"正整数"——从 1 开始的整数；"小于"——不包含等于
容易忽略0 不是正整数；集合要用花括号 { } 表示

二、这题准备怎么走

  1. 先搞清楚"正整数"到底有哪些
  2. 再确定 π 在数轴上的位置（在 3 和 4 之间）
  3. 逐一检查每个正整数，看是否小于 π
  4. 用花括号写出集合
  5. 检查边界，确认没有多写或少写

三、关键想法

核心动作：不需要精确计算 π 的值，只需要知道 π 夹在哪两个整数之间。
π ≈ 3.14，所以 3 < π < 4。
这意味着正整数里比 π 小的，最大的是 3。

四、标准解法

第 1 步：写出正整数序列

正整数从 1 开始，依次为：1, 2, 3, 4, 5, 6, …

为什么：正整数就是大于 0 的整数，即 1, 2, 3, …注意 0 不是正整数。

第 2 步：定位 π 的大小

π ≈ 3.14，所以 3 < π < 4。

为什么：π 在数轴上位于 3 和 4 之间，不需要记住精确值，记住这个范围就够了。

第 3 步：逐一筛选

检查每个正整数：

1 < π ✓   →   保留

2 < π ✓   →   保留

3 < π ✓   →   保留

4 > π ✗   →   排除

5, 6, … 都大于 π   →   全部排除

为什么：从 4 开始就已经大于 π，后面的更大，不用再检查。

第 4 步：写出集合

用列举法：{1, 2, 3}

为什么：列举法要求用花括号把所有元素写出来，元素之间用逗号隔开。

{1, 2, 3}

六、易错提醒

易错点 1：把 0 放进正整数集合
正整数 = 大于 0 的整数 = {1, 2, 3, …}。0 是自然数，但不是正整数。
易错点 2：忘记写花括号
集合必须用 { } 包裹，写成"1, 2, 3"是错的，要写成 {1, 2, 3}。
易错点 3：把 4 写进去
题目要求"小于 π"，而 4 > π，所以 4 不满足条件。

七、一句话总结

做题口诀：先定元素属性（正整数？整数？），再定上界位置（夹在哪个整数之间），最后逐一列举写进花括号。
