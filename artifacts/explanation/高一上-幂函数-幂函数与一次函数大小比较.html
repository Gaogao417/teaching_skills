学生版讲解：幂函数与 y = x 的位置比较

主题：幂函数 · 不等式恒成立 · 指数函数单调性

原题

幂函数 y = xs 在 x > 1 时位于 y = x 下方，求 s 的取值范围。

一、先把题目拆开

已知条件对一切 x > 1，都有 xs < x
要求目标求 s 的取值范围
关键词"位于下方"= 值更小；"恒成立"= 对所有符合条件的 x
容易忽略不等号是严格的（<），不是 ≤；x > 1 而不是 x ≥ 1

二、这题准备怎么走

  1. 先用一个具体数字（比如 x = 2）感受一下问题
  2. 把不等式 xs < x 变形——两边除以 x
  3. 利用"底数大于 1 时，指数越大值越大"这个事实得出结论
  4. 检查边界值 s = 1 是否符合要求

三、关键想法

不等式 xs < x 看起来有两个未知数，但我们可以把 x 消掉——两边除以 x，变成只和 s 有关的问题。

四、标准解法

第 1 步：用具体数字建立直觉

取 x = 2，不等式变成 2s < 2。

试试几个值：

s2s和 2 比满足？
388 > 2×
122 = 2×
1/2√2 ≈ 1.411.41 < 2✓
011 < 2✓
-11/20.5 < 2✓

看起来 s < 1 时满足，s ≥ 1 时不满足。但这是 x = 2 的情况，需要推广。

为什么：先用具体数字"看见"答案，再推广到一般情形，这样不容易迷失。

第 2 步：变形不等式

题目要求：对一切 x > 1，都有 xs < x。

因为 x > 1 > 0，两边同除以 x，不等号方向不变：

xs - 1 < 1

为什么：x > 0 所以可以放心除，不会变号。这一步把不等式从"两个未知数"变成了"一个参数 s"。

第 3 步：利用底数大于 1 的性质

把 1 写成 x0：

xs - 1 < x0

因为 x > 1，所以函数 t → xt 是递增的——指数越大，值越大。

要使 xs - 1 < x0 恒成立，只需指数满足：

s - 1 < 0   即   s < 1

为什么：底数大于 1 时，指数函数是"增函数"，值的大小关系和指数的大小关系一致，所以可以直接比较指数。

第 4 步：检查边界

当 s = 1 时，xs = x1 = x，与 y = x 重合，不是"位于下方"。

题目要求的是严格小于（<），所以 s = 1 要排除。

为什么：题目说"位于下方"，意味着严格小于；如果题意是"不超过"，答案才会包含 s = 1。

五、边讲边问

想一想 1

当 x = 2 时，不等式 xs < x 变成了什么？

想一想 2

x > 1 时，x 是正数还是负数？能不能两边同时除以 x？

想一想 3

x > 1 时，x3 和 x2 哪个大？指数变大，值怎么变？

想一想 4

如果题意改成"不超过"（≤），答案会变成什么？

六、易错提醒

易错点 1：把 xs - 1 < 1 误看成 xs - 1 < x1，得出 s - 1 < 1 即 s < 2 的错误答案。
正确做法：1 = x0，不是 x1。
易错点 2：忘记检验 s = 1 是否满足题意。s = 1 时两函数重合，不满足"位于下方"。

七、一句话总结

底数大于 1 时：幂恒小于底数 → 除一下 → 指数恒小于 0。
