学生版讲解：证明两个函数共用同一个对称中心

主题标签：中心对称 · f(a-t)+f(a+t) 验证法 · 交点对称求和

原题

f(x) = 1/x + 1/(x - 2) + 1/(x - 4) + 1

g(x) = (3·2x - 10) / (2x-2 + 1)

函数 y = f(x) 的图像是一个中心对称图形。若函数 y = f(x) 与函数 y = g(x) 的图像交点分别为 (x1,y1), (x2,y2), ..., (xm,ym)，其中 m 为正整数，求 ∑(xi + yi)。

一、先把题目拆开

已知条件f(x) 是中心对称图形；f 和 g 有 m 个交点
要求目标求所有交点坐标之和 ∑(xi + yi)
关键词"中心对称图形"、"交点"
容易忽略不要硬解交点；关键是证明 f 和 g 共用同一个对称中心

二、这题准备怎么走

  1. 猜 f(x) 的对称中心横坐标（观察分母的特殊位置）
  2. 用 f(a-t) + f(a+t) 验证 f 的中心对称，确定纵坐标
  3. 用同样的方法证明 g(x) 也关于同一个点中心对称
  4. 推出交点成对对称，从而求和

三、关键想法

不要硬解交点！两个函数如果共用同一个对称中心，那么交点也成对关于这个点对称，一对交点的 (x+y) 贡献是固定的。

任务 3：证明 g(x) 也关于同一个点中心对称

g(x) 中出现 2x-2，所以也优先围绕 x = 2 做左右对称验证。令 u = 2t。

计算 g(2 + t)

因为 2(2+t)-2 = 2t = u，22+t = 4·2t = 4u，所以：

g(2+t) = (12u - 10) / (u + 1)

g(2-t) = (12 - 10u) / (u + 1)

七、易错提醒

易错点 1：只看出横坐标中心是 2，但没算纵坐标中心。真正要算的是 f(2-t)+f(2+t) 的值。
易错点 2：误以为共同中心是 (2,0)。按当前题面应是 (2,1)。
易错点 3：硬解交点。题目要的是交点集合的对称性，不是每个交点的具体坐标。

八、一句话总结

证明共用对称中心 → 交点成对对称 → 一对贡献固定 → 乘以 m 即得总和。

九、本节核心公式

f(a-t) + f(a+t) = 2b ⟺ f(x) 关于点 (a, b) 中心对称
关于 (a,b) 对称的一对点：(x,y) 和 (2a-x, 2b-y)
一对交点的 (x+y) 贡献：2a + 2b，平均每个点贡献 a + b
教师用完整答案

f(2-t)+f(2+t) = 2，中心 (2,1)。g(2+t)+g(2-t) = 2，中心 (2,1)。一对交点贡献 [x+y]+[(4-x)+(2-y)] = 6，平均每个点 3，总和 3m。
