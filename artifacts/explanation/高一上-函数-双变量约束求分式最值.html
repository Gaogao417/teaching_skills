学生版讲解：约束条件下求分式最值

主题：消元 · 换元 · 配方求最值 | 适合高中数学

原题

正实数 a, b 满足 a + 1/(2b) = 1，求 a/b 的最大值。

一、先把题目拆开

已知条件a > 0，b > 0，且 a + 1/(2b) = 1
要求目标a/b 的最大值
关键词正实数、约束条件、最大值
容易忽略a > 0 蕴含 b > 1/2，即 b 有下界

二、这题准备怎么走

  1. 由约束条件解出 a，用 b 表示（消元）
  2. 把 a 的表达式代入 a/b，化简
  3. 换元：令 t = 1/b，把分式变成二次函数
  4. 对二次函数配方，求最大值
  5. 验证最值点是否在定义域内，代回验算

三、关键想法

两个变量 → 一个变量：题目给了一个等式，可以把其中一个变量用另一个表示出来。这样"求两个变量的分式最值"就变成了"求一个变量的函数最值"，用配方就能解决。

四、标准解法

1消元：用 b 表示 a

由 a + 1/(2b) = 1，直接解出：

a = 1 − 1/(2b)

因为 a > 0，所以 1 − 1/(2b) > 0，即 b > 1/2。

为什么：一个等式连着两个变量，就能把一个变量"消掉"，把双变量问题变成单变量问题。

2代入化简 a/b

把 a = 1 − 1/(2b) 代入目标 a/b：

a/b = (1 − 1/(2b)) / b = 1/b − 1/(2b²)

为什么：目标里也含 a，代入后目标就只含 b 了。现在的任务变成了：在 b > 1/2 的条件下，求这个式子的最大值。

3换元：令 t = 1/b

式子里到处是 1/b，不如直接令 t = 1/b：

因为 b > 1/2，所以 0 < t < 2。

目标变成：

g(t) = t − t²/2， 0 < t < 2

为什么：换元后，分式变成了简单的二次函数 t − t²/2，用配方法就能求最值。别忘了写 t 的范围！

4配方求最值

对 g(t) = t − t²/2 提取系数 −1/2：

g(t) = −1/2 · (t² − 2t)

对括号内配方：t² − 2t = (t − 1)² − 1

所以：

g(t) = 1/2 − (t − 1)²/2 ≤ 1/2

等号当 t = 1 时取到。

为什么：二次项系数为负，开口向下，配方后立刻看出最大值。(t−1)² ≥ 0，所以 1/2 减去一个非负数，最大就是 1/2。

5验证与回代

t = 1 在范围 (0, 2) 内 ✓

回代：b = 1/t = 1，a = 1 − 1/(2×1) = 1/2

验算：a + 1/(2b) = 1/2 + 1/2 = 1 ✓

a/b = (1/2)/1 = 1/2 ✓

为什么：求出最值点后，一定要检查它是否在合法范围内（a, b 都是正实数），并且代回原条件验算。

最终答案：a/b 的最大值为 1/2，当 a = 1/2，b = 1 时取到。

五、边讲边问

想一想 1

题目说 a 是正实数。从等式 a + 1/(2b) = 1 能推出 b 必须满足什么条件？

想一想 2

化简到 a/b = 1/b − 1/(2b²) 之后，这个式子看着不太方便。你想到怎么换元了吗？令什么等于什么？

想一想 3

配方得到 g(t) = 1/2 − (t−1)²/2 ≤ 1/2。最大值对应的 t = 1，它在 t 的取值范围 (0, 2) 内吗？如果不在会怎样？

六、易错提醒

易错点 1：忘记写 b 的范围。由 a > 0 推出 b > 1/2，换元后 t = 1/b 的范围是 (0, 2) 而非全体实数。如果最值点不在范围内，就不能取到等号。
易错点 2：换元后不写新变量的范围。令 t = 1/b 时，必须同步写出 0 < t < 2，否则后续配方求出的最值点可能不在定义域内。
易错点 3：代回验算跳过。算完 t = 1 后，必须回代检查 a, b 是否满足原条件和正实数要求。这是确保答案正确的最后一步。

七、一句话总结

消元 → 代入 → 换元 → 配方 → 验算。遇到"双变量 + 等式约束 + 求最值"，第一步永远是：用一个变量表示另一个变量。
