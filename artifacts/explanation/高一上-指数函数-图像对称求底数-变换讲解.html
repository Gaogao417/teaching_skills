学生版讲解：图像变换后求解析式

主题标签：函数图像变换 · 对称 · 平移 · 追点法求解析式

原题

已知函数 y = ax 与函数 y = 3x 的图像关于 y 轴对称，求 a。

一、先把题目拆开

已知条件y = ax 与 y = 3x 的图像关于 y 轴对称
要求目标求 a 的值
关键词"关于 y 轴对称"
容易忽略不只考偶函数，核心是"图像变换后求解析式"的通用方法

二、这题准备怎么走

  1. 在原图像上设一个点 P(t, 3t)
  2. 写出这个点变换后的新坐标
  3. 把新坐标代入新函数
  4. 整理等式，求出 a

三、关键想法

最稳的方法不是背"把 x 变成 -x"，而是：设点 → 变换坐标 → 代入新函数。

这个三步法适用于所有图像变换：对称、平移、翻折，不限于 y 轴对称。

四、标准解法

第 1 步：设原图像上一点

原图像 y = 3x 上任意一点可以怎样表示？

设它的横坐标为 t，则点为：

P（____，____）

用参数 t 表示任意点，比用具体数字更一般，能推导出通用结论。

第 2 步：写出变换后的坐标

点 P(t, 3t) 关于 y 轴对称后，横坐标和纵坐标分别怎样变？

(t, 3t) →（____，____）

关于 y 轴对称：横坐标取反，纵坐标不变。

第 3 步：代入新函数

这个新点在 y = ax 上。它的横坐标是 -t，纵坐标是 3t，所以有：

a(____) = 3t

新点在新函数的图像上，把新横坐标代入新函数应该得到新纵坐标。

第 4 步：整理求 a

整理：

a-t = 3t　　所以：(1/a)t = 3t

因此：

1/a = ____　　a = ____

五、边讲边问

想一想 1

如果两个函数关于 x 轴对称，设原图像上一点 (t, y)，对称后的新点坐标是什么？

想一想 2

如果两个函数关于原点中心对称，设原图像上一点 (t, y)，对称后的新点坐标是什么？

六、易错提醒

易错点 1：把 y 取反而不是 x 取反。"关于 y 轴对称"改的是 x 坐标，y 不动。
易错点 2：化简 a-t = 3t 时写成 -at = 3t。记住：a-t = (1/a)t，负号在指数上。

七、一句话总结

图像变换求解析式三步法：设点 → 变换坐标 → 代入新函数。

八、本节核心公式

关于 y 轴对称：(t, y) → (-t, y)
关于 x 轴对称：(t, y) → (t, -y)
关于原点中心对称：(t, y) → (-t, -y)
a-t = (1/a)t
教师用答案

P(t, 3t) → (-t, 3t)，新点在 y = ax 上：a-t = 3t，(1/a)t = 3t，所以 a = 1/3。
