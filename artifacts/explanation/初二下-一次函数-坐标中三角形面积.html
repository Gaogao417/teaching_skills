学生版讲解：三角形面积求点坐标

主题：直角三角形与面积分割  |  解析几何基础

原题

已知 A(0, 1)、B(b, 0)（b > 0）， ∠ABO = 30°， 点 P(1, p)， S△ABP = 2√3， 求 P 的坐标。

一、先把题目拆开

已知条件 A(0,1)；B(b,0) 且 b>0；∠ABO = 30°；P(1,p)；三角形 ABP 面积 = 2√3
要求目标求 P 的坐标，即求 p 的值
有几个未知数b 和 p，两个未知数需要两个条件
观察关键A(0,1) 在 y 轴上，B(b,0) 在 x 轴上，P(1,p) 在直线 x=1 上
容易忽略A、B 分别在两条坐标轴上&arr; ∠AOB = 90°

二、这题准备怎么走

  1. A 在 y 轴、B 在 x 轴&arr; ∠AOB = 90°，直角三角形中用 tan 求 b
  2. 求出 B 的坐标后，求直线 AB 的解析式
  3. 求 AB 与 x = 1 的交点 M
  4. 用 S = ½ · PM · |xA - xB| 列方程求 p

三、关键想法

A 在 y 轴，B 在 x 轴&arr; ∠AOB = 90°
坐标轴互相垂直。A 在 y 轴上意味着 OA 沿 y 轴方向，B 在 x 轴上意味着 OB 沿 x 轴方向，所以 OA ⊥ OB。
直角三角形中，角度直接给出边长比：
tan∠ABO = OA / OB = 1 / b = tan30° = √3/3
不需要向量公式，直接一步出结果。
P 在直线 x = 1 上&arr; 找 AB 与 x = 1 的交点 M，做面积分割：
S△ABP = S△APM + S△BPM = ½ · PM · |xA - xB|
因为 A、B 分居 x = 1 两侧，两个小三角形的高都是 PM，底合起来恰好是 |xA - xB|。

四、标准解法

第 1 步：观察直角

A(0,1) 在 y 轴上，B(b,0) 在 x 轴上，O(0,0) 是原点。

OA 沿 y 轴方向，OB 沿 x 轴方向，两条坐标轴互相垂直，所以 ∠AOB = 90°

即 △AOB 是以 O 为直角顶点的直角三角形。

为什么先看这个：一旦发现是直角三角形，就可以用 tan 直接求边长，跳过向量公式。

第 2 步：用 tan30° 求 b

在直角 △AOB 中：
OA = 1（A 到原点的距离）
OB = b（B 到原点的距离）
∠ABO = 30°

tan30° = OA / OB = 1/b

√3/3 = 1/b

b = √3

所以 B(√3,\; 0)

为什么用 tan：在直角三角形中，tanα = 对边/邻边，直接把角度变成边长比，一步到位。

第 3 步：求直线 AB 的解析式

AB 过 A(0, 1) 和 B(√3,\; 0)。

斜率 k = (0 - 1) / (√3 - 0) = -1/√3 = -√3/3

A(0,1) 在 y 轴上，截距为 1。

AB 的解析式：y = -(√3/3)x + 1
第 4 步：求 AB 与 x = 1 的交点 M

P 在直线 x = 1 上。把 x = 1 代入 AB 的方程：

y = -√3/3 × 1 + 1 = 1 - √3/3

M(1,\; 1 - √3/3)

为什么找 M：M 把 AB 分成两段，这样 △ABP 可以拆成 △APM 和 △BPM，面积好算。

第 5 步：用面积分割公式列方程

A 的 x 坐标 = 0，B 的 x 坐标 = √3。因为 0 < 1 < √3，A、B 分居直线 x=1 两侧。

所以 S△APM = ½ · |0 - 1| · PM = ½ PM
S△BPM = ½ · |√3 - 1| · PM = ½(√3 - 1)PM

S△ABP = ½PM + ½(√3-1)PM = ½ · PM · √3

即 S△ABP = ½ · PM · |xA - xB|

代入已知面积和 PM：

PM = |p - (1 - √3/3)|

½ · |p - 1 + √3/3| · √3 = 2√3

|p - 1 + √3/3| = 4

为什么 A、B 在两侧很重要：只有两侧时，两个小三角形的底合起来恰好等于 |x_A - x_B|，公式才成立。

第 6 步：解绝对值方程

p - 1 + √3/3 = 4 → p = 5 - √3/3

p - 1 + √3/3 = -4 → p = -3 - √3/3

两个解均满足题意（P 可以在 AB 上方也可以在下方）。

五、边讲边问

想一想 1

A(0,1) 在哪条坐标轴上？B(b,0) 在哪条坐标轴上？∠AOB 等于多少度？

想一想 2

在直角 △AOB 中，∠ABO = 30°，OA = 1，OB = b。tan30° 等于什么？列出等式。

想一想 3

P 在直线 x = 1 上，AB 与 x = 1 的交点 M 怎么求？M 的坐标是什么？

想一想 4

S△ABP = S△APM + S△BPM。这两个小三角形的高都是什么？底分别是多少？合起来底等于什么？

想一想 5

为什么要求 A、B 分居 x = 1 两侧？如果都在同一侧，这个公式还成立吗？

六、易错提醒

易错点 1：没看出直角
A 在 y 轴、B 在 x 轴&arr; OA ⊥ OB &arr; ∠AOB = 90°。没发现这一点就无法用 tan，只好绕远路。
易错点 2：面积公式忘 ½
S = ½ · PM · |xA - xB|，不是 PM · |xA - xB|。三角形面积必须有 ½。
易错点 3：不检查 A、B 是否在 x=1 两侧
面积分割公式 S = ½ · PM · |xA - xB| 只在 A、B 分居两侧时成立。要养成验证 0 < 1 < √3 的习惯。
易错点 4：绝对值漏掉一个解
|p - 1 + √3/3| = 4，去掉绝对值后是两个方程，不是只写一个。

七、一句话总结

坐标轴上的点&arr; 直角三角形&arr; tan 求边长；
竖直线上的点&arr; 找交点 M&arr; S = ½ · PM · |xA - xB|。

最终答案

P(1,\; 5 - √3/3) 或 P(1,\; -3 - √3/3)
即 P(1, (15-√3)/3) 或 P(1, -(9+√3)/3)
