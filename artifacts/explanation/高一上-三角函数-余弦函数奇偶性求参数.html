学生版讲解：余弦函数为奇函数求参数

主题标签：三角函数 · 奇偶性 · 和角公式 · 偶部归零

原题

若 y = cos(x + φ) 为奇函数，求 sin φ 的值。

一、先把题目拆开

已知条件y = cos(x + φ) 是奇函数
要求目标求 sin φ 的值
关键词"奇函数"、"cos(x + φ)"
容易忽略奇函数要求偶函数部分必须消失；sin φ 有两个可能的值（±1）

二、这题准备怎么走

  1. 回顾：什么是奇函数？cos x 和 sin x 分别是偶函数还是奇函数？
  2. 展开 cos(x + φ) 的和角公式
  3. 把展开结果分成"偶函数部分"和"奇函数部分"
  4. 偶函数部分必须为零，求出 φ
  5. 由 φ 的值算出 sin φ

三、关键想法

偶函数 + 奇函数 = 奇函数 → 偶函数部分必须为零。

也就是说：如果一个函数由"偶函数部分"加"奇函数部分"组成，它要是奇函数，那偶函数部分必须消失。

类比：一杯水是"热的"，但它由"冷的部分 + 热的部分"组成——那冷的部分必须是零。

四、标准解法

第 1 步：确认基础知识

奇函数定义：f(-x) = -f(x)，图像关于原点对称。

cos x 是偶函数（因为 cos(-x) = cos x）。

sin x 是奇函数（因为 sin(-x) = -sin x）。

后续展开后要识别"偶部"和"奇部"，先确认基本构件。

第 2 步：展开和角公式

利用 cos(α + β) = cos α cos β - sin α sin β：

cos(x + φ) = cos x · cos φ - sin x · sin φ

展开是为了分离出偶函数项和奇函数项。

第 3 步：识别偶部和奇部

展开后的结果有两部分：

· cos x · cos φ：cos x 是偶函数，乘以常数 cos φ 还是偶函数 → 这是偶函数部分

· -sin x · sin φ：sin x 是奇函数，乘以常数 -sin φ 还是奇函数 → 这是奇函数部分

所以：cos(x + φ) = 偶函数部分 + 奇函数部分

分离偶部和奇部，是本题的核心动作。

第 4 步：偶部归零

题目说 cos(x + φ) 是奇函数。

而它由"偶部 + 奇部"组成，要是奇函数，偶部必须为零：

cos φ = 0

所以 φ = π/2 + kπ（k 为整数）。

偶函数加奇函数不可能是纯奇函数，除非偶函数部分恰好为零。

第 5 步：求 sin φ

由 cos φ = 0，得 φ = π/2 + kπ。

当 k 为偶数时：φ = π/2, 5π/2, ... → sin φ = sin(π/2) = 1

当 k 为奇数时：φ = 3π/2, 7π/2, ... → sin φ = sin(3π/2) = -1

sin φ = ±1

k 的奇偶性导致 sin φ 取两个不同的值，必须都写出来。

五、边讲边问

想一想 1

cos(-x) 等于什么？sin(-x) 等于什么？由此判断它们分别是偶函数还是奇函数。

想一想 2

展开 cos(x + π/3)，指出哪部分是偶函数、哪部分是奇函数。

想一想 3

取 φ = π/2，算一算 cos(x + π/2) = ？验证它是否真的是奇函数。

想一想 4

如果只写 sin φ = 1，会漏掉什么？为什么 sin φ = -1 也是对的？

六、易错提醒

易错点 1：展开和角公式时符号出错。cos(α + β) = cos α cos β - sin α sin β，中间是减号，不是加号。
易错点 2：算出 cos φ = 0 后只写 sin φ = 1，漏掉 sin φ = -1。φ = π/2 和 φ = 3π/2 都满足 cos φ = 0，但 sin 的值不同。
易错点 3：不知道"偶部归零"的思路，改用 f(-x) = -f(x) 两边展开再整理。虽然也能做，但步骤多、容易在符号上出错。

七、一句话总结

展开和角公式 → 分出偶部和奇部 → 偶部归零 → 得参数。

八、本节核心公式

cos(x + φ) = cos x cos φ - sin x sin φ
偶函数 + 奇函数 = 奇函数 → 偶函数部分 = 0
cos φ = 0 → φ = π/2 + kπ → sin φ = ±1
