# 结构分析：函数与坐标轴交点给坐标条件

## 原题
函数求点坐标专题中的“函数与坐标轴交点”：题目给出一次函数与 $x$ 轴、$y$ 轴交点，要求求交点坐标、反求函数解析式，或把截距坐标作为后续点坐标条件继续求点。

## 一、题目场景
- 数学对象：一次函数 $L:y=kx+b$，与 $x$ 轴交点 $A$，与 $y$ 轴交点 $B$，后续点 $P,Q$。
- 变量/参数：$x$ 截距、$y$ 截距、函数参数 $k,b$，后续点坐标。
- 函数/图形：一次函数图像、坐标轴。
- 已知条件：点在坐标轴上，或函数与某坐标轴相交。
- 要求目标：求截距点坐标、由截距反求解析式、把截距坐标传递给后续点。

## 二、核心结构
### 2.1 表层信息
- 表面考点：一次函数与坐标轴交点、代入求值、待定系数。
- 题型功能：`basic_skill`
- 是否值得完整 structural analysis：是；因为本专题要锁定“坐标轴交点怎样给求点条件”，不是只会算截距。
- 一句话问题模式：与 $x$ 轴交点给出 $y=0$，与 $y$ 轴交点给出 $x=0$，截距点还能作为后续求点的已知坐标。

### 2.2 结构表达
#### 判别条件表（概念辨析题用；不适用则写“无”）
- 必要条件：点在 $x$ 轴上等价于纵坐标为 $0$；点在 $y$ 轴上等价于横坐标为 $0$。
- 充分条件：点同时满足坐标轴零坐标条件和函数解析式。
- 常见干扰项：把 $x$ 轴、$y$ 轴对应的零坐标记反；直线过原点时误以为有两个不同截距点。
- 最短检查动作：先写 $x$ 轴点 $(a,0)$、$y$ 轴点 $(0,b)$。

#### 情景量表（应用题用；不适用则写“无”）
无

#### 命题网络（所有题型都写；简单题写简版）
- P1（题设）：一次函数 $L:y=kx+b$。
- P2（定义）：$A$ 是 $L$ 与 $x$ 轴交点，所以 $A$ 满足 $y=0$ 且在 $L$ 上。
- P3（定义）：$B$ 是 $L$ 与 $y$ 轴交点，所以 $B$ 满足 $x=0$ 且在 $L$ 上。
- P4（可推）：令 $y=0$ 求 $A$，令 $x=0$ 求 $B$。
- P5（可推）：若已知 $A,B$，可代入 $y=kx+b$ 反求 $k,b$。
- P6（可推）：截距点的某个坐标可作为后续点 $P,Q$ 的横坐标或纵坐标条件。
- R1：P1 + P2 -> P4，方法：$x$ 轴交点令 $y=0$。
- R2：P1 + P3 -> P4，方法：$y$ 轴交点令 $x=0$。
- R3：P4 + 后续函数条件 -> P6，方法：把截距坐标传递到新点并代入新函数。
- 目标：得到截距点或由截距推出的后续点坐标。

### 2.3 解题主链
```text
识别坐标轴 -> 写零坐标条件 -> 代入函数求截距 -> 需要时把截距坐标传给后续点 -> Answer
```

### 2.4 模型标签
- model_id：linear_function_intercepts_condition_transfer
- model_name：一次函数截距条件与后续坐标传递
- configuration：已知一次函数或截距点，要求截距坐标、解析式或后续点。
- 可迁移方向：截距直接求点、截距反求解析式、截距传递到另一条函数图像求点、截距三角形面积。
- 非同构边界：若直线与某坐标轴重合或平行导致交点不唯一/不存在，要单独讨论。

## 二点五、知识点/模型锚点
- 建议讲义标题：模型二：坐标轴交点与零坐标条件
- 知识点/模型名称：一次函数与坐标轴交点
- 核心公式/定理：与 $x$ 轴交点令 $y=0$；与 $y$ 轴交点令 $x=0$。
- 使用条件：题干出现“与 $x$ 轴交于”“与 $y$ 轴交于”。
- 入口信号：交点在坐标轴上，或者点坐标形如 $(a,0)$、$(0,b)$。
- 易混边界：$x$ 轴不是令 $x=0$，而是所有点纵坐标为 $0$。
- 本题如何体现：截距不仅能作为答案，还能继续提供后续点的横纵坐标。
- 可作为例题的结构层级：直接求截距、由截距反求解析式、截距传递求新点。

## 三、关键转化
- 最关键的转化：把“与坐标轴交点”翻译成一个坐标为 $0$。
- 为什么降低计算量：直接代入 $0$ 可少列方程组；$y$ 轴交点直接给出 $b$。
- 不转化时的低效路径：把交点当普通未知点同时设两个坐标，计算冗余。

## 四、标准路径骨架
1. 先做什么：判断交的是哪条坐标轴。
2. 再做什么：写出对应零坐标。
3. 建立什么关系：代入一次函数解析式。
4. 如何求解：解一元一次方程或直接读出截距。
5. 需要检查什么：是否把零坐标写反，是否出现退化截距。

## 四点五、标准完整解与验算
- 关键交点/关键量：一次函数 $y=-2x+6$ 与两轴交点。
- 面积/方程/关系式：$x$ 轴交点令 $0=-2x+6$；$y$ 轴交点令 $x=0$。
- 完整求解过程：令 $y=0$ 得 $x=3$，所以 $A(3,0)$；令 $x=0$ 得 $y=6$，所以 $B(0,6)$。
- 最终答案：$A(3,0),B(0,6)$。
- 排除值：若 $k=0$ 且 $b\ne0$，无 $x$ 轴交点；若 $b=0$，两轴交点可能同为原点。
- 退化情形：直线与坐标轴重合时交点不唯一，不作为本组基础题。
- 验算：$A,B$ 均满足函数式并分别在对应坐标轴上。
- 本题最短可靠路径：写零坐标，再代入。

## 五、出题人逻辑
- 诱导学生硬算的位置：不知道先令哪个量为 $0$，或由截距反求解析式时不先读 $b$。
- 真正的捷径：坐标轴本身已经给出一个坐标条件。
- 训练的可迁移能力：把图像语言翻译成坐标条件，并把已求坐标继续传递给后续对象。

## 六、学生卡点预测
- 读题/入手动作卡点：把 $x$ 轴交点误写成 $x=0$。
- 建模/关系入口卡点：由 $B(0,b)$ 不能快速读出截距。
- 求解/检查卡点：后续点的“横坐标等于 A 的横坐标”与“纵坐标等于 B 的纵坐标”传递混乱。

## 七、变式原则
- 核心不变量：坐标轴交点提供零坐标条件。
- 表层特征：与坐标轴交于 $A,B$，或给出截距点坐标。
- 可变维度：正向求截距、反向求解析式、截距坐标传递、截距面积。
- 深化阶梯：直接求两截距 -> 已知截距反求解析式 -> 截距作为后续点条件 -> 接入面积或不等式。
- 允许的变换：函数斜率和截距换为简单整数/分数；后续点在另一条一次函数上。
- 禁止的变换：引入坐标轴重合导致交点不唯一而不说明。
- 表征切换：解析式、截距点、坐标轴条件、后续点坐标条件。
- 包装方式：把截距点命名为 $A,B$，再说 $P$ 的横坐标等于 $A$ 的横坐标。
- 近迁移例子：求 $y=-2x+6$ 的两个截距。
- 远迁移例子：先求截距，再把截距作为后续函数上的点坐标约束。
- 反例/伪变式：只给坐标轴上一点但不涉及函数图像求点，不能训练完整链条。

## 八、计算复杂度预算
- 原题计算层级：一次代入和一元一次方程。
- 允许小步上升到：一个简单分数斜率或两个后续点代入。
- 禁止引入的计算负担：多参数方程组、复杂面积方程。
- 必须保留的可见支架：明确交哪条轴，后续点继承哪个坐标。

## 九、推荐讲题任务包
- 建议的本轮教学入口：用 $(a,0)$、$(0,b)$ 固定坐标轴点形式。
- 本题讲解目标：学生能把坐标轴交点作为求点条件而不是死记截距。
- 不要直接讲的抽象话：不要只说“截距法”。
- 必须先问的问题：这个点在哪条轴上？哪一个坐标必为 $0$？
- 关键讲解顺序：零坐标条件 -> 代入函数 -> 反求/传递。
- 最适合的具体数值例子：$y=-2x+6$ 的 $A,B$。
- 讲到哪里停下来让学生回答：写出 $A(a,0)$、$B(0,b)$ 后让学生代入。

## 十、推荐练题任务包
- 若卡在读题/入手动作，出什么题：直接求 $y=-2x+6$ 与两轴交点。
- 若卡在建模或关系入口，出什么题：已知两截距点反求 $y=kx+b$。
- 若卡在求解和检查，出什么题：截距坐标传递给 $P,Q$ 后继续求点。
- 若原题已稳，如何小步迁移：接入截距三角形面积或图像上方/下方范围。
- 若结构识别已稳，如何深化/抽象/包装：让截距先成为后续 relation 的输入。
- 禁止出的跑偏变式：坐标轴交点不唯一却不提示退化。

## 十点五、推荐图形请求包（可选）
- 是否需要图：否
- 图形类型：`function_graph`
- 用图意图：`student_explanation`
- 需要出现的对象：坐标轴和直线可手画。
- 需要突出给学生看的关系：$x$ 轴上 $y=0$，$y$ 轴上 $x=0$。
- 图中不能暗示的错误性质：不要把两截距画得都非零而忽略过原点情形。
- 图失败时的降级方案：用坐标轴点形式说明。
