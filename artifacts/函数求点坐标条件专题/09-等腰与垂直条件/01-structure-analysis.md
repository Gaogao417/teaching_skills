# 结构分析：函数求点坐标条件专题 09 · 等腰与垂直条件

## 原题
本专题研究函数求点坐标题中，题目用“等腰、垂直、等腰直角、45°关系”等几何条件限制待求点坐标。典型题面为：已知定点 $A,B$，动点 $P$ 在一次函数轨迹上，给出 $PA=PB$、$AP\perp BP$ 或相关垂直/等腰条件，求 $P$ 的坐标。

## 一、题目场景
- 数学对象：坐标点、一次函数轨迹、线段、等腰三角形、垂直关系。
- 变量/参数：动点 $P(t,kt+b)$，距离平方，向量或斜率。
- 函数/图形：一次函数图像、两点连线、垂线或等距轨迹。
- 已知条件：定点、动点轨迹、等腰/垂直条件。
- 要求目标：把等长或垂直转为方程，求所有可能点。

## 二、核心结构
### 2.1 表层信息
- 表面考点：距离公式、斜率乘积、数量积、等腰分类。
- 题型功能：`existence_or_parameter`
- 是否值得完整 structural analysis：是；等腰/垂直是几何语言给坐标方程的典型入口，且容易漏分类。
- 一句话问题模式：先把动点设在函数轨迹上，再把等腰或垂直条件写成距离平方、数量积或斜率方程。

### 2.2 结构表达
#### 判别条件表（概念辨析题用；不适用则写“无”）
- 必要条件：等腰要明确哪两边相等；垂直要明确哪两条线段或直线垂直。
- 充分条件：两段距离平方相等可推出等长；方向向量数量积为 0 可推出垂直。
- 常见干扰项：只做 $PA=PB$，漏 $PA=AB$、$PB=AB$；用斜率法漏掉竖直线。
- 最短检查动作：先列分类表或方向向量，再列方程。

#### 情景量表（应用题用；不适用则写“无”）
| 量 | 类型/单位 | 题设关系 | 未知/已知 |
|---|---|---|---|
| $A,B$ | Point2D | 定点 | 已知 |
| $P$ | Point2D | 在一次函数或水平/竖直轨迹上 | 待求 |
| 等腰条件 | Constraint | 两边相等，可能有三类 | 已知 |
| 垂直条件 | Constraint | 线段或直线垂直 | 已知 |

#### 命题网络（所有题型都写；简单题写简版）
- P1（题设）：已知定点 $A,B$ 和动点轨迹 $L$。
- P2（构造）：设 $P(t,f(t))$。
- P3（定义）：等腰条件等价于某两段距离平方相等。
- P4（定义）：垂直条件可用方向向量数量积为 0，或非竖直线斜率乘积为 $-1$。
- P5（计算状态）：得到关于 $t$ 的方程。
- P6（目标）：求候选点并判断对应条件。
- R1：P1 + P2 + P3 -> P5，方法：距离平方方程与分类讨论。
- R2：P1 + P2 + P4 -> P5，方法：数量积或斜率方程。
- R3：P5 -> P6，方法：解方程、代回轨迹、检查退化。
- 目标：让几何条件成为待求点坐标的方程来源。

### 2.3 解题主链
```text
设轨迹点 -> 选择等长/垂直表达 -> 列方程 -> 解候选点 -> 说明分类与验算
```

### 2.4 模型标签
- model_id：`coordinate_isosceles_perpendicular_condition`
- model_name：一次函数轨迹上的等腰与垂直坐标条件
- configuration：已知定点和动点轨迹，给等腰或垂直条件，要求候选点。
- 可迁移方向：等腰分类、垂直数量积、等腰直角旋转、45°候选直线。
- 非同构边界：纯几何证明题暂不入 v0；没有轨迹或筛选却要求唯一坐标的题不稳定。

## 二点五、知识点/模型锚点
- 建议讲义标题：模型九：等腰与垂直条件怎样变成坐标方程
- 知识点/模型名称：距离平方方程与垂直数量积方程
- 核心公式/定理：$PA^2=PB^2$ 表示等距；$\vec{PA}\cdot\vec{PB}=0$ 表示垂直；等腰要分三类。
- 使用条件：点坐标已设出；垂直对象明确；等腰对应边明确或可分类。
- 入口信号：“等腰三角形”“垂直”“斜率乘积”“45°”“等腰直角”。
- 易混边界：斜率法遇到竖直线要小心；等腰不一定是顶点在中垂线。
- 本题如何体现：几何形状不是画出来的，而是通过方程限制动点坐标。
- 可作为例题的结构层级：等腰分类 -> 垂直方程 -> 函数交点/垂直/等腰综合。

## 三、关键转化
- 最关键的转化：等腰 $\rightarrow$ 距离平方相等；垂直 $\rightarrow$ 数量积为 0 或斜率乘积为 $-1$。
- 为什么降低计算量：用距离平方避免根号，用数量积避免斜率不存在的漏解。
- 不转化时的低效路径：凭图形猜对称点，或只按一种等腰情况求。

## 四、标准路径骨架
1. 先做什么：把动点坐标设在给定轨迹上。
2. 再做什么：读清等腰/垂直对应对象。
3. 建立什么关系：距离平方相等、数量积为 0、或垂直斜率关系。
4. 如何求解：解一元方程，得到候选点。
5. 需要检查什么：是否漏分类、是否出现点重合、是否满足轨迹和题意。

## 四点五、标准完整解与验算
- 关键交点/关键量：定点坐标、动点参数、方向向量、距离平方。
- 面积/方程/关系式：$PA^2=PB^2$，$\vec{PA}\cdot\vec{PB}=0$，$k_1k_2=-1$。
- 完整求解过程：设点、列分类、写方程、解参数、回代坐标、说明对应边或垂直对象。
- 最终答案：候选点集，通常需要说明每个点对应哪类等腰或垂直。
- 排除值：点与定点重合导致边长为 0；斜率法分母为 0 的特殊情况。
- 退化情形：三角形面积为 0，或所谓等腰来自重合点。
- 验算：代回距离平方或数量积。
- 本题最短可靠路径：优先数量积/距离平方，少用斜率硬套。

## 五、出题人逻辑
- 诱导学生硬算的位置：让学生只看图形名称，忽略分类。
- 真正的捷径：先设点，再统一转为坐标方程。
- 训练的可迁移能力：把几何性质作为函数点坐标的约束。

## 六、学生卡点预测
- 读题/入手动作卡点：不知道等腰到底哪两条边相等。
- 建模/关系入口卡点：不会用数量积处理垂直，或用斜率漏竖直情况。
- 求解/检查卡点：解出候选后不说明对应条件，不排除退化。

## 七、变式原则
- 核心不变量：几何性质转成代数方程，动点轨迹提供参数。
- 表层特征：等腰、垂直、等腰直角、45°条件。
- 可变维度：轨迹类型、定点位置、等腰分类、垂直对象、筛选条件。
- 深化阶梯：单一等腰关系 -> 三类等腰 -> 单一垂直 -> 函数交点后垂直 -> 等腰/垂直综合验算。
- 允许的变换：换点、换函数、把斜率法改为数量积法。
- 禁止的变换：无轨迹却要求唯一点；纯几何证明模型强行入库。
- 表征切换：距离平方表、向量表、斜率条件、45°旋转走法。
- 包装方式：先求交点或截距，再用垂直/等腰条件求第三点。
- 近迁移例子：$P$ 在 $y=x$ 上，$AP\perp BP$，求 $P$。
- 远迁移例子：由等腰直角构造候选直线，再与一次函数求交点。
- 反例/伪变式：只把点坐标换大数，仍只套同一个距离公式。

## 八、计算复杂度预算
- 原题计算层级：一元一次或可因式分解的一元二次。
- 允许小步上升到：一个函数交点或截距作为前置输入。
- 禁止引入的计算负担：高次方程、复杂根式、多个动点同时未知。
- 必须保留的可见支架：明确垂直对象或等腰边，动点轨迹显性。

## 九、推荐讲题任务包
- 建议的本轮教学入口：先写“等腰三类表”或“垂直向量表”。
- 本题讲解目标：让学生知道几何条件怎样给坐标方程。
- 不要直接讲的抽象话：不要只说“利用距离公式和斜率公式”。
- 必须先问的问题：哪个点是动点？哪两条边相等或哪两条线垂直？
- 关键讲解顺序：设点 -> 分类/对象确认 -> 方程 -> 解点 -> 验算。
- 最适合的具体数值例子：$A(0,2),B(4,0),P$ 在 $y=x$ 上，$AP\perp BP$。
- 讲到哪里停下来让学生回答：写出两个向量后，让学生算数量积。

## 十、推荐练题任务包
- 若卡在读题/入手动作，出什么题：等腰三类分类题。
- 若卡在建模或关系入口，出什么题：动点在函数上，用数量积列垂直方程。
- 若卡在求解和检查，出什么题：候选点逐一说明对应等腰边或垂直对象。
- 若原题已稳，如何小步迁移：先求函数交点或截距，再接入垂直条件。
- 若结构识别已稳，如何深化/抽象/包装：等腰直角 90° 旋转或 45° 候选线，再与函数求交。
- 禁止出的跑偏变式：纯证明题、复杂圆轨迹题、无法手算的二次方程。

## 十点五、推荐图形请求包（可选）
- 是否需要图：否。
- 图形类型：`coordinate_geometry`
- 用图意图：`student_explanation`
- 需要出现的对象：定点、动点轨迹、两条线段或垂线。
- 需要突出给学生看的关系：等长用距离平方，垂直用方向向量。
- 图中不能暗示的错误性质：不要把等腰默认画成 $PA=PB$。
- 图失败时的降级方案：用分类表和向量表替代图。
