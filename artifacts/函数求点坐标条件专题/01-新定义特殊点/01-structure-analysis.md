# 结构分析：新定义特殊点给坐标条件

## 原题
函数求点坐标专题中的“新定义特殊点”：题目先定义一类点，例如“到 $x$ 轴和到 $y$ 轴距离相等的点叫做特殊点”，再要求在一次函数图像或含参函数图像上求满足定义的点坐标或参数。

## 一、题目场景
- 数学对象：坐标平面中的点 $P(x,y)$、一次函数图像、由题目定义的特殊点条件。
- 变量/参数：点坐标 $x,y$，有时含函数参数 $a,k,b$。
- 函数/图形：一次函数 $y=kx+b$；特殊点条件常来自距离、坐标和、坐标倍数、象限等文字定义。
- 已知条件：点在函数图像上；点满足新定义；可能附加象限、正负、横坐标固定或参数固定条件。
- 要求目标：求点坐标、求参数，或列出所有满足条件的候选点。

## 二、核心结构
### 2.1 表层信息
- 表面考点：一次函数图像上的点坐标、新定义阅读、绝对值或简单方程。
- 题型功能：`existence_or_parameter`
- 是否值得完整 structural analysis：是；因为关键不在求一次函数值，而在“怎样把新定义转成可联立的坐标条件”。
- 一句话问题模式：新定义条件给出坐标约束，函数图像给出轨迹，两者联立后再用筛选条件确定点或候选点。

### 2.2 结构表达
#### 判别条件表（概念辨析题用；不适用则写“无”）
- 必要条件：点必须同时满足新定义和函数图像条件。
- 充分条件：候选点代回定义、函数式、附加范围均成立。
- 常见干扰项：把“距离”直接当有符号坐标；只解一个绝对值分支；把定义条件当作固定公式而不读题。
- 最短检查动作：先写点的一般坐标，再逐字翻译定义。

#### 情景量表（应用题用；不适用则写“无”）
无

#### 命题网络（所有题型都写；简单题写简版）
- P1（题设）：点 $P(x,y)$ 在一次函数 $L:y=kx+b$ 上。
- P2（定义）：$P$ 满足题目给出的特殊点定义，可翻译为坐标约束 $G(x,y)=0$ 或若干分支条件。
- P3（题设）：题目给出象限、正负、横坐标固定、参数范围等筛选条件。
- P4（可推）：由 P1 得到轨迹方程 $y=kx+b$ 或参数化点 $P(t,kt+b)$。
- P5（可推）：由 P2 + P4 得到候选点集或参数候选集。
- P6（检查）：用 P3 筛选候选点，并代回定义和函数式。
- R1：P1 -> P4，方法：函数图像上的点代入或设参。
- R2：P2 + P4 -> P5，方法：把新定义坐标约束与函数轨迹联立。
- R3：P5 + P3 -> P6，方法：象限、范围、正负或题目附加条件筛选。
- 目标：由 P6 写出点坐标或参数；若筛选不足，保留所有候选点。

### 2.3 解题主链
```text
读新定义 -> 翻译坐标约束 -> 写函数图像条件 -> 联立求候选 -> 用象限/范围筛选 -> Answer
```

### 2.4 模型标签
- model_id：special_definition_point_on_linear_function
- model_name：新定义特殊点与函数图像联立求坐标
- configuration：已知函数轨迹和特殊点定义，要求满足条件的点或参数。
- 可迁移方向：定义从“到两轴距离相等”换成坐标和、坐标倍数、对称距离；函数可换为含参一次函数。
- 非同构边界：只给定义不限定轨迹时通常不是函数求点坐标题；定义需要二次距离或圆时会引入更高计算负担。

## 二点五、知识点/模型锚点
- 建议讲义标题：模型一：新定义特殊点与坐标约束
- 知识点/模型名称：定义翻译 + 函数图像条件联立
- 核心公式/定理：点 $(x,y)$ 到 $x$ 轴距离为 $|y|$，到 $y$ 轴距离为 $|x|$；点在 $y=kx+b$ 上等价于 $y=kx+b$。
- 使用条件：题干先定义特殊点，再说点在某条函数图像上或给出横纵坐标关系。
- 入口信号：“我们把……叫做……点”“点 $P$ 是……点”“点 $P$ 在函数图像上”。
- 易混边界：距离要取绝对值；第一象限可把 $|x|=|y|$ 化为 $x=y$，但不能在其他象限直接这样写。
- 本题如何体现：特殊点定义先变成坐标约束，再与一次函数联立。
- 可作为例题的结构层级：同结构换数、含参函数、定义包装。

## 三、关键转化
- 最关键的转化：把“特殊点”的文字定义翻译成坐标方程或分支约束。
- 为什么降低计算量：先翻译定义能直接得到 $x=y$、$x=-y$ 或简单绝对值方程，避免盲目代数试算。
- 不转化时的低效路径：只在函数上随意取点，或把定义当作图形直觉，无法保证答案完整。

## 四、标准路径骨架
1. 先做什么：写出点的一般坐标或由函数设为 $P(t,kt+b)$。
2. 再做什么：逐字把新定义转成坐标约束。
3. 建立什么关系：将定义约束与函数图像条件联立。
4. 如何求解：解一次方程、绝对值方程或简单分支方程。
5. 需要检查什么：象限、正负、参数范围、是否漏分支、是否真的在函数图像上。

## 四点五、标准完整解与验算
- 关键交点/关键量：特殊点 $P(x,y)$ 的坐标约束。
- 面积/方程/关系式：典型等轴点满足 $|x|=|y|$；若 $P$ 在第一象限，则 $x=y$。
- 完整求解过程：以 $P$ 在 $y=2x-3$ 且为第一象限等轴点为例，等轴点给 $x=y$，函数给 $y=2x-3$，联立得 $x=3,y=3$。
- 最终答案：$P(3,3)$。
- 排除值：不满足第一象限或不满足定义的分支要排除。
- 退化情形：若定义允许多个分支而题目未给筛选，必须列出所有候选点。
- 验算：代回 $y=2x-3$ 得 $3=2\cdot3-3$，且 $|3|=|3|$。
- 本题最短可靠路径：定义翻译在前，函数代入在后，最后筛选。

## 五、出题人逻辑
- 诱导学生硬算的位置：看见新名词后以为要记新公式，或看到距离就忘记绝对值。
- 真正的捷径：新定义只是坐标条件，先翻译即可。
- 训练的可迁移能力：读陌生定义、建立坐标约束、与函数轨迹联立。

## 六、学生卡点预测
- 读题/入手动作卡点：不知道新定义要先翻译，不会写点到坐标轴距离。
- 建模/关系入口卡点：不能把“点在函数上”和“点满足定义”同时列出。
- 求解/检查卡点：漏掉绝对值另一个分支，或忘记用象限筛选。

## 七、变式原则
- 核心不变量：定义条件产生坐标约束，函数图像产生轨迹，答案来自二者交集。
- 表层特征：新名词、距离/坐标关系、函数图像上的点。
- 可变维度：定义内容、函数解析式、给定横坐标/纵坐标、是否含参、筛选条件。
- 深化阶梯：同结构换数 -> 定义换表述 -> 含参函数 -> 条件包装 -> 反向构造参数。
- 允许的变换：把等轴点换为“横纵坐标和为定值的点”“纵坐标是横坐标 2 倍的点”等。
- 禁止的变换：同时引入复杂二次方程和隐藏筛选；定义不足以确定有限候选却要求唯一答案。
- 表征切换：文字定义、坐标方程、函数图像上的交点。
- 包装方式：把定义命名为“好点”“平衡点”“等轴点”，但仍要能回到坐标约束。
- 近迁移例子：一次函数上的等轴点坐标。
- 远迁移例子：含参函数的横坐标固定点满足新定义，反求参数。
- 反例/伪变式：只问“某点是否为特殊点”而不与函数图像求点结合，训练目标会变窄。

## 八、计算复杂度预算
- 原题计算层级：一次方程或一个绝对值方程。
- 允许小步上升到：两个分支的绝对值方程或一个简单含参一次方程。
- 禁止引入的计算负担：复杂二次方程、根式距离、多参数联立。
- 必须保留的可见支架：定义原文、点在函数图像上的条件、筛选条件。

## 九、推荐讲题任务包
- 建议的本轮教学入口：先讲“新定义就是坐标条件”。
- 本题讲解目标：学生能把定义、函数、筛选分成三行。
- 不要直接讲的抽象话：不要只说“数形结合”或“转化思想”。
- 必须先问的问题：这个新定义对点 $(x,y)$ 到底限制了什么？
- 关键讲解顺序：定义翻译 -> 函数设点 -> 联立 -> 筛选验算。
- 最适合的具体数值例子：$P$ 在 $y=2x-3$ 上且为第一象限等轴点。
- 讲到哪里停下来让学生回答：写出 $|x|=|y|$ 后，让学生根据象限改写。

## 十、推荐练题任务包
- 若卡在读题/入手动作，出什么题：给具体点 $P(a,-3)$，让学生翻译定义并求 $a$。
- 若卡在建模或关系入口，出什么题：点在 $y=2x-3$ 上且为第一象限等轴点。
- 若卡在求解和检查，出什么题：含参函数横坐标固定，绝对值方程产生两个参数。
- 若原题已稳，如何小步迁移：换一个新定义但仍为一次坐标约束。
- 若结构识别已稳，如何深化/抽象/包装：让学生解释为什么筛选不足时应保留候选点集。
- 禁止出的跑偏变式：只做定义判断或引入圆距离公式。

## 十点五、推荐图形请求包（可选）
- 是否需要图：否
- 图形类型：`function_graph`
- 用图意图：`student_explanation`
- 需要出现的对象：可口头说明坐标轴距离，不必出图。
- 需要突出给学生看的关系：定义条件与函数轨迹的交集。
- 图中不能暗示的错误性质：不要暗示只有第一象限一种情况。
- 图失败时的降级方案：直接使用坐标代数讲解。
