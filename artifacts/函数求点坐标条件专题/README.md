# 函数求点坐标条件专题

本专题按“题目怎样给条件”来训练函数题中的求点坐标，而不是按函数类型分类。

## 基础条件

1. 新定义特殊点：把“特殊点”定义翻译成坐标方程，再与函数图像联立。
2. 函数与坐标轴交点：x 轴交点令 y=0，y 轴交点令 x=0。
3. 两个函数交于一点：交点满足两个函数表达式同时成立，也是不等式比较的分界点。
4. 函数平移后的点：平移后斜率不变，结合“经过某点”或“与坐标轴交于某点”重建条件。
5. 对称点与中点条件：对称看坐标符号变化，中点看横纵坐标平均。
6. 横竖线段距离：水平距离看横坐标差，竖直距离看纵坐标差，点到坐标轴距离看绝对值。

## 进阶条件

7. 面积条件：把面积转化成底和高，底高通常来自横竖距离。
8. 平行四边形与特殊平行四边形：平行四边形优先看对角线中点，特殊四边形再叠加直角或等边条件。
9. 等腰与垂直条件：等腰转化为距离相等，垂直转化为斜率关系或横竖结构。
