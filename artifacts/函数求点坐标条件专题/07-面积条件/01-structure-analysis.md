# 结构分析：函数求点坐标条件专题 07 · 面积条件

## 原题
本专题研究一次函数与坐标几何中，题目用“面积相等、面积为定值、底高关系”来限制待求点坐标的题型。典型题面为：已知两点 $A,B$，动点 $P$ 在一次函数或坐标轴上，给出 $\triangle ABP$ 的面积，求 $P$ 的坐标。

## 一、题目场景
- 数学对象：坐标平面中的定点、一次函数轨迹上的动点、三角形面积。
- 变量/参数：动点 $P(t,kt+b)$ 或 $P(a,t)$，目标面积 $S$。
- 函数/图形：一次函数图像、坐标轴、以两定点为底的三角形。
- 已知条件：底边端点、动点轨迹、面积值或等面积关系。
- 要求目标：由面积条件反推出候选点坐标，并按位置或题意筛选。

## 二、核心结构
### 2.1 表层信息
- 表面考点：三角形面积、一次函数图像上点、绝对值方程。
- 题型功能：`existence_or_parameter`
- 是否值得完整 structural analysis：是；面积条件是函数求点坐标中最常见的“几何条件给方程”入口。
- 一句话问题模式：把“面积”翻译成“底边长度 $\times$ 点到直线的距离”，再把距离写成坐标差或函数表达式。

### 2.2 结构表达
#### 判别条件表（概念辨析题用；不适用则写“无”）
无。

#### 情景量表（应用题用；不适用则写“无”）
| 量 | 类型/单位 | 题设关系 | 未知/已知 |
|---|---|---|---|
| $A,B$ | Point2D | 底边端点 | 已知 |
| $P$ | Point2D | 在一次函数或坐标轴上 | 待求 |
| $S_{\triangle ABP}$ | Area | 给定面积或等面积 | 已知 |
| $h$ | 距离 | 点 $P$ 到底边所在直线的距离 | 由面积推出 |

#### 命题网络（所有题型都写；简单题写简版）
- P1（题设）：已知底边两端点 $A,B$。
- P2（题设）：动点 $P$ 在一次函数轨迹或坐标轴上。
- P3（题设）：$\triangle ABP$ 的面积为目标面积 $S$，或与另一个三角形面积相等。
- P4（可推）：底边长度与高满足 $\dfrac12\cdot |AB|\cdot h=S$。
- P5（可推）：高 $h$ 可写成坐标差的绝对值，如 $|y_P-y_0|$ 或铅垂线距离。
- P6（计算状态）：由 $P$ 的轨迹表达和 $h$ 得到关于参数 $t$ 的方程。
- P7（目标）：解出候选点集，并按位置条件、线段范围或题意筛选。
- R1：P1 + P3 -> P4，方法：三角形面积公式反求高。
- R2：P2 + P4 + P5 -> P6，方法：把函数轨迹上的点参数化，列绝对值方程。
- R3：P6 + 题设筛选 -> P7，方法：分类求解并代回检查。
- 目标：由 P1、P2、P3 经 R1-R3 推出待求点坐标。

### 2.3 解题主链
```text
定底边 A,B + 面积 S -> 高/距离 -> 动点参数方程 -> 候选点集 -> 筛选/验算
```

### 2.4 模型标签
- model_id：`vertical_line_area_inverse_point`
- model_name：面积条件反求一次函数轨迹上的动点
- configuration：已知底边、动点轨迹、面积，要求候选点坐标。
- 可迁移方向：等底等面积、坐标轴截距三角形、面积反向筛选参数。
- 非同构边界：没有动点轨迹或底边方向不可转成简单距离时，不宜强套铅垂线法。

## 二点五、知识点/模型锚点
- 建议讲义标题：模型七：面积条件怎样变成坐标方程
- 知识点/模型名称：坐标面积反求动点
- 核心公式/定理：$S=\dfrac12\cdot\text{底}\cdot\text{高}$；函数图像上点可设为 $P(t,kt+b)$。
- 使用条件：底边长度可求，高能转成横向或纵向坐标差；动点轨迹明确。
- 入口信号：题干出现“面积为”“面积相等”“底边在坐标轴上”“点在一次函数图像上”。
- 易混边界：距离是绝对值，通常有两侧候选；唯一答案必须来自筛选条件。
- 本题如何体现：面积不是最终目标，而是给待求点坐标的一条方程。
- 可作为例题的结构层级：先坐标轴底边，再一次函数轨迹，再等面积两侧分类。

## 三、关键转化
- 最关键的转化：面积条件 $\rightarrow$ 点到底边所在直线的距离 $\rightarrow$ 坐标差绝对值方程。
- 为什么降低计算量：避免点到直线距离公式，直接用横竖距离和一次函数代入。
- 不转化时的低效路径：只盯着三角形形状，或用复杂距离公式硬算。

## 四、标准路径骨架
1. 先做什么：判断哪条线段适合作底边，并求底边长度。
2. 再做什么：设动点坐标，如 $P(t,kt+b)$。
3. 建立什么关系：用面积公式列出含绝对值的距离方程。
4. 如何求解：分正负两种情况解参数。
5. 需要检查什么：候选点是否在轨迹上，是否满足位置限制，是否出现退化。

## 四点五、标准完整解与验算
- 关键交点/关键量：底边长度、点到直线的横竖距离。
- 面积/方程/关系式：$\dfrac12\cdot |AB|\cdot |y_P-y_{AB}|=S$，或铅垂线法中的 $\dfrac12\cdot\text{横向宽度}\cdot\text{竖直距离}=S$。
- 完整求解过程：设点、列面积方程、解绝对值、写候选坐标、逐一代回。
- 最终答案：通常为一个或两个候选点；若题面没有筛选，必须保留所有可能点。
- 排除值：面积为负不合法；动点不在给定轨迹或不满足位置限制要舍去。
- 退化情形：底边长度为 0，点落在底边所在直线上导致面积为 0。
- 验算：代回函数解析式和面积公式。
- 本题最短可靠路径：先底高，再参数化，最后筛选。

## 五、出题人逻辑
- 诱导学生硬算的位置：看到面积就想套一般距离公式，或只找图上同侧一点。
- 真正的捷径：把面积值反推成“距离值”，再用函数轨迹求坐标。
- 训练的可迁移能力：识别几何条件如何给函数点坐标方程。

## 六、学生卡点预测
- 读题/入手动作卡点：不知道面积条件到底限制了点的哪个坐标。
- 建模/关系入口卡点：忘记距离要取绝对值，漏掉另一侧候选。
- 求解/检查卡点：解出参数后不代回轨迹或不说明位置筛选。

## 七、变式原则
- 核心不变量：面积条件转为底高关系，动点轨迹转为参数表达。
- 表层特征：底边可在坐标轴、水平线、竖直线或一般一次函数上。
- 可变维度：面积值、底边位置、动点轨迹、筛选条件、等面积参照点。
- 深化阶梯：同底定面积 -> 一次函数轨迹 -> 等面积两侧 -> 加位置筛选 -> 接入后续平行四边形或交点条件。
- 允许的变换：换底边端点、换函数解析式、由定面积改为等面积。
- 禁止的变换：同时隐藏底边和加入复杂二次方程；没有轨迹却要求唯一点。
- 表征切换：文字条件、图像条件、函数解析式条件之间切换。
- 包装方式：用“三角形面积相等”“面积是另一个三角形的一半”等包装目标面积。
- 近迁移例子：$AB$ 在 $x$ 轴上，$P$ 在 $y=2x-1$ 上，面积为 10。
- 远迁移例子：先由面积求候选点，再把候选点作为平行四边形第四点或函数交点的输入。
- 反例/伪变式：只改变面积数字但仍只需要代入一次，不能训练“条件给坐标方程”的识别。

## 八、计算复杂度预算
- 原题计算层级：一元一次绝对值方程，至多两个候选点。
- 允许小步上升到：一个等面积参照点，或一个位置筛选。
- 禁止引入的计算负担：复杂二次方程、点到一般直线距离公式、多个未知点联立。
- 必须保留的可见支架：底边、轨迹、面积值三者至少两个要显性给出。

## 九、推荐讲题任务包
- 建议的本轮教学入口：先问“面积给的是底还是高？”
- 本题讲解目标：让学生把面积条件主动翻译成坐标方程。
- 不要直接讲的抽象话：不要只说“数形结合”“建模思想”。
- 必须先问的问题：哪条边作底最省事？这个点到这条底边的距离怎么写？
- 关键讲解顺序：底边长度 -> 高的绝对值 -> 设点 -> 方程 -> 筛选。
- 最适合的具体数值例子：$A(1,0),B(5,0),P$ 在 $y=2x-1$ 上，面积为 10。
- 讲到哪里停下来让学生回答：列出 $|2t-1|=5$ 后，让学生分两类求点。

## 十、推荐练题任务包
- 若卡在读题/入手动作，出什么题：底边在坐标轴、动点在轴上的面积反求题。
- 若卡在建模或关系入口，出什么题：动点在一次函数上，要求设 $P(t,kt+b)$。
- 若卡在求解和检查，出什么题：含位置筛选的两候选点题。
- 若原题已稳，如何小步迁移：把定面积改成与另一个三角形等面积。
- 若结构识别已稳，如何深化/抽象/包装：面积求候选点后接入平行四边形或函数交点条件。
- 禁止出的跑偏变式：无轨迹、无筛选却要求唯一点；计算过重的斜底距离题。

## 十点五、推荐图形请求包（可选）
- 是否需要图：否。
- 图形类型：`function_graph`
- 用图意图：`student_explanation`
- 需要出现的对象：坐标轴、底边、动点轨迹。
- 需要突出给学生看的关系：高是横竖距离，不是斜边。
- 图中不能暗示的错误性质：不要只画同侧一点导致漏解。
- 图失败时的降级方案：用文字说明底边与高的对应关系。
