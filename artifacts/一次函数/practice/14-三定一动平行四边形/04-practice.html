三定一动——平行四边形存在性

班级：________  姓名：________  日期：________

难度：E 档（基础巩固）  限时：25 分钟

练习说明

本节目标：掌握"三定一动"平行四边形存在性问题的核心方法——中点坐标法。

核心公式：平行四边形对角线互相平分，因此两条对角线的中点坐标相等。若对角线端点为 $(x_1,y_1)$、$(x_2,y_2)$，则中点为 $\left(\dfrac{x_1+x_2}{2},\;\dfrac{y_1+y_2}{2}\right)$。

解题口诀：三个定点，两两搭配为对角线，$C_4^2 = 3$ 种情况，每种列一组中点相等的方程，不遗漏不重复。

练习 1 直接运用中点坐标法（6 分）

在平面直角坐标系 $xOy$ 中，已知点 $A(4,0)$、$B(0,6)$、$C(2,3)$。点 $P$ 为平面内一点，以 $A$、$B$、$C$、$P$ 为顶点的四边形是平行四边形，求所有满足条件的点 $P$ 的坐标。

解题区：

点击查看提示

三个定点 $A(4,0)$、$B(0,6)$、$C(2,3)$，需要分三种情况讨论哪条线段作为对角线：

情况一：以 $AB$ 为对角线 $\;\Rightarrow\;$ 中点 $AB$ = 中点 $CP$

情况二：以 $BC$ 为对角线 $\;\Rightarrow\;$ 中点 $BC$ = 中点 $AP$

情况三：以 $AC$ 为对角线 $\;\Rightarrow\;$ 中点 $AC$ = 中点 $BP$

每种情况列两个方程：$\dfrac{x_1+x_2}{2}=\dfrac{x_3+x_4}{2}$，$\dfrac{y_1+y_2}{2}=\dfrac{y_3+y_4}{2}$，解出 $P(x,y)$。

点击查看解答步骤

情况一：以 $AB$ 为对角线

中点 $AB = \left(\dfrac{4+0}{2},\;\dfrac{0+6}{2}\right) = (2,\;3)$

中点 $CP = \left(\dfrac{2+x_P}{2},\;\dfrac{3+y_P}{2}\right)$

由中点相等：$\dfrac{2+x_P}{2}=2$，$\dfrac{3+y_P}{2}=3$

解得：$x_P = 2$，$y_P = 3$，即 $P(2,3)$。

（此时 $P$ 与 $C$ 重合，四边形退化为三角形，舍去。）

情况二：以 $BC$ 为对角线

中点 $BC = \left(\dfrac{0+2}{2},\;\dfrac{6+3}{2}\right) = (1,\;\dfrac{9}{2})$

中点 $AP = \left(\dfrac{4+x_P}{2},\;\dfrac{0+y_P}{2}\right)$

由中点相等：$\dfrac{4+x_P}{2}=1$，$\dfrac{y_P}{2}=\dfrac{9}{2}$

解得：$x_P = -2$，$y_P = 9$，即 $P(-2,9)$。

情况三：以 $AC$ 为对角线

中点 $AC = \left(\dfrac{4+2}{2},\;\dfrac{0+3}{2}\right) = (3,\;\dfrac{3}{2})$

中点 $BP = \left(\dfrac{0+x_P}{2},\;\dfrac{6+y_P}{2}\right)$

由中点相等：$\dfrac{x_P}{2}=3$，$\dfrac{6+y_P}{2}=\dfrac{3}{2}$

解得：$x_P = 6$，$y_P = -3$，即 $P(6,-3)$。

综上：满足条件的点 $P$ 的坐标为 $P(-2,9)$ 或 $P(6,-3)$。

练习 2 结合一次函数求定点后运用中点坐标法（8 分）

在平面直角坐标系 $xOy$ 中，一次函数 $y = -3x + 6$ 的图象与 $x$ 轴交于点 $A$，与 $y$ 轴交于点 $B$。

（1）求点 $A$、$B$ 的坐标；（2 分）

（2）点 $P$ 为平面内一点，以 $O$（原点）、$A$、$B$、$P$ 为顶点的四边形是平行四边形，求所有满足条件的点 $P$ 的坐标。（6 分）

解题区：

点击查看提示

第（1）问：令 $y=0$ 求 $A$（与 $x$ 轴交点），令 $x=0$ 求 $B$（与 $y$ 轴交点）。

第（2）问：三个定点为 $O(0,0)$、$A$、$B$。分三种情况讨论：

情况一：以 $OA$ 为对角线；

情况二：以 $OB$ 为对角线；

情况三：以 $AB$ 为对角线。

每种情况用中点坐标法列方程求解。

点击查看解答步骤

第（1）问：

令 $y=0$：$-3x+6=0$，$x=2$，$\therefore A(2,0)$。

令 $x=0$：$y=6$，$\therefore B(0,6)$。

第（2）问：

三个定点：$O(0,0)$、$A(2,0)$、$B(0,6)$。

情况一：以 $OA$ 为对角线

中点 $OA = (1,\;0)$，中点 $BP = \left(\dfrac{x_P}{2},\;\dfrac{6+y_P}{2}\right)$

$\dfrac{x_P}{2}=1$，$\dfrac{6+y_P}{2}=0$ $\;\Rightarrow\;$ $P(2,-6)$

情况二：以 $OB$ 为对角线

中点 $OB = (0,\;3)$，中点 $AP = \left(\dfrac{2+x_P}{2},\;\dfrac{y_P}{2}\right)$

$\dfrac{2+x_P}{2}=0$，$\dfrac{y_P}{2}=3$ $\;\Rightarrow\;$ $P(-2,6)$

情况三：以 $AB$ 为对角线

中点 $AB = (1,\;3)$，中点 $OP = \left(\dfrac{x_P}{2},\;\dfrac{y_P}{2}\right)$

$\dfrac{x_P}{2}=1$，$\dfrac{y_P}{2}=3$ $\;\Rightarrow\;$ $P(2,6)$

综上：$P(2,-6)$ 或 $P(-2,6)$ 或 $P(2,6)$。

练习 3 完整三定一动综合题（10 分）

在平面直角坐标系 $xOy$ 中，一次函数 $y = -x + 4$ 的图象与 $x$ 轴交于点 $A$，与 $y$ 轴交于点 $B$。点 $M$ 是线段 $AB$ 的中点。

（1）求点 $A$、$B$ 的坐标及点 $M$ 的坐标；（3 分）

（2）点 $P$ 为平面内一点，以 $A$、$B$、$M$、$P$ 为顶点的四边形是平行四边形，请直接写出所有满足条件的点 $P$ 的坐标。（7 分）

解题区：

点击查看提示

第（1）问：常规求交点。点 $M$ 是 $AB$ 中点，用中点坐标公式即可。

第（2）问：三个定点为 $A$、$B$、$M$。注意 $M$ 恰好是 $AB$ 中点，这一特殊关系会导致某种对角线搭配时 $P$ 与 $M$ 重合（需舍去）。

仍然分三种情况讨论对角线，注意检验每种情况得到的 $P$ 是否与已知点重合。

点击查看解答步骤

第（1）问：

令 $y=0$：$-x+4=0$，$x=4$，$\therefore A(4,0)$。

令 $x=0$：$y=4$，$\therefore B(0,4)$。

中点 $M = \left(\dfrac{4+0}{2},\;\dfrac{0+4}{2}\right) = (2,2)$。

第（2）问：

三个定点：$A(4,0)$、$B(0,4)$、$M(2,2)$。

情况一：以 $AB$ 为对角线

中点 $AB = (2,2)$，中点 $MP = \left(\dfrac{2+x_P}{2},\;\dfrac{2+y_P}{2}\right)$

$\dfrac{2+x_P}{2}=2$，$\dfrac{2+y_P}{2}=2$ $\;\Rightarrow\;$ $P(2,2)$

（$P$ 与 $M$ 重合，四边形退化，舍去。）

情况二：以 $AM$ 为对角线

中点 $AM = (3,1)$，中点 $BP = \left(\dfrac{x_P}{2},\;\dfrac{4+y_P}{2}\right)$

$\dfrac{x_P}{2}=3$，$\dfrac{4+y_P}{2}=1$ $\;\Rightarrow\;$ $P(6,-2)$

情况三：以 $BM$ 为对角线

中点 $BM = (1,3)$，中点 $AP = \left(\dfrac{4+x_P}{2},\;\dfrac{y_P}{2}\right)$

$\dfrac{4+x_P}{2}=1$，$\dfrac{y_P}{2}=3$ $\;\Rightarrow\;$ $P(-2,6)$

综上：$P(6,-2)$ 或 $P(-2,6)$。

教师备注

设计意图：三道题构成梯度递进。

  • 练习 1（E 档）：直接给出三个定点坐标，纯粹练习中点坐标法，确保方法本身掌握到位。
  • 练习 2（E 档）：需要先求一次函数与坐标轴的交点，再运用中点坐标法。与原题第(3)问结构一致，但定点更简单（含原点），降低计算负担。
  • 练习 3（E→D 档）：增加"中点"这一中间步骤，且第(2)问需要识别退化情况（$P$ 与 $M$ 重合需舍去），考查学生的检验意识。

常见错误预警：

  • 遗漏情况：只找到 1~2 个 $P$ 点，忘记 $C_4^2=3$ 的系统分类。
  • 未检验退化：未检查 $P$ 是否与已知点重合。
  • 中点公式错误：分母漏写 2 或坐标对应关系搞反。

差异化教学建议：

  • 若学生在练习 1 就出错，回到课堂讲解，重新演示中点坐标法的列方程过程。
  • 若学生完成练习 1、2 但在练习 3 忘记舍去退化情况，说明方法已掌握但检验习惯不足，需强化。
  • 若三题全部正确，可进入 D 档：引入非坐标轴上的定点或结合面积条件。
