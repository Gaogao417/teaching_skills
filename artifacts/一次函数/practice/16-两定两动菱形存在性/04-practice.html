两定两动—菱形存在性
练习说明

本练习围绕菱形存在性的分类讨论展开。核心方法：

  1. 参数化两个动点——将动点坐标表示为直线上的参数形式；
  2. 分两大类讨论——两个定点的连线是菱形的边还是对角线；
  3. 利用距离相等 + 平行条件（或对角线垂直平分）列方程组求解；
  4. 验证约束条件（如坐标的正负、是否在指定半轴上等）。

每道题后附有提示、解题步骤和答案，供你自主检查。

第 1 题（巩固：正向应用，结构不变）

在平面直角坐标系中，直线 $l_1$ 过 $A(0,0)$ 和 $B(3,3)$，直线 $l_2$ 过 $C(0,6)$ 和 $D(4,0)$。点 $M$ 在直线 $l_1$ 上，点 $N$ 在直线 $l_2$ 上。若存在以 $A$、$P(0,p)$（$p < 0$）、$M$、$N$ 为顶点的菱形，求点 $P$ 的坐标。

提示一

先写出两条直线的方程：$l_1$ 过 $(0,0)$ 和 $(3,3)$，斜率为 $1$，即 $y = x$；$l_2$ 过 $(0,6)$ 和 $(4,0)$，斜率为 $\dfrac{0-6}{4-0} = -\dfrac{3}{2}$，即 $y = -\dfrac{3}{2}x + 6$。然后参数化 $M(m, m)$ 和 $N\!\left(n,\, -\dfrac{3}{2}n + 6\right)$。

提示二

$A(0,0)$ 和 $P(0,p)$ 都在 $y$ 轴上，$AP$ 竖直。两大类：

  • AP 是边：对边 $MN$ 也竖直 $\Rightarrow$ $m = n$，用四边等长 + 对边平行列方程；
  • AP 是对角线：中点重合 + $MN \perp AP$。$AP$ 竖直 $\Rightarrow$ $MN$ 水平 $\Rightarrow$ $y_M = y_N$。
解题步骤
准备：参数化

$l_1: y = x$，$l_2: y = -\dfrac{3}{2}x + 6$。

$M(m, m)$，$N\!\left(n,\, -\dfrac{3}{2}n + 6\right)$。

$|AP| = |p|$（$p < 0$，所以 $|AP| = -p$）。

情况一：AP 是菱形的边

$AP$ 竖直 $\Rightarrow$ 对边 $MN$ 也竖直 $\Rightarrow$ $m = n$。

$|AM| = |AP|$：$\sqrt{m^2 + m^2} = -p$，即 $2m^2 = p^2$ …… ①

$AM \parallel PN$：方向 $(m, m) \parallel \left(m,\; -\dfrac{3}{2}m + 6 - p\right)$，得

$m = -\dfrac{3}{2}m + 6 - p$，即 $p = 6 - \dfrac{5}{2}m$ …… ②

② 代入 ①：$2m^2 = \left(6 - \dfrac{5}{2}m\right)^{\!2} = 36 - 30m + \dfrac{25}{4}m^2$。

两边乘 $4$：$8m^2 = 144 - 120m + 25m^2$，即 $17m^2 - 120m + 144 = 0$。

$\Delta = 14400 - 9792 = 4608 = 2304 \times 2$，$\sqrt{\Delta} = 48\sqrt{2}$。

$m = \dfrac{120 \pm 48\sqrt{2}}{34} = \dfrac{60 \pm 24\sqrt{2}}{17}$。

  • $m = \dfrac{60 + 24\sqrt{2}}{17} \Rightarrow p = 6 - \dfrac{5}{2} \cdot \dfrac{60 + 24\sqrt{2}}{17} = \dfrac{-48 - 60\sqrt{2}}{17} < 0$ ✓
  • $m = \dfrac{60 - 24\sqrt{2}}{17} \Rightarrow p = \dfrac{-48 + 60\sqrt{2}}{17} \approx 2.17 > 0$，舍去。
情况二：AP 是菱形的对角线

$AP$ 竖直 $\Rightarrow$ $MN$ 水平 $\Rightarrow$ $y_M = y_N$，即 $m = -\dfrac{3}{2}n + 6$ …… ①

中点重合：$AP$ 中点 $\left(0, \dfrac{p}{2}\right)$，$MN$ 中点 $\left(\dfrac{m+n}{2},\; \dfrac{y_M + y_N}{2}\right)$。

$x$ 方向：$\dfrac{m+n}{2} = 0 \Rightarrow n = -m$ …… ②

② 代入 ①：$m = \dfrac{3}{2}m + 6 \Rightarrow -\dfrac{1}{2}m = 6 \Rightarrow m = -12$，$n = 12$。

$y$ 方向中点：$\dfrac{p}{2} = \dfrac{-12 + (-12)}{2} = -12 \Rightarrow p = -24 < 0$ ✓

验证：$M(-12,-12)$，$N(12,-12)$，$|AM|^2 = 288$，$|AN|^2 = 288$，$|MN| = 24$，$|AP| = 24$。✓

教师备注（E档）：本题与原题结构高度一致：两个定点在 $y$ 轴上（$A$ 和 $P$），两个动点分别在两条直线上。重点观察学生是否：(1) 主动分两大类讨论；(2) 在情况一中同时使用距离相等和平行条件（而非只用距离）；(3) 在情况二中正确利用"竖直对角线 ⇒ 水平对角线"简化计算；(4) 最后验证 $p < 0$ 舍去不符合条件的解。

答案：$P\!\left(0,\; \dfrac{-48-60\sqrt{2}}{17}\right)$ 或 $P(0,\,-24)$

第 2 题（变式：换情境，结构不变）

在平面直角坐标系中，$A(-2, 0)$、$B(2, 0)$、$C(0, -4)$、$D(0, 2)$。直线 $AC$ 与直线 $BD$ 交于点 $P$。点 $M$ 在直线 $PA$ 上，点 $N$ 在直线 $PB$ 上。若存在以 $C$、$E(0, e)$（$e > 0$）、$M$、$N$ 为顶点的菱形，求 $e$ 的值。

提示一

先求直线方程。$AC$ 过 $(-2,0)$ 和 $(0,-4)$，斜率 $= -2$，即 $y = -2x - 4$。$BD$ 过 $(2,0)$ 和 $(0,2)$，斜率 $= -1$，即 $y = -x + 2$。联立求 $P$ 坐标，再参数化 $M$、$N$。

提示二

$C(0,-4)$ 和 $E(0,e)$ 都在 $y$ 轴上 $\Rightarrow$ $CE$ 竖直。两大类：

  • CE 是边：$MN$ 也竖直 $\Rightarrow$ $x_M = x_N$，用距离 + 平行列方程；
  • CE 是对角线：中点重合 + $MN$ 水平。解出后必须验证四边是否真的相等！
解题步骤
准备：求关键坐标和参数化

$AC: y = -2x - 4$，$BD: y = -x + 2$。

联立：$-2x - 4 = -x + 2 \Rightarrow x = -6$，$y = 8$。$P(-6, 8)$。

$PA$ 即 $AC$：$y = -2x - 4$，参数化 $M(m,\, -2m - 4)$。

$PB$ 即 $BD$：$y = -x + 2$，参数化 $N(n,\, -n + 2)$。

$|CE| = e - (-4) = e + 4$。

情况一：CE 是菱形的边

$CE$ 竖直 $\Rightarrow$ $MN$ 竖直 $\Rightarrow$ $m = n$。

$|CM| = |CE|$：$m^2 + (-2m)^2 = (e+4)^2$，即 $\boxed{5m^2 = (e+4)^2}$ …… ①

$CM \parallel EN$：方向 $(m, -2m) \parallel (m, -m + 2 - e)$，得 $-2m = -m + 2 - e$，即 $\boxed{e = m + 2}$ …… ②

② 代入 ①：$5m^2 = (m + 6)^2 = m^2 + 12m + 36 \Rightarrow 4m^2 - 12m - 36 = 0 \Rightarrow m^2 - 3m - 9 = 0$。

$m = \dfrac{3 \pm \sqrt{9 + 36}}{2} = \dfrac{3 \pm 3\sqrt{5}}{2}$。

  • $m = \dfrac{3 + 3\sqrt{5}}{2} \Rightarrow e = \dfrac{7 + 3\sqrt{5}}{2} \approx 6.85 > 0$ ✓
  • $m = \dfrac{3 - 3\sqrt{5}}{2} \approx -1.85 \Rightarrow e = \dfrac{7 - 3\sqrt{5}}{2} \approx 0.15 > 0$ ✓
情况二：CE 是菱形的对角线

$CE$ 竖直 $\Rightarrow$ $MN$ 水平 $\Rightarrow$ $y_M = y_N$，即 $-2m - 4 = -n + 2 \Rightarrow n = 2m + 6$ …… ①

中点重合（$x$ 方向）：$\dfrac{m + n}{2} = 0 \Rightarrow n = -m$ …… ②

①② 联立：$2m + 6 = -m \Rightarrow 3m = -6 \Rightarrow m = -2$，$n = 2$。

中点（$y$ 方向）：$\dfrac{-4 + e}{2} = \dfrac{-2(-2)-4 + (-2+2)}{2} = \dfrac{0 + 0}{2} = 0 \Rightarrow e = 4$。

验证四边等长：$M(-2, 0)$，$N(2, 0)$，$C(0, -4)$，$E(0, 4)$。

$|CM|^2 = 4 + 16 = 20$，$|MN| = 4$，$|NE|^2 = 4 + 16 = 20$。$|CM| = 2\sqrt{5} \neq 4 = |MN|$。不是菱形，舍去。

教师备注（E档）：本题将"抛物线情境"简化为直接给坐标，但核心结构完全一致：$CE$ 在 $y$ 轴上（竖直），动点 $M$、$N$ 分别在两条直线上。关键陷阱在情况二：对角线条件（中点重合 + 垂直）只能保证四边形是"筝形"，不一定满足四边相等。若学生忘记验证就直接写出答案，说明对菱形与筝形的区别理解不足。

答案：$e = \dfrac{7 + 3\sqrt{5}}{2}$ 或 $e = \dfrac{7 - 3\sqrt{5}}{2}$

第 3 题（迁移：三定一动变体）

在平面直角坐标系中，$A(0, 0)$、$B(4, 0)$、$C(2, 3)$。点 $D$ 在直线 $y = 2x - 1$ 上。是否存在以 $A$、$B$、$C$、$D$ 为顶点的菱形？若存在，求点 $D$ 的坐标；若不存在，请说明理由。

提示一

先计算已知点之间的距离：$|AB| = 4$，$|AC| = |BC| = \sqrt{13}$。注意 $|AC| = |BC|$，但 $|AB| \neq |AC|$。

菱形四边等长。如果某两个已知点是菱形的邻边顶点，则边长等于它们的距离。$|AB| = 4$，$|AC| = |BC| = \sqrt{13}$——边长只能是 $4$ 或 $\sqrt{13}$。

提示二

枚举策略：先试"某两个已知点的连线是菱形的边"（需要边长相等），再试"某两个已知点的连线是菱形的对角线"（中点重合 + 垂直平分）。设 $D(d,\, 2d - 1)$，对每种情况列方程并验证。

注意：四个顶点的排列顺序可以不同，需要系统枚举所有可能的排列。

解题步骤
准备：已知距离

$|AB| = 4$，$|AC| = \sqrt{2^2 + 3^2} = \sqrt{13}$，$|BC| = \sqrt{2^2 + 3^2} = \sqrt{13}$。

设 $D(d,\, 2d - 1)$。

枚举：两个已知点的连线是边

AB 是边，边长 $= 4$：$|AC| = \sqrt{13} \neq 4$，所以 $C$ 不能是以 $A$、$B$ 为邻边的菱形的顶点。排除。

AC 是边，边长 $= \sqrt{13}$：$|AB| = 4 \neq \sqrt{13}$，排除。

BC 是边，边长 $= \sqrt{13}$：同理，排除。

枚举：两个已知点的连线是对角线

AB 是对角线：$AB$ 中点 $(2, 0)$。$CD$ 是另一条对角线 $\Rightarrow$ 中点也是 $(2, 0)$，且 $CD \perp AB$。$AB$ 水平 $\Rightarrow$ $CD$ 竖直 $\Rightarrow$ $x_D = x_C = 2$。$D(2, 3) = C$，与 $C$ 重合，舍去。

AC 是对角线：$AC$ 中点 $(1,\, 1.5)$。$BD$ 中点 $\left(\dfrac{4+d}{2},\; \dfrac{2d-1}{2}\right)$。$\dfrac{4+d}{2} = 1 \Rightarrow d = -2$，但 $\dfrac{2d-1}{2} = 1.5 \Rightarrow d = 2$，矛盾。无解。

BC 是对角线：$BC$ 中点 $(3,\, 1.5)$。$\dfrac{d}{2} = 3 \Rightarrow d = 6$，但 $\dfrac{2d-1}{2} = 1.5 \Rightarrow d = 2$，矛盾。无解。

继续枚举顶点排列顺序

上述排除了"两个已知点相邻"和"两个已知点是对角"的标准情况。还需要考虑其他顶点排列。

排列 $A$-$C$-$B$-$D$：$|AC| = |CB| = \sqrt{13}$（已满足两条邻边等长），需 $|BD| = |DA| = \sqrt{13}$。

$|BD|^2 = (d-4)^2 + (2d-1)^2 = 5d^2 - 12d + 17 = 13 \Rightarrow 5d^2 - 12d + 4 = 0$。

$d = \dfrac{12 \pm \sqrt{144 - 80}}{10} = \dfrac{12 \pm 8}{10}$，$d = 2$ 或 $d = \dfrac{2}{5}$。

$d = 2 \Rightarrow D(2, 3) = C$，舍去。$d = \dfrac{2}{5} \Rightarrow |DA|^2 = \dfrac{4}{25} + \dfrac{1}{25} = \dfrac{1}{5} \neq 13$，不成立。

其他排列（$A$-$B$-$C$-$D$、$A$-$B$-$D$-$C$、$A$-$C$-$D$-$B$、$A$-$D$-$C$-$B$）经逐一验证，均因边长不匹配而无解。

教师备注（E档）：本题从"两定两动"迁移到"三定一动"，枚举维度增加。学生容易犯的错误：(1) 只试了一两种排列就下结论"不存在"；(2) 在"AB 是对角线"时没有发现 $D$ 与 $C$ 重合；(3) 在 $A$-$C$-$B$-$D$ 排列中，满足 $|BD| = \sqrt{13}$ 后忘记验证 $|DA| = \sqrt{13}$。若学生能系统地枚举所有情况并正确判断"不存在"，说明已掌握菱形存在性问题的完整分析框架。

答案：不存在满足条件的菱形。

答案汇总

教学交接说明

当前档位：E 档（基础入门）

练习目标：巩固"两定两动菱形存在性"的参数化 + 两大类讨论 + 列方程 + 验证约束的完整流程。

第 1 题：与原题结构高度一致，检验学生能否独立完成两大类讨论。重点观察是否遗漏"对角线"情况，以及是否忘记验证 $p < 0$。

第 2 题：情境变化（给坐标而非抛物线），核心结构不变。关键陷阱：情况二对角线条件解出 $e = 4$ 后，验证发现不是菱形（筝形）。若学生能正确舍去，说明理解了菱形与筝形的区别。

第 3 题：从两定两动迁移到三定一动，枚举维度增加。答案为"不存在"，考验学生是否系统地枚举了所有排列，而非过早下结论。

升级条件：若学生三题全部独立完成（包括正确舍去不合条件的解、验证筝形与菱形的区别）→ 升至 F 档（引入正方形存在性或旋转类菱形）。

降级信号：若学生在第 1 题就只写出一种情况（漏掉"对角线"类）→ 回到讲解阶段，强化"两大类讨论"的框架意识。
