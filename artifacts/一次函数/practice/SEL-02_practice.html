自适应练习：用待定系数法求一次函数解析式

教师判断

当前档位：D档。

判断依据：初二学生，已掌握一次函数概念，本次教学目标是掌握待定系数法基本步骤。学生应能理解"设→代→列→解→写"的流程，主要风险在于代入时写错位置、解方程组时正负号出错。因此练习侧重巩固流程，同时做轻微迁移（换条件表述方式），不增加新的知识点。

练习说明

本组共 3 题，只练一个动作：用待定系数法求一次函数解析式。每题都按"设→代→列→解→写"五步走，做完后对照参考答案检查。

第1题（巩固·基本代入）

已知一个一次函数的图像经过点 (1, 5) 和点 (4, −4)。求这个函数的解析式。

训练目标：独立完成"设→代→列→解→写"全流程。

预期卡点：代入时 y 值正负号弄错（−4 容易漏掉负号），解方程组时减法正负号出错。

提示一

先设 y = kx + b（k ≠ 0），然后把两个点分别代入，看看能得到什么方程。

提示二

代入 (1, 5)：k + b = 5……①；代入 (4, −4)：4k + b = −4……②。用 ② − ① 消去 b。

第2题（巩固·换条件表述）

已知一次函数 y = kx + b 的图像经过点 (0, 3)，且当 x = 2 时，y = −1。求这个函数的解析式。

训练目标：适应不同的条件表述方式——一个点给坐标，一个用"当x=…时y=…"给出。

预期卡点：注意到 x = 0 时 y = 3 其实就是 b = 3（y轴截距），但不会直接利用，还是老老实实列方程组。

提示一

"图像经过点 (0, 3)"和"当 x = 0 时 y = 3"是同一件事。把 (0, 3) 和 (2, −1) 分别代入 y = kx + b。

提示二

代入 (0, 3)：0 × k + b = 3，所以 b = 3。直接代入 b = 3，再由 (2, −1) 求 k：2k + 3 = −1。

第3题（轻微迁移·理解截距）

已知一次函数 y = kx + b 的图像与 y 轴交于点 (0, −2)，且经过点 (1, 4)。求 k 的值，并写出解析式。

训练目标：识别"与y轴交于(0, b)"就是 b 的值，能省一步直接用。

预期卡点：不认识"与y轴交于(0, −2)"的含义，或者知道 b = −2 但在后续计算中又忘记负号。

提示一

图像与 y 轴的交点，x 坐标一定是 0。所以交点是 (0, −2)，代入 y = kx + b 可得 b = ?

提示二

b = −2。把 b = −2 和点 (1, 4) 代入 y = kx − 2，得 k − 2 = 4，所以 k = 6。

参考答案与升级/降级判断

第1题

设 y = kx + b（k ≠ 0）

代入 (1, 5)：k + b = 5   ……①

代入 (4, −4)：4k + b = −4  ……②

② − ①：3k = −9，k = −3

代入①：−3 + b = 5，b = 8

解析式：y = −3x + 8

升级判断：若学生过程完整、答案正确且检验通过，说明 D 档巩固到位，可升级到 E 档（做迁移变式）。

降级判断：若代入出错（把 y = −4 写成 4）或解方程组正负号算错，退回 C 档，重新做"代入列方程"的专项练习。

第2题

设 y = kx + b（k ≠ 0）

代入 (0, 3)：b = 3  （捷径：与 y 轴交点的 y 值就是 b）

代入 (2, −1)：2k + 3 = −1，2k = −4，k = −2

解析式：y = −2x + 3

升级判断：若学生发现 b = 3 可以直接得出，说明对截距有初步理解，进入第3题不会困难。

降级判断：若学生没有识别出 b = 3 的捷径，而是列完整方程组（不影响正确性），则保持 D 档。

第3题

与 y 轴交于 (0, −2)，所以 b = −2

代入 (1, 4)：k + (−2) = 4，k = 6

解析式：y = 6x − 2

升级判断：若学生正确识别"与 y 轴交点"的含义并快速求出解析式，说明已掌握 E 档能力，下次可做结构隐藏型变式（如"已知函数图像经过某两点且 k > 0，求解析式"）。

降级判断：若学生对"与 y 轴交于"不理解，退回 D 档，先复习 y 轴上点的坐标特征。

下一轮建议：根据三题完成情况更新学生画像。若三题全对 → E档变式；若第1题出错 → C档专项；若第2、3题出错 → 回到讲解页补充截距概念。
