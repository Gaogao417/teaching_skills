自适应练习：两定两动菱形存在性

教师判断

当前档位：E 档（会做原题，但需要迁移巩固）

判断依据：学生已掌握平行四边形存在性，理解坐标几何基本工具（斜率、距离公式、参数化）。通过讲解阶段学习了"两定两动菱形存在性"的两大类讨论框架（QC 是边 vs QC 是对角线），但尚未独立完成过此类问题。需要同源平级变式来巩固"参数化 → 分大类 → 列方程 → 验证约束"的流程。

练习说明

本组共 3 题，每题都练习同一个动作：参数化两个动点 → 分"两个定点是邻边"和"两个定点是对角线"两大类 → 每类列方程求待定坐标 → 验证约束条件。

请按顺序完成，每题写出完整的参数化、分类讨论、列式和验证过程。

第 1 题（巩固：正向应用，结构不变）

在平面直角坐标系中，直线 l1 过 A(0, 0) 和 B(3, 3)，直线 l2 过 C(0, 6) 和 D(4, 0)。点 M 在直线 l1 上，点 N 在直线 l2 上。若存在以 A、P(0, −2)、M、N 为顶点的菱形，求点 P 的坐标（P 在 y 轴负半轴上）。

训练目标：独立完成两大类讨论，每步写出"AP 是边"和"AP 是对角线"的列式过程。

预期卡点：① 漏掉"AP 是对角线"的情况；② 参数化 M、N 时代入错误；③ 忘记验证 P 在 y 轴负半轴（y < 0）。

提示一

先写出两条直线的方程：l1 过 (0,0) 和 (3,3) → y = x；l2 过 (0,6) 和 (4,0) → y = −(3/2)x + 6。然后参数化 M(m, m) 和 N(n, −3n/2 + 6)。

提示二

两大类：(a) AP 是边 → AP 竖直 → MN 也竖直 → m = n，四边等长 + 对边平行列方程；(b) AP 是对角线 → 中点重合 + 垂直平分。注意 A(0,0) 和 P(0,p) 都在 y 轴上，AP 竖直。

答题区域

第 2 题（变式：换情境，结构不变）

抛物线 y = x² − 4 与 x 轴交于 A(−2, 0) 和 B(2, 0)，与 y 轴交于 C(0, −4)。直线 AC 与直线 BD（D(0, 2)）交于点 P。点 M 在直线 PA 上，点 N 在直线 PB 上。是否存在以 C、E(0, e)（e > 0）、M、N 为顶点的菱形？若存在，求 e 的值。

训练目标：将同样的分类讨论方法应用到新图形（抛物线）上，体会方法的通用性。

预期卡点：虽然图形变了，但核心方法完全一样。学生可能被"抛物线"干扰，但实际上只需要相关直线和关键点的坐标。

提示一

先求直线 AC 方程：过 (−2,0) 和 (0,−4) → y = −2x − 4。直线 BD 过 (2,0) 和 (0,2) → y = −x + 2。联立求 P 坐标。然后参数化 M、N。

提示二

C(0,−4) 和 E(0,e) 都在 y 轴上 → CE 竖直。两大类：(a) CE 是边 → MN 竖直 → m = n；(b) CE 是对角线 → MN 水平。

答题区域

第 3 题（提升：结构隐藏——三个定点 + 一个动点变体）

在平面直角坐标系中，A(0, 0)、B(4, 0)、C(2, 3)。点 D 在直线 y = 2x − 1 上。是否存在以 A、B、C、D 为顶点的菱形？若存在，求点 D 的坐标。

训练目标：从"两定两动"迁移到"三定一动"的菱形存在性，体会分类策略的变化。

预期卡点：① 三个定点中选哪两个作为菱形的"边"或"对角线"的选择变多了；② 需要先判断哪两个定点的连线可以作为菱形的边或对角线（距离必须合理）；③ 参数化只有一个变量，但约束条件可能更复杂。

提示一

先算 AB、BC、AC 的长度：|AB|=4，|BC|=√13，|AC|=√13。注意 |BC|=|AC|！这意味着 A 和 B 关于直线 x=2 不对称，但 C 到 A 和 B 的距离相等。菱形四边等长 → 边长必须等于某条已知线段的长度。

提示二

枚举：哪两个已知点的连线是菱形的边？由于 |BC|=|AC|=√13 而 |AB|=4，边长只能是 √13 或 4。试 (a) AB 是边，边长=4；(b) AC 是边，边长=√13；(c) BC 是边，边长=√13。每种情况用距离相等 + D 在直线上列方程。

答题区域

参考答案与升级/降级判断

第 1 题

已知：A(0, 0)、P(0, p)，p < 0
l1: y = x，l2: y = −(3/2)x + 6
M(m, m)，N(n, −3n/2 + 6)
|AP| = |p|（p < 0 所以 = −p）

情况一：AP 是菱形的边

AP 竖直 → MN 竖直 → m = n

|AM| = |AP| → √(m² + m²) = −p → 2m² = p² …… ①

|PN| = |AP| → √(m² + (−3m/2 + 6 − p)²) = −p

→ m² + (−3m/2 + 6 − p)² = p² …… ②

AM ∥ PN：方向 (m, m) ∥ (m, −3m/2 + 6 − p)

→ m / m = m / (−3m/2 + 6 − p) → −3m/2 + 6 − p = m → p = 6 − 5m/2 …… ③

③ 代入 ①：2m² = (6 − 5m/2)² = 36 − 30m + 25m²/4

两边乘 4：8m² = 144 − 120m + 25m²

17m² − 120m + 144 = 0

Δ = 14400 − 9792 = 4608 = 2304 × 2

m = (120 ± 48√2) / 34 = (60 ± 24√2) / 17

  • m = (60 + 24√2)/17 → p = 6 − 5(60 + 24√2)/(34) = 6 − (300 + 120√2)/34 = (204 − 300 − 120√2)/34 = (−96 − 120√2)/34 = (−48 − 60√2)/17 < 0 ✓
  • m = (60 − 24√2)/17 → p = (−48 + 60√2)/17 ≈ (−48 + 84.85)/17 ≈ 2.17 > 0 舍去

情况二：AP 是对角线

AP 竖直 → MN 水平 → yM = yN → m = −3n/2 + 6 …… ①

中点重合：AP 中点 (0, p/2)，MN 中点 ((m+n)/2, (m−3n/2+6)/2)

(m + n)/2 = 0 → n = −m …… ②

② 代入 ①：m = 3m/2 + 6 → −m/2 = 6 → m = −12

y 中点：p/2 = (−12 − 3×12/2 + 6)/2 = (−12 − 18 + 6)/2 = −24/2 = −12

→ p = −24 < 0 ✓

验证四边等长：M(−12, −12)，N(12, −12)，|AM|² = 144+144 = 288，|AN|² = 144+144 = 288，|MN|² = 576，|AP|² = 576。✓

第 1 题答案：P(0, (−48−60√2)/17) 或 P(0, −24)

升级判断：若学生两类都独立完成 → 升至 F 档。若情况二的"中点重合+垂直平分"列式出错 → 保持在 E 档，重点复习对角线条件。

第 2 题

已知：A(−2, 0)、B(2, 0)、C(0, −4)、D(0, 2)

AC: 过 (−2,0) 和 (0,−4) → 斜率 = −4/2 = −2 → y = −2x − 4

BD: 过 (2,0) 和 (0,2) → 斜率 = (2−0)/(0−2) = −1 → y = −x + 2

联立求 P：−2x − 4 = −x + 2 → x = −6, y = 8 → P(−6, 8)

PA 过 P(−6,8) 和 A(−2,0) → 斜率 = (0−8)/(−2+6) = −2 → 就是 AC：y = −2x − 4

PB 过 P(−6,8) 和 B(2,0) → 斜率 = (0−8)/(2+6) = −1 → 就是 BD：y = −x + 2

参数化：M(m, −2m − 4) 在 PA 上，N(n, −n + 2) 在 PB 上

C(0, −4)，E(0, e)，|CE| = e + 4

情况一：CE 是菱形的边

CE 竖直 → MN 竖直 → m = n

|CM| = |CE| → m² + (−2m)² = (e + 4)² → 5m² = (e + 4)² …… ①

|EN| = |CE| → m² + (−m + 2 − e)² = (e + 4)² …… ②

CM ∥ EN：方向 (m, −2m) ∥ (m, −m + 2 − e) → −2m = −m + 2 − e → e = 2 + m …… ③

③ 代入 ①：5m² = (6 + m)² = 36 + 12m + m² → 4m² − 12m − 36 = 0 → m² − 3m − 9 = 0

m = (3 ± √45)/2 = (3 ± 3√5)/2

  • m = (3 + 3√5)/2 → e = (7 + 3√5)/2 ≈ 6.85 > 0 ✓
  • m = (3 − 3√5)/2 ≈ −1.85 → e = (7 − 3√5)/2 ≈ 0.15 > 0 ✓

情况二：CE 是菱形的对角线

CE 竖直 → MN 水平 → yM = yN → −2m − 4 = −n + 2 → n = 2m + 6 …… ①

中点重合：CE 中点 (0, (e − 4)/2)，MN 中点 ((m+n)/2, (−2m−4−n+2)/2)

(m + n)/2 = 0 → n = −m …… ②

①② 联立：2m + 6 = −m → 3m = −6 → m = −2

代入中点 y：e − 4 = −2(−2) − 4 − (−2) + 2 = 4 − 4 + 2 + 2 = 4 → e = 8

验证四边等长：M(−2, 0)，N(2, 0)，C(0, −4)，E(0, 8)

|CM|² = 4 + 16 = 20，|MN|² = 16，|NE|² = 4 + 64 = 68。|CM| ≠ |MN| → 不是菱形，舍去。

第 2 题答案：e = (7 + 3√5)/2 或 e = (7 − 3√5)/2

升级判断：若学生完成情况二后发现"不是菱形"并正确舍去 → 升至 F 档。若学生直接认为情况二一定成立而未验证 → 保持在 E 档。

第 3 题

已知：A(0,0)，B(4,0)，C(2,3)，D 在 y=2x−1 上

|AB| = 4，|AC| = √(4+9) = √13，|BC| = √(4+9) = √13

设 D(d, 2d−1)

枚举菱形情况（三定一动）

菱形四边等长。三个已知点中，如果某两点是菱形的相邻顶点，则边长等于它们的距离。

情况一：AB 是菱形的边，边长 = 4

C 和 D 分别在 AB 的两侧。AC = √13 ≠ 4 → C 不是以 A、B 为邻边的菱形的顶点。排除。

情况二：AC 是菱形的边，边长 = √13

|AB| = 4 ≠ √13 → B 不是以 A、C 为邻边的菱形的顶点。排除。

情况三：BC 是菱形的边，边长 = √13

同上，|AB| ≠ √13。排除。

情况四：AB 是菱形的对角线

AB 中点 (2, 0)。CD 是另一条对角线 → 中点也是 (2, 0)，且 CD ⊥ AB。

AB 水平 → CD 竖直 → D 的 x 坐标 = C 的 x 坐标 = 2

D(2, 2×2−1) = (2, 3) = C！与 C 重合，舍去。

情况五：AC 是菱形的对角线

AC 中点 (1, 1.5)。BD 是另一条对角线 → 中点也是 (1, 1.5)。

(4+d)/2 = 1 → d = −2；(0+2d−1)/2 = 1.5 → 2d−1 = 3 → d = 2。矛盾！无解。

情况六：BC 是菱形的对角线

BC 中点 (3, 1.5)。AD 是另一条对角线 → 中点也是 (3, 1.5)。

(0+d)/2 = 3 → d = 6；(0+2d−1)/2 = 1.5 → 2d−1 = 3 → d = 2。矛盾！无解。

重新思考——边长不一定要等于已知线段长度

菱形 ABCD 中，A、B、C 不一定两两相邻。比如 A 和 C 是对角。

如果 AC 是对角线（情况五已排除）。

如果四个顶点的顺序是 A−D−B−C（A 和 B 不相邻）：

AD = DC = CB = BA？不对，菱形是四条边等长。

重新枚举四边形顶点顺序和菱形条件：

A−B−C−D：|AB|=|BC|=|CD|=|DA|。|AB|=4, |BC|=√13 ≠ 4。不成立。

A−B−D−C：|AB|=4, |BD|=|DC|=|CA|=√13。4≠√13。不成立。

A−C−B−D：|AC|=√13, |CB|=√13, |BD|=|DA|=√13 → 需 |BD|=|DA|=√13

|BD|² = (d−4)²+(2d−1)² = d²−8d+16+4d²−4d+1 = 5d²−12d+17 = 13

→ 5d²−12d+4 = 0 → d = (12±√(144−80))/10 = (12±8)/10 → d=2 或 d=2/5

  • d=2 → D(2,3) = C，舍去
  • d=2/5 → D(2/5, −1/5)

验证 |DA|² = (2/5)²+(−1/5)² = 5/25 = 1/5 ≠ 13。不成立！

（注意：顺序 A−C−B−D 中，AC 和 BD 是对边不是对角线）

继续枚举：A−C−D−B

|AC|=√13，|CD|=|DB|=|BA| → |CD|=|DB|=|BA|=4

|CD|² = (d−2)²+(2d−1−3)² = (d−2)²+(2d−4)² = d²−4d+4+4d²−16d+16 = 5d²−20d+20 = 16

→ 5d²−20d+4 = 0 → d = (20±√(400−80))/10 = (20±√320)/10 = (20±8√5)/10 = (10±4√5)/5

|DB|² = (d−4)²+(2d−1)² = 5d²−12d+17 = 16

→ 5d²−12d+1 = 0 → d = (12±√(144−20))/10 = (12±√124)/10 = (12±2√31)/10 = (6±√31)/5

需要两个方程的 d 相同，但 (10±4√5)/5 ≠ (6±√31)/5。无公共解。

A−D−C−B

|AD|=|DC|=|CB|=|BA| → AD=BA=4 且 DC=CB=√13

|AD|² = d²+(2d−1)² = 5d²−4d+1 = 16 → 5d²−4d−15 = 0 → d = (4±√(16+300))/10 = (4±√316)/10 = (4±2√79)/10 = (2±√79)/5

|DC|² = 5d²−20d+20 = 13 → 5d²−20d+7 = 0 → d = (20±√(400−140))/10 = (20±√260)/10 = (20±2√65)/10 = (10±√65)/5

同样无公共解。不成立。

第 3 题答案：不存在满足条件的菱形。

升级判断：若学生系统地枚举了所有可能的顶点排列并正确判断不存在 → 升至 F 档。若学生只试了一两种排列就下结论 → 保持在 E 档，强调"枚举所有排列"的重要性。
