自适应练习：一次函数综合压轴（五星）

一次函数综合压轴（五星）

练习说明

本练习围绕一次函数与坐标轴交点、含特殊角的直角三角形面积建模、等腰三角形存在性分类讨论三大核心考点展开。共 3 题，建议完成时间 40–50 分钟。

  • 每题均包含提示和详细解答，请先独立完成后再对照。
  • 注意区分"线段"与"射线"条件对动点范围的影响。
  • 遇到卡点可参考提示；完成全部练习后填写自检表并交给老师。

当前档位：E 档（基础巩固，侧重标准路径与计算准确性）。

第 1 题

直线 $y = -\dfrac{\sqrt{3}}{3}\,x + 3$ 与 $y$ 轴、$x$ 轴分别交于 $A$、$B$ 两点。

(1) 求点 $A$、$B$ 的坐标和 $\angle BAO$ 的度数；

(2) 点 $C$、$D$ 分别在线段 $OA$、$AB$ 上（不与端点重合），且 $CD = DA$。设 $OC = x$，$\triangle OCD$ 的面积为 $y$，求 $y$ 关于 $x$ 的函数关系式及定义域；

(3) 若点 $C$、$D$ 分别在射线 $OA$、射线 $BA$ 上，且 $CD = DA$，当 $\triangle ODB$ 为等腰三角形时，求点 $C$ 的坐标。

提示

第(1)问：令 $x=0$ 得 $A$ 坐标；令 $y=0$ 解方程得 $B$ 坐标；$\tan\angle BAO = \dfrac{OB}{OA}$。

第(2)问：$C$ 在 $OA$（$y$ 轴）上，$C(0, x)$。由 $CD=DA$ 知 $\triangle CDA$ 是以 $A$ 为顶点的等腰三角形，作 $DF \perp OA$ 于 $F$，则 $F$ 是 $CA$ 中点。

第(3)问：等腰三角形分三类讨论：$OD = OB$、$OD = DB$、$OB = DB$。注意"射线"意味着 $C$ 可以在 $O$ 下方或 $A$ 上方。

详细解答

第(1)问

令 $x = 0$：$y = 3$，故 $A(0, 3)$。

令 $y = 0$：$-\dfrac{\sqrt{3}}{3}\,x + 3 = 0$，$x = 3\sqrt{3}$，故 $B(3\sqrt{3},\, 0)$。

$OA = 3$，$OB = 3\sqrt{3}$，$\tan\angle BAO = \dfrac{OB}{OA} = \sqrt{3}$，所以 $\angle BAO = 60°$。

第(2)问

$C$ 在线段 $OA$ 上，$OC = x$，故 $C(0, x)$，其中 $0 < x < 3$。

$CA = OA - OC = 3 - x$。

$CD = DA$，$\triangle CDA$ 是等腰三角形。作 $DF \perp OA$ 于 $F$，则 $F$ 是 $CA$ 的中点。

$AF = \dfrac{CA}{2} = \dfrac{3 - x}{2}$，$OF = OA - AF = 3 - \dfrac{3-x}{2} = \dfrac{3 + x}{2}$。

在 $\text{Rt}\triangle ADF$ 中，$\tan\angle BAO = \tan 60° = \dfrac{DF}{AF}$，所以 $DF = \sqrt{3} \cdot \dfrac{3-x}{2} = \dfrac{\sqrt{3}\,(3-x)}{2}$。

$S_{\triangle OCD} = \dfrac{1}{2} \cdot OC \cdot DF = \dfrac{1}{2} \cdot x \cdot \dfrac{\sqrt{3}\,(3-x)}{2} = \dfrac{\sqrt{3}\,x\,(3-x)}{4}$。

即 $y = \dfrac{-\sqrt{3}}{4}\,x^2 + \dfrac{3\sqrt{3}}{4}\,x$，定义域为 $0 < x < 3$。

第(3)问

$C$ 在射线 $OA$ 上，设 $C(0, m)$（$m$ 可正可负）。$D$ 在射线 $BA$ 上，$CD = DA$。

类似第(2)问的分析：作 $DF \perp OA$ 于 $F$，$F$ 仍为 $CA$ 中点。

$CA = |3 - m|$，$AF = \dfrac{|3-m|}{2}$。

当 $m < 3$ 时（$C$ 在 $O$ 下方或 $OA$ 上），$CA = 3 - m$，$AF = \dfrac{3-m}{2}$，$DF = \sqrt{3} \cdot \dfrac{3-m}{2}$。

$D$ 的坐标为 $\left(\dfrac{\sqrt{3}\,(3-m)}{2},\; \dfrac{3+m}{2}\right)$。

$OD^2 = \left(\dfrac{\sqrt{3}\,(3-m)}{2}\right)^2 + \left(\dfrac{3+m}{2}\right)^2 = \dfrac{3(3-m)^2 + (3+m)^2}{4} = \dfrac{3(9-6m+m^2) + 9+6m+m^2}{4} = \dfrac{27 - 18m + 3m^2 + 9 + 6m + m^2}{4} = \dfrac{4m^2 - 12m + 36}{4} = m^2 - 3m + 9$。

情形 i：$OD = OB$

$OB = 3\sqrt{3}$，$OD^2 = 27$，即 $m^2 - 3m + 9 = 27$，$m^2 - 3m - 18 = 0$，$(m-6)(m+3) = 0$。

$m = 6$ 或 $m = -3$。

当 $m = 6$ 时，$CA = -3 < 0$，与 $m < 3$ 矛盾，舍去。验证：$m=6 > 3$，属于 $m > 3$ 的情况，后续单独讨论。

当 $m = -3$ 时，$D\left(3\sqrt{3},\, 0\right) = B$，此时 $\triangle ODB$ 退化为线段，不合题意，舍去。

情形 ii：$OD = DB$

$DB^2 = \left(3\sqrt{3} - \dfrac{\sqrt{3}(3-m)}{2}\right)^2 + \left(0 - \dfrac{3+m}{2}\right)^2 = \left(\dfrac{\sqrt{3}(3+m)}{2}\right)^2 + \left(\dfrac{3+m}{2}\right)^2 = \dfrac{3(3+m)^2 + (3+m)^2}{4} = (3+m)^2$。

$OD^2 = m^2 - 3m + 9$，$DB^2 = (3+m)^2 = m^2 + 6m + 9$。

$m^2 - 3m + 9 = m^2 + 6m + 9$，$-3m = 6m$，$m = 0$。

当 $m = 0$ 时，$C$ 与 $O$ 重合，$D$ 也在 $O$ 处（$D(0, \frac{3}{2})$ 处，$\triangle ODB$ 非退化），但 $C$ 在射线 $OA$ 上且 $m=0$ 意味着 $C$ 与 $O$ 重合。验证 $OD = \sqrt{0 - 0 + 9} = 3$，$DB = |3+0| = 3$，$OD = DB = 3 \neq OB = 3\sqrt{3}$，故 $m = 0$ 时 $\triangle ODB$ 为等腰三角形（$OD = DB$），$C(0, 0)$。

情形 iii：$OB = DB$

$OB = 3\sqrt{3}$，$DB = |3+m|$（因 $DB^2 = (3+m)^2$ 且 $DB \geq 0$），$3\sqrt{3} = |3+m|$。

$3 + m = 3\sqrt{3}$，$m = 3\sqrt{3} - 3$；或 $3 + m = -3\sqrt{3}$，$m = -3 - 3\sqrt{3}$。

$m = 3\sqrt{3} - 3 \approx 2.196 < 3$，符合。此时 $C\left(0,\; 3\sqrt{3} - 3\right)$。

$m = -3 - 3\sqrt{3} \approx -8.196 < 3$，符合。此时 $C\left(0,\; -3 - 3\sqrt{3}\right)$。

当 $m > 3$ 时（$C$ 在 $A$ 上方），$CA = m - 3$，$D$ 在 $BA$ 延长线上。

$AF = \dfrac{m-3}{2}$，$DF = \sqrt{3} \cdot \dfrac{m-3}{2}$，$D\left(\dfrac{\sqrt{3}(m-3)}{2},\; \dfrac{3+m}{2}\right)$。

$OD^2 = \dfrac{3(m-3)^2 + (m+3)^2}{4} = \dfrac{4m^2 - 12m + 36}{4} = m^2 - 3m + 9$（与 $m < 3$ 结果一致）。

情形 i（$OD = OB$）：$m^2 - 3m + 9 = 27$，$m = 6$ 或 $m = -3$。$m = 6 > 3$，符合。此时 $C(0, 6)$，$D\left(\dfrac{3\sqrt{3}}{2},\; \dfrac{9}{2}\right)$，验证 $OD = \sqrt{36 - 18 + 9} = \sqrt{27} = 3\sqrt{3} = OB$，成立。

情形 ii（$OD = DB$）：$m^2 - 3m + 9 = (m+3)^2 = m^2 + 6m + 9$，$m = 0$，不满足 $m > 3$，舍去。

情形 iii（$OB = DB$）：$|m+3| = 3\sqrt{3}$，$m = 3\sqrt{3} - 3 \approx 2.196 < 3$（舍去），$m = -3 - 3\sqrt{3} < 3$（舍去）。

综上，点 $C$ 的坐标为：

$C(0, 0)$，$C\left(0,\; 3\sqrt{3} - 3\right)$，$C\left(0,\; -3 - 3\sqrt{3}\right)$，$C(0, 6)$。

教师备注：本题第(3)问完整解涉及 4 个坐标，难度较大。E 档学生若完成(1)(2)两问即可视为达标；能写出第(3)问的分类框架（分三种情况讨论）即为良好。$m = 6$ 的解需要 $D$ 在射线延长线上，是最容易遗漏的解。

第 2 题

直线 $y = -\sqrt{3}\,x + 3\sqrt{3}$ 与 $y$ 轴交于点 $A$，与 $x$ 轴交于点 $B$。

(1) 求点 $A$、$B$ 的坐标及 $\angle BAO$ 的度数；

(2) 点 $C$、$D$ 分别在线段 $OA$、$AB$ 上（不与端点重合），且 $CD = CA$。设 $OC = x$，$S_{\triangle OCD} = y$，求 $y$ 关于 $x$ 的函数关系式及定义域；

(3) 若点 $C$、$D$ 分别在射线 $OA$、射线 $BA$ 上，且 $CD = CA$，当 $\triangle ODB$ 为等腰三角形时，求点 $C$ 的坐标。

提示

第(1)问：令 $x=0$ 得 $A(0, 3\sqrt{3})$；令 $y=0$ 得 $B(3, 0)$。$\tan\angle BAO = \dfrac{OB}{OA} = \dfrac{3}{3\sqrt{3}} = \dfrac{\sqrt{3}}{3}$，故 $\angle BAO = 30°$。

第(2)问：$CD = CA$ 意味着 $\triangle CDA$ 以 $C$ 为顶点。作 $DE \perp OA$ 于 $E$，利用 $30°$ 角和等腰条件找 $DE$ 与 $x$ 的关系。

第(3)问：等腰三分类：$OD = OB$、$OD = DB$、$OB = DB$。注意本题特殊角为 $30°$，$\tan 30° = \dfrac{\sqrt{3}}{3}$。

详细解答

第(1)问

令 $x = 0$：$y = 3\sqrt{3}$，$A(0, 3\sqrt{3})$。

令 $y = 0$：$-\sqrt{3}\,x + 3\sqrt{3} = 0$，$x = 3$，$B(3, 0)$。

$OA = 3\sqrt{3}$，$OB = 3$，$\tan\angle BAO = \dfrac{OB}{OA} = \dfrac{3}{3\sqrt{3}} = \dfrac{\sqrt{3}}{3}$，$\angle BAO = 30°$。

第(2)问

$C$ 在线段 $OA$ 上，$OC = x$，故 $C(0, x)$，$0 < x < 3\sqrt{3}$。

$CA = OA - OC = 3\sqrt{3} - x$。

$CD = CA$，$\triangle CDA$ 是以 $C$ 为顶点的等腰三角形。作 $DE \perp OA$ 于 $E$。

$\angle CDA = \angle CAD$，且 $\angle CAD = \angle BAO = 30°$（$A$、$D$、$B$ 共线），所以 $\angle CDA = 30°$。

$\angle DCE = 90° - 30° = 60°$。在 $\text{Rt}\triangle CDE$ 中，$\tan\angle DCE = \dfrac{DE}{CE} = \tan 60° = \sqrt{3}$，$DE = \sqrt{3}\cdot CE$。

又 $CE = CD \cdot \cos 60° = CA \cdot \dfrac{1}{2} = \dfrac{3\sqrt{3} - x}{2}$。

$DE = \sqrt{3} \cdot \dfrac{3\sqrt{3} - x}{2} = \dfrac{9 - \sqrt{3}\,x}{2}$。

$S_{\triangle OCD} = \dfrac{1}{2} \cdot OC \cdot DE = \dfrac{1}{2} \cdot x \cdot \dfrac{9 - \sqrt{3}\,x}{2} = \dfrac{9x - \sqrt{3}\,x^2}{4}$。

即 $y = \dfrac{-\sqrt{3}}{4}\,x^2 + \dfrac{9}{4}\,x$，定义域为 $0 < x < 3\sqrt{3}$。

第(3)问

设 $C(0, m)$，$D$ 在射线 $BA$ 上。$CD = CA = |3\sqrt{3} - m|$。

与第 1 题类似分析（利用 $30°$ 角的特殊性）：

当 $m < 3\sqrt{3}$ 时，$CE = \dfrac{3\sqrt{3} - m}{2}$，$DE = \sqrt{3} \cdot \dfrac{3\sqrt{3} - m}{2} = \dfrac{9 - \sqrt{3}\,m}{2}$。

$OE = OC + CE = m + \dfrac{3\sqrt{3} - m}{2} = \dfrac{m + 3\sqrt{3}}{2}$。

$D\left(\dfrac{9 - \sqrt{3}\,m}{2},\; \dfrac{m + 3\sqrt{3}}{2}\right)$。

$OD^2 = \left(\dfrac{9 - \sqrt{3}\,m}{2}\right)^2 + \left(\dfrac{m + 3\sqrt{3}}{2}\right)^2 = \dfrac{81 - 18\sqrt{3}\,m + 3m^2 + m^2 + 6\sqrt{3}\,m + 27}{4} = \dfrac{4m^2 - 12\sqrt{3}\,m + 108}{4} = m^2 - 3\sqrt{3}\,m + 27$。

情形 i：$OD = OB = 3$

$m^2 - 3\sqrt{3}\,m + 27 = 9$，$m^2 - 3\sqrt{3}\,m + 18 = 0$。

$\Delta = 27 - 72 = -45 < 0$，无实数解。

情形 ii：$OD = DB$

$DB^2 = \left(3 - \dfrac{9 - \sqrt{3}\,m}{2}\right)^2 + \left(\dfrac{m + 3\sqrt{3}}{2}\right)^2 = \left(\dfrac{\sqrt{3}\,m - 3}{2}\right)^2 + \left(\dfrac{m + 3\sqrt{3}}{2}\right)^2 = \dfrac{3m^2 - 6\sqrt{3}\,m + 9 + m^2 + 6\sqrt{3}\,m + 27}{4} = \dfrac{4m^2 + 36}{4} = m^2 + 9$。

$m^2 - 3\sqrt{3}\,m + 27 = m^2 + 9$，$-3\sqrt{3}\,m = -18$，$m = \dfrac{18}{3\sqrt{3}} = 2\sqrt{3}$。

$2\sqrt{3} \approx 3.46 < 3\sqrt{3} \approx 5.20$，符合。

验证：$OD^2 = 12 - 18 + 27 = 21$，$DB^2 = 12 + 9 = 21$，$OD = DB = \sqrt{21}$，$OB = 3 \neq \sqrt{21}$，成立。$C(0, 2\sqrt{3})$。

情形 iii：$OB = DB = 3$

$m^2 + 9 = 9$，$m = 0$。

$C(0, 0) = O$，此时 $D\left(\dfrac{9}{2},\; \dfrac{3\sqrt{3}}{2}\right)$，验证 $OD = \sqrt{\dfrac{81}{4} + \dfrac{27}{4}} = \sqrt{27} = 3\sqrt{3}$，$DB = \sqrt{0 + 9} = 3 = OB$，$\triangle ODB$ 中 $OB = DB = 3$，成立。

但 $C$ 与 $O$ 重合，$C(0, 0)$。

当 $m > 3\sqrt{3}$ 时，$C$ 在 $A$ 上方，$CA = m - 3\sqrt{3}$，$D$ 在 $BA$ 延长线上。

$CE = \dfrac{m - 3\sqrt{3}}{2}$，$DE = \sqrt{3} \cdot \dfrac{m - 3\sqrt{3}}{2} = \dfrac{\sqrt{3}\,m - 9}{2}$。

$OE = OC - CE = m - \dfrac{m - 3\sqrt{3}}{2} = \dfrac{m + 3\sqrt{3}}{2}$。

$D\left(\dfrac{\sqrt{3}\,m - 9}{2},\; \dfrac{m + 3\sqrt{3}}{2}\right)$。

$OD^2 = \dfrac{(\sqrt{3}\,m - 9)^2 + (m + 3\sqrt{3})^2}{4} = \dfrac{3m^2 - 18\sqrt{3}\,m + 81 + m^2 + 6\sqrt{3}\,m + 27}{4} = \dfrac{4m^2 - 12\sqrt{3}\,m + 108}{4} = m^2 - 3\sqrt{3}\,m + 27$（与上面一致）。

情形 i（$OD = OB$）：$m^2 - 3\sqrt{3}\,m + 18 = 0$，$\Delta < 0$，无解。

情形 ii（$OD = DB$）：$m = 2\sqrt{3} < 3\sqrt{3}$，不满足 $m > 3\sqrt{3}$，舍去。

情形 iii（$OB = DB$）：$DB^2 = \dfrac{(\sqrt{3}\,m - 9 - 6)^2 + (m + 3\sqrt{3})^2}{4}$。由于 $D$ 在延长线上，$DB^2$ 的计算方式不同。更简洁地，由 $DB^2 = (m+3)^2$ 的推导在 $m > 3\sqrt{3}$ 时依然成立（$DB^2 = m^2 + 9$ 对所有 $m$ 成立，因为这是代数恒等式）。所以 $m^2 + 9 = 9$，$m = 0$，不符合 $m > 3\sqrt{3}$。

综上，点 $C$ 的坐标为：

$C(0, 0)$，$C(0, 2\sqrt{3})$。

教师备注：本题将特殊角从 $60°$ 改为 $30°$，等腰条件从 $CD=DA$ 改为 $CD=CA$（顶点从 $A$ 变为 $C$）。第(2)问的关键变化是 $CE$ 的计算方向不同。第(3)问只有 2 个解（比第 1 题少），$OD=OB$ 无解是一个值得让学生思考的结论。E 档学生完成(1)(2)即可视为达标。

第 3 题

直线 $y = -x + 4$ 与 $y$ 轴交于点 $A$，与 $x$ 轴交于点 $B$。矩形 $OEFG$ 的顶点 $E$ 在 $AB$ 上，$F$ 在 $x$ 轴正半轴上，$G$ 在 $y$ 轴正半轴上。

(1) 求点 $A$、$B$ 的坐标及 $\angle BAO$ 的度数；

(2) 设 $OG = t$，用含 $t$ 的式子表示矩形 $OEFG$ 的面积 $S$，并求 $S$ 的最大值；

(3) 点 $P$ 在线段 $AB$ 上，且 $\triangle OPA$ 为等腰三角形，求点 $P$ 的坐标。

提示

第(1)问：$A(0, 4)$，$B(4, 0)$。$\triangle AOB$ 是等腰直角三角形，$\angle BAO = 45°$。

第(2)问：矩形内接于等腰直角三角形，$G(0, t)$，$E$ 在 $AB$ 上。由 $AB$ 的方程可求 $E$ 的横坐标。面积 $S = OG \times OF$。

第(3)问：$\triangle OPA$ 为等腰三角形，分三类：$OP = OA$、$OP = PA$、$OA = PA$。注意 $A$ 在 $y$ 轴上、$O$ 为原点的几何关系。

详细解答

第(1)问

令 $x = 0$：$y = 4$，$A(0, 4)$。

令 $y = 0$：$-x + 4 = 0$，$x = 4$，$B(4, 0)$。

$OA = 4 = OB$，$\triangle AOB$ 为等腰直角三角形，$\angle BAO = 45°$。

第(2)问

$G(0, t)$，$EG \perp OA$（$OA$ 在 $y$ 轴上），故 $E$ 的纵坐标为 $t$。

$E$ 在 $AB$：$y = -x + 4$ 上，$t = -x_E + 4$，$x_E = 4 - t$。

$OEGF$ 为矩形，$OF = x_E = 4 - t$，$OG = t$。

$S = OG \cdot OF = t(4 - t) = -t^2 + 4t$。

$S = -(t^2 - 4t + 4) + 4 = -(t - 2)^2 + 4$。

当 $t = 2$ 时，$S$ 取最大值 $4$。此时 $G(0, 2)$，$E(2, 2)$，$F(2, 0)$。

第(3)问

$P$ 在线段 $AB$ 上，设 $P(a, 4-a)$，$0 \leq a \leq 4$。

$OA = 4$。

$OP^2 = a^2 + (4-a)^2 = a^2 + 16 - 8a + a^2 = 2a^2 - 8a + 16$。

$PA^2 = a^2 + (4-a-4)^2 = a^2 + a^2 = 2a^2$。

情形 i：$OP = OA = 4$

$OP^2 = 16$，$2a^2 - 8a + 16 = 16$，$2a^2 - 8a = 0$，$2a(a - 4) = 0$。

$a = 0$：$P = A = (0, 4)$，$\triangle OPA$ 退化为线段，舍去。

$a = 4$：$P = B = (4, 0)$，$OP = \sqrt{16} = 4 = OA$，$PA = \sqrt{32} = 4\sqrt{2} \neq 4$，成立。$P(4, 0)$。

情形 ii：$OP = PA$

$OP^2 = PA^2$，$2a^2 - 8a + 16 = 2a^2$，$-8a + 16 = 0$，$a = 2$。

$P(2, 2)$，$OP = \sqrt{8} = 2\sqrt{2}$，$PA = \sqrt{8} = 2\sqrt{2}$，$OA = 4$，成立。

情形 iii：$OA = PA = 4$

$PA^2 = 16$，$2a^2 = 16$，$a^2 = 8$，$a = 2\sqrt{2}$。

$0 < 2\sqrt{2} \approx 2.83 < 4$，符合。

$P(2\sqrt{2},\; 4 - 2\sqrt{2})$，验证 $PA = \sqrt{8 + 8} = \sqrt{16} = 4 = OA$，$OP = \sqrt{8 + (4-2\sqrt{2})^2} = \sqrt{8 + 16 - 16\sqrt{2} + 8} = \sqrt{32 - 16\sqrt{2}} \neq 4$（$32 - 16\sqrt{2} \approx 32 - 22.63 = 9.37 \neq 16$），成立。

综上，点 $P$ 的坐标为：

$P(4, 0)$，$P(2, 2)$，$P\!\left(2\sqrt{2},\; 4 - 2\sqrt{2}\right)$。

教师备注：本题引入 $45°$ 等腰直角三角形，矩形面积建模比三角形更直观，第(3)问从等腰存在性角度切入。与第 1、2 题相比，特殊角为 $45°$（$\tan 45° = 1$），计算最简洁，适合作为过渡练习。E 档学生完成(1)(2)即可，第(3)问能分清三类情况即可。
