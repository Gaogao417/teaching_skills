等腰梯形存在性

练习说明

本练习围绕等腰梯形存在性问题展开，共 3 道题。每道题都是"已知三个顶点坐标，求第四个顶点使四边形为等腰梯形"。

核心方法：枚举三组平行对边 × 每组列两个方程（斜率相等 + 腰长相等）× 验证不退化为平行四边形。

建议先做第 1 题，做完再对答案；全部完成后填写自检表。

答案与解析

第 1 题答案

备用数据：$k_{OA}=0$，$|OA|^2=4$；$k_{AB}=1$，$|AB|^2=8$；$k_{OB}=\dfrac{1}{2}$，$|OB|^2=20$。

情况一：$OA \parallel BC$，腰为 $OB$ 和 $AC$

平行：$k_{BC}=0$，即 $y_C = y_B = 2$。

等腰：$|OB|^2 = |AC|^2$，即 $20 = (x_C-2)^2 + 4$。

$$(x_C-2)^2 = 16 \implies x_C - 2 = \pm 4 \implies x_C = 6 \text{ 或 } x_C = -2$$

$C(6,2)$：检查 $k_{AC} = \dfrac{2-0}{6-2} = \dfrac{1}{2} = k_{OB}$，两组对边都平行，退化为平行四边形，舍去。

$C(-2,2)$：检查 $k_{AC} = \dfrac{2-0}{-2-2} = -\dfrac{1}{2} \neq \dfrac{1}{2}$，不平行，有效。

$$\boxed{C(-2,\,2)}$$

情况二：$AB \parallel OC$，腰为 $OA$ 和 $BC$

平行：$k_{OC} = k_{AB} = 1$，即 $\dfrac{y_C}{x_C} = 1$，$y_C = x_C$。

等腰：$|OA|^2 = |BC|^2$，即 $4 = (x_C-4)^2 + (x_C-2)^2$。

$$(x_C-4)^2 + (x_C-2)^2 = 4$$

$$x_C^2 - 8x_C + 16 + x_C^2 - 4x_C + 4 = 4$$

$$2x_C^2 - 12x_C + 16 = 0$$

$$x_C^2 - 6x_C + 8 = 0 \implies (x_C-2)(x_C-4) = 0$$

$x_C = 2$ 时 $C(2,2)$：检查 $k_{BC} = 0 \neq 0 = k_{OA}$...等等，$k_{BC} = \dfrac{2-2}{2-4} = 0 = k_{OA}$，退化为平行四边形（$OA \parallel BC$ 且 $AB \parallel OC$），舍去。

$x_C = 4$ 时 $C(4,4)$：检查 $k_{BC} = \dfrac{4-2}{4-4}$ 分母为零（$C$ 和 $B$ 重合），四边形退化，舍去。

因此情况二无有效解。

情况三：$OB \parallel AC$，腰为 $OA$ 和 $BC$

平行：$k_{AC} = k_{OB} = \dfrac{1}{2}$，即 $\dfrac{y_C}{x_C-2} = \dfrac{1}{2}$，$y_C = \dfrac{1}{2}(x_C-2)$。

等腰：$|OA|^2 = |BC|^2$，即 $4 = (x_C-4)^2 + \left(y_C-2\right)^2$。

代入 $y_C = \dfrac{1}{2}x_C - 1$：

$$(x_C-4)^2 + \left(\frac{x_C}{2} - 3\right)^2 = 4$$

$$x_C^2 - 8x_C + 16 + \frac{x_C^2}{4} - 3x_C + 9 = 4$$

$$\frac{5}{4}x_C^2 - 11x_C + 21 = 0$$

$$5x_C^2 - 44x_C + 84 = 0$$

$$\Delta = 1936 - 1680 = 256$$

$$x_C = \frac{44 \pm 16}{10} \implies x_C = 6 \text{ 或 } x_C = \dfrac{14}{5}$$

$x_C = 6$ 时 $y_C = 2$，$C(6,2)$：检查 $k_{BC} = 0 = k_{OA}$，退化为平行四边形，舍去。

$x_C = \dfrac{14}{5}$ 时 $y_C = \dfrac{1}{2}\!\left(\dfrac{14}{5}-2\right) = \dfrac{2}{5}$，$C\!\left(\dfrac{14}{5},\,\dfrac{2}{5}\right)$。检查 $k_{BC} = \dfrac{\frac{2}{5}-2}{\frac{14}{5}-4} = \dfrac{-8/5}{-6/5} = \dfrac{4}{3} \neq 0 = k_{OA}$，不平行，有效。

$$\boxed{C\!\left(\dfrac{14}{5},\,\dfrac{2}{5}\right)}$$

第 1 题最终答案

$$C(-2,\,2)\;\text{或}\; C\!\left(\dfrac{14}{5},\,\dfrac{2}{5}\right)$$

第 2 题答案

备用数据：$k_{AB}=1$，$|AB|^2=18$；$k_{BC}=-1$，$|BC|^2=18$；$k_{AC}=0$，$|AC|^2=36$。

注意 $|AB|^2 = |BC|^2 = 18$，即 $B$ 在 $AC$ 的垂直平分线上。

情况一：$AB \parallel CD$，腰为 $AC$ 和 $BD$

平行：$k_{CD}=k_{AB}=1$，即 $\dfrac{y_D-0}{x_D-6}=1$，$y_D = x_D - 6$。

等腰：$|AC|^2 = |BD|^2$，即 $36 = (x_D-3)^2 + (y_D-3)^2$。

代入 $y_D = x_D-6$：

$$(x_D-3)^2 + (x_D-9)^2 = 36$$

$$2x_D^2 - 24x_D + 90 = 0$$

$$x_D^2 - 12x_D + 45 = 0$$

$\Delta = 144 - 180 = -36 < 0$，无实数解，情况一不成立。

情况二：$AC \parallel BD$，腰为 $AB$ 和 $CD$

平行：$k_{BD}=k_{AC}=0$，即 $y_D = y_B = 3$。

等腰：$|AB|^2 = |CD|^2$，即 $18 = (x_D-6)^2 + 9$。

$$(x_D-6)^2 = 9 \implies x_D - 6 = \pm 3 \implies x_D = 9 \text{ 或 } x_D = 3$$

$D(3,3)$：与 $B$ 重合，四边形退化，舍去。

$D(9,3)$：检查 $k_{CD} = \dfrac{3-0}{9-6} = 1 = k_{AB}$，退化为平行四边形，舍去。

因此情况二无有效解。

情况三：$BC \parallel AD$，腰为 $AB$ 和 $CD$

平行：$k_{AD}=k_{BC}=-1$，即 $\dfrac{y_D}{x_D}=-1$，$y_D = -x_D$。

等腰：$|AB|^2 = |CD|^2$，即 $18 = (x_D-6)^2 + y_D^2$。

代入 $y_D = -x_D$：

$$(x_D-6)^2 + x_D^2 = 18$$

$$2x_D^2 - 12x_D + 18 = 0$$

$$x_D^2 - 6x_D + 9 = 0 \implies (x_D-3)^2 = 0$$

$x_D = 3$，$y_D = -3$，$D(3,-3)$。检查 $k_{CD} = \dfrac{-3-0}{3-6} = 1 = k_{AB}$，退化为平行四边形，舍去。

因此情况三无有效解。

重新检查：是否遗漏了等腰条件的配对？

等腰梯形中，两腰相等。上面的分析中，"腰"的选择是固定的：底的对边就是腰。但等腰梯形的等腰条件也可以写成"对角线相等"，即 $|AD| = |BC$|（当底为 $AB$ 和 $CD$ 时）。让我们用对角线相等的条件重新检查情况一：

情况一（$AB \parallel CD$），等腰条件改为 $|AD| = |BC|$，即 $|AD|^2 = |BC|^2 = 18$：

$$(x_D-0)^2 + (y_D-0)^2 = 18$$

代入 $y_D = x_D - 6$：

$$x_D^2 + (x_D - 6)^2 = 18 \implies 2x_D^2 - 12x_D + 18 = 0$$

$$(x_D - 3)^2 = 0 \implies x_D = 3,\; y_D = -3$$

$D(3,-3)$：检查 $k_{AD} = \dfrac{-3}{3} = -1 = k_{BC}$，$AD \parallel BC$ 且 $AB \parallel CD$，退化为平行四边形，舍去。

结论：这道题不存在等腰梯形的解。这是因为 $A(0,0)$、$B(3,3)$、$C(6,0)$ 关于 $B$ 对称，无论 $D$ 怎么取，只要满足一组对边平行，另一组对边也必然平行。

这道题的设计意图是：先让学生完整走一遍三组讨论的流程，最后发现没有有效解——这是一种重要体验。不是每道分类讨论题都一定有答案。

第 2 题最终答案

$$\boxed{\text{不存在使 } ABCD \text{ 为等腰梯形的点 } D}$$

第 3 题答案

备用数据：$k_{PQ}=1$，$|PQ|^2=20$；$k_{QR}=-1$，$|QR|^2=20$；$k_{PR}=0$，$|PR|^2=36$。

注意 $P(1,1)$、$R(7,1)$ 都在直线 $y=1$ 上，$Q(3,5)$ 在上方。$|PQ|^2=|QR|^2=20$，关于 $QR$ 的垂直平分线对称。

第(1)问：$PQ \parallel SR$

平行：$k_{SR}=k_{PQ}=1$，即 $\dfrac{y_S-1}{x_S-7}=1$，$y_S = x_S - 6$。

等腰梯形 $PQRS$ 中，底为 $PQ$ 和 $SR$，腰为 $PS$ 和 $QR$。

等腰条件：$|PS|^2 = |QR|^2 = 20$。

$$(x_S-1)^2 + (y_S-1)^2 = 20$$

代入 $y_S = x_S - 6$：

$$(x_S-1)^2 + (x_S-7)^2 = 20$$

$$2x_S^2 - 16x_S + 50 = 0$$

$$x_S^2 - 8x_S + 25 = 0$$

$\Delta = 64 - 100 = -36 < 0$，用腰 $PS=QR$ 无解。

换一种等腰方式：对角线 $|PR|=|QS|$。

$$|PR|^2 = 36 = |QS|^2 = (x_S-3)^2 + (y_S-5)^2$$

代入 $y_S = x_S - 6$：

$$(x_S-3)^2 + (x_S-11)^2 = 36$$

$$2x_S^2 - 28x_S + 130 = 0$$

$$x_S^2 - 14x_S + 65 = 0$$

$\Delta = 196 - 260 = -64 < 0$，对角线相等也无解。

第(1)问答案：不存在使 $PQ \parallel SR$ 且 $PQRS$ 为等腰梯形的点 $S$。

$$\boxed{\text{第(1)问：不存在这样的点 } S}$$

第(2)问：不指定平行边，求所有可能的 $S$

由第(1)问知 $PQ \parallel SR$ 无解。讨论另外两组：

情况 A：$PR \parallel QS$（底为 $PR$ 和 $QS$，腰为 $PQ$ 和 $RS$）

平行：$k_{QS} = k_{PR} = 0$，即 $y_S = y_Q = 5$。

等腰：$|PQ|^2 = |RS|^2 = 20$，即 $20 = (x_S-7)^2 + (5-1)^2 = (x_S-7)^2 + 16$。

$$(x_S-7)^2 = 4 \implies x_S - 7 = \pm 2 \implies x_S = 9 \text{ 或 } x_S = 5$$

$S(9,5)$：检查 $k_{RS} = \dfrac{5-1}{9-7} = 2 \neq 1 = k_{PQ}$，不平行，有效。

$S(5,5)$：检查 $k_{RS} = \dfrac{5-1}{5-7} = -2 \neq 1 = k_{PQ}$，不平行，有效。

$$\boxed{S(9,\,5)\;\text{或}\;S(5,\,5)}$$

情况 B：$QR \parallel PS$（底为 $QR$ 和 $PS$，腰为 $PQ$ 和 $RS$）

平行：$k_{PS} = k_{QR} = -1$，即 $\dfrac{y_S-1}{x_S-1} = -1$，$y_S = -x_S + 2$。

等腰：$|PQ|^2 = |RS|^2 = 20$，即 $20 = (x_S-7)^2 + (y_S-1)^2$。

代入 $y_S = -x_S + 2$：

$$(x_S-7)^2 + (-x_S+1)^2 = 20$$

$$x_S^2 - 14x_S + 49 + x_S^2 - 2x_S + 1 = 20$$

$$2x_S^2 - 16x_S + 30 = 0$$

$$x_S^2 - 8x_S + 15 = 0 \implies (x_S-3)(x_S-5) = 0$$

$x_S = 3$ 时 $y_S = -1$，$S(3,-1)$：检查 $k_{RS} = \dfrac{-1-1}{3-7} = \dfrac{1}{2} \neq 0 = k_{PR}$，不平行，有效。

$x_S = 5$ 时 $y_S = -3$，$S(5,-3)$：检查 $k_{RS} = \dfrac{-3-1}{5-7} = 2 \neq 0 = k_{PR}$，不平行，有效。

$$\boxed{S(3,\,-1)\;\text{或}\;S(5,\,-3)}$$

第 3 题最终答案

$$\text{第(1)问：不存在这样的点 } S$$

$$\text{第(2)问：}\;S(9,\,5)\;\text{或}\;S(5,\,5)\;\text{或}\;S(3,\,-1)\;\text{或}\;S(5,\,-3)$$

教师注记

  • 第 1 题（同源平级）：换了一组点 $O(0,0)$、$A(2,0)$、$B(4,2)$，完全独立走三组讨论。难点在于情况三的二次方程计算量稍大（系数为分数），学生可能出现计算错误。如果学生只能完成 1-2 种情况，说明还在 D 档水平。
  • 第 2 题（换情境）：这道题没有解。设计目的是让学生体验"完整走完流程后发现无解"的情况。$A(0,0)$、$B(3,3)$、$C(6,0)$ 中 $B$ 在 $AC$ 垂直平分线上，对称结构导致任何满足平行条件的 $D$ 都会让另一组对边也平行。如果学生不做就猜"无解"，应追问理由；如果学生做了全部三组才得出结论，说明方法掌握扎实。
  • 第 3 题（结构隐藏）：第(1)问指定平行边但无解，第(2)问放开后找到 4 个解。这道题的等腰条件有两种理解方式（两腰相等 vs. 对角线相等），第(1)问中两种都无解，这是一个重要的辨析点。学生需要学会在一种方式无解时尝试另一种。第(2)问情况 A 中 $|PQ|^2 = |RS|^2$ 得出两个解，情况 B 同样两个解，共 4 个。
  • 分档建议：E 档学生完成第 1 题即可；D 档学生完成第 1、2 题；C 档及以上完成全部三题。

答案速查

  • 第 1 题：$C(-2,2)$ 或 $C\!\left(\dfrac{14}{5},\,\dfrac{2}{5}\right)$（共 2 个解）
  • 第 2 题：不存在使 $ABCD$ 为等腰梯形的点 $D$（无解）
  • 第 3 题(1)：无解
  • 第 3 题(2)：$S(9,5)$ 或 $S(5,5)$ 或 $S(3,-1)$ 或 $S(5,-3)$（共 4 个解）

阶段交接

本阶段你掌握了什么：

  • 等腰梯形存在性问题的核心方法：枚举三组平行对边 × 斜率相等列方程 × 腰长相等列方程 × 验证不退化为平行四边形。
  • 不是每道题都有解——有些对称结构下不存在等腰梯形。
  • 等腰条件有两种等价表达：两腰相等或对角线相等，一种无解时可以尝试另一种。
  • 退化检查是必做步骤：两组对边都平行就不是梯形了。

下一阶段预告：将等腰梯形扩展为直角梯形或一般梯形（不需要腰长相等的条件，但需要加直角条件或其他约束），进一步体会坐标几何中存在性问题的通用策略。
