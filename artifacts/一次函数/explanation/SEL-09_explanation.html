学生版讲解：一次函数与正比例函数的图象符号分析

一、这题先看懂什么

题目给了两个函数：

  • 一次函数 y = ax + b —— 这是一条直线，斜率是 a，与 y 轴的交点是 (0, b)
  • 正比例函数 y = abx —— 这也是一条直线，过原点，斜率是 ab

关键条件：ab ≠ 0，意思是 a 和 b 都不等于 0，所以它们要么都正、要么都负、要么一正一负。

题目问：四个选项各画了两条直线，哪一组能同时代表 y = ax + b 和 y = abx？

二、第一步该做什么

不要先画图，先想符号。

a 和 b 各有两个可能：正或负。一共只有 4 种组合。对每种组合，我们都能同时确定两条直线长什么样。然后看哪个选项匹配上了。

想一想：为什么"先看符号"比"先画图"快？

三、关键转化怎么想

核心问题其实只有两个：

  1. 走向（斜率正负）：斜率 > 0 → 从左下到右上；斜率 < 0 → 从左上到右下
  2. 截距（b 的正负）：b > 0 → 直线与 y 轴交在原点上方；b < 0 → 交在原点下方

正比例函数 y = abx 没有截距（过原点），只看走向。它的斜率是 ab，所以走向由 a 和 b 同号还是异号决定。

四、标准解法

第一步：列四组符号

组别abab
第①组> 0> 0> 0
第②组> 0< 0< 0
第③组< 0> 0< 0
第④组< 0< 0> 0

第二步：每组写出两条直线的特征

第①组（a>0, b>0）：

  • y = ax + b：a>0 → 走向↗；b>0 → 截距在上方。直线从左下往右上走，交 y 轴正半轴。
  • y = abx：ab>0 → 走向↗。过原点，从左下往右上走。

第②组（a>0, b<0）：

  • y = ax + b：a>0 → 走向↗；b<0 → 截距在下方。直线从左下往右上走，交 y 轴负半轴。
  • y = abx：ab<0 → 走向↘。过原点，从左上往右下走。

第③组（a<0, b>0）：

  • y = ax + b：a<0 → 走向↘；b>0 → 截距在上方。直线从左上往右下走，交 y 轴正半轴。
  • y = abx：ab<0 → 走向↘。过原点，从左上往右下走。

第④组（a<0, b<0）：

  • y = ax + b：a<0 → 走向↘；b<0 → 截距在下方。直线从左上往右下走，交 y 轴负半轴。
  • y = abx：ab>0 → 走向↗。过原点，从左下往右上走。

第三步：用排除法看选项

看每个选项中两条直线各自的走向和截距，对照上面的四组：

  • 如果选项中的一次函数走向↗、截距在上方，正比例函数走向↗ → 匹配第①组 ✓
  • 如果选项中两条线都是↗走向，但一次函数截距在下方 → 不匹配任何组 ✗（第①组截距在上方，第②组正比例函数↘）

只有两条线的特征同时属于同一组的选项才是对的。

五、边讲边问

  1. 如果 a = 2, b = 3，那么 y = ax + b 和 y = abx 的走向各是什么？
  2. 如果 y = ax + b 的走向是↗，截距在下方，那 a 和 b 各是什么符号？
  3. ab > 0 时，y = abx 走向是什么？a 和 b 同号还是异号？
  4. 如果 y = ax + b 走向↘，y = abx 也走向↘，那 (a, b) 属于第几组？
  5. 为什么不能只看 y = ax + b 的图象就选答案？

六、易错提醒

  • 别忘了 ab≠0 意味着 a 和 b 都不为 0。 不能假设 a=1 或 b=1，必须讨论符号。
  • ab 的符号 ≠ a 的符号。 ab 是 a 和 b 相乘的结果，同号为正、异号为负，要单独算。
  • 两条线要同时匹配同一组条件。 不能一条线匹配第①组、另一条线匹配第③组——它们用的是同一组 a、b。

七、一句话总结

分四组讨论 a、b 的符号 → 每组同时确定两条直线的走向和截距 → 排除矛盾选项。
