学生版讲解：一次函数增减性分类讨论

题目：已知一次函数 y = kx + b 的自变量的取值范围是 −3 到 6，相应的函数值的取值范围是 −5 到 −2，求这个一次函数的解析式。

一、这题先看懂什么

题目告诉了我们两件事：

  1. x 的范围：从 −3 到 6（最小值 −3，最大值 6）
  2. y 的范围：从 −5 到 −2（最小值 −5，最大值 −2）

我们的目标是：把 y = kx + b 里的 k 和 b 都确定下来。

想一想：知道 x 的两个端点和 y 的两个端点，一共 4 个数，是不是能把 k 和 b 算出来？

二、第一步该做什么——为什么要分类讨论？

你可能会想：直接把最小的 x 配最小的 y，最大的 x 配最大的 y，列方程组不就行了吗？

这确实是一种配法，但它对应的是直线从左下往右上走（k > 0，递增）的情况。

如果直线是从左上往右下走（k < 0，递减）呢？那最小的 x 对应的不是最小的 y，而是最大的 y！

想一想：k > 0 时，x 越大，y 越大还是越小？k < 0 呢？

所以，因为题目没有告诉我们 k 是正数还是负数，我们必须分两种情况来讨论，否则就会漏掉一个解。

核心判断：k 的正负未知 → 必须分 k > 0 和 k < 0 两种情况讨论

三、关键转化——端点怎么配对？

这是本题最关键的一步。我们把 x 和 y 的端点列出来：

最小值 最大值
x −3 6
y −5 −2

情况一：k > 0（直线递增，从左下到右上）

x 变大 → y 也变大。所以：最小的 x 对应最小的 y，最大的 x 对应最大的 y。

x = −3 时，y = −5 x = 6 时，y = −2

情况二：k < 0（直线递减，从左上到右下）

x 变大 → y 反而变小。所以：最小的 x 对应最大的 y，最大的 x 对应最小的 y。

x = −3 时，y = −2 x = 6 时，y = −5
想一想：如果 k = 0（水平直线），y 的范围会不会是 −5 到 −2？

四、标准解法

情况一：k > 0（递增）

端点配对：x = −3 对应 y = −5，x = 6 对应 y = −2

代入 y = kx + b：

① −3k + b = −5    ② 6k + b = −2

求解：② − ① 得 9k = 3，所以 k = 1/3

代入①得 b = −5 + 3 × (1/3) = −5 + 1 = −4，所以 b = −4

解一：y = (1/3)x − 4

情况二：k < 0（递减）

端点配对：x = −3 对应 y = −2，x = 6 对应 y = −5

代入 y = kx + b：

③ −3k + b = −2    ④ 6k + b = −5

求解：④ − ③ 得 9k = −3，所以 k = −1/3

代入③得 b = −2 + 3 × (−1/3) = −2 − 1 = −3，所以 b = −3

解二：y = −(1/3)x − 3

验证

对解一 y = (1/3)x − 4：x = −3 时 y = −5，x = 6 时 y = −2，符合条件。

对解二 y = −(1/3)x − 3：x = −3 时 y = −2，x = 6 时 y = −5，符合条件。

五、边讲边问

  1. 题目中 x 有两个端点、y 有两个端点，一共 4 个数。但我们只需要确定 k 和 b 两个量。为什么不能随便挑两对数列方程组？
  2. 如果题目改成"y 的范围是 −5 到 −2，且 k > 0"，还需要分类讨论吗？为什么？
  3. 在本题中，为什么 k 不可能等于 0？
  4. 如果 x 的范围是 1 到 3，y 的范围是 −2 到 2，你觉得这样的题会有几个解？为什么？

六、易错提醒

易错 1：忘记分类讨论，只算一种情况。
很多同学看到"取值范围"就直接把最小的 x 和最小的 y 配在一起，漏掉了 k < 0 的解。记住：k 的正负没告诉你，就必须讨论！
易错 2：k < 0 时端点配反。
k < 0 意味着 x 大了 y 反而小，所以 x 最小值对应的应该是 y 最大值。如果配反了，解出来的 k 符号就不对。
易错 3：解完不验证。
两种情况都算完之后，一定要把 x 的两个端点代进去检查 y 的值是否在题目给定的范围内。

七、一句话总结

k 正负不知→分两类→递增同向配、递减反向配→分别列方程组求解→验证
