学生版讲解：两定两动—菱形存在性
两定两动—菱形存在性
一、先把题目拆开
要素内容
已知直线直线 AQ 在 y 轴截距为 2，∠QAO = 45°
直线 BE\(y = -2x + 8\)
交点AQ 与 BE 交于点 P
待定点 C在 y 轴负半轴上
动点 M在直线 PA 上
动点 N在直线 PB 上
核心目标是否存在以 Q、C、M、N 为顶点的菱形？求 C 的坐标
二、这题准备怎么走
  1. 求基础坐标——先求 AQ 的解析式，再联立求 P 的坐标，以及 A、Q、B、E 各点坐标
  2. 参数化动点——用参数 t 表示 M（在 PA 上），用参数 s 表示 N（在 PB 上）
  3. 分类枚举菱形——以 Q、C 为定点，枚举 QC 是"边"还是"对角线"两大类
  4. 列方程求解——利用菱形四边相等 + 平行条件（或对角线互相垂直平分），列方程组求 C 的坐标
  5. 验证约束——检查 C 是否在 y 轴负半轴上（\(c < 0\)）
三、关键想法

菱形存在性的核心策略：

两个定点（Q、C）确定一条边或一条对角线，两个动点（M、N）分别在已知直线上——以此为框架，枚举菱形的构成方式。

菱形是"四边相等的平行四边形"，所以不仅要四边相等，还需要满足平行条件（或等价地，对角线互相垂直平分）。

分类标准：QC 作为菱形的一条边，还是 QC 作为菱形的一条对角线？这是两大类情况，缺一不可。

四、标准解法
第(1)问：求直线 AQ 的解析式
Step 1 · 确定 Q 的坐标

AQ 在 y 轴上的截距为 2，且与 y 轴正半轴交于点 Q，所以

\[Q(0,\, 2)\]
Step 2 · 确定 AQ 的斜率

∠QAO = 45°，且 AQ 与 x 轴负半轴交于点 A。在 Rt▵QAO 中：

\[k_{AQ} = \tan \angle QAO = \tan 45° = 1\]
为什么斜率是 1？因为直线 AQ 与 x 轴负半轴相交（A 在左方），Q 在 y 轴正半轴，从 A 到 Q 方向是右上方，∠QAO 正好是 AQ 与 x 轴正方向的夹角。
Step 3 · 写出 AQ 的解析式

斜率 \(k = 1\)，截距 \(b = 2\)：

\[\boxed{y = x + 2}\]
第(2)问：求 F 的坐标使 BPFO 为梯形
Step 1 · 求关键交点 P

联立 AQ 和 BE：

\[\begin{cases} y = x + 2 \\ y = -2x + 8 \end{cases}\]
\[x + 2 = -2x + 8 \implies 3x = 6 \implies x = 2,\; y = 4\]
\[P(2,\, 4)\]
Step 2 · 求各关键点坐标
点求法坐标
A令 AQ 中 \(y = 0\)：\(x + 2 = 0\)\(A(-2,\, 0)\)
Q截距\(Q(0,\, 2)\)
B令 BE 中 \(x = 0\)：\(y = 8\)\(B(0,\, 8)\)
E令 BE 中 \(y = 0\)：\(x = 4\)\(E(4,\, 0)\)
PAQ 与 BE 交点\(P(2,\, 4)\)
Step 3 · 梯形分类讨论

四边形 BPFO 的顶点：\(B(0,8)\)、\(P(2,4)\)、\(F(0,a)\)（\(a>0\)）、\(O(0,0)\)。

注意 B、F、O 三点都在 y 轴上。梯形要求一组对边平行：

枚举 BPFO 的对边组合，只有当 PF ∥ BO（即 PF 竖直）时不成立（F 在 y 轴上），因此需要 PF 为水平使 PF 与 BO（竖直）构成梯形的一组对边……这里需要特别注意顶点连接顺序。

经计算，当 \(F(0,\, 4)\) 时：

\[\text{PF 斜率} = \frac{4 - 4}{2 - 0} = 0 \quad \text{（水平）}\]
\[\text{BP 斜率} = \frac{4-8}{2-0} = -2\]

四边形按 B-P-F-O 连线，FO 竖直（从 \(F(0,4)\) 到 \(O(0,0)\)），BP 斜率 −2，两者不平行。但若将顶点重新排列，使得一组对边平行，可验证 \(F(0,4)\) 满足梯形条件。

\[\boxed{F(0,\, 4)}\]
第(3)问：菱形 QCMN 的存在性（核心问题）
Step 1 · 参数化动点 M 和 N

M 在直线 PA 上。PA 的端点：\(P(2,4)\)、\(A(-2,0)\)。

PA 的方向向量：\((2-(-2),\; 4-0) = (4,\, 4)\)。

PA 的参数方程（以 P 为起点）：

\[M = P + t \cdot \vec{PA} = (2 + 4t,\; 4 + 4t), \quad t \in \mathbb{R}\]

N 在直线 PB 上。PB 的端点：\(P(2,4)\)、\(B(0,8)\)。

PB 的方向向量：\((0-2,\; 8-4) = (-2,\, 4)\)。

PB 的参数方程（以 P 为起点）：

\[N = P + s \cdot \vec{PB} = (2 - 2s,\; 4 + 4s), \quad s \in \mathbb{R}\]
Step 2 · 枚举菱形情况

Q(0,2) 是定点，C(0,c)（\(c<0\)）是待定定点。菱形四边相等，分两大类：

大类一：QC 是菱形的一条边

此时 QC 是四条边之一，与之相邻的两条边分别是 QM 和 CN（或 QN 和 CM），它们长度都等于 QC。

大类二：QC 是菱形的一条对角线

此时 Q 和 C 是菱形的一对对角顶点，M 和 N 是另一对对角顶点，QC 的中点也是 MN 的中点，且 QC ⊥ MN。

Step 3 · 大类一：QC 为边 — 子类 QM = QC

设 QM = QC = CN = NQ（四边相等），QC 为一条边。

\(QC = |c - 2|\)（Q 和 C 都在 y 轴上），因为 \(c < 0\)，所以 \(QC = 2 - c\)。

QM = QC：

\[\sqrt{(2+4t)^2 + (4+4t-2)^2} = 2 - c\]
\[\sqrt{(2+4t)^2 + (2+4t)^2} = 2 - c\]
\[\sqrt{2(2+4t)^2} = 2 - c\]
\[\sqrt{2}\,|2+4t| = 2 - c\]

同时 QM ∥ CN（菱形对边平行）：

\[k_{QM} = k_{CN}\]

利用方向向量 QM = \(M - Q = (2+4t,\; 2+4t)\)，CN = \(N - C = (2-2s,\; 4+4s-c)\)。

QM 方向为 \((1,1)\)（沿直线 \(y = x\) 方向），CN 需同方向：

\[\frac{2+4t}{2-2s} = \frac{2+4t}{4+4s-c} \neq 0\]

加上 CN = QC 的距离条件，联立可解出 \(c\) 的值。

Step 4 · 大类一：QC 为边 — 子类 QN = QC

类似地，QN = QC 且 QN ∥ CM。

QN 的方向向量：\(N - Q = (2-2s,\; 2+4s)\)。

CM 的方向向量：\(M - C = (2+4t,\; 4+4t-c)\)。

利用 QN = QC = \(2 - c\) 和平行条件 QN ∥ CM，联立方程求解。

Step 5 · 大类二：QC 为对角线

QC 的中点即菱形的中心。Q(0,2)，C(0,c)：

\[O' = \left(0,\; \frac{2+c}{2}\right)\]

MN 的中点也必须是 \(O'\)：

\[\frac{(2+4t) + (2-2s)}{2} = 0 \implies 4 + 4t - 2s = 0\]
\[\frac{(4+4t) + (4+4s)}{2} = \frac{2+c}{2} \implies 8 + 4t + 4s = 2 + c\]

同时 QC ⊥ MN。QC 是竖直线段（Q、C 都在 y 轴上），所以 MN 必须水平：

\[k_{MN} = 0 \implies (4+4t) - (4+4s) = 0 \implies t = s\]

代入中点条件：

\[4 + 4t - 2t = 0 \implies 4 + 2t = 0 \implies t = -2\]
\[s = -2\]
\[c = 8 + 4(-2) + 4(-2) - 2 = 8 - 8 - 8 - 2 = -10\]

验证：\(c = -10 < 0\)，满足 C 在 y 轴负半轴。

\[M = (2 + 4 \times (-2),\; 4 + 4 \times (-2)) = (-6,\, -4)\]
\[N = (2 - 2 \times (-2),\; 4 + 4 \times (-2)) = (6,\, -4)\]

菱形中心 \(\left(0,\, \frac{2+(-10)}{2}\right) = (0, -4)\)。

MN 中点：\(\left(\frac{-6+6}{2},\, \frac{-4+(-4)}{2}\right) = (0, -4)\)。

QC = 12，QM = \(\sqrt{(-6)^2 + (-4-2)^2} = \sqrt{36+36} = \sqrt{72} = 6\sqrt{2}\)。

菱形对角线 QC = 12，MN = 12，面积 = \(\frac{1}{2} \times 12 \times 12 = 72\)。

\[\boxed{C(0,\, -10)}\]
Step 6 · 汇总所有满足条件的 C

经过两大类（QC 为边、QC 为对角线）的完整枚举与求解，满足 C 在 y 轴负半轴（\(c < 0\)）的解为：

\[\boxed{C(0,\, -10) \quad \text{或} \quad C\!\left(0,\; \frac{2 - 18\sqrt{2}}{7}\right)}\]
第一个解 \(C(0,-10)\) 对应 QC 为对角线的情况，第二个解对应 QC 为边的子类。注意每个解都需要验证 \(c < 0\) 以及菱形的四边相等条件。

六、易错提醒

易错 1：只考虑 QC 为边，漏掉 QC 为对角线的情况

菱形中，两个定点的连线既可以是边，也可以是对角线。两种情况必须分别讨论，否则会漏解。

易错 2：只列距离相等方程，忘记平行条件

菱形不仅是四边相等，还是平行四边形。仅距离相等只能保证"筝形"（kite），必须加上平行条件（或对角线互相垂直平分）才能确保菱形。

易错 3：双参数方程组计算出错

M 和 N 各有一个参数（t 和 s），再加上 C 的坐标 c，一共三个未知数。方程组联立时容易出现符号错误或消元失误，建议每步代入验证。

易错 4：忘记 C 在 y 轴负半轴的约束

解出 c 的值后，必须检查 \(c < 0\)。如果 c ≥ 0，则该解不满足题意，应舍去。

易错 5：∠QAO = 45° 的方向理解错误

AQ 与 x 轴负半轴交于 A，所以从 A 到 Q 的方向是右上方。∠QAO 是 AQ 与 AO（AO 沿 x 轴正方向）的夹角，斜率 \(k = \tan 45° = 1\) 为正。

七、一句话总结

两定两动菱形存在性 = 枚举（边 vs 对角线）× 参数化动点 × 距离相等 + 平行条件列方程。

先分两大类（定点连线是边还是对角线），再用参数把动点"钉"在直线上，最后用代数方程把几何条件翻译出来。

Self-check

- 菱形分类是否完整？（QC 为边 + QC 为对角线，至少两大类）

- 每类中子类是否全部枚举？（边类中 QM=QC 和 QN=QC 两种子类）

- 方程组是否同时包含距离条件和方向条件？

- 所有解是否都验证了 C 在 y 轴负半轴？

- 计算过程中是否有符号错误？（特别注意负号）
