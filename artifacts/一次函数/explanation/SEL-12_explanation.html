学生版讲解：直线与坐标三角形的面积与等腰问题

原题

$A$、$B$ 分别是 $x$ 轴上位于原点左右两侧的点，点 $P(2,p)$ 在第一象限，直线 $PA$ 交 $y$ 轴于点 $C(0,2)$，直线 $PB$ 交 $y$ 轴于点 $D$，$\triangle AOP$ 的面积为 $6$。

（1）求点 $A$ 的坐标；

（2）求点 $P$ 的坐标；

（3）若 $\triangle BOP$ 是以 $OP$ 为腰的等腰三角形，直接写出点 $D$ 坐标。

一、这题先看懂什么

闭上眼睛想象一下这幅图：

  • x 轴上，原点左边有个点 $A$，右边有个点 $B$。
  • 第一象限里有个点 $P$，横坐标是 $2$，纵坐标是个未知数 $p$。
  • 从 $P$ 向左连线到 $A$，这条直线会穿过 $y$ 轴上的点 $C(0,2)$。
  • 从 $P$ 向右连线到 $B$，这条直线穿过 $y$ 轴于点 $D$（$D$ 的坐标是我们最终要求的东西之一）。
  • $\triangle AOP$（由 $A$、原点 $O$、$P$ 三个点组成的三角形）的面积是 $6$。

想一想：$A$ 在原点左侧，所以 $A$ 的横坐标是负数。$B$ 在原点右侧，横坐标是正数。画图时这一点千万别搞反。

二、第一步该做什么

三道小题是接力的：第（1）问的结果帮第（2）问，第（2）问的结果帮第（3）问。

所以我们要从三角形面积入手。面积公式你知道：$\dfrac{1}{2} \times \text{底} \times \text{高}$。

关键问题：$\triangle AOP$ 的"底"和"高"分别对应哪些坐标分量？

我们选择以 $OA$ 为底（$OA$ 在 $x$ 轴上），那么高就是 $P$ 点到 $x$ 轴的距离，也就是 $P$ 的纵坐标 $p$。

于是： $$S_{\triangle AOP} = \frac{1}{2} \times |OA| \times p = 6$$

其中 $|OA|$ 是 $A$ 到原点的距离（$A$ 在负半轴，所以 $|OA| = -x_A$）。

这个方程里有两个未知量：$|OA|$ 和 $p$。一个方程解不出两个未知数——我们还需要另一个条件。

另一个条件就是：$C(0,2)$ 在直线 $PA$ 上。

三、关键转化怎么想

题目给了一条直线上的两个已知信息：

  • 这条直线过 $C(0,2)$
  • 这条直线还过 $A$（横坐标待求）和 $P(2,p)$

当一条直线过两个点时，斜率相等就是最直接的条件。我们用 $C$ 和 $P$ 的坐标写出斜率，再用 $A$ 和 $P$ 的坐标写出斜率，令它们相等：

$$k_{CP} = k_{AP}$$

但是 $A$ 的坐标和 $p$ 都是未知的——似乎还是两个未知数？

别急，别忘了面积方程 $\dfrac{1}{2} \times |OA| \times p = 6$ 已经给出了 $|OA|$ 和 $p$ 的关系。把 $|OA|$ 用 $p$ 表示，代入斜率方程，就能求出 $p$。

核心思路：面积方程把"距离"和"坐标"连起来，斜率方程把"共线"和"坐标"连起来——两个方程，两个未知数，搞定。

四、标准解法

第（1）问：求 $A$ 的坐标

Step 1：列出面积方程。

$A$ 在 $x$ 轴负半轴，设 $A(-a,0)$，其中 $a>0$。

$$\frac{1}{2} \times a \times p = 6 \quad \Rightarrow \quad a \cdot p = 12 \quad \cdots (*)$$

Step 2：利用 $C(0,2)$ 在直线 $PA$ 上，列斜率等式。

$$\frac{2 - p}{0 - 2} = \frac{0 - p}{-a - 2}$$

Step 3：化简斜率方程。

$$\frac{2-p}{-2} = \frac{-p}{-a-2}$$

$$\frac{2-p}{-2} = \frac{p}{a+2}$$

交叉相乘：

$$(2-p)(a+2) = -2p$$

展开：

$$2a + 4 - pa - 2p = -2p$$

$$2a + 4 - pa = 0$$

$$a(2 - p) = -4$$

$$a = \frac{-4}{2-p} = \frac{4}{p-2} \quad \cdots (**)$$

Step 4：联立 $(*)$ 和 $(**)$。

由 $(*)$：$a = \dfrac{12}{p}$，代入 $(**)$：

$$\frac{12}{p} = \frac{4}{p-2}$$

交叉相乘：

$$12(p-2) = 4p$$

$$12p - 24 = 4p$$

$$8p = 24$$

$$p = 3$$

Step 5：代回求 $a$。

$$a = \frac{12}{3} = 4$$

所以 $A(-4,0)$。

想一想：如果我把以 $OA$ 为底改为以 $OP$ 为底，面积方程会变成什么样？答案是 $\frac{1}{2} \times 2 \times (\text{$A$的纵坐标})$，但 $A$ 在 $x$ 轴上，纵坐标为 $0$——这说明底和高必须垂直，选底时要看哪个边是水平/竖直的。

第（2）问：求 $P$ 的坐标

第（1）问已经求出 $p=3$，所以：

$$P(2,3)$$

第（3）问：等腰三角形分类讨论求 $D$

条件：$\triangle BOP$ 以 $OP$ 为腰的等腰三角形。

"以 $OP$ 为腰"的意思是 $OP$ 是等腰三角形的一条相等的边。等腰三角形有两条相等的边，所以有两种情况：

情况一：$OP = OB$

先算 $OP$ 的长度：

$$OP = \sqrt{2^2 + 3^2} = \sqrt{13}$$

所以 $OB = \sqrt{13}$，$B$ 在 $x$ 轴正半轴，$B(\sqrt{13}, 0)$。

求直线 $PB$：过 $P(2,3)$ 和 $B(\sqrt{13},0)$。

斜率：$k = \dfrac{0-3}{\sqrt{13}-2} = \dfrac{-3}{\sqrt{13}-2}$

直线方程（点斜式）：$y - 3 = \dfrac{-3}{\sqrt{13}-2}(x - 2)$

令 $x=0$：

$$y = 3 + \dfrac{-3}{\sqrt{13}-2}(-2) = 3 + \dfrac{6}{\sqrt{13}-2}$$

分子分母同乘 $(\sqrt{13}+2)$：

$$y = 3 + \dfrac{6(\sqrt{13}+2)}{13-4} = 3 + \dfrac{6(\sqrt{13}+2)}{9} = 3 + \dfrac{2(\sqrt{13}+2)}{3}$$

$$y = \frac{9 + 2\sqrt{13} + 4}{3} = \frac{13 + 2\sqrt{13}}{3}$$

所以 $D\left(0, \dfrac{13+2\sqrt{13}}{3}\right)$。

情况二：$OP = PB$

$OP = \sqrt{13}$，所以 $PB = \sqrt{13}$。

设 $B(b,0)$（$b>0$），则：

$$(b-2)^2 + (0-3)^2 = 13$$

$$(b-2)^2 + 9 = 13$$

$$(b-2)^2 = 4$$

$b-2 = 2$ 或 $b-2 = -2$

$b = 4$ 或 $b = 0$

$b=0$ 就是原点 $O$，排除。所以 $B(4,0)$。

求直线 $PB$：过 $P(2,3)$ 和 $B(4,0)$。

斜率：$k = \dfrac{0-3}{4-2} = -\dfrac{3}{2}$

直线方程：$y - 3 = -\dfrac{3}{2}(x-2)$

令 $x=0$：$y = 3 + 3 = 6$

所以 $D(0,6)$。

综上：

$$D(0,6) \quad \text{或} \quad D\left(0, \frac{13+2\sqrt{13}}{3}\right)$$

五、边讲边问

  1. $\triangle AOP$ 中，如果以 $OA$ 为底，高是什么？如果以 $OP$ 为底，高是什么？为什么我们选择以 $OA$ 为底？

  2. 面积方程 $\frac{1}{2} \times a \times p = 6$ 里有两个未知量，为什么我们不能从这里直接求出 $a$ 和 $p$？还需要什么条件？

  3. "等腰三角形以 $OP$ 为腰"这句话，你能想到哪两种等腰三角形的形状？画个草图试试。

  4. 第（3）问中，$OP=PB$ 时解出 $b=4$ 和 $b=0$，为什么 $b=0$ 要舍去？

  5. 如果题目改为"以 $OB$ 为腰"的等腰三角形，情况会变成哪几种？

六、易错提醒

  • 底高的选取：面积公式中底和高必须垂直。$\triangle AOP$ 中 $OA$ 在 $x$ 轴上（水平的），高就是竖直方向——$P$ 的纵坐标 $p$。不要把"高"想成 $OP$ 的长度。
  • 分类讨论要全面：等腰三角形有两条相等的边。"以 $OP$ 为腰"意味着 $OP$ 是其中一条腰，另一条腰可以是 $OB$ 也可以是 $BP$，不要漏掉任何一种情况。
  • 根式化简要仔细：$OP=OB$ 的情况中，求 $D$ 的纵坐标涉及分母有理化，交叉相乘和符号容易出错。建议每一步都验算一下。

七、一句话总结

面积方程定关系，斜率方程定坐标；等腰分两类，腰相等要全列，舍去不合理的解。
