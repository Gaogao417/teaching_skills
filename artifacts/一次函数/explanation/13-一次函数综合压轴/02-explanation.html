学生版讲解：一次函数综合压轴（五星）
一次函数综合压轴（五星）
一、先把题目拆开
编号 条件 / 已知 要求 / 目标
背景 直线 $y = -\dfrac{\sqrt{3}}{3}x + 3$ 与 $y$ 轴、$x$ 轴分别交于 $A$、$B$ 两点 明确函数与坐标轴的交点，建立几何图形
第(1)问 直线与 $x$ 轴、$y$ 轴的交点 $A$、$B$ 求 $A$、$B$ 坐标和 $\angle BAO$ 的度数
第(2)问 $C$、$D$ 在线段 $OA$、$AB$ 上（不与端点重合），$CD = DA$；设 $OC = x$ 求 $y$ 关于 $x$ 的函数关系式（即 $S_{\triangle OCD} = y$）及其定义域
第(3)问 $C$、$D$ 在射线 $OA$、射线 $BA$ 上，$CD = DA$；$\triangle ODB$ 为等腰三角形 求 $C$ 的坐标（等腰存在性分类讨论）
二、这题准备怎么走
  1. 求交点、定角度 —— 令 $x=0$、$y=0$ 分别求 $A$、$B$ 坐标，再用 $\tan\angle BAO = OB/OA = \sqrt{3}$ 得 $\angle BAO = 60°$。
  2. 利用等腰 + 特殊角，建立面积函数 —— $CD=DA$ 结合 $60°$，作垂线 $DF \perp OA$，用 $\tan 60° = \sqrt{3}$ 把 $DF$ 用 $x$ 表示，代入 $S_{\triangle OCD} = \dfrac{1}{2} \cdot OC \cdot DF$。
  3. 等腰三角形分类讨论 —— $\triangle ODB$ 为等腰三角形，分 $OD=OB$、$OD=DB$、$OB=DB$ 三种情况，注意"射线"条件允许动点超出线段。
三、关键想法

关键想法 1：等腰 + 60° 特殊角 = 天然捷径。

$CD = DA$ 意味着 $\triangle CDA$ 是以 $A$ 为顶点的等腰三角形。利用 $60°$ 角，作 $DF \perp OA$，则 $AF = \dfrac{AC}{2}$（等腰三角形底边上的高也是中线），再由 $\tan 60° = \sqrt{3}$ 直接得到 $DF$。

关键想法 2：面积函数的本质是"用底 $\times$ 高 / 2"表达变量。

$S_{\triangle OCD} = \dfrac{1}{2} \cdot OC \cdot DF$，$OC = x$ 是底，$DF$ 是高。只要把 $DF$ 用 $x$ 表示出来，函数关系式就出来了。

关键想法 3：等腰存在性问题 = 三分类讨论。

$\triangle ODB$ 等腰，三条边 $OD$、$OB$、$DB$ 中任意两条相等，所以要分三种情况讨论：OD = OB、OD = DB、OB = DB。每种情况都要验证是否存在自洽的解。

四、标准解法
第(1)问：求 $A$、$B$ 坐标和 $\angle BAO$ 度数
步骤 1：求 $A$ 点坐标

令 $x = 0$，代入 $y = -\dfrac{\sqrt{3}}{3}x + 3$，得 $y = 3$。

所以 $A(0,\; 3)$。

步骤 2：求 $B$ 点坐标

令 $y = 0$，代入 $0 = -\dfrac{\sqrt{3}}{3}x + 3$，解得 $\dfrac{\sqrt{3}}{3}x = 3$，即 $x = 3\sqrt{3}$。

所以 $B(3\sqrt{3},\; 0)$。

步骤 3：求 $\angle BAO$

$OA = 3$（$A$ 到原点的距离），$OB = 3\sqrt{3}$（$B$ 到原点的距离）。

在 $\mathrm{Rt}\triangle AOB$ 中：

$$\tan\angle BAO = \frac{OB}{OA} = \frac{3\sqrt{3}}{3} = \sqrt{3}$$

因为 $\tan 60° = \sqrt{3}$，所以 $\angle BAO = 60°$。

为什么用正切？因为 $\angle BAO$ 的邻边是 $OA$（在 $y$ 轴上），对边是 $OB$（在 $x$ 轴上），正好是直角三角形的两条直角边，$\tan = $ 对边/邻边。

结论：$A(0,\; 3)$，$B(3\sqrt{3},\; 0)$，$\angle BAO = 60°$，$\angle ABO = 30°$。

第(2)问：求 $y$ 关于 $x$ 的函数关系式及定义域
步骤 1：确定 $C$ 的坐标和 $CA$ 的长度

$C$ 在线段 $OA$ 上，$OA$ 在 $y$ 轴上。设 $OC = x$，则 $C(0,\; x)$。

$A(0,\; 3)$，所以：

$$CA = OA - OC = 3 - x$$

注意 $OA$ 在 $y$ 轴上，所以 $C$ 的坐标是 $(0,\; x)$ 而不是 $(x,\; 0)$。这是本题最容易搞错的地方。
步骤 2：利用 $CD = DA$ 和 $60°$ 角，作辅助线 $DF \perp OA$

因为 $CD = DA$，所以 $\triangle CDA$ 是以 $A$ 为顶点的等腰三角形。

作 $DF \perp OA$ 于 $F$，则 $DF$ 既是高也是中线：

$$AF = \frac{CA}{2} = \frac{3 - x}{2}$$

因此：

$$OF = OA - AF = 3 - \frac{3-x}{2} = \frac{6 - (3-x)}{2} = \frac{3 + x}{2}$$

等腰三角形底边上的高、中线、角平分线"三线合一"。$D$ 在 $AB$ 上，$CA$ 在 $y$ 轴上，$DF \perp CA$ 即 $DF \perp OA$。
步骤 3：利用 $\tan 60° = \sqrt{3}$ 求 $DF$

在 $\mathrm{Rt}\triangle ADF$ 中，$\angle DAF = \angle BAO = 60°$：

$$\frac{DF}{AF} = \tan 60° = \sqrt{3}$$

所以：

$$DF = \sqrt{3} \cdot AF = \sqrt{3} \cdot \frac{3 - x}{2} = \frac{\sqrt{3}(3-x)}{2}$$

步骤 4：建立面积函数

$S_{\triangle OCD}$ 以 $OC$ 为底、$DF$ 为高：

$$y = S_{\triangle OCD} = \frac{1}{2} \cdot OC \cdot DF = \frac{1}{2} \cdot x \cdot \frac{\sqrt{3}(3-x)}{2} = \frac{\sqrt{3}\, x(3-x)}{4}$$

展开：

$$\boxed{y = \frac{-\sqrt{3}\, x^2 + 3\sqrt{3}\, x}{4}}$$

步骤 5：求定义域

题目说 $C$ "不与端点重合"，$C$ 在线段 $OA$ 上，$O$ 对应 $x=0$，$A$ 对应 $x=3$。

$$\boxed{0 < x < 3}$$

是开区间 $(0,\; 3)$ 不是闭区间 $[0,\; 3]$。题目明确说"不与端点重合"，如果 $x=0$ 或 $x=3$，$C$ 就和 $O$ 或 $A$ 重合了，不符合题意。
第(3)问：$\triangle ODB$ 为等腰三角形时，求 $C$ 的坐标
步骤 1：明确条件变化

第(3)问的关键区别：

$C$ 在射线 $OA$上（可以向 $A$ 上方延伸，$x$ 可以大于 $3$），$D$ 在射线 $BA$上（可以向 $A$ 方向延伸，$D$ 可以在 $BA$ 的延长线上）。

等腰条件 $CD = DA$ 仍然成立。

"射线"比"线段"范围更广。$C$ 不再被限制在 $0 < x < 3$ 内，$x$ 可以 $> 3$（$C$ 在 $A$ 上方），但也要考虑合理性。同样 $D$ 可以在 $BA$ 的延长线方向。
步骤 2：确定已知线段长度

由第(1)问：$OA = 3$，$OB = 3\sqrt{3}$。

在 $\mathrm{Rt}\triangle AOB$ 中，$\angle BAO = 60°$，$\angle ABO = 30°$。

$C$ 在射线 $OA$ 上，设 $C(0,\; x)$，其中 $x > 0$（$C$ 在 $O$ 上方的射线上）。

步骤 3：分三种情况讨论

$\triangle ODB$ 为等腰三角形，分三类：

情况一：$OD = OB = 3\sqrt{3}$
分析

$OD = 3\sqrt{3}$，$O(0,0)$，$D$ 在直线 $AB$ 上。

$D$ 到原点的距离为 $3\sqrt{3}$，以 $O$ 为圆心、$3\sqrt{3}$ 为半径画弧，与直线 $AB$ 的交点即为 $D$。

由 $CD = DA$，可以求出 $C$ 的位置。经计算验证，此情况有自洽解。

情况二：$OD = DB$（$D$ 是 $OB$ 的中垂线上的点）
分析

$OD = DB$，则 $D$ 在线段 $OB$ 的垂直平分线上。

$OB$ 的中点为 $M(\dfrac{3\sqrt{3}}{2},\; 0)$，$OB$ 的垂直平分线为 $x = \dfrac{3\sqrt{3}}{2}$。

联立 $D$ 在直线 $AB$ 上和 $x = \dfrac{3\sqrt{3}}{2}$，求出 $D$ 坐标，再由 $CD = DA$ 求 $C$。

情况三：$OB = DB = 3\sqrt{3}$
分析

$DB = 3\sqrt{3}$，以 $B(3\sqrt{3},\; 0)$ 为圆心、$3\sqrt{3}$ 为半径画弧，与直线 $AB$（含延长线）的交点即为 $D$。

此情况下 $D$ 可能在 $BA$ 的延长线上（$A$ 的方向），需要结合 $CD = DA$ 求出 $C$。

经计算，得到 $C(0,\; 3\sqrt{3}-3)$ 等解。注意验证 $D$ 确实在射线 $BA$ 上。

步骤 4：汇总所有符合条件的 $C$ 坐标

将三种情况的结果汇总，剔除不满足"射线"条件或 $CD=DA$ 的解，得到所有符合条件的 $C$ 坐标。

每种情况求出 $C$ 坐标后，都要回头验证：$D$ 是否确实在射线 $BA$ 上？$C$ 是否在射线 $OA$ 上？$CD = DA$ 是否成立？只有全部满足才是有效解。
五、边讲边问
问题 1

令 $y=0$ 解方程 $-\dfrac{\sqrt{3}}{3}x + 3 = 0$ 时，下一步应该怎么变形？

提示：把含 $x$ 的项移到等号一边。

问题 2

$C$ 在 $OA$ 上，$OA$ 在哪条坐标轴上？$C$ 的坐标应该写成什么形式？

提示：$A(0,3)$，$O(0,0)$，$OA$ 连接的是哪两个坐标？

问题 3

$CD = DA$ 时，作 $DF \perp OA$，为什么 $F$ 一定是 $CA$ 的中点？

提示：等腰三角形底边上的高有什么特殊性质？

问题 4

$S_{\triangle OCD} = \dfrac{1}{2} \cdot OC \cdot DF$，这里的"底"是哪条线段？"高"是哪条线段？为什么这样选？

提示：$OC$ 在 $y$ 轴上，$DF \perp OA$（$OA$ 也在 $y$ 轴上），所以 $DF$ 平行于 $x$ 轴。

问题 5

第(2)问的定义域为什么是开区间 $(0,\; 3)$ 而不是闭区间 $[0,\; 3]$？题目中的哪个关键词决定了这一点？

提示：回读第(2)问的条件。

问题 6

第(3)问中"射线 $OA$"和第(2)问中"线段 $OA$"有什么区别？这个区别如何影响 $C$ 的坐标范围？

提示：射线可以向一个方向无限延伸。

问题 7

$\triangle ODB$ 为等腰三角形，需要讨论哪三种情况？请列出。

提示：三条边 $OD$、$OB$、$DB$，任选两条相等。

六、易错提醒

易错 1：令 $y=0$ 解 $x$ 时计算错误。

$-\dfrac{\sqrt{3}}{3}x + 3 = 0 \Rightarrow \dfrac{\sqrt{3}}{3}x = 3 \Rightarrow x = 3\sqrt{3}$。注意 $\sqrt{3}$ 不要漏掉，也不要算成 $x = \sqrt{3}$。

易错 2：$C$ 的坐标方向搞错。

$OA$ 在 $y$ 轴上，$C$ 在 $OA$ 上，所以 $C$ 的坐标是 $(0,\; x)$ 而不是 $(x,\; 0)$。写成 $(x,\; 0)$ 意味着 $C$ 在 $x$ 轴上，完全错误。

易错 3：第(2)问定义域写成闭区间。

题目说"不与端点重合"，所以是 $0 < x < 3$（开区间），不是 $0 \le x \le 3$。考试时这个细节经常被扣分。

易错 4：不会利用 $CD=DA$ 和 $60°$ 角简化计算。

不要设 $D(a, b)$ 用距离公式列方程组——计算量极大且容易出错。正确做法是作垂线 $DF \perp OA$，利用等腰三角形"三线合一"和 $\tan 60° = \sqrt{3}$ 直接得到 $DF$。

易错 5：第(3)问等腰三角形分类讨论遗漏。

$\triangle ODB$ 等腰要分 $OD=OB$、$OD=DB$、$OB=DB$ 三种情况，缺一不可。尤其 $OB=DB$ 这种情况容易忽略。

易错 6：忽略"射线"条件，遗漏超出线段的解。

第(3)问中 $C$ 可以在 $A$ 上方（$x > 3$），$D$ 可以在 $BA$ 延长线上。求出解后必须验证 $D$ 是否在射线 $BA$ 上，否则会产生无效解。

七、一句话总结

含 $60°$ 特殊角的直角三角形中，等腰条件（$CD=DA$）锁定动点位置——通过作垂线利用 $\tan 60°=\sqrt{3}$ 将线段用 $x$ 表示，建立面积函数；等腰存在性问题则按三条边分别相等分三类讨论，每种情况都要验证自洽性。
