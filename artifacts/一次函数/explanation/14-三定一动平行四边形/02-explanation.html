学生版讲解：三定一动—平行四边形存在性

三定一动—平行四边形存在性

一、先把题目拆开

小问 已知条件 要求目标 关键词
第(1)问 一次函数 \(y = -2x + 4\) 的图象与 \(x\) 轴交于点 A，与 \(y\) 轴交于点 B 求点 A、点 B 的坐标 坐标轴交点
第(2)问 C 在线段 AB 上，O 为原点，D 在第二象限，四边形 BCOD 为菱形 求点 D 的坐标 菱形、四边相等
第(3)问 在(2)的条件下，以 B、D、A、P 为顶点的四边形是平行四边形 直接写出所有满足条件的 P 点坐标 平行四边形存在性、三定一动

二、这题准备怎么走

  1. 第(1)问：令 \(y = 0\) 解出 A 的横坐标，令 \(x = 0\) 解出 B 的纵坐标 —— 基础送分题。
  2. 第(2)问：设 C 在 AB 上用参数表示，由菱形四边相等（\(|BC| = |CO|\)）列方程求出 C，再由 \(|OC| = |OD|\) 且 D 在第二象限求出 D。
  3. 第(3)问：核心步骤！三个顶点 B、D、A 已知，用中点坐标法分别以 BD、DA、AB 为对角线，求 P 的三种情况，共 3 个解。

三、关键想法

中点坐标法是平行四边形存在性问题的"万能钥匙"。

平行四边形有一个核心性质：对角线互相平分。也就是说，两条对角线的中点坐标一定相等。

对于三定一动问题（三个顶点已知，求第四个顶点 P），从 3 个定点中任选 2 个作为一条对角线的端点，剩下的 1 个定点和 P 构成另一条对角线 —— 共有 \(C_3^2 = 3\) 种搭配方式，每种搭配列一组中点相等的方程，就能求出 P 的全部可能坐标。

为什么不用画图找点？画图数格子容易遗漏，尤其是 P 点落在第二、第三象限时很难精确读出坐标。中点坐标法是纯代数方法，保证不遗漏、不重复。

四、标准解法

第(1)问：求 A、B 坐标

Step 1：令 \(y = 0\)，求 A 点坐标

把 \(y = 0\) 代入 \(y = -2x + 4\)：

\[0 = -2x + 4 \quad \Longrightarrow \quad x = 2\]

所以 \(\boxed{A(2,\ 0)}\)。

Step 2：令 \(x = 0\)，求 B 点坐标

把 \(x = 0\) 代入 \(y = -2x + 4\)：

\[y = -2 \times 0 + 4 = 4\]

所以 \(\boxed{B(0,\ 4)}\)。

第(2)问：求 D 坐标（菱形 BCOD）

Step 1：参数化线段 AB 上的点 C

A(2, 0) 到 B(0, 4) 的方向向量为 \((-2,\ 4)\)。设 C 将 AB 分为比例 \(s : (1-s)\)（\(0 \leq s \leq 1\)），则：

\[C = (2 - 2s,\ 4s)\]

Step 2：由菱形条件 \(|BC| = |CO|\) 列方程

菱形四边相等，所以邻边 \(|BC| = |CO|\)：

\[ |BC|^2 = (-2s)^2 + (4 - 4s)^2 = 4s^2 + 16(1-s)^2 \]

\[ |CO|^2 = (2-2s)^2 + (4s)^2 = 4(1-s)^2 + 16s^2 \]

令 \(|BC| = |CO|\)，即 \(|BC|^2 = |CO|^2\)：

\[ 4s^2 + 16(1-s)^2 = 4(1-s)^2 + 16s^2 \]

\[ 12(1-s)^2 = 12s^2 \quad \Longrightarrow \quad (1-s)^2 = s^2 \]

\[ 1 - 2s + s^2 = s^2 \quad \Longrightarrow \quad s = \frac{1}{2} \]

代入得 \(C(1,\ 2)\)。

Step 3：由 \(|OC| = |OD|\) 且 D 在第二象限求 D

\(|OC| = \sqrt{1^2 + 2^2} = \sqrt{5}\)。菱形中 \(|OD| = |OC| = \sqrt{5}\)，且 \(|CD| = |BC| = \sqrt{(-1)^2 + 2^2} = \sqrt{5}\)。

由于 O(0, 0)、C(1, 2)，菱形对角线互相垂直平分，中点为 \((\frac{1}{2},\ 1)\)。D 关于中点与 O 对称：

\[ D = (1 - 0,\ 2 - 0) = (1,\ 2) \quad \text{方向延长} \]

更直接地：O 和 D 关于中点 \((\frac{1}{2},\ 1)\) 对称，所以 \(D = (2 \times \frac{1}{2} - 0,\ 2 \times 1 - 0) = (1,\ 2)\)？不对，需要仔细核实。

正确做法：C(1, 2)，中点 M(1/2, 1)。D 满足 \(|OD| = \sqrt{5}\) 且 \(|MD| = |MC| = \frac{\sqrt{5}}{2}\)，D 在第二象限。

\[ D = (2 \times \tfrac{1}{2} - 0,\ 2 \times 1 - 0) = (1,\ 2) \quad \text{不行，这不满足对称关系} \]

重新推导：菱形 BCOD 中，对角线 BD 与 OC 互相平分。O(0,0) 和 C(1,2) 的中点为 M(1/2, 1)。B(0,4) 和 D 的中点也为 M，所以：

\[ \frac{0 + x_D}{2} = \frac{1}{2},\quad \frac{4 + y_D}{2} = 1 \]

\[ x_D = 1,\quad y_D = -2 \]

但这与"D 在第二象限"矛盾（\(y_D = -2\) 在第三象限方向）。重新检查：菱形 BCOD 的对角线是哪两条？

菱形 BCOD 的四条边是 BC、CO、OD、DB。对角线是 BO 和 CD。中点 M 同时是 BO 和 CD 的中点：

\[ M_{BO} = \left(\frac{0+0}{2},\ \frac{4+0}{2}\right) = (0,\ 2) \]

\[ M_{CD} = \left(\frac{1+x_D}{2},\ \frac{2+y_D}{2}\right) = (0,\ 2) \]

\[ \frac{1+x_D}{2} = 0 \Longrightarrow x_D = -1;\quad \frac{2+y_D}{2} = 2 \Longrightarrow y_D = 2 \]

所以 \(\boxed{D(-1,\ 2)}\)，\(x_D = -1 < 0\)，\(y_D = 2 > 0\)，确实在第二象限，验证通过。

第(3)问：求所有 P 点坐标（平行四边形 BDAP 存在性）

已知三个定点：\(B(0,\ 4)\)、\(D(-1,\ 2)\)、\(A(2,\ 0)\)。要求以 B、D、A、P 为顶点的四边形是平行四边形，求所有 P。

核心方法：中点坐标法 —— 平行四边形对角线互相平分。

情况一：以 BD 为一条对角线，AP 为另一条对角线

此时 B 和 D 是对角线的两个端点，A 和 P 是另一条对角线的两个端点。两条对角线的中点相同：

\[ M_{BD} = \left(\frac{0+(-1)}{2},\ \frac{4+2}{2}\right) = \left(-\frac{1}{2},\ 3\right) \]

\[ M_{AP} = \left(\frac{2+x_P}{2},\ \frac{0+y_P}{2}\right) = \left(-\frac{1}{2},\ 3\right) \]

\[ \frac{2+x_P}{2} = -\frac{1}{2} \Longrightarrow x_P = -3;\quad \frac{y_P}{2} = 3 \Longrightarrow y_P = 6 \]

所以 \(\boxed{P_1(-3,\ 6)}\)。

情况二：以 DA 为一条对角线，BP 为另一条对角线

\[ M_{DA} = \left(\frac{-1+2}{2},\ \frac{2+0}{2}\right) = \left(\frac{1}{2},\ 1\right) \]

\[ M_{BP} = \left(\frac{0+x_P}{2},\ \frac{4+y_P}{2}\right) = \left(\frac{1}{2},\ 1\right) \]

\[ \frac{x_P}{2} = \frac{1}{2} \Longrightarrow x_P = 1;\quad \frac{4+y_P}{2} = 1 \Longrightarrow y_P = -2 \]

所以 \(\boxed{P_2(1,\ -2)}\)。

情况三：以 AB 为一条对角线，DP 为另一条对角线

\[ M_{AB} = \left(\frac{0+2}{2},\ \frac{4+0}{2}\right) = (1,\ 2) \]

\[ M_{DP} = \left(\frac{-1+x_P}{2},\ \frac{2+y_P}{2}\right) = (1,\ 2) \]

\[ \frac{-1+x_P}{2} = 1 \Longrightarrow x_P = 3;\quad \frac{2+y_P}{2} = 2 \Longrightarrow y_P = 2 \]

所以 \(\boxed{P_3(3,\ 2)}\)。

汇总

情况 对角线配对 P 的坐标
一 BD 与 AP \(P(-3,\ 6)\)
二 DA 与 BP \(P(1,\ -2)\)
三 AB 与 DP \(P(3,\ 2)\)

六、易错提醒

错误 1：混淆菱形的对角线

菱形 BCOD 的四条边依次是 B→C→O→D→B，对角线是 BO 和 CD，不是 BD 和 OC。搞错对角线会导致中点算错，D 的坐标完全不对。

方法：菱形中，对角线连接的是"隔一个顶点"的两个顶点。画图标注四条边的顺序后，自然能看清对角线。

错误 2：只找 1~2 个 P 点就停了

画图数格子时，\(P(3,\ 2)\) 在第一象限很好找，但 \(P(-3,\ 6)\) 在第二象限、\(P(1,\ -2)\) 在第四象限，很容易漏掉。

方法：用中点坐标法，系统列出 3 种配对，每种算一个方程组，一个都不会漏。

错误 3：忘记检查象限条件

题目说"D 在第二象限"，如果求出的 D 不在第二象限，说明中间某个步骤出了问题。类似的，第(3)问的三个 P 点分别在第二、第四、第一象限，需要逐一确认。

错误 4：参数化时范围不清

第(2)问设 C 在 AB 上，参数 \(s\) 的范围是 \(0 \leq s \leq 1\)。如果解出来的 \(s\) 不在这个范围内，说明该解无效（C 不在线段 AB 上）。

七、一句话总结

三定一动平行四边形：3 个定点配出 \(C_3^2 = 3\) 条对角线，每条对角线对应一个中点方程，中点坐标法一拉就出，不遗漏不重复。
