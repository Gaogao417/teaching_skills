学生版讲解：一次函数值与不等式

题目：已知一次函数解析式是 \(y = \dfrac{1}{2}x - 3\)
(1) 当 \(x\) 取何值时，\(y = 2\)？
(2) 当 \(x\) 取何值时，\(y > 2\)？
(3) 当 \(x\) 取何值时，\(y < 2\)？
(4) 当 \(x\) 取何值时，\(0 < y < 2\)？

一、这题先看懂什么

这道题给了一个一次函数的式子，然后问四件事。前三个问题很好懂：

  • 第(1)问：让函数值等于 2，问此时 \(x\) 是多少。
  • 第(2)问：让函数值大于 2，问 \(x\) 在什么范围内。
  • 第(3)问：让函数值小于 2，问 \(x\) 在什么范围内。

第(4)问多了一层——要求函数值在 0 和 2 之间，同时满足两个条件。

二、第一步该做什么

很多同学看到 \(y = 2\)、\(y > 2\)，会犹豫：这是算函数，还是解不等式？

答案是：把 \(y\) 的条件，全部"塞回"函数式子里去。

因为题目给了 \(y = \dfrac{1}{2}x - 3\)，所以把 2 代替 \(y\) 的位置，问题就变成：

\(\dfrac{1}{2}x - 3 \quad\text{分别}\quad = 2,\; > 2,\; < 2\)

这样，一道函数题就变成了你学过的方程和不等式。

三、关键转化怎么想

核心动作就一句话："把 y 换成条件数"。

具体来说：

  • \(y = 2\) 代入后 \(\longrightarrow\) \(\dfrac{1}{2}x - 3 = 2\)（一元一次方程）
  • \(y > 2\) 代入后 \(\longrightarrow\) \(\dfrac{1}{2}x - 3 > 2\)（一元一次不等式）
  • \(y < 2\) 代入后 \(\longrightarrow\) \(\dfrac{1}{2}x - 3 < 2\)（一元一次不等式）
  • \(0 < y < 2\) 代入后 \(\longrightarrow\) \(0 < \dfrac{1}{2}x - 3 < 2\)（复合不等式）

注意：复合不等式要拆成两半分别解，最后取交集。

四、标准解法

第(1)问：\(y = 2\) 时，\(x =\)？

做：把 \(y = 2\) 代入 \(y = \dfrac{1}{2}x - 3\)：

\(2 = \dfrac{1}{2}x - 3\)
\(\dfrac{1}{2}x = 5\)
\(x = 10\)

为什么这么做："\(y = \)" 就是告诉你函数值等于多少，把它塞进去就是一元一次方程，直接解。

\(x = 10\) 是后面的"分界点"，要记住。

第(2)问：\(y > 2\) 时，\(x\) 的范围？

做：把 \(y = 2\) 改成 \(y > 2\)：

\(\dfrac{1}{2}x - 3 > 2\)
\(\dfrac{1}{2}x > 5\)
\(x > 10\)

为什么不等号不翻转：两边乘 2（正数），不等号方向不变。

快捷想法：\(x = \frac{1}{2}x - 3\) 的系数 \(\frac{1}{2} > 0\)，函数是递增的。所以 \(y\) 大了，\(x\) 也应该大 \(\longrightarrow\) 直接写 \(x > 10\)。

第(3)问：\(y < 2\) 时，\(x\) 的范围？

做：同理：

\(\dfrac{1}{2}x - 3 < 2 \;\Longrightarrow\; x < 10\)

快捷想法：递增函数，\(y\) 小了，\(x\) 也小 \(\longrightarrow\) \(x < 10\)。

第(4)问：\(0 < y < 2\) 时，\(x\) 的范围？

做：代入后得复合不等式：

\(0 < \dfrac{1}{2}x - 3 < 2\)

拆成两半分别解：

左半：\(0 < \dfrac{1}{2}x - 3 \;\Longrightarrow\; \dfrac{1}{2}x > 3 \;\Longrightarrow\; x > 6\)
右半：\(\dfrac{1}{2}x - 3 < 2 \;\Longrightarrow\; \dfrac{1}{2}x < 5 \;\Longrightarrow\; x < 10\)

取交集：两个条件同时满足 \(\Longrightarrow\) \(6 < x < 10\)。

检查：\(x = 6\) 时 \(y = 0\)（下界取等），\(x = 10\) 时 \(y = 2\)（上界取等），所以都是开区间，正确。

五、边讲边问

  1. 函数 \(y = \dfrac{1}{2}x - 3\) 的系数 \(\dfrac{1}{2}\) 是正数还是负数？这说明函数图像是上升还是下降？
  2. 为什么第(2)问可以直接写 \(x > 10\)，而不需要重新从头算？
  3. 复合不等式 \(0 < \dfrac{1}{2}x - 3 < 2\) 拆成两半后，为什么要取"交集"而不是"并集"？
  4. 如果函数式改成 \(y = -\dfrac{1}{2}x - 3\)（系数变负），第(2)问的答案会变成什么？为什么？
  5. 第(4)问的答案 \(6 < x < 10\) 中，能不能取等号（即 \(x = 6\) 或 \(x = 10\)）？为什么？

六、易错提醒

  • 忘代入：看到 \(y > 2\) 就想画图，忘记第一步应该是把 2 代入式子变成关于 \(x\) 的不等式。画图可以辅助理解，但计算要走代入。
  • 复合不等式只解一半：\(0 < y < 2\) 要拆成两个不等式分别解，然后取交集。如果只解了 \(y < 2\) 得到 \(x < 10\)，就漏掉了 \(y > 0\) 的要求。
  • 不等号方向搞错：如果系数是负数，乘（或除以）负数时不等号要翻转。本题系数 \(\frac{1}{2} > 0\)，方向不变。但换题后要特别注意。

七、一句话总结

y 的条件代回式子，方程求点，不等式求范围，复合不等式拆开解再取交集。
