# 结构分析：垂径定理·平行弦分类讨论

## 原题

已知 $\odot O$ 半径 $OB=13$，$AB$、$CD$ 是弦，$AB\parallel CD$，$AB=10$，$CD=24$，则 $AB$ 与 $CD$ 的距离是\_\_\_\_。

> 图形描述：以 $O$ 为圆心、$13$ 为半径的圆中，弦 $AB$ 和弦 $CD$ 互相平行。两弦长度分别为 $10$ 和 $24$。需要求两平行弦之间的距离。

## 一、题目场景
- 数学对象：圆（$\odot O$）、弦（$AB$、$CD$）、弦心距（圆心到弦的垂线段）、平行线间距离
- 变量/参数：半径 $R=13$、弦长 $AB=10$、$CD=24$
- 函数/图形：圆的几何图形
- 已知条件：半径 $OB=13$，$AB\parallel CD$，$AB=10$，$CD=24$
- 要求目标：求 $AB$ 与 $CD$ 的距离

## 二、核心结构
- 表面考点：圆中平行弦距离计算
- 本质考点：**垂径定理 + 分类讨论（同侧/异侧）**
- 一句话问题模式：**已知半径和弦长，用垂径定理求弦心距，再对两弦相对圆心的位置进行分类讨论。**

## 三、关键转化
- 最关键的转化：将"两弦距离"转化为"圆心到两弦的垂线段长度之差或之和"
- 为什么降低计算量：垂径定理将弦长问题直接转化为直角三角形的勾股问题，计算量极小（只需 $\sqrt{13^2-5^2}$ 和 $\sqrt{13^2-12^2}$）
- 不转化时的低效路径：试图用坐标系建系求直线方程再求距离，计算量大且容易出错

## 四、标准路径骨架
1. **作辅助线**：从圆心 $O$ 分别作 $OM\perp AB$ 于 $M$，$ON\perp CD$ 于 $N$
2. **平行弦共线**：$AB\parallel CD$ 且 $OM\perp AB$，故 $OM$ 与 $ON$ 共线（即两弦的公垂线）
3. **垂径定理求半弦长**：$AM=\frac{1}{2}AB=5$，$CN=\frac{1}{2}CD=12$
4. **勾股定理求弦心距**：在 $\mathrm{Rt}\triangle OAM$ 中 $OM=\sqrt{13^2-5^2}=12$；在 $\mathrm{Rt}\triangle OCN$ 中 $ON=\sqrt{13^2-12^2}=5$
5. **分类讨论求距离**：同侧时 $|OM-ON|=|12-5|=7$；异侧时 $OM+ON=12+5=17$

## 四点五、标准完整解与验算

**完整求解过程：**

- 关键辅助线：作 $OM\perp AB$ 于 $M$，$ON\perp CD$ 于 $N$
- 关键量：
  - 半弦长：$AM=5$，$CN=12$
  - 弦心距：$OM=\sqrt{13^2-5^2}=\sqrt{169-25}=\sqrt{144}=12$；$ON=\sqrt{13^2-12^2}=\sqrt{169-144}=\sqrt{25}=5$
- 完整求解过程：
  - 作 $OM\perp AB$ 于 $M$，$ON\perp CD$ 于 $N$
  - 由垂径定理：$AM=\frac{1}{2}AB=5$，$CN=\frac{1}{2}CD=12$
  - 由 $AB\parallel CD$ 且 $OM\perp AB$，$ON\perp CD$，知 $OM$ 与 $ON$ 共线
  - 在 $\mathrm{Rt}\triangle OAM$ 中：$OM=\sqrt{OA^2-AM^2}=\sqrt{13^2-5^2}=12$
  - 在 $\mathrm{Rt}\triangle OCN$ 中：$ON=\sqrt{OC^2-CN^2}=\sqrt{13^2-12^2}=5$
  - **情形一**：$AB$、$CD$ 在圆心同侧，距离 $d=OM-ON=12-5=7$
  - **情形二**：$AB$、$CD$ 在圆心异侧，距离 $d=OM+ON=12+5=17$
- 最终答案：**$7$ 或 $17$**
- 排除值：无（两种情形均合法）
- 退化情形：
  - 若 $AB=CD$（等长平行弦），则 $OM=ON$，同侧时距离为 $0$（两弦重合，不可能）；异侧时距离为 $2OM$，只有一个答案
  - 若某一弦恰好为直径（如 $CD=26$），则 $ON=0$，同侧/异侧退化
- 验算：
  - $OM^2+AM^2=144+25=169=13^2=OA^2$ ✔
  - $ON^2+CN^2=25+144=169=13^2=OC^2$ ✔
  - 同侧距离 $7 < 26$（直径），合理 ✔
  - 异侧距离 $17 < 26$（直径），合理 ✔
- 本题最短可靠路径：作两条垂线段 → 垂径定理 → 勾股定理 → 分类讨论 → 两答案

## 五、出题人逻辑
- 诱导学生硬算的位置：学生可能试图建系、求直线方程、用两平行线距离公式，绕大弯
- 真正的捷径：垂径定理直接给弦心距，只需两次勾股运算，极其简洁
- 训练的可迁移能力：
  - 垂径定理的标准辅助线作法（过圆心作弦的垂线段）
  - 圆中"平行弦共垂线"的隐含条件
  - 几何中的分类讨论意识（同侧/异侧）

## 六、学生卡点预测
- 基础薄弱学生：
  - 不知道作辅助线（不会"过圆心作弦的垂线段"这一标准动作）
  - 不理解垂径定理"垂径→平分"的双向关系
  - 忘记分类讨论，只算出一种情形就停了
- 中等学生：
  - 知道垂径定理，但不理解为什么 $OM$ 和 $ON$ 共线（平行弦的公垂线性质）
  - 能算出弦心距，但分类讨论时把同侧和异侧搞反
  - 算出 $12-5=7$ 和 $12+5=17$，但不确定哪个是同侧、哪个是异侧
- 较强学生：
  - 能完整解题，但无法解释"为什么一定是这两条垂线段共线"
  - 可能忽略验证两种情形是否都合法

## 七、变式原则
- 核心不变量：垂径定理（半径、半弦长、弦心距的勾股关系）+ 分类讨论意识
- 表层特征：两平行弦在圆中的位置关系
- 可变维度：
  - 半径和两弦长度的数值（保持 $\le 2R$ 且形成勾股数以简化计算）
  - 弦的数量（可推广到三平行弦）
  - 隐藏条件（不给弦长，给弦心距）
  - 反向构造（给距离和一条弦长，求另一条弦长）
  - 位置追问（哪条弦离圆心更近？弦心距比值？）
- 深化阶梯：
  1. 同结构换数：换半径和弦长，仍保持勾股数
  2. 同结构换问法：已知距离求弦长
  3. 条件包装：将弦长藏进周长、面积等间接条件中
  4. 结构部分隐藏：不直接说"平行弦"，而是用"梯形内接于圆"等包装
  5. 反向构造：给定距离和部分条件，判断能否构成
- 允许的变换：更换半径和弦长数值（保持为勾股数组合以控制计算量）、增加弦的数量、互换已知与所求
- 禁止的变换：引入圆周角、切线等新知识点（跑偏）、数值复杂到需要计算器
- 表征切换：纯几何图形 → 坐标系表达（建系后弦是平行直线）
- 包装方式：
  - "圆柱形管道截面中两根平行管道的距离"
  - "圆桌面上两根平行筷子的间距"
- 近迁移例子：
  - 圆中两弦 $AB\parallel CD$，$AB=16$，$CD=12$，半径 $10$，求距离（$\sqrt{10^2-8^2}=6$，$\sqrt{10^2-6^2}=8$，答案 $2$ 或 $14$）
- 远迁移例子：
  - 圆内接等腰梯形，已知上下底和圆的半径，求梯形的高（本质相同：两底即两平行弦）
- 反例/伪变式：
  - 出成切线与弦的距离（引入切线性质，跑了新知识点，不算本结构变式）

## 八、计算复杂度预算
- 原题计算层级：两次勾股运算，均为整除（$\sqrt{144}=12$，$\sqrt{25}=5$），最后加减法
- 允许小步上升到：弦心距出现 $\sqrt{3}$、$2\sqrt{3}$ 等简单无理数；距离结果为无理数
- 禁止引入的计算负担：需要近似计算的弦心距、复杂的三次以上方程
- 必须保留的可见支架：垂径定理的几何图形标注（垂线段、半弦长、半径）

## 九、推荐讲题任务包
- 适合的学习层级：L2（建模层）到 L4（迁移层）
- 本题讲解目标：
  - 掌握垂径定理的标准辅助线作法
  - 理解平行弦共垂线
  - 建立分类讨论的几何直觉
- 不要直接讲的抽象话："要分类讨论"——应通过画图让学生自己发现两种位置
- 必须先问的问题：
  - "两根弦平行，过圆心作它们的垂线，这两条垂线段有什么关系？"（引出共线）
  - "圆心到这两根弦的垂线段长度一定不一样吗？为什么？"（引出弦长不同→弦心距不同）
- 关键讲解顺序：
  1. 画圆，画出两种位置（同侧、异侧）
  2. 引导作辅助线（过圆心作两弦的垂线段）
  3. 垂径定理求半弦长和弦心距
  4. 在图上直接看出距离是"差"还是"和"
  5. 计算
- 最适合的具体数值例子：原题的 $13$、$10$、$24$ 已经是最优组合（勾股数 $5-12-13$，计算全为整数）
- 讲到哪里停下来让学生回答：
  - 作完辅助线后，问"$OM$ 和 $ON$ 在一条直线上吗？为什么？"（核心共线性质）
  - 算出 $OM=12$、$ON=5$ 后，问"看图，距离应该是几？"（让学生自己发现两种情形）

## 十、推荐练题任务包
- 若学生在 L0-L1，出什么题：
  - 基础垂径定理：已知半径和弦长，求弦心距（单弦，不涉及分类讨论）
  - 例：$\odot O$ 半径 $10$，弦 $AB=12$，求圆心到 $AB$ 的距离
- 若学生在 L2，出什么题：
  - 同结构换数：$\odot O$ 半径 $10$，$AB\parallel CD$，$AB=16$，$CD=12$，求距离（答案 $2$ 或 $14$）
- 若学生在 L3，出什么题：
  - 同结构换问法：$\odot O$ 半径 $13$，$AB\parallel CD$，$AB=10$，两弦距离 $7$，求 $CD$ 的长（需要学生倒推 $ON$ 再求弦长）
- 若学生在 L4，如何迁移：
  - 圆内接梯形：$\odot O$ 半径 $10$，内接等腰梯形 $ABCD$，$AD\parallel BC$，$AD=16$，$BC=12$，求梯形高（本质相同）
- 若学生达到 L5-L6，如何深化/抽象/包装：
  - 反向构造：$\odot O$ 半径 $R$，两平行弦 $AB$、$CD$ 长分别为 $a$、$b$，两弦距离为 $d$，求 $R$（用 $d$ 的两个值分别表示，建立方程组消元求 $R$）
  - 证明题：证明两平行弦的弦心距之和为定值（异侧时），或证明弦心距之差为定值（同侧时）
- 禁止出的跑偏变式：
  - 引入切线（如"切线与平行弦的距离"——新知识点）
  - 引入圆周角（如"弦所对的圆周角"——跑了题）
  - 三条以上平行弦（计算量膨胀，不适合本题层级）
