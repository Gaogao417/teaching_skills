提示一

已知面积和圆心角，目标是周长。面积 = (1/2)r²θ，这里 θ 已知，可以解出 r。

参考答案

第 1 题答案
θ = l/r = 6/3 = 2 弧度
第 2 题答案

面积 (1/2)rl = 6 → rl = 12 → l = 12/r

代入周长：2r + 12/r = 10

两边乘 r：2r² − 10r + 12 = 0 → r² − 5r + 6 = 0

(r − 2)(r − 3) = 0，r = 2 或 r = 3

r = 2：l = 12/2 = 6，θ = 6/2 = 3

r = 3：l = 12/3 = 4，θ = 4/3

θ = 3 弧度或 4/3 弧度。

验算：r=2,l=6：周长=4+6=10 ✓，面积=6 ✓；r=3,l=4：周长=6+4=10 ✓，面积=6 ✓

第 3 题答案

面积 S = (1/2)r²θ = (1/2)r²×2 = r² = 8

r = 2√2（r > 0）

l = rθ = 2√2 × 2 = 4√2

周长 = 2r + l = 4√2 + 4√2 = 8√2
第 4 题答案

两个解：r = 1 时 l = 4；r = 2 时 l = 2。

弧长之比 = 4 : 2 = 2 : 1

几何含义：小半径（r=1）的扇形弧更长、角度更大，是一个"扁扇形"；大半径（r=2）的扇形弧更短、角度更小，是一个"窄扇形"。两者的周长和面积恰好相同。
