参考答案

第 1 题答案

(t, 2t) → (t, -2t)，所以 g(t) = -2t，即 g(x) = -2x。

第 2 题答案

(t, t2+1) → (-t, -t2-1)，令 x = -t，t = -x，所以 g(x) = -(-x)2 - 1 = -x2 - 1。

第 3 题答案

(t, t2) → (4-t, t2)，令 x = 4-t，t = 4-x，所以 g(x) = (4-x)2 = (x-4)2。

第 4 题答案

(t, t2) → (2-t, 4-t2)，令 x = 2-t，t = 2-x，所以 g(x) = 4 - (2-x)2。

第 5 题答案

(t, t2) → (t+3, t2)，令 x = t+3，t = x-3，所以 g(x) = (x-3)2。

第 6 题答案

(t, √t) → (t-2, √t) → (t-2, -√t)，令 x = t-2，t = x+2，所以 g(x) = -√(x+2)，定义域 x ≥ -2。
