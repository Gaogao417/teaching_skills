参考答案

第 1 题答案

(1) sin(πx) = sin(π(x + 1/3))

情形一：πx = π(x + 1/3) + 2kπ → m = −1/3（不合）

情形二：πx = π − π(x + 1/3) + 2kπ → 2πx = 2π/3 + 2kπ → x = 1/3 + k

(2) P0 = (1/3, sin(π/3)) = (1/3, √3/2)

  P1 = (4/3, sin(4π/3)) = (4/3, −√3/2)

  P2 = (7/3, sin(7π/3)) = (7/3, √3/2)

第 2 题答案

(1) xk+1 − xk = (1−m)/2 + (k+1) − (1−m)/2 − k = 1 ✓

(2) yk = sin(πxk) = sin(π(1−m)/2 + kπ) = sin(π(1−m)/2)cos(kπ) + cos(π(1−m)/2)sin(kπ) = (−1)ky0 ✓

(3) P0 = ((1−m)/2, y0)，P1 = ((1−m)/2+1, −y0)，P2 = ((1−m)/2+2, y0)

第 3 题答案

(1) |P0P1| = √(1+4a2)，|P1P2| = √(1+4a2)，|P0P2| = 2

(2) √(1+4a2) = 2 → a2 = 3/4 → a = √3/2

(3) |P0P1|2 + |P1P2|2 = |P0P2|2 → 2(1+4a2) = 4 → a2 = 1/4 → a = 1/2

第 4 题答案

交点通项：x = (1−m)/2 + k，纵坐标交替 ±y0

等腰直角三角形（直角在 P1）：2(1+4y02) = 4 → y02 = 1/4 → |y0| = 1/2

sin(π(1−m)/2) = ±1/2 → π(1−m)/2 = π/6 或 −π/6（主值范围）

(1−m)/2 = 1/6 → m = 2/3；(1−m)/2 = −1/6 → m = 4/3

答：m = 2/3 或 4/3
