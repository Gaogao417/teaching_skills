自适应练习：幂函数与 y = x 的位置比较

核心动作：将函数大小关系转化为指数不等式

参考答案

第 1 题答案

(1) x3 > x  (2) x0 = 1 < x  (3) x-2 = 1/x2 < 1 < x  (4) x1 = x

规律：a > 1 时 xa > x；a = 1 时相等；a < 1 时 xa < x。

第 2 题答案

a 的取值范围是 (-∞, 1)。

解法与原题完全一致：xa < x → xa-1 < 1 = x0 → 因为 x > 2 > 1 → a-1 < 0 → a < 1。

第 3 题答案

s 的取值范围是 (1, +∞)。

解法：xs < x → xs-1 < 1 = x0。因为 0 < x < 1，指数函数递减，所以不等号反转：s-1 > 0，即 s > 1。
验证：s=2, x=1/2 → (1/2)^2 = 1/4 < 1/2 ✓；s=0, x=1/2 → (1/2)^0 = 1 > 1/2 ✗（排除正确）。

第 4 题答案

m < n。

解法：xm < xn → 两边除以 xn（>0）→ xm-n < 1 = x0 → x > 1 → m-n < 0 → m < n。
