参考答案

第 1 题

(1) y = −x + 6 交 x 轴：令 y = 0 → x = 6 → A(6, 0)

交 y 轴：令 x = 0 → y = 6 → B(0, 6)

OA = |6| = 6，OD = 1/2 × 6 = 3，D 在 x 轴正半轴 → D(3, 0)

(2) AD⃗ = (3−6, 0) = (−3, 0)

C = B + AD⃗ = (0−3, 6+0) = (−3, 6)

验证：DC⃗ = (−3−3, 6−0) = (−6, 6)，AB⃗ = (0−6, 6−0) = (−6, 6) ✓

第 2 题

(1) BD 过 B(0, 6) 和 D(3, 0)

k = (0−6)/(3−0) = −6/3 = −2

过 B(0, 6) → b = 6

直线 BD：y = −2x + 6

(2) F = ((0+3)/2, (6+0)/2) = (3/2, 3)

第 3 题

(1) E = (t, 0)，t > 0

(2) 直线 EF 过 E(t, 0) 和 F(3/2, 3)

斜率 k = (3−0)/(3/2−t) = 3/((3−2t)/2) = 6/(3−2t)

EF 方程：y = 6(x−t)/(3−2t)

令 y = 6：6 = 6(x−t)/(3−2t) → 3−2t = x−t → x = 3−t

H = (3−t, 6)
