第 4 题：结构隐藏型——合并后分离

若 y = sin(x + φ) + cos(x + φ) 为奇函数，求 tan φ 的值。

提示一

分别展开 sin(x + φ) 和 cos(x + φ) 的和角公式，然后合并同类项。

提示二

sin(x+φ) + cos(x+φ) = (sin x cos φ + cos x sin φ) + (cos x cos φ - sin x sin φ) = (cos φ - sin φ) sin x + (sin φ + cos φ) cos x。偶部归零 → sin φ + cos φ = 0 → tan φ = -1。

参考答案

第 1 题答案

cos(x + π/4) = cos x cos(π/4) - sin x sin(π/4) = (√2/2)cos x - (√2/2)sin x

偶函数部分：(√2/2)cos x

奇函数部分：-(√2/2)sin x

第 2 题答案

sin(x + φ) = sin x cos φ + cos x sin φ

奇函数部分：sin x cos φ（sin x 是奇函数）

偶函数部分：cos x sin φ（cos x 是偶函数）

要成为偶函数，奇部必须为零：cos φ = 0

所以 cos φ = 0。

第 3 题答案

由 cos(x + φ) 为奇函数，得 cos φ = 0，即 φ = π/2 + kπ。

在 [0, 2π] 内：k = 0 时 φ = π/2；k = 1 时 φ = 3π/2。

验证：φ = π/2 时 y = -sin x（奇函数）；φ = 3π/2 时 y = sin x（奇函数）。

答案：φ = π/2 或 φ = 3π/2。

第 4 题答案

sin(x+φ) + cos(x+φ) = (sin x cos φ + cos x sin φ) + (cos x cos φ - sin x sin φ)

= (cos φ - sin φ)sin x + (sin φ + cos φ)cos x

偶部（cos x 项）归零：sin φ + cos φ = 0 → tan φ = -1

答案：tan φ = -1。
