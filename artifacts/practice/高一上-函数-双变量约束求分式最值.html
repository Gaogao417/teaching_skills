第 4 题：改变目标表达式

正实数 x, y 满足 x + 2/y = 3，求 x·y 的最大值。

提示一

还是先消元。解出 x = 3 − 2/y，目标变为 x·y = 3y − 2。

提示二

参考答案

第 1 题答案
f(x) = x − x²/2 = −1/2(x² − 2x) = −1/2[(x−1)² − 1] = 1/2 − (x−1)²/2

因为 (x−1)² ≥ 0，所以 f(x) ≤ 1/2。

x = 1 ∈ (0, 2) ✓。

最大值为 1/2。

第 2 题答案

x = 2 − 1/y，由 x > 0 得 y > 1/2。

x/y = (2 − 1/y)/y = 2/y − 1/y²。

令 t = 1/y，则 0 < t < 2。

x/y = 2t − t² = −(t−1)² + 1 ≤ 1

t = 1 ∈ (0, 2) ✓，回代 y = 1，x = 1。验算：1 + 1 = 2 ✓。

最大值为 1，当 x = 1，y = 1 时取到。

第 3 题答案

m = (6 − 3/n)/2 = 3 − 3/(2n)，由 m > 0 得 n > 1/2。

m/n = 3/n − 3/(2n²)。

令 t = 1/n，则 0 < t < 2。

m/n = 3t − 3t²/2 = −3/2(t−1)² + 3/2 ≤ 3/2

t = 1 ∈ (0, 2) ✓，回代 n = 1，m = 3 − 3/2 = 3/2。验算：2×(3/2) + 3/1 = 3 + 3 = 6 ✓。

最大值为 3/2，当 m = 3/2，n = 1 时取到。

第 4 题答案

x = 3 − 2/y，由 x > 0 得 y > 2/3。

x·y = (3 − 2/y)·y = 3y − 2。

这是一次函数，且系数 3 > 0，所以 y 越大，x·y 越大。

但 y 只有下界（> 2/3），没有上界，所以 y → +∞ 时 x·y → +∞。

x·y 没有最大值。
