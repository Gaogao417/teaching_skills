参考答案

第 1 题

设 y = −2x + b，代入 (1, 4)：4 = −2 × 1 + b，b = 6。

解析式：y = −2x + 6

第 2 题

与 y = 3x − 1 平行，故斜率 k = 3；y 轴截距为 5，即 b = 5。

解析式：y = 3x + 5

第 3 题

设 y = kx + b，代入两点：

{ 3 = k + b    { −1 = −k + b

两式相减：4 = 2k，k = 2；b = 1。

解析式：y = 2x + 1

第 4 题

b = −6，设 y = kx − 6，代入 (4, −2)：

−2 = 4k − 6，4k = 4，k = 1。

解析式：y = x − 6

第 5 题

与 y = −2x + 1 平行，故 k = −2。设 y = −2x + b，代入 (1, −4)：

−4 = −2 + b，b = −2。

解析式：y = −2x − 2

第 6 题

(1) 设 y = −x + b，代入 (−2, 5)：5 = 2 + b，b = 3。解析式：y = −x + 3。

(2) 令 y = 0，得 x = 3；令 x = 0，得 y = 3。

面积：S = 12 × 3 × 3 = 92

第 7 题

(1) 设 y = kx + b，代入两点：7 = 2k + b，1 = −k + b。相减：k = 2，b = 3。解析式：y = 2x + 3。

(2) 令 y = 0，x = −32；令 x = 0，y = 3。

面积：S = 12 × 32 × 3 = 94

第 8 题

(1) 与 y = −3x 平行，k = −3；截距 b = 1。解析式：y = −3x + 1。

(2) 令 y = 0，x = 13；令 x = 0，y = 1。

面积：S = 12 × 13 × 1 = 16

第 9 题

(1) 设原函数 y = 12x + b。向右平移 4 个单位后：y = 12(x − 4) + b。

过 (1, 0)：0 = 12(1 − 4) + b = −32 + b，b = 32。原函数：y = 12x + 32。

(2) 令 y = 0，x = −3；令 x = 0，y = 32。

面积：S = 12 × 3 × 32 = 94

第 10 题

(1) 设 y = kx + b，代入两点：−3 = k + b，3 = −2k + b。相减：k = −2，b = −1。解析式：y = −2x − 1。

(2) 令 y = 0，x = −12；令 x = 0，y = −1。

三角形顶点为 (0, 0)、(−12, 0)、(0, −1)。

面积：S = 12 × 12 × 1 = 14
