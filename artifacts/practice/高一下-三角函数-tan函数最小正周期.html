参考答案

第 1 题

(1) ω = 3，T = π/|3| = π/3

(2) ω = 1/2，T = π/|1/2| = π ÷ (1/2) = 2π

第 2 题

(1) ω = 2，T = π/|2| = π/2（-π/3 不影响周期）

(2) ω = -4，T = π/|-4| = π/4（+5 不影响周期）

第 3 题

(1) y = sin(2x+1)：基础周期 2π，T = 2π/|2| = π

(2) y = cos(2x+1)：基础周期 2π，T = 2π/|2| = π

(3) y = tan(2x+1)：基础周期 π，T = π/|2| = π/2

发现：ω 相同时，sin 和 cos 周期相同，tan 的周期是它们的一半。

第 4 题

T = π/|ω| = π/ω（因为 ω > 0）

π/ω = 2π → ω = π/(2π) = 1/2

满足 ω > 0，所以 ω = 1/2。
