第 4 题：变式——解集为全体实数

不等式 (x + a)/(x − 2) > 0 的解集为全体实数（x ≠ 2 时恒成立），求 a 的值。

提示一

和"解集为空"的思路一样：先找零点。但这次要求的是"对所有 x（除分母零点外）分式都 > 0"。

提示二

分子零点 x = −a，分母零点 x = 2。如果零点不同，两个零点之间分式 < 0，不可能全体 > 0。所以零点必须重合，但重合后分式 = 1（> 0），解集确实为全体（除 x=2）。

参考答案

第 1 题答案

(1) 分子零点 x = −4，分母零点 x = 2

(2) 分子零点 x = 3，分母零点 x = −3

(3) 分子零点 x = a，分母零点 x = −5

第 2 题答案

(1) 零点 −4 和 2 不同，解集非空（解集为 (−4, 2)）

(2) 零点重合于 x = 1，(x−1)/(x−1) = 1（x≠1），1 < 0 不成立，解集为空

(3) 零点重合于 x = 1，(x−1)/(x−1) = 1（x≠1），1 > 0 成立，解集为 (−∞,1)∪(1,+∞)

第 3 题答案

移项通分：(3x + a)/(x + 2) − 1 = (3x + a − x − 2)/(x + 2) = (2x + a − 2)/(x + 2) < 0

分子零点 x = (2 − a)/2，分母零点 x = −2

令零点重合：(2 − a)/2 = −2，解得 2 − a = −4，a = 6

验证：a = 6 时，(3x + 6)/(x + 2) = 3(x + 2)/(x + 2) = 3（x≠−2），3 < 1 不成立，解集为空。正确。

第 4 题答案

分子零点 x = −a，分母零点 x = 2

若 −a ≠ 2（即 a ≠ −2），则两个零点之间分式 < 0，不可能全体 > 0

令零点重合：−a = 2，a = −2

验证：a = −2 时，(x − 2)/(x − 2) = 1（x≠2），1 > 0 恒成立，解集为 (−∞,2)∪(2,+∞)。正确。
