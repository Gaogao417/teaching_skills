第 2 题 平行 + y 轴截距

第 3 题 经过两个点

已知一次函数经过点 (−2, 5) 和点 (3, −5)。求该函数的解析式。

第 4 题 y 轴截距 + 过一点

预期卡点−2 = 3k + 4 中移项出错

答案：y = −2x + 4

第 5 题 平行 + 过一点

已知一次函数与 y = 12x − 3 平行，且经过点 (4, 6)。求该函数的解析式。

解答：

训练目标平行确定 k，再用一点求 b；含分数系数

预期卡点斜率为分数时代入计算出错；k = ½，6 = ½ × 4 + b = 2 + b

答案：y = 12x + 4

第 10 题 经过两个点 → 面积

参考答案

第 1 题

设 y = 3x + b，代入 (2, −1)：−1 = 6 + b，b = −7。

解析式：y = 3x − 7

第 2 题

与 y = −x + 4 平行，故斜率 k = −1；y 轴截距为 −3，即 b = −3。

解析式：y = −x − 3

第 3 题

设 y = kx + b，代入两点：

k = −5 − 53 − (−2) = −105 = −2

代入 (−2, 5)：5 = 4 + b，b = 1。

解析式：y = −2x + 1

第 4 题

b = 4，设 y = kx + 4，代入 (3, −2)：

−2 = 3k + 4，3k = −6，k = −2。

解析式：y = −2x + 4

第 5 题

与 y = 12x − 3 平行，故 k = 12。设 y = 12x + b，代入 (4, 6)：

6 = 12 × 4 + b = 2 + b，b = 4。

解析式：y = 12x + 4

第 6 题

(1) 设 y = 2x + b，代入 (−1, 5)：5 = −2 + b，b = 7。解析式：y = 2x + 7。

(2) 令 y = 0，0 = 2x + 7，x = −72；令 x = 0，y = 7。

面积：S = 12 × 72 × 7 = 494

第 7 题

(1) 设 y = kx + b，代入两点：

k = −4 − 4−3 − 1 = −8−4 = 2

代入 (1, 4)：4 = 2 + b，b = 2。解析式：y = 2x + 2。

(2) 令 y = 0，x = −1；令 x = 0，y = 2。

面积：S = 12 × 1 × 2 = 1

第 8 题

(1) 与 y = −4x + 2 平行，k = −4；截距 b = 3。解析式：y = −4x + 3。

(2) 令 y = 0，0 = −4x + 3，x = 34；令 x = 0，y = 3。

面积：S = 12 × 34 × 3 = 98

第 9 题

(1) 设原函数 y = −12x + b。向右平移 2 个单位后：

y = −12(x − 2) + b = −12x + 1 + b

过 (4, 0)：0 = −12 × 4 + 1 + b = −2 + 1 + b，b = 1。原函数：y = −12x + 1。

(2) 令 y = 0，x = 2；令 x = 0，y = 1。

面积：S = 12 × 2 × 1 = 1

第 10 题

(1) 设 y = kx + b，代入两点：

k = 4 − (−5)2 − (−1) = 93 = 3

代入 (−1, −5)：−5 = −3 + b，b = −2。解析式：y = 3x − 2。

(2) 令 y = 0，0 = 3x − 2，x = 23；令 x = 0，y = −2。

三角形顶点为 (0, 0)、(23, 0)、(0, −2)。

面积：S = 12 × 23 × 2 = 23
