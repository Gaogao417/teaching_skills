\eduSectionTitle{例题 1 — 一线三垂直求等腰直角第三顶点 C}

\begin{eduProblemBlock}[题目]
直线 $y = -\dfrac{\sqrt{3}}{3}x + 1$ 与 $x$ 轴、$y$ 轴分别交于 $A$、$B$，以 $AB$ 为直角边在第一象限作等腰 Rt$\triangle ABC$，$\angle BAC = 90°$。
(1) 求点 $C$ 的坐标。

\end{eduProblemBlock}









\begin{eduAuxItem}{思路导航}
令 $y=0$ 求 $A(\sqrt{3},\,0)$，令 $x=0$ 求 $B(0,\,1)$$\;\to\;$过 $A$ 做 $x$ 轴辅助线，过 $B$、$C$ 做垂线$\;\to\;$标出三个直角，证明 $\triangle ABB_1 \cong \triangle CC_1A$$\;\to\;$由对应边：$AB_1 \to CC_1$，$BB_1 \to AC_1$$\;\to\;$求出 $C$ 坐标，验证等腰 $+$ 直角\end{eduAuxItem}




\begin{eduAuxItem}{核心思路}
\textbf{一线三垂直全等模型（核心工具）}：
\begin{enumerate}
  \item 过直角顶点 $A$ 沿坐标轴方向做辅助线（本题 $A$ 在 $x$ 轴上，直接用 $x$ 轴）
  \item 从两个锐角顶点 $B$、$C$ 分别做辅助线的垂线，垂足为 $B_1$、$C_1$
  \item 辅助线上出现三个直角：$\angle AB_1B = 90°$、$\angle BAC = 90°$、$\angle AC_1C = 90°$
  \item 由互补关系证 $\angle ABB_1 = \angle CAC_1$，加上 $AB = AC$，得 $\triangle ABB_1 \cong \triangle CC_1A$（AAS）
  \item 全等 $\Rightarrow$ 对应边相等：水平位移与竖直位移互换
\end{enumerate}
\end{eduAuxItem}




\begin{eduSubQuestionItem}{(1)}
求点 $C$ 的坐标。\end{eduSubQuestionItem}

\begingroup
\setlength{\parskip}{2.5mm}
令 $y = 0$：$-\dfrac{\sqrt{3}}{3}x + 1 = 0 \implies x = \sqrt{3}$，$A(\sqrt{3},\,0)$；令 $x = 0$：$B(0,\,1)$\par
过直角顶点 $A$ 沿 $x$ 轴做辅助线。过 $B$ 做 $x$ 轴垂线，垂足 $B_1(0,\,0)$；过 $C$ 做 $x$ 轴垂线，垂足 $C_1(x_C,\,0)$\par
三个直角：$\angle AB_1B = 90°$（$B_1$ 处），$\angle BAC = 90°$（$A$ 处），$\angle AC_1C = 90°$（$C_1$ 处）\par
在 $\triangle ABB_1$ 中 $\angle ABB_1 = 90° - \angle B_1AB$；又 $\angle BAC = 90° = \angle B_1AB + \angle CAC_1$，所以 $\angle ABB_1 = \angle CAC_1$\par
$AB = AC$（等腰直角），$\angle AB_1B = \angle AC_1C = 90°$，$\angle ABB_1 = \angle CAC_1$ $\implies$ $\triangle ABB_1 \cong \triangle CC_1A$（AAS）\par
由全等：$AB_1 = CC_1 = \sqrt{3}$ $\implies$ $y_C = \sqrt{3}$；$BB_1 = AC_1 = 1$ $\implies$ $x_C = \sqrt{3} + 1$\par
$C(\sqrt{3}+1,\,\sqrt{3})$，验证：$AB = \sqrt{3+1} = 2$，$AC = \sqrt{1+3} = 2$ $\checkmark$\par
\endgroup

\vspace{2mm}
\eduSubTitle{注意}
\begin{eduBulletList}
  \item $\dfrac{1}{\sqrt{3}/3} = \dfrac{3}{\sqrt{3}} = \sqrt{3}$，不要算成 $3\sqrt{3}$。  \item 全等对应：$A \leftrightarrow C$，$B \leftrightarrow A$，$B_1 \leftrightarrow C_1$，所以 $AB_1$ 对应 $CC_1$，$BB_1$ 对应 $AC_1$。\end{eduBulletList}




\begin{eduBottomGrid}
  \begin{eduSummaryLeft}
    \begin{eduSummaryBlock}{\eduIconCheck}{答案}
    \begin{eduPlainList}
      \item $C(\sqrt{3}+1,\,\sqrt{3})$    \end{eduPlainList}
    \end{eduSummaryBlock}
  \end{eduSummaryLeft}
  \eduSummaryDivider
  \begin{eduSummaryRight}
    \begin{eduSummaryBlock}{\eduIconPin}{方法提醒}
    \begin{eduBulletList}
      \item 一线三垂直标准六步：求交点 $\to$ 做辅助线 $\to$ 做垂线 $\to$ 证全等 $\to$ 对应边互换 $\to$ 求坐标。      \item 全等后水平位移变成竖直位移、竖直位移变成水平位移，注意方向。    \end{eduBulletList}
    \end{eduSummaryBlock}
  \end{eduSummaryRight}
\end{eduBottomGrid}




\begin{eduMistakeBox}
\eduSubTitle{全等对应边搞反}$\triangle ABB_1 \cong \triangle CC_1A$ 的对应关系是 $A \leftrightarrow C$、$B \leftrightarrow A$、$B_1 \leftrightarrow C_1$。
所以 $AB_1$（水平）对应 $CC_1$（竖直），$BB_1$（竖直）对应 $AC_1$（水平）——水平与竖直互换。
如果搞反了，$AB_1$ 对应 $AC_1$，就会得到 $x_C = \sqrt{3} + \sqrt{3}$ 的错误结果。
\end{eduMistakeBox}




\begin{eduMistakeBox}
\eduSubTitle{辅助线方向选错}过直角顶点做辅助线时，选坐标轴方向（水平或竖直）最方便，因为垂线也沿坐标轴方向，坐标计算最简单。
如果选斜线做辅助线，垂线是斜的，坐标计算会变复杂。
\end{eduMistakeBox}




\begin{eduAuxItem}{思路启发}
$A$ 在 $x$ 轴上，过 $A$ 做辅助线选什么方向最方便？为什么？\end{eduAuxItem}




\begin{eduAuxItem}{思路启发}
如果直角顶点不在坐标轴上（比如 $A(2, 3)$），辅助线该怎么画？全等还成立吗？\end{eduAuxItem}

\eduSectionTitle{例题 2 — 条件求点 + 一线三垂直构造 P}

\begin{eduProblemBlock}[题目]
直线 $AB$ 过 $A(-1,1)$、$B(2,0)$，交 $y$ 轴于 $C$。$D(0,n)$ 在 $C$ 上方，$S_{\triangle ABD} = 2$ 时，求 $P$ 使 $BP = BD$ 且 $BP \perp BD$。

\end{eduProblemBlock}









\begin{eduAuxItem}{思路导航}
求直线 $AB$ 解析式和 $C$ 坐标$\;\to\;$用 $CD$ 作竖直底边，拆成 $\triangle ACD$ 和 $\triangle BCD$ 求 $n$$\;\to\;$对 $\triangle BDP$（等腰直角，$B$ 为直角顶点）套一线三垂直$\;\to\;$由全等对应边求 $P$ 坐标（注意两解）\end{eduAuxItem}




\begin{eduSubQuestionItem}{第一步}
求直线 $AB$ 解析式和点 $D$ 坐标。\end{eduSubQuestionItem}

\begingroup
\setlength{\parskip}{2.5mm}
$k = \dfrac{0 - 1}{2 - (-1)} = -\dfrac{1}{3}$，$y = -\dfrac{1}{3}x + \dfrac{2}{3}$ $\implies$ $C\!\left(0,\,\dfrac{2}{3}\right)$\par
因为 $D$ 在 $C$ 上方，$CD=n-\dfrac{2}{3}$。以 $CD$ 为公共底，$A$ 到 $y$ 轴的距离为 $1$，$B$ 到 $y$ 轴的距离为 $2$\par
$S_{\triangle ABD}=S_{\triangle ACD}+S_{\triangle BCD}=\dfrac12\cdot CD\cdot1+\dfrac12\cdot CD\cdot2=\dfrac32\left(n-\dfrac23\right)$\par
$\dfrac32\left(n-\dfrac23\right)=2 \implies n=2$，所以 $D(0,\,2)$\par
\endgroup





\begin{eduSubQuestionItem}{第二步}
用一线三垂直求 $P$ 坐标。\end{eduSubQuestionItem}

\begingroup
\setlength{\parskip}{2.5mm}
$BP = BD$ 且 $BP \perp BD$ $\implies$ $\triangle BDP$ 等腰直角，$B$ 为直角顶点\par
过直角顶点 $B$ 作水平辅助线（即 $x$ 轴）。过锐角顶点 $D(0,2)$ 作垂线，垂足 $D_1(0,0)$，则 $BD_1=2$，$DD_1=2$\par
再过另一个锐角顶点 $P$ 作辅助线的垂线，垂足为 $P_1$。三个直角是 $\angle BD_1D$、$\angle DBP$、$\angle BP_1P$\par
由一线三垂直，$\triangle BDD_1 \cong \triangle PP_1B$（AAS），所以 $BP_1=DD_1=2$，$PP_1=BD_1=2$\par
若 $P_1$ 在 $B$ 右侧，则 $x_P=2+2=4$，且 $P$ 在 $x$ 轴上方，$P(4,\,2)$\par
若 $P_1$ 在 $B$ 左侧，则 $x_P=2-2=0$，且 $P$ 在 $x$ 轴下方，$P(0,\,-2)$\par
检验：两点都满足 $BP=BD=2\sqrt2$，且一线三垂直的构造已经保证 $BP\perp BD$\par
\endgroup

\vspace{2mm}
\eduSubTitle{后两问如何承接}
\begin{eduBulletList}
  \item $P(0,\,-2)$ 时，$P$ 的垂足 $P_1(0,0)$ 恰好与 $D_1$ 重合——这是垂足重合的退化情形，不要漏掉。\end{eduBulletList}




\begin{eduBottomGrid}
  \begin{eduSummaryLeft}
    \begin{eduSummaryBlock}{\eduIconCheck}{答案}
    \begin{eduPlainList}
      \item $D(0,\,2)$，$P(4,\,2)$ 或 $P(0,\,-2)$    \end{eduPlainList}
    \end{eduSummaryBlock}
  \end{eduSummaryLeft}
  \eduSummaryDivider
  \begin{eduSummaryRight}
    \begin{eduSummaryBlock}{\eduIconPin}{方法提醒}
    \begin{eduBulletList}
      \item 条件构造型：先用条件（面积）确定已知点，再套一线三垂直。      \item 全等求坐标时注意两侧都有解，检查退化情形。    \end{eduBulletList}
    \end{eduSummaryBlock}
  \end{eduSummaryRight}
\end{eduBottomGrid}




\begin{eduMistakeBox}
\eduSubTitle{漏掉 P(0, -2)}$P$ 可以在 $B$ 的左侧。全等 $\triangle BDD_1 \cong \triangle BP_1P$ 中，$P_1$ 可以在 $B$ 的左侧（$x_P = 0$），此时 $y_P = -2$。
不要因为 $D_1$ 和 $P_1$ 重合就认为这个解不成立。
\end{eduMistakeBox}




\begin{eduMistakeBox}
\eduSubTitle{面积别套高中公式}不要直接套行列式面积。这里 $C$、$D$ 都在 $y$ 轴上，先看 $CD=n-\frac23$，再把 $\triangle ABD$ 拆成 $\triangle ACD$ 和 $\triangle BCD$。
两个小三角形的高分别是 $1$ 和 $2$，因此面积是 $\frac12CD\cdot1+\frac12CD\cdot2$。
\end{eduMistakeBox}




\begin{eduAuxItem}{思路启发}
观察 $\triangle BDD_1$，$BD_1 = DD_1 = 2$，它本身是什么三角形？这个发现能帮你更快求 $P$ 吗？\end{eduAuxItem}

\eduSectionTitle{例题 3 — 手拉手：两正方形全等证垂直}

\begin{eduProblemBlock}[题目]
以 $\triangle ABC$ 的边 $AC$、$AB$ 向外作正方形 $ABEF$ 和 $ACNM$，联结 $BM$、$CF$。
(1) 证明 $BM = CF$，$BM \perp CF$；
(2) 若 $AB = 3$，$AC = 2$，$BC = 4$，求 $FM$。

\end{eduProblemBlock}









\begin{eduAuxItem}{思路导航}
证 $\triangle FAC \cong \triangle BAM$（SAS）$\;\to\;$由全等 + 旋转 $90°$ 的几何意义证 $BM \perp CF$$\;\to\;$建系求 $C$ 坐标，再求 $F$、$M$ 坐标$\;\to\;$距离公式求 $FM$\end{eduAuxItem}




\begin{eduAuxItem}{核心思路}
\textbf{手拉手模型核心}：两个正方形共顶点 $A$，各贡献一个 $90°$ 角。
$\angle FAC = \angle FAB + \angle BAC = 90° + \angle BAC = \angle BAM$，从而 SAS 全等。

\textbf{证垂直的简洁方法}：$\triangle FAC$ 绕 $A$ 旋转 $90°$ 得到 $\triangle BAM$（因为两个正方形各贡献 $90°$）。
旋转 $90°$ 使对应线段 $CF$ 和 $BM$ 的夹角也是 $90°$，所以 $BM \perp CF$。

\textbf{第(2)问建系}：设 $A$ 在原点，$AB$ 沿 $x$ 轴，由距离公式求 $C$ 坐标，再用正方形方向确定 $F$、$M$ 坐标。
\end{eduAuxItem}




\begin{eduSubQuestionItem}{(1)}
证明 $BM = CF$，$BM \perp CF$。\end{eduSubQuestionItem}

\begingroup
\setlength{\parskip}{2.5mm}
$AB = AF$（正方形 $ABEF$），$AC = AM$（正方形 $ACNM$）\par
$\angle FAC = \angle FAB + \angle BAC = 90° + \angle BAC = \angle BAM$\par
$\therefore \triangle FAC \cong \triangle BAM$（SAS）$\implies$ $BM = CF$\par
$\triangle FAC$ 绕 $A$ 旋转 $90°$ 得 $\triangle BAM$（两个正方形各转 $90°$），旋转 $90°$ 使对应线段夹角为 $90°$\par
$\therefore BM \perp CF$\par
\endgroup





\begin{eduSubQuestionItem}{(2)}
$AB = 3$，$AC = 2$，$BC = 4$，求 $FM$。\end{eduSubQuestionItem}

\begingroup
\setlength{\parskip}{2.5mm}
设 $A(0,0)$，$B(3,0)$。由 $AC = 2$、$BC = 4$ 列方程求 $C$：$x_C^2 + y_C^2 = 4$，$(x_C-3)^2 + y_C^2 = 16$ $\implies$ $C\!\left(-\dfrac{1}{2},\,\dfrac{\sqrt{15}}{2}\right)$\par
正方形 $ABEF$ 向外（远离 $C$，$C$ 在上方故 $F$ 在下方）：$F(0,\,-3)$\par
正方形 $ACNM$ 向外（远离 $B$，$B$ 在右侧故 $M$ 在左下）：$M\!\left(-\dfrac{\sqrt{15}}{2},\,-\dfrac{1}{2}\right)$\par
$FM = \sqrt{\left(\dfrac{\sqrt{15}}{2}\right)^2 + \left(\dfrac{5}{2}\right)^2} = \sqrt{\dfrac{15}{4} + \dfrac{25}{4}} = \sqrt{10}$\par
\endgroup
