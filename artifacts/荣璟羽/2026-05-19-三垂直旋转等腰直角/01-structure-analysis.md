# 结构分析：一线三垂直 / 手拉手 / 等腰直角存在性

## 原题

**例题1**: 直线 $y = -\dfrac{\sqrt{3}}{3}x + 1$ 与 $x$ 轴、$y$ 轴分别交于 $A$、$B$，以 $AB$ 为直角边在第一象限作等腰 Rt$\triangle ABC$，$\angle BAC = 90°$。求点 $C$ 的坐标。

**例题2**: 直线 $AB$ 过 $A(-1,1)$、$B(2,0)$，交 $y$ 轴于 $C$。$D(0,n)$ 在 $C$ 上方，$S_{\triangle ABD} = 2$ 时，求 $P$ 使 $BP = BD$ 且 $BP \perp BD$。

**例题3**: 以 $\triangle ABC$ 的边 $AC$、$AB$ 向外作正方形 $ABEF$ 和 $ACNM$，联结 $BM$、$CF$。(1) 证明 $BM = CF$，$BM \perp CF$；(2) 若 $AB = 3$，$AC = 2$，$BC = 4$，求 $FM$。

## 约束

**全程禁止**高中向量表述：不得出现 $\overrightarrow{}$、向量旋转、向量点积、$\cdot$（点乘）等。所有推导只用初中几何（全等、旋转、勾股、坐标距离公式）。

## 一、题目场景

### 例题1
- 数学对象：直线与坐标轴交点、等腰直角三角形、第三顶点坐标
- 变量/参数：直线斜率含 $\sqrt{3}$、交点坐标含根式
- 函数/图形：直线 $y = -\sqrt{3}/3 \cdot x + 1$
- 已知条件：直线方程、$\angle BAC = 90°$、$AB = AC$、$C$ 在第一象限
- 要求目标：求 $C$ 的坐标

### 例题2
- 数学对象：直线解析式、坐标面积公式、等腰直角构造第三点
- 已知条件：$A(-1,1)$、$B(2,0)$、$D(0,n)$ 在 $C$ 上方、$S_{\triangle ABD} = 2$、$BP = BD$ 且 $BP \perp BD$
- 要求目标：先求 $D$ 坐标，再求 $P$ 坐标（两解）

### 例题3
- 数学对象：双正方形手拉手模型、全等证明、垂直证明、线段长度
- 已知条件：正方形 $ABEF$、$ACNM$ 向外、$AB = 3$、$AC = 2$、$BC = 4$
- 要求目标：(1) 证 $BM = CF$ 且 $BM \perp CF$；(2) 求 $FM$

## 二、核心结构

### 例题1
- 表面考点：等腰直角三角形第三顶点坐标
- 本质考点：**一线三垂直全等模型**——在坐标轴上构造三个直角，利用全等将水平位移与竖直位移互换
- 一句话问题模式：已知等腰直角三角形的直角顶点和一锐角顶点坐标，用全等求另一锐角顶点

### 例题2
- 表面考点：面积条件求点 + 等腰直角构造
- 本质考点：分两步——先用坐标面积公式确定已知点，再用一线三垂直构造第三点
- 一句话问题模式：条件先行确定已知点坐标，再套一线三垂直标准流程

### 例题3
- 表面考点：手拉手模型（全等 + 垂直 + 长度）
- 本质考点：共顶点双正方形 → SAS 全等 → 旋转 90° 得垂直 → 建系求长度
- 一句话问题模式：两个正方形共顶点，对应边全等，对应线段旋转 90° 后互相垂直

## 三、关键转化

### 例题1
- 最关键的转化：过直角顶点 $A$ 做水平辅助线（$x$ 轴），从 $B$、$C$ 向辅助线做垂线，构造"一线三直角"，证明 $\triangle ABB_1 \cong \triangle CC_1A$，将 $A$ 到 $B$ 的水平/竖直距离"翻转"得到 $A$ 到 $C$ 的位移
- 为什么降低计算量：只需算交点坐标和简单加减，无需解方程组或算旋转矩阵
- 不转化时的低效路径：硬设 $C(m,n)$ 列 $AB^2 = AC^2$ 和 $AB \perp AC$ 两个方程联立求解，根式运算量大

### 例题2
- 最关键的转化：先用面积条件锁定 $D$ 坐标，再对 $\triangle BDP$（等腰直角，$B$ 为直角顶点）套一线三垂直
- 为什么降低计算量：面积公式是代数入口，一线三垂直是几何入口，各司其职

### 例题3
- 最关键的转化：$\angle FAC = \angle BAM = 90° + \angle BAC$，SAS 全等后，由旋转 90° 的几何意义直接得出 $BM \perp CF$
- 为什么降低计算量：纯几何角度推导代替坐标计算

## 四、标准路径骨架

### 例题1（一线三垂直标准六步）
1. 求交点：令 $y = 0$ 得 $A(\sqrt{3}, 0)$，令 $x = 0$ 得 $B(0, 1)$
2. 做辅助线：过直角顶点 $A$ 沿 $x$ 轴做水平辅助线
3. 做垂线：过 $B$ 做辅助线垂线，垂足 $B_1(0,0)$；过 $C$ 做辅助线垂线，垂足 $C_1(x_C, 0)$
4. 证全等：$\angle AB_1B = 90°$，$\angle BAC = 90°$，$\angle AC_1C = 90°$（一线三垂直），由 $\angle ABB_1 = \angle CAC_1$ 及 $AB = AC$ 得 $\triangle ABB_1 \cong \triangle CC_1A$（AAS）
5. 对应位移：$AB_1 = \sqrt{3}$ → $CC_1 = \sqrt{3}$（$y_C = \sqrt{3}$）；$BB_1 = 1$ → $AC_1 = 1$（$x_C = \sqrt{3} + 1$）
6. 答案：$C(\sqrt{3} + 1,\, \sqrt{3})$

### 例题2
1. 求直线 $AB$ 解析式：$y = -\frac{1}{3}x + \frac{2}{3}$，得 $C(0, \frac{2}{3})$
2. 坐标面积公式求 $n$：$\frac{1}{2}|3n - 2| = 2$，$n = 2$（$n > \frac{2}{3}$），$D(0, 2)$
3. 一线三垂直求 $P$：$B(2,0)$ 为直角顶点，做 $x$ 轴辅助线
4. $D$ 的垂足 $D_1(0,0)$，$BD_1 = 2$，$DD_1 = 2$
5. 证 $\triangle BDD_1 \cong \triangle BP_1P$，得 $P_1$ 在 $B$ 右侧 $2$ 个单位处，$P$ 在辅助线上方 $2$ 个单位处
6. $P(4, 2)$；另一解 $P(0, -2)$（垂足与 $D_1$ 重合的退化情形）

### 例题3
1. 证全等：$AB = AF$，$AC = AM$，$\angle FAC = \angle BAM = 90° + \angle BAC$，$\triangle FAC \cong \triangle BAM$（SAS）
2. 证垂直：$\triangle FAC$ 绕 $A$ 旋转 $90°$ 得到 $\triangle BAM$，旋转 $90°$ 使对应线段 $CF$ 和 $BM$ 的夹角为 $90°$，即 $BM \perp CF$
3. 建系求 $FM$：设 $A(0,0)$，$B(3,0)$，$C(-\frac{1}{2}, \frac{\sqrt{15}}{2})$，得 $F(0, -3)$，$M(-\frac{\sqrt{15}}{2}, -\frac{1}{2})$
4. 距离公式：$FM = \sqrt{10}$

## 四点五、标准完整解与验算

### 例题1
- 关键交点：$A(\sqrt{3}, 0)$，$B(0, 1)$
- 全等对应：$B_1 \leftrightarrow C_1$，$A \leftrightarrow A$，$B \leftrightarrow C$（不对，应该是 $\triangle ABB_1 \cong \triangle CC_1A$，即 $A \leftrightarrow C$，$B \leftrightarrow C_1$，$B_1 \leftrightarrow A$）
  - 更正：$A \leftrightarrow C$，$B \leftrightarrow A$，$B_1 \leftrightarrow C_1$
  - 对应边：$AB_1 = CC_1$，$BB_1 = AC_1$，$AB = CA$
- 完整求解过程：如上六步
- 最终答案：$C(\sqrt{3} + 1,\, \sqrt{3})$
- 排除值：$C(\sqrt{3} - 1,\, \sqrt{3})$ 代入不满足 $\angle BAC = 90°$，舍去
- 验算：$AB = \sqrt{3 + 1} = 2$，$AC = \sqrt{1 + 3} = 2$ ✓；$AB_1 \cdot BB_1 = \sqrt{3} \cdot 1$，$AC_1 \cdot CC_1 = 1 \cdot \sqrt{3}$，比值为 $\sqrt{3}$ 一致 ✓
- 本题最短可靠路径：一线三垂直六步法

### 例题2
- 关键量：$k = -1/3$，$C(0, 2/3)$，$D(0, 2)$，$BD = 2\sqrt{2}$
- 面积方程：$\frac{1}{2}|(-1)(0 - n) + 2(n - 1) + 0(1 - 0)| = \frac{1}{2}|3n - 2| = 2$
- $n = 2$ 或 $n = -2/3$（舍，$n > 2/3$）
- 一线三垂直求 $P$：
  - $D_1(0, 0)$，$BD_1 = 2$，$DD_1 = 2$
  - $\triangle BDD_1$ 中 $\angle BD_1D = 90°$，$BD_1 = DD_1 = 2$，等腰直角
  - $\triangle BDP$ 也是等腰直角（$B$ 为直角顶点），$\triangle BDD_1 \cong \triangle BP_1P$
  - $P_1$ 在 $B$ 右侧：$x_P = 4$，$y_P = 2$ → $P(4, 2)$
  - $P_1$ 在 $B$ 左侧：$x_P = 0$，$y_P = -2$ → $P(0, -2)$
- 最终答案：$P(4, 2)$ 或 $P(0, -2)$
- 验算：$P(4,2)$: $BP = \sqrt{4+4} = 2\sqrt{2} = BD$ ✓，$BD^2 + BP^2 = 8+8 = 16 = DP^2$ → 勾股验证 $BD \perp BP$ ✓
  $P(0,-2)$: $BP = \sqrt{4+4} = 2\sqrt{2}$ ✓，同理 ✓

### 例题3
- 全等对应：$F \leftrightarrow B$，$A \leftrightarrow A$，$C \leftrightarrow M$
- 建系：$A(0,0)$，$B(3,0)$，$C(-\frac{1}{2}, \frac{\sqrt{15}}{2})$
- $F$：正方形 $ABEF$ 向外（远离 $C$），$C$ 在 $x$ 轴上方，$F$ 在 $x$ 轴下方，$F(0, -3)$
- $M$：正方形 $ACNM$ 向外（远离 $B$），$B$ 在右侧，$M$ 在左下，$M(-\frac{\sqrt{15}}{2}, -\frac{1}{2})$
- $FM = \sqrt{\left(\frac{\sqrt{15}}{2}\right)^2 + \left(\frac{5}{2}\right)^2} = \sqrt{\frac{15}{4} + \frac{25}{4}} = \sqrt{10}$
- 最终答案：(1) $BM = CF$，$BM \perp CF$；(2) $FM = \sqrt{10}$
- 验算：$BF = 3\sqrt{2}$（正方形对角线），$AM = 2\sqrt{2}$（正方形对角线），$\triangle ABM$ 与 $\triangle FAC$ 面积相等 ✓

## 五、出题人逻辑

### 例题1
- 诱导学生硬算的位置：设 $C(m,n)$ 列 $AB^2 = AC^2$ 和 $\perp$ 两个方程联立，根式运算复杂
- 真正的捷径：一线三垂直全等模型，直接将位移"翻转"
- 训练的可迁移能力：识别等腰直角 + 坐标条件 → 一线三垂直

### 例题2
- 诱导学生硬算的位置：面积公式容易算错绝对值
- 真正的捷径：面积锁定 $D$ 后，观察 $\triangle BDO$ 等腰直角简化计算（$DO = BO = 2$）
- 训练的可迁移能力：条件分步处理 + 一线三垂直

### 例题3
- 诱导学生硬算的位置：证垂直时迷失在角度推导中
- 真正的捷径：全等后直接用"旋转 90°"的几何意义，一句话证垂直
- 训练的可迁移能力：共顶点正方形 → 手拉手模型

## 六、学生卡点预测

### 基础薄弱学生
- 例题1：不会求含 $\sqrt{3}$ 的交点坐标（$-1/\sqrt{3}/3$ 化简出错）
- 例题1：不理解为什么要做辅助线和垂线
- 例题2：坐标面积公式不熟练
- 例题3：找不到全等的角条件（$\angle FAC = \angle BAM$）

### 中等学生
- 例题1：能做辅助线但不能完成全等证明（对应关系搞混）
- 例题2：求出 $D$ 后不会转一线三垂直
- 例题3：能证全等但证垂直卡住

### 较强学生（荣璟羽水平：好）
- 例题1：能完成但对应边关系写反（$AB_1$ 对应 $AC_1$ 还是 $CC_1$？）
- 例题2：可能漏掉 $P(0, -2)$ 这个解
- 例题3：证垂直角度推导不够简洁

## 七、变式原则

### 例题1 变式
- 核心不变量：等腰直角 + 坐标 → 一线三垂直全等
- 表层特征：直线与坐标轴交点
- 可变维度：直线斜率（换数字）、直角顶点位置（$A$ 或 $B$）、求 $C$ 的条件（给 $A$ 和 $B$ → 求哪个顶点）
- 深化阶梯：
  1. 换干净数字（$y = -x + 4$，$A(4,0)$，$B(0,4)$）
  2. 换直角顶点（$\angle ABC = 90°$ 而非 $\angle BAC = 90°$）
  3. 换问法（已知 $C$ 求 $AB$ 所在直线解析式）
  4. 不给象限条件，让学生自己讨论多解
- 允许的变换：改数字、改直角顶点、增加讨论
- 禁止的变换：引入向量、引入三角函数
- 表征切换：从"求交点"换成"已知两顶点坐标"

### 例题2 变式
- 核心不变量：分步——先用条件锁定已知点，再一线三垂直
- 可变维度：面积值、$D$ 的位置条件、$P$ 的约束

### 例题3 变式
- 核心不变量：双正方形共顶点 → SAS 全等 → 旋转 90° 证垂直
- 可变维度：正方形换成等边三角形（手拉手变体）、给不同边长、换问法

## 八、计算复杂度预算

- 原题计算层级：L3（含根式交点计算、坐标面积公式）
- 允许小步上升到：L4（换数字后讨论多解、换直角顶点）
- 禁止引入的计算负担：向量运算、三角函数、旋转矩阵
- 必须保留的可见支架：辅助线画法、全等证明的完整书写、对应边标注

## 九、推荐讲题任务包

- 适合的学习层级：L3-L4（荣璟羽水平好）
- 本题讲解目标：掌握一线三垂直标准六步法，能独立完成全等证明和坐标计算
- 不要直接讲的抽象话："旋转90°"——应改为"利用一线三垂直全等，将水平位移和竖直位移互换"
- 必须先问的问题：例题1——"$A$ 在坐标轴上，你觉得过 $A$ 做什么方向的辅助线最方便？"
- 关键讲解顺序：
  1. 先画图，标出 $A$、$B$ 的坐标
  2. 问学生：怎么找 $C$？需要什么关系？
  3. 引出一线三垂直：过 $A$ 做辅助线，从 $B$、$C$ 做垂线
  4. 标出三个直角，问学生"这三个直角能帮你证明什么？"
  5. 完成全等证明
  6. 由对应边得出坐标
- 最适合的具体数值例子：先讲 $y = -x + 4$ 的干净版本热身，再讲含 $\sqrt{3}$ 的原题
- 讲到哪里停下来让学生回答：全等对应关系（让学生自己配对）、对应边的具体数值

## 十、推荐练题任务包

- 若学生在 L0-L1：先练求交点坐标 + 画辅助线标注三个直角（不要求证全等）
- 若学生在 L2：练一线三垂直的完整六步，换干净数字
- 若学生在 L3：练例题2（分步 + 一线三垂直），练手拉手全等证明
- 若学生在 L4：练多解讨论（不给象限条件）、练直角顶点变化
- 若学生达到 L5-L6：练"已知 $C$ 反求直线解析式"、练判断等腰直角三角形个数
- 禁止出的跑偏变式：含向量表述的题、含三角函数的题
