# 结构分析：菱形存在性（动点轨迹型）

## 原题

本套为题组生成任务：为荣璟羽出一份**标准压轴式**的「菱形存在性」专题作业，共 10 题，难度**阶梯过渡**。核心形态是：给定两个定点 $A,B$（常为一次函数与坐标轴的交点），动点 $P$ 在某条轨迹（坐标轴 / 直线 / 中垂线）上运动，问是否存在 $P$（及对应第四点 $Q$），使 $A,B,P,Q$ 构成菱形。

## 题目场景

- 学生：荣璟羽。
- 主题：菱形存在性的**两类构成方式分类**。
- 训练目标：让学生形成固定的解题骨架——
  1. 先判断 $AB$ 在菱形中的角色：是**边**（邻边）还是**对角线**；
  2. 按“$AB$ 为边”和“$AB$ 为对角线”两类分别列方程求动点 $P$；
  3. 每个 $P$ 都要**补出第四点 $Q$ 并验证四边相等**（这是存在性验算，不是列完距离方程就结束）；
  4. 排除退化（$P$ 与 $A$ 或 $B$ 重合、四点共线）。

## 一、题目场景
- 数学对象：菱形 $ABQP$（四边相等）；动点 $P$、第四点 $Q$。
- 变量/参数：动点坐标 $t$，直线轨迹方程。
- 函数/图形：一次函数直线与坐标轴的交点；中垂线 $x=\text{常数}$。
- 已知条件：两定点 $A,B$（常给出直线方程）；动点 $P$ 所在轨迹；有时给面积或边长定解。
- 要求目标：存在性判断 + 全部符合条件的 $P$（及 $Q$）坐标，分类讨论后逐个验算。

## 二、核心结构

### 2.1 表层信息
- 表面考点：菱形存在性、分类讨论、坐标两点距离、中点坐标。
- 题型功能：`existence_or_parameter`（存在性/参数讨论）。
- 是否值得完整 structural analysis：是；理由：菱形存在性压轴题的关键不是“会算距离”，而是“知道要分两类讨论、且每类算完都要补点验四边”，需要锁定这个分类骨架。
- 一句话问题模式：定点 $A,B$ + 动点 $P$ 在轨迹上，问使 $A,B,P,Q$ 为菱形的 $P$——**分“$AB$ 为边”和“$AB$ 为对角线”两类，每类求候选后必补 $Q$ 验四边相等**。

### 2.2 结构表达

> 菱形存在性的统一骨架：先定 $AB$ 的角色，再分类求 $P$，最后补 $Q$ 验算。

#### 命题网络（核心）

- $P_1$（题设）：已知定点 $A,B$，动点 $P$ 在轨迹 $L$ 上。
- $P_2$（定义）：四点构成菱形 $\Leftrightarrow$ 四边相等；等价地说，先为平行四边形再加邻边相等（或对角线互相垂直平分）。
- $P_3$（分类，关键）：$AB$ 在菱形中只有两种角色——
  - 情况一（**$AB$ 为边**）：$AB$ 与 $AP$（或 $BP$）是邻边，需 $|AP|=|AB|$（或 $|BP|=|AB|$）。
  - 情况二（**$AB$ 为对角线**）：$AB$ 与 $PQ$ 是两条对角线，需 $PQ$ 被 $AB$ 垂直平分（即 $P,Q$ 关于 $AB$ 中点对称，且 $PQ\perp AB$）。
- $R_1: P_1 + P_3(\text{情况一}) \to P_4$，方法：用 $|AP|=|AB|$（或 $|BP|=|AB|$）列方程解 $P$。
- $R_2: P_1 + P_3(\text{情况二}) \to P_5$，方法：用“$P,Q$ 关于 $AB$ 中点对称 + $PQ\perp AB$”列方程解 $P$。
- $P_6$（检查，必须）：对每个候选 $P$，**补出 $Q$**（情况一用平行四边形 $Q=B+P-A$；情况二用对称 $Q=2M-P$，$M$ 为 $AB$ 中点），再**验证四边相等**。
- $R_3: P_4/P_5 + P_6 \to P_7$，方法：四边平方比较 + 排除退化（点重合、共线）。
- 目标：由 $P_3\to P_4/P_5\to P_6\to P_7$ 得到全部合法解。

#### 关键纠错点（出题人埋的坑）

- **只列 $|AP|=|AB|$ 不验四边**：满足距离条件的 $P$ 不一定是菱形顶点（如 Q3 中 $P=(-4,0)$ 满足 $|BP|=|AB|$ 但补出 $Q$ 后四边不全等）。这是存在性题的“筛选”环节，不可省。
- **漏掉某一类**：只算“$AB$ 为边”或只算“$AB$ 为对角线”，会丢解。
- **顶点顺序写错**：情况一顺序为 $A\text{-}B\text{-}Q\text{-}P$（$AB$、$AP$ 为邻边）；情况二顺序为 $A\text{-}P\text{-}B\text{-}Q$（$AB$、$PQ$ 为对角线）。

### 2.3 解题主链
```text
定AB角色 → [情况一:|AP|=|AB|列方程] ∪ [情况二:PQ⊥AB且被平分列方程] → 每个P补Q → 验四边相等+排退化 → 列全部解
```

### 2.4 模型标签
- model_id：`coordinate_quadrilateral_existence`
- model_name：坐标四边形存在性（菱形）
- configuration：已知定点 $A,B$ + 动点轨迹，要求菱形顶点 $P,Q$。
- 可迁移方向：菱形 $\to$ 正方形 $\to$ 矩形存在性；“$AB$ 为边 / 为对角线”的二分法可迁移到一切坐标四边形存在性。
- 非同构边界：纯几何判定题（无动点、无坐标）不属于本套存在性题；只验证一组对边平行不算菱形（non-example）。

## 二点五、知识点/模型锚点

- 建议讲义标题：菱形存在性专题——$AB$ 为边还是为对角线？
- 知识点/模型名称：菱形存在性分类讨论 + 两点距离公式 + 中点对称 + 对角线互相垂直。
- 核心公式/定理：
  - 两点距离：$|P_1P_2|=\sqrt{(x_1-x_2)^2+(y_1-y_2)^2}$。
  - 情况一补点：$Q=B+P-A$（平行四边形对角线中点重合）。
  - 情况二补点：$Q=2M-P$，$M=\dfrac{A+B}{2}$；并要求 $PQ\perp AB$。
  - 菱形面积：$=\dfrac12\,|AB|\cdot|PQ|$（对角线乘积一半）。
- 使用条件：动点轨迹已知；题目问“是否存在点 $P$”。
- 入口信号：出现“在直线/坐标轴上是否存在点 $P$……使……为菱形”。
- 易混边界：
  - 把“$AB$ 为对角线”和“$AB$ 为边”混为一谈，只算一类。
  - 列完 $|AP|=|AB|$ 就当求完了，忘记补 $Q$ 验四边。
  - 忘记排除 $P$ 与 $A,B$ 重合的退化情形。
- 本题如何体现：先用 $AB$ 角色二分，每类求候选，再统一进入“补点 + 验四边”的筛选。
- 可作为例题的结构层级：边解脚手架 $\to$ 对角线脚手架 $\to$ 两类完整分类 $\to$ 含面积/边长反求的综合。

## 三、关键转化

- 最关键的转化：把“构成菱形”转化为“$AB$ 是边还是对角线”的二选一，每一类都能落到一个可解的距离/垂直方程上。
- 为什么降低计算量：分类后每类只需一个方程，避免对每个候选都重解二次方程组。
- 不转化时的低效路径：直接对四点列四边相等的方程组硬解，既要处理顶点顺序又要联立，计算量大且极易漏解。

## 四、标准路径骨架

1. 写出定点 $A,B$ 的坐标和 $|AB|$，标出动点 $P$ 的轨迹。
2. 分两类：情况一 $AB$ 为边；情况二 $AB$ 为对角线。
3. 情况一：用 $|AP|=|AB|$（必要时也试 $|BP|=|AB|$）列方程解 $P$。
4. 情况二：设 $P=(t,\ast)$，用 $P,Q$ 关于 $AB$ 中点对称 + $PQ\perp AB$ 列方程解 $t$。
5. 对每个候选 $P$：补出 $Q$，验证四边平方全等，排除退化。
6. 汇总全部解；多解时逐个代回。

## 四点五、标准完整解与验算

典型例（$AB$ 为边 + $AB$ 为对角线，完整 3 解）：

直线 $y=-\dfrac34 x+3$ 与 $x$ 轴交于 $A(4,0)$，与 $y$ 轴交于 $B(0,3)$，$|AB|=5$。点 $P$ 在 $x$ 轴上运动（异于 $A$），是否存在 $P,Q$ 使 $A,B,P,Q$ 为菱形顶点？

**情况一（$AB$ 为边）：**
- 由 $|AP|=|AB|=5$：$|(t,0)-(4,0)|=5\Rightarrow |t-4|=5\Rightarrow t=9$ 或 $t=-1$。
  - $t=9$：$P=(9,0)$，补 $Q=B+P-A=(0,3)+(9,0)-(4,0)=(5,3)$。四边 $AB^2=BQ^2=QP^2=PA^2=25$，是菱形。✓
  - $t=-1$：$P=(-1,0)$，补 $Q=(-5,3)$。四边均为 $5$，是菱形。✓
- 由 $|BP|=|AB|=5$：$t^2+9=25\Rightarrow t=\pm4$。$t=4$ 即 $A$（退化舍去）；$t=-4$：$P=(-4,0)$，补 $Q=(-8,3)$，四边 $AB=5$ 但 $BQ=8\ne5$，**不是菱形，舍去**。

**情况二（$AB$ 为对角线）：**
- $AB$ 中点 $M=(2,\tfrac32)$。$P=(t,0)$，则 $Q=2M-P=(4-t,3)$。菱形要求 $PQ\perp AB$：
  $\overrightarrow{PQ}=(4-2t,\,3)$，$\overrightarrow{AB}=(-4,3)$，点积 $-4(4-2t)+9=8t-7=0\Rightarrow t=\dfrac78$。
  - $P=(\tfrac78,0)$，$Q=(\tfrac{25}{8},3)$，四边平方均为 $\dfrac{25}{8}$ 的同值，是菱形。✓

**结论：** 共 3 个 $P$：$(9,0)$、$(-1,0)$、$(\tfrac78,0)$。

退化检查：$t=4$（$P=A$）已舍；$t=-4$ 验四边后舍。

本题最短可靠路径：$AB$ 角色二分 $\to$ 每类列方程 $\to$ 补 $Q$ 验四边 $\to$ 汇总。

## 五、出题人逻辑

- 诱导学生硬算的位置：直接对四点列四边相等方程组。
- 真正的捷径：先 $AB$ 角色二分，每类一个方程，再统一补点验算。
- 训练的可迁移能力：分类意识 + “候选 + 筛选（补点验四边）”的存在性套路 + 对角线互相垂直平分的坐标表达。

## 六、学生卡点预测

- 读题/入手动作卡点：看不出要分“$AB$ 为边 / 为对角线”两类；把动点在直线上的题当定点题硬凑。
- 建模/关系入口卡点：不会用对角线互相垂直平分来列情况二的方程；情况一里只列 $|AP|=|AB|$ 就停，忘记补 $Q$。
- 求解/检查卡点：漏掉某一类；满足距离的 $P$ 没验四边就当解；解出 $t$ 没检查是否与 $A,B$ 重合；顶点顺序写反导致四边算错。

## 七、变式原则

- 核心不变量：始终是“定点 $A,B$ + 动点轨迹 + 菱形条件 $\to$ 分两类求 $P$ 并补点验算”。
- 表层特征：题目常包装成“直线与坐标轴交于 $A,B$，点 $P$ 在 $x$ 轴/$y$ 轴/直线上”。
- 可变维度：动点所在轨迹（$x$ 轴 / $y$ 轴 / 中垂线 / 一般直线）、是否给面积或边长定解、$AB$ 是否水平、解的个数（2 解 / 3 解）。
- 深化阶梯：
  1. 原题复现：单一类（仅“$AB$ 为边”）+ 水平 $AB$，整数解。
  2. 同结构换数：单一类（仅“$AB$ 为对角线”）+ 给面积定解。
  3. 同结构换轨迹：动点从坐标轴换到中垂线（对角线天然成立）。
  4. 两类完整分类：边解 + 对角线解并存，3 解。
  5. 条件包装：含一次函数轨迹（非水平 $AB$），解中出现分数。
  6. 结构部分隐藏：动点在另一坐标轴，部分候选被验算筛掉（强调筛选）。
  7. 反向构造：给边长或面积反求动点 / 对角线。
- 允许的变换：换轨迹、换 $AB$ 是否水平、给面积/边长定解、换动点所在坐标轴。
- 禁止的变换：引入二次函数轨迹、圆、立体几何；引入判别式为负或系数繁琐的方程。
- 表征切换：纯坐标计算 $\leftrightarrow$ 含一次函数轨迹 $\leftrightarrow$ 含面积/边长反求。
- 包装方式：矩形/直线交点背景、中垂线背景。
- 近迁移例子：水平 $AB$ + 动点在坐标轴（2 解）。
- 远迁移例子：一次函数轨迹（非水平 $AB$）+ 两类完整分类（3 解，含分数解）。
- 反例/伪变式：
  - 只列 $|AP|=|AB|$ 不补 $Q$ 验四边（伪完成）。
  - 只算一类就下结论（漏解）。
  - 把动点轨迹当定点处理。

## 八、计算复杂度预算

- 原题计算层级：一次绝对值/平方方程 + 一次点积（垂直）。
- 允许小步上升到：解中出现简单分数（如 $\tfrac78,\tfrac74,-\tfrac76$），或给面积反求半对角线。
- 禁止引入的计算负担：二次函数轨迹、含根号系数的方程组、三参数讨论、圆与直线联立。
- 必须保留的可见支架：两类分类必须显式写出；每个候选 $P$ 必须显式补 $Q$ 验四边。

## 九、推荐讲题任务包

- 建议的本轮教学入口：先建立“$AB$ 是边还是对角线”的二分，再练每类怎么列方程、怎么补点验算。
- 本题讲解目标：让学生形成“二分 + 候选 + 补点验四边 + 排退化”的四步骨架。
- 不要直接讲的抽象话：不要只说“数形结合”“分类讨论思想”，要落到“$AB$ 是边还是对角线”“补出 $Q$ 再比四边”。
- 必须先问的问题：$AB$ 在这个菱形里是边还是对角线？动点 $P$ 在哪条轨迹上？
- 关键讲解顺序：角色二分 $\to$ 情况一列方程 $\to$ 情况二列方程 $\to$ 补点验四边。
- 最适合的具体数值例子：$AB$ 水平、整数解的题。
- 讲到哪里停下来让学生回答：列出情况一的候选后，让学生判断哪个要补点、补点后哪个成立。

## 十、推荐练题任务包

- 若卡在读题/入手动作，出什么题：水平 $AB$ + 动点在坐标轴，单一“$AB$ 为边”类，2 解（Q1、Q6）。
- 若卡在建模或关系入口，出什么题：单一“$AB$ 为对角线”类 + 给面积定解（Q2、Q5）。
- 若卡在求解和检查，出什么题：含一次函数轨迹的两类完整分类，强制补点验算（Q3、Q4、Q7）。
- 若原题已稳，如何小步迁移：动点换到另一坐标轴，含被筛掉的候选（Q8）。
- 若结构识别已稳，如何深化/抽象/包装：反向构造——给边长/面积反求（Q9、Q10）。
- 禁止出的跑偏变式：含二次函数轨迹、圆、立体几何的菱形题；判别式为负或系数繁琐的方程。

10 题分组（每组不超过 3 题，覆盖多个深化阶梯）：

- 第一组（脚手架，2 题）：Q1 水平 $AB$ + 动点在 $y$ 轴，仅边解 2 解；Q2 $AB$ 为对角线 + 给面积定解。
- 第二组（两类完整分类，2 题）：Q3、Q4 一次函数轨迹，边解 + 对角线解各 3 解。
- 第三组（轨迹/角色强化，2 题）：Q5 中垂线（对角线天然成立）+ 面积；Q6 水平 $AB$ + 动点在 $y$ 轴，对角线无解（强化“无解也要说明”）。
- 第四组（迁移与筛选，2 题）：Q7 非水平轨迹完整 3 解（含分数解）；Q8 动点在 $y$ 轴、部分候选被筛掉。
- 第五组（反向构造，2 题）：Q9 给边长 + 一对角线反求另一对角线与面积；Q10 给边长反求动点（对角线 + 边长约束）。
