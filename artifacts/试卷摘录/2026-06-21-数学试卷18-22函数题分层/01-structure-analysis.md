## 核心结构

这批题不是一个单题模型，而是一组八年级函数题的分层训练集合。主线结构分为四类：

1. 一次函数基础判断：斜率、截距、平移、经过象限、函数值范围。
2. 图像与代数互译：交点、方程组解、函数图像解不等式。
3. 参数与坐标几何：含参数直线、坐标轴交点、面积、定点、特殊点。
4. 综合建模：一次函数或反比例函数用于交通、输液、滑梯等实际情境，或分段函数新定义。

## 关键转化

- 图像经过象限 -> 判断斜率和截距符号。
- 两函数交点 -> 联立方程，或用交点横坐标作为不等式分界点。
- 直线平移/平行 -> 斜率不变，只改常数项或截距位置。
- 表格/图像建模 -> 先取两点求一次函数，再代入或列不等式。
- 反比例函数应用 -> 先确定点坐标，再由 $xy=k$ 建立模型。
- 分段函数新定义 -> 先判断 $x$ 的符号，再代入对应表达式。

## 出题人逻辑

易题主要检查入口动作是否熟练：读截距、读斜率、代入、平移、判断区间。

中题在易题动作上增加一个条件组合：参数范围、图像大小关系、实际问题中的单位换算或坐标几何条件。

难题把多个动作串联：先求解析式或模型，再处理面积、存在性、比例、分段或决策条件。

## 变式原则

本轮不重新编造变式题，而是把原题库按难度与题型重排：

- 学生练习卷：纳入题库中适合独立练习且题面完整的基础与中等题。
- 教师练习卷：与学生卷题目一致，但保留答案、解析和简要教学备注。
- hard 讲义：每道 hard 题作为一个讲解单元，按“知识点陈述 -> 例题 -> 路线 -> 双栏讲解 -> 易错提醒 -> 方法提醒”组织。

## 推荐练题任务包

练习卷按题型和难度排序：

1. 填空题：按内部难度从基础到综合排序，训练直接计算与判断。
2. 选择题：只纳入选项完整、学生能独立作答的题。
3. 解答题：训练完整链条书写。

选项缺失或依赖原卷图像选项的选择题不进入正式学生卷，保留在题库源文件中等待后续补图或补选项。

## 推荐讲题任务包

hard 题讲义分四个单元：

1. 两条一次函数交点、面积与坐标轴存在性。
2. 输液剩余量图像与实际流速换算。
3. 交通流量一次函数与可变车道决策。
4. 新定义分段函数与矩形交点个数分类。

## 计算复杂度预算

- 练习卷不增加原题外计算负担，保留原题难度。
- 教师版解析以短步骤为主，避免把简单题讲成大段讲义。
- hard 讲义每题只展开关键链条，不做额外变式。
