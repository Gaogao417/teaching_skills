\begin{question}[points=4]
  （原第3题）俄罗斯方块是一款经典休闲益智游戏，如图是假设俄罗斯方块时某一时刻的截图，若在以 $O$ 为原点建立的平面直角坐标系中，小宇将上方的方块先向左移动 $2$ 个格子，再向下移动 $6$ 个格子后，点 $A$ 恰好落在点 $B(3,1)$ 处，则上方的方块移动前点 $A$ 所在位置的坐标为（\quad）。
  \begin{choices}
    \item $(4,7)$
    \item $(5,6)$
    \item $(5,7)$
    \item $(7,5)$
  \end{choices}
\end{question}



\begin{question}[points=4]
  （原第4题）下列命题，其中是假命题的是（\quad）。
  \begin{choices}
    \item 对角线相等的菱形是正方形
    \item 对角线互相垂直平分的四边形是菱形
    \item 有一个角是直角且对角线互相平分的四边形是矩形
    \item 一组对角相等且一组对边相等的四边形是平行四边形
  \end{choices}
\end{question}



\begin{question}[points=4]
  （原第5题）已知直线 $l_1:y=hx+b\ (h\ne0)$ 与直线 $l_2:y=kx-6\ (k<0)$ 在第三象限交于点 $M$，若直线 $l_1$ 与 $x$ 轴的交点为 $B(3,0)$，则 $h$ 的取值范围是（\quad）。
  \begin{choices}
    \item $-2<h<2$
    \item $-2<h<0$
    \item $0<h<4$
    \item $0<h<2$
  \end{choices}
\end{question}



\begin{question}[points=4]
  （原第6题）如图，在平行四边形 $ABCD$ 中，$AD=2AB$，$M$ 是 $AD$ 中点，作 $BE\perp CD$ 交 $CD$ 于 $E$，连接 $EM$ 并延长，与 $BA$ 的延长线交于 $F$。下列结论中，不正确的是（\quad）。
  \begin{choices}
    \item $BM$ 平分 $\angle ABC$
    \item $BM=MF$
    \item $\angle AMB=3\angle MED$
    \item $S_{\triangle BCE}=2S_{\triangle ABE}$
  \end{choices}
\end{question}



\begin{question}[points=4]
  （原第11题）已知函数 $y=\dfrac{2}{3}x+1$，如果函数值 $y>5$，那么相应的自变量 $x$ 的取值范围是\underline{\hspace{32mm}}。
\end{question}



\begin{question}[points=4]
  （原第12题）我国古代数学著作《算法统宗》中有这样一个问题：平地竿子未起，斜竿离地一尺。送行二步与人齐，五尺人高曾记。良工商士张好奇，算出竿长有几？（$1$ 步 $=5$ 尺）这段话的意思是：竿子静止时，斜竿离地面 $1$ 尺高；将竿子的脚跟向前推动 $2$ 步 $10$ 尺时，斜竿就和推竿子的人一样高，同为 $5$ 尺。小明根据上述信息画出了如图所示的示意图，则可求竿子的长度为\underline{\hspace{28mm}}尺。
\end{question}



\begin{question}[points=4]
  （原第13题）如图，在平面直角坐标系 $xOy$ 中，已知点 $A(4,3)$，$AB\perp y$ 轴于点 $B$。以 $AB$ 为边作菱形 $ABCD$，若点 $C$ 在 $x$ 轴上，则点 $D$ 的坐标为\underline{\hspace{36mm}}。
\end{question}



\begin{question}[points=4]
  （原第14题）如图，在矩形 $ABCD$ 中，$AB=3$，$BC=4$，点 $P$ 是边 $AB$ 上一动点，连接 $CP$，将 $\triangle BCP$ 沿着 $CP$ 翻折后得到 $\triangle ECP$，若 $BE$、$EC$ 与边 $AD$ 分别交于点 $F$、$G$，且 $AP=BF$，则 $AP$ 的长为\underline{\hspace{28mm}}。
\end{question}



\begin{question}[points=4]
  （原第15题）定义：如果直线 $y_1=k_1x+b_1\ (k_1\ne0)$ 与直线 $y_2=k_2x+b_2\ (k_2\ne0)$ 满足如下条件：$k_1\cdot k_2=-1$ 且 $b_1+b_2=0$，那么我们就说这两条直线具有“和谐关系”。例如，直线 $y_1=2x+3$ 与直线 $y_2=-\dfrac12x-3$，它们具有“和谐关系”。如果直线 $y_1=h_1x+2\ (h_1>0)$ 与直线 $y_2=kx-2$ 具有“和谐关系”，且这两条直线与 $y$ 轴围成的三角形面积为 $\dfrac{16}{5}$，则 $h_1=$\underline{\hspace{28mm}}。
\end{question}



\begin{question}[points=4]

\end{question}



\begin{question}[points=4]

\end{question}



\begin{question}[points=4]
