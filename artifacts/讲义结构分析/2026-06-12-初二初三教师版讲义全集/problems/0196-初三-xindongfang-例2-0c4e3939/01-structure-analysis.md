# 结构分析：初三-xindongfang-例2-二次函数、函数综合

## 原题
(1) 求抛物线的表达式及点C的坐标；

(2) 点P是Y轴的负半轴上的一点，如果 $ \Delta P B C $与 $ \Delta B C D $相似，且相似比不为1，求点P的坐标；

(3) 将 $ \angle C B D $绕着点B逆时针方向旋转，使射线BC经过点A，另一边与抛物线交于点E （点E在对称轴的右侧），求点E的坐标.

## 五、出题人逻辑
- 诱导学生硬算的位置：图形/图像信息多，容易先猜图或堆辅助线。
- 真正的捷径：先找与所求直接相连的不变量或基本关系。
- 训练的可迁移能力：从题面识别结构、建立关系、求解后检查。

## 六、学生卡点预测
- 读题/入手动作卡点：无法把截图、选项、分问或前置条件合并成完整题意。
- 建模/关系入口卡点：不知道该用代入、比例、相似、三角比、向量法则还是方程。
- 求解/检查卡点：忽略定义域、增根、对应顺序、范围、单位或退化情况。

## 七、变式原则
- 核心不变量：把根、顶点、开口、对称轴转为顶点式/一般式中的参数关系。
- 表层特征：二次函数、函数综合、相似三角形、图形运动；题号 例2；来源 初三数学春季——10 二模二次函数与相似三角形。
- 可变维度：数值、所求目标、图文表征、条件包装、是否给选项。
- 深化阶梯：原题复现 → 同结构换数 → 同结构换问法 → 同结构换表征 → 条件包装 → 结构部分隐藏。
- 允许的变换：只改变一个主维度，并保留同一个核心关系入口。
- 禁止的变换：加入题面没有的新性质，或让计算复杂度掩盖结构训练。
- 表征切换：文字题、函数图像、几何图、表格、选项判断之间切换。
- 包装方式：把核心条件放入实际情境、图形描述或选择题干扰项。
- 近迁移例子：保留题型结构，仅替换干净数字。
- 远迁移例子：把同一转化迁移到新的函数/几何/应用场景。
- 反例/伪变式：只保留相似背景但核心关系换成另一知识点的题，不应作为同构练习。

## 八、计算复杂度预算
- 原题计算层级：需复核；练习阶段先按中等以下计算量控制。
- 允许小步上升到：多一个中间量、一次简单代入或一个分类分支。
- 禁止引入的计算负担：复杂根式运算、多重参数讨论、超纲定理或冗长辅助线。
- 必须保留的可见支架：对象标注、核心条件、关系式入口、答案检查点。

## 九、推荐讲题任务包
- 建议的本轮教学入口：find_entry → build_relation → solve_and_check。
- 本题讲解目标：让学生明确第一步转化来自哪个条件，并能完成候选答案检查。
- 不要直接讲的抽象话：不要只说“数形结合”“观察可得”“灵活运用”。
- 必须先问的问题：题目给了哪些对象？所求目标和哪个条件最近？有没有定义域/图形限制？
- 关键讲解顺序：读题标对象 → 圈关键条件 → 建关系 → 求解 → 验算/排除。
- 最适合的具体数值例子：用原题中的最小整数或最清楚的一组量先演示。
- 讲到哪里停下来让学生回答：关系式刚建立后、候选答案刚出现后。

## 十、推荐练题任务包
- 若卡在读题/入手动作，出什么题：给同结构题，只要求画/列对象关系表。
- 若卡在建模或关系入口，出什么题：给半成品条件，让学生补关键方程、比例或对应关系。
- 若卡在求解和检查，出什么题：给已列式的题，专练求解、验根、范围和选项排除。
- 若原题已稳，如何小步迁移：同结构换一个干净数字或把填空改成选择。
- 若结构识别已稳，如何深化/抽象/包装：把核心条件藏入图形描述、函数图像或应用情境。
- 禁止出的跑偏变式：缺少原图却要求猜性质，或加入新知识点导致核心结构改变。
