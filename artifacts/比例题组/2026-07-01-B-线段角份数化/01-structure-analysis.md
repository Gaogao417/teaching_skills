# 结构分析：线段与角的份数化

## 原题
围绕“几何条件 -> 比例/份数 -> 调用 A 组比例工具 -> 求一个目标量”设计 B 组训练。每题只求一个线段或一个角，避免目标过多遮蔽底层动作。

典型题：

1. 点 $A,B,C$ 在同一直线上，$B$ 在 $A,C$ 之间。若 $AB=2BC$，$AC=18$，求 $BC$。
2. 点 $A,B,C$ 在同一直线上，$B$ 在 $A,C$ 之间。若 $4AB-3BC=0$，$AC=21$，求 $AB$。
3. 射线 $OB$ 在 $\angle AOC$ 内部。若 $\angle AOB: \angle BOC=2:3$，$\angle AOC=100^\circ$，求 $\angle BOC$。
4. 射线 $OB$ 在 $\angle AOC$ 内部。若 $5\angle AOB-2\angle BOC=0$，$\angle AOC=112^\circ$，求 $\angle AOB$。

## 一、题目场景
- 数学对象：同一直线上的相邻线段、同一个角内的相邻角。
- 变量/参数：线段长、角度。
- 函数/图形：无函数；简单线段图或角图。
- 已知条件：整体量、部分量之间的倍数/比例/方程式关系。
- 要求目标：每题只求一个指定部分量。

## 二、核心结构
### 2.1 表层信息
- 表面考点：线段和角的比例计算。
- 题型功能：`basic_skill_transfer`。
- 是否值得完整 structural analysis：是；核心不是新计算，而是把几何语言转成 A 组比例工具能处理的形式。
- 一句话问题模式：先找整体与部分，再把条件转成份数或比例，最后用 A 组方法选择完成计算。

### 2.2 结构表达
#### 判别条件表
- 必要条件：两个部分必须组成同一个整体，如 $AC=AB+BC$ 或 $\angle AOC=\angle AOB+\angle BOC$。
- 充分条件：部分量比例与整体量已知，即可求目标部分量。
- 常见干扰项：$4a=3b$ 错写成 $a:b=4:3$；忘记题目只问一个量。
- 最短检查动作：先写“整体 = 部分 + 部分”，再翻译条件。

#### 情景量表
| 量 | 类型/单位 | 题设关系 | 未知/已知 |
|---|---|---|---|
| $AB,BC$ | 线段 | $AB+BC=AC$ | 一个目标量未知 |
| $\angle AOB,\angle BOC$ | 角 | 相加为 $\angle AOC$ | 一个目标角未知 |
| 整体量 | 长度/角度 | 对应总份数 | 已知 |

#### 命题网络
- P1（题设）：两个部分量组成同一个整体。
- P2（题设）：部分量之间有倍数、比例或方程式关系。
- P3（转化）：把 P2 转成部分量份数比。
- P4（转化）：整体量对应总份数。
- P5（调用 A）：根据份数关系选择倍数/份数或交叉相乘。
- R1：P1 -> 写整体式。
- R2：P2 -> P3，方法：条件转比例；方程系数需要反向成比。
- R3：P3 + P1 -> P4，方法：总份数相加。
- R4：P4 + 目标量份数 -> 目标量。
- 目标：求一个指定部分量。

### 2.3 解题主链
```text
读目标 -> 找整体和部分 -> 条件转比例/份数 -> 总份数对应整体 -> 求目标部分
```

### 2.4 模型标签
- model_id：segment_angle_share_with_proportion_tool
- model_name：线段角整体-部分份数化
- configuration：已知整体和部分关系，求一个部分量。
- 可迁移方向：相似图形对应边比例、面积比链。
- 非同构边界：两个量不属于同一整体时不能加份数。

## 二点五、知识点/模型锚点
- 建议讲义标题：线段和角：先翻译，再调用比例工具
- 知识点/模型名称：整体-部分结构、方程式关系转比例、份数统一。
- 核心公式/定理：若 $4a=3b$，则 $a:b=3:4$；整体份数等于两个部分份数之和。
- 使用条件：两个部分能组成同一个整体。
- 入口信号：“B 在 A,C 之间”“OB 在角内部”“总长”“总角”。
- 易混边界：方程系数和比例份数相反；题目问一个量时不必写两个答案。
- 本题如何体现：线段和角只是表面不同，底层都先翻译为份数。
- 可作为例题的结构层级：A 组比例工具的几何迁移。

## 三、关键转化
- 最关键的转化：$4a-3b=0 \Rightarrow 4a=3b \Rightarrow a:b=3:4$。
- 为什么降低计算量：转成份数后只需“总份数 -> 一份 -> 目标份数”。
- 不转化时的低效路径：设两个未知数列方程组，或把系数顺序直接抄成比例。

## 四、标准路径骨架
1. 先做什么：读清题目只求哪个目标量。
2. 再做什么：写整体式，如 $AC=AB+BC$。
3. 建立什么关系：把倍数、比例、方程式条件转成份数。
4. 如何求解：整体量除以总份数，再乘目标份数。
5. 需要检查什么：目标量是否只写一个；部分相加是否回到整体。

## 四点五、标准完整解与验算
- $AB=2BC, AC=18$：$AB:BC=2:1$，总 $3$ 份，$BC=6$。
- $4AB-3BC=0, AC=21$：$4AB=3BC$，$AB:BC=3:4$，总 $7$ 份，$AB=9$。
- $\angle AOB:\angle BOC=2:3, \angle AOC=100^\circ$：总 $5$ 份，$\angle BOC=60^\circ$。
- $5\angle AOB-2\angle BOC=0, \angle AOC=112^\circ$：$\angle AOB:\angle BOC=2:5$，总 $7$ 份，$\angle AOB=32^\circ$。
- 排除值：长度和角度为正。
- 验算：目标量与另一个部分相加回整体。
- 本题最短可靠路径：翻译成份数，再调用 A 组比例工具。

## 五、出题人逻辑
- 诱导学生硬算的位置：一看到几何名词就忘记它其实只是比例工具迁移。
- 真正的捷径：先找整体，再把条件翻译成份数。
- 训练的可迁移能力：把题面语言转换成可计算比例式。

## 六、学生卡点预测
- 读题/入手动作卡点：没先看目标量，导致每题都求两个量。
- 建模/关系入口卡点：找不到整体；方程式关系转比例反了。
- 求解/检查卡点：总份数忘加，或者答案写多。

## 七、变式原则
- 核心不变量：两个部分共享一个整体。
- 表层特征：线段、角、比例、倍数、方程式包装。
- 可变维度：已知整体/已知部分、目标量位置、系数形式。
- 深化阶梯：直接比例 -> 倍数关系 -> 方程式转比例 -> 已知部分反求整体 -> 角度同构。
- 允许的变换：小整数比例、整洁角度。
- 禁止的变换：不共整体的量直接相加；一题同时问过多目标。
- 表征切换：文字、比例式、方程式。
- 包装方式：把比例条件藏在等式中。
- 近迁移例子：C 组相似矩形对应边比例。
- 远迁移例子：D 组面积拆分后的份数统一。
- 反例/伪变式：$AB$ 与无关线段 $DE$ 的比例不能求 $AC$。

## 八、计算复杂度预算
- 原题计算层级：一层份数。
- 允许小步上升到：已知部分反求整体、方程式条件转比例。
- 禁止引入的计算负担：多变量方程组。
- 必须保留的可见支架：目标量、整体、条件翻译、份数。

## 九、推荐讲题任务包
- 建议的本轮教学入口：先回扣 A 组：比例工具已经会了，现在学题面翻译。
- 本题讲解目标：看出几何条件背后仍是比例式。
- 不要直接讲的抽象话：不要只说“按比例分配”。
- 必须先问的问题：目标量是谁？它属于哪个整体？条件给的是哪两个部分的关系？
- 关键讲解顺序：线段直接关系 -> 线段方程关系 -> 角度同构。
- 最适合的具体数值例子：$4AB-3BC=0, AC=21$。
- 讲到哪里停下来让学生回答：转出 $AB:BC=3:4$ 后停。

## 十、推荐练题任务包
- 若卡在读题/入手动作，出什么题：每题只求一个目标量的线段题。
- 若卡在建模或关系入口，出什么题：方程式关系转比例的线段/角混排。
- 若卡在求解和检查，出什么题：已知部分反求整体或求另一个部分。
- 若原题已稳，如何小步迁移：角度题同构迁移。
- 若结构识别已稳，如何深化/抽象/包装：把条件写成 $ma-nb=0$。
- 禁止出的跑偏变式：角平分线这类过于直给、无法训练比例翻译的题。
