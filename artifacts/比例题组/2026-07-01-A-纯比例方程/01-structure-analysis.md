# 结构分析：纯比例方程

## 原题
围绕“比例式本身怎么解”设计 A 组工具训练。讲解只设一个大题，含 4 个小问：

1. $2,4,3,x$ 成比例，求 $x$。
2. 求 $3,12,5$ 的第四比例项 $x$。
3. $7:x=3:5$，求 $x$。
4. $\dfrac{x+2}{5}=\dfrac{3}{4}$，求 $x$。

每小问必须先判断：能不能用倍数/份数直接解；如果不能，再交叉相乘。

## 一、题目场景
- 数学对象：四个数成比例、第四比例项、比例方程、含一次式的比例式。
- 变量/参数：$x$。
- 函数/图形：无。
- 已知条件：两个比相等，或文字给出“成比例”“第四比例项”“比例中项”。
- 要求目标：求未知数，并说明优先方法。

## 二、核心结构
### 2.1 表层信息
- 表面考点：比例方程。
- 题型功能：`basic_skill`。
- 是否值得完整 structural analysis：是；本组不是堆题，而是训练方法选择。
- 一句话问题模式：把文字/分数式翻译成比例式，先判断能否倍数或份数求解；不适合时才交叉相乘。

### 2.2 结构表达
#### 判别条件表
- 必要条件：比例顺序、分子分母位置、未知数位置清楚。
- 充分条件：两个比相等，即可建立比例方程。
- 常见干扰项：不看倍数关系，所有题都机械交叉相乘；或者把第四比例项的顺序写错。
- 最短检查动作：写完比例式后，先看一组对应量是否有整数倍或简单份数。

#### 情景量表
无。

#### 命题网络
- P1（题设）：题面给出四个量成比例、第四比例项、比例中项或分数式相等。
- P2（定义）：$a,b,c,x$ 成比例表示 $a:b=c:x$；$x$ 是 $a,b$ 的正比例中项表示 $a:x=x:b$，即 $x^2=ab$。
- P3（策略）：若两侧有明显整数倍或简单份数，优先倍数/份数法。
- P4（策略）：若比例中含一次式、没有明显倍数或未知数位置不便观察，使用交叉相乘。
- R1：P1 + P2 -> 写出比例式。
- R2：比例式 + P3 -> 倍数/份数求解。
- R3：比例式 + P4 -> 交叉相乘求解。
- 目标：完成“结构识别 -> 方法选择 -> 计算”的闭环。

### 2.3 解题主链
```text
题面文字/分数式 -> 写比例式 -> 判断有无整数倍/简单份数 -> 选择方法 -> 计算 -> 代回检查
```

### 2.4 模型标签
- model_id：proportion_equation_method_choice
- model_name：比例方程解法选择
- configuration：已知比例关系，求未知量。
- 可迁移方向：线段/角份数化、相似矩形对应边比例、面积比链。
- 非同构边界：复杂分式方程、二次方程不是本组训练目标。

## 二点五、知识点/模型锚点
- 建议讲义标题：比例方程：先判断方法，再动手计算
- 知识点/模型名称：成比例、第四比例项、比例中项、倍数法、交叉相乘。
- 核心公式/定理：$a:b=c:d \Leftrightarrow ad=bc$；若能看出对应倍数，可直接保持同倍数。
- 使用条件：比例对象顺序明确，分母不为 $0$。
- 入口信号：“成比例”“第四比例项”“比例中项”“两个比相等”。
- 易混边界：$a,b,c,x$ 成比例的顺序是 $a:b=c:x$，不是任意配对。
- 本题如何体现：四个小问分别训练顺序识别、第四比例项、无整数倍比例、含一次式比例。
- 可作为例题的结构层级：基础工具型。

## 三、关键转化
- 最关键的转化：把文字条件先变成标准比例式。
- 为什么降低计算量：能倍数/份数时直接算；没有合适倍数时再交叉相乘，避免一招走天下。
- 不转化时的低效路径：题面还没读清就列方程，容易把顺序、分母或第四比例项位置写错。

## 四、标准路径骨架
1. 先做什么：写出比例式。
2. 再做什么：判断是否有整数倍或简单份数。
3. 建立什么关系：对应量保持同倍数；不便观察则用交叉乘积相等。
4. 如何求解：能倍数/份数就直接求；不能就交叉相乘。
5. 需要检查什么：比例顺序、分母不为 $0$、答案代回是否成立。

## 四点五、标准完整解与验算
- $2:4=3:x$，$2\to4$ 是 $\times2$，所以 $x=6$。
- 第四比例项：$3:12=5:x$，$3\to12$ 是 $\times4$，所以 $x=20$。
- $7:x=3:5$，无整数倍直接对应，交叉得 $3x=35$，所以 $x=\dfrac{35}{3}$。
- $\dfrac{x+2}{5}=\dfrac{3}{4}$，交叉得 $4(x+2)=15$，所以 $x=\dfrac{7}{4}$。
- 排除值：作分母的量不为 $0$。
- 验算：代回原比例式。
- 本题最短可靠路径：先判断方法，再计算。

## 五、出题人逻辑
- 诱导学生硬算的位置：看到比例式就不加判断地交叉相乘。
- 真正的捷径：先看是否有整数倍/简单份数。
- 训练的可迁移能力：后续 B/C/D 的比例计算都要先调用这个工具。

## 六、学生卡点预测
- 读题/入手动作卡点：不知道第四比例项的顺序。
- 建模/关系入口卡点：不能区分“可倍数”和“该交叉相乘”。
- 求解/检查卡点：含一次式时漏括号，或比例中项漏掉“正数”限制。

## 七、变式原则
- 核心不变量：两个比相等。
- 表层特征：文字成比例、第四比例项、比例中项、分数式、冒号比例式。
- 可变维度：未知数位置、是否有整数倍、是否含一次式。
- 深化阶梯：明显整数倍 -> 第四比例项 -> 无整数倍 -> 含一次式 -> 比例中项辨析。
- 允许的变换：小整数、简单分数、一次式。
- 禁止的变换：复杂通分、二次方程求根训练。
- 表征切换：文字、冒号、分数式。
- 包装方式：第四比例项、正比例中项。
- 近迁移例子：B 组中 $4AB=3BC$ 转比例后求份数。
- 远迁移例子：C 组相似矩形中判断对应边后建立比例式。
- 反例/伪变式：无比例关系的等式求值。

## 八、计算复杂度预算
- 原题计算层级：小整数、简单分数、一次式。
- 允许小步上升到：正比例中项的平方关系。
- 禁止引入的计算负担：复杂根式、二次方程训练。
- 必须保留的可见支架：比例式、方法选择、选择理由。

## 九、推荐讲题任务包
- 建议的本轮教学入口：一个大题四个小问，逐问比较方法选择。
- 本题讲解目标：知道何时用倍数/份数，何时交叉相乘。
- 不要直接讲的抽象话：不要只说“用比例性质”。
- 必须先问的问题：这一题有没有明显整数倍？未知数在什么位置？
- 关键讲解顺序：倍数法 -> 第四比例项 -> 无整数倍交叉相乘 -> 含一次式交叉相乘。
- 最适合的具体数值例子：$2:4=3:x$ 与 $7:x=3:5$ 对比。
- 讲到哪里停下来让学生回答：写出比例式后，让学生先判断方法。

## 十、推荐练题任务包
- 若卡在读题/入手动作，出什么题：成比例与第四比例项混排。
- 若卡在建模或关系入口，出什么题：比例中项、分数式、冒号式混排。
- 若卡在求解和检查，出什么题：无整数倍与含一次式的交叉相乘。
- 若原题已稳，如何小步迁移：放进 B 组线段/角条件中。
- 若结构识别已稳，如何深化/抽象/包装：给出文字定义而不直接写比例式。
- 禁止出的跑偏变式：独立两未知量表达题作为本轮主训练。
