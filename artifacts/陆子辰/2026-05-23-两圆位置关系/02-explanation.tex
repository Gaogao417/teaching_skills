\begin{eduProblemBlock}[题目]
已知大圆半径为 $R$，小圆半径为 $r$（即 $R \ge r$），两圆圆心距为 $d$。
\begin{enumerate}[label=(\arabic*)]
  \item \textbf{【已知半径与圆心距求位置关系】} 已知 $\odot O_1$ 的半径为 $3\text{cm}$，$\odot O_2$ 的半径为 $4\text{cm}$。
  \begin{enumerate}[label=\arabic*.]
    \item 若圆心距 $O_1O_2 = 8\text{cm}$，求两圆的位置关系；
    \item 若圆心距 $O_1O_2 = 5\text{cm}$，求两圆的位置关系。
  \end{enumerate}
  \item \textbf{【已知半径与位置关系求圆心距】} 已知两圆的半径分别为 $2$ 和 $5$，若两圆\textbf{相切}，求两圆的圆心距 $d$。
  \item \textbf{【已知单半径、圆心距与位置关系求另一半径】} 已知 $\odot A$ 的半径为 $4$，$\odot A$ 与 $\odot B$ \textbf{相切}，两圆的圆心距为 $3$，求 $\odot B$ 的半径 $r$。
\end{enumerate}

\end{eduProblemBlock}









\begin{eduAuxItem}{思路导航}
算关键边界：和值 $R+r$ 与差值 $R-r$$\;\to\;$将 $d$ 与边界值对比判定位置关系$\;\to\;$相切双解：外切代入 $d=R+r$，内切代入 $d=|R-r|$$\;\to\;$内切大圆讨论：讨论 $r_A > r$ 或 $r > r_A$$\;\to\;$过滤非法解（半径必大于0）并汇总答案\end{eduAuxItem}




\begin{eduAuxItem}{核心思路}
\textbf{数轴边界模型（把位置关系变成区间）}：
我们可以把两圆的圆心距 $d$ 放在一条数轴上。和值 $R+r$ 与差值 $R-r$ 就是两个\textbf{黄金边界点}。
- 边界点 1：$d = R + r$ $\implies$ 外切（相切的第一种情况）
- 边界点 2：$d = R - r$ $\implies$ 内切（相切的第二种情况）
只要题目中提到“相切”，你的第一反应必须是：\textbf{它代表数轴上的这两个孤立分界点}，所以必须写出 $d = R+r$ 或 $d = R-r$ 的分类。
而“相交”则是夹在差与和之间的\textbf{开区间}（$R-r < d < R+r$）；“内含”则是小于差值的区间（$0 \le d < R-r$）。
\end{eduAuxItem}




\begin{eduSubQuestionItem}{(1)}
已知 $\odot O_1$ 的半径为 $3\text{cm}$，$\odot O_2$ 的半径为 $4\text{cm}$。
若圆心距 $O_1O_2 = 8\text{cm}$，求两圆的位置关系；若圆心距 $O_1O_2 = 5\text{cm}$，求两圆的位置关系。
\end{eduSubQuestionItem}

\begin{eduSolutionBox}{解答}
  \begingroup
  \setlength{\parskip}{2.5mm}
由题意，两圆半径分别为 $R = 4\text{cm}, r = 3\text{cm}$，则 $R+r = 7\text{cm}$，$R-r = 1\text{cm}$。\par
① 当圆心距 $d = 8\text{cm}$ 时，因为 $d > R+r$，所以两圆的位置关系是\textbf{外离}。\par
② 当圆心距 $d = 5\text{cm}$ 时，因为 $R-r < d < R+r$，所以两圆的位置关系是\textbf{相交}。\par
  \endgroup
  \vspace{2.5mm}
  {\color{edu-blue}\bfseries\small 核心}\par\vspace{1.5mm}
  \begin{eduBulletList}
    \item 做题时一定要先写出“和”与“差”的数值，它们是判定位置关系的基准边界线。  \end{eduBulletList}
\end{eduSolutionBox}




\begin{eduSubQuestionItem}{(2)}
已知两圆的半径分别为 $2$ 和 $5$，若两圆相切，求两圆的圆心距 $d$。\end{eduSubQuestionItem}

\begin{eduSolutionBox}{解答}
  \begingroup
  \setlength{\parskip}{2.5mm}
因为两圆相切，所以分外切和内切两种几何情况讨论：\par
① 当两圆外切时，圆心距等于两半径之和：$d = R+r = 5+2=7$；\par
② 当两圆内切时，圆心距等于两半径之差：$d = R-r = 5-2=3$。\par
综上所述，两圆的圆心距 $d$ 为 $3$ 或 $7$。\par
  \endgroup
  \vspace{2.5mm}
  {\color{edu-blue}\bfseries\small 关键}\par\vspace{1.5mm}
  \begin{eduBulletList}
    \item 绝不能漏解！在没有明确指明时，“相切”是“外切”和“内切”的合并统称。  \end{eduBulletList}
\end{eduSolutionBox}




\begin{eduSubQuestionItem}{(3)}
已知 $\odot A$ 的半径为 $4$，$\odot A$ 与 $\odot B$ 相切，两圆的圆心距为 $3$，求 $\odot B$ 的半径 $r$。\end{eduSubQuestionItem}

\begin{eduSolutionBox}{解答}
  \begingroup
  \setlength{\parskip}{2.5mm}
设 $\odot B$ 的半径为 $r(r>0)$。因为 $\odot A$ 与 $\odot B$ 相切，分以下两种情况讨论：\par
① 若两圆外切，则圆心距 $d = r_A + r \implies 3 = 4 + r \implies r = -1 < 0$（不合题意，舍去）；\par
② 若两圆内切，根据两圆大半径未知，分以下两种子情况：\par
\quad 子情况 1（$\odot A$ 为大圆，即 $4 > r$）：$d = r_A - r \implies 3 = 4 - r \implies r = 1$；\par
\quad 子情况 2（$\odot B$ 为大圆，即 $r > 4$）：$d = r - r_A \implies 3 = r - 4 \implies r = 7$。\par
综上所述，$\odot B$ 的半径为 $1$ 或 $7$。\par
  \endgroup
  \vspace{2.5mm}
  {\color{edu-blue}\bfseries\small 警示}\par\vspace{1.5mm}
  \begin{eduBulletList}
    \item 即使外切无解，内切仍然有两解！这是中考填空题第18题的超高频陷阱，一定要熟练掌握大小圆分类讨论。  \end{eduBulletList}
\end{eduSolutionBox}

\begin{eduMistakeBox}
\eduSubTitle{漏掉内切情况}很多同学在做“两圆相切”问题时，下意识只在脑海中画出两圆在外面挨着的图像，只写了外切 $d = R+r$，而完全把圆藏在圆里面的内切 $d = R-r$ 忘得一干二净。
\textbf{防坑策略}：只要题目中没有明确指出是“外切”还是“内切”，只要看见\textbf{“相切”}，草稿纸上立刻写下“外切”和“内切”两个大字。
\end{eduMistakeBox}




\begin{eduMistakeBox}
\eduSubTitle{忽略负半径必须舍去}在例题 3 的外切讨论中，我们列出了 $3 = 4+r \implies r=-1$。有的同学会顺手把 $-1$ 写在答案里，或者怀疑自己算错了。
\textbf{防坑策略}：几何图形中，圆的半径 $r$ 是线段长度，\textbf{必须是严格大于 0 的实数}。算出来的负数解一定要勇敢地写上“舍去”。
\end{eduMistakeBox}




\begin{eduMistakeBox}
\eduSubTitle{漏掉内切中的大圆分类}在例题 3 的内切讨论中，很多同学只写了 $3 = 4-r \implies r=1$，就以为完事了。他们默认了已知圆 $\odot A$ 必定是那个大圆。但实际上，新求的圆 $\odot B$ 的半径完全可以是 $7$，把 $\odot A$ 包在里面。
\textbf{防坑策略}：当内切时，只要\textbf{不知道哪一个是已知半径的大圆}，必须列出两个内切公式：$d = r_A - r_B$ 和 $d = r_B - r_A$。
\end{eduMistakeBox}
