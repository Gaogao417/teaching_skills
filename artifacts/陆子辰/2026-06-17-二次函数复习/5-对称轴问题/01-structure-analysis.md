# 结构分析：二次函数对称轴反求（y 相等 ⇒ 对称）

## 原题（本知识点模型题）
抛物线上有四个点 $(1,1),(3,1),(4,-1),(a,-1)$，求 $a$ 的值。

## 一、题目场景
- 数学对象：二次函数（抛物线），具有竖直对称轴。
- 变量/参数：点的横坐标 $x_1,x_2,a$；参数 $a,b,c,h$。
- 函数/图形：$y=f(x)$，对称轴 $x=h$。
- 已知条件：抛物线上存在两组 $y$ 值分别相等的点。
- 要求目标：反求对称轴 $h$；反求未知横坐标；结合一般式反求 $b$ 与 $a$ 的关系。

## 二、核心结构
### 2.1 表层信息
- 表面考点：二次函数的对称性。
- 题型功能：`basic_skill`（两等点求轴）/ `composite_problem`（四点分组 + 结合一般式）。
- 是否值得完整 structural analysis：是；理由：陆子辰对称性识别弱、四点题容易乱，需锁定"分组—等值—对称"的动作链。
- 一句话问题模式：**抛物线上 $y$ 相等的两点关于对称轴对称，故对称轴 = 两点横坐标的平均数**。

### 2.2 结构表达
#### 判别条件表
- 必要条件：两点在同一抛物线上、且 $y$ 值相等。
- 充分条件：上述条件 + $x_1\ne x_2$ ⇒ 该两点关于对称轴对称。
- 常见干扰项：误以为 $y$ 相等⇒两点重合；误把 $\frac{x_1+x_2}{2}$ 算成 $\frac{|x_1-x_2|}{2}$。
- 最短检查动作：找两个 $y$ 相等的点 → 取横坐标平均数。

#### 命题网络
- P1（定理）：抛物线 $y=f(x)$ 有竖直对称轴 $x=h$，且 $f(h+t)=f(h-t)$。
- P2（题设）：$(x_1,y_0),(x_2,y_0)$ 在抛物线上，$x_1\ne x_2$。
- P3（可推结论）：$x_1=h-t,\ x_2=h+t$，即 $h=\dfrac{x_1+x_2}{2}$。
- R1：P1 + P2 -> P3，方法：对称性等距（设 $t=\frac{|x_2-x_1|}{2}$）。
- P4（定义）：一般式 $y=ax^2+bx+c$ 的对称轴 $h=-\dfrac{b}{2a}$。
- R2：P3 + P4 -> P5，方法：令 $-\dfrac{b}{2a}=\dfrac{x_1+x_2}{2}$，得 $b=-a(x_1+x_2)=-2ah$。
- 目标：由 R1 得 $h$；由 R2 得 $b$ 与 $a$ 的关系；四点题先分组再各用一次 R1。

### 2.3 解题主链
```text
两等点 (x1,y0),(x2,y0) -> h=(x1+x2)/2  -> [四点题：对另一组等值点再列 (xa+xb)/2=h 解未知坐标]
                                            -> [一般式题：-b/(2a)=h -> b=-2ah]
```

### 2.4 模型标签
- model_id：`quad-symmetry-from-equal-y`
- model_name：二次函数 y 相等反求对称轴
- configuration：已知两组等 $y$ 点的横坐标（其中含未知），求对称轴 / 未知坐标 / $b,a$ 关系。
- 可迁移方向：顶点式 $f(x)=a(x-h)^2+k$ 中 $f(h+t)=f(h-t)$；绝对值函数 $y=|x-h|$ 同构。
- 非同构边界：一次函数无对称轴；两点 $y$ 相等但不在抛物线上则结论失效。

## 二点五、知识点/模型锚点
- 建议讲义标题：对称轴反求：y 相等的两点关于对称轴对称。
- 知识点/模型名称：二次函数对称性反求（equal-y ⇒ symmetry）。
- 核心公式/定理：$f(x_1)=f(x_2),\ x_1\ne x_2 \Rightarrow h=\dfrac{x_1+x_2}{2}$；一般式 $h=-\dfrac{b}{2a}\Rightarrow b=-2ah$。
- 使用条件：两点必须同在一条二次函数图像上，且 $y$ 值相等、横坐标不等。
- 入口信号：题面出现"$f(m)=f(n)$""两点纵坐标相同""$(x_1,y_0)$ 与 $(x_2,y_0)$"。
- 易混边界：$y$ 相等 ≠ 两点重合；不要把平均数算成差或半差；四点题必须先分组。
- 本题如何体现：$(1,1),(3,1)$ 一组定轴 $x=2$，$(4,-1),(a,-1)$ 一组用轴反求 $a$。
- 可作为例题的结构层级：基础（两等点求轴）→ 必考（四点求 $a$）→ 复合（结合一般式求 $b$）。

## 三、关键转化
- 最关键的转化：把"$y$ 相等"翻译成"关于对称轴对称"。
- 为什么降低计算量：免去求解析式、免去联立方程组，一步平均数得轴。
- 不转化时的低效路径：四点全部代入 $y=ax^2+bx+c$ 列四个方程硬解 $a,b,c,a$ —— 计算爆炸且易错。

## 四、标准路径骨架
1. 先做什么：扫描题面，找出所有 $y$ 值相等的点对。
2. 再做什么：把等值点对分组，每组两点 $(x_i,y_0),(x_j,y_0)$。
3. 建立什么关系：$h=\dfrac{x_i+x_j}{2}$。
4. 如何求解：含已知横坐标的组先定 $h$；含未知坐标的组再代入 $h$ 反解。
5. 需要检查什么：$x_i\ne x_j$；求出的未知点不与已给点重合导致退化。

## 四点五、标准完整解与验算（必考四点题）
- 关键量：$h$（对称轴），$a$（未知横坐标）。
- 完整求解过程：
  - $(1,1),(3,1)$：$y$ 相等 ⇒ $h=\dfrac{1+3}{2}=2$。
  - $(4,-1),(a,-1)$：$y$ 相等 ⇒ 两点关于 $x=2$ 对称 ⇒ $\dfrac{4+a}{2}=2$。
  - 解得 $4+a=4$ ⇒ $a=0$。
- 最终答案：$a=0$。
- 排除值：无（线性方程唯一解）。
- 退化情形：若 $a=4$ 则两点重合，与"$x_1\ne x_2$"矛盾，舍去（本题不触发）。
- 验算：令 $f(x)=p(x-2)^2+q$，由 $(1,1)$ 得 $p+q=1$，由 $(4,-1)$ 得 $4p+q=-1$，解得 $p=-\tfrac23,q=\tfrac53$，$f(0)=-\tfrac23\cdot4+\tfrac53=-1$ ✓。
- 本题最短可靠路径：分组 → 平均数定轴 → 平均数反求 $a$，全程不解析式。

## 五、出题人逻辑
- 诱导学生硬算的位置：四个点都给出，诱导列方程组求 $a,b,c$。
- 真正的捷径：识别两组等 $y$，两次平均数搞定。
- 训练的可迁移能力：对称性识别 + 分组意识。

## 六、学生卡点预测
- 读题/入手动作卡点：陆子辰看到四个点不知道要先"找 $y$ 相等的对"。
- 建模/关系入口卡点：不会把"$y$ 相等"翻译成"关于对称轴对称"。
- 求解/检查卡点：四点题忘记分组，把四个点混在一起；把 $\frac{x_1+x_2}{2}$ 算成 $\frac{x_2-x_1}{2}$。

## 七、变式原则
- 核心不变量：$f(x_1)=f(x_2)\Rightarrow h=\frac{x_1+x_2}{2}$。
- 表层特征：给点、给解析式、给顶点式。
- 可变维度：点数（2→4→含参）、是否结合一般式求 $b$、是否要求画图验证。
- 深化阶梯：① 两等点求轴 → ② 四点求未知坐标 → ③ 结合一般式求 $b$ 与 $a$ 关系 → ④ 顶点式包装。
- 允许的变换：换数字、换所求（求轴/求点/求 $b$）、换表征（解析式↔表格↔图像）。
- 禁止的变换：把两点放在不同函数上；让 $x_1=x_2$；引入高次。
- 表征切换：文字"$f(1)=f(5)$"↔坐标点 $(1,y_0),(5,y_0)$。
- 包装方式：藏进"图像上两点纵坐标相同"一句话。
- 近迁移例子：$f(2)=f(8)\Rightarrow h=5$。
- 远迁移例子：$y=|x-h|$ 上两点函数值相等同样适用。
- 反例/伪变式：两点 $y$ 相等但在直线 $y=kx+b$ 上（一次函数无对称轴）——非同构。

## 八、计算复杂度预算
- 原题计算层级：一次平均数 + 一次一元一次方程。
- 允许小步上升到：解二元一次方程组（顶点式验算）。
- 禁止引入的计算负担：四元方程组硬解 $a,b,c$；含参二次方程。
- 必须保留的可见支架：分组动作、$\frac{x_1+x_2}{2}$ 公式显式写出。

## 九、推荐讲题任务包
- 建议的本轮教学入口：先讲"$y$ 相等 ⇒ 对称"这条定理本身（例 A），再上四点题。
- 本题讲解目标：让陆子辰形成"见等 $y$ 就取平均数"的条件反射。
- 不要直接讲的抽象话："数形结合思想""对称美"。
- 必须先问的问题：这两个点的 $y$ 一样，说明它们在对称轴的什么位置？
- 关键讲解顺序：定理 → 两点求轴 → 四点分组 → 结合一般式。
- 最适合的具体数值例子：$(1,1),(3,1)$ 这种小整数。
- 讲到哪里停下来让学生回答：四点题分组后，让学生自己说出第二组也要用平均数。

## 十、推荐练题任务包
- 若卡在读题/入手动作，出什么题：只给两个等 $y$ 点求轴（例 A 同型）。
- 若卡在建模或关系入口，出什么题：给三个点其中一对等 $y$，求轴。
- 若卡在求解和检查，出什么题：四点含字母，求字母（例 B 同型换数）。
- 若原题已稳，如何小步迁移：结合一般式 $y=ax^2+bx+c$，给 $f(m)=f(n)$ 求 $b$。
- 若结构识别已稳，如何深化/抽象/包装：顶点式包装 $y=2(x-h)^2+1$。
- 禁止出的跑偏变式：跨函数两点；$x_1=x_2$；引入非二次。
