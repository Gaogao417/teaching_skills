# 结构分析：二次函数图像平移——知二推一（原解析式 / 平移 / 新解析式）

## 原题
本知识点为复习专题，无单一原题。三类"知二推一"题模板如下：
- (A) 已知原解析式（顶点式）+ 平移方式 → 求新解析式。
- (B) 已知原解析式 + 新解析式 → 求平移方式（向哪移几个单位）。
- (C) 已知新解析式 + 平移方式 → 反推原解析式。

## 一、题目场景
- 数学对象：顶点式 $y=a(x-h)^2+k$ 下的抛物线。
- 变量/参数：顶点 $(h,k)$、开口系数 $a$、平移量 $m$（左右）、$n$（上下）。
- 函数/图形：抛物线，顶点随平移移动，形状与开口方向不变。
- 已知条件：原/新解析式中两个 + 平移方式中一个（共给出两要素，求第三要素）。
- 要求目标：解析式、顶点坐标或平移方式。

## 二、核心结构

### 2.1 表层信息
- 表面考点：二次函数图像平移。
- 题型功能：`basic_skill`（配少量 `concept_check`）。
- 是否值得完整 structural analysis：是；理由：左右平移符号方向是高频易错点，需要锁定"只动顶点不动 a"与"左加右减"的命题链。
- 一句话问题模式：平移只移动顶点 $(h,k)$，$a$ 不变；用口诀把平移量翻译成 $x$ 与 $k$ 的代换。

### 2.2 结构表达

#### 判别条件表（概念辨析用）
- 必要条件：解析式已是或可写成顶点式 $y=a(x-h)^2+k$。
- 充分条件：已知顶点 $(h,k)$ 与 $a$，即可唯一写出顶点式。
- 常见干扰项：把"向左 $m$"错写成 $x-m$；平移时去改 $a$；把左右平移的量加到 $k$ 上。
- 最短检查动作：先读顶点 $(h,k)$ 与 $a$，再看平移方向，只对括号内 $x$ 或对 $k$ 做代换。

#### 命题网络
- P1（定义）：顶点式 $y=a(x-h)^2+k$，顶点为 $(h,k)$，开口与形状由 $a$ 决定。
- P2（定理）：图像平移只改变顶点位置，不改变 $a$（开口方向与宽窄不变）。
- P3（方法，左右）：向左 $m$ 个单位 $\Rightarrow$ 把括号内的 $x$ 换成 $x+m$；向右 $m$ 个单位 $\Rightarrow x-m$。口诀"左加右减（对 $x$）"。
- P4（方法，上下）：向上 $n$ 个单位 $\Rightarrow k+n$；向下 $n$ 个单位 $\Rightarrow k-n$。口诀"上加下减（对 $k$）"。
- P5（题设/目标，A 类）：已知 $y_{原}=a(x-h)^2+k$ 与平移 $(m,n)$。
- P6（可推，A 类目标）：$y_{新}=a(x-h+m)^2+(k+n)$ 当向左 $m$、向上 $n$ 时（向右/向下取负号）。
- R1：P1 + P2 + P3 + P4 -> P6，方法：对 $x$ 和 $k$ 分别做平移代换，$a$ 原样抄。
- R2（B 类）：P1 + P2 -> 读出原顶点 $(h_原,k_原)$ 与新顶点 $(h_新,k_新)$，方法：对比顶点坐标差。
- R3（B 类目标）：$\Delta x=h_新-h_原$，$\Delta y=k_新-k_原$；$\Delta x>0$ 向右，$\Delta x<0$ 向左；$\Delta y>0$ 向上，$\Delta y<0$ 向下。
- R4（C 类）：P1 + P2 + P3 + P4 的逆用，方法：由"新 = 原经平移"反推原顶点：$h_原=h_新-m_{左右}$（左移记正），$k_原=k_新-n_{上下}$（上移记正）。
- 目标：根据题型在 A/B/C 三类中分别由 P5、R2、R4 推出答案。

### 2.3 解题主链
```text
读顶点 (h,k) 与 a -> 判断题型（A/B/C）->
  A: 把平移代换代入 x 与 k，a 不动 -> y_新
  B: 对比新旧顶点坐标差 -> 方向与单位数
  C: 由新顶点反向加减平移量 -> 原顶点 -> y_原
-> Answer
```

### 2.4 模型标签
- model_id：`quadratic-vertex-shift-know-two`
- model_name：顶点式抛物线平移·知二推一
- configuration：三要素（原解析式顶点、平移方式、新解析式顶点）知二推一；$a$ 始终不变。
- 可迁移方向：任意顶点式抛物线；可推广到 $y=a|x-h|+k$、圆 $(x-h)^2+(y-k)^2=r^2$ 的平移。
- 非同构边界：一般式 $y=ax^2+bx+c$ 的平移不能直接对 $x$、常数项代换，必须先配方成顶点式——那属于其它知识点。

## 二点五、知识点/模型锚点
- 建议讲义标题：知识点 6：二次函数图像平移——只动顶点，不动 $a$。
- 知识点/模型名称：顶点式抛物线平移·知二推一。
- 核心公式/定理：$y=a(x-h)^2+k$；向左 $m$：$x\to x+m$；向右 $m$：$x\to x-m$；向上 $n$：$k\to k+n$；向下 $n$：$k\to k-n$；$a$ 不变。
- 使用条件：解析式已写成（或可写成）顶点式。
- 入口信号：题面出现"平移/向左/向右/向上/向下/平移几个单位"，且给出或要求顶点式。
- 易混边界：左右平移只改括号内 $x$，且"向左是加"；不要改 $a$；不要把左右量加到 $k$ 上。
- 本题如何体现：通过 A/B/C 三类例题，反复训练"读顶点→平移代换/反推→$a$ 不变"的同一命题链。
- 可作为例题的结构层级：A 类顺推、B 类对比顶点、C 类逆推原顶点。

## 三、关键转化
- 最关键的转化：把"平移方向 + 单位数"翻译成对 $x$ 与 $k$ 的代换（或其逆）。
- 为什么降低计算量：顶点式下平移只动两个参数，不必展开、不必配方、不必联立。
- 不转化时的低效路径：把解析式展开成一般式再做坐标变换，计算量大且易错符号。

## 四、标准路径骨架
1. 先做什么：读出（或设出）顶点 $(h,k)$ 与 $a$。
2. 再做什么：判断题型 A/B/C。
3. 建立什么关系：A 类对 $x$、$k$ 做平移代换；B 类对比新旧顶点差；C 类由新顶点反推原顶点。
4. 如何求解：写出新/原顶点式，$a$ 原样保留。
5. 需要检查什么：左右符号方向（左加右减）；$a$ 是否被动；顶点坐标是否与平移一致。

## 四点五、标准完整解与验算

例题 1（A 类）
- 原式：$y=2(x-3)^2+1$，顶点 $(3,1)$，$a=2$。
- 平移：向左 2、向上 3。
- 求解：$x\to x+2$，$k\to k+3$；$y_{新}=2(x-3+2)^2+(1+3)=2(x-1)^2+4$，顶点 $(1,4)$。
- 验算：$(3,1)$ 左 2、上 3 $\to(3-2,1+3)=(1,4)$，与解析式一致。
- 最终答案：$y=2(x-1)^2+4$。

例题 2（B 类）
- 原：$y=-(x+1)^2+2$，顶点 $(-1,2)$，$a=-1$。
- 新：$y=-(x-3)^2+5$，顶点 $(3,5)$，$a=-1$。
- 求解：$\Delta x=3-(-1)=4>0$ 向右 4；$\Delta y=5-2=3>0$ 向上 3。
- 验算：$(-1,2)$ 右 4、上 3 $\to(3,5)$，与 $y=-(x-3)^2+5$ 一致。
- 最终答案：向右平移 4 个单位，再向上平移 3 个单位。

例题 3（C 类）
- 新式：$y=3(x-2)^2-1$，新顶点 $(2,-1)$，$a=3$。
- 平移（原经此平移得新）：向左 3、向下 2。
- 反推：原顶点 $(h,k)$ 经左 3、下 2 得 $(2,-1)$，故 $h-3=2\Rightarrow h=5$，$k-2=-1\Rightarrow k=1$。
- 原式：$y=3(x-5)^2+1$，$a=3$ 不变。
- 验算：$y=3(x-5)^2+1$ 经左 3（$x-5+3=x-2$）、下 2（$1-2=-1$）$\to y=3(x-2)^2-1$，与新式一致。
- 最终答案：原解析式为 $y=3(x-5)^2+1$。
- 排除值：无。
- 退化情形：无（$a\ne0$ 已隐含）。
- 本题最短可靠路径：始终只动顶点，$a$ 原样抄。

## 五、出题人逻辑
- 诱导学生硬算的位置：把顶点式展开成一般式再平移，或对一般式直接平移后重新配方。
- 真正的捷径：守住顶点式，只对 $x$ 和 $k$ 代换。
- 训练的可迁移能力：把几何变换（平移）翻译成代数代换的能力，以及对"方向↔符号"的稳定映射。

## 六、学生卡点预测
- 读题/入手动作卡点：看到一般式想不到先配方（本知识点不练配方，但需提醒用顶点式）；分不清"已知新求原"的方向。
- 建模/关系入口卡点：左右符号搞反（向左误写成 $x-m$）；误把左右平移量加到 $k$ 上。
- 求解/检查卡点：展开后符号出错；忘记 $a$ 不变；B 类中 $\Delta x$ 正负与方向对应错。

## 七、变式原则
- 核心不变量：平移只动顶点 $(h,k)$，$a$ 不变；三要素知二推一。
- 表层特征：顶点式系数、平移单位数、方向组合。
- 可变维度：$a$ 的正负与大小；$h,k,m,n$ 的正负与数值；题型在 A/B/C 间切换。
- 深化阶梯：A 类同结构换数 → B 类对比顶点 → C 类反推 → 混合两步平移 → 把条件藏进"顶点在某直线上"等包装。
- 允许的变换：换 $a$、换顶点、换单位数、换方向、换问法（A↔B↔C）。
- 禁止的变换：引入一般式配方（跨知识点）；引入求解析式、对称轴方程；改变 $a$（破坏模型不变量）。
- 表征切换：顶点式 ↔ 顶点坐标 ↔ 图像示意。
- 包装方式：给"新顶点在某直线上"反推平移量。
- 近迁移例子：A 类换数为 $y=\tfrac12(x+2)^2-3$ 向右 5、向下 1。
- 远迁移例子：C 类给"新顶点 + 两步平移"反推原顶点式。
- 反例/伪变式：让学生对 $y=x^2+bx+c$ 直接平移而不配方——属跨知识点，本讲不练。

## 八、计算复杂度预算
- 原题计算层级：单步代换 / 两数相减。
- 允许小步上升到：两步平移（左右 + 上下）同时进行；C 类需解两个一次方程。
- 禁止引入的计算负担：展开一般式、重新配方、联立方程组、求对称轴。
- 必须保留的可见支架：顶点 $(h,k)$ 与 $a$ 显式标出；平移代换写成 $x\to x\pm m$、$k\to k\pm n$。

## 九、推荐讲题任务包
- 建议的本轮教学入口：先锁定顶点式三要素（顶点、$a$、平移），再讲口诀"左加右减对 $x$，上加下减对 $k$"。
- 本题讲解目标：让学生能用同一套命题链处理 A/B/C 三类题。
- 不要直接讲的抽象话：不讲"函数图像变换的一般理论"。
- 必须先问的问题：这条抛物线的顶点是谁？$a$ 是几？平移是往哪、几个单位？
- 关键讲解顺序：顶点式要素 → 口诀 → A 类顺推 → B 类对比 → C 类逆推 → 易错收口。
- 最适合的具体数值例子：见例题 1/2/3。
- 讲到哪里停下来让学生回答：每次写出原顶点后，停一下让学生说新顶点；C 类让学生先反推原顶点。

## 十、推荐练题任务包
- 若卡在读题/入手动作，出什么题：给顶点式 + 单方向单步平移，求新解析式（A 类最简）。
- 若卡在建模或关系入口，出什么题：左右平移专项（只左右、不上下），强化"左加右减"。
- 若卡在求解和检查，出什么题：B 类对比顶点，训练 $\Delta x/\Delta y$ 正负与方向对应。
- 若原题已稳，如何小步迁移：C 类反推；混合两步平移。
- 若结构识别已稳，如何深化/抽象/包装：把新顶点藏进"新顶点在 $y=x+2$ 上"等条件。
- 禁止出的跑偏变式：要求对一般式配方后再平移；求对称轴方程；改变 $a$。
