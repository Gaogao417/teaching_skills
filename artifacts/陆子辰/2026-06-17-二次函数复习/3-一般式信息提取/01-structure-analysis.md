# 结构分析：一般式 y=ax²+bx+c 的顶点与对称轴（公式法）

## 原题
本知识点为复习型单题型，无单一原题。讲义目标：已知一般式 $y=ax^2+bx+c$（$a\ne0$），用公式直接求对称轴与顶点。

## 一、题目场景
- 数学对象：一般式写成的二次函数。
- 变量/参数：$a,b,c$。
- 函数/图形：抛物线，对称轴 $x=-\frac{b}{2a}$，顶点 $\left(-\frac{b}{2a},\frac{4ac-b^2}{4a}\right)$。
- 已知条件：解析式是一般式。
- 要求目标：对称轴方程、顶点坐标。

## 二、核心结构
### 2.1 表层信息
- 表面考点：一般式求对称轴、顶点（公式法）。
- 题型功能：`basic_skill`。
- 是否值得完整 structural analysis：是；理由：$-\frac{b}{2a}$ 的负号、代入算 $k$ 的符号是高频卡点，且是一般式信息提取的唯一稳定入口。
- 一句话问题模式：对称轴 $x=-\frac{b}{2a}$；顶点纵坐标把对称轴 $x$ 代回原式，或用 $k=\frac{4ac-b^2}{4a}$。

### 2.2 结构表达
#### 判别条件表
- 必要条件：解析式是 $y=ax^2+bx+c$，$a\ne0$。
- 充分条件：按公式 $h=-\frac{b}{2a}$ 求 $h$，再代入或用公式求 $k$。
- 常见干扰项：$-\frac{b}{2a}$ 漏负号；代入算 $k$ 时漏算或符号错；把对称轴写成 $x=\frac{b}{2a}$。
- 最短检查动作：先写 $h=-\frac{b}{2a}$（带负号），再把 $h$ 代回原式算 $k$。

#### 命题网络
- P1（定义）：一般式 $y=ax^2+bx+c$，$a\ne0$。
- P2（定理）：对称轴 $h=-\dfrac{b}{2a}$（注意负号）。
- P3（方法 A）：顶点纵坐标 $k=$ 把 $x=h$ 代入原式计算。
- P4（方法 B）：顶点纵坐标 $k=\dfrac{4ac-b^2}{4a}$。
- P5（目标）：顶点 $(h,k)$，对称轴 $x=h$。
- R1：P1 + P2 -> $h$，方法：代公式。
- R2：$h$ + P3（或 P4）-> $k$，方法：代入计算或代公式。
- R3：$h,k$ -> P5，方法：写出顶点与对称轴。
- 目标：由 R1、R2、R3 得到对称轴和顶点。

### 2.3 解题主链
```text
y=ax²+bx+c -> h=-b/(2a) -> 把 h 代入算 k（或用 k=(4ac-b²)/(4a)）-> 顶点(h,k)、对称轴 x=h
```

## 二点五、知识点/模型锚点
- 建议讲义标题：一般式求对称轴与顶点（公式法）。
- 知识点/模型名称：一般式信息提取（公式法）。
- 核心公式/定理：对称轴 $x=-\frac{b}{2a}$；顶点 $\left(-\frac{b}{2a},\frac{4ac-b^2}{4a}\right)$。
- 使用条件：解析式是一般式，$a\ne0$。
- 入口信号：题目给一般式，要求对称轴或顶点。
- 易混边界：$-\frac{b}{2a}$ 的负号；代入算 $k$ 时符号；把 $b$ 的符号带入时容易错。
- 本题如何体现：三例覆盖 $a=1/a\ne1$、$b$ 正/负、顶点纵坐标正/负。
- 可作为例题的结构层级：basic_skill。

## 三、关键转化
- 最关键的转化：把"求顶点"转化为"先求对称轴 $x=h$，再代入求 $k$"。
- 为什么降低计算量：避免了配方的多步变形，一步公式得 $h$，一次代入得 $k$。
- 不转化时的低效路径：每次都配方（知识点 1），步骤多、易错；或只背 $k$ 公式但记错符号。

## 四、标准路径骨架
1. 先做什么：写对称轴公式 $x=-\frac{b}{2a}$（带负号）。
2. 再做什么：代入 $a,b$ 算出 $h$。
3. 建立什么关系：把 $x=h$ 代入原式算 $k$（或用 $k=\frac{4ac-b^2}{4a}$）。
4. 如何求解：写出对称轴 $x=h$ 和顶点 $(h,k)$。
5. 需要检查什么：$-\frac{b}{2a}$ 负号；$k$ 的符号；$a,b$ 代入时符号。

## 四点五、标准完整解与验算
例题 1：$y=x^2-4x+3$
- 对称轴：$h=-\dfrac{-4}{2\cdot1}=2$，对称轴 $x=2$。
- 顶点纵坐标：代入 $x=2$，$k=2^2-4\cdot2+3=4-8+3=-1$。
- 顶点 $(2,-1)$。用公式验：$k=\dfrac{4\cdot1\cdot3-(-4)^2}{4\cdot1}=\dfrac{12-16}{4}=-1$ ✓。

例题 2：$y=2x^2+8x+5$
- 对称轴：$h=-\dfrac{8}{2\cdot2}=-2$，对称轴 $x=-2$。
- 顶点纵坐标：代入 $x=-2$，$k=2\cdot(-2)^2+8\cdot(-2)+5=8-16+5=-3$。
- 顶点 $(-2,-3)$。用公式验：$k=\dfrac{4\cdot2\cdot5-8^2}{4\cdot2}=\dfrac{40-64}{8}=-3$ ✓。

例题 3：$y=-x^2+6x-7$
- 对称轴：$h=-\dfrac{6}{2\cdot(-1)}=3$，对称轴 $x=3$。
- 顶点纵坐标：代入 $x=3$，$k=-3^2+6\cdot3-7=-9+18-7=2$。
- 顶点 $(3,2)$。用公式验：$k=\dfrac{4\cdot(-1)\cdot(-7)-6^2}{4\cdot(-1)}=\dfrac{28-36}{-4}=2$ ✓。
- 排除值：$a\ne0$。
- 退化情形：无。
- 本题最短可靠路径：$h=-\frac{b}{2a}$ → 代入算 $k$。

## 五、出题人逻辑
- 诱导学生硬算的位置：配方（多步）或死记 $k=\frac{4ac-b^2}{4a}$ 但符号记反。
- 真正的捷径：$h=-\frac{b}{2a}$ 一步得轴，代入算 $k$ 比记 $k$ 公式更稳。
- 训练的可迁移能力：公式代入的符号稳定性；对称轴与顶点的桥梁。

## 六、学生卡点预测
- 读题/入手动作卡点：陆子辰记不住对称轴公式带负号。
- 建模/关系入口卡点：$b$ 为负时 $-\frac{b}{2a}$ 双重负号处理错；$a$ 为负时易错。
- 求解/检查卡点：代入算 $k$ 时漏负号或算错；只写对称轴漏写顶点。

## 七、变式原则
- 核心不变量：$h=-\frac{b}{2a}$，顶点 $(h, f(h))$。
- 表层特征：$a,b,c$ 的正负与数值。
- 可变维度：$a$ 的正负与是否为 1、$b$ 的符号。
- 深化阶梯：$a=1,b$ 为负 → $a\ne1$ → $a$ 为负 → 系数含分数。
- 允许的变换：换 $a,b,c$ 数值。
- 禁止的变换：配方（知识点 1）、顶点式（知识点 2）、求解析式（知识点 4）。
- 表征切换：一般式 ↔ 对称轴方程 + 顶点坐标。
- 近迁移例子：$y=x^2+6x+5$。
- 远迁移例子：$y=-\frac12 x^2+3x-1$。
- 反例/伪变式：给顶点式求顶点（知识点 2）。

## 八、计算复杂度预算
- 原题计算层级：一次除法 + 一次代入（含平方和乘法）。
- 允许小步上升到：$a$ 为负、$b$ 为负的双重符号；系数含简单分数。
- 禁止引入的计算负担：配方多步、联立方程。
- 必须保留的可见支架：$-\frac{b}{2a}$ 显式带负号写出；代入过程显式列式。
