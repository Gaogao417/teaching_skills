# 结构分析：顶点式 y=a(x-h)²+k 的顶点与对称轴信息提取

## 原题
本知识点为复习型单题型，无单一原题。讲义目标：已知二次函数顶点式
$y=a(x-h)^2+k$（$a\ne0$），直接读出顶点 $(h,k)$、对称轴 $x=h$、开口方向。

## 一、题目场景
- 数学对象：顶点式写成的二次函数。
- 变量/参数：$a,h,k$，其中 $a\ne0$。
- 函数/图形：抛物线，一条对称轴 $x=h$，一个顶点 $(h,k)$。
- 已知条件：解析式已经是 $y=a(x-h)^2+k$ 形式。
- 要求目标：读出顶点坐标、对称轴方程、开口方向。

## 二、核心结构
### 2.1 表层信息
- 表面考点：顶点式信息直接读取。
- 题型功能：`concept_check` / `basic_skill`（识图式读取）。
- 是否值得完整 structural analysis：是（单题型复习单元，符号判别有稳定卡点）。
- 一句话问题模式：括号里是 $x-h$，对称轴就是 $x=h$；顶点就是 $(h,k)$；开口由 $a$ 正负决定。

### 2.2 结构表达
#### 判别条件表
- 必要条件：解析式必须是 $a(x-h)^2+k$ 形式，即平方项系数先化成 1（$a$ 提到括号外）。
- 充分条件：括号内是 $x-h$（$h$ 可正可负），外面加常数 $k$。
- 常见干扰项：$a$ 未提出（如 $y=2x^2+...$）；括号内写成 $x+h$ 误判符号；平方项里含系数（如 $y=(2x-3)^2$ 非本知识点标准顶点式）。
- 最短检查动作：先确认括号内是 $x$ 减去一个数（不是 $2x$、不是 $x^2$）；再把"减去的数"当 $h$。

#### 命题网络
- P1（定义）：顶点式 $y=a(x-h)^2+k$，$a\ne0$，括号内变量项系数为 1。
- P2（定义）：对称轴是过顶点且与 $y$ 轴平行的直线。
- P3（定义）：开口方向由 $a$ 正负决定。
- P4（目标/可推结论）：顶点 $(h,k)$。
- P5（目标/可推结论）：对称轴 $x=h$。
- P6（目标/可推结论）：$a>0$ 开口向上，$a<0$ 开口向下。
- R1：P1 + 符号规则 -> P4，方法：括号内写成 $x-h$，取 $h$ 为顶点横坐标，$k$ 为纵坐标。
- R2：P1 + P2 -> P5，方法：对称轴方程直接取顶点横坐标 $x=h$。
- R3：P1 -> P6，方法：看 $a$ 的正负。
- 目标：由 P1 经 R1/R2/R3 推出 P4、P5、P6。

### 2.3 解题主链
```text
P1(识别顶点式) -> 把括号写成 x-h -> 顶点(h,k) + 对称轴 x=h + a定开口 -> Answer
```

### 2.4 模型标签
- model_id：`vertex-form-readout`
- model_name：顶点式信息提取
- configuration：已知 $a,h,k$，要求顶点、对称轴、开口方向。
- 可迁移方向：一般式转顶点式（配方法）、平移、求解析式（其它知识点）。
- 非同构边界：括号内变量系数不是 1（如 $(2x-3)^2$）不是本知识点标准形式；一般式 $ax^2+bx+c$ 不在此列。

## 二点五、知识点/模型锚点
- 建议讲义标题：顶点式 $y=a(x-h)^2+k$ 的顶点、对称轴与开口方向
- 知识点/模型名称：顶点式信息提取
- 核心公式/定理：$y=a(x-h)^2+k$，顶点 $(h,k)$，对称轴 $x=h$，$a>0$ 向上 / $a<0$ 向下。
- 使用条件：解析式已经是顶点式，括号内变量项系数为 1。
- 入口信号：看到 $y=a(x\ \Box\ h)^2+k$ 的结构（$\Box$ 为 $+$ 或 $-$）。
- 易混边界：括号内 $x-h$ 与对称轴 $x=h$ 的符号关系；顶点横坐标是 $h$ 不是 $-h$。
- 本题如何体现：三个例题分别覆盖 $h,k$ 正负与 $a$ 正负的全部组合。
- 可作为例题的结构层级：concept_check / basic_skill。

## 三、关键转化
- 最关键的转化：把括号内的 $x\pm n$ 统一认成 $x-h$（即 $x+3=x-(-3)$）。
- 为什么降低计算量：顶点式无需配方或求导，所有几何信息直接读出。
- 不转化时的低效路径：展开成一般式再用 $x=-\frac{b}{2a}$ 求对称轴，计算量大且易错。

## 四、标准路径骨架
1. 先做什么：确认括号内是 $x-h$ 形式（变量系数为 1）。
2. 再做什么：读出 $h$（注意 $x+n$ 时 $h=-n$）和 $k$。
3. 建立什么关系：顶点 $(h,k)$，对称轴 $x=h$。
4. 如何求解：直接写出顶点、对称轴、开口方向。
5. 需要检查什么：$x+n$ 型不要把对称轴写成 $x=n$；顶点横坐标不要写成 $-h$。

## 四点五、标准完整解与验算
- 例题1 $y=2(x-3)^2+1$：顶点 $(3,1)$，对称轴 $x=3$，开口向上。验算 $x=3$：$2\cdot0+1=1=k$ ✓
- 例题2 $y=-(x+2)^2-4$：即 $y=-(x-(-2))^2-4$，顶点 $(-2,-4)$，对称轴 $x=-2$，开口向下。验算 $x=-2$：$-0-4=-4=k$ ✓
- 例题3 $y=-\dfrac12(x+3)^2+5$：顶点 $(-3,5)$，对称轴 $x=-3$，开口向下。验算 $x=-3$：$-\dfrac12\cdot0+5=5=k$ ✓
- 排除值：$a\ne0$。
- 退化情形：无。
- 本题最短可靠路径：认括号 → 读 $h,k$ → 写顶点和对称轴 → 看 $a$ 正负定开口。

## 五、出题人逻辑
- 诱导学生硬算的位置：把顶点式展开成一般式再用 $x=-\frac{b}{2a}$。
- 真正的捷径：直接读 $h,k$。
- 训练的可迁移能力：从结构（而不是计算）中读信息；符号规则的一致性。

## 六、学生卡点预测
- 读题/入手动作卡点：括号内是 $x+h$ 时不知道要变成 $x-(-h)$。
- 建模/关系入口卡点：把对称轴 $x=h$ 的 $h$ 与顶点横坐标 $h$ 搞混符号（陆子辰具体弱项：$x+3$ 写成 $x=3$）。
- 求解/检查卡点：顶点横坐标写成 $-h$；开口方向由 $h$ 而不是 $a$ 决定。

## 七、变式原则
- 核心不变量：括号内 $x-h$ 与对称轴 $x=h$、顶点 $(h,k)$ 的对应关系。
- 表层特征：$a,h,k$ 的正负、绝对值大小。
- 可变维度：$a$ 的正负与大小、$h$ 的正负、$k$ 的正负、括号符号（$-$ 或 $+$）。
- 深化阶梯：同结构换数 → 括号由 $-$ 换 $+$ → 给顶点和对称轴反写顶点式。
- 允许的变换：换 $a,h,k$ 的数值；括号 $x-h$ 与 $x+h$ 切换。
- 禁止的变换：引入配方法、一般式、平移、求解析式（其它知识点）。
- 表征切换：解析式 ↔ 图像（顶点位置 + 对称轴 + 开口）。
- 近迁移例子：$y=3(x-1)^2+2$。
- 远迁移例子：$y=-2(x+5)^2-7$。
- 反例/伪变式：$y=(2x-3)^2+1$（括号内变量系数非 1，不属于本知识点标准顶点式）；$y=x^2-6x+5$（一般式，属其它知识点）。

## 八、计算复杂度预算
- 原题计算层级：直接读取，0 步计算。
- 允许小步上升到：把 $x+n$ 改写成 $x-(-n)$ 这一步符号处理。
- 禁止引入的计算负担：配方、展开、求 $-\frac{b}{2a}$。
- 必须保留的可见支架：明确写出"括号里减去的数 = $h$ = 对称轴 = 顶点横坐标"。

## 九、推荐讲题任务包
- 建议的本轮教学入口：先认结构 $y=a(x-h)^2+k$，再读 $h,k$，再判 $a$。
- 本题讲解目标：让学生看到顶点式能 3 秒读出顶点、对称轴、开口。
- 不要直接讲的抽象话："数形结合""对称性"。
- 必须先问的问题：括号里是 $x$ 加一个数还是减一个数？

- 最适合的具体数值例子：例题1 $h,k$ 都正打底；例题2/3 专攻 $x+n$ 的负号。
- 讲到哪里停下来让学生回答：给 $y=-(x+4)^2+2$，让学生说顶点和对称轴。
