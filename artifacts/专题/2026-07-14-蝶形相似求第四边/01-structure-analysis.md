# 结构分析：蝶形相似求第四边

## 原题
如图，直线 $AB$ 与 $CD$ 相交于点 $O$，点序分别为 $A-O-B$、$C-O-D$，且 $\angle OAC=\angle ODB$，明确 $AC\nparallel BD$。已知 $AO=6$，$OC=8$，$OD=9$，求 $OB$。

## 一、题目场景
- 数学对象：交叉构型中的 $\triangle AOC$ 与 $\triangle DOB$。
- 变量/参数：未知边长 $OB$。
- 函数/图形：两条直线交于点 $O$，外观接近“8 字形”的蝶形。
- 已知条件：$A,O,B$ 共线且 $O$ 在线段 $AB$ 上；$C,O,D$ 共线且 $O$ 在线段 $CD$ 上；$\angle OAC=\angle ODB$；$AC\nparallel BD$；$AO=6$，$OC=8$，$OD=9$。
- 要求目标：求 $OB$。

本专题的首要训练是形成条件反应：题目一旦给出一组角相等，就立即寻找第二组角并尝试判相似；相似成立后，再按“等角顶点 -> 对应顶点 -> 对应边”识别边，不被“像 8 字形”的外观带偏。

## 二、核心结构
### 2.1 表层信息
- 表面考点：对顶角、两角分别相等判定相似、相似三角形对应边成比例。
- 题型功能：`basic_skill`
- 是否值得完整 structural analysis：是；理由：外形很像常见的“平行线 8 字型”，但本题明确不能借平行线，需要把判定依据和对应顺序重新锁定。
- 一句话问题模式：看到题设等角 -> 到交点寻找对顶角 -> 判定相似 -> 由等角顺序锁定对应边 -> 三边求第四边。

### 2.2 结构表达
#### 判别条件表（概念辨析题用；不适用则写“无”）
- 必要条件：必须有两组分别相等的角；图形外观本身不是相似依据。
- 充分条件：$\angle AOC=\angle DOB$ 且 $\angle OAC=\angle ODB$，由两角分别相等可判定 $\triangle AOC\sim\triangle DOB$。
- 常见干扰项：看到“8 字形”就臆造 $AC\parallel BD$；按视觉位置误写 $A\leftrightarrow B$。
- 最短检查动作：先逐角写出等角，再按角的顶点确定 $A\leftrightarrow D$、$O\leftrightarrow O$、$C\leftrightarrow B$。

#### 情景量表（应用题用；不适用则写“无”）
无。

#### 命题网络（所有题型都写；简单题写简版）
- P1（题设）：$A,O,B$ 共线，$C,O,D$ 共线，且两条直线交于 $O$。
- P2（定理）：$\angle AOC=\angle DOB$，理由是对顶角相等。
- P3（题设）：$\angle OAC=\angle ODB$。
- P4（可推结论）：$\triangle AOC\sim\triangle DOB$，对应关系为 $A\leftrightarrow D$、$O\leftrightarrow O$、$C\leftrightarrow B$。
- P5（可推结论）：$\dfrac{AO}{DO}=\dfrac{OC}{OB}=\dfrac{AC}{DB}$。
- P6（题设）：$AO=6$，$OC=8$，$DO=9$。
- P7（目标）：$OB$ 的长。
- R1：P1 -> P2，方法：对顶角相等。
- R2：P2 + P3 -> P4，方法：两角分别相等判定两三角形相似。
- R3：P4 -> P5，方法：相似三角形对应边成比例。
- R4：P5 + P6 -> P7，方法：代入比例式并解一元一次方程。
- 目标：由 P5、P6 列 $6:9=8:OB$，求出 $OB=12$。

### 2.3 解题主链
```text
看到 ∠OAC=∠ODB -> 寻找第二组角 -> 交点 O 处对顶角相等
-> △AOC∽△DOB -> A↔D，O↔O，C↔B
-> AO↔DO，OC↔OB，AC↔DB -> 6:9=8:OB -> OB=12
```

### 2.4 模型标签
- model_id：`butterfly-similarity-aa-fourth-side`
- model_name：蝶形非平行条件下的 AA 相似求第四边
- configuration：已知交点处对顶角、另一组显式等角和三条对应边，求第四条边。
- 可迁移方向：更换三条已知边；把未知量换到任一组对应边；把等角条件换为另一组不导致平行的显式等角。
- 非同构边界：只给交叉外形而没有第二组等角；把 $AC\parallel BD$ 当作默认条件；第二组角选成会反推平行的 $\angle OAC=\angle OBD$。

## 二点五、知识点/模型锚点
- 建议讲义标题：蝶形相似：先用角锁对应，再列边比
- 知识点/模型名称：对顶角 + 显式等角判定相似；相似三角形三已知边求第四边。
- 核心公式/定理：若两三角形有两组角分别相等，则两三角形相似；本题 $\dfrac{AO}{DO}=\dfrac{OC}{OB}=\dfrac{AC}{DB}$。
- 使用条件：两条直线确实相交于 $O$；另一组等角必须由题设明确给出；对应顺序必须跟随等角顶点。
- 入口信号：“两直线相交”先找对顶角；“另有一组角相等”补足 AA 判定；“三边已知求第四边”转入对应边比例。
- 易混边界：蝶形外观不等于平行线构型；$AC\nparallel BD$ 时不能写同位角或内错角；相似式应写 $\triangle AOC\sim\triangle DOB$，不能机械写成 $\triangle AOC\sim\triangle BOD$。
- 本题如何体现：$O$ 处角由对顶角得到，$A$ 与 $D$ 处角由题设得到，继而锁定 $OC\leftrightarrow OB$。
- 可作为例题的结构层级：单题型基础例题，重点放在判定依据和对应顺序，不增加分段或代数参数。

## 三、关键转化
- 最关键的转化：把“像 8 字”的视觉印象改写成两条可核验的等角，再把角的顶点顺序翻译成对应边。
- 为什么降低计算量：一旦写对 $AO:DO=OC:OB$，只需一次交叉相乘。
- 不转化时的低效路径：凭图猜对应边，或试图寻找不存在的平行线和角关系。

## 四、标准路径骨架
1. 先做什么：由两直线交于 $O$，写 $\angle AOC=\angle DOB$。
2. 再做什么：结合题设 $\angle OAC=\angle ODB$。
3. 建立什么关系：判定 $\triangle AOC\sim\triangle DOB$，写清 $A\leftrightarrow D$、$C\leftrightarrow B$。
4. 如何求解：列 $AO:DO=OC:OB$，代入三条已知边。
5. 需要检查什么：比例两侧是否保持相同对应顺序；全过程是否误用了 $AC\parallel BD$。

## 四点五、标准完整解与验算
- 关键交点/关键量：交点为 $O$，未知量为 $OB$。
- 面积/方程/关系式：$\dfrac{AO}{DO}=\dfrac{OC}{OB}$。
- 完整求解过程：因为直线 $AB$ 与 $CD$ 相交于 $O$，所以 $\angle AOC=\angle DOB$。又因为 $\angle OAC=\angle ODB$，所以 $\triangle AOC\sim\triangle DOB$。由对应边成比例，得 $\dfrac{AO}{DO}=\dfrac{OC}{OB}$，即 $\dfrac69=\dfrac8{OB}$，解得 $OB=12$。
- 最终答案：$OB=12$。
- 排除值：边长必须为正，本题结果满足。
- 退化情形：两直线不能重合；各三角形不能三点共线；交叉角不能为 $0^\circ$ 或 $180^\circ$。
- 验算：$6:9=2:3$，$8:12=2:3$，两组对应边比相等。
- 本题最短可靠路径：对顶角 + 题设等角 -> AA 相似 -> 对应边比例。

## 五、出题人逻辑
- 诱导学生硬算的位置：利用熟悉的“8 字形”外观诱导学生补出平行条件，或把左右位置相近的点直接配对。
- 真正的捷径：用两组角逐顶点锁定相似顺序，再只选含未知边的一条比例式。
- 训练的可迁移能力：区分图形外观与题设事实；从判定条件反推对应关系。

## 六、学生卡点预测
- 读题/入手动作卡点：只看到蝶形，不主动找交点处对顶角。
- 建模/关系入口卡点：把 $A$ 对到 $B$，写成错误相似顺序；或擅自使用平行线性质。
- 求解/检查卡点：比例式上下对应不一致，例如误写 $AO:DO=OB:OC$。

## 七、变式原则
- 核心不变量：对顶角与另一组显式等角共同完成 AA 判定，并由角顶点锁定三组对应边。
- 表层特征：两条直线交叉、两个三角形分居交点两侧，外观接近 8 字形。
- 可变维度：三条已知边的数值、未知边所在的对应组、等角条件的表述位置。
- 深化阶梯：原题复现 -> 同结构换数 -> 未知边换位 -> 把等角写进角度和条件 -> 让学生判断一个给定比例式是否对应正确。
- 允许的变换：保持 $A\leftrightarrow D$、$O\leftrightarrow O$、$C\leftrightarrow B$ 的结构，单独更换数值或所求。
- 禁止的变换：添加 $AC\parallel BD$；用会直接推出 $AC\parallel BD$ 的角条件替代题设等角；同时加入分段和多步方程。
- 表征切换：从图形条件切换为“等角顶点对应表”，再切换为对应边比例。
- 包装方式：把第二组等角写成两个已知角度相等，但必须明确是哪两个角。
- 近迁移例子：已知 $AO=10$、$DO=15$、$OB=18$，求 $OC$。
- 远迁移例子：只给角条件和四个边长选项，判断哪一条比例式正确。
- 反例/伪变式：只有 $A,O,B$ 共线、$C,O,D$ 共线而无第二组等角，不能仅凭对顶角判定相似。

## 八、计算复杂度预算
- 原题计算层级：一个干净分数比例，一次交叉相乘，结果为正整数。
- 允许小步上升到：比例化简后出现一位小数或简单分数；未知边换到比例式任一位置。
- 禁止引入的计算负担：根式边长、复杂分段、二次方程、多重相似链。
- 必须保留的可见支架：两组等角、相似顺序、对应点表或对应边表。

## 九、推荐讲题任务包
- 建议的本轮教学入口：先圈出题设 $\angle OAC=\angle ODB$，追问“已经有一组角相等，交点 $O$ 还能提供哪一组角？”
- 本题讲解目标：学生看到角相等会主动寻找第二组角和相似三角形；判定相似后能按等角顶点找到对应边，而不是靠平行或图形位置猜测。
- 不要直接讲的抽象话：不要只说“蝶形必相似”或“8 字形就这样配”。
- 必须先问的问题：图中哪条性质来自题设，哪条只来自视觉？$AC$ 与 $BD$ 能否当作平行？
- 关键讲解顺序：对顶角 -> 题设等角 -> 对应顶点 -> 相似顺序 -> 对应边比例 -> 计算验算。
- 最适合的具体数值例子：$AO=6$、$OC=8$、$OD=9$，求 $OB=12$。
- 讲到哪里停下来让学生回答：写出两组等角后，让学生独立说出 $A$、$C$ 分别对应哪个点。

## 十、推荐练题任务包
- 若卡在读题/入手动作，出什么题：只要求圈出对顶角和题设等角，不计算。
- 若卡在建模或关系入口，出什么题：给出多个相似顺序，让学生选择并说明对应顶点。
- 若卡在求解和检查，出什么题：保留相似式，只换三条干净边长求第四边。
- 若原题已稳，如何小步迁移：把未知量从 $OB$ 换成 $AO$ 或 $DB$，每次只换一个目标。
- 若结构识别已稳，如何深化/抽象/包装：隐藏“对顶角”字样，只保留交点条件，让学生主动补出第一组等角。
- 禁止出的跑偏变式：依赖平行线判定相似，或同时加入中点、角平分线、面积比等新知识点。
