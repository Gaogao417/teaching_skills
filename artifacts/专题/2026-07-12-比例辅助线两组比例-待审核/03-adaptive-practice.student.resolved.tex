  \begin{minipage}[t]{\dimexpr\linewidth-60mm-6mm\relax}
    \vspace{0pt}
    如图，在 $\triangle ABC$ 中，点 $D$ 在线段 $BC$ 上，点 $E$ 在线段 $AC$ 上，
线段 $AD$\ \ 与 $BE$ 相交于点 $P$。已知
\[
AE:EC=1:2,\qquad AP:PD=3:4,
\]
求 $BP:PE$。

\ \begin{enumerate}[label=(\arabic*)]
  \item 写出两条已知比和所求比分别在哪三条直线上，并指出这三条直线围成的三角形；
\ \  \item 选择这个三角形的一个顶点，在图中作对边的平行线，并写清它与第四条比例线相交于哪一点；
  \item 写出辅助线形成的两组 A/8\ \ 字型相似三角形；
  \item 用两组相似所得的比例求 $BP:PE$。
\end{enumerate}

    \begin{hintbox}
\textbf{提示：}先把“两条已知比＋所求比”落到三条直线上，再看它们围成哪个三角形。\end{hintbox}
\begin{hintbox}
\textbf{提示：}作线后先找两组相似；两组比例要通过同一条中间线段接起来。\end{hintbox}
