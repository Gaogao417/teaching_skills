# 比例辅助线：母图、辅助线与两组相似的绘图指导

## 1. 这次日志暴露的问题

| job | 日志现象 | 结论 |
|---|---|---|
| `auxiliary-student-q1/q2/q3-prompt` | 三幅原题图都在第 1 轮通过 | 原题母图只要使用一致的确定性坐标，`D、E、P` 不独立随机取点，就很稳定 |
| `auxiliary-teacher-q1/q3-solution` | 第 1 轮通过，且 reuse check 确认原始点未漂移 | 解答图必须锁定 prompt 的 `A～P`，只新增 F 和辅助线 |
| `auxiliary-teacher-q2-solution` | 到第 4 轮才通过；四轮几何坐标完全相同，主要在调整 P、F 标签位置 | 这类图的主要风险不是数学退化，而是 F、P、E 靠近时的标签和高亮拥挤 |
| `explanation-example-similar-b/e` | 分别经历多轮视觉修复 | 在小尺寸图中同时高亮多个重叠三角形容易乱；讲解图宜一轮只强调一组相似 |

核心经验：**先由比例确定唯一母图，再由“已知比＋所求比”的三条比例线唯一决定辅助线。不得独立随机取 P，也不得随便让平行线与某条线相交。**

## 2. 唯一母图的坐标构造

设：

```text
AE:EC = r:s
BD:DC = m:n
```

先选一个宽高适中的外三角形：

```text
A = (0, H)
B = (-b, 0)
C = (c, 0)
```

推荐 `6≤H≤8`、`3≤b≤5`、`3≤c≤5`。然后按比例计算：

```text
D = (n B + m C) / (m+n)
E = (s A + r C) / (r+s)
```

P 不能另取，应由 `AD` 与 `BE` 的交点确定。用重心坐标可直接写成：

```text
P = (s m A + n r B + m r C) / (s m + n r + m r)
```

这套构造自动保证：

```text
A-E-C
B-D-C
A-P-D
B-P-E
```

并且：

```text
AP:PD = r(m+n) : s m
BP:PE = m(r+s) : n r
```

所以当题目给的是四个比中的任意两个时，应先在代数层恢复一组一致的 `r:s`、`m:n`，再生成坐标；不要把四个比当作四条互不相关的绘图约束。

## 3. 已验证可用的母图种子

### 3.1 第 1 类

```text
A=(0,6), B=(-3,0), C=(3,0)
D=(1,0), E=(1,4), P=(3/7,24/7)
```

对应：

```text
AE:EC=1:2
BD:DC=2:1
AP:PD=3:4
BP:PE=6:1
```

### 3.2 第 2 类

```text
A=(0,6), B=(-3,0), C=(3,0)
D=(1,0), E=(2,2), P=(3/4,3/2)
```

对应：

```text
AE:EC=2:1
BD:DC=2:1
AP:PD=3:1
BP:PE=3:1
```

这一构型中辅助点 F 会靠近 P，必须预先安排标签偏移。

### 3.3 第 3 类

```text
A=(0,8), B=(-3,0), C=(5,0)
D=(2,0), E=(25/9,32/9), P=(4/3,8/3)
```

对应：

```text
AP:PD=2:1
BP:PE=3:1
BD:DC=5:3
AE:EC=5:4
```

## 4. 四条比例线与四种唯一辅助线

四条比例线固定为：

```text
AC：AE:EC
AD：AP:PD
BE：BP:PE
BC：BD:DC
```

两条已知比和一个所求比所在的三条直线先围成三角形；剩下的一条是“第四条比例线”。辅助线必须由这个三角形和第四条比例线共同决定。

| 三条比例线 | 围成的三角形 | 作法 | 第四条线上的交点 | 两组相似 |
|---|---|---|---|---|
| `AC, AD, BC` | `△ACD` | 过 C 作 `CF∥AD` | `F=CF∩BE` | `△EAP∼△ECF`；`△BDP∼△BCF` |
| `AC, AD, BE` | `△AEP` | 过 A 作 `AF∥EP` | `F=AF∩BC` | `△ACF∼△ECB`；`△ADF∼△PDB` |
| `AC, BE, BC` | `△ECB` | 过 E 作 `EF∥CB` | `F=EF∩AD` | `△AEF∼△ACD`；`△PEF∼△PBD` |
| `AD, BE, BC` | `△PDB` | 过 P 作 `PF∥DB` | `F=PF∩AC` | `△APF∼△ADC`；`△EPF∼△EBC` |

这张表是绘图构造表，不是要求学生背诵的解题表。学生讲解仍按“找三条比例线—围三角形—过顶点作对边平行线—与第四条比例线相交”的现场判断顺序展开。

## 5. F 点必须解析求交，不能随便放

设要过顶点 V 作与边 XY 平行的直线，并与第四条比例线 UW 相交。应求解：

```text
V + t(Y-X) = U + q(W-U)
```

得到的交点才是 F。禁止：

- 先在画面上随便放 F，再声称平行；
- 只让视觉上“差不多平行”；
- 让 F 与错误的比例线相交；
- 为了版面好看而移动 prompt 中的 A～P。

本次成功解答图的 F 为：

```text
过 A 作 AF∥EP，F=AF∩BC：F=(-6,0)
过 E 作 EF∥CB，F=EF∩AD：F=(2/3,2)
过 P 作 PF∥DB，F=PF∩AC：F=(10/3,8/3)
```

F 可能落在原三角形外部，这是正确构型，不应为了“放进三角形”而改错。只需扩大 bounding box，避免裁切。

## 6. prompt 与 solution 的不可变关系

### 6.1 prompt / clean 图

只画：

- `△ABC`；
- `D∈BC`、`E∈AC`；
- `AD`、`BE` 及交点 P；
- A、B、C、D、E、P 标签。

不得画 F、辅助线、平行标记、相似高亮、比例答案或任何提示。

### 6.2 solution / annotated 图

- 必须使用 `reuse_geometry_from` 复用对应 prompt。
- A、B、C、D、E、P 的坐标全部锁定。
- 只新增 F、辅助线和平行标记。
- `solution_reuse_check` 必须显示 `locked_points_same: true`，且 `drift`、`missing` 为空。

可直接使用的约束框架：

```yaml
semantic_constraints:
  given_constraints:
    - reuse the complete original geometry from <prompt-slot-id>
    - V,F line is parallel to X,Y line
    - F lies on the fourth ratio line U,W
    - F is the exact intersection of those two lines
    - show the auxiliary line and one parallel marker
```

## 7. 防退化与防拥挤检查

### 7.1 母图检查

- `△ABC` 的高度至少为底边长度的 `60%`，避免外三角形过扁。
- D、E 都严格在线段内部；若题目比例允许，离端点至少保留对应边长的 `15%`。
- P 严格在 AD、BE 内部，且不要与 A、D、B、E 中任一点几乎重合。
- 四组点序必须在预览中可直接看出，不能只在 JSON 中成立。

### 7.2 新增 F 后的检查

- 先计算 F，再扩展画布；F 在三角形外时不能裁掉。
- 检查 F 与 P、D、E 的屏幕距离。若小于约两个标签字高，必须显式指定标签偏移。
- 标签放在线段两侧：相邻的 E、P、F 不得全部放在同一侧。
- 平行标记放在线段中部，避开交点和标签。

日志中的 `auxiliary-teacher-q2-solution` 四轮使用完全相同的几何坐标，最终只是把 P、F 标签移开才通过。这说明标签位置也应进入绘图请求，而不是留给默认布局碰运气。

## 8. 相似三角形高亮规则

- 原始骨架统一用深色细线；辅助线用紫色或另一种单一强调色。
- 第一组相似可用浅蓝，第二组用浅黄；透明度建议 `0.15～0.30`。
- 高亮多边形不能遮住原始线、点和标签。
- 小尺寸 sidecar 中优先“一张图只高亮一组相似”；讲解步骤图分别展示两组，稳定性高于一张图同时堆四个三角形。
- 若教师答案图必须同时展示两组，使用同色系不同描边，并在 PNG 中人工检查重叠区域。

## 9. 推荐工作流

1. 根据比例先在代数层确定一致的 `r:s`、`m:n`。
2. 用公式生成 A、B、C、D、E、P 的确定性母图。
3. 先生成并审核 prompt；确认点序和标签。
4. 根据三条比例线查明唯一的三角形、顶点、对边和第四条比例线。
5. 解析求交得到 F。
6. solution 复用 prompt，只新增 F 和辅助对象。
7. gate 后查看最终 PNG，特别检查 F 是否被裁切、E/P/F 是否拥挤。
8. 讲解图按两组相似分步高亮；教师最终图再决定是否合并。

## 10. 最终验收清单

- [ ] D、E、P 由同一组比例和交线关系计算，不是独立随机点。
- [ ] A-E-C、A-P-D、B-P-E、B-D-C 四个点序都清楚。
- [ ] 两条已知比和所求比所在的三条线确实围成所声明的三角形。
- [ ] 平行线经过该三角形的指定顶点，并平行于其对边。
- [ ] F 位于第四条比例线上，是解析求得的交点。
- [ ] 两组相似三角形与构造表一致。
- [ ] solution 完整复用 prompt，A～P 无漂移。
- [ ] F 在图外时未被裁切，E/P/F 标签不重叠。
- [ ] prompt 无答案泄露，solution 的高亮不遮挡原始骨架。
