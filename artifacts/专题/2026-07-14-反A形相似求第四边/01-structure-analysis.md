# 结构分析：反 A 形相似三角形求第四边

## 原题
如图，点 $P,A,C$ 依次共线，点 $P,B,D$ 依次共线，且 $\angle PAB=\angle PDC$。已知 $PA=6$，$PB=8$，$PD=9$，求 $PC$。

图形文字描述：从点 $P$ 引出两条射线，一条射线上依次为 $P,A,C$，另一条射线上依次为 $P,B,D$；连接 $AB$、$CD$，形成一小一大的反 A 形构型。

## 一、题目场景
- 数学对象：$\triangle PAB$ 与 $\triangle PDC$。
- 变量/参数：待求边长 $PC$。
- 函数/图形：两条同顶点射线上的一小一大两个三角形。
- 已知条件：$P,A,C$ 依次共线，$P,B,D$ 依次共线，$\angle PAB=\angle PDC$，$PA=6$，$PB=8$，$PD=9$。
- 要求目标：先判定 $\triangle PAB\sim\triangle PDC$，再利用三条已知边求第四条边 $PC$。

本专题的首要训练不是计算，而是建立两个固定反应：看到题目给出一组角相等，立即在两个候选三角形中寻找第二组等角并尝试判相似；相似成立后，按“等角顶点 -> 对应顶点 -> 对应边”写出三组对应边，不凭图形位置猜边。

## 二、核心结构
### 2.1 表层信息
- 表面考点：相似三角形的判定、对应顶点与对应边、相似三角形对应边成比例。
- 题型功能：`basic_skill`
- 是否值得完整 structural analysis：是；理由：计算很短，但“同序写三角形名称”和“对应边不串位”是后续所有相似边长题的基础。
- 一句话问题模式：题目给出角相等 -> 主动寻找第二组等角 -> 判定相似 -> 用角锁定顶点对应 -> 写出对应边 -> 求第四边。

### 2.2 结构表达
#### 判别条件表（概念辨析题用；不适用则写“无”）
无。本题是基础计算题，但需要先完成 AA 判定和对应关系识别。

#### 情景量表（应用题用；不适用则写“无”）
无。

#### 命题网络（所有题型都写；简单题写简版）
- P1（题设）：$P,A,C$ 依次共线，$P,B,D$ 依次共线。
- P2（可推）：$\angle APB=\angle DPC$，因为两角由同两条射线组成。
- P3（题设）：$\angle PAB=\angle PDC$。
- P4（可推结论）：$\triangle PAB\sim\triangle PDC$，对应顶点为 $P\leftrightarrow P$、$A\leftrightarrow D$、$B\leftrightarrow C$。
- P5（定理）：相似三角形的对应边成比例，即 $PA:PD=PB:PC=AB:DC$。
- P6（题设）：$PA=6$，$PB=8$，$PD=9$。
- P7（计算状态）：$6:9=8:PC$。
- P8（目标）：$PC=12$。
- R1：P1 -> P2，方法：由共线点确定同向射线，识别顶点 $P$ 处的公共角。
- R2：P2 + P3 -> P4，方法：两角分别相等判定两三角形相似（AA）。
- R3：P4 + P5 -> 对应边比例，方法：按同序顶点读取对应边。
- R4：对应边比例 + P6 -> P7，方法：选取同时包含三个已知量和一个未知量的两组对应边。
- R5：P7 -> P8，方法：交叉相乘解比例方程。
- 目标：由 P1、P3 判定相似，结合 P6 建立比例并求得 P8。

### 2.3 解题主链
```text
看到 ∠PAB=∠PDC -> 寻找另一组角 -> 由共线找到 ∠APB=∠DPC
-> AA 相似 -> 由等角写 P↔P，A↔D，B↔C
-> 由顶点写 PA↔PD，PB↔PC，AB↔DC -> 6:9=8:PC -> PC=12
```

### 2.4 模型标签
- model_id：similarity_reverse_a_fourth_side
- model_name：反 A 形 AA 相似与三已知求一未知
- configuration：已知共线关系、两角相等以及两组对应边中的三条边，要求第四条边。
- 可迁移方向：换数；改求 $PD$、$PB$ 或另一组对应边；隐藏相似三角形名称后要求学生自行按角定位对应顶点。
- 非同构边界：若题设角不能与顶点 $P$ 处的等角组成 AA，或点的共线顺序改变后角的射线关系不成立，则不能机械套用本模型。

## 二点五、知识点/模型锚点
- 建议讲义标题：反 A 形相似：先对顶点，再用三条已知边求第四边
- 知识点/模型名称：AA 判定与相似三角形对应边比例。
- 核心公式/定理：若 $\triangle PAB\sim\triangle PDC$，且 $P\leftrightarrow P$、$A\leftrightarrow D$、$B\leftrightarrow C$，则
  $$\frac{PA}{PD}=\frac{PB}{PC}=\frac{AB}{DC}.$$
- 使用条件：必须先确认两组角分别相等，并保持三角形名称中的顶点顺序一致。
- 入口信号：两组三点共线形成同顶点的一小一大两个三角形；另给一组跨三角形等角。
- 易混边界：$A$ 对应 $D$，不是按字母位置或图上远近猜对应；因此 $PA$ 对应 $PD$，$PB$ 对应 $PC$。
- 本题如何体现：题设角锁定 $A\leftrightarrow D$，公共顶点锁定 $P\leftrightarrow P$，于是自动得到 $B\leftrightarrow C$。
- 可作为例题的结构层级：原题复现层，支架完整，适合训练“角—顶点—边”的顺序链。

## 三、关键转化
- 最关键的转化：把角相等翻译成顶点对应关系，再由顶点对应关系读取边的对应关系。
- 为什么降低计算量：无需使用勾股定理或求角度，只需选两组对应边列一个比例方程。
- 不转化时的低效路径：直接凭图形位置配边，容易把 $PB$ 错配给 $PD$；或先计算相似比却没有说明分子、分母属于哪个三角形。

## 四、标准路径骨架
1. 先做什么：由两组三点共线指出 $\angle APB=\angle DPC$。
2. 再做什么：结合 $\angle PAB=\angle PDC$，用 AA 判定 $\triangle PAB\sim\triangle PDC$。
3. 建立什么关系：写清 $P\leftrightarrow P$、$A\leftrightarrow D$、$B\leftrightarrow C$，从而得到 $PA:PD=PB:PC$。
4. 如何求解：代入 $6:9=8:PC$，交叉相乘。
5. 需要检查什么：检查比例左右是否都是“小三角形边 : 大三角形对应边”，并用大小关系验算。

## 四点五、标准完整解与验算
- 关键交点/关键量：共同顶点为 $P$；对应顶点为 $P\leftrightarrow P$、$A\leftrightarrow D$、$B\leftrightarrow C$。
- 面积/方程/关系式：$PA:PD=PB:PC$。
- 完整求解过程：因为 $P,A,C$ 依次共线，$P,B,D$ 依次共线，所以 $\angle APB=\angle DPC$。又因为 $\angle PAB=\angle PDC$，所以 $\triangle PAB\sim\triangle PDC$。因此
  $$\frac{PA}{PD}=\frac{PB}{PC},\qquad \frac{6}{9}=\frac{8}{PC}.$$
  交叉相乘得 $6PC=72$，所以 $PC=12$。
- 最终答案：$PC=12$。
- 排除值：边长为正；比例方程分母 $PD$、$PC$ 均不能为 $0$，由非退化三角形自然满足。
- 退化情形：两条射线不能重合，且 $A,B,C,D$ 均不能与 $P$ 重合，否则三角形退化。
- 验算：$PA:PD=6:9=2:3$，$PB:PC=8:12=2:3$；且 $PC=12>PB=8$，符合大三角形对应边较长的图形关系。
- 本题最短可靠路径：两角判相似 -> 写顶点对应 -> 选含未知量的对应边比例 -> 解比例方程。

## 五、出题人逻辑
- 诱导学生硬算的位置：图形像一大一小两个三角形，容易仅凭视觉猜比例或误配“同一条射线上的边”。
- 真正的捷径：题设角先锁定 $A\leftrightarrow D$，公共顶点再锁定 $P\leftrightarrow P$，剩余对应关系无需猜测。
- 训练的可迁移能力：把角的对应转成顶点的对应，再把顶点的对应转成边的对应。

## 六、学生卡点预测
- 读题/入手动作卡点：看见共线只读出点的位置，没有读出顶点 $P$ 处的等角。
- 建模/关系入口卡点：能判定相似，却把三角形写成错误顺序，随后对应边全部错位。
- 求解/检查卡点：比例式一边写“小:大”，另一边写成“大:小”；求出 $PC<8$ 后也未用图形大小关系排错。

## 七、变式原则
- 核心不变量：两组三点共线提供顶点 $P$ 处等角，另一组等角完成 AA；顶点对应始终为 $P\leftrightarrow P$、$A\leftrightarrow D$、$B\leftrightarrow C$。
- 表层特征：反 A 形的一小一大两个三角形，三条边已知、第四条边未知。
- 可变维度：边长数值、未知边位置、比例式的表达形式、是否直接给出相似结论。
- 深化阶梯：原题复现 -> 同结构换数 -> 改求另一条对应边 -> 隐去相似三角形名称让学生自行写同序名称 -> 用文字条件包装共线关系。
- 允许的变换：保持相似比为简单整数比或分数比；每次只改一个未知量位置。
- 禁止的变换：同时引入周长、面积、根式计算或额外辅助线，导致训练重点从“对应关系”漂移。
- 表征切换：可把 $PA:PD=2:3$ 改写为“大三角形边是小三角形对应边的 $1.5$ 倍”。
- 包装方式：将“$P,A,C$ 依次共线”改写为“点 $A$ 在线段 $PC$ 上”，将“$P,B,D$ 依次共线”改写为“点 $B$ 在线段 $PD$ 上”。
- 近迁移例子：已知 $PA=10$、$PD=15$、$PC=18$，求 $PB$。
- 远迁移例子：只给边长表格，让学生先根据角对应填写三组对应边，再选择一组比例计算。
- 反例/伪变式：只保留两组三点共线而删除 $\angle PAB=\angle PDC$；此时通常只有一组等角，不能保证两三角形相似。

## 八、计算复杂度预算
- 原题计算层级：一个两项比例方程，整数交叉相乘。
- 允许小步上升到：相似比为简单分数，或未知量出现在比例式分母中。
- 禁止引入的计算负担：复杂根式、多步分式方程、周长面积混合运算。
- 必须保留的可见支架：先写对应顶点，再写三组对应边，最后圈出含三个已知量和一个未知量的两组边。

## 九、推荐讲题任务包
- 建议的本轮教学入口：先圈出题设给出的 $\angle PAB=\angle PDC$，追问“已经有一组等角，还缺哪一组角才能找相似？”
- 本题讲解目标：形成“看到角相等就找第二组角、看到相似就按角找对应边”的条件反射，稳定完成“题设等角 -> 第二组等角 -> 相似 -> 顶点 -> 边 -> 比例”的动作链。
- 不要直接讲的抽象话：不要只说“对应边成比例”，必须把三组对应边逐组写出。
- 必须先问的问题：$\angle PAB$ 的顶点是谁？与它相等的 $\angle PDC$ 的顶点是谁？
- 关键讲解顺序：识别 $P$ 处等角 -> AA 判相似 -> 写同序三角形名称 -> 列三组对应边 -> 选比例求解 -> 大小验算。
- 最适合的具体数值例子：$PA=6$、$PB=8$、$PD=9$，求 $PC$；相似比 $2:3$，答案 $12$。
- 讲到哪里停下来让学生回答：写出 $P\leftrightarrow P$、$A\leftrightarrow D$ 后停下，让学生补出 $B\leftrightarrow C$ 及两组所需对应边。

## 十、推荐练题任务包
- 若卡在读题/入手动作，出什么题：只要求圈出两组等角并标出角的顶点，不计算。
- 若卡在建模或关系入口，出什么题：给出相似结论，只要求填写三组对应边。
- 若卡在求解和检查，出什么题：保留对应关系表，换一组整洁整数，要求列式并用大小关系验算。
- 若原题已稳，如何小步迁移：把未知量换成 $PD$ 或 $PB$，其余结构不变。
- 若结构识别已稳，如何深化/抽象/包装：用“点在线段上”的文字条件替代直接列出三点共线，并要求学生自行写同序相似式。
- 禁止出的跑偏变式：加入面积比、周长比或需要辅助线的新模型。
