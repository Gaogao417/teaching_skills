# 结构分析：比例辅助线边比

## 原题
在 $\triangle ABC$ 中，点 $D$ 在边 $AB$ 上，点 $E$ 在边 $AC$ 上，直线 $CD$ 与 $BE$ 相交于点 $P$。四条线上分别看分点比例：

- $AB$ 线上：$AD:DB$；
- $AC$ 线上：$AE:EC$；
- $CD$ 线上：$CP:PD$；
- $BE$ 线上：$BP:PE$。

已知其中两条线上的比例，求另一条线上的比例。学生讲解只使用“作平行线、找相似三角形、接比例”的路径，不出现定理名称。

## 一、题目场景
- 数学对象：$\triangle ABC$，$D\in AB$，$E\in AC$，$P=CD\cap BE$。
- 变量/参数：$AD:DB$、$AE:EC$、$CP:PD$、$BP:PE$。
- 函数/图形：无函数；平面几何比例关系。
- 已知条件：四条线中的两条分点比例。
- 要求目标：求剩余线中的一条分点比例。

## 二、核心结构
### 2.1 表层信息
- 表面考点：平行辅助线、相似三角形、对应边比例。
- 题型功能：`proof_model`
- 是否值得完整 structural analysis：是；本题重点不是记公式，而是看出辅助线如何把交叉线转成两组三角形相似。
- 一句话问题模式：在两条边上取点、两条连线相交的构型中，用一条平行辅助线把“未知线上的比例”转成“已知边上的比例”。

### 2.2 结构表达
#### 判别条件表
- 必要条件：$D,A,B$ 共线，$E,A,C$ 共线，$P,C,D$ 共线，$P,B,E$ 共线。
- 充分条件：过 $C$ 作 $CF\parallel AB$，且 $F$ 在 $BE$ 延长线上，则可以读出两组相似三角形。
- 常见干扰项：把 $AD:DB$ 直接当成 $AB:DB$；把 $AE:EC$ 与 $CE:AE$ 写反。
- 最短检查动作：先看目标比例在哪条线，再选包含这条线的三角形。

#### 情景量表
无。

#### 命题网络
- P1（题设）：$D\in AB$，$E\in AC$，$P=CD\cap BE$。
- P2（构造）：若目标为 $CP:PD$，过 $C$ 作 $CF\parallel AB$，交 $BE$ 延长线于 $F$。
- P3（可推）：$\triangle EAB\sim\triangle ECF$，所以 $CF:AB=CE:AE$。
- P4（可推）：$\triangle PDB\sim\triangle PCF$，所以 $CP:PD=CF:DB$。
- P5（可推）：$CP:PD=(AB:DB)\times(CE:AE)$。
- P6（构造）：若目标为 $BP:PE$，可换到以 $BE$ 为目标线的相似链。
- P7（可推）：$BP:PE=(AC:CE)\times(BD:AD)$。
- R1：P1 + P2 -> P3，方法：平行线对应角相等，得相似。
- R2：P1 + P2 -> P4，方法：同一条平行线再构成第二组相似。
- R3：P3 + P4 -> P5，方法：用中间量 $CF$ 接比例。
- 目标：把已知两条线上的比例转成目标线上的比例。

### 2.3 解题主链
```text
目标线 -> 选三角形 -> 过顶点作平行线 -> 找交点 -> 两组相似 -> 接比例 -> 化简
```

### 2.4 模型标签
- model_id：parallel_helper_ratio_bridge
- model_name：平行辅助线接比例
- configuration：$D\in AB$，$E\in AC$，$P=CD\cap BE$。
- 可迁移方向：正向求 $CP:PD$ 或 $BP:PE$，反向求 $AD:DB$ 或 $AE:EC$。
- 非同构边界：若 $P$ 不是 $CD$ 与 $BE$ 的交点，或缺少共线条件，则不能套用本比例链。

## 二点五、知识点/模型锚点
- 建议讲义标题：比例辅助线：用平行线接两组相似
- 知识点/模型名称：平行辅助线接比例
- 核心公式/定理：
  - $CP:PD=(AB:DB)\times(CE:AE)$。
  - $BP:PE=(AC:CE)\times(BD:AD)$。
- 使用条件：题图中必须有 $D\in AB$、$E\in AC$、$P=CD\cap BE$。
- 入口信号：题目问 $CD$ 或 $BE$ 上的分点比例。
- 易混边界：先把边上小段比转成整段比，如 $AD:DB=2:1$ 时，$AB:DB=3:1$。
- 本题如何体现：例题用 $AD:DB=1:1$、$AE:EC=1:2$ 求 $CP:PD$。
- 可作为例题的结构层级：同结构换数。

## 三、关键转化
- 最关键的转化：过 $C$ 作 $CF\parallel AB$，把未知的 $CP:PD$ 转为 $CF:DB$。
- 为什么降低计算量：不求点坐标，不设多段长度，只做比例乘法。
- 不转化时的低效路径：盲目设坐标求交点 $P$。

## 四、标准路径骨架
1. 先做什么：标出已知比例和目标比例。
2. 再做什么：目标在 $CD$ 上，先看 $\triangle ACD$。
3. 建立什么关系：过 $C$ 作 $CF\parallel AB$，找 $F=CF\cap BE$。
4. 如何求解：由两组相似得到 $CF:AB=CE:AE$ 与 $CP:PD=CF:DB$。
5. 需要检查什么：比例顺序是否与题目一致，整段比是否转对。

## 四点五、标准完整解与验算
- 关键交点/关键量：$F=CF\cap BE$，且 $CF\parallel AB$。
- 关系式：
  \[
  CF:AB=CE:AE,\qquad CP:PD=CF:DB.
  \]
- 完整求解过程：若 $AD:DB=1:1$，则 $AB:DB=2:1$；若 $AE:EC=1:2$，则 $CE:AE=2:1$。所以
  \[
  CP:PD=(AB:DB)\times(CE:AE)=2\times2=4:1.
  \]
- 最终答案：$CP:PD=4:1$。
- 排除值：所有线段长度为正。
- 退化情形：$D,E$ 不与端点重合，$P$ 不与端点重合。
- 验算：设 $DB=1$、$AD=1$、$AE=1$、$EC=2$，则 $AB=2$，$CE:AE=2:1$，目标比为 $4:1$。
- 本题最短可靠路径：先写两组相似比例，再接 $CF$。

## 五、出题人逻辑
- 诱导学生硬算的位置：交点 $P$ 看起来复杂，容易想设坐标。
- 真正的捷径：加平行线，把交点比例转成相似三角形比例。
- 训练的可迁移能力：从目标线倒推辅助线和相似三角形。

## 六、学生卡点预测
- 读题/入手动作卡点：不知道 $CD$ 上的比例是 $CP:PD$。
- 建模/关系入口卡点：不知道为什么过 $C$ 作平行线。
- 求解/检查卡点：整段比和分段比混淆，最终比写反。

## 七、变式原则
- 核心不变量：一条平行辅助线接两组相似三角形。
- 表层特征：三角形、边上点、两条连线相交、求分点比例。
- 可变维度：已知哪两条线，求哪一条线。
- 深化阶梯：先正向求 $CP:PD$，再求 $BP:PE$，最后反向求 $AD:DB$ 或 $AE:EC$。
- 允许的变换：整数比例、反向比例方程。
- 禁止的变换：引入复杂坐标或多个未知比例同时求。
- 表征切换：图上标线、公式填空、完整解答。
- 包装方式：题目只说“四条线上的比例”，让学生先翻译为具体小段比。
- 近迁移例子：换 $AD:DB$ 与 $AE:EC$，仍求 $CP:PD$。
- 远迁移例子：给 $CP:PD$ 与 $AE:EC$，反求 $AD:DB$。
- 反例/伪变式：没有交点 $P$ 或不共线的图形。

## 八、计算复杂度预算
- 原题计算层级：一次整段比转换 + 一次比例乘法。
- 允许小步上升到：一个一元比例方程。
- 禁止引入的计算负担：坐标求交点、二次方程、根式。
- 必须保留的可见支架：题图和目标线提示。

## 九、推荐讲题任务包
- 建议的本轮教学入口：直接从五步图进入。
- 本题讲解目标：学生能复述“先选三角形，再作平行线，再用两组相似接比例”。
- 不要直接讲的抽象话：不讲定理名，不先背公式。
- 必须先问的问题：目标比例在哪条线上？
- 关键讲解顺序：标图 -> 作线 -> 找交点 -> 写两组相似 -> 接比例。
- 最适合的具体数值例子：$AD:DB=1:1$、$AE:EC=1:2$。
- 讲到哪里停下来让学生回答：写出第一组相似后，让学生补第二组相似。
