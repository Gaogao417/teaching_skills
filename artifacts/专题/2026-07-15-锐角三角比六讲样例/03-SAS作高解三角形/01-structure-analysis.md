# 结构分析：SAS 先从非已知角顶点作高

## 原题
在三角形 $ABC$ 中，已知 $AB=10$，$AC=13$，夹角 $\angle A=30^\circ$。不使用正弦定理、余弦定理，从点 $B$ 向 $AC$ 作高 $BH$，解这个三角形：求边 $BC$ 以及 $\angle B$、$\angle C$（角度精确到 $0.01^\circ$）。

图形文字描述：$H$ 在线段 $AC$ 上，$BH\perp AC$。辅助线 $BH$ 把原三角形分成 $\operatorname{Rt}\triangle ABH$ 与 $\operatorname{Rt}\triangle BCH$。本题是标准 SAS：已知两边 $AB,AC$ 及其夹角 $\angle A$，三角形唯一确定。

## 一、题目场景
- 数学对象：一般三角形 $ABC$、从顶点 $B$ 向边 $AC$ 所作的高 $BH$、两个直角三角形 $ABH$ 与 $BCH$。
- 变量/参数：已知 $AB=10$、$AC=13$、$\angle A=30^\circ$；待求 $BH,AH,CH,BC,\angle B,\angle C$。
- 函数/图形：一般三角形通过一条高拆成两个直角三角形；在直角三角形中使用锐角三角比与勾股定理。
- 已知条件：两边及其夹角，即标准 SAS 条件。
- 要求目标：只使用作高、锐角三角比、勾股定理和三角形内角和解出其余边角。

## 二、核心结构
### 2.1 表层信息
- 表面考点：用锐角三角比解一般三角形。
- 题型功能：`application_modeling`
- 是否值得完整 structural analysis：是；理由：SAS 的关键不是直接在一般三角形中套公式，而是选对作高顶点，形成“先解一个 Rt 三角形，再解另一个 Rt 三角形”的稳定链条。
- 一句话问题模式：已知两边及夹角时，从不是已知角的一个顶点作高，先用含已知角的直角三角形求高和投影，再用线段差进入第二个直角三角形。

### 2.2 结构表达
#### 判别条件表（概念辨析题用；不适用则写“无”）
| 检查项 | 本题判断 | 作用 |
|---|---|---|
| 是否为标准 SAS | $AB,AC$ 及夹角 $\angle A$ 已知 | 唯一确定三角形，不分类“两解” |
| 从哪个顶点作高 | 从 $B$ 或 $C$ 作高，不能从已知角顶点 $A$ 作高 | 让已知角进入第一个直角三角形 |
| 垂足在线段还是延长线 | $AH=5\sqrt3<13=AC$ | $H$ 在线段 $AC$ 上，故 $CH=AC-AH$ |

- 必要条件：所给角必须是两条已知边的夹角，才是标准 SAS。
- 充分条件：两条正边和一个小于 $180^\circ$ 的夹角唯一确定三角形。
- 常见干扰项：把“已知两边和一个角”一律当成 SAS；从已知角顶点 $A$ 作高；未判断点序就机械写 $CH=AC-AH$；把 SSA 的两解误写到 SAS。
- 最短检查动作：先圈出夹角两边，再从其中一个非夹角顶点向另一条已知边所在直线作高。

#### 情景量表（应用题用；不适用则写“无”）
| 量 | 类型/单位 | 题设关系 | 未知/已知 |
|---|---|---|---|
| $AB$ | 长度 | 第一个直角三角形的斜边 | 已知 $10$ |
| $AC$ | 长度 | 高的落边，也是 $AH+CH$ | 已知 $13$ |
| $\angle A$ | 角度 | $AB$ 与 $AC$ 的夹角 | 已知 $30^\circ$ |
| $BH$ | 长度 | 两个直角三角形共用的高 | 未知 |
| $AH,CH$ | 长度 | $AC=AH+CH$ | 未知 |
| $BC$ | 长度 | 第二个直角三角形的斜边 | 目标 |
| $\angle B,\angle C$ | 角度 | 原三角形其余两角 | 目标 |

#### 命题网络（所有题型都写；简单题写简版）
- P1（题设）：$AB=10$，$AC=13$，$\angle A=30^\circ$，且 $\angle A$ 是两条已知边的夹角。
- P2（构造）：从 $B$ 作 $BH\perp AC$ 于 $H$，得到 $\operatorname{Rt}\triangle ABH$ 与 $\operatorname{Rt}\triangle BCH$。
- P3（可推）：在 $\operatorname{Rt}\triangle ABH$ 中，$BH=AB\sin A=5$，$AH=AB\cos A=5\sqrt3$。
- P4（检查/可推）：$0<5\sqrt3<13$，故 $H$ 在线段 $AC$ 上，$CH=13-5\sqrt3$。
- P5（可推）：在 $\operatorname{Rt}\triangle BCH$ 中，$BC=\sqrt{BH^2+CH^2}=\sqrt{269-130\sqrt3}$。
- P6（可推）：$\tan C=BH/CH=5/(13-5\sqrt3)$，故 $\angle C\approx49.04^\circ$。
- P7（可推）：$\angle B=180^\circ-\angle A-\angle C\approx100.96^\circ$。
- P8（目标）：得到三角形的其余一边与两角，并完成边角合理性检查。
- R1：P1 + P2 -> P3，方法：在第一个直角三角形中用正弦、余弦求高和投影。
- R2：P1 + P3 -> P4，方法：比较投影 $AH$ 与已知边 $AC$，判定垂足位置后作线段加减。
- R3：P2 + P4 -> P5，方法：在第二个直角三角形中用勾股定理求斜边。
- R4：P2 + P4 -> P6，方法：在第二个直角三角形中用正切求锐角 $C$。
- R5：P1 + P6 -> P7，方法：用三角形内角和求最后一个角。
- R6：P5 + P6 + P7 -> P8，方法：边角范围、内角和与大边对大角联合验算。
- 目标：由 P1 经“作高—第一个 Rt—线段差—第二个 Rt—内角和”推出 P8。

### 2.3 解题主链
```text
SAS 两边及夹角
-> 从非已知角顶点 B 作 BH⊥AC
-> Rt△ABH：求 BH、AH
-> 比较 AH 与 AC，得到 CH
-> Rt△BCH：求 BC、∠C
-> 内角和求 ∠B
-> 边角关系验算
```

### 2.4 模型标签
- model_id：triangle_sas_altitude_two_right_triangles
- model_name：SAS 作高后依次解两个直角三角形
- configuration：已知 $AB,AC$ 和夹角 $A$，从非已知角顶点 $B$ 向 $AC$ 作高，要求其余边角。
- 可迁移方向：交换从 $B$ 或 $C$ 作高；改变夹角大小；改变投影与底边的大小关系；在同一构造下改求面积或高。
- 非同构边界：所给角不是两条已知边的夹角时属于 SSA，不得沿用“SAS 唯一解”结论；正弦定理、余弦定理不属于本课允许工具。

## 二点五、知识点/模型锚点
- 建议讲义标题：SAS：从非已知角顶点作高，顺次解两个 Rt 三角形
- 知识点/模型名称：两边及夹角条件下的作高解三角形模型。
- 核心公式/定理：$BH=AB\sin A$，$AH=AB\cos A$；判定点序后 $CH=AC-AH$；$BC^2=BH^2+CH^2$；$\tan C=BH/CH$；$B=180^\circ-A-C$。
- 使用条件：两条已知边共用的顶点处所夹的角已知；高必须从不是该已知角的顶点作出。
- 入口信号：题面给“两边及夹角”，并要求不用正弦定理、余弦定理解一般三角形。
- 易混边界：SAS 唯一确定，不出现两解；SSA 才需要单独讨论无解、一解或两解。线段差必须先由数值或图形条件确认垂足位置。
- 本题如何体现：$30^\circ$ 让第一个直角三角形的高与投影可精确求出，$AC>AH$ 让垂足落在线段内，再顺畅进入第二个直角三角形。
- 可作为例题的结构层级：SAS 标准母题，保留根式但计算链清楚，适合作为后续题库的数字设计基准。

## 三、关键转化
- 最关键的转化：把一般三角形转成两个共享高 $BH$ 的直角三角形，并规定先解含已知夹角的 $\triangle ABH$。
- 为什么降低计算量：第一个直角三角形中有“斜边 + 锐角”，可直接求高和投影；第二个直角三角形随即获得两条直角边。
- 不转化时的低效路径：试图在一般三角形中直接寻找对边、邻边、斜边；或引入本课禁止的正弦定理、余弦定理。

## 四、标准路径骨架
1. 先做什么：辨认标准 SAS，确认 $\angle A$ 是 $AB,AC$ 的夹角；从非已知角顶点 $B$ 作 $BH\perp AC$。
2. 再做什么：在 $\operatorname{Rt}\triangle ABH$ 中用 $\sin A$、$\cos A$ 求 $BH,AH$。
3. 建立什么关系：比较 $AH$ 与 $AC$，确认 $H$ 在线段 $AC$ 上后写 $CH=AC-AH$。
4. 如何求解：在 $\operatorname{Rt}\triangle BCH$ 中用勾股求 $BC$，用 $\tan C$ 求 $\angle C$，再用内角和求 $\angle B$。
5. 需要检查什么：垂足位置、角度范围、三角形内角和以及最大边是否对应最大角。

## 四点五、标准完整解与验算
- 关键交点/关键量：$H$ 是 $B$ 到 $AC$ 的垂足；$BH$ 是两个直角三角形的公共直角边。
- 面积/方程/关系式：$BH=10\sin30^\circ$，$AH=10\cos30^\circ$，$CH=13-AH$，$BC^2=BH^2+CH^2$，$\tan C=BH/CH$。
- 完整求解过程：
  1. 作 $BH\perp AC$ 于 $H$。在 $\operatorname{Rt}\triangle ABH$ 中，$AB=10$ 为斜边，$\angle BAH=30^\circ$。
  2. $BH=10\sin30^\circ=5$，$AH=10\cos30^\circ=5\sqrt3$。
  3. 因为 $0<5\sqrt3<13$，所以 $H$ 在线段 $AC$ 上，$CH=AC-AH=13-5\sqrt3$。
  4. 在 $\operatorname{Rt}\triangle BCH$ 中，
     $$BC=\sqrt{BH^2+CH^2}=\sqrt{25+(13-5\sqrt3)^2}=\sqrt{269-130\sqrt3}\approx6.62.$$
  5. $$\tan C=\frac{BH}{CH}=\frac{5}{13-5\sqrt3},$$
     所以 $\angle C\approx49.04^\circ$。
  6. $\angle B=180^\circ-30^\circ-49.04^\circ\approx100.96^\circ$。
- 最终答案：$BC=\sqrt{269-130\sqrt3}\approx6.62$，$\angle C\approx49.04^\circ$，$\angle B\approx100.96^\circ$。
- 排除值：两条已知边必须为正，夹角满足 $0^\circ<A<180^\circ$；本题均满足。
- 退化情形：若夹角为 $0^\circ$ 或 $180^\circ$，三点共线；不属于三角形。
- 验算：$30^\circ+49.04^\circ+100.96^\circ=180^\circ$；最大边 $AC=13$ 对应最大角 $B\approx100.96^\circ$；$|13-10|<6.62<13+10$。
- 本题最短可靠路径：从 $B$ 作高 -> 用 $AB,\angle A$ 求 $BH,AH$ -> 作线段差求 $CH$ -> 在第二个 Rt 三角形求边角。

## 五、出题人逻辑
- 诱导学生硬算的位置：学生可能盯着一般三角形的三条边，试图直接找“斜边”，或从已知角顶点作高却得不到可解的第一个直角三角形。
- 真正的捷径：让高避开已知角顶点，从而把“已知边 + 已知角”完整保留在第一个直角三角形中。
- 训练的可迁移能力：把一般三角形主动拆成两个直角三角形，并把第一个三角形求出的高和投影传给第二个三角形。

## 六、学生卡点预测
- 读题/入手动作卡点：分不清 SAS 与 SSA；只看到“两边一角”，没有检查角是否为夹角。
- 建模/关系入口卡点：从 $A$ 作高；或作出 $BH$ 后先解 $\triangle BCH$，但这个直角三角形中暂时没有足够已知量。
- 求解/检查卡点：未判断 $H$ 的位置就写 $CH=AC-AH$；把 $AH$ 与 $CH$ 混淆；计算最后一个角后不核对最大边与最大角。

## 七、变式原则
- 核心不变量：标准 SAS 唯一解；高从非已知角顶点作出；先解含已知夹角的直角三角形，再解另一个。
- 表层特征：两条边和一个夹角；辅助高落在其中一条已知边所在直线上。
- 可变维度：边长、夹角、作高方向、垂足在线段内部或延长线、最终所求量。
- 深化阶梯：照搬本题数值 -> 同结构换数 -> 交换从 $B/C$ 作高 -> 改变夹角使垂足位置变化 -> 隐去“作高”提示让学生自主构造。
- 允许的变换：使用 $30^\circ,45^\circ,60^\circ$ 等特殊角；让 $AC$ 与投影形成整洁的和或差；把所求改为面积、高或某个角。
- 禁止的变换：把所给角换成非夹角后仍标成 SAS；引入正弦定理或余弦定理；一次同时增加垂足外置和复杂根式近似两种负担。
- 表征切换：从带图题切换到纯文字 SAS 条件，让学生自行画草图并决定作高顶点。
- 包装方式：用测距、屋架或坡面语境给出两段长度与夹角，但核心仍保持 SAS。
- 近迁移例子：$AB=8,AC=10,\angle A=60^\circ$，从 $B$ 作高，按同一链条求 $BC$ 和其余两角。
- 远迁移例子：只给出同一组 SAS 数据，要求先判断从 $B$、$C$ 两种作高方案中哪一种计算更简洁并说明理由。
- 反例/伪变式：已知 $AB,AC$ 与 $\angle B$；角 $B$ 不是两边的夹角，这是 SSA，可能出现不同解数，不能归入本模型。

## 八、计算复杂度预算
- 原题计算层级：一个特殊角精确三角比、一次根式线段差、一次勾股开方、一次反三角函数近似。
- 允许小步上升到：垂足落在延长线时改用线段和；或加入一个面积/高的附加目标。
- 禁止引入的计算负担：非特殊角的多轮近似传播、繁复分母有理化、正弦定理、余弦定理以及多种作高分类同时出现。
- 必须保留的可见支架：明确标出已知夹角；第一批题给出“从非已知角顶点作高”的提示；要求每次写清正在解哪个 $\operatorname{Rt}\triangle$。

## 九、推荐讲题任务包
- 建议的本轮教学入口：先让学生圈出两条已知边及夹角，再问“如果从 $A$ 作高，第一个直角三角形里还剩哪些已知量？”由此筛掉错误作法。
- 本题讲解目标：形成 SAS 的固定动作链，并明确标准 SAS 唯一解。
- 不要直接讲的抽象话：不说“套解三角形公式”；每一步都落到具体的 $\operatorname{Rt}\triangle ABH$ 或 $\operatorname{Rt}\triangle BCH$。
- 必须先问的问题：所给角是不是两条已知边的夹角？作高顶点是否避开了已知角顶点？第一个要解的直角三角形是哪一个？
- 关键讲解顺序：识别 SAS -> 从 $B$ 作高 -> 解 $ABH$ -> 判点序并求 $CH$ -> 解 $BCH$ -> 内角和与边角关系验算。
- 最适合的具体数值例子：$\angle A=30^\circ,AB=10,AC=13$；第一个直角三角形的高、投影整洁，第二个直角三角形保留适量根式。
- 讲到哪里停下来让学生回答：算出 $BH,AH$ 后暂停，先让学生比较 $AH$ 与 $AC$，口述为什么此处用 $CH=AC-AH$。

## 十、推荐练题任务包
- 若卡在读题/入手动作，出什么题：给多组“两边一角”条件，只判断是 SAS 还是 SSA，并圈出夹角。
- 若卡在建模或关系入口，出什么题：只要求选择从哪个顶点作高，并指出第一个可解的直角三角形及其已知量。
- 若卡在求解和检查，出什么题：保留 $30^\circ,45^\circ,60^\circ$ 特殊角，专练“求高—求投影—判点序—求剩余投影”。
- 若原题已稳，如何小步迁移：交换两条已知边的长度，使垂足位置发生变化，但其余步骤保持不变。
- 若结构识别已稳，如何深化/抽象/包装：不给辅助线，要求学生比较两种作高方案并选择计算更短者；再将目标改为面积或高。
- 禁止出的跑偏变式：使用正弦定理/余弦定理的题；把 SSA 两解判断混入 SAS 单元；数字导致连续多次近似且掩盖结构动作的题。
