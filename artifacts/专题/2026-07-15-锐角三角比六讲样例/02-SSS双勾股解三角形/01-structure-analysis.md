# 结构分析：SSS 条件下作高并用双勾股解三角形

## 原题
本讲用两个例题覆盖高的两种落足位置。

1. 锐角三角形：在 $\triangle ABC$ 中，$AB=13$，$AC=15$，$BC=14$。过 $A$ 作 $AD\perp BC$ 于 $D$。求 $AD$ 及三角形的三个角（精确到 $0.1^\circ$）。
2. 钝角三角形：在 $\triangle ABC$ 中，$AB=13$，$AC=20$，$BC=11$，$\angle B$ 为钝角。过 $A$ 作 $AD\perp BC$ 所在直线于 $D$。求 $AD$ 及三角形的三个角（精确到 $0.1^\circ$）。

限制：只用勾股定理、锐角三角比及计算器反求锐角，不用正弦定理、余弦定理。

## 一、题目场景
- 数学对象：已知三边的普通三角形、从顶点 $A$ 向边 $BC$ 所在直线作出的高 $AD$、两个直角三角形。
- 变量/参数：底边分段 $BD=x$；$AD$ 不设为未知数，而由勾股表示。
- 函数/图形：锐角三角形时 $D$ 在线段 $BC$ 上；$\angle B$ 为钝角时 $D$ 在 $BC$ 的反向延长线上。
- 已知条件：SSS；例 1 为 $13,15,14$，例 2 为 $13,20,11$ 且 $\angle B$ 为钝角。
- 要求目标：求高，再由两个直角三角形中的锐角三角比求原三角形各角。

## 二、核心结构
### 2.1 表层信息
- 表面考点：SSS 解三角形、作高、勾股定理、锐角三角比。
- 题型功能：`composite_problem`
- 是否值得完整 structural analysis：是；关键不是套一条边角公式，而是先把普通三角形构造成两个直角三角形，并根据高的落足位置正确写底边分段。
- 一句话问题模式：三边已知 $\rightarrow$ 作高拆成两个 Rt 三角形 $\rightarrow$ 设底边分段 $x$ $\rightarrow$ 双勾股消去高 $\rightarrow$ 用锐角三角比求角。

### 2.2 结构表达
#### 判别条件表（概念辨析题用；不适用则写“无”）
- 必要条件：先有高 $AD\perp BC$，才能在两个直角三角形内使用勾股和锐角三角比。
- 充分条件：三边已知且点序分支明确时，两个勾股式可确定底边分段与高。
- 常见干扰项：不判断 $D$ 的位置便直接写 $DC=BC-BD$；把高设为 $x$，导致底边两段还需另设未知数；直接对原三角形使用锐角三角比。
- 最短检查动作：先画高并判断 $D$ 在线段内还是延长线上，再写 $DC=BC-BD$ 或 $DC=BC+BD$。

#### 情景量表（应用题用；不适用则写“无”）
无。

#### 命题网络（所有题型都写；简单题写简版）
- P1（题设）：$AB,AC,BC$ 三边已知。
- P2（构造）：过 $A$ 作 $AD\perp BC$ 所在直线，得到 Rt$\triangle ABD$ 与 Rt$\triangle ACD$。
- P3（检查）：锐角分支中 $D\in BC$，有 $DC=BC-BD$；$B$ 钝角分支中点序为 $D-B-C$，有 $DC=BD+BC$。
- P4（计算状态）：设 $BD=x$，分别由两个直角三角形得到两个含 $AD^2$ 的勾股式。
- P5（可推）：两式相等或相减消去 $AD^2$，求出 $x$，再由勾股求 $AD$。
- P6（可推）：在两个 Rt 三角形中写锐角三角比，并用计算器反求锐角。
- P7（目标）：得到原三角形的三角、三边及高；钝角分支用补角还原 $\angle B$。
- R1：P1 + P2 + P3 -> P4，方法：作高、按点序表示底边另一段。
- R2：P4 -> P5，方法：双勾股消元。
- R3：P2 + P5 -> P6，方法：在直角三角形内使用锐角三角比。
- R4：P3 + P6 -> P7，方法：锐角分支直接对应；钝角分支用 $180^\circ-\angle ABD$。
- 目标：由 P2、P5、P6、P7 完成 SSS 解三角形。

### 2.3 解题主链
```text
SSS -> 作 AD⊥BC -> 判断 D 的位置 -> 设 BD=x -> 表示 DC -> 双勾股消去 AD² -> 求 AD -> 在两个 Rt 三角形中求锐角 -> 还原原三角形
```

### 2.4 模型标签
- model_id：`triangle_sss_altitude_double_pythagoras`
- model_name：SSS 作高—底边分段—双勾股解三角形
- configuration：已知 $AB,AC,BC$；求底边分段、高与三个角。
- 可迁移方向：换一组能产生干净分段与高的三边；改变钝角所在顶点；改求面积或指定锐角的三角比。
- 非同构边界：只给两边或边角条件不再是 SSS；直接使用正弦定理、余弦定理不属于本课模型；不作高便对普通三角形使用锐角三角比是错误路径。

## 二点五、知识点/模型锚点
- 建议讲义标题：SSS：先作高，再用双勾股解两个 Rt 三角形
- 知识点/模型名称：高的落足分支与共高双勾股。
- 核心公式/定理：$AB^2=AD^2+BD^2$，$AC^2=AD^2+DC^2$；锐角三角比只在直角三角形中按对边、邻边、斜边取比。
- 使用条件：三边已知；已作 $AD\perp BC$；点序已经判断。
- 入口信号：题目给普通三角形三边，要求角、高或完整解三角形。
- 易混边界：锐角分支 $DC=BC-BD$；$B$ 钝角分支 $DC=BC+BD$；$\angle ABD$ 是锐角但不是原三角形的 $\angle B$，二者互补。
- 本题如何体现：例 1 得 $BD=5,DC=9,AD=12$；例 2 得 $BD=5,DC=16,AD=12$。
- 可作为例题的结构层级：例 1 建立主流程，例 2 只改变落足位置这一维度。

## 三、关键转化
- 最关键的转化：作 $AD\perp BC$，把一个普通三角形拆成两个共享高 $AD$ 的直角三角形。
- 为什么降低计算量：设底边分段 $BD=x$ 后，另一段由点序直接表示；两个勾股式共享 $AD^2$，相减即可消元。
- 不转化时的低效路径：同时设高和两个底边分段为未知数，方程冗长；或者试图在普通三角形里直接写 sin/cos/tan。

## 四、标准路径骨架
1. 先做什么：过 $A$ 作 $AD\perp BC$ 所在直线，把原三角形拆成两个 Rt 三角形。
2. 再做什么：判断 $D$ 的位置；设 $BD=x$，不要设高为 $x$。
3. 建立什么关系：按点序表示 $DC$，分别在 Rt$\triangle ABD$、Rt$\triangle ACD$ 中列勾股式。
4. 如何求解：两式消去 $AD^2$ 求 $x$，再由一个勾股式求 $AD$；随后在两个 Rt 三角形中用锐角三角比求角。
5. 需要检查什么：底边分段为正且点序一致；钝角分支中原 $\angle B=180^\circ-\angle ABD$；三角形内角和为 $180^\circ$。

## 四点五、标准完整解与验算
### 例 1：锐角三角形，垂足在线段内
- 关键交点/关键量：$D\in BC$，设 $BD=x$，则 $DC=14-x$。
- 面积/方程/关系式：
  $$AD^2=13^2-x^2=15^2-(14-x)^2.$$
- 完整求解过程：
  $$169-x^2=225-(14-x)^2=29+28x-x^2,$$
  所以 $x=5$。故 $BD=5,DC=9$，
  $$AD=\sqrt{13^2-5^2}=12.$$
  在 Rt$\triangle ABD$ 中，$\tan B=\dfrac{12}{5}$，故 $B\approx67.4^\circ$；在 Rt$\triangle ACD$ 中，$\tan C=\dfrac{12}{9}=\dfrac43$，故 $C\approx53.1^\circ$。于是
  $$A=180^\circ-B-C\approx59.5^\circ.$$
- 最终答案：$AD=12$，$A\approx59.5^\circ$，$B\approx67.4^\circ$，$C\approx53.1^\circ$。
- 排除值：$0<x<14$；所得 $x=5$ 合法。
- 退化情形：无，三角形三边满足严格三角不等式。
- 验算：$5^2+12^2=13^2$，$9^2+12^2=15^2$；三角约为 $180.0^\circ$。
- 本题最短可靠路径：设 $BD=x$，令两个 $AD^2$ 表达式相等。

### 例 2：钝角三角形，垂足在延长线上
- 关键交点/关键量：点序为 $D-B-C$，设 $BD=x$，则 $DC=x+11$。
- 面积/方程/关系式：
  $$AD^2=13^2-x^2=20^2-(x+11)^2.$$
- 完整求解过程：
  $$169-x^2=400-(x+11)^2=279-22x-x^2,$$
  所以 $x=5$。故 $BD=5,DC=16$，
  $$AD=\sqrt{13^2-5^2}=12.$$
  在 Rt$\triangle ABD$ 中，$\tan\angle ABD=\dfrac{12}{5}$，所以 $\angle ABD\approx67.4^\circ$。由于 $\angle ABC$ 与 $\angle ABD$ 互补，
  $$B\approx180^\circ-67.4^\circ=112.6^\circ.$$
  在 Rt$\triangle ACD$ 中，$\tan C=\dfrac{12}{16}=\dfrac34$，故 $C\approx36.9^\circ$；于是 $A\approx30.5^\circ$。
- 最终答案：$AD=12$，$A\approx30.5^\circ$，$B\approx112.6^\circ$，$C\approx36.9^\circ$。
- 排除值：$x>0$；所得 $x=5$ 与点序 $D-B-C$ 一致。
- 退化情形：无，三角形三边满足严格三角不等式。
- 验算：$5^2+12^2=13^2$，$16^2+12^2=20^2$；三角约为 $180.0^\circ$。
- 本题最短可靠路径：点序先行，再写 $DC=BD+BC$，双勾股消元。

## 五、出题人逻辑
- 诱导学生硬算的位置：把高设为 $x$，再另设底边分段；或未判断垂足位置就机械套 $DC=BC-BD$。
- 真正的捷径：设底边分段为 $x$，让高通过勾股出现为同一个 $AD^2$，两式相减消去。
- 训练的可迁移能力：先构造直角三角形，再在 Rt 三角形内部使用三角比；根据图形点序选择加法或减法。

## 六、学生卡点预测
- 读题/入手动作卡点：不知道普通三角形不能直接按“对边/邻边/斜边”写比，忘记先作高。
- 建模/关系入口卡点：把高设为 $x$；不会按 $D$ 的位置写 $DC=14-x$ 或 $DC=x+11$。
- 求解/检查卡点：双勾股没有消去同一个 $AD^2$；把钝角例中的 $\angle ABD$ 当成原 $\angle B$。

## 七、变式原则
- 核心不变量：作一条高得到两个共高 Rt 三角形，设底边分段，双勾股消元，再用锐角三角比求角。
- 表层特征：三边数字、钝角所在顶点、所求可以是高/面积/角/某锐角三角比。
- 可变维度：每次只改变数字、落足位置、问题目标或图形表征中的一个。
- 深化阶梯：先复现 $13$-$14$-$15$；再同结构换数；再把所求改为面积或角；再隐藏垂足位置让学生先判断。
- 允许的变换：选取能产生整数底边分段与整数高的三边；把钝角移到 $C$ 并相应调整点序。
- 禁止的变换：同时加入角平分线、中线、相似等新结构；引入正弦定理或余弦定理；让数字导致繁重根式或反三角函数计算。
- 表征切换：从带高的图切换为只给三边的图，让学生自己补高。
- 包装方式：问“求面积后求三个角”，仍保持同一主链。
- 近迁移例子：$AB=15,AC=13,BC=14$，高仍为 $12$，但底边分段交换。
- 远迁移例子：只说明某角为钝角而不画延长线，要求学生自行补出外落高。
- 反例/伪变式：给 SAS 条件后仍称为 SSS 双勾股；这会改变条件网络，不是本讲的同构变式。

## 八、计算复杂度预算
- 原题计算层级：一元一次消元、两次整数勾股、两个反正切近似。
- 允许小步上升到：底边分段出现简单整数或一位小数，角保留到 $0.1^\circ$。
- 禁止引入的计算负担：复杂根式、二次方程多解筛选、非特殊角的手算三角函数、任何正弦定理或余弦定理。
- 必须保留的可见支架：明确的作高动作、点序、$BD=x$、两个勾股式和钝角的补角还原。

## 九、推荐讲题任务包
- 建议的本轮教学入口：先问“锐角三角比在哪一种三角形里定义”，再要求学生把普通三角形切成两个 Rt 三角形。
- 本题讲解目标：稳定完成“作高—判落足—设底边段—双勾股—解两个 Rt 三角形”。
- 不要直接讲的抽象话：不要先给“SSS 解三角形公式”；不要使用正弦定理或余弦定理。
- 必须先问的问题：高落在线段内还是延长线上？此时 $DC$ 应该用减法还是加法？
- 关键讲解顺序：锐角主例建立流程；钝角例只突出点序和补角这两个变化。
- 最适合的具体数值例子：锐角 $13,15,14$；钝角 $13,20,11$。
- 讲到哪里停下来让学生回答：作高后停下，让学生决定设哪个量为 $x$；画出钝角外落高后停下，让学生写 $DC$。

## 十、推荐练题任务包
- 若卡在读题/入手动作，出什么题：只要求在普通三角形中补一条高并圈出两个 Rt 三角形。
- 若卡在建模或关系入口，出什么题：给出锐/钝两幅图，只填 $BD=x$ 时 $DC$ 的表达式。
- 若卡在求解和检查，出什么题：给出两个勾股式，让学生只做消元并检验分段长度。
- 若原题已稳，如何小步迁移：保持落足位置不变，只换一组整数勾股数字。
- 若结构识别已稳，如何深化/抽象/包装：不画高，要求学生判断从哪个顶点作高最便于使用已知三边。
- 禁止出的跑偏变式：正弦定理、余弦定理、含复杂根式或另加相似/圆结构的题。
