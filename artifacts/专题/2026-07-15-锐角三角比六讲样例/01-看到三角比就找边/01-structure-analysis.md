# 结构分析：看到三角比就找对应的两条边

## 原题

本讲用两组例子训练同一个动作。

例 1：在 $\mathrm{Rt}\triangle ABC$ 中，$\angle A=90^\circ$，$AD\perp BC$ 于 $D$。不计算，只写出
$\sin B$、$\cos\angle BAD$、$\tan\angle CAD$、$\cot\angle ABD$ 分别是哪两条边的比。

例 2：在 $\mathrm{Rt}\triangle EFG$ 中，$\angle G=90^\circ$，$EG=\sqrt2$，$FG=\sqrt6$，求 $\sin E$。

## 一、题目场景

- 数学对象：直角三角形、目标锐角、对边、邻边、斜边、四种锐角三角比。
- 变量/参数：目标角与所求函数可变；两条已知边可带相同整数或根式公因子。
- 函数/图形：简单直角三角形或含多个直角三角形的复合图。
- 已知条件：题中能找到含目标角的直角三角形；计算题给出其中两条边。
- 要求目标：看到 $\sin/\cos/\tan/\cot$ 后，立即找对所需两类边并列比；必要时先化简边长。

## 二、核心结构

### 2.1 表层信息

- 表面考点：锐角三角比定义、勾股定理、根式约分。
- 题型功能：`basic_skill`
- 是否值得完整 structural analysis：是；理由：本讲锁定后续所有锐角三角比题共用的入口动作，不训练孤立的边名辨析。
- 一句话问题模式：函数名决定两类边，目标角决定这些边在图中的位置。

### 2.2 结构表达

#### 判别条件表

| 看到 | 立即找 |
|---|---|
| $\sin\theta$ | 对边、斜边 |
| $\cos\theta$ | 邻边、斜边 |
| $\tan\theta$ | 对边、邻边 |
| $\cot\theta$ | 邻边、对边 |

- 必要条件：先找到一个包含目标锐角的直角三角形。
- 充分条件：在该直角三角形中认出函数所需的两类边，边长已知或可由勾股得到。
- 常见干扰项：看到函数名却先盲算第三边；在一般三角形中直接认斜边；忽略两条已知边的公因子；目标角改变后沿用旧边名。
- 最短检查动作：读函数名定两类边，再在图中圈出同时包含目标角和这两类边的 $\mathrm{Rt}\triangle$。

#### 情景量表

无。

#### 命题网络

- P1（题设）：题中存在包含目标锐角 $\theta$ 的直角三角形。
- P2（定义）：$\sin/\cos/\tan/\cot$ 分别对应对斜、邻斜、对邻、邻对。
- P3（可推）：在选定的直角三角形中确定所需两条边。
- P4（计算状态）：若已知边同含正公因子 $k$，先把相关边同时除以 $k$，三角比不变。
- P5（计算状态）：必要时用勾股补出所需边。
- P6（目标）：列出正确边比并化简。
- R1：P1 + P2 -> P3，方法：函数名定边类，目标角与直角共同锁定三角形。
- R2：P3 + P4 -> P5，方法：相似缩放与勾股定理。
- R3：P2 + P5 -> P6，方法：锐角三角比定义。
- 目标：由 P1、P2、P3、P4、P5 得到 P6。

### 2.3 解题主链

```text
看函数名 -> 定所需边类 -> 找含目标角的 Rt 三角形 -> 认边
-> 有公因子先约分 -> 缺所需边再勾股 -> 列比并化简
```

### 2.4 模型标签

- model_id：right_triangle_ratio_reaction
- model_name：锐角三角比找边反应链
- configuration：给目标锐角与一种三角比，图中可找到直角三角形；可能另给两边。
- 可迁移方向：换目标角、换函数名、换参照直角三角形、给两直角边或斜边与一直角边、加入可约公因子。
- 非同构边界：不作高就在一般三角形中直接认对邻斜；使用正弦定理或余弦定理；反求一般三角形。

## 二点五、知识点/模型锚点

- 建议讲义标题：看到 $\sin/\cos/\tan/\cot$，先找哪两条边？
- 知识点/模型名称：函数名到边类的即时反应；直角三角形定位；边长整体约分。
- 核心公式/定理：$\sin=\text{对}/\text{斜}$，$\cos=\text{邻}/\text{斜}$，$\tan=\text{对}/\text{邻}$，$\cot=\text{邻}/\text{对}$；勾股定理。
- 使用条件：只在选定的直角三角形中认对边、邻边、斜边；目标角必须是锐角。
- 入口信号：题目出现 $\sin/\cos/\tan/\cot$ 或“求某锐角的三角比”。
- 易混边界：函数名先决定要找的边类；不需要每次把三条边都完整命名。只有缺少所需边时才补边。
- 本题如何体现：例 1 训练函数名到线段比；例 2 训练先约根式公因子再计算。
- 可作为例题的结构层级：基础反应 + 同结构数值简化。

## 三、关键转化

- 最关键的转化：把“求某函数”立即翻译成“两条边的比”，再回图中找这两条边。
- 为什么降低计算量：只寻找函数真正需要的边；两边先整体约去公因子，可把根式勾股化成简单数勾股。
- 不转化时的低效路径：先机械补齐三边、详细命名所有边，再回头猜函数定义。

## 四、标准路径骨架

1. 先做什么：看到函数名，先说出分子和分母分别是哪类边。
2. 再做什么：在图中找包含目标角的直角三角形。
3. 建立什么关系：在该直角三角形中定位所需两条边。
4. 如何求解：已知边有公因子先同时约分；缺所需边时再用勾股；最后列比。
5. 需要检查什么：所用三角形是否为直角三角形；分子分母是否与函数名一致；约分是否对所有相关边同倍缩放。

## 四点五、标准完整解与验算

- 关键交点/关键量：例 1 中直角分别在 $A$ 或 $D$；例 2 中直角在 $G$。
- 面积/方程/关系式：例 2 将 $EG,FG$ 同除以 $\sqrt2$，得到邻边与对边之比为 $1:\sqrt3$，归一后的斜边为 $2$。
- 完整求解过程：例 1 得 $\sin B=AC/BC$，$\cos\angle BAD=AD/AB$，$\tan\angle CAD=CD/AD$，$\cot\angle ABD=BD/AD$。例 2 中 $EG:FG=1:\sqrt3$，由勾股得归一后的斜边 $EF=2$，故 $\sin E=\sqrt3/2$。
- 最终答案：上述四个线段比；$\boxed{\sin E=\dfrac{\sqrt3}{2}}$。
- 排除值：边长为正，公因子必须为正。
- 退化情形：两条直角边不能为 $0$。
- 验算：$0<\sin E<1$；也可用原边长计算 $EF=2\sqrt2$，得 $\sqrt6/(2\sqrt2)=\sqrt3/2$。
- 本题最短可靠路径：函数名定边 -> 圈 Rt -> 认边 -> 先约公因子 -> 必要时勾股 -> 列比。

## 五、出题人逻辑

- 诱导学生硬算的位置：一看到两边就立刻代入勾股，忽略 $\sqrt2$ 公因子。
- 真正的捷径：先问函数需要哪两条边，并把可同倍缩小的边先化简。
- 训练的可迁移能力：在复合图中主动找合适的直角三角形；对带公因子的边长先做数值降复杂度。

## 六、学生卡点预测

- 读题/入手动作卡点：看见函数名不能立即说出分子、分母边类。
- 建模/关系入口卡点：不能从复合图中找到同时包含目标角和所需边的直角三角形。
- 求解/检查卡点：有公因子仍直接做复杂根式勾股；列比后不约分。

## 七、变式原则

- 核心不变量：函数名定边类，直角三角形定边的位置，三角比对整体缩放不变。
- 表层特征：点名、方向、目标角、函数名、已知边组合、公共因子。
- 可变维度：一次只换函数名、目标角、参照三角形或公因子之一。
- 深化阶梯：直接线段比 -> 简单整数边 -> 带整数公因子 -> 带根式公因子 -> 复合图中选 Rt。
- 允许的变换：$3,3\sqrt3$；$2\sqrt5,6\sqrt5$；斜边与一直角边同含公因子。
- 禁止的变换：同时加入复杂复合图、复杂根式与反求角；引入一般三角形定理。
- 表征切换：图中标边、文字给边、只填线段比、计算具体值。
- 包装方式：不提示“先约分”，让学生主动发现共同因子。
- 近迁移例子：两直角边为 $2\sqrt5,6\sqrt5$，求 $\cos$。
- 远迁移例子：在含高线的图中指定小直角三角形内的角，求一种三角比。
- 反例/伪变式：在一般三角形中不作高直接询问“斜边”。

## 八、计算复杂度预算

- 原题计算层级：一步认边；一次整体约分；一次简单勾股；一次列比。
- 允许小步上升到：约分后得到常见勾股数组或 $1:\sqrt3:2$、$1:1:\sqrt2$。
- 禁止引入的计算负担：复杂二次根式、有理化多步运算、反三角函数、正弦定理、余弦定理。
- 必须保留的可见支架：四函数到边比的 $2\times2$ 格；“有公因子先约”的动作提醒。

## 九、推荐讲题任务包

- 建议的本轮教学入口：教师随机说一个函数名，学生立刻说分子、分母边类。
- 本题讲解目标：学生形成“函数名 -> 两类边 -> 找 Rt -> 找线段 -> 简化计算”的连续动作。
- 不要直接讲的抽象话：不展开单位圆、函数图像或一般三角形中的三角关系。
- 必须先问的问题：“看到这个函数，你现在只需要找哪两类边？”
- 关键讲解顺序：2×2 格 -> 复合图快速写比 -> 根式公因子例题。
- 最适合的具体数值例子：$\sqrt2,\sqrt6$ 同除以 $\sqrt2$ 得 $1,\sqrt3$。
- 讲到哪里停下来让学生回答：每出现一个函数名就暂停，让学生先只说边类，不先计算。

## 十、推荐练题任务包

- 若卡在读题/入手动作，出什么题：只给函数名，口答“哪类边除以哪类边”。
- 若卡在建模或关系入口，出什么题：在复合图中只要求写线段比，不算数值。
- 若卡在求解和检查，出什么题：给带共同整数或根式因子的两边，要求先写约分后的边长比。
- 若原题已稳，如何小步迁移：随机更换目标角与函数名。
- 若结构识别已稳，如何深化/抽象/包装：隐藏“先约分”提示，观察学生是否主动缩放。
- 禁止出的跑偏变式：特殊角背值、一般三角形解法、正弦定理或余弦定理。
