# 结构分析：SSA 两解辨析——作高后看点的两个位置

## 原题
在 $\triangle ABC$ 中，已知 $\angle A=30^\circ$，$AB=8$，$BC=5$。不预设点 $C$ 的位置唯一，求所有可能的 $AC$，并说明为什么会出现两个三角形。

限制：只能使用作高、锐角三角比、勾股定理和线段加减；不使用正弦定理、余弦定理。

本题是标准 SSA：已知两边 $AB,BC$ 以及其中一边 $BC$ 的对角 $\angle A$。SSA 并不总能唯一确定三角形，可能有 $0$、$1$ 或 $2$ 个解。

## 一、题目场景
- 数学对象：一般三角形 $ABC$；已知角 $A$；已知角的对边 $BC$；另一条已知边 $AB$；从非已知角顶点 $B$ 向 $AC$ 所在直线作的高 $BH$。
- 变量/参数：点 $C$ 在从 $A$ 出发、与 $AB$ 成 $30^\circ$ 的射线上的位置；$AC$ 的长度；垂足 $H$ 与 $A,C$ 的点序。
- 函数/图形：射线与定长条件的两个可能交点；作高后得到共用高 $BH$ 的两个直角三角形。不使用三角函数图像。
- 已知条件：$\angle A=30^\circ$，$AB=8$，$BC=5$。
- 要求目标：求所有可能的 $AC$；验证两个三角形均存在；说明两个解分别对应哪一种点序。

## 二、核心结构
### 2.1 表层信息
- 表面考点：锐角三角比；勾股定理；一般三角形作高；分类讨论；线段和差。
- 题型功能：`composite_problem`
- 是否值得完整 structural analysis：是；理由：SSA 的难点不是一次计算，而是“同一个垂直距离可能对应高足两侧的两个点”。若默认图形唯一，就会漏掉一个合法三角形。
- 一句话问题模式：固定一个角和角的一边后，用另一条已知边去截该角的另一条射线；作高把位置问题转成“垂足两侧各取同一段”的线段和差问题。

### 2.2 结构表达
#### 判别条件表（概念辨析题用；不适用则写“无”）
以下判别表只针对 $A$ 为锐角、已知 $a=BC$（$A$ 的对边）与 $c=AB$（另一已知边）的标准 SSA。令
\[
h=c\sin A.
\]

| 条件 | 三角形个数 | 几何含义 |
|---|---:|---|
| $a<h$ | $0$ | 定长 $a$ 够不到角的另一边 |
| $a=h$ | $1$ | 恰好在垂足处相切，得到一个直角三角形 |
| $h<a<c$ | $2$ | 垂足两侧各有一个合法点，且两个点都在射线上 |
| $a\ge c$ | $1$ | 垂足靠近 $A$ 一侧的候选点退化或落到反向延长线上，只剩一个合法点 |

- 必要条件：必须明确哪一边是已知角的对边；两解只可能出现在已知角为锐角且 $h<a<c$ 时。
- 充分条件：对本题，$h=AB\sin A=4$，而 $4<BC=5<AB=8$，故必有两个解。
- 常见干扰项：默认点 $C$ 只有一个位置；从已知角顶点 $A$ 作高，不能直接形成固定高；只写 $AC=AH+HC$ 而漏掉 $AC=AH-HC$；把“垂足在线段外”误判为三角形不存在。
- 最短检查动作：先算 $h=AB\sin A$，比较 $h,BC,AB$；若 $h<BC<AB$，立即预判两解，再作高分别写线段差与线段和。

#### 情景量表（应用题用；不适用则写“无”）
无。

#### 命题网络（所有题型都写；简单题写简版）
- P1（题设）：$\angle A=30^\circ$，$AB=8$，$BC=5$。
- P2（构造）：从非已知角顶点 $B$ 向 $AC$ 所在直线作 $BH\perp AC$ 于 $H$。
- P3（可推）：在 $\mathrm{Rt}\triangle ABH$ 中，$BH=AB\sin30^\circ=4$，$AH=AB\cos30^\circ=4\sqrt3$。
- P4（检查）：$BH=4<BC=5<AB=8$，符合锐角 SSA 两解条件。
- P5（可推）：在 $\mathrm{Rt}\triangle BCH$ 中，$HC=\sqrt{BC^2-BH^2}=\sqrt{25-16}=3$。
- P6（分支一）：若 $A-C_1-H$，则 $AC_1=AH-HC_1=4\sqrt3-3$。
- P7（分支二）：若 $A-H-C_2$，则 $AC_2=AH+HC_2=4\sqrt3+3$。
- P8（检查）：$4\sqrt3-3>0$，且两组边长都满足三角形两边之和大于第三边；两个候选点均在从 $A$ 出发的同一射线上。
- P9（目标）：$AC$ 有两个可能值 $4\sqrt3-3$ 与 $4\sqrt3+3$，对应两个不同三角形。
- R1：P1 + P2 -> P3，方法：在直角三角形 $ABH$ 中用正弦、余弦求高与投影。
- R2：P3 + P1 -> P4，方法：比较高、已知对边与另一已知边，预判 SSA 解的个数。
- R3：P2 + P1 + P3 -> P5，方法：在直角三角形 $BCH$ 中用勾股定理求垂足到候选点的距离。
- R4：P3 + P5 + 点序 -> P6/P7，方法：按垂足在线段 $AC$ 外或内分别作线段差、线段和。
- R5：P6 + P7 + P8 -> P9，方法：正值、射线位置与三角形不等式联合验算。
- 目标：由 R1、R2 识别两解，再由 R3、R4 分支求值，最后由 R5 保留两个合法答案。

### 2.3 解题主链
```text
识别标准 SSA -> 从非已知角顶点作高 -> 在第一个 Rt 三角形求高与投影
-> 比较 h、已知对边、另一已知边，预判解数
-> 在第二个 Rt 三角形勾股求 HC
-> 按 C 在 H 两侧分类：AH-HC 或 AH+HC
-> 检查点序、正值与三角形存在 -> Answer
```

### 2.4 模型标签
- model_id：`acute_ssa_altitude_two_positions`
- model_name：锐角 SSA 作高后的两位置判别
- configuration：已知锐角 $A$、其对边 $a=BC$ 与另一边 $c=AB$，求第三边 $b=AC$ 及三角形个数。
- 可迁移方向：保持 $A$ 为锐角，改变 $a$ 相对 $h=c\sin A$ 与 $c$ 的大小，分别生成 $0/1/2$ 解；已知角和字母换位；由求第三边迁移到补全其余角。
- 非同构边界：SSS、SAS、ASA、AAS 的唯一构型；使用正弦定理或余弦定理直接列式；已知角为直角或钝角时套用本表的“两解”判断；只画一个看似合理的三角形便宣布唯一。

## 二点五、知识点/模型锚点
- 建议讲义标题：SSA 不是唯一条件：作高后检查垂足两侧。
- 知识点/模型名称：SSA 的 $0/1/2$ 解；从非已知角顶点作高；共用高的两个直角三角形；线段和差分类。
- 核心公式/定理：
  - $h=c\sin A$；当 $A$ 为锐角时，$a<h$ 为无解，$a=h$ 为一解，$h<a<c$ 为两解，$a\ge c$ 为一解；
  - 本题 $BH=AB\sin A$，$AH=AB\cos A$；
  - $HC=\sqrt{BC^2-BH^2}$；
  - 两解分别为 $AC=AH-HC$ 与 $AC=AH+HC$，但必须结合点序与正值检查。
- 使用条件：标准 SSA 必须是“两边及其中一边的对角”；上述阈值表要求已知角 $A$ 为锐角；三角形边长均为正且点 $C$ 位于角 $A$ 的指定射线上。
- 入口信号：题目给出一个角、该角的对边、另一条边；没有给夹角；问“所有可能”“是否唯一”“解三角形”。
- 易混边界：SAS 的已知角夹在两条已知边之间，通常唯一；SSA 的已知角不夹在两条已知边之间，可能不唯一。垂足在线段延长线上并不自动表示无解。
- 本题如何体现：$BH=4<BC=5<AB=8$，圆心在 $B$、半径为 $5$ 的定长条件在射线上给出 $C_1,C_2$ 两个位置；二者关于垂足 $H$ 的位置距离均为 $3$。
- 可作为例题的结构层级：两解主例；数值设计让第一个直角三角形含 $30^\circ$，第二个直角三角形为 $3$-$4$-$5$，把认结构置于复杂计算之前。

## 三、关键转化
- 最关键的转化：不直接在一般三角形中硬算，而是从非已知角顶点 $B$ 作高，把原题拆成 $\mathrm{Rt}\triangle ABH$ 与 $\mathrm{Rt}\triangle BCH$；再把“两个三角形”转成“点 $C$ 在垂足 $H$ 两侧的两个位置”。
- 为什么降低计算量：$AB=8$ 与 $30^\circ$ 立即给出 $BH=4,AH=4\sqrt3$；$BC=5$ 与 $BH=4$ 立即形成 $3$-$4$-$5$ 直角三角形，两个答案只差最后一步的加减。
- 不转化时的低效路径：只凭一张单构型草图列式，容易漏解；设一般未知量列复杂方程，会掩盖两个交点与线段和差的几何来源；正弦定理、余弦定理不在本课允许范围内。

## 四、标准路径骨架
1. 先做什么：识别为 SSA，并指出已知角 $A$ 不是两条已知边 $AB,BC$ 的夹角。
2. 再做什么：从不在已知角上的顶点 $B$ 向 $AC$ 所在直线作 $BH\perp AC$。
3. 建立什么关系：在 $\mathrm{Rt}\triangle ABH$ 中求 $BH,AH$；比较 $BH,BC,AB$ 判断解数；在 $\mathrm{Rt}\triangle BCH$ 中求 $HC$。
4. 如何求解：分别按 $A-C_1-H$ 和 $A-H-C_2$ 写 $AC_1=AH-HC_1$、$AC_2=AH+HC_2$。
5. 需要检查什么：两个 $AC$ 是否为正；两个候选点是否都在角 $A$ 的同一射线上；是否满足三角形不等式；是否把垂足在线段外的合法构型误删。

## 四点五、标准完整解与验算
- 关键交点/关键量：作 $BH\perp AC$ 所在直线于 $H$；$BH=4$，$AH=4\sqrt3$，$HC_1=HC_2=3$。
- 面积/方程/关系式：$BH=8\sin30^\circ=4$；$AH=8\cos30^\circ=4\sqrt3$；$HC=\sqrt{5^2-4^2}=3$。
- 完整求解过程：
  1. 从 $B$ 向 $AC$ 所在直线作 $BH\perp AC$ 于 $H$。在 $\mathrm{Rt}\triangle ABH$ 中，$BH=AB\sin A=8\cdot\frac12=4$，$AH=AB\cos A=8\cdot\frac{\sqrt3}{2}=4\sqrt3$。
  2. 因为 $BH=4<BC=5<AB=8$，所以点 $C$ 可位于 $H$ 的两侧，预判有两个解。
  3. 在 $\mathrm{Rt}\triangle BCH$ 中，$HC=\sqrt{BC^2-BH^2}=\sqrt{25-16}=3$。
  4. 近点位置 $C_1$ 满足 $A-C_1-H$，所以 $AC_1=AH-C_1H=4\sqrt3-3$。
  5. 远点位置 $C_2$ 满足 $A-H-C_2$，所以 $AC_2=AH+HC_2=4\sqrt3+3$。
- 最终答案：$\boxed{AC=4\sqrt3-3\ \text{或}\ 4\sqrt3+3}$，共有两个三角形。
- 排除值：无；两个值均为正。不能把“$\pm$”留作未经解释的代数符号，必须对应两种点序。
- 退化情形：若 $BC=BH=4$，两候选点重合于 $H$，只有一个直角三角形；若 $BC<BH$，无交点；若近点退到 $A$ 或射线反向，则该分支退化或无效。
- 验算：
  - $4\sqrt3-3\approx3.928>0$，且 $(4\sqrt3-3)+5>8$；
  - $4\sqrt3+3\approx9.928$，且 $8+5>4\sqrt3+3$；
  - 两分支中均有 $BH^2+HC^2=4^2+3^2=5^2=BC^2$，且 $\tan30^\circ=BH/AH=4/(4\sqrt3)=1/\sqrt3$。
- 本题最短可靠路径：作 $BH$ -> 算 $BH=4,AH=4\sqrt3$ -> 比较 $4<5<8$ 确认两解 -> 勾股得 $HC=3$ -> 写 $AH\pm HC$ 并解释点序。

## 五、出题人逻辑
- 诱导学生硬算的位置：题面只写一个 $\triangle ABC$，学生容易照惯常草图只画一个 $C$，然后把 $AC$ 固定写成 $AH+HC$ 或 $AH-HC$。
- 真正的捷径：先比较 $h,a,c$ 预判解数；再让两个候选点共用同一条高，计算只做一次。
- 训练的可迁移能力：看到 SSA 先问“是否唯一”；能用作高和点序解释 $0/1/2$ 解，而不是背一句“SSA 可能两解”。

## 六、学生卡点预测
- 读题/入手动作卡点：分不清 SAS 与 SSA；默认三个已知条件一定唯一；把 $BC$ 忘记识别为 $\angle A$ 的对边。
- 建模/关系入口卡点：从已知角顶点 $A$ 作高，导致固定高无法直接求出；不会把 $BC$ 看作从 $B$ 到候选点 $C$ 的定长；不知道应比较 $BH,BC,AB$。
- 求解/检查卡点：只保留 $AH+HC$；见到 $H$ 在线段 $AC$ 延长线上就删掉近点分支；写 $AC=AH\pm HC$ 却不说明两种点序；忽略近点分支的三角形不等式。

## 七、变式原则
- 核心不变量：标准 SSA + 从非已知角顶点作高 + 比较 $h,a,c$ + 垂足两侧分类 + 线段和差。
- 表层特征：顶点字母、图形朝向、已知锐角、两边数值、所求边、点序表述。
- 可变维度：优先每题只改变 $a$ 相对 $h,c$ 的一个区间，使学生把解数变化与几何位置对应起来。
- 深化阶梯：复现本题两解 -> 同结构换数 -> 保持 $A,c$ 只改变 $a$ 得 $0/1/2$ 解 -> 隐去“可能有两个”提示 -> 要求反向设计一个两解 SSA。
- 允许的变换：保留特殊角和整洁勾股数组；例如固定 $A=30^\circ,c=8,h=4$，依次取 $a=3,4,5,8,9$ 观察 $0,1,2,1,1$ 解。
- 禁止的变换：同时引入复杂特殊角、非整洁根式、角度近似、多个未知参数；使用正弦定理或余弦定理；把已知角改成夹角后仍声称属于 SSA 两解。
- 表征切换：三角形图 <-> 射线与定长圆的两个交点 <-> 高足两侧的点序图 <-> $h,a,c$ 比较表。
- 包装方式：题面不说 SSA，只给“一个角、它的对边、另一边”，要求先判断三角形个数再求边。
- 近迁移例子：固定 $A=30^\circ,AB=8$，将 $BC$ 改为 $4$，说明为何只有一个直角三角形。
- 远迁移例子：给定 $A,c$，反向选择 $a$ 的范围，使题目恰有两个解，并用图解释。
- 反例/伪变式：已知 $A,AB,AC$ 的 SAS 题改几个数字；它由夹角直接确定唯一三角形，不保留 SSA 的核心关系。

## 八、计算复杂度预算
- 原题计算层级：两个特殊角三角比；一次勾股；一次线段和差；两个正值与三角形不等式检查。
- 允许小步上升到：保留一步简单根式；用计算器求一个锐角近似值以补全三角形，但应在“两个位置”已建立后进行。
- 禁止引入的计算负担：正弦定理、余弦定理；复杂二次方程；多层根式；多个角度近似连锁；把分类讨论埋在长计算中。
- 必须保留的可见支架：明确标出已知角；高 $BH$；点序 $A-C_1-H-C_2$；比较式 $BH<BC<AB$；两个线段式 $AH-HC$ 与 $AH+HC$。

## 九、推荐讲题任务包
- 建议的本轮教学入口：先不给完整三角形，只固定 $AB$ 与从 $A$ 出发的 $30^\circ$ 射线，问“距离 $B$ 为 $5$ 的点 $C$ 可能有几个”；再把两个交点分别连回 $B$。
- 本题讲解目标：学生能识别 SSA 不保证唯一；会从非已知角顶点作高；能用两个直角三角形算出两个解并解释点序。
- 不要直接讲的抽象话：不先背“SSA 两解公式”；不使用正弦定理、余弦定理；不把 $\pm$ 当作两解原因。
- 必须先问的问题：“已知角夹在两条已知边之间吗？从哪个非已知角顶点作高，才能让高直接可求？$BC$ 与高 $BH$ 谁更长？”
- 关键讲解顺序：识别 SSA -> 作 $BH$ -> 求 $BH,AH$ -> 比较 $4<5<8$ -> 勾股求 $HC$ -> 画出 $C_1,C_2$ -> 分别作差、作和 -> 验算。
- 最适合的具体数值例子：$A=30^\circ,AB=8,BC=5$；高为 $4$，另一直角三角形是 $3$-$4$-$5$，已知量全为整数，答案结构清楚。
- 讲到哪里停下来让学生回答：得到 $BH=4,AH=4\sqrt3,HC=3$ 后暂停，只让学生依据点序写出两个 $AC$，先不代数值。

## 十、推荐练题任务包
- 若卡在读题/入手动作，出什么题：只判断给定条件属于 SSS/SAS/SSA 中哪一种，并圈出“已知角的对边”。
- 若卡在建模或关系入口，出什么题：给 $A,c,a$，只要求作正确的高并写出 $h=c\sin A$，不继续计算。
- 若卡在求解和检查，出什么题：固定 $A=30^\circ,c=8$，依次给 $a=3,4,5,8$，只判 $0/1/2$ 解并解释。
- 若原题已稳，如何小步迁移：换字母、旋转图形或隐藏“两解”提示，仍保留特殊角与整洁勾股数。
- 若结构识别已稳，如何深化/抽象/包装：给定 $A,c$，要求写出 $a$ 取何范围时有两解；或让学生反向构造一组有两解的整数数据。
- 禁止出的跑偏变式：用正弦定理/余弦定理求解；在未学相关工具时要求复杂反三角函数近似；混入圆、相似或坐标几何新知识点。

## 十点五、推荐图形请求包（可选）
- 是否需要图：是；两解的本质是同一射线上点 $C$ 有两个位置，必须通过图展示点序与高足位置。
- 图形类型：`synthetic_geometry`
- 用图意图：`student_explanation`
