# 结构分析：AAS 从未给角的顶点作高

## 原题
在三角形 $ABC$ 中，已知 $\angle A=30^\circ$，$\angle C=45^\circ$，$AB=8$。不使用正弦定理、余弦定理，从唯一未作为已知角顶点的点 $B$ 向 $AC$ 作高 $BH$，解这个三角形：求 $AC$、$BC$ 和 $\angle B$。

图形文字描述：$H$ 在线段 $AC$ 上，$BH\perp AC$。辅助线 $BH$ 把原三角形分成 $\operatorname{Rt}\triangle ABH$ 与 $\operatorname{Rt}\triangle BCH$。本题给出两角 $A,C$ 及一条非夹边 $AB$，属于 AAS，三角形唯一确定。

## 一、题目场景
- 数学对象：一般三角形 $ABC$、从顶点 $B$ 向边 $AC$ 所作的高 $BH$、两个直角三角形 $ABH$ 与 $BCH$。
- 变量/参数：已知 $\angle A=30^\circ$、$\angle C=45^\circ$、$AB=8$；待求 $BH,AH,CH,AC,BC,\angle B$。
- 函数/图形：一般三角形通过一条高拆成两个共享直角边的直角三角形；分别使用锐角三角比。
- 已知条件：两角及一条非夹边，即 AAS 条件。
- 要求目标：只使用作高、锐角三角比、勾股关系和三角形内角和解出其余边角。

## 二、核心结构
### 2.1 表层信息
- 表面考点：用锐角三角比解决 AAS 型一般三角形。
- 题型功能：`application_modeling`
- 是否值得完整 structural analysis：是；理由：AAS 中作高顶点的选择有明确约束。高应从唯一未给角的顶点作出，使两个已知角分别落入两个直角三角形，而公共高成为两边计算的传递量。
- 一句话问题模式：已知两个角和一条非夹边时，从未给角的顶点作高，先在含已知边的直角三角形中求公共高，再用另一个已知角解第二个直角三角形。

### 2.2 结构表达
#### 判别条件表（概念辨析题用；不适用则写“无”）
| 检查项 | 本题判断 | 作用 |
|---|---|---|
| 是否为 AAS | 已知角为 $A,C$，两角夹边是 $AC$，但已知边是 $AB$ | 确认“两角及一条非夹边” |
| 从哪个顶点作高 | 从唯一未给角的顶点 $B$ 作高 | 两个已知角分别进入两个 Rt 三角形 |
| 先解哪个 Rt | $\triangle ABH$ 含已知斜边 $AB=8$ 与已知锐角 $A=30^\circ$ | 先求公共高 $BH$ 和投影 $AH$ |
| 第二个 Rt 的入口 | $\triangle BCH$ 已知锐角 $C=45^\circ$ 与对边 $BH=4$ | 求 $CH,BC$ |

- 必要条件：两个已知角必须是原三角形的内角，且和小于 $180^\circ$；已知边必须为正。
- 充分条件：两个内角及任意一条边唯一确定三角形；本题因已知边不是两角夹边而记为 AAS。
- 常见干扰项：把 $AB$ 看成两角 $A,C$ 的夹边；从已知角顶点 $A$ 或 $C$ 作高；先解没有已知边的 $\triangle BCH$；误把 AAS 写成可能两解。
- 最短检查动作：圈两个已知角，剩下的顶点就是作高顶点；再看哪一个直角三角形含已知边，决定先解它。

#### 情景量表（应用题用；不适用则写“无”）
| 量 | 类型/单位 | 题设关系 | 未知/已知 |
|---|---|---|---|
| $\angle A$ | 角度 | 左侧直角三角形中的锐角 | 已知 $30^\circ$ |
| $\angle C$ | 角度 | 右侧直角三角形中的锐角 | 已知 $45^\circ$ |
| $AB$ | 长度 | $\operatorname{Rt}\triangle ABH$ 的斜边 | 已知 $8$ |
| $BH$ | 长度 | 两个直角三角形的公共高 | 未知传递量 |
| $AH,CH$ | 长度 | $AC=AH+CH$ | 未知 |
| $AC,BC,\angle B$ | 边角 | 原三角形其余量 | 目标 |

#### 命题网络（所有题型都写；简单题写简版）
- P1（题设）：$\angle A=30^\circ$，$\angle C=45^\circ$，$AB=8$，已知边 $AB$ 不是两角 $A,C$ 的夹边。
- P2（构造）：从唯一未给角的顶点 $B$ 作 $BH\perp AC$ 于 $H$，得到 $\operatorname{Rt}\triangle ABH$ 与 $\operatorname{Rt}\triangle BCH$。
- P3（可推）：在 $\operatorname{Rt}\triangle ABH$ 中，$BH=AB\sin A=4$，$AH=AB\cos A=4\sqrt3$。
- P4（可推）：在 $\operatorname{Rt}\triangle BCH$ 中，$\tan C=BH/CH=1$，故 $CH=4$。
- P5（可推）：在 $\operatorname{Rt}\triangle BCH$ 中，$BC=BH/\sin C=4\sqrt2$。
- P6（可推）：$AC=AH+CH=4\sqrt3+4$。
- P7（可推）：$\angle B=180^\circ-\angle A-\angle C=105^\circ$。
- P8（目标）：得到三角形的其余两边与最后一个角，并完成边角关系验算。
- R1：P1 + P2 -> P3，方法：在含已知边 $AB$ 的第一个直角三角形中用正弦、余弦求高和投影。
- R2：P1 + P2 + P3 -> P4，方法：把公共高 $BH$ 传给第二个直角三角形，用正切求另一段投影。
- R3：P1 + P2 + P3 -> P5，方法：在第二个直角三角形中用正弦求斜边。
- R4：P3 + P4 -> P6，方法：用共线线段相加还原原三角形边 $AC$。
- R5：P1 -> P7，方法：用三角形内角和求第三角。
- R6：P5 + P6 + P7 -> P8，方法：内角和、大边对大角与三边范围联合验算。
- 目标：由 P1 经“选未给角顶点作高—先解含已知边的 Rt—公共高传递—解第二个 Rt—还原原边”推出 P8。

### 2.3 解题主链
```text
AAS：∠A、∠C 与非夹边 AB
-> 从唯一未给角顶点 B 作 BH⊥AC
-> Rt△ABH：由 AB、∠A 求 BH、AH
-> Rt△BCH：由 BH、∠C 求 CH、BC
-> AC=AH+CH
-> 内角和求 ∠B
-> 边角关系验算
```

### 2.4 模型标签
- model_id：triangle_aas_altitude_shared_height
- model_name：AAS 从未给角顶点作高、用公共高连接两个直角三角形
- configuration：已知两个角 $A,C$ 及非夹边 $AB$，从第三个顶点 $B$ 向两已知角的夹边 $AC$ 作高，要求其余边角。
- 可迁移方向：更换特殊角组合；把非夹边改为 $BC$ 并调整先解方向；改变所求为面积或高；从有图过渡到自主作图。
- 非同构边界：已知边若为两角夹边 $AC$，属于 ASA；只给一个角和两边属于 SAS 或 SSA；本模型不使用正弦定理、余弦定理。

## 二点五、知识点/模型锚点
- 建议讲义标题：AAS：从未给角的顶点作高，让公共高连接两个 Rt 三角形
- 知识点/模型名称：两角及非夹边条件下的作高解三角形模型。
- 核心公式/定理：$BH=AB\sin A$，$AH=AB\cos A$；$CH=BH/\tan C$，$BC=BH/\sin C$；$AC=AH+CH$；$B=180^\circ-A-C$。
- 使用条件：已知两个内角及一条非夹边；两个已知角之和小于 $180^\circ$；高从唯一未给角的顶点作出。
- 入口信号：题面同时给两个角和一条不连接这两个角顶点的边，且要求用锐角三角比解一般三角形。
- 易混边界：AAS 的已知边不是两已知角的夹边；作高顶点不是任取，而是唯一未作为已知角顶点的顶点；AAS 唯一解。
- 本题如何体现：从 $B$ 作高后，$AB=8$ 与 $A=30^\circ$ 先确定公共高 $BH=4$；$C=45^\circ$ 再把公共高转成 $CH=4,BC=4\sqrt2$。
- 可作为例题的结构层级：两个特殊角使两次三角比均有精确值，重点完全落在作高选择和计算顺序上。

## 三、关键转化
- 最关键的转化：把“两个分散在原三角形两端的已知角”分别放入两个直角三角形，并用公共高 $BH$ 把第一个直角三角形的结果传给第二个。
- 为什么降低计算量：第一个直角三角形有“斜边 + 锐角”，可直接求高；第二个直角三角形随即有“锐角 + 对边”，也可直接求其余边。
- 不转化时的低效路径：在一般三角形中寻找对边、邻边、斜边；从已知角顶点作高导致另一个已知角不能自然进入第二个直角三角形；或引入本课禁止的正弦定理、余弦定理。

## 四、标准路径骨架
1. 先做什么：辨认 AAS，圈出两个已知角 $A,C$，从剩余顶点 $B$ 作 $BH\perp AC$。
2. 再做什么：在含已知边 $AB=8$ 的 $\operatorname{Rt}\triangle ABH$ 中用 $\sin A,\cos A$ 求 $BH,AH$。
3. 建立什么关系：把公共高 $BH$ 作为 $\operatorname{Rt}\triangle BCH$ 中相对于 $\angle C$ 的对边。
4. 如何求解：用 $\tan C,\sin C$ 求 $CH,BC$，再由 $AC=AH+CH$ 和内角和求剩余量。
5. 需要检查什么：作高顶点是否正确；先解的直角三角形是否含已知边；内角和和最大边对最大角是否一致。

## 四点五、标准完整解与验算
- 关键交点/关键量：$H$ 是 $B$ 到 $AC$ 的垂足；$BH$ 是两个直角三角形共享的传递量。
- 面积/方程/关系式：$BH=8\sin30^\circ$，$AH=8\cos30^\circ$，$CH=BH/\tan45^\circ$，$BC=BH/\sin45^\circ$，$AC=AH+CH$。
- 完整求解过程：
  1. 作 $BH\perp AC$ 于 $H$。在 $\operatorname{Rt}\triangle ABH$ 中，$AB=8$ 为斜边，$\angle BAH=30^\circ$。
  2. $BH=8\sin30^\circ=4$，$AH=8\cos30^\circ=4\sqrt3$。
  3. 在 $\operatorname{Rt}\triangle BCH$ 中，$\angle BCH=45^\circ$，且 $BH=4$。由 $\tan45^\circ=BH/CH=1$，得 $CH=4$。
  4. $BC=BH/\sin45^\circ=4/(\sqrt2/2)=4\sqrt2$。
  5. $AC=AH+CH=4\sqrt3+4$。
  6. $\angle B=180^\circ-30^\circ-45^\circ=105^\circ$。
- 最终答案：$AC=4\sqrt3+4$，$BC=4\sqrt2$，$\angle B=105^\circ$。
- 排除值：一般 AAS 中两已知角均须大于 $0^\circ$ 且和小于 $180^\circ$，已知边须为正；本题满足。
- 退化情形：若两已知角之和大于或等于 $180^\circ$，不能组成非退化三角形。
- 验算：$30^\circ+45^\circ+105^\circ=180^\circ$；$AC=4\sqrt3+4\approx10.93$ 为最大边，对应最大角 $B=105^\circ$；$AB=8>BC=4\sqrt2\approx5.66$，对应 $C=45^\circ>A=30^\circ$。
- 本题最短可靠路径：从未给角顶点 $B$ 作高 -> 由 $AB,A$ 求公共高与左投影 -> 由公共高、$C$ 求右投影与斜边 -> 投影相加还原 $AC$。

## 五、出题人逻辑
- 诱导学生硬算的位置：题面有两个角，学生容易先求 $\angle B$ 后停住，或误以为一般三角形中的 $AB$ 可直接作为某个已知角的“斜边”。
- 真正的捷径：从未给角顶点作高，让两个已知角分别成为两个直角三角形的锐角，并使公共高成为传递量。
- 训练的可迁移能力：看到 AAS 后主动完成“选作高顶点—确定先解哪一个 Rt—用公共高连接第二个 Rt”的完整链条。

## 六、学生卡点预测
- 读题/入手动作卡点：分不清 AAS 与 ASA；没有检查已知边是否为两角夹边。
- 建模/关系入口卡点：从已知角顶点 $A$ 或 $C$ 作高；或不知道应先解含已知边 $AB$ 的 $\triangle ABH$。
- 求解/检查卡点：在 $\triangle BCH$ 中把 $BH$ 误认作 $\angle C$ 的邻边；忘记 $AC=AH+CH$；得到 $B=105^\circ$ 后因图形看起来像锐角三角形而擅自改答案。

## 七、变式原则
- 核心不变量：AAS 唯一解；从唯一未给角的顶点作高；先解含已知边的直角三角形；公共高传给第二个直角三角形。
- 表层特征：两个已知角、一条非夹边、连接未给角顶点与两已知角夹边的高。
- 可变维度：特殊角组合、已知非夹边位于左侧或右侧、最终所求、是否给出辅助线提示。
- 深化阶梯：照搬原题 -> 同结构换数 -> 交换已知非夹边 -> 隐去作高提示 -> 让学生比较 AAS 与 ASA 的作高结构。
- 允许的变换：优先使用 $30^\circ,45^\circ,60^\circ$ 的组合；改变 $AB$ 的整数倍；把所求改为面积、高或周长。
- 禁止的变换：把已知边换成夹边后仍标 AAS；从任意顶点作高；引入正弦定理或余弦定理；在基础题中加入非特殊角和连续近似。
- 表征切换：从图形题切换为文字条件，让学生先画出两角、非夹边并标出唯一未给角顶点。
- 包装方式：用测量两个视角与一条非夹边长度的情境包装，但保留同一作高结构。
- 近迁移例子：$\angle A=45^\circ,\angle C=60^\circ,AB=6$，从 $B$ 作高，依次求 $BH,AH,CH,BC,AC$。
- 远迁移例子：给出 AAS 与 ASA 两道相似题，只要求画出最有效的高并说明先解哪个直角三角形。
- 反例/伪变式：已知 $\angle A,\angle C$ 与夹边 $AC$；这是 ASA，已知边不属于任一作高后直角三角形的斜边，计算入口不同。

## 八、计算复杂度预算
- 原题计算层级：两次特殊角三角比、一次公共高传递、一次线段相加，全部保留精确根式。
- 允许小步上升到：把一个特殊角换成可查表的锐角并只做一次近似；或增加面积、周长中的一个目标。
- 禁止引入的计算负担：两个非特殊角同时近似、繁复分母有理化、正弦定理、余弦定理以及垂足外置分类。
- 必须保留的可见支架：标出两个已知角；第一批题提示“从未给角的顶点作高”；每步写明所在的直角三角形和公共高角色。

## 九、推荐讲题任务包
- 建议的本轮教学入口：先让学生圈 $A,C$ 两个已知角，再问“哪个顶点没有给角？”用答案 $B$ 直接确定作高顶点。
- 本题讲解目标：把 AAS 的作高选择和“公共高传递”变成稳定流程，而不是背诵一般三角形公式。
- 不要直接讲的抽象话：不说“两个角已知所以都能算”；必须具体指出第一个直角三角形为什么可解、第二个直角三角形何时才可解。
- 必须先问的问题：已知边是不是两角夹边？唯一未给角的顶点是谁？作高后哪个直角三角形同时含已知边和已知角？
- 关键讲解顺序：识别 AAS -> 从 $B$ 作高 -> 解 $ABH$ 得公共高 -> 解 $BCH$ -> 投影相加 -> 内角和与边角验算。
- 最适合的具体数值例子：$A=30^\circ,C=45^\circ,AB=8$；两次三角比均有精确值，清晰呈现 $BH=4$ 的传递作用。
- 讲到哪里停下来让学生回答：求出 $BH=4,AH=4\sqrt3$ 后暂停，让学生在第二个直角三角形中指出相对于 $C$ 的对边、邻边、斜边。

## 十、推荐练题任务包
- 若卡在读题/入手动作，出什么题：混合给出 AAS 与 ASA 条件，只判断已知边是否为两角夹边，并圈出未给角顶点。
- 若卡在建模或关系入口，出什么题：每题只要求画高、写出两个直角三角形，并标明先解哪一个及理由。
- 若卡在求解和检查，出什么题：保持 $30^\circ,45^\circ,60^\circ$，专练“已知边和第一角求高—高与第二角求边—投影相加”。
- 若原题已稳，如何小步迁移：把已知非夹边从 $AB$ 换成 $BC$，仍从 $B$ 作高，但改为先解右侧直角三角形。
- 若结构识别已稳，如何深化/抽象/包装：不给图和作高提示，让学生从 AAS 条件自行决定作高顶点与先解方向。
- 禁止出的跑偏变式：使用正弦定理、余弦定理；把 SSA 的解数分类混入 AAS；数字使三角比计算掩盖作高结构。
