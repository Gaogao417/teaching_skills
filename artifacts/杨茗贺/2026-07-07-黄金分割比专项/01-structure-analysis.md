# 结构分析：黄金分割比中的线段辨认与列式

## 原题
一条装饰线全长为 $30\text{ cm}$，按黄金分割分成较长、较短两段。求较长一段的长度。

本专项还需迁移到以下问法：已知全长、长段或短段中的一个量，求另外一个量或全长。

## 一、题目场景
- 数学对象：一条被分成较长段 $\text{长}$、较短段 $\text{短}$ 的线段，全长记为 $\text{全}$。
- 变量/参数：$\text{长}>\text{短}>0$，$\text{全}=\text{长}+\text{短}$，黄金分割比 $k=\dfrac{\sqrt5-1}{2}$，短段与全长之比 $q=\dfrac{3-\sqrt5}{2}$。
- 函数/图形：不涉及函数；线段图只起辨认长段、短段、全长的作用。
- 已知条件：$\dfrac{\text{短}}{\text{长}}=\dfrac{\text{长}}{\text{全}}=k$，$\dfrac{\text{短}}{\text{全}}=q$；原题给出 $\text{全}=30\text{ cm}$。
- 要求目标：原题求 $\text{长}$；专项迁移要求能根据已知量与所求量选择正确关系。

## 二、核心结构
### 2.1 表层信息
- 表面考点：黄金分割比、二次根式乘除、线段和差。
- 题型功能：`basic_skill`
- 是否值得完整 structural analysis：是；理由：计算本身不难，真正需要锁定的是三类线段的辨认、两两比值的方向与乘除选择。
- 一句话问题模式：确定已知边和所求边 -> 写出这两条边按“所求/已知”或“已知/所求”组成的比值 -> 根据已知量在分母还是分子决定乘或除。

### 2.2 结构表达
#### 判别条件表（概念辨析题用；不适用则写“无”）
- 必要条件：先确认题目中的线段是全长、长段还是短段。
- 充分条件：已知量与所求量能对应到 $\text{短}/\text{长}=k$、$\text{长}/\text{全}=k$、$\text{短}/\text{全}=q$ 中的一组。
- 常见干扰项：把“长短”误写成 $\text{长}/\text{短}$；混淆比值的方向；不看已知量在分子还是分母就决定乘除。
- 最短检查动作：写出“已知：___；所求：___”，写出这两条边的有向比值，再看已知量处在分子还是分母。

#### 情景量表（应用题用；不适用则写“无”）
| 量 | 类型/单位 | 题设关系 | 未知/已知 |
|---|---|---|---|
| 全长 $\text{全}$ | 长度 | $\text{长}/\text{全}=k$，$\text{短}/\text{全}=q$ | 原题已知 $30\text{ cm}$ |
| 长段 $\text{长}$ | 长度 | $\text{短}/\text{长}=k$，$\text{长}/\text{全}=k$ | 原题所求 |
| 短段 $\text{短}$ | 长度 | $\text{短}/\text{长}=k$，$\text{短}/\text{全}=q$ | 原题未直接求 |

#### 命题网络（所有题型都写；简单题写简版）
- P1（定义）：$k=\dfrac{\sqrt5-1}{2}$，$q=k^2=\dfrac{3-\sqrt5}{2}$，且 $0<q<k<1$。
- P2（定义）：黄金分割线段满足 $\dfrac{\text{短}}{\text{长}}=\dfrac{\text{长}}{\text{全}}=k$，并且 $\dfrac{\text{短}}{\text{全}}=q$。
- P3（题设）：原题 $\text{全}=30\text{ cm}$。
- P4（计算状态）：$\text{长}=kT$。
- P5（目标）：$\text{长}=15(\sqrt5-1)\text{ cm}$。
- P6（检查）：$\text{全}>\text{长}>\text{短}>0$。
- R1：P2 + P3 -> P4，方法：确定已知 $\text{全}$、所求 $\text{长}$，选取“长比全” $\text{长}/\text{全}=k$。
- R2：P1 + P4 -> P5，方法：代入并化简二次根式。
- R3：P3 + P5 -> P6，方法：用 $\text{短}=\text{全}-\text{长}$ 与 $0<k<1$ 检查大小关系。
- 目标：由 P1、P2、P3 依次推出 P4、P5，并用 P6 作数量级检查。

### 2.3 解题主链
```text
确定已知边和所求边 -> 确定两边的比值 -> 根据已知量的位置选择乘或除 -> 化简
```

### 2.4 模型标签
- model_id：`golden_ratio_segment_mapping`
- model_name：黄金分割线段三类量映射
- configuration：已知 $\text{全},\text{长},\text{短}$ 中一个量，求另一个量或全长。
- 可迁移方向：三类线段两两互求；更换生活情境不改变三组有向比值。
- 非同构边界：只给出“看起来协调”而未明确黄金分割；黄金矩形折叠辨认属于相邻迁移题，必须单独建立矩形宽长比，不能把矩形宽、长机械套成线段题的 $\text{短},\text{长},\text{全}$。

## 二点五、知识点/模型锚点
- 建议讲义标题：黄金分割比：先认三类线段，再选比例
- 知识点/模型名称：黄金分割线段中的 $\text{全},\text{长},\text{短}$ 对应关系。
- 核心公式/定理：$\dfrac{\text{短}}{\text{长}}=\dfrac{\text{长}}{\text{全}}=\dfrac{\sqrt5-1}{2}$，$\dfrac{\text{短}}{\text{全}}=\dfrac{3-\sqrt5}{2}$。
- 使用条件：题目明确给出黄金分割点或长、短两段成黄金分割比，并能判断哪一段较长。
- 入口信号：“全长”“较长一段”“较短一段”“短段和长段成黄金分割比”。
- 易混边界：“短长”指 $\text{短}/\text{长}$，“长全”指 $\text{长}/\text{全}$，两者都是 $k$；“短全”指 $\text{短}/\text{全}$，等于 $q$。比值有方向，倒数不能仍写成同一个数。
- 本题如何体现：已知 $\text{全}$、求 $\text{长}$，直接使用 $\text{长}/\text{全}=k$。
- 可作为例题的结构层级：例 1 用 $\text{全}$ 求 $\text{长}$，演示已知分母时乘比值；例 2 用 $\text{短}$ 求 $\text{全}$，演示已知分子时除以比值。

## 三、关键转化
- 最关键的转化：把情境里的线段名称统一标为 $\text{全},\text{长},\text{短}$，确定这两边的有向比值，再依据已知量在分子还是分母选择乘除。
- 为什么降低计算量：三组两两比值覆盖所有问法，不需要先求中间边，也不必背六套情境公式。
- 不转化时的低效路径：看到 $k$ 就盲目乘或除，再靠小数大小猜答案是否合理。

## 四、标准路径骨架
1. 确定已知边和要求的边：分别标为全长 $\text{全}$、长段 $\text{长}$、短段 $\text{短}$。
2. 确定这两条边的比值：从 $\text{短}/\text{长}=k$、$\text{长}/\text{全}=k$、$\text{短}/\text{全}=q$ 中选一条，并写清分子分母。
3. 确定乘还是除：已知分母就“分母 $\times$ 比值 = 分子”；已知分子就“分子 $\div$ 比值 = 分母”。

## 四点五、标准完整解与验算
- 关键交点/关键量：$\text{全}=30\text{ cm}$，$k=\dfrac{\sqrt5-1}{2}$。
- 面积/方程/关系式：不涉及面积；使用 $\dfrac{\text{长}}{\text{全}}=k$。
- 完整求解过程：
  \[
  \frac{\text{长}}{30}=\frac{\sqrt5-1}{2},
  \qquad
  \text{长}=30\times\frac{\sqrt5-1}{2}=15(\sqrt5-1)\text{ cm}.
  \]
- 最终答案：较长一段为 $15(\sqrt5-1)\text{ cm}$。
- 排除值：无方程增根；不能取负长度。
- 退化情形：若题目未能区分长段与短段，必须先依据文字条件澄清，不能仅凭示意图猜测。
- 验算：$k\approx0.618$，故 $\text{长}\approx18.54\text{ cm}$；短段约为 $11.46\text{ cm}$，满足 $30>18.54>11.46>0$。
- 本题最短可靠路径：写清 $\text{全}=30$、求 $\text{长}$，直接列 $\text{长}/\text{全}=k$。

## 五、出题人逻辑
- 诱导学生硬算的位置：把同一结构包装成花盘、书签、舞台、园林、照片或窗格，使学生误以为每种情境公式不同。
- 真正的捷径：删去情境词，只保留“已知是 $\text{全}/\text{长}/\text{短}$ 中哪个，所求是哪个”。
- 训练的可迁移能力：把文字对象映射到统一变量，再按量之间的直接关系列式。

## 六、学生卡点预测
- 读题/入手动作卡点：没有先区分全长、长段、短段，把“较远边”“留白区”等情境名称直接带入计算。
- 建模/关系入口卡点：没有掌握 $\text{短}/\text{长}$、$\text{长}/\text{全}$、$\text{短}/\text{全}$ 三组有向比值，或把 $\text{短}/\text{长}$ 误写成 $\text{长}/\text{短}$。
- 求解/检查卡点：已知量在分子时仍乘比值，得到比已知边更小的错误结果。

## 七、变式原则
- 核心不变量：$\text{短}/\text{长}=\text{长}/\text{全}=k$，$\text{短}/\text{全}=q$，以及统一的“已知分母乘、已知分子除”。
- 表层特征：线段名称、生活情境、单位和给定数值。
- 可变维度：一次只改变已知量/所求量、数值或情境表述中的一个。
- 深化阶梯：原题复现 -> 同结构换数 -> 已知/所求互换 -> 隐去“全长/长段/短段”标签、要求从语义辨认。
- 允许的变换：用全长求短段、用长段求全长、用短段求全长；整数换为简单小数。
- 禁止的变换：在基础六题中同时引入复杂相似、无理方程与多个未知点；黄金矩形折叠只作为独立收束题，不与基础计算题混讲。
- 表征切换：文字情境与标有 $\text{全},\text{长},\text{短}$ 的线段示意之间切换。
- 包装方式：花盘半径、书签分区、光带、路径、照片边长、窗格等。
- 近迁移例子：较短一段为 $6\text{ m}$，求较长一段和全长。
- 远迁移例子：由折叠等距与勾股定理推出矩形的宽长比，再辨认黄金矩形。
- 反例/伪变式：普通按 $2:3$ 分段不是黄金分割；黄金矩形长宽比的对象定义不同，不能不加说明直接套线段命名。

## 八、计算复杂度预算
- 原题计算层级：一次乘法与二次根式化简。
- 允许小步上升到：一次除以 $k$，并用共轭有理化；再做一次线段加减。
- 禁止引入的计算负担：复杂分式方程、多重根式、近似角度或跨知识点几何证明。
- 必须保留的可见支架：$\text{全},\text{长},\text{短}$ 标记、三组两两比值、已知分母乘/已知分子除。

## 九、推荐讲题任务包
- 建议的本轮教学入口：直接给出“全长 $30$，求长段”的例题，先让学生写“已知 $\text{全}$、所求 $\text{长}$”。
- 本题讲解目标：看到题目后能把线段归类，并选择包含已知量和所求量的关系。
- 不要直接讲的抽象话：不要先背六种问法的公式表，也不要先讲黄金分割的历史与审美背景。
- 必须先问的问题：题目给的是全长、长段还是短段？要求的又是哪一类？
- 关键讲解顺序：直接从真实数值例题进入；把“确定两边、确定比值、确定乘除”放在小贴士中，其中第一题的小贴士完整说清三组两两比值；正式解答只保留一条规范列式。例题前不放入口题、假例题或完整公式块。
- 最适合的具体数值例子：$\text{全}=30$；迁移例用 $\text{短}=6$。
- 讲到哪里停下来让学生回答：写出已知边和所求边后，停下来让学生写出这两边的有向比值，并判断已知量在分子还是分母。
