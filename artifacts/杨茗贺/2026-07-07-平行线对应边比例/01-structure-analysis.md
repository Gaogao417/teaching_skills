# 结构分析：平行线对应边比例求线段

## 原题
在三角形 $ACE$ 中，点 $B$ 在 $AC$ 上，点 $D$ 在 $AE$ 上，且 $BD\parallel CE$。已知 $AD=3$，$BC=5$，$AC=7$，求 $DE$。

## 一、题目场景
- 数学对象：大三角形 $ACE$，小三角形 $ABD$，其中 $BD\parallel CE$。
- 变量/参数：$AB,BC,AD,DE,AE,BD,CE$。
- 函数/图形：无函数；核心是平行线得到相似三角形。
- 已知条件：$BD\parallel CE$，$AD=3$，$BC=5$，$AC=7$。
- 要求目标：求 $DE$。

## 二、核心结构
### 2.1 表层信息
- 表面考点：平行线分线段成比例、相似三角形对应边。
- 题型功能：`basic_skill`
- 是否值得完整 structural analysis：是；这是“先定目标比、再改已知、最后乘已知量”的典型入口题。
- 一句话问题模式：要求平行线夹出的旁边小段，先把目标段和已知段配成比例，再把已知条件改写到这个比例上。

### 2.2 结构表达
#### 判别条件表（概念辨析题用；不适用则写“无”）
无。

#### 情景量表（应用题用；不适用则写“无”）
无。

#### 命题网络（所有题型都写；简单题写简版）
- P1（题设）：$B\in AC$，$D\in AE$，且 $BD\parallel CE$。
- P2（定理）：由 $BD\parallel CE$ 得 $\triangle ABD\sim\triangle ACE$。
- P3（可推）：对应边比例 $\dfrac{AD}{AE}=\dfrac{AB}{AC}=\dfrac{BD}{CE}$。
- P4（计算状态）：$AB=AC-BC=7-5=2$。
- P5（可推）：$\dfrac{AE}{AD}=\dfrac{AC}{AB}$，所以 $\dfrac{DE}{AD}=\dfrac{AE-AD}{AD}=\dfrac{AC-AB}{AB}=\dfrac{BC}{AB}$。
- P6（目标）：由 $\dfrac{DE}{AD}=\dfrac{5}{2}$ 和 $AD=3$ 求 $DE$。
- R1：P1 + 平行线定理 -> P2，方法：同位角/内错角相等得到两三角形相似。
- R2：P2 -> P3，方法：相似三角形对应边成比例。
- R3：P3 + P4 -> P5，方法：把整体比改写成差量比。
- R4：P5 + $AD=3$ -> P6，方法：目标比乘已知量。
- 目标：$DE=AD\cdot\dfrac{BC}{AB}=3\cdot\dfrac{5}{2}=\dfrac{15}{2}$。

### 2.3 解题主链
```text
BD∥CE -> △ABD∼△ACE -> 先定 DE:AD -> DE:AD=BC:AB -> AB=AC-BC -> DE
```

### 2.4 模型标签
- model_id：parallel-side-corresponding-segment-ratio
- model_name：过一边平行线的对应边比例
- configuration：已知平行线、同边整体/部分和一条边上的线段，要求另一条边上的相邻线段。
- 可迁移方向：换目标边、换已知边、给 $BD:CE$、反求 $AD$ 或 $DE$。
- 非同构边界：没有平行线、点不在同两边上、误把不共线或不共三角形的边当对应边。

## 二点五、知识点/模型锚点
- 建议讲义标题：平行线里的对应边：先找共线边，再找共三角形边
- 知识点/模型名称：平行线分线段成比例与相似三角形对应边。
- 核心公式/定理：若 $BD\parallel CE$，则 $\triangle ABD\sim\triangle ACE$，$\dfrac{AD}{AE}=\dfrac{AB}{AC}=\dfrac{BD}{CE}$；进一步 $\dfrac{DE}{AD}=\dfrac{BC}{AB}$。
- 使用条件：$B,D$ 分别在大三角形的两边上，且连接线 $BD$ 与第三边 $CE$ 平行。
- 入口信号：题面出现“一条内部线段平行于三角形的一边”，并且要求旁边被截出来的小段。
- 易混边界：$DE$ 不是直接和 $BC$ 同一个三角形里的对应边，但它们都是“整体减小段”得到的差量；先从 $AD$ 对 $AB$、$AE$ 对 $AC$ 入手。
- 本题如何体现：要求 $DE$，已知 $AD$，所以先想 $DE:AD$；再把它改成 $BC:AB$。
- 可作为例题的结构层级：单题型核心例题。

## 三、关键转化
- 最关键的转化：先定目标比 $DE:AD$，再利用相似把它改成 $BC:AB$。
- 为什么降低计算量：不用先求 $AE$，只要先补出 $AB=AC-BC$，再乘 $AD$。
- 不转化时的低效路径：先求相似比，再求 $AE$，最后做 $DE=AE-AD$，步骤更绕。

## 四、标准路径骨架
1. 先做什么：明确要求 $DE$，且已知 $AD$，所以目标比定为 $DE:AD$。
2. 再做什么：由 $BD\parallel CE$ 找对应边，$AD$ 对 $AB$，$AE$ 对 $AC$。
3. 建立什么关系：$DE:AD=BC:AB$。
4. 如何求解：由 $AB=AC-BC=2$，得 $DE:AD=5:2$，所以 $DE=3\cdot\frac52$。
5. 需要检查什么：点序必须是 $B$ 在 $AC$ 上、$D$ 在 $AE$ 上；对应边要来自共线边或共三角形边，不能乱配。

## 四点五、标准完整解与验算
- 关键交点/关键量：$AB=AC-BC=2$。
- 面积/方程/关系式：$DE:AD=BC:AB=5:2$。
- 完整求解过程：因为 $BD\parallel CE$，所以 $\triangle ABD\sim\triangle ACE$，对应边成比例。要求 $DE$，已知 $AD$，先看 $DE:AD$。由相似可得差量比 $DE:AD=BC:AB$。又 $AB=7-5=2$，所以 $DE:3=5:2$，$DE=\frac{15}{2}$。
- 最终答案：$DE=\dfrac{15}{2}$。
- 排除值：无。
- 退化情形：若 $AB=0$ 或 $B=C$，比例失效；本题 $AB=2>0$。
- 验算：若 $AD=3$，$DE=7.5$，则 $AE=10.5$；$AD:AE=3:10.5=2:7$，与 $AB:AC=2:7$ 一致。
- 本题最短可靠路径：$DE:AD=BC:AB$，补出 $AB$ 后直接乘。

## 五、出题人逻辑
- 诱导学生硬算的位置：学生容易直接找 $DE$ 和 $AD$，但不知道如何把它变成已知。
- 真正的捷径：把目标比转成同一组相似三角形里的对应边差量比。
- 训练的可迁移能力：看到过一边平行线，先找对应边；对应边一般是共线、共三角形的边。

## 六、学生卡点预测
- 读题/入手动作卡点：不知道先想 $DE:AD$，会盯着 $DE$ 单独发呆。
- 建模/关系入口卡点：知道平行线会比例，但把 $DE$ 直接和 $BD$ 或 $CE$ 乱配。
- 求解/检查卡点：忘记 $AB=AC-BC$，或把 $BC:AB$ 写反。

## 七、变式原则
- 核心不变量：一条内部线段平行于三角形一边，形成小三角形与大三角形相似；差量边与对应差量边成比例。
- 表层特征：三角形、内平行线、给三条线段求第四条。
- 可变维度：目标边可换成 $DE,AD,BC,AB$；已知比例可换成 $BD:CE$；数字可换成整除或简单分数。
- 深化阶梯：先给 $AC,BC,AD$ 求 $DE$；再给 $AB,BC,DE$ 求 $AD$；再给 $BD:CE$ 求差量边；最后反求整体或小段。
- 允许的变换：换点名、换左右边数据、给 $BD:CE$、反求 $AD$。
- 禁止的变换：取消平行关系、让点序不清、引入角平分线或中位线等额外模型。
- 表征切换：文字条件、图形条件、$BD:CE$ 比值条件。
- 包装方式：把 $AB$ 隐藏在 $AC-BC$ 中，把 $BC:AB$ 隐藏在 $BD:CE$ 中。
- 近迁移例子：$BD\parallel CE, AD=4, AC=15, BC=9$，求 $DE$。
- 远迁移例子：给 $BD:CE=5:8$ 和 $AD=15$，求 $DE$。
- 反例/伪变式：给 $BD$ 不平行 $CE$ 时不能使用本比例。

## 八、计算复杂度预算
- 原题计算层级：一次减法、一个比例乘法，可出现简单分数。
- 允许小步上升到：两步比例转换，或由 $BD:CE$ 转出 $BC:AB$。
- 禁止引入的计算负担：复杂根式、二次方程、多个嵌套相似三角形。
- 必须保留的可见支架：前三题保留“要求边/对应边”的提示。

## 九、推荐讲题任务包
- 建议的本轮教学入口：从“目标量 : 已知量”开始。
- 本题讲解目标：让学生会先定 $DE:AD$，再找对应边改写。
- 不要直接讲的抽象话：不要只说“相似所以比例”，要说清对应边怎么找。
- 必须先问的问题：要求哪条边？题里已经知道哪条同边上的边？它们各自对应到哪条边？
- 关键讲解顺序：先定目标比，再改已知，再乘已知量。
- 最适合的具体数值例子：原题 $AD=3,BC=5,AC=7$。
- 讲到哪里停下来让学生回答：停在 $DE:AD=BC:AB$，让学生自己补 $AB$ 并计算。

## 十、推荐练题任务包
- 若卡在读题/入手动作，出什么题：题面直接提示“要求边/对应边”的三角形平行线题。
- 若卡在建模或关系入口，出什么题：要求学生填写 $DE:AD=BC:AB$ 或 $AD:DE=AB:BC$。
- 若卡在求解和检查，出什么题：数字整除，重点练 $AB=AC-BC$。
- 若原题已稳，如何小步迁移：给 $BD:CE$，让学生先转出 $AB:AC$。
- 若结构识别已稳，如何深化/抽象/包装：反求 $AD$ 或整体 $AE$。
- 禁止出的跑偏变式：把平行线比例混成角平分线比例。
