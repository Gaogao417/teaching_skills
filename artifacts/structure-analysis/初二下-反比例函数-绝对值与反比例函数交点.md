# 结构分析：y=|x| 与 y=1/x 的交点

## 原题
y = |x| 和 y = 1/x 有几个交点？

## 一、题目场景
- 数学对象：绝对值函数、反比例函数
- 变量/参数：x（自变量）
- 函数/图形：y = |x|（V 形折线）、y = 1/x（双曲线）
- 已知条件：两个函数表达式
- 要求目标：求两条曲线交点个数（注：原题"焦点"应为"交点"之误）

## 二、核心结构
- 表面考点：函数图像交点
- 本质考点：绝对值函数的"分情况讨论"思想；反比例函数的定义域限制
- 一句话问题模式：联立含绝对值的函数方程，分象限讨论，注意定义域

## 三、关键转化
- 最关键的转化：|x| 的分类讨论 → x > 0 和 x < 0 分别联立
- 为什么降低计算量：避免凭直觉画图出错；直接列方程 + 检验定义域，比画图更可靠
- 不转化时的低效路径：只靠画草图猜，容易漏掉或画错第一象限交点位置

## 四、标准路径骨架
1. 先明确"交点" = 联立方程的解
2. 按 x > 0 和 x < 0 分两种情况
3. 每种情况下联立方程，解出 x
4. 回代求 y，写出交点坐标
5. 检验解是否在该情况的定义域内（尤其 x ≠ 0）

## 四点五、标准完整解与验算
- 关键交点/关键量：
- 面积/方程/关系式：

  **情况一：x > 0**（第一象限）
  - |x| = x，联立：x = 1/x
  - x² = 1，x = 1（x = -1 舍去，因为 x > 0）
  - y = 1
  - 交点：(1, 1) ✓

  **情况二：x < 0**（第二象限）
  - |x| = -x，联立：-x = 1/x
  - -x² = 1，即 x² = -1
  - 无实数解

  **x = 0**：y = 1/0 无意义，不讨论

- 完整求解过程：见上
- 最终答案：1 个交点，坐标为 (1, 1)
- 排除值：x = -1（在情况一中被 x > 0 排除）、x = 0（1/x 无定义）
- 退化情形：无
- 验算：将 (1,1) 代入 y=|x| → 1=|1|=1 ✓；代入 y=1/x → 1=1/1=1 ✓
- 本题最短可靠路径：分 x>0、x<0 两种情况联立方程

## 五、出题人逻辑
- 诱导学生硬算的位置：无，本题计算量很小
- 真正的捷径：直接分情况联立，无需画图
- 训练的可迁移能力：含绝对值的方程要分情况讨论；解完后要检验是否满足前提条件

## 六、学生卡点预测
- 基础薄弱学生：不理解"交点"对应"联立方程"；不会分 x>0 和 x<0 讨论；忘记 x=0 时 1/x 无意义
- 中等学生：能分情况但可能在情况二（x<0）算出 x²=-1 后不确定是否"无解"
- 较强学生：快速完成，但可以追问"画图验证"或变式

## 七、变式原则
- 必须保留：绝对值 + 反比例函数的组合；分情况讨论的核心动作
- 可以变化：换函数（如 y=|2x| 与 y=k/x，讨论 k 对交点个数的影响）；换条件（如求交点围成的面积）
- 容易跑偏：变式引入二次函数与绝对值联立（对初二过早）

## 八、计算复杂度预算
- 原题计算层级：极低（一次方程 + 平方）
- 允许小步上升到：引入参数 k（y=|x| 与 y=k/x 的交点讨论）
- 禁止引入的计算负担：复杂的二次方程、判别式讨论
- 必须保留的可见支架：分情况的书写格式

## 九、推荐讲题任务包
- 适合的学生档位：B 档（中等偏下，能听懂但需要引导）
- 本题讲解目标：建立"含绝对值要分情况"的意识；会写分情况讨论的完整格式
- 不要直接讲的抽象话："数形结合""分类讨论"
- 必须先问的问题："交点是什么意思？""|x| 在 x>0 和 x<0 时分别等于什么？"
- 关键讲解顺序：先问交点含义 → 再问 |x| 如何拆 → 分情况联立 → 检验 → 总结
- 最适合的具体数值例子：直接用原题
- 讲到哪里停下来让学生回答：分完情况后，让学生分别列方程

## 十、推荐练题任务包
- 若学生卡在 A/B 档，出什么题：y=|x| 与 y=2 的交点（纯绝对值分类练习，不涉及反比例）
- 若学生卡在 C 档，出什么题：y=|x| 与 y=1/(2x) 的交点（同构，换系数）
- 若学生卡在 D 档，出什么题：y=|x| 与 y=k/x，讨论 k>0、k<0 时交点个数
- 若学生达到 E/F 档，如何变式：求 y=|x|、y=1/x 与 x 轴围成的面积（综合）
- 禁止出的跑偏变式：y=|x²-1| 类复合绝对值；联立二次函数
