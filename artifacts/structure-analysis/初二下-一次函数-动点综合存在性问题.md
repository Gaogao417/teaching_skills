# 结构分析：一次函数与平行四边形中的动点综合

## 原题
直线 y = 2x + 6 的图像交 x 轴于点 A，交 y 轴于点 B，点 D 在 x 轴正半轴上，且 OD = 8/3·OA，以 AB、AD 为边作平行四边形 ABCD。
1. 求点 A、B 的坐标
2. 求出直线 BD 的函数解析式
3. 点 E 从 O 点出发，以每秒一个单位的速度沿 x 轴正方向移动，点 E 运动时间记为 t，点 F 是线段 BD 的中点，连接 EF 并延长，交直线 BC 于点 H，请直接写出：当 t 为何值时，四边形 ABHE 是平行四边形 / 等腰梯形
4. 点 Q 是直线 BD 上的一个动点，在 y 轴上找一点 P，连接 AP、AQ、PQ，当三角形 APQ 是以 PQ 为斜边的等腰直角三角形时，请直接写出点 P 的坐标

## 一、题目场景
- 数学对象：一次函数图像、坐标系中的平行四边形、动点问题、等腰直角三角形
- 变量/参数：t（时间参数，即点 E 的横坐标）、q（点 Q 在 BD 上的自由参数）、p（点 P 的纵坐标）
- 函数/图形：y = 2x + 6（已给定）、BD 所在直线（待求）、y = 6（直线 BC）
- 已知条件：直线 y = 2x + 6；OD = 8/3·OA；AB、AD 为平行四边形的边；E 的速度为 1 单位/秒；F 是 BD 中点
- 要求目标：(1) A、B 坐标 (2) BD 解析式 (3) t 取值使 ABHE 为平行四边形/等腰梯形 (4) P 的坐标使 APQ 为等腰直角三角形

## 二、核心结构
- 表面考点：一次函数求交点、待定系数法求直线、动点中的特殊四边形判定、等腰直角三角形存在性
- 本质考点：坐标法处理几何关系、一线三垂直构造全等三角形处理等腰直角三角形
- 一句话问题模式：**坐标系中基于一次函数的平行四边形构型 + 动点参数化 + 特殊图形存在性探究**

## 三、关键转化
- 最关键的转化：第(3)问中 H = (8−t, 6) 的显式参数化 + 第(4)问中"一线三垂直"构造全等三角形（AAS）
- 为什么降低计算量：第(3)问一旦参数化 H，平行四边形/等腰梯形退化为解一元方程；第(4)问用全等直接把几何条件化为简单的线段等式
- 不转化时的低效路径：第(3)问用斜率条件硬算 → 产生分式方程容易出错；第(4)问用距离公式 AP=AQ+AP⊥AQ → 两个方程联立消元 → 高次方程

## 四、标准路径骨架
1. 求交点 A、B：令 y=0 得 A，令 x=0 得 B
2. 由 OD = 8/3·OA 确定D坐标，再用平行四边形确定 C 坐标
3. 用待定系数法求 BD 直线
4. 第(3)问：参数化 E=(t,0)，求直线 EF 与 y=6 的交点 H，得到 H=(8−t,6)
5. 第(3)问平行四边形：AB=HE 向量相等 → t=5/2
6. 第(3)问等腰梯形：BH∥EA（恒成立）+ |AB|=|HE| + AB∦HE → t=11/2
7. 第(4)问：过 P、Q 向 x 轴作垂线，一线三垂直 → 互补角关系 → AAS 全等 → 线段等式 → 两种情况两个解

## 四点五、标准完整解与验算

### 前置：确定四边形 ABCD 各点坐标

**A、B 坐标（第1问）：**
- y=0: 0=2x+6, x=−3 → A(−3, 0)
- x=0: y=6 → B(0, 6)

**D 坐标：**
- OA = 3（A 到原点距离）
- OD = 8/3 × 3 = 8
- D 在 x 轴正半轴 → D(8, 0)

**C 坐标（平行四边形 ABCD，AB、AD 为邻边）：**
- AB⃗ = B−A = (3, 6)
- AD⃗ = D−A = (11, 0)
- C = A + AB⃗ + AD⃗ = (−3, 0) + (3, 6) + (11, 0) = (11, 6)
- 验证：DC⃗ = (3, 6) = AB⃗ ✓，BC⃗ = (11, 0) = AD⃗ ✓

**答案1：A(−3, 0)，B(0, 6)**

### BD 直线解析式（第2问）

B(0, 6)，D(8, 0)
斜率 k = (0−6)/(8−0) = −3/4
y = −3/4 x + 6

**答案2：y = −3/4 x + 6**

### 第(3)问完整求解

**参数化：**
- E = (t, 0)，t > 0
- F 是 BD 中点：F = ((0+8)/2, (6+0)/2) = (4, 3)

**直线 EF 与 BC 的交点 H：**
- 直线 BC：y = 6（水平线）
- 直线 EF 过 E(t,0) 和 F(4,3)，斜率 = 3/(4−t)
- 方程：y = 3/(4−t) · (x − t)
- 令 y = 6：6 = 3(x−t)/(4−t) → 2(4−t) = x−t → x = 8−t
- 统一公式（含 t=4 的垂直情况）：**H = (8−t, 6)**

**验证 t=4：EF 垂直（x=4），交 y=6 于 H(4,6)，代入公式 8−4=4 ✓**

**关键各点坐标汇总：**
| 点 | 坐标 |
|----|------|
| A | (−3, 0) |
| B | (0, 6) |
| H | (8−t, 6) |
| E | (t, 0) |

**各边向量：**
- AB⃗ = (3, 6)
- BH⃗ = (8−t, 0)  ← 恒为水平
- HE⃗ = (2t−8, −6)
- EA⃗ = (−3−t, 0) ← 恒为水平

**平行四边形判定（ABHE 顶点顺序 A→B→H→E）：**
需要 AB⃗ = HE⃗（或等价：AB ∥ HE 且 BH ∥ EA）
- BH ∥ EA：恒成立（均水平）
- AB⃗ = HE⃗：3 = 2t−8 且 6 = −6 → 第二式 6=−6 不成立！

**重新审视：四边形 ABHE 顶点顺序应为 AB ∥ EH 且 BH ∥ EA**
- HE⃗ 反向即 EH⃗ = (8−2t, 6)
- AB⃗ = EH⃗：(3, 6) = (8−2t, 6) → 8−2t = 3 → **t = 5/2**

验证：t=5/2 时，E=(5/2,0)，H=(11/2,6)
- AB⃗ = (3,6)，HE⃗ = (2·5/2−8, −6) = (−3,−6) = −AB⃗ → AB∥HE ✓
- BH⃗ = (11/2, 0)，EA⃗ = (−11/2, 0) → BH∥EA ✓

**答案3平行四边形：t = 5/2**

**等腰梯形判定：**
四边形 ABHE 中，BH ∥ EA 恒成立。只需：
- AB ∦ HE（即不是平行四边形，t ≠ 5/2）
- |AB| = |HE|

|AB| = √(9+36) = 3√5
|HE| = √((2t−8)² + 36)

3√5 = √((2t−8)²+36) → 45 = (2t−8)²+36 → (2t−8)² = 9
2t−8 = ±3 → t = 11/2 或 t = 5/2

排除 t=5/2（平行四边形），**t = 11/2**

验证：t=11/2 时，E=(11/2,0)，H=(5/2,6)
- AB⃗=(3,6)，HE⃗=(3,−6)，3/3≠6/−6，不平行 ✓
- |AB|=3√5，|HE|=√(9+36)=3√5，相等 ✓

**答案3等腰梯形：t = 11/2**

### 第(4)问完整求解

**条件：** △APQ 以 PQ 为斜边的等腰直角三角形 → 直角在 A，AP = AQ，∠PAQ = 90°

**一线三垂直全等法核心：** 过 P 向 x 轴作垂线（垂足 O），过 Q 向 x 轴作垂线（垂足 M），利用互补角关系 + AAS 全等

**设未知数：**
- P = (0, p) 在 y 轴上
- Q = (q, −3q/4 + 6) 在 BD：y = −3/4x + 6 上
- A = (−3, 0) 在 x 轴上
- 过 P 作 PO ⊥ x 轴于 O(0, 0)（因为 P 在 y 轴上，垂足恰好是原点）
- 过 Q 作 QM ⊥ x 轴于 M(q, 0)

**关键角关系（一线三垂直）：**
A、O、M 都在 x 轴上，且 O、M 都在 A 的右侧（因为 q > −3 且 0 > −3），所以射线 AO 和 AM 同向。
因为 ∠PAQ = 90°（直角在 A），所以 ∠PAO + ∠MAQ = 90°。
在 Rt△APO 中，∠PAO + ∠APO = 90°。
所以 **∠APO = ∠MAQ**。

**Case 1：P 在 x 轴下方（p < 0），Q 在 x 轴上方**

此时 QM = −3q/4 + 6 > 0（向上），PO = |p| = −p（向下）。
A 在 O 和 M 之间？不，A(−3) < O(0) < M(q)，A 在最左边。
射线 AO 方向向右 = 射线 AM 方向向右。

∠AOP = ∠QMA = 90°（均垂直于 x 轴）✓
∠APO = ∠MAQ（互补角关系）✓
AP = AQ（已知等腰条件）——这是两个三角形中，对应于 90° 角和 ∠APO/∠MAQ 的"非夹边"

由 AAS：**△APO ≅ △QMA**

对应边：AO = QM，PO = AM，AP = QA

AO = 3 → QM = 3 → −3q/4 + 6 = 3 → q = 4
PO = AM → −p = q + 3 = 7 → p = −7

P = (0, −7)，Q = (4, 3)

验证：APO 中 AO=3, PO=7, AP=√58；QMA 中 QM=3, MA=7, QA=√58 ✓
∠PAQ=90°：tan(∠PAO)=7/3, tan(∠MAQ)=3/7, 互余 ✓

**Case 2：P 在 x 轴上方（p > 0），Q 在 x 轴下方**

此时 PO = p（向上），QM = 3q/4 − 6 > 0（向下，即 |y_Q| = −y_Q）。
同样 A(−3) < O(0) < M(q)，射线 AO = 射线 AM 同向。

∠QAP = 90° → ∠QAO + ∠OAP = 90°
Rt△APO：∠OAP + ∠APO = 90° → ∠QAO = ∠APO
又 ∠QAO = ∠QAM（AO 和 AM 同向）
所以 **∠APO = ∠QAM**

考虑全等：∠AOP = ∠AMQ = 90°，∠APO = ∠QAM
对应关系变了：**△OAP ≅ △MAQ**（AAS）

对应边：OA = MQ，OP = AM，PA = AQ

OA = 3 → MQ = 3 → 3q/4 − 6 = 3 → q = 12
OP = AM → p = q + 3 = 15

P = (0, 15)，Q = (12, −3)

验证：OAP 中 OA=3, OP=15, AP=√234；MAQ 中 MQ=3, AM=15, AQ=√234 ✓
∠PAQ=90°：tan(∠QAO)=3/15=1/5, tan(∠OAP)=15/3=5, 互余 ✓

**答案4：P(0, −7) 或 P(0, 15)**

### 排除值与退化情形
- 第(3)问 t > 0（E 在 x 轴正半轴移动），t ≠ 8（H 与 B 重合）
- 第(4)问 q 为任意实数，两个解均合法

### 验算
- 平行四边形 ABCD：AB⃗=(3,6)=DC⃗ ✓，AD⃗=(11,0)=BC⃗ ✓
- 第(3)问 t=5/2：四顶点不共线、不退化 ✓
- 第(3)问 t=11/2：H 在 BC 线段上（x=5/2∈[0,11]）✓，四边形不自交 ✓
- 第(4)问两个解均满足 AP⊥AQ + AP=AQ + PQ²=2·AP² ✓

### 本题最短可靠路径
1. 交点法确定 A、B → OA=3 → OD=8 → D(8,0)
2. 平行四边形向量法：C = B + AD⃗ = (11,6)
3. 待定系数法求 BD
4. 第(3)问参数化 H=(8−t,6) → 向量相等/长度相等列方程
5. 第(4)问一线三垂直全等 → AAS 全等得 AO=QM, PO=AM → 两个简单方程直接解出

## 五、出题人逻辑
- 诱导学生硬算的位置：第(3)问如果不用参数化，而是每一步都展开斜率/截距 → 分式运算易错；第(4)问如果用距离公式联立 → 二元二次方程
- 真正的捷径：第(3)问发现 BH、EA 恒水平 → 直接看另一组对边；第(4)问一线三垂直将几何条件化为两组线段等式
- 训练的可迁移能力：坐标系中几何量的参数化表达、一线三垂直全等处理等腰直角三角形存在性

## 六、学生卡点预测
- 基础薄弱学生：求 D 坐标时 OD=8/3·OA 的 OA 取绝对值还是长度混淆；平行四边形 C 点坐标确定错误（向量加法方向搞反）
- 中等学生：第(3)问找不到 H 的参数表达式，或列向量方程时方向搞反；等腰梯形忘了排除平行四边形情形
- 较强学生：第(4)问想不到一线三垂直，陷入解二元二次方程；或只考虑一种情况（P 在上 / P 在下）漏解

## 七、变式原则
- 必须保留：一次函数与平行四边形的组合、动点参数化、一线三垂直全等处理等腰直角三角形
- 可以变化：直线方程、OD 与 OA 的比值、E 的运动路径（改到其他边上）、第(4)问直角顶点的位置
- 容易跑偏：把 OD/OA 比值改得过于复杂导致坐标难算；第(4)问改底边导致方程无整数解

## 八、计算复杂度预算
- 原题计算层级：一次方程为主，最高涉及一元二次方程（第(3)问 |AB|=|HE| 展开）
- 允许小步上升到：分数系数的一次方程组、含参数的线段长度公式
- 禁止引入的计算负担：高次方程、复杂根式化简、三角函数
- 必须保留的可见支架：坐标系、平行四边形各点坐标表、H 的参数表达式

## 九、推荐讲题任务包
- 适合的学生档位：初二下学期，已完成一次函数与全等三角形基础
- 本题讲解目标：掌握"坐标参数化+坐标条件"处理动点几何的存在性问题
- 不要直接讲的抽象话："数形结合"、"分类讨论"
- 必须先问的问题："四边形ABHE的四个顶点按什么顺序连接？""等腰梯形的哪两条边平行？""∠PAQ=90° 时，和旁边的直角有什么关系？"
- 关键讲解顺序：(1)(2)热身 → 第(3)问先参数化H → 平行四边形→等腰梯形 → 第(4)问先画辅助线（一线三垂直）→ 互补角关系 → AAS全等 → 两种情况
- 最适合的具体数值例子：画一个简单的直角三角形，过直角顶点作水平线，从另两个顶点向水平线作垂线，让学生看到两个直角三角形
- 讲到哪里停下来让学生回答：求出 H=(8−t,6) 后让学生自己列平行四边形条件；第(4)问画完辅助线后让学生找角关系

## 十、推荐练题任务包
- 若学生卡在 A/B 档：求一次函数与坐标轴交点、确定平行四边形第四顶点坐标
- 若学生卡在 C 档：动点参数化练习——给定 E(t,0)，求直线过定点交定直线交点坐标
- 若学生卡在 D 档：等腰直角三角形旋转构造——给定直角顶点和一条边的端点，求第三点坐标
- 若学生达到 E/F 档：将第(4)问变式为"直角在 P 或 Q"的情形；或将第(3)问的 E 改到 y 轴上运动
- 禁止出的跑偏变式：引入椭圆/双曲线轨迹；将 OD/OA 改为无理数比
