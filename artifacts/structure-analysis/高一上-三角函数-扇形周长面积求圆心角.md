# 结构分析：扇形周长面积求圆心角

## 原题
扇形周长为 6，面积为 2，求圆心角的弧度数。

## 一、题目场景
- 数学对象：扇形
- 变量/参数：半径 r，弧长 l，圆心角 θ
- 函数/图形：扇形（圆的一部分）
- 已知条件：周长 2r + l = 6，面积 (1/2)rl = 2
- 要求目标：求圆心角 θ 的弧度数

## 二、核心结构
- 表面考点：扇形的周长公式、面积公式
- 本质考点：由两个方程消元解三个未知数；由 r 和 l 求 θ 的关系 θ = l/r
- 一句话问题模式：两个含 r、l 的方程，消元解出 r，再求 θ = l/r

## 三、关键转化
- 最关键的转化：从面积得 rl = 4，从而 l = 4/r，代入周长方程消去 l，得到只含 r 的方程
- 为什么降低计算量：避免解三元方程组，把问题变成一元二次方程
- 不转化时的低效路径：先分别列出 θ = l/r 的等式，变成三个方程三个未知数，增加消元混乱

## 四、标准路径骨架
1. 写出两个公式：周长 2r + l = 6，面积 (1/2)rl = 2
2. 由面积方程得 rl = 4，即 l = 4/r
3. 代入周长方程，得 2r + 4/r = 6
4. 化为二次方程 2r² − 6r + 4 = 0，因式分解或求根公式解 r
5. 用 θ = l/r 求圆心角，两个解都合法

## 四点五、标准完整解与验算
- 关键交点/关键量：r = 1 或 r = 2，对应 l = 4 或 l = 2
- 面积/方程/关系式：

  **求解过程：**

  由面积 (1/2)rl = 2，得 rl = 4，即 **l = 4/r**。

  代入周长 2r + l = 6：

  2r + 4/r = 6

  两边乘 r（r > 0）：

  2r² − 6r + 4 = 0

  因式分解：2(r² − 3r + 2) = 0，即 **2(r − 1)(r − 2) = 0**

  所以 **r = 1 或 r = 2**。

  情形一：r = 1，l = 4/1 = 4，θ = l/r = 4/1 = **4**
  情形二：r = 2，l = 4/2 = 2，θ = l/r = 2/2 = **1**

- 最终答案：θ = 1 或 θ = 4（弧度）
- 排除值：无，r = 1 和 r = 2 均为正值，对应的 θ = 1 和 θ = 4 均为合法弧度
- 退化情形：r = 0 时扇形退化为一个点，但 r = 0 不是方程的解
- 验算：
  - r = 1, l = 4：周长 = 2×1 + 4 = 6 ✓，面积 = (1/2)×1×4 = 2 ✓
  - r = 2, l = 2：周长 = 2×2 + 2 = 6 ✓，面积 = (1/2)×2×2 = 2 ✓
- 本题最短可靠路径：面积 → rl = 4 → l = 4/r → 代入周长 → 解二次方程 → θ = l/r

## 五、出题人逻辑
- 诱导学生硬算的位置：先分别列三个方程，然后混乱消元
- 真正的捷径：面积先算出 rl = 4，直接把 l 用 r 表示，代入周长变一元方程
- 训练的可迁移能力：多方程消元策略；弧长公式 l = rθ 的灵活运用；一题两解的理解

## 六、学生卡点预测
- 基础薄弱学生：记不住扇形周长公式和面积公式；不知道 θ = l/r；不会由 rl = 4 写出 l = 4/r
- 中等学生：能列方程但消元出错；解二次方程时因式分解失败；算出一个 r 值就停下，漏掉另一个解
- 较强学生：快速完成，可以追问为什么有两个解（几何含义：小半径大弧度和 大半径小弧度都能满足条件）

## 七、变式原则
- 必须保留：扇形周长+面积两个公式联立的核心结构
- 可以变化：周长和面积的数值；改为已知半径和面积求圆心角；改为已知周长和圆心角求面积
- 容易跑偏：引入扇形内接三角形或组合图形（偏离消元核心）

## 八、计算复杂度预算
- 原题计算层级：低（一元二次方程，整数根，简单因式分解）
- 允许小步上升到：根非整数但可因式分解的二次方程；简单的分数运算
- 禁止引入的计算负担：判别式含根号、需要配方法才能分解的方程、复杂分数运算
- 必须保留的可见支架：扇形公式表（周长、面积、弧长）

## 九、推荐讲题任务包
- 适合的学生档位：B 档（能记公式但不会联立消元）
- 本题讲解目标：掌握扇形周长和面积公式的联立消元方法
- 不要直接讲的抽象话："消元""联立方程组"
- 必须先问的问题："扇形的周长包括哪几部分？""面积公式里的 (1/2)rl 是什么意思？"
- 关键讲解顺序：确认公式 → 面积得出 rl → 用 r 表示 l → 代入周长 → 解方程 → 求 θ
- 最适合的具体数值例子：直接用原题
- 讲到哪里停下来让学生回答：面积公式化简得到 rl = 4 后，让学生自己写出 l = 4/r 并代入
