# 结构分析：小于 π 的正整数集合

## 原题

写出小于 π 的正整数集合。

答案：{1, 2, 3}

解析：正整数从 1 开始。因为 3 < π < 4，所以小于 π 的正整数只有 1, 2, 3。易错点：不要把 0 放进"正整数"集合。

## 一、题目场景

- 数学对象：集合（有限集）、正整数、无理数 π
- 变量/参数：π ≈ 3.14159…
- 函数/图形：数轴上的区间定位
- 已知条件：上限为 π；元素必须是正整数
- 要求目标：用列举法写出满足条件的集合

## 二、核心结构

- 表面考点：集合的列举法表示
- 本质考点：正整数定义 + 无理数大小比较
- 一句话问题模式：给定一个实数上界，找出满足特定属性（正整数）的所有整数

## 三、关键转化

- 最关键的转化：将"小于 π"转化为"小于 4 的正整数"，再逐一筛选
- 为什么降低计算量：π 是无理数，无法精确列举，但 3 < π < 4 是基本事实，直接锁定范围
- 不转化时的低效路径：试图精确计算 π 的值再逐一比较，浪费时间且容易出错

## 四、标准路径骨架

1. 先明确"正整数"的定义：1, 2, 3, 4, 5, …
2. 再定位 π 的大小：3 < π < 4
3. 在数轴上找到 π 的位置
4. 取正整数中所有小于 π 的值
5. 用列举法写出集合 {1, 2, 3}，检查边界

## 四点五、标准完整解与验算

- 关键交点/关键量：π ≈ 3.14，与整数 3 和 4 的关系
- 面积/方程/关系式：正整数集 Z⁺ = {1, 2, 3, 4, …}；约束条件 x < π
- 完整求解过程：
  - 正整数从 1 开始：1, 2, 3, 4, 5, …
  - π ≈ 3.14，所以 3 < π < 4
  - 小于 π 的正整数：1, 2, 3
  - 写成集合：{1, 2, 3}
- 最终答案：{1, 2, 3}
- 排除值：0（非正整数）；4, 5, …（不小于 π）
- 退化情形：若上界改为正整数（如"小于 4 的正整数"），则答案相同 {1, 2, 3}；若上界 ≤ 1，则集合为空集 ∅
- 验算：1 < π ✓，2 < π ✓，3 < π ✓，4 > π ✗（排除正确）
- 本题最短可靠路径：正整数从 1 开始 → 3 < π < 4 → {1, 2, 3}

## 五、出题人逻辑

- 诱导学生硬算的位置：π 的近似值，诱导学生去"计算"π 而非直接利用 3 < π < 4
- 真正的捷径：π 的大小关系是常识（3 < π < 4），无需精确计算
- 训练的可迁移能力：无理数与整数的大小比较、正整数定义、集合列举法

## 六、学生卡点预测

- 基础薄弱学生：分不清正整数与自然数/非负整数的区别，可能把 0 写进去；不知道 π 大约是 3.14
- 中等学生：知道正整数定义，能写出 {1, 2, 3}，但不确定是否要包含边界（π 不是整数，所以不存在边界问题，但学生可能犹豫）
- 较强学生：本题没有难度，可能忽略"列举法"格式要求（如忘记写花括号）

## 七、变式原则

- 必须保留：正整数的定义约束、与某个常数的大小比较、列举法写集合
- 可以变化：上界改为其他无理数（如 √10、e）、上界改为整数、改为不小于某数的正整数、改为绝对值不等式
- 容易跑偏：引入复杂的无理数运算（如 π²），或同时要求用描述法表示，偏离"大小比较 + 集合列举"的核心

## 八、计算复杂度预算

- 原题计算层级：零计算，纯概念判断
- 允许小步上升到：简单无理数的大小估计（如 √2, √10, 2√3）
- 禁止引入的计算负担：无理数的精确运算、复杂的根式化简
- 必须保留的可见支架：数轴位置感、正整数序列

## 九、推荐讲题任务包

- 适合的学生档位：A-B 档（初学集合概念的学生）
- 本题讲解目标：建立"正整数"的精确含义，以及无理数与整数的快速比较
- 不要直接讲的抽象话：不要说"显然 π 在 3 和 4 之间"，要让学生在数轴上找到位置
- 必须先问的问题：
  1. "正整数有哪些？从几开始？"
  2. "π 大概是多少？在 3 和 4 之间吗？"
  3. "那么小于 π 的正整数，最大的一个是几？"
- 关键讲解顺序：正整数定义 → π 的大小估计 → 逐一筛选 → 列举法格式
- 最适合的具体数值例子：先问"小于 4 的正整数有哪些"，再过渡到"小于 π 的"
- 讲到哪里停下来让学生回答：确认 π 的大致位置后，让学生自己列举

## 十、推荐练题任务包

- 若学生卡在 A/B 档，出什么题：
  1. 写出小于 5 的正整数集合
  2. 写出小于 3.5 的正整数集合
  3. 写出大于 2 且小于 6 的正整数集合
- 若学生卡在 C 档，出什么题：
  1. 写出小于 √10 的正整数集合（√10 ≈ 3.16）
  2. 写出小于 e 的正整数集合（e ≈ 2.72）
- 若学生卡在 D 档，出什么题：
  1. 写出小于 π² 的正整数集合（π² ≈ 9.87）
  2. 写出绝对值小于 3 的正整数集合
- 若学生达到 E/F 档，如何变式：
  1. 写出满足 |x - 2| < 1 的正整数集合
  2. 写出使不等式 n < √(n+10) 成立的正整数 n 的集合
- 禁止出的跑偏变式：涉及无理数四则运算后比较大小（偏离本题核心）
