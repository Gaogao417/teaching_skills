# 结构分析：y = ln(x - 1) 的定义域

## 原题
求函数 y = ln(x - 1) 的定义域。
答案：(1, +∞)

## 一、题目场景
- 数学对象：对数函数 ln(x - 1)，复合函数（内层为一次函数 x - 1）
- 变量/参数：自变量 x
- 函数/图形：y = ln u，其中 u = x - 1
- 已知条件：函数表达式 y = ln(x - 1)
- 要求目标：求定义域，即所有使函数有意义的 x 的集合

## 二、核心结构
- 表面考点：求对数函数的定义域
- 本质考点：对数函数真数必须大于零 → 列不等式 → 解一元一次不等式
- 一句话问题模式：给一个含对数的函数，让真数 > 0，解出 x 的范围。

## 三、关键转化
- 最关键的转化：将"求定义域"转化为"令真数 > 0，解不等式 x - 1 > 0"
- 为什么降低计算量：这是一元一次不等式，一步移项即可，无需复杂运算
- 不转化时的低效路径：试图画图或代入数值试算，效率低且容易遗漏

## 四、标准路径骨架
1. 识别函数中的对数结构：y = ln(x - 1)，对数符号 ln 后面跟着的是真数
2. 写出真数大于零的条件：x - 1 > 0
3. 解不等式：x > 1
4. 用区间表示法定义域：(1, +∞)
5. 检查：x = 1 代入得 ln(0) 无意义，x > 1 时 x - 1 > 0 有意义

## 四点五、标准完整解与验算
- 关键量：真数 u = x - 1
- 关系式：u > 0，即 x - 1 > 0
- 完整求解过程：
  1. 对数函数 y = ln(x - 1) 要求真数 x - 1 > 0
  2. 解不等式：x - 1 > 0 → x > 1
  3. 定义域为 (1, +∞)
- 最终答案：(1, +∞)
- 排除值：x = 1（ln(0) 无意义），x < 1（真数为负，对数无定义）
- 退化情形：无
- 验算：取 x = 2 → y = ln(1) = 0，有意义 ✓；取 x = 0 → y = ln(-1)，无意义 ✓；取 x = 1 → y = ln(0)，无意义 ✓
- 本题最短可靠路径：真数 > 0 → x - 1 > 0 → x > 1 → (1, +∞)

## 五、出题人逻辑
- 诱导学生硬算的位置：可能让学生纠结对数函数的图像或性质，而非直接列不等式
- 真正的捷径：认出"求对数函数定义域 = 令真数 > 0"这一个动作
- 训练的可迁移能力：从任意复合对数函数中提取真数、列不等式、解不等式的能力

## 六、学生卡点预测
- 基础薄弱学生：
  - 不理解"定义域"的含义
  - 不知道对数函数对真数的要求（真数 > 0）
  - 解不等式 x - 1 > 0 出错（写成 x > -1 或 x < 1）
- 中等学生：
  - 能列出 x - 1 > 0 但写答案时忘记用区间表示法
  - 对 ln(0) 是否有意义模糊
  - 定义域写成 [1, +∞) 或 {x | x > 1} 但不会转区间
- 较强学生：
  - 能快速完成，但面对 ln(x² - 1) 等复合结构时可能忘记真数整体 > 0 的条件
  - 不等式求解无障碍

## 七、变式原则
- 必须保留：对数函数结构、"真数 > 0"这一核心转化
- 可以变化：内层函数从 x - 1 变为更复杂的一次函数（如 2x - 3）、二次函数（如 x² - 4）、分式函数（如 (x+1)/(x-2)）等
- 容易跑偏：变成求值域或求反函数定义域，或引入对数运算法则（如 ln a + ln b）使题目偏离"求定义域"的核心

## 八、计算复杂度预算
- 原题计算层级：一元一次不等式（最低层级）
- 允许小步上升到：一元二次不等式（如 ln(x² - 4)）、分式不等式（如 ln((x+1)/(x-2))），但每步计算量仍保持简单
- 禁止引入的计算负担：复合对数运算（如 ln(ln(x))）、对数方程求解、复杂的因式分解
- 必须保留的可见支架："真数 > 0"这一关键不等式的明确写出

## 九、推荐讲题任务包
- 适合的学生档位：A~D 档（从完全不会到基本掌握）
- 本题讲解目标：建立"求定义域 → 令真数 > 0 → 解不等式"的三步操作链
- 不要直接讲的抽象话："定义域是使函数有意义的自变量的取值集合"（太抽象），应替换为"把 x 代进去算一算，看看什么数让这个式子算得出来"
- 必须先问的问题：
  - "ln 这个符号你认识吗？ln(2) 大概是多少？"
  - "ln(0) 算得出来吗？ln(-1) 呢？"
  - "那要让 ln(括号里的东西) 有意义，括号里必须满足什么条件？"
- 关键讲解顺序：
  1. 先确认学生知道 ln(正数) 有意义，ln(0) 和 ln(负数) 无意义
  2. 再让学生把"括号里的东西 > 0"写成不等式
  3. 最后解不等式，写成区间
- 最适合的具体数值例子：ln(1) = 0, ln(e) = 1, ln(0) = 报错, ln(-1) = 报错
- 讲到哪里停下来让学生回答：
  - 写出 x - 1 > 0 之后停下来，让学生解出 x
  - 解出 x > 1 之后停下来，让学生用区间表示

## 十、推荐练题任务包
- 若学生卡在 A/B 档，出什么题：
  - 求 y = ln(x + 2) 的定义域
  - 求 y = ln(2x) 的定义域
  - 求 y = log₅(x - 3) 的定义域
- 若学生卡在 C 档，出什么题：
  - 求 y = ln(3x - 6) 的定义域（内层系数不为 1）
  - 求 y = lg(x² - 1) 的定义域（引入二次真数）
- 若学生卡在 D 档，出什么题：
  - 求 y = √(ln(x - 1)) 的定义域（叠加根号条件：ln(x-1) ≥ 0 且 x-1 > 0）
  - 求 y = ln(2x - 3) + ln(x + 1) 的定义域（真数各自 > 0，取交集）
- 若学生达到 E/F 档，如何变式：
  - 求 y = ln(x² - 2x - 3) 的定义域（因式分解 + 二次不等式）
  - 求 y = ln((x - 1)/(x + 2)) 的定义域（分式真数 > 0）
- 禁止出的跑偏变式：
  - 求 y = ln(x) 的反函数（偏离"求定义域"核心）
  - 解方程 ln(x - 1) = 2（变成了方程求解）
  - 含参数的讨论题（如 y = ln(ax - 1) 对 a 的讨论，计算负担过重）
