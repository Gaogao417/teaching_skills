# 结构分析：正实数 a,b 求最值

## 原题
正实数 $a, b$ 满足 $a + \dfrac{1}{2b} = 1$，求 $\dfrac{a}{b}$ 的最大值。

## 一、题目场景
- 数学对象：正实数对 $(a, b)$，满足线性约束条件
- 变量/参数：$a$（正实数），$b$（正实数，且由条件可推出 $b > 1/2$）
- 函数/图形：目标函数 $f = a/b$，在约束条件下的最值
- 已知条件：$a > 0$，$b > 0$，$a + 1/(2b) = 1$
- 要求目标：$a/b$ 的最大值

## 二、核心结构
- 表面考点：约束条件下求分式最值
- 本质考点：消元转化为一元二次函数，利用配方法求最值
- 一句话问题模式：**约束消元 → 一元二次配方 → 在限定区间内求最值**

## 三、关键转化
- 最关键的转化：将双变量问题消元为一元函数。由 $a = 1 - 1/(2b)$ 代入 $a/b$，得 $a/b = 1/b - 1/(2b^2)$。令 $t = 1/b$（$0 < t < 2$），化为 $t - t^2/2$。
- 为什么降低计算量：双变量消为单变量后，只需对二次函数配方即可，无需处理偏导或拉格朗日乘数。
- 不转化时的低效路径：直接对双变量用不等式放缩（如 AM-GM），容易走弯路或计算复杂；或用拉格朗日乘数法，超出高中范围。

## 四、标准路径骨架
1. 先做什么：由约束条件解出 $a = 1 - 1/(2b)$，明确 $b > 1/2$（因 $a > 0$）。
2. 再做什么：将 $a$ 的表达式代入目标 $a/b$，化简为 $1/b - 1/(2b^2)$。
3. 建立什么关系：令换元 $t = 1/b$，则 $0 < t < 2$，目标化为 $g(t) = t - t^2/2$。
4. 如何求解：对 $g(t)$ 配方：$g(t) = 1/2 - (t-1)^2/2$，当 $t = 1$（在定义域内）时取最大值 $1/2$。
5. 需要检查什么：验证 $t = 1$ 在 $(0, 2)$ 内（是），回代 $b = 1$，$a = 1 - 1/2 = 1/2$，$a/b = 1/2$，均满足正实数约束。

## 四点五、标准完整解与验算
- 关键量：$b$ 的范围 $b > 1/2$，换元 $t = 1/b \in (0, 2)$
- 目标表达式：$a/b = 1/b - 1/(2b^2) = t - t^2/2$
- 完整求解过程：
  1. 由 $a + 1/(2b) = 1$ 得 $a = 1 - 1/(2b)$
  2. 因 $a > 0$，故 $1 - 1/(2b) > 0$，即 $b > 1/2$
  3. $a/b = (1 - 1/(2b))/b = 1/b - 1/(2b^2)$
  4. 令 $t = 1/b$，则 $0 < t < 2$，目标为 $g(t) = t - t^2/2$
  5. $g(t) = -\frac{1}{2}(t^2 - 2t) = -\frac{1}{2}[(t-1)^2 - 1] = \frac{1}{2} - \frac{(t-1)^2}{2}$
  6. 因 $(t-1)^2 \geq 0$，故 $g(t) \leq 1/2$，等号当 $t = 1$ 时取到
  7. $t = 1 \Rightarrow b = 1$，$a = 1 - 1/2 = 1/2$，验证 $a + 1/(2b) = 1/2 + 1/2 = 1$ ✓
- 最终答案：$a/b$ 的最大值为 $1/2$
- 排除值：$b$ 不能取 $1/2$（此时 $a = 0$，不满足正实数），$b$ 不能趋近无穷（$t \to 0$，$g(t) \to 0$）
- 退化情形：$a \to 0^+$ 时 $b \to 1/2^+$，$a/b \to 0$；$b \to +\infty$ 时 $a \to 1$，$a/b \to 0$。目标函数在边界趋近0，内部取最大值。
- 验算：$a = 1/2, b = 1$ 代入原条件：$1/2 + 1/(2 \times 1) = 1/2 + 1/2 = 1$ ✓；$a/b = 1/2$ ✓
- 本题最短可靠路径：消元求出 $a = 1 - 1/(2b)$ → 代入化简 $a/b = 1/b - 1/(2b^2)$ → 换元 $t = 1/b$ → 配方求二次函数最值 → 检验端点和最值点

## 五、出题人逻辑
- 诱导学生硬算的位置：直接尝试用不等式（AM-GM 等）放缩求 $a/b$ 的最大值，容易因形式不对称而卡住。
- 真正的捷径：消元 + 换元 + 配方，是最基本也最可靠的高中方法。
- 训练的可迁移能力：（1）双变量约束条件下的消元意识；（2）分式化为二次函数的换元技巧；（3）定义域验证意识。

## 六、学生卡点预测
- 基础薄弱学生：不会从 $a + 1/(2b) = 1$ 解出 $a$；不会代入化简 $a/b$；不知道求最值用什么方法。
- 中等学生：能消元化简，但不做换元，面对 $1/b - 1/(2b^2)$ 不知道如何求最值；或换元后忘记确定 $t$ 的范围（$0 < t < 2$）。
- 较强学生：能完成全过程，但可能忽略对 $t = 1$ 是否在定义域内的验证，或验算环节不够严谨。

## 七、变式原则
- 必须保留：双变量约束条件 + 求某个分式的最值 + 消元配方的基本框架
- 可以变化：约束条件的具体形式（如 $a + k/b = c$）、目标表达式（如 $ab$、$a^2 + b^2$）、变量个数（加第三个变量但仍消元）、最值方向（最小值）
- 容易跑偏：引入微积分求导方法（超出高中大纲）、使用拉格朗日乘数法、约束条件过于复杂导致消元困难

## 八、计算复杂度预算
- 原题计算层级：分式化简 + 二次配方，约3-4步运算
- 允许小步上升到：分母含更高次项（如 $b^2$ 项出现），或换元后需要更复杂的配方
- 禁止引入的计算负担：三次及以上多项式、根式运算、三角换元、微积分
- 必须保留的可见支架：消元后的目标函数化简过程、换元后的定义域标注、配方的中间步骤

## 九、推荐讲题任务包
- 适合的学生档位：B-D 档（有基础代数运算能力，但对最值方法不熟练的学生）
- 本题讲解目标：建立"约束消元→一元函数→配方求最值"的完整动作链
- 不要直接讲的抽象话："数形结合思想""函数思想"——应具体化为"把两个变量变成一个变量""凑成完全平方的形式"
- 必须先问的问题："条件里有几个变量？""能不能把 $a$ 用 $b$ 表示出来？""$a/b$ 代入后变成什么样子？"
- 关键讲解顺序：消元 → 化简 → 换元（重点讲为什么换元） → 配方 → 验证
- 最适合的具体数值例子：取 $b = 1$ 时 $a = 1/2$，$a/b = 1/2$；取 $b = 3/4$ 时 $a = 1 - 2/3 = 1/3$，$a/b = 4/9 \approx 0.44 < 0.5$；取 $b = 2$ 时 $a = 1 - 1/4 = 3/4$，$a/b = 3/8 = 0.375 < 0.5$
- 讲到哪里停下来让学生回答：换元后问"现在要求 $t - t^2/2$ 的最大值，用什么方法？"；配方完成后问"最大值在 $t$ 等于多少时取到？这个 $t$ 值在范围内吗？"

## 十、推荐练题任务包
- 若学生卡在 A/B 档，出什么题：纯一元二次函数配方求最值（无约束条件），如"求 $2x - x^2$ 在 $0 < x < 3$ 上的最大值"
- 若学生卡在 C 档，出什么题：简单的约束消元题，如"已知 $x + y = 1$，$x, y > 0$，求 $xy$ 的最大值"
- 若学生卡在 D 档，出什么题：类似本题但约束条件更简单的分式最值，如"正实数 $a, b$ 满足 $a + 1/b = 2$，求 $a/b$ 的最大值"
- 若学生达到 E/F 档，如何变式：增加约束条件的复杂性，如"正实数 $x, y$ 满足 $2x + 3/y = 5$，求 $x/y$ 的最大值"；或改变目标为 $ab$ 的最大值
- 禁止出的跑偏变式：含三角函数的约束、微积分方法、拉格朗日乘数法、超过三个变量的约束
