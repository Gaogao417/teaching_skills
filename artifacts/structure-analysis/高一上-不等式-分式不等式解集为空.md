# 结构分析：分式不等式解集为空

## 原题

不等式 (2x + a)/(x − 1) < 1 的解集为空，求 a 的值。

## 一、题目场景

- 数学对象：分式不等式（含参数）
- 变量/参数：x 为未知数，a 为待定参数
- 函数/图形：f(x) = (2x + a)/(x − 1) − 1，本质上是 y = (x + a + 1)/(x − 1) < 0 的图像在 x 轴下方的部分
- 已知条件：不等式的解集为空集
- 要求目标：求参数 a 的值

## 二、核心结构

- 表面考点：分式不等式的求解
- 本质考点：分式不等式 f(x)/g(x) < 0 解集为空的条件——分子分母零点重合
- 一句话问题模式：给定 f(x)/g(x) < 0 解集为空，反推参数使得分子分母零点重合。

## 三、关键转化

- 最关键的转化：移项通分，把 (2x + a)/(x − 1) < 1 变为 (x + a + 1)/(x − 1) < 0。此时分子零点 x = −(a+1)，分母零点 x = 1，问题变成"两个零点何时重合"。
- 为什么降低计算量：移项后只需令分子零点 = 分母零点解一个方程，无需展开分式或对 x 求解。
- 不转化时的低效路径：展开 2x + a < x − 1（忘记分母正负讨论），陷入分母 x − 1 的正负分类讨论，最终只能解出具体的 x 范围而无法利用"解集为空"这一全局条件。

## 四、标准路径骨架

1. 先做什么：移项，把所有项移到不等式同侧，通分化为 f(x)/g(x) < 0 的标准形式。
2. 再做什么：识别分子零点和分母零点。
3. 建立什么关系：分式 f(x)/g(x) < 0 的解集结构——如果两个零点不同，解集一定非空（取两个零点之间的区间）；要让解集为空，必须两个零点重合。
4. 如何求解：令分子零点 = 分母零点，解出 a。
5. 需要检查什么：代入 a 的值回原不等式，验证解集确实为空。

## 四点五、标准完整解与验算

- 关键交点/关键量：分子零点 x₀ = −(a+1)，分母零点 x₁ = 1
- 面积/方程/关系式：

  移项：(2x + a)/(x − 1) − 1 < 0

  通分：(2x + a − x + 1)/(x − 1) < 0

  化简：(x + a + 1)/(x − 1) < 0

- 完整求解过程：

  **关键推理**：分式 (x + a + 1)/(x − 1) < 0，分子零点为 x = −(a+1)，分母零点为 x = 1。

  若 −(a+1) ≠ 1（即 a ≠ −2），则两个零点将数轴分成三段，分式在其中一段取负值，解集非空。

  因此要让解集为空，必须 −(a+1) = 1，解得 a = −2。

- 最终答案：a = −2

- 排除值：无。a = −2 不是分母零点的排除值，分母零点是 x = 1。

- 退化情形：a = −2 时，(x + a + 1)/(x − 1) = (x − 1)/(x − 1) = 1（x ≠ 1 时），不等式 1 < 0 不成立，解集为空。验证通过。

- 验算：a = −2 代入原不等式：(2x − 2)/(x − 1) < 1，x ≠ 1 时左边 = 2，2 < 1 不成立，解集为空。正确。

- 本题最短可靠路径：移项通分 → 识别两个零点 → 两个零点不同则解集非空 → 令两零点相等解 a → 代入验证。

## 五、出题人逻辑

- 诱导学生硬算的位置：直接展开 2x + a < x − 1，忽略分母正负，得到 x < −a − 1，然后发现与"解集为空"矛盾，陷入困惑。
- 真正的捷径：移项通分后看分子分母零点关系，发现解集为空等价于零点重合。
- 训练的可迁移能力：分式不等式解集结构的整体理解——两个零点决定解集形态，而不是对每个 x 穷举。

## 六、学生卡点预测

- 基础薄弱学生：不会移项通分；可能直接交叉相乘而不讨论分母正负；看到"解集为空"不知道从何入手。
- 中等学生：能移项通分，但不理解"解集为空"与"零点重合"的关系，可能去解 x 的范围然后发现无法匹配"解集为空"的条件。
- 较强学生：能完成推理，但在"为什么两个零点不同时解集一定非空"这一步缺乏严格的论证，可能只是直觉判断。

## 七、变式原则

- 必须保留：分式不等式 + 参数 + 解集为特殊集合（空集或全集）的结构
- 可以变化：不等号方向（> 0）、解集条件（解集为 R）、分子次数（二次）、参数位置（分子和分母中都有参数）
- 容易跑偏：变成纯计算题（给具体 a 值求解集）——失去了反推参数的思维训练价值；引入高次不等式——计算负担过大

## 八、计算复杂度预算

- 原题计算层级：一次分式化简 + 一次方程求解（线性）
- 允许小步上升到：二次分子（如 (x² + ax + b)/(x − 1) < 0），解集为空时需要判别式为负或零点重合
- 禁止引入的计算负担：三次以上多项式、根式、绝对值分式、需要分类讨论3种以上情况的分式
- 必须保留的可见支架：移项通分的步骤痕迹，分子分母零点的标注

## 九、推荐讲题任务包

- 适合的学生档位：B/C/D档（能看懂分式但不会利用解集条件）
- 本题讲解目标：让学生理解"分式不等式解集由分子分母零点决定"这一核心结构
- 不要直接讲的抽象话："零点重合"、"解集为空的条件"
- 必须先问的问题："(x+3)/(x−1) < 0 的解集是什么？"——用具体数字让学生看到两个零点不同时解集非空
- 关键讲解顺序：
  1. 先用具体例子 (x+3)/(x−1) < 0 求解，让学生看到解集 = (−3, 1)
  2. 问：什么情况下解集会消失？
  3. 引导发现：两个零点合为一个就没有"之间"了
  4. 回到原题，移项通分，令零点重合
  5. 代入验证
- 最适合的具体数值例子：(x + 3)/(x − 1) < 0，零点 −3 和 1，解集 (−3, 1)
- 讲到哪里停下来让学生回答：问"如果分子变成 (x−1)，不等式 (x−1)/(x−1) < 0 的解集是什么？"

## 十、推荐练题任务包

- 若学生卡在 A/B 档，出什么题：给定 (x + 2)/(x − 3) > 0，求解集（练习找零点+写区间）
- 若学生卡在 C 档，出什么题：已知 (x + a)/(x − 2) < 0 的解集为空，求 a（同结构，数字更简单）
- 若学生卡在 D 档，出什么题：已知 (2x + a)/(x + 1) > 0 的解集为 R，求 a（反向：解集为全集）
- 若学生达到 E/F 档，如何变式：已知 (x² + ax − 1)/(x − 1) < 0 的解集为空，求 a 的取值范围（二次分子 + 判别式）
- 禁止出的跑偏变式：含绝对值的分式不等式、需要分3段以上讨论的不等式
