# 结构分析：余弦函数为奇函数求参数

## 原题
若 y = cos(x + φ) 为奇函数，求 sin φ 的值。

## 一、题目场景
- 数学对象：三角函数 cos(x + φ) 的奇偶性
- 变量/参数：x 为自变量，φ 为待定参数
- 函数/图形：f(x) = cos(x + φ)
- 已知条件：f(x) 为奇函数
- 要求目标：求 sin φ 的值

## 二、核心结构
- 表面考点：三角函数的奇偶性判断
- 本质考点：奇函数定义的代数展开——f(-x) = -f(x) 配合和角公式拆分偶部与奇部，令偶部系数为零
- 一句话问题模式：一个含参数的三角函数具有特定奇偶性，通过展开并分离偶部/奇部来确定参数。

## 三、关键转化
- 最关键的转化：cos(x + φ) = cos x cos φ - sin x sin φ，分离出偶函数部分 cos x cos φ 和奇函数部分 -sin x sin φ
- 为什么降低计算量：利用 cos x 为偶函数、sin x 为奇函数的性质，直接读出偶部系数必须为零，一步得到 cos φ = 0
- 不转化时的低效路径：代入 f(-x) = -f(x) 后展开两边再整理，虽然也能做，但步骤多、容易在符号上出错

## 四、标准路径骨架
1. 先做什么：写出奇函数定义 f(-x) = -f(x)
2. 再做什么：展开 cos(x + φ) 的和角公式
3. 建立什么关系：识别展开式中哪些项是偶函数（cos x 乘常数）、哪些是奇函数（sin x 乘常数）
4. 如何求解：偶部系数为零 → cos φ = 0 → φ = π/2 + kπ → sin φ = ±1
5. 需要检查什么：sin φ 是否有遗漏（φ = π/2 + kπ 时 sin φ 交替为 1 和 -1，两个值都合法）

## 四点五、标准完整解与验算
- 关键交点/关键量：φ 的值，sin φ 的值
- 面积/方程/关系式：cos φ = 0
- 完整求解过程：
  - 由奇函数定义：f(-x) = -f(x)，即 f(-x) + f(x) = 0
  - 展开和角公式：cos(x + φ) = cos x cos φ - sin x sin φ
  - cos x cos φ 是偶函数部分，-sin x sin φ 是奇函数部分
  - 偶函数 + 奇函数 = 奇函数，则偶函数部分必须为零
  - 所以 cos φ = 0，即 φ = π/2 + kπ（k ∈ Z）
  - 当 φ = π/2 + 2kπ 时，sin φ = sin(π/2) = 1
  - 当 φ = π/2 + (2k+1)π 时，sin φ = sin(3π/2) = -1
  - 综合得 sin φ = ±1
- 最终答案：sin φ = ±1
- 排除值：无排除值，cos φ = 0 的所有解都合法
- 退化情形：无退化情形
- 验算：
  - 取 φ = π/2：y = cos(x + π/2) = -sin x，确实是奇函数。
  - 取 φ = -π/2：y = cos(x - π/2) = sin x，确实是奇函数。
  - 取 φ = 3π/2：y = cos(x + 3π/2) = sin x，确实是奇函数。
- 本题最短可靠路径：展开 cos(x + φ) → 分离偶部奇部 → 偶部系数为零 → cos φ = 0 → sin φ = ±1

## 五、出题人逻辑
- 诱导学生硬算的位置：可能从 f(-x) = -f(x) 出发，两边都展开成 cos 项，然后化简，步骤多且容易符号出错
- 真正的捷径：利用和角公式直接分离偶部和奇部，"偶部归零"一步出 cos φ = 0
- 训练的可迁移能力：(1) 三角和角公式的展开能力；(2) 函数奇偶性分解为偶部+奇部的结构意识；(3) "要成为纯奇函数，偶部必须消失"的推理模式

## 六、学生卡点预测
- 基础薄弱学生：不知道奇函数的定义 f(-x) = -f(x)；不会展开 cos(x + φ) 的和角公式；不知道 cos x 是偶函数、sin x 是奇函数
- 中等学生：能展开但不会分离偶部和奇部；可能直接用 f(-x) = -f(x) 代入两边然后化简，走了弯路；算出 cos φ = 0 后忘记考虑 k 的奇偶导致漏掉 sin φ = -1
- 较强学生：能快速完成但说不清"偶部为什么必须为零"的严格理由；可能认为 sin φ = 1 是唯一答案而漏掉 -1

## 七、变式原则
- 必须保留：展开和角公式 + 分离偶部/奇部的核心动作
- 可以变化：函数类型（改为 sin(x + φ) 为偶函数）；目标（改为求 φ 的取值集合或 cos φ 的值）；增加约束（如 φ ∈ [0, 2π]）
- 容易跑偏：引入复合函数 f(x) = cos(2x + φ) 的奇偶性（增加周期缩放干扰）；引入多个参数

## 八、计算复杂度预算
- 原题计算层级：一次和角公式展开 + 一次偶部归零 + 一次 cos φ = 0 求解 + sin φ 的计算，极低
- 允许小步上升到：sin(x + φ) 为偶函数（换用正弦展开）；φ 限定在某个区间内求具体值
- 禁止引入的计算负担：复合函数如 cos(2x + φ) 的奇偶性；联立方程组；高次三角方程
- 必须保留的可见支架：偶函数/奇函数的识别、和角公式展开、偶部归零的推理

## 九、推荐讲题任务包
- 适合的学生档位：C 档（会列式但容易操作失误或遗漏）
- 本题讲解目标：掌握"展开和角公式 → 分离偶部/奇部 → 偶部归零"的核心动作
- 不要直接讲的抽象话：不要说"利用奇偶性分解"，直接教"偶函数部分必须消失"
- 必须先问的问题：cos x 是偶函数还是奇函数？sin x 呢？
- 关键讲解顺序：(1) 确认 cos x 是偶函数、sin x 是奇函数 → (2) 展开和角公式 → (3) 识别偶部和奇部 → (4) 偶部归零得 cos φ = 0 → (5) 求 sin φ
- 最适合的具体数值例子：取 φ = π/2，cos(x + π/2) = -sin x，验证是奇函数
- 讲到哪里停下来让学生回答：展开 cos(x + φ) 后，让学生指出哪部分是偶函数、哪部分是奇函数

## 十、推荐练题任务包
- 若学生卡在 A/B 档，出什么题：判断 cos x、sin x、cos 2x 的奇偶性；展开 cos(x + π/3) 并识别偶部和奇部
- 若学生卡在 C 档，出什么题：若 sin(x + φ) 为偶函数，求 cos φ 的值（同结构换函数）
- 若学生卡在 D 档，出什么题：若 cos(x + φ) 为奇函数，且 φ ∈ [0, 2π]，求 φ 的值（加区间约束防漏解）
- 若学生达到 E/F 档，如何变式：若 f(x) = sin(x + φ) + cos(x + φ) 为奇函数，求 tan φ 的值（结构隐藏型：需要合并后分离偶部奇部）
- 禁止出的跑偏变式：cos(2x + φ) 的奇偶性（引入周期缩放干扰）；f(x) = a sin x + b cos x 为奇函数（虽然可以，但换成了待定系数法，偏离本题核心路径）
