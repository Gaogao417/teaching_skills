# 结构分析：斜面与竖直杆的影长

## 原题

一个斜面的底角为 θ = 20°。在水平地面上竖直立一根长为 1 米的杆，其在水平地面上的影子长为 0.5 米。现将一根同样长为 1 米的杆竖直插在斜面上，求这根杆在斜面上的影子长度（精确到 0.01 米）。

## 一、题目场景

- 数学对象：斜面（倾斜平面）、竖直杆、阳光（平行光线）、影子（光线与斜面/地面的交线段）
- 变量/参数：斜面底角 θ = 20°，杆高 h = 1 米，水平影长 l = 0.5 米，斜面影长 s（待求）
- 函数/图形：阳光方向与水平面的夹角 β（仰角），斜面截面三角形
- 已知条件：
  - 斜面底角 θ = 20°
  - 杆长（高）h = 1 米
  - 杆在水平地面上的影子长 0.5 米
  - 第二根杆竖直插在斜面上，影子沿斜面向下
- 要求目标：求第二根杆在斜面上的影子长度 s

## 二、核心结构

- 表面考点：相似三角形、锐角三角函数、影子问题
- 本质考点：**确定阳光方向后，在斜面上建立水平/竖直分量关系，列方程求解影长**
- 一句话问题模式：**已知水平面上杆与影的比例确定阳光仰角，求同一阳光在斜面上的杆影长度**

## 三、关键转化

- 最关键的转化：先用水平面上的影子确定阳光仰角 β（tan β = 杆高/影长），再将斜面上的影子问题分解为水平分量和竖直分量，建立含有 s 的方程。
- 为什么降低计算量：阳光方向一旦确定，斜面上的问题就变成一个关于 s 的简单一次方程，不需要建立复杂的立体几何模型。
- 不转化时的低效路径：试图画三维立体图、建立空间坐标系、或用向量投影来做，增加不必要的复杂度。

## 四、标准路径骨架

1. **先做什么**：由水平面上的杆和影子求阳光仰角 β，tan β = 1/0.5 = 2
2. **再做什么**：画斜面截面图，标注第二根杆的位置、影子方向（沿斜面向下）
3. **建立什么关系**：杆顶发出的光线到达斜面时，光线在水平方向的投影 = 杆底的水平偏移量 + 影子的水平分量；竖直方向的下降量 = 杆高 + 影子沿斜面下降的竖直分量
4. **如何求解**：利用阳光方向的 tan β = 竖直下降量 / 水平投影量 列方程，解 s
5. **需要检查什么**：s > 0，答案合理（应比水平影长 0.5 米略长，因为斜面"帮"了杆的忙），精确到 0.01 米

## 四点五、标准完整解与验算

### 关键交点/关键量

- 阳光仰角 β：tan β = 1/0.5 = 2，β = arctan 2 ≈ 63.43°
- 斜面底角 θ = 20°
- 斜面上影子长度 s（待求）
- 影子的水平分量：s cos 20°
- 影子的竖直分量：s sin 20°
- 杆顶到影末的总竖直下降：1 + s sin 20°
- 杆顶到影末的总水平距离：s cos 20°

### 面积/方程/关系式

阳光方向的约束（tan β = 竖直下降 / 水平距离）：

$$\frac{1 + s\sin 20°}{s\cos 20°} = 2$$

### 完整求解过程

展开方程：

$$1 + s\sin 20° = 2s\cos 20°$$

$$1 = 2s\cos 20° - s\sin 20°$$

$$1 = s(2\cos 20° - \sin 20°)$$

$$s = \frac{1}{2\cos 20° - \sin 20°}$$

代入数值：cos 20° ≈ 0.9397，sin 20° ≈ 0.3420

$$s = \frac{1}{2 \times 0.9397 - 0.3420} = \frac{1}{1.8794 - 0.3420} = \frac{1}{1.5374} \approx 0.65046$$

精确到 0.01 米：s ≈ 0.65 米。

### 最终答案

s ≈ 0.65 米

### 排除值

无排除值。s = 1/(2cos20° − sin20°) > 0 因为 2cos20° ≈ 1.8794 > sin20° ≈ 0.3420。

### 退化情形

- 若 θ = 0°（水平面），s = 1/2 = 0.5 米，与水平面上的影子一致。
- 若 θ 增大到使 2cosθ = sinθ，即 tanθ = 2，此时 s → +∞，光线平行于斜面，杆在斜面上无有限影长。
- 若 θ > arctan 2，光线从杆顶打向斜面上方，影子方向反转。

### 验算

- tan β = 2，β ≈ 63.43°。杆高 1 米、水平影 0.5 米：1/0.5 = 2 ✓
- s ≈ 0.6505，水平分量 = 0.6505 × cos20° ≈ 0.6505 × 0.9397 ≈ 0.6114 米
- 竖直下降 = 1 + 0.6505 × sin20° ≈ 1 + 0.6505 × 0.3420 ≈ 1.2225 米
- tan β = 1.2225 / 0.6114 ≈ 1.9995 ≈ 2 ✓

### 本题最短可靠路径

水平影子 → tan β = 2 → 画斜面图分解 s 的水平/竖直分量 → 列方程 (1+s sinθ)/(s cosθ) = 2 → 解 s → 取近似值。

## 五、出题人逻辑

- 诱导学生硬算的位置：可能试图建立三维坐标系或用复杂的空间向量运算。
- 真正的捷径：这道题本质是二维截面问题，只需在同一竖直平面内分析阳光方向和斜面截面，用分解水平/竖直分量的方式列一个简单方程。
- 训练的可迁移能力：
  1. 从已知影子信息反推阳光方向
  2. 在斜面上分解向量的水平/竖直分量
  3. 建立比例关系列方程

## 六、学生卡点预测

- 基础薄弱学生：
  - 看不懂题目场景：不知道"竖直插在斜面上"是什么意思，画不出截面图
  - 不会由水平影长反推阳光方向（tan β = 杆高/影长）
- 中等学生：
  - 能确定阳光方向，但不知道如何在斜面上分解影子的水平分量和竖直分量
  - 分解正确但列方程时把分子分母搞反（把水平/竖直比等于 tan β 而非竖直/水平比）
  - 算不出精确度（不知道四舍五入到 0.01）
- 较强学生：
  - 能解出答案但不会考虑退化情形（θ 变大时光线可能平行于斜面）
  - 可能忽略对答案合理性的检验（s 应该比 0.5 大因为斜面帮杆分担了部分"遮挡"）

## 七、变式原则

- 必须保留：阳光方向确定（由水平影长反推）、斜面上的杆影分解、tan β 的比例关系
- 可以变化：斜面底角 θ 取不同值、杆高不同、水平影长不同、影子方向改为沿斜面向上（θ 足够大时）
- 容易跑偏：引入多个斜面叠加、引入非竖直的杆、要求求光线与斜面的夹角等偏题

## 八、计算复杂度预算

- 原题计算层级：一个一次方程、代入 cos/sin 的数值、除法取近似。整体为低复杂度。
- 允许小步上升到：θ 为特殊角（如 30°、45°）使计算更简洁、令杆高为参数 a 而非具体数值
- 禁止引入的计算负担：需要解二次方程的情况、复合角的三角函数（如 cos(2θ+β)）、多次迭代的数值计算
- 必须保留的可见支架：tan β = 对边/邻边的定义、向量分解（水平分量 = s cosθ，竖直分量 = s sinθ）

## 九、推荐讲题任务包

- 适合的学生档位：B 档（看懂图形但不会找关键量）至 C 档（会找关键量但不会选关系）
- 本题讲解目标：让学生掌握"先定阳光方向，再在斜面上分解分量，最后用 tan β 列方程"的完整链条
- 不要直接讲的抽象话："利用相似三角形的比例关系"；应改为具体动作："阳光的竖直下降量除以水平移动量等于 2"
- 必须先问的问题："杆在水平地面上影子 0.5 米、高 1 米，这个信息能推出什么？"
- 关键讲解顺序：
  1. 先由水平影子求阳光方向 tan β = 2
  2. 画斜面截面图，标出杆的位置和影子的方向
  3. 让学生分解影子 s 的水平分量和竖直分量
  4. 列方程并求解
  5. 检验答案合理性
- 最适合的具体数值例子：先问"如果地面是平的（θ = 0°），影子多长？"答 0.5 米；再问"如果斜面很陡，影子会变长还是变短？"让学生直觉判断
- 讲到哪里停下来让学生回答：在"分解影子的水平分量和竖直分量"这一步，让学生写出 s cosθ 和 s sinθ 分别代表什么；在"列方程"之前，让学生说出"竖直下降量除以水平距离等于什么"

## 十、推荐练题任务包

- 若学生卡在 A/B 档，出什么题：
  - 一根高 2 米的杆在水平面上影子长 1 米，求阳光仰角 β（答案：tan β = 2）
  - 画一根竖直杆和一个斜面的截面图，标出斜面底角和杆的位置
- 若学生卡在 C 档，出什么题：
  - 已知阳光仰角的 tan β = 3，杆高 1 米竖直插在 θ = 30° 的斜面上，求斜面影长（答案：s = 1/(3cos30° − sin30°) = 1/(3×√3/2 − 1/2) = 1/(2.598 − 0.5) ≈ 0.48 米）
- 若学生卡在 D 档，出什么题：
  - 原题中若 θ = arctan 2 ≈ 63.43°，解释为什么影子长度趋近无穷大
- 若学生达到 E/F 档，如何变式：
  - 杆不竖直而是垂直于斜面（即杆沿法线方向），杆长 1 米，求影子在斜面上的长度
- 禁止出的跑偏变式：
  - 引入多个光源或非平行光
  - 杆在曲面上（如球面）的影子
  - 要求计算影子的面积而非长度
