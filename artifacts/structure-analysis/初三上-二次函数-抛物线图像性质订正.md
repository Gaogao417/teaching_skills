# 结构分析：抛物线基础题订正

## 原题
图片中第 11-20 题，主题为二次函数图像、顶点式、对称轴、象限、交点和平移。第 14 题按图中抛物线与 x 轴交于 -3 和 1 处理。

## 一、题目场景
- 数学对象：二次函数与抛物线。
- 变量/参数：x, y, m, k, a, b, c。
- 函数/图形：顶点式、一般式、图像平移、x 轴交点。
- 已知条件：对称轴、顶点、开口方向、图像上的点、图像截距或平移方向。
- 要求目标：求参数、表达式、取值范围、交点、顶点和对称轴。

## 二、核心结构
- 表面考点：二次函数图像性质。
- 本质考点：把图像条件翻译成顶点、对称轴、零点或平移公式。
- 一句话问题模式：看到抛物线条件，先判断用顶点式、对称轴公式、代点还是平移替换。

## 三、关键转化
- 最关键的转化：顶点式 y=a(x-h)^2+k 的顶点是 (h,k)，对称轴是 x=h。
- 为什么降低计算量：避免把所有式子先展开，再从一般式重新配方。
- 不转化时的低效路径：盲目展开、混淆 x 方向平移时的 x 替换、把 y>0 的端点误写成闭区间。

## 四、标准路径骨架
1. 先做什么：识别题型属于顶点、对称轴、交点、象限还是平移。
2. 再做什么：选用对应公式。
3. 建立什么关系：把题目条件列成等式或不等式。
4. 如何求解：代入、比较或解方程。
5. 需要检查什么：开口方向、严格不等号、端点是否包含、平移方向是否反向替换。

## 四点五、标准完整解与验算
- 关键交点/关键量：对称轴 x=h；顶点 (h,k)；零点由 y=0 求。
- 面积/方程/关系式：本页无面积关系，主要是图像性质关系。
- 完整求解过程：
  - 11. 若 y=(x+m)^2+m^2-5，对称轴 x=-m，给 x=1，所以 m=-1。
  - 12. y=(x-k)^2+2-k 的顶点为 (k,2-k)，在第四象限需 k>0 且 2-k<0，所以 k>2。
  - 13. 与 y=-2x^2 形状大小、开口方向相同，a=-2，顶点 (3,-2)，所以 y=-2(x-3)^2-2。
  - 14. 图像与 x 轴交于 -3 和 1，且开口向下，y>0 在两根之间，所以 -3<x<1。
  - 15. y=-x^2-2x+m 的顶点横坐标为 -1，顶点纵坐标为 m+1；在 y=2x 上，纵坐标为 -2，所以 m=-3。
  - 16. a>0,b<0,c=0 时，y=x(ax+b)，两个零点为 0 和 -b/a>0，图像经过第一、第二、第四象限，不经过第三象限。
  - 17. -x^2-2x+3=0，即 x^2+2x-3=0，得 x=1 或 x=-3，交点为 (1,0),(-3,0)。
  - 18. y1=a+b+c，y2=a-b+c，所以 y1-y2=2b。若 y1-y2=-2，则 b=-1。
  - 19(1). y=2(x+1)^2+5，开口向上，对称轴 x=-1，顶点 (-1,5)。
  - 19(2). y=4x^2-2x+1，开口向上，对称轴 x=1/4，顶点 (1/4,3/4)。
  - 20. 向右平移 4 个单位、再向下平移 5 个单位，得 y=-1/2(x-4)^2+3(x-4)+1/2-5=-1/2x^2+7x-49/2。
- 最终答案：见完整求解过程。
- 排除值：第 14 题 y>0 时端点不包含；第 12 题 k=2 不在第四象限。
- 退化情形：本页题目未涉及 a=0 的退化二次函数。
- 验算：第 20 题原顶点 (3,5)，平移后为 (7,0)，新式 y=-1/2(x-7)^2，等价于 -1/2x^2+7x-49/2。
- 本题最短可靠路径：顶点式直接读图像性质；一般式用公式或代点；平移用 x 替换和整体加减。

## 五、出题人逻辑
- 诱导学生硬算的位置：把顶点式展开后再求顶点；把 y>0 区间写成闭区间。
- 真正的捷径：顶点式直接读顶点与对称轴。
- 训练的可迁移能力：图像条件与代数式之间互译。

## 六、学生卡点预测
- 基础薄弱学生：记不住顶点式顶点，左右平移替换方向混乱。
- 中等学生：会算但容易漏掉严格不等号或象限条件。
- 较强学生：可能在一般式顶点纵坐标和配方验算上粗心。

## 七、变式原则
- 必须保留：顶点式、对称轴、图像平移、交点区间判断。
- 可以变化：参数字母、顶点坐标、平移距离、开口系数。
- 容易跑偏：引入复杂根式、分式参数或超出本页目标的综合压轴题。

## 八、计算复杂度预算
- 原题计算层级：初中二次函数基础性质。
- 允许小步上升到：含一个参数的顶点或交点判断。
- 禁止引入的计算负担：高次方程、复杂判别式讨论。
- 必须保留的可见支架：顶点式模板、对称轴公式、平移规则。

## 九、推荐讲题任务包
- 适合的学生档位：默认预测为基础到中等。
- 本题讲解目标：先会从式子读图像，再会由图像写式子。
- 不要直接讲的抽象话：不要只说“数形结合”而不落到顶点、轴、零点。
- 必须先问的问题：这个式子是顶点式还是一般式？
- 关键讲解顺序：顶点式读信息 -> 一般式转信息 -> 图像区间 -> 平移。
- 最适合的具体数值例子：y=-2(x-3)^2-2。
- 讲到哪里停下来让学生回答：读出顶点和开口后，让学生自己补表达式常数项。

## 十、推荐练题任务包
- 若学生卡在 A/B 档，出什么题：只读顶点式的顶点、对称轴、开口。
- 若学生卡在 C 档，出什么题：一般式求顶点和 x 轴交点。
- 若学生卡在 D 档，出什么题：由顶点、开口系数写解析式。
- 若学生达到 E/F 档，如何变式：加入简单参数范围和图像平移组合。
- 禁止出的跑偏变式：复杂判别式分类讨论。
