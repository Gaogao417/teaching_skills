## 1. 小于 pi 的正整数集合

答案：`{1,2,3}`

解析：正整数从 1 开始。因为 `3 < pi < 4`，所以小于 `pi` 的正整数只有 `1,2,3`。  
易错点：不要把 `0` 放进“正整数”集合。

## 2. `y = ln(x - 1)` 的定义域

答案：`(1,+∞)`

解析：对数函数要求真数大于 0，所以 `x - 1 > 0`，得 `x > 1`。  
因此定义域为 `(1,+∞)`。

## 3. 已知 `sin alpha = 1/3`，求 `cos 2alpha`

答案：`7/9`

解析：用二倍角公式：

```text
cos 2alpha = 1 - 2 sin^2 alpha
            = 1 - 2 * (1/3)^2
            = 1 - 2/9
            = 7/9
```

这里不需要判断 `alpha` 的象限，因为公式只用到了 `sin^2 alpha`。

## 4. `y = a^x` 与 `y = 3^x` 关于 y 轴对称

答案：`1/3`

解析：关于 `y` 轴对称，就是把 `x` 换成 `-x`。  
`y = 3^x` 关于 `y` 轴的对称图像为：

```text
y = 3^(-x) = (1/3)^x
```

所以 `a = 1/3`。

## 5. `y = tan(2x + 1)` 的最小正周期

答案：`pi/2`

解析：`tan x` 的周期是 `pi`。当自变量变成 `2x + 1` 时，只看 `x` 的系数 `2`：

```text
T = pi / |2| = pi/2
```

平移量 `+1` 不影响周期。

## 6. 若 `y = cos(x + phi)` 为奇函数，求 `sin phi`

答案：`±1`

解析：展开：

```text
cos(x + phi) = cos x cos phi - sin x sin phi
```

其中 `cos x cos phi` 是偶函数部分，`-sin x sin phi` 是奇函数部分。  
整个函数要成为奇函数，偶函数部分必须消失，所以：

```text
cos phi = 0
```

因此 `sin phi = ±1`。

## 7. 扇形周长为 6，面积为 2，求圆心角弧度数

答案：`1 或 4`

解析：设半径为 `r`，弧长为 `l`，圆心角为 `theta`。  
扇形周长：

```text
2r + l = 6
```

扇形面积：

```text
(1/2)rl = 2
```

所以：

```text
rl = 4
l = 4/r
```

代入周长：

```text
2r + 4/r = 6
2r^2 - 6r + 4 = 0
r^2 - 3r + 2 = 0
(r - 1)(r - 2) = 0
```

所以 `r = 1` 或 `r = 2`。又因为 `l = r theta`，即 `theta = l/r = 4/r^2`。  
当 `r = 1` 时，`theta = 4`；当 `r = 2` 时，`theta = 1`。

## 8. 幂函数 `y = x^s` 在 `x > 1` 时位于 `y = x` 下方

答案：`(-∞,1)`

解析：题意是对所有 `x > 1`，都有：

```text
x^s < x
```

因为 `x > 1`，两边除以 `x` 得：

```text
x^(s - 1) < 1
```

当 `x > 1` 时，指数小于 0 才能保证 `x^(s - 1) < 1` 恒成立，所以：

```text
s - 1 < 0
s < 1
```

因此 `s` 的取值范围是 `(-∞,1)`。

## 9. 不等式 `(2x + a)/(x - 1) < 1` 的解集为空

答案：`-2`

解析：先移项：

```text
(2x + a)/(x - 1) - 1 < 0
```

通分：

```text
(2x + a - x + 1)/(x - 1) < 0
(x + a + 1)/(x - 1) < 0
```

如果分子零点和分母零点不同，这个分式一定会在两个零点之间取负值，解集不会为空。  
所以要让解集为空，必须让分子分母的零点重合：

```text
x + a + 1 = x - 1
a + 1 = -1
a = -2
```

此时原式变成：

```text
(2x - 2)/(x - 1) < 1
```

在 `x != 1` 时左边恒为 `2`，所以 `2 < 1` 不成立，解集为空。

## 10. 正实数 `a,b` 满足 `a + 1/(2b) = 1`，求 `a/b` 最大值

答案：`1/2`

解析：由条件得：

```text
a = 1 - 1/(2b)
```

因为 `a > 0`，所以 `b > 1/2`。于是：

```text
a/b = 1/b - 1/(2b^2)
```

令 `t = 1/b`，则 `0 < t < 2`，问题变成求：

```text
t - t^2/2
```

配方：

```text
t - t^2/2 = 1/2 - (t - 1)^2/2
```

所以最大值为 `1/2`，当 `t = 1`，即 `b = 1`、`a = 1/2` 时取到。

## 11. 斜面与竖直杆的影长问题

答案：`0.65` 米

解析：斜面底角 `theta = 20°`，杆长为 `1` 米。

先利用水平面上的影子确定阳光方向。影子在水平面上长 `0.5` 米，杆高 `1` 米，所以阳光与水平面的夹角满足：

```text
tan beta = 1 / 0.5 = 2
```

也就是说，光线每水平前进 `1`，竖直下降 `2`。

再设另一根杆在斜面上的影长为 `s`。影子沿斜面向下，则它的水平分量是：

```text
s cos 20°
```

竖直下降量除了杆本身的 `1` 米，还包括沿斜面下降的：

```text
s sin 20°
```

所以光线斜率关系为：

```text
(1 + s sin 20°) / (s cos 20°) = 2
```

整理：

```text
1 + s sin 20° = 2s cos 20°
s(2cos20° - sin20°) = 1
s = 1 / (2cos20° - sin20°)
```

计算得：

```text
s ≈ 0.65046
```

结果精确到 `0.01` 米，所以影长为 `0.65` 米。

## 12. `y = sin pi x` 左移 `m` 后，相邻三个交点形成正三角形

答案：`1/3 或 5/3`

解析：左移 `m` 个单位后：

```text
f(x) = sin pi(x + m)
```

两个图像交点满足：

```text
sin pi x = sin pi(x + m)
```

由 `sin A = sin B`，主要得到：

```text
pi x = pi - pi(x + m) + 2kpi
```

化简：

```text
2x + m = 1 + 2k
x = (1 - m)/2 + k
```

所以相邻交点的横坐标相差 `1`。设相邻三个交点为：

```text
P0 = (x0, y)
P1 = (x0 + 1, -y)
P2 = (x0 + 2, y)
```

因为 `sin pi(x + 1) = -sin pi x`，所以纵坐标会正负交替。

若这三个点构成正三角形，则底边 `P0P2` 长为 `2`，两腰也应为 `2`：

```text
P0P1 = sqrt(1^2 + (2y)^2) = 2
```

平方得：

```text
1 + 4y^2 = 4
y^2 = 3/4
|y| = sqrt(3)/2
```

而：

```text
y = sin(pi * (1 - m)/2)
```

由于 `0 < m < 2`，所以 `(1 - m)/2` 在 `(-1/2, 1/2)` 内。要使正弦绝对值为 `sqrt(3)/2`，只能有：

```text
(1 - m)/2 = 1/3   或   (1 - m)/2 = -1/3
```

解得：

```text
m = 1/3   或   m = 5/3
```
